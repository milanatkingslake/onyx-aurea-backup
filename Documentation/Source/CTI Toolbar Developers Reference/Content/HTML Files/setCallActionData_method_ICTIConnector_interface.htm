setCallActionData method

The setCallActionData method of the ICTIConnector interface sets the record display instructions (the ActionData XML structure) and attaches the instructions to the identified call. The attached instructions define automated OEP record actions to perform when the call is transferred or conferenced.

All methods of the ICTIConnector interface are commented and explained in the sample connector code file. Full path: YourOEPwebsite\cti\test\cticonnector.js where YourOEPwebsite is your OEP installation directory.

Syntax (JavaScript)

function setCallActionData(

psCallId,

psCallActionDataXml)

View JavaScript parameter descriptions

psCallId

[in] (String data type) The ID of the call for attachment of the ActionData.

psCallActionDataXml

[in] (XML String) ActionData XML structure.

Return values

This method returns a return code enumeration value corresponding to the status of the method call.

Return code enumeration values are listed in the returnCodeEnums section of the enumeration file.
