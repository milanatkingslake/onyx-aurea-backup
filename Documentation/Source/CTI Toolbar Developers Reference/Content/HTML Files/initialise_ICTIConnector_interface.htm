initialise method

The initialise method of the ICTIConnector interface instantiates and initializes all necessary CTI system components (unless the instantiation and initialization has already been performed). The method also assigns the poCtiConnection reference passed to a local variable, which enables the ICTIConnector interface to perform callbacks into Onyx CTI.

All methods of the ICTIConnector interface are commented and explained in the sample connector code file. Full path: YourOEPwebsite\cti\test\cticonnector.js where YourOEPwebsite is your OEP installation directory.

Syntax (JavaScript)

function initialise(poCtiConnection)

View JavaScript parameter descriptions

poCtiConnection

[in] (CtiConnectionObject data type) A reference to the parent CtiConnection class that calls this method.  The connector object holds a reference to the parent CtiConnection, so it can call the event methods on the CtiConnection object.

Return values

This method returns a return code enumeration value corresponding to the status of the method call.

Return code enumeration values are listed in the returnCodeEnums section of the enumeration file.
