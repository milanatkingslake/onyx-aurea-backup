ctiFrequentPhones object

The ctiFrequentPhones represents a list of frequent phone records.

Methods

The ctiFrequentPhones object has two methods, retrieveCollection and saveCollection, which behave as standard OEAS retrieveCollection and saveCollection methods. For details on the standard OEAS retrieveCollection and saveCollection methods, see the OEAS Technical Reference.

Properties

Property Description

active

Indicates active status; only active objects are displayed in the Onyx CTI interface. For details on this property, see the OEAS Technical Reference.

connectorId

Identifies the connector associated with the identified phone number.

description

Describes the listed phone number.

insertBy

Creator of the record. For details on this property, see the OEAS Technical Reference.

insertDate

Date of creation. For details on this property, see the OEAS Technical Reference.

phoneNumber

Lists the phone number (string).

primaryId

Unique identifier assigned to the object when it is first saved to the database.

updateBy

Modifier of the record. For details on this property, see the OEAS Technical Reference.

updateDate

Date of modification. For details on this property, see the OEAS Technical Reference.
