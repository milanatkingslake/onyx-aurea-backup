Resource string files

All connectors use one set of resource string files to define the resource string IDs and the text (strings) associated with the supported enumeration values.

Text strings associated with this enumeration value Appear in this area of the Onyx CTI interface

Agent state enumeration value

Agent status display in the Onyx CTI toolbar

Call state enumeration value

Active call selection list in the Onyx CTI toolbar

Return code enumeration value

Error message window

The resource string files must include any unique resource strings for your connector to ensure that Onyx CTI can display the correct information to the users. We recommend that you use copies of the default resource string files as your starting point when building a new connector.

To confirm that your changes appear as expected, delete the files in Internet Explorer's Temporary Internet Files folder.

Full paths of the default resource string files, where YourOEPwebsite is your OEP installation directory and language is the language-specific folder (such as "eng"):

  • YourOEPwebsite\language\cti\ctitoolbar.xml
  • YourOEPwebsite\language\cti\ctidialogs.xml

Example (ctidialogs.xml file):

<strings>

   <!-- Error strings -->

   <string name="ctiError" text="Error"/>

   <string name="ctiWarning" text="Warning"/>

   ...

</strings>

The following table describes the elements of the resource string files:

Element Description

strings

Root element. Contains the collection of string elements.

string

Contains the definition information for the identified resource string.

The following table describes the attributes of the resource string files:

Attribute Description

name

Identifies the resource string (lists the resource string ID).

When modifying or creating resource string IDs for agent states, call states, or return codes, ensure that the resource strings are also represented in the enumeration file (messageName attribute). The enumeration file defines the enumerations supported by your custom connector.

text

Specifies the text string to be displayed in the Onyx CTI interface for this resource string.
