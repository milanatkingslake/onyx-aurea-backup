ctiConnector object

The ctiConnector object contains domain data about a CTI connector. (A CTI connector connects Onyx CTI to a CTI system.)

Methods

The ctiConnector object has two methods, retrieveCollection and saveCollection, which behave as standard OEAS retrieveCollection and saveCollection methods. For details on the standard OEAS retrieveCollection and saveCollection methods, see the OEAS Technical Reference.

Properties

The ctiConnector object has one key property (primaryId) that uniquely identifies the connector in the database.

Property Description

active

Indicates active status; only active objects are displayed in the Onyx CTI interface. For details on this property, see the OEAS Technical Reference.

className

The name of the JavaScript class that implements the ICTIConnector Interface.

connectorFolder

The name of the custom connector folder that contains the connector files.

description

Description of the connector.

insertBy

Creator of the record. For details on this property, see the OEAS Technical Reference.

insertDate

Date of creation. For details on this property, see the OEAS Technical Reference.

primaryId

Primary ID.

updateBy

Modifier of the record. For details on this property, see the OEAS Technical Reference.

updateDate

Date of modification. For details on this property, see the OEAS Technical Reference.
