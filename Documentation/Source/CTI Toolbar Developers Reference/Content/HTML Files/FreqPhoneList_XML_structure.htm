FreqPhoneList XML structure

The FreqPhoneList XML structure contains the list of phone numbers frequently used for placing and transferring calls.

If the CTI system supports frequently used numbers, the connector must return this structure in response to the getFreqPhoneList method.

Example:

<freqPhoneList>

   <freqPhone number="18001234567" Name="Shipping Status Hotline" />

   <freqPhone number="18001112222" Name="RMA Status Hotline" />

   <freqPhone number="18883334444" Name="Service Repair Status Hotline" />

</freqPhoneList>

The following table describes the elements of the FreqPhoneList XML:

Element Description

freqPhoneList

Root element. Contains the collection of freqPhone elements.

freqPhone

Contains the data for the identified frequently used phone number.

The following table describes the attributes of the FreqPhoneList XML:

Attribute Description

number

Specifies the frequently used phone number.

name

Specifies the name of the frequently used phone number for display in the Onyx CTI interface.
