conferenceCallCold method

The conferenceCallCold method of the ICTIConnector interface performs a one-step cold conference, in which the incoming call is not put on hold while the conference call is made to a third party. When the third party answers the call, a three-way conference call is established among the agent, original incoming caller, and the third party.

All methods of the ICTIConnector interface are commented and explained in the sample connector code file. Full path: YourOEPwebsite\cti\test\cticonnector.js where YourOEPwebsite is your OEP installation directory.

Syntax (JavaScript)

function conferenceCallCold(

psCallId,

psDestination)

View JavaScript parameter descriptions

psCallId

[in] (String data type) The ID of the call to conference.

psDestination

[in] (String data type) Destination of the call. Contains an Agent ID for internal calls; contains a PSTN number for external calls.

Return values

This method returns a return code enumeration value corresponding to the status of the method call.

Return code enumeration values are listed in the returnCodeEnums section of the enumeration file.
