ctiAgentAssociation object

The ctiAgentAssociation object lists connector associations with agents (by both Onyx user ID and CTI system user ID). The ctiAgentAssociation object is intended for use with CTI systems that use multiple connectors.

The cti_agent_association table is programmatically maintained with this object. Onyx CTI checks this table for existence of association records when OEP is loaded. If an association record exists for the current agent, Onyx CTI loads the specified connector. If no association records exist for the current agent, Onyx CTI loads the default connector specified in the main configuration file (full path: YourOEPwebsite\cti\cti_config.xml).

Methods

The ctiAgentAssociation object has two methods, retrieveCollection and saveCollection, which behave as standard OEAS retrieveCollection and saveCollection methods. For details on the standard OEAS retrieveCollection and saveCollection methods, see the OEAS Technical Reference.

Properties

The ctiAgentAssociation object does not have a primaryId property. The userId and connectorId properties are used as key properties for unique identification.

Property Description

active

Indicates active status; only active objects are displayed in the Onyx CTI interface. For details on this property, see the OEAS Technical Reference.

agentId

Identifies the agent (CTI system user ID) associated with the identified Onyx user and connector.

connectorId

Identifies the connector associated with the identified agent and Onyx user.

insertBy

Creator of the record. For details on this property, see the OEAS Technical Reference.

insertDate

Date of creation. For details on this property, see the OEAS Technical Reference.

stationId

Identifies the station at which the agent is located.

updateBy

Modifier of the record. For details on this property, see the OEAS Technical Reference.

updateDate

Date of modification. For details on this property, see the OEAS Technical Reference.

userId

Identifies the Onyx user associated with the identified agent and connector.
