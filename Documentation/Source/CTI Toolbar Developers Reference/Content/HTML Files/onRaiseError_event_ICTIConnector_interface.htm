onRaiseError event

The onRaiseError event of the ICTIConnector interface notifies Onyx CTI when an error has occurred in the connector or the CTI system.

Syntax (JavaScript)

onRaiseError(piReturnCode)

View JavaScript parameter descriptions

piReturnCode

[in] (CtiReturnCode) A return code indicating success or failure.  

Return values

Events of the ICTIConnector interface do not return any values.
