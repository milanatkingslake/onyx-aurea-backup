StatusDetails XML structure

The StatusDetails XML structure contains detailed status information about the CTI system.

The connector must return this structure in response to the getStatusDetail method.

Example:

<detailInfo>

   <detail caption="Sample Data" data="See Below" />

   <detail caption="Calls in Queue" data="5" />

   <detail caption="Wait Time" data="2:21" />

   <detail caption="Call Time" data="5:43" />

   <detail caption="Customer Phone Number" data="555-555-5555" />

   <detail caption="Customer Incident ID" data="12345" />

   <detail caption="Customer ID" data="123456789" />

</detailInfo>

The following table describes the elements of the StatusDetails XML:

Element Description

detailInfo

Root element. Contains the collection of detail elements.

detail

Contains the status details data for the CTI system.

The following table describes the attributes of the StatusDetails XML:

Attribute Description

caption

Specifies the caption of the status detail data for display in the Onyx CTI interface.

data

Specifies the status detail data.
