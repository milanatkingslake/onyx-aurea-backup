StatusSummary XML structure

The StatusDetails XML structure contains summary status information about the CTI system.

The connector must return this structure in response to the getStatusSummary method.

Example:

<summaryInfo>

   <summary caption="Calls in Queue" data="5" />

   <summary caption="Wait Time" data="2:21" />

   <summary caption="Agents Logged In" data="43" />

</summaryInfo>

The following table describes the elements of the StatusSummary XML:

Element Description

summaryinfo

Root element. Contains the collection of detail elements.

detail

Contains the status summary data for the CTI system.

The following table describes the attributes of the StatusSummary XML:

Attribute Description

caption

Specifies the caption of the status summary data for display in the Onyx CTI interface.

data

Contains the status summary data.
