getCalls method

The getCalls method of the ICTIConnector interface retrieves the Calls XML structure containing all calls for the logged-on agent.

All methods of the ICTIConnector interface are commented and explained in the sample connector code file. Full path: YourOEPwebsite\cti\test\cticonnector.js where YourOEPwebsite is your OEP installation directory.

Syntax (JavaScript)

Not applicable. No input parameters.

Return values

This method returns an XML string representing the Calls XML structure.
