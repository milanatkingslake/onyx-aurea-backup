Onyx CTI Toolbar

This topic describes the Onyx CTI toolbar displayed in OEP after the installation of Onyx CTI. The Onyx CTI toolbar displays current agent and system status and includes buttons and an active call selection list. You can configure the state and availability of selected toolbar buttons by editing the configuration file for your connector.

For instructions on installing Onyx CTI and configuring toolbar access, see the Onyx Computer Telephony Integration Implementation Guide.

The active call selection list is a drop-down list of all current phone calls received or placed by the agent. Each phone call includes the call state and the phone number. Agent phone numbers may be represented by agent IDs. Agents can select another phone call on the list to activate that phone call.

Examples of information included in the active call selection list:

ACD - 111

Outbound - 18001234567

Status indicated on the Onyx CTI toolbar includes the following:

  • Agent status listing the current agent name, station ID, and state.

    • Example:

    Joe Smith (33) - On Call

  • CTI system status listing the real-time data pre-configured for the current agent and CTI system.

    • Example:

    Calls in queue - 5 Wait time - 60 seconds

The buttons on the Onyx CTI toolbar are described in the following table.

Button Default tooltip Description

Logon

Displays the Log on window for entry of logon credentials to connect the agent to the CTI system. Logon credentials include agent ID, agent PIN, and station ID. (This button also serves as an indicator that the agent is not currently connected.)

Logoff

Logs the agent off the CTI system after the agent confirms the operation. (This button also serves as an indicator that the agent is currently connected.)

Open CTI Test Driver

Displays the test driver interface.

Set Agent Ready

Sets the agent state to Ready.

Set Agent Idle

Sets the agent state to Idle after the agent selects the applicable reason (such as Break or Meeting) from the Idle Reason window.

Idle reasons are supplied by your custom connector. For more information, see IdleReasonList XML structure.

Set Agent Wrap Up

Sets the agent state to Wrap Up.

Place Call

Places a phone call to the specified destination.

Disconnect Current Call

Hangs up the current call.

Hold Current Call

Places the current call on hold.

Resume Current Call

Resumes the current held call.

Transfer Current Call

Transfers the current call to the specified destination.

Conference Current Call

Conferences the current call to the specified destination.

View CTI System Details

Displays the system-specific status in the CTI System Details window. Example status elements: calls in queue, wait time, call time, IVR prompts, customer phone number, customer incident ID, and customer ID.

This status is supplied by your custom connector. For more information, see StatusDetails XML structure.

Refresh Toolbar

Synchronizes the Onyx CTI toolbar with the CTI system.
