getStatusSummary method

The getStatusSummary method of the ICTIConnector interface retrieves the StatusSummary XML structure containing summary information from the CTI system.

This method is typically called in response to the connector firing an onStatusSummaryChange event. That event and the subsequent call to this method allow the CTI system to display CTI system summary information in the toolbar on a recurring basis.

All methods of the ICTIConnector interface are commented and explained in the sample connector code file. Full path: YourOEPwebsite\cti\test\cticonnector.js where YourOEPwebsite is your OEP installation directory.

Syntax (JavaScript)

Not applicable. No input parameters.

Return values

This method returns an XML string representing the StatusSummary XML structure.
