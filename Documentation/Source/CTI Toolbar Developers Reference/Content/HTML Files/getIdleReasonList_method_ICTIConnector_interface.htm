getIdleReasonList method

The getIdleReasonList method of the ICTIConnector interface retrieves the IdleReasonList XML structure that contains the list of reasons agents can select when changing to the Idle state.  

All methods of the ICTIConnector interface are commented and explained in the sample connector code file. Full path: YourOEPwebsite\cti\test\cticonnector.js where YourOEPwebsite is your OEP installation directory.

Syntax (JavaScript)

function getIdleReasonList()

Return values

This method returns an XML string representing the IdleReasonList XML structure.
