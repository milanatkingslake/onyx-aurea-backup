What's New

Onyx CTI has been updated to leverage the new SQL Generation (SQLGen) architecture of the OEAS 6.0 platform. For more information about the SQL generation platform, see the OEAS Technical Reference.

The following changes have been made to the Onyx CTI business objects:

  • Existing methods have been replaced with the saveCollection and retrieveCollection methods as is standard for Onyx 6.0. These methods provide additional functionality for the objects.
  • The ctiAgentAssociation object is now a domain data object (rather than a transactional object) and contains an active property as is standard for Onyx 6.0.
  • The ctiConnector object no longer has the user1..N properties. These properties were removed due to the improved handling of object and table customization in the OEAS 6.0 platform. For more information about customizing objects and tables, see the OEAS Technical Reference.
