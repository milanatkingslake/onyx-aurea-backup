QueueList XML structure

The QueueList XML structure contains the list of available queues to which a call can be transferred.

The connector must return this structure in response to the getQueueList method.

Example:

<queueList>

   <queue queueId="1" queueName="Service Escalations" />

   <queue queueId="2" queueName="Service RMA" />

   <queue queueId="3" queueName="Service Repairs" />

</queueList>

The following table describes the elements of the QueueList XML:

Element Description

queueList

Root element. Contains the collection of queue elements.

queue

Contains the data for the identified available queue.

The following table describes the attributes of the QueueList XML:

Attribute Description

queueId

Specifies the ID of the available queue.

queueName

Specifies the name of the available queue.
