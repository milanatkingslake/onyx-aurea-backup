Glossary

Term

Definition

ACD

Automatic Call Distributor. Queues and distributes calls to agents in a Contact Center environment. Usually integrated with the PBX. Onyx integrates with the ACD through the CTI interface.

Agent

Onyx user who uses a third-party CTI system to respond to incoming calls.

Agent state

The current state of the agent as defined by the CTI system. Typical agent states include Ready, Idle, On Call, Available, and Wrap Up.

Call state

The current state of the agent's call. Typical call states include On Call and On Hold.

CTI connector

Computer Telephony Integration connector. A collection of code that implements the ICTIConnector interface for communication between your CTI system and Onyx CTI. The CTI connector makes calls to Onyx CTI to display OEP records and perform other actions from the Onyx CTI toolbar displayed in OEP.

CTI system

Computer Telephony Integration system. Installed call center solution that enables computers (such as billing systems) to control telephony functions, including voice systems (such as PBX, IVR, and ACD). Examples include Avaya, Aspect, and Cisco.

Idle reason

Explanation for the agent's current Idle status (why the agent is not taking calls). Typically used for reporting purposes.
