onCallDataOffered event

The onCallDataOffered event of the ICTIConnector interface notifies Onyx CTI when call data is offered to the agent.

Syntax (JavaScript)

onCallDataOffered(psCallData)

View JavaScript parameter descriptions

psCallData

[in] (XML string) A string containing the CallData XML structure.

Return values

Events of the ICTIConnector interface do not return any values.
