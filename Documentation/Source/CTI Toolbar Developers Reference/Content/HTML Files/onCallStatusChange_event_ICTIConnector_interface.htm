onCallStatusChange event

The onCallStatusChange event of the ICTIConnector interface notifies Onyx CTI when the status of a call changes.

Syntax (JavaScript)

onCallStatusChange(

piReturnCode

psCallData)

View JavaScript parameter descriptions

piReturnCode

[in] (CtiReturnCode) A return code indicating success or failure.  

psCallData

[in] (XML string) A string containing the CallData XML structure.

Return values

Events of the ICTIConnector interface do not return any values.
