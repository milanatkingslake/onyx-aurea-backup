onRefreshToolbar event

The onRefreshToolbar event of the ICTIConnector interface reloads agent and call information from the CTI system.

Use this event when the toolbar becomes out-of-sync with the CTI system, such as when the CTI system generates unhandled errors.

Syntax (JavaScript)

onRefreshToolbar()

Return values

Events of the ICTIConnector interface do not return any values.
