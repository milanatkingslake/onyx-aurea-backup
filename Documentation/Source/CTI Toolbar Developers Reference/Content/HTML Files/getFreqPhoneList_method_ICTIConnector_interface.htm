getFreqPhoneList method

The getFreqPhoneList method of the ICTIConnector interface retrieves the FreqPhoneList XML structure containing the list of available numbers for placing or transferring calls. Available numbers are the list of frequently dialed numbers that are presented to agents when performing these actions.

If your CTI system does not support the "frequently used numbers" concept, ensure that the getFreqPhoneList method returns an empty string to trigger the appropriate OEAS call from Onyx CTI to retrieve this information from the OEDB.

All methods of the ICTIConnector interface are commented and explained in the sample connector code file. Full path: YourOEPwebsite\cti\test\cticonnector.js where YourOEPwebsite is your OEP installation directory.

Syntax (JavaScript)

function getFreqPhoneList()

Return values

This method returns an XML string representing the FreqPhoneList XML structure.
