AgentList XML structure

The AgentList XML structure contains the list of available agents for transferring and conferencing calls.

The connector must return this structure in response to the getAgentList method.

Example:

<agentList>

   <agent agentId="111" agentName="John Smith" />

   <agent agentId="112" agentName="Mary Jones" />

   <agent agentId="113" agentName="Lyle Miller" />

</agentList>

The following table describes the elements of the AgentList XML:

Element Description

agentList

Root element. Contains the collection of agent elements.

agent

Contains the data for the identified available agent.

The following table describes the attributes of the AgentList XML:

Attribute Description

agentId

Specifies the ID of the available agent.

agentName

Specifies the name of the available agent.
