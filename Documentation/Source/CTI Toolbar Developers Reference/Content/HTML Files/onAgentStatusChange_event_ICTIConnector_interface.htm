onAgentStatusChange event

The onAgentStatusChange event of the ICTIConnector interface notifies Onyx CTI when the status of an agent changes.

Syntax (JavaScript)

onAgentStatusChange(

piReturnCode

psAgentData)

View JavaScript parameter descriptions

piReturnCode

[in] (CtiReturnCode) A return code indicating success or failure.

psAgentData

[in] (XML String) A string containing the AgentData XML structure.

Return values

Events of the ICTIConnector interface do not return any values.
