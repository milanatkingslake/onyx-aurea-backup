AgentData XML structure

The AgentData XML structure contains the details about the identified agent.

The connector must return this XML structure in response to a call to the getAgentData method.

Example:

<root>

   <agent loggedOn=1 agentId=777 stationId=9202 agentName=Todd agentState=2 idleCode=11 />

   ...

</root>

The following table describes the elements of the AgentData XML:

Element Description

root

Root element.

agent

Contains the agent information.

The following table describes the attributes of the AgentData XML:

Attribute Description

loggedOn

Indicates agent logon status. Boolean value: 1 indicates that the agent is logged on. 0 indicates that the agent is not logged on.

agentId

Specifies the agent ID of the logged-on agent.

stationId

Specifies the station ID of the logged-on agent.

agentName

Specifies the name of the logged-on agent.

agentState

Specifies the current state of the agent. Integer value corresponding to an agent state enumeration value.

idleCode

Specifies the Idle Reason code if the agent state is Idle. Integer value corresponding to an Idle Reason code returned by the getIdleReasonList method on the ICtiConnector interface. Empty if the agent is not in the Idle state.
