getAgentList method

The getAgentList method of the ICTIConnector interface retrieves the AgentList XML structure that contains the list of available agents for transferring calls and for conferencing.

All methods of the ICTIConnector interface are commented and explained in the sample connector code file. Full path: YourOEPwebsite\cti\test\cticonnector.js where YourOEPwebsite is your OEP installation directory.

Syntax (JavaScript)

function getAgentList()

Return values

This method returns an XML string representing the AgentList XML structure.
