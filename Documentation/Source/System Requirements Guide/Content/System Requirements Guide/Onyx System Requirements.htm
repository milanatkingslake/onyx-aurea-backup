Onyx System Requirements

The requirements listed in this document provide the requirements for a basic implementation. Hardware requirements depends on various factors, including the number of concurrent users, server activity, network connection speed, and so on. For sizing guidelines tailored to your needs, contact a certified Onyx CRM professional.

This document contains hardware and software system requirements to successfully install all components of the Onyx system. The topology diagram depicts the recommended basic configuration.

Important considerations:

  • Components of the Onyx system can be installed on a single computer or on multiple computers, based on your organization's needs.
  • Businesses that handle large volumes of activity and data transfer may require a network of Web farms with multiple server computers for certain Onyx components.
  • If you intend to install multiple components on one server computer, ensure that you fulfill the hardware and software requirements for each component.
  • For requirements such as disk space and memory, ensure that the cumulative requirements for all components are fulfilled to minimize any potential performance losses.
  • For a complete list of components available in Onyx, see List of Components on page 1.
  • Avolin CRM products are supported on both physical servers and virtualized servers. For more information on virtualization, see Onyx CRM Product Virtualization.
