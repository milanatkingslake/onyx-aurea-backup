Email Accelerator Server

The following requirements apply to Email Accelerator server computers. If all system requirements are fulfilled, the Email Accelerator server can be installed on the same server computer as Processing Server (OPS) or OEAS.

  • Email Accelerator Internal Server
  • Email Accelerator External Server

Email Accelerator Internal Server

The following requirements apply to a computer that hosts the internal server for Email Accelerator, as depicted in the topology diagram. The internal server is also called the "Email Accelerator Client Server."

Hardware Value

Memory

1GB-4GB depending on the number of concurrent users. Because this application sends bulk mail, the memory usage depends on both the volume of emails being processed and the SMTP services.

Processor

Core i3, 1 GHz

Disk space

Minimum 250 MB (additional 122 MB during setup)

 

Operating Systems
  • Microsoft Windows Server 2012
  • Microsoft Windows Server 2012 R2
  • Microsoft Windows Server 2016 Standard

 

Operating System Components

Microsoft Internet Information Services (IIS) 8.0 or later

Simple Mail Transport Protocol (SMTP) services

Email Accelerator External Server

The following requirements apply to a machine that hosts the external server for Email Accelerator. The external server is also called the "tracking server."

Hardware Value

Memory

1GB-4GB depending on the number of concurrent users. Memory usage depends on the amount of Web traffic to this server. We recommend that you use the internal server to host the tracking website.

Processor

Core i3, 1GHz

Disk space

Minimum 250 MB (additional 122 MB during setup)
