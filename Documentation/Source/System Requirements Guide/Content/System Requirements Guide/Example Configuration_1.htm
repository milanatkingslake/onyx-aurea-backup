Email Server for Processing Server

You must set up one IMAP-compliant email server, such as Microsoft Exchange or Lotus Domino, for use with OPS. This can be an existing email server that you currently use within your organization. A dedicated email server is not necessary. 

Mail server options if you are using Microsoft Exchange - One of the following

Microsoft Exchange 2007

Microsoft Exchange 2010

Microsoft Exchange 2013

For hardware specifications and operating system requirements to install email server, visit the email server vendor website.
