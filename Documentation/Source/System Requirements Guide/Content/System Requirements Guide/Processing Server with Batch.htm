Processing Server with Batch Subsystem

The following requirements, which apply to a server computer dedicated to the Batch subsystem, are the equivalent of a low-end server-class computer. For example, the hardware requirements for the OPS /OES Administration client computer are sufficient to install OPS with Batch subsystem.

Online data
  • OPS server with local, dedicated OEAS

  • 100-Mbps network connectivity between the OPS-OEAS server and OEDB server

High capacity
  • OPS server with local, dedicated OEAS
  • OPS-OEAS server: 4 Xeon processors at 500 MHz with 1 GB of RAM
  • 100-Mbps network connectivity between the OPS-OEAS server and OEDB server
  • Processing capacity: Approximately 50,000 OEAS calls processed per hour
  • Ten Batch subsystems
