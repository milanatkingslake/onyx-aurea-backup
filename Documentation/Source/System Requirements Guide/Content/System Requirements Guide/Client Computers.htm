Client Computers

This section provides system requirements for client computers.

  • Email Client
  • Enterprise Studio and Processing Server Administration Clients
  • Onyx Employee Portal
  • Onyx UI Client
  • CTI Toolbar

You can use a single client computer to access Onyx UI, Enterprise Studio Administration, Processing Server Administration, and Email clients, as long as all system requirements are satisfied.

Email Client

An email client computer is required if you intend to install Email Add-in for Microsoft Outlook. Any computer that uses Microsoft Outlook 2007 or later to access email messages and has the .NET 4.6.1 or later Framework installed can be used for this purpose. This can be the same client computer as the one used for the Onyx UI. This email client connects to your corporate email server. For information on email server requirements for Onyx, see Email Server for Processing Server.

Enterprise Studio and Processing Server Administration Clients

The following requirements apply to client computers that are used to administer Onyx Enterprise Studio (OES) and Onyx Processing Server (OPS).

  • You can set up OES and OPS administration clients on the same client computer for Onyx
  • If OES and OPS are installed to physical servers (as opposed to virtual servers), the server computers can also serve as the administration clients.
  • Provided all system requirements are met, the OES and OPS administration clients can reside on the same client computer as the Onyx UI Client on page 1.
