Processing Server Computers

This section provides information on the system requirements for Processing Server and its various subsystems.

  • Processing Server with all the Subsystems
  • Email Server for Processing Server

Processing Server with all the Subsystems

The following requirements apply to a single server computer hosting Processing Server (OPS) and all available subsystems: Email Monitoring Subsystem (EMS), Fulfillment, and Batch.

To view requirements for a server computer dedicated to a single subsystem, see:

  • Processing Server with Email Monitoring Subsystem (EMS)
  • Processing Server with Fulfillment Subsystem
  • Processing Server with Batch Subsystem
Hardware Minimum

Processor

Single-processor Xeon or equivalent, 700 MHz

Memory

2 GB

Disk space

250 MB

 

Operating Systems
  • Microsoft Windows Server 2012
  • Microsoft Windows Server 2012 R2
  • Microsoft Windows Server 2016 Standard

 

Operating System Components

Microsoft Internet Information Services (IIS) 8.0 or later
Microsoft XML Parser (MSXML) 4.0 SP3 or MSXML 6.0 SP2

Microsoft Exchange System Management Tools for the version of Exchange Server you are using

 

Web Browser
  • Microsoft Internet Explorer 8.0 or later
  • Microsoft Internet Explorer 11.0 / Microsoft Edge ( 32 / 64 Bit )

Processing Server with Email Monitoring Subsystem (EMS)

The following requirements apply to a server computer dedicated to the EMS.

Hardware Minimum

Memory

512 MB

Processor

Single-processor Xeon, 700 MHz

Disk space

250 MB

 

Operating System
  • Microsoft Windows Server 2012
  • Microsoft Windows Server 2012 R2
  • Microsoft Windows Server 2016 Standard

 

Operating System Components

Microsoft Internet Information Services (IIS) 8.0 or later
Microsoft XML Parser (MSXML) 4.0 SP3 or MSXML 6.0 SP2

Exchange System Management Tools for the version of Exchange you are using

 

Web Browser
  • Microsoft Internet Explorer 8.0 or later
  • Microsoft Internet Explorer 11.0 / Microsoft Edge ( 32 / 64 Bit )

 

Email Server

IMAP-compliant email server, such as Microsoft Exchange or Lotus Domino. For detailed Email server requirements, see Email Server for Processing Server on page 1.
