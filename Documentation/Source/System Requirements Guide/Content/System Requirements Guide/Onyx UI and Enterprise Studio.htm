Onyx Extraction Engine Server

If you are installing Insight Extraction Engine on a corporate intranet that only internal users can access, we recommend that you use a dedicated server computer to run Extraction Engine.

For an installation where both internal and external uses can use Insight Extraction Engine, we recommend you use a second Web server that is separated from the corporate network by a firewall.

I fall system requirements are fulfilled, Insight Extraction Engine can be installed on the same server computer as Processing Server (OPS).
