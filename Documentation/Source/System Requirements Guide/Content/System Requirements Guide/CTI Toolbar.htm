CTI Toolbar

CTI Toolbar that is installed on the Onyx UI does not require a separate client computer. It requires the following environment.

CTI Toolbar Requirements

Onyx OEP UI

Compatible CTI connector that is configured to integrate your phone system with CTI Toolbar. For more information, see CTI Toolbar Implementation Guide.

Because the ICTIConnector interface used by CTI has not changed, connectors built for ONYX CTI are also compatible with CTI Toolbar 7.9.
