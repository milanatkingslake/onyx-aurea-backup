Onyx Mobile Web Prerequisites

Server-side Prerequisites

Onyx Mobile Web server-side components need to be installed on the same machine where OEAS is installed.

Onyx Mobile Web Server-side prerequisites

Onyx Enterprise Application Server (OEAS)

Onyx Gateway Service

Onyx Gateway Proxy

Onyx Employee Portal

IIS 7 and above

Client-side Prerequisites

Onyx Mobile Web client-side components need to be installed on the client machine from where the application server is accessed.

Operating System
  • Microsoft Windows 7
  • Microsoft Windows 8.1
  • Microsoft Windows 10 Anniversary Update

 

Browsers
  • IE11
  • Chrome
  • Mozilla Firefox
  • Safari
  • Microsoft Edge
