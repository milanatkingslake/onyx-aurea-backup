Onyx 360 Prerequisites

Server-side Prerequisites

Onyx 360 server-side components need to be installed on the same machine where OEAS is installed.

Onyx 360 Server-side prerequisites

Onyx 7.9 Enterprise Application Server (OEAS)

Onyx 7.9 Web Services (OWS) configured to work under Onyx and Windows Authentication

Microsoft .NET Framework 4.6.1 or later

Setup Onyx Employee Portal file upload (attachment) folder structure to support the Onyx360 Add Attachment feature

Setup Onyx File Agent (OFA) to support Onyx360 email attachments.

Grant necessary permissions to the attachment file-share to be accessible from users’ desktop (same permission as OEP file attachment).

Client-side Prerequisites

Onyx 360 client-side components need to be installed on the client machine from where the application server is accessed.

Onyx 360 Client-side Prerequisites

Microsoft Outlook 2010 or 2013 or 2016

Microsoft .NET Framework 4.6.1 or later (is included along with the client installer)

Visual Studio Tools for Office Runtime (VSTO runtime libraries, included along with the client installer)
