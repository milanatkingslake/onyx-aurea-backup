Database and Storage Server Computers

This section provides information on the system requirements for database and storage server computers.

  • Onyx Enterprise Database Server
  • BI for Microsoft SSRS Report Server
  • File Attachment Storage Server

Depending on your business needs, if all the system requirements are met, all the components listed above can reside on the same server computer.

BI for Microsoft SSRS Report Server

The following requirements apply to a server computer that hosts the BI for Microsoft SSRS report server:

  • Microsoft SQL Server 2012/2014/2016 Reporting Services (SSRS) installed. For information about installing SSRS, see Microsoft SQL Server 2012 /2014/2016 Books Online on the MSDN website.
  • Access to the Onyx database.
  • The Application Server should have access to use the Web services exposed by the report server.

Based on the scale of your business, you can set up BI for SSRS on the same server computer as the Onyx database or on a separate one. The server load from SSRS depends on the volume and complexity of your reports. For more information about SSRS server load and server distribution and how server load may impact the Onyx database server (if installed to the same computer), review the documentation on planning deployments in Microsoft SQL Server 2012/2014/2016 Books Online on the MSDN website at msdn.microsoft.com.
