OEAS object model changes for Onyx 7.8

This document provides a summary of the changes between the original, non-customized Onyx CRM 7.7 OED and the Onyx CRM 7.8 OED.

To understand the differences between your customized version of the OED and the newly-installed OED, upgrade to OEAS 7.8 and then consult the log.txt file.

Additionally, the following differences are included in the log.txt file that is generated when you upgrade OEAS, and are not listed in this document:

  • Changes to the descriptions of the objects, methods, and properties
  • Changes to the version numbers, if the only difference between one object and another is the version number

For a summary of changes made to the database (OEDB), refer to the Database changes document.

  • Changes to the OEAS object model
