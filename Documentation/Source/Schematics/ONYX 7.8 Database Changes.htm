Database changes for Onyx 7.8

This document provides a summary of changes made to Onyx databases for Onyx 7.8. For a summary of changes made to the middle tier, refer the OEAS object model changes document.

This document is important to you if you are upgrading to Onyx 7.8 and have:

  • Customized the business logic in the database (such as customizing and implementing configurable stored procedures).
  • Implemented other customizations, such as building a custom user interface or integration component that communicates directly with the database instead of through the middle tier.

The following sections describe the differences between the 7.7 and 7.8 versions of the Onyx databases, including:

Schema changes

  • Changes to the Onyx Persistence Database
  • Changes to the Onyx Transaction Database

Additional information

  • APTEAN Support

Printing tips

The width of the following tables may exceed the width of a normal printed page. To print this document:

  1. Copy the document into Microsoft Word or a similar program.
  2. Change the printing orientation of the document from portrait to landscape.
  3. Adjust the size of the table columns to the desired width.
  4. Issue a print command.
