Modifying an email monitor subsystem

After making changes to a subsystem, you must refresh any running instances of OPS Administration for the changes to appear in the navigation area.

To modify a subsystem:

  1. In the navigation area, select the subsystem that you want to modify.
  2. Click to stop the subsystem.
  3. In the edit area, make the desired changes.

    4. For descriptions of the attributes in the edit area, see Adding and configuring email monitor subsystems.

  4. Click to save your changes.
  5. Click to restart the subsystem.
