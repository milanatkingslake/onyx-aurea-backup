Viewing OPS service information

You can view the OPS service name and status in the navigation area or as described in the following procedure.

To view OPS service information:

  1. In the navigation area, select the Service node.

    2. The service information appears in the edit area.

  2. You can now view the following non-editable Service Information fields:
  • OPS Server Name (the name of the server for the OPS service)
  • Status (the status of the OPS service: Started, Paused, Stopped, or Error)
