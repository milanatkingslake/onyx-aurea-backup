Viewing Completed Batches

The following procedure describes how to view Completed Batches. All action batches with a status of completed are returned.

The total number of lines shown depends on the Maximum Rows for Completed Batches setting in Queue Preferences. For more information on queue settings, see Configuring queue preferences.

To view completed action batches:

  1. In the navigation area, expand the connection node.
  2. Expand the Batches Queues node.
  3. Select Completed Batches.

    4. The queue appears in the edit area. Action batches are shown with a status of Completed.
