Managing literature order lines

Using OPS Administration, you can change an order line’s status, printer, or priority.

When you view the New Order Lines queue or Completed Order Lines, the following columns appear:

 

  • Priority
  • Batch ID
  • Order ID
  • Line Number
  • Status
  • Type
  • Printer
  • Product
  • Description
  • Quantity Ordered
  • Order Owner Type
  • Order Owner ID
  • First Name
  • Last Name
  • Company Name
  • Email To
  • Order Date
  • Ship Date
  • Subsystem
  • Insert By

The following sections provide procedures for:

  • Viewing New Order Lines queue
  • Viewing Completed Order Lines
  • Sorting the queue

  • Modifying an order line
