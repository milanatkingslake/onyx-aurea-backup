Modifying mailboxes

After you modify mailbox settings, you must refresh any running instances of OPS Administration for the changes to appear in the navigation area.

To modify a monitored mailbox:

  1. In the navigation area, select the mailbox that you want to modify.
  2. In the edit area, make the desired changes. For descriptions of the attributes in the edit area, see Adding and configuring monitored mailboxes.
  3. Click to save your changes.
