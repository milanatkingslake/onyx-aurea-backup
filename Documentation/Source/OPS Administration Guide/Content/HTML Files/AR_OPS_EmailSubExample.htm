Example: Email monitor subsystem

The following example shows the edit area for an EMS called Q1 Promotions.

In this example, Q1 Promotions is scheduled to monitor mailboxes for inbound email 24 hours a day, 7 days a week (Start Time 00:00, Stop Time 00:00). After each inbound email that the EMS processes, it pauses for the 100 millisecond governor interval before processing another email. After the processing cycle completes, the EMS enters sleep mode for the run interval of 5 minutes before restarting. Error notification emails are sent to the Administrator’s inbox.

To learn about configuring a monitored mailbox for Q1 Promotions, see Example: Monitored mailbox.
