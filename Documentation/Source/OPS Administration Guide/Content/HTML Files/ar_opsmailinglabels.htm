Mailing labels

To use OPS mailing labels, first determine:

  • What type of labels you need.
  • Which literature products need mailing labels. Generally, literature products with a fulfillment type of Send as Printed Document or Send as Physical Item have labels associated with them for mailing purposes.

Once you have determined the type of labels you need, you first need to create templates and then to create a literature product that references the template.

For instance, you might create a mailing label literature product named “Label 5163” and associate the label with the following:

  • Thank-you letter
  • New customer welcome package
  • Sales brochure
  • Demonstration software CD
