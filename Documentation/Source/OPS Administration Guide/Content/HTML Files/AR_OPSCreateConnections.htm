Create OPS connections

Step 1: Create OPS connections

Normally, you configure an OPS connection by running the OPS Configuration setup immediately after running the OPS setup for the first time. If you have not already done so, you must create at least one OPS connection before adding and configuring subsystems. If you want to configure additional OPS connections or change settings for existing connections, you can re-run the OPS Configuration setup from the OEAS installation package. Each connection must point to a different OEDB.
