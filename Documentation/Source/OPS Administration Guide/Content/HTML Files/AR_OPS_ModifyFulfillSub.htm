Modifying a subsystem

After making changes to a subsystem, restart the subsystem for the changes to take effect. For more information on the attributes, see Adding and configuring a print subsystem or Adding and configuring an email subsystem.

To implement changes to a subsystem:

  1. In the navigation area, select the subsystem that you want to modify.
  2. Click to stop the subsystem.
  3. In the edit area, make the desired changes.
  4. Click to save your changes.
  5. Click to restart the subsystem.
