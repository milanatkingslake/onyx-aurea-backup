Subsystems

After you decide which literature products you need, determine what types of subsystems are required to process the order lines. The literature product's fulfillment type determines what types of subsystems are required. A typical installation includes fax, email, and print subsystems.

The following table shows the fulfillment types processed by the different subsystems.

Subsystem

Fulfillment type

Fax

Send as Fax (supports merging)

Send as Fax (no merging)

Email

Send as Email Body (supports merging)

Send as Email Attachment (supports merging)

Send as Email Attachment (no merging)

Print
