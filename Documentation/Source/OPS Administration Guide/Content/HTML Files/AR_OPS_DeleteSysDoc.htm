Deleting a system document

Deleting a system document in OPS Administration only deletes the association between the literature product and the subsystem. To permanently delete a system document, first delete the system document from all subsystems then delete the product in Product Administration.

To delete a system document from a subsystem:

  1. In the navigation area, select the system document that you want to delete.
