Activating or deactivating a subsystem

To activate a subsystem:

  1. In the navigation area, select the subsystem that you want to activate.
  2. In the edit area, check the Activate box.
  3. Click to save your changes.
  4. Click to restart the subsystem if the subsystem was stopped.

To deactivate a subsystem:

  1. In the navigation area, select the subsystem that you want to deactivate.
  2. Click to stop the subsystem.
  3. In the edit area, clear the Activate box.
  4. Click to save your changes.
