Step 9: Add batch subsystems

Batch subsystems process List Manager actions in action batches. The subsystem processes the action batch according to subsystem settings that you specify using OPS Administration. When you add a subsystem to an OPS connection, you configure the subsystem settings that determine how the actions will be processed. You can have multiple batch subsystems, with each subsystem configured independently.

For more information, see Adding a batch subsystem.
