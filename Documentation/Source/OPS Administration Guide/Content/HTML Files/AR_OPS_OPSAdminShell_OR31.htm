Modifying order line status

When a subsystem retrieves an order line, the order line is marked with the subsystem ID. An order line must have this to be processed and the line can only be processed by the reserving subsystem. Because of this, the follow table describes the allowable status updates.

Current status

Allowable status updates

Comments

New

None

The order line does not have a subsystem ID and so is not processed.

In Process

New

The order line is processed by the next available subsystem.

Completed

New or In Process

If updated from Completed to New, the order line is processed by the next available subsystem.

If updated from Completed to In Process, the order line is processed by the reserving subsystem. If that subsystem is unavailable, the line is not processed.

Error

New or In Process

If updated from Error to New, the order line is processed by the next available subsystem.

If updated from Error to In Process, the order line is processed by the reserving subsystem. If that subsystem is unavailable, the line is not processed.

All other statuses

New

The order line is processed by the next available subsystem.

The following procedure describes how to modify an order line.

To modify an order line:

  1. Navigate to and expand the Fulfillment Queues node.
  2. Select New Order Lines or Completed Order Lines.

    3. The queue appears in the edit area.

  3. Click Show Edit.
  4. Check the box to the left of each order line that you want to modify.

    5. OR

    To select all order lines, check the box in the queue header.

  5. Do one of the following:
  • To change the printer, select a printer from the Printer list.
  • To change the priority, select a priority from the Priority list.
  • To change the status, select a status from the Status list.
  • To reprocess an order line, select New.
  1. For more information on updating status, see Status updates.
  2. Select New from the Literature Order Line Status list.
  3. Click Update.
