Example: Monitored mailbox

The following example shows the edit area for a new mailbox called Promotion responses. This mailbox will be monitored by the EMS called Q1 Promotions.

In this example, the inbox for Promotion responses is monitored for inbound emails for a maximum processing time of 5 minutes and receives only Standard Format emails. During email processing, new emails are flagged for follow-up, so that internal users can make them active in Task Manager. During post-processing, emails are moved to the archive folder.
