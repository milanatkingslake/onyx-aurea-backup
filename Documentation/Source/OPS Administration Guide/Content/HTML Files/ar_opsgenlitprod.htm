General literature products

To determine the literature products that can be ordered from the Literature feature, perform the following steps:

  1. First, determine the literature products that your users need. The following are some examples:
    • Thank-you letter
    • Thank-you fax
    • Thank-you email
    • New customer welcome package
    • Sales brochure
    • Demonstration software CD
    • Sales contract
    • Software program
  2. Next, for each literature product, decide if you need to merge information into the literature product. This determines the fulfillment type and customization required for the template.
  3. Finally, for each literature product, decide how to fulfill it. You can configure multiple fulfillment types for each literature product so that the user can select how they want the literature product to be fulfilled. Note, however, that if you want to provide the option for a literature product to be fulfilled as fax, email, and print documents, you need to create templates for each: one Extensible Stylesheet Language (XSL) template for fax, one XSL template for email, and one Microsoft Word Visual Basic for Applications (VBA) template for print.

Note: Some fulfillment types do not support data merging. For more information, see Fulfillment types.
