Viewing OPS connection information

A connection between OPS and OEAS is a prerequisite for adding subsystems. Use the following procedure to view the application name and site ID for the connection.

To view an OPS connection:

  1. In OPS Administration, select a connection in the navigation area.

    2. The connection information appears in the edit area.

  2. You can now view the following non-editable Connection Information fields:
    • Application Name (the name of the OEAS logical application for the OPS service)
    • Site ID (the database site ID that was configured in OPS Configuration)
