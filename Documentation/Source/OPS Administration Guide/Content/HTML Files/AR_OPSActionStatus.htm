Step 10: Configure action batch status

The action batch status is used to track the status of an action batch. This status is displayed on the New Batches queue and Completed Batches in OPS Administration. Configuring action batch status is an optional step.

The following table details the default action batch status data installed with the OEDB.

actionBatchStatusCode

Status

Description

0

New

Note: Do not modify this status data

Processing of the batch has not begun.

1

In Process

Note: Do not modify this status data

The batch is being processed.

2

Completed

Note: Do not modify this status data

The processing is complete.

3

Error

Note: Do not modify this status data

There were batch errors during processing.
