Viewing New Order Lines queue

In the edit area, under New Order Lines, the total number of lines shown depends on the Maximum Rows for New Order Lines setting in Queue Preferences. For more information on queue settings, see Configuring queue preferences.

To view order lines:

  1. In the navigation area, expand the connection node.
  2. Expand the Fulfillment Queues node.
  3. Select New Order Lines.

    4. The queue appears in the edit area.
