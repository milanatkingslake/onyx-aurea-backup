Fax literature products

Fax literature products are literature products that have a fulfillment type of Send as Fax (supports merging) or Send as Fax (no merging). You configure these to be ordered from the Literature feature and Faxed to a customer. You can create Fax literature products that merge customer specific data into the Fax, or you can create Fax literature products that contain finished content that you want to send with no further modification.

To create a Fax literature product personalized with merged customer data:

  1. Create an XSL template to perform data merging. Create the template by customizing one of the OPS sample XSL templates or by creating a new template. For more information on creating and customizing templates, see Creating templates.
  2. In Product Administration, create a literature product that references the template, making the following changes in the edit area:

Active Literature: Check this box to indicate that the literature product is active.

File Path: Enter the file path to the location where the XSL template is located.

Printer: Leave this box empty.

Is a Label: Leave this box empty.

Label Product: Leave this box empty.

Fulfillment Types: Select the following fulfillment type:

    • Send as Fax (supports merging)

You can also create a Fax literature product that does not require data merging and send it as an Fax attachment.

To create a Fax literature product of finished content:

  1. Create the finished content file (pdf, gif, and so on).
  2. In Product Administration, create a literature product that references the file , making the following changes in the edit area:

    3. Active Literature: Check this box to indicate that the literature product is active.

    File Path: Enter the file path to the location where the file is located.

    Printer: Leave this box empty.

    Is a Label: Leave this box empty.

    Label Product: Leave this box empty.

    Fulfillment Types: Select the following fulfillment type:

  • Send as Fax Attachment (no merging)

Note: For details on adding literature products, see Product Administration Help.
