UI Text ID prefixes Definition

caption

Text identifying section titles

data

Text regarding data entry

label

Text identifying fields

msg

Text inside error messages or dialog boxes

save

Text regarding the save status of the entry

tip

(tooltip) Text briefly displayed when the user points to the button

title

Text identifying a Header Bar button

ucf

(UI Control Fields) The text in text boxes, drop down lists, tabs, toolbars, and so on

WorkbenchPageName

Text identifying the page to which you are adding UI text items
  1. Assign the text an ID in the Name field. You can enter any string of alphanumeric and underscore characters. For example, customMessageText. Remember the name of the message ID.
  2. In the Text field, type the desired message.
  3. Click to confirm your change and close the UI Text Editor window.
  4. Click in the page to save your changes.
