Resources used for common Navigator functions

The following table identifies the security resources for controlling various common functions in Navigator.

This Resource... Allows users to...

UI:OEP.searchFilters.configure

Configure Navigator search criteria.

UI:OEP.Navigator.PublicFolder.ShowHide

Access public queries folder, save queries, and run saved public queries.

UI:OEP.Navigator.PrivateFolder.ShowHide

Access private queries folder, save queries, and run saved private queries.

UI:OEP.resultgrid.configure

Configure Navigator result grid columns.

UI:OEP.resultGrid.print

Print Navigator search results.

UI:OEP.resultGrid.export.csv

Export Navigator search results to a CSV file.

UI:OEP.resultgrid.configure.maxRecord

Modify the value for maximum search records to display in Navigator.

UI:OEP.Navigator.PublicQuery.Manage

Delete public queries, clone public queries, rename public queries, edit public queries, add public folders, rename public folders, delete public folders, and move public and private queries.

UI:OEP.Navigator.ResultGridAdminConsole.AvailableList.View

 

 Enable the Available Section of Search Result Field Selection and Administration Window in Navigator search result.

UI:OEP.Navigator.ResultGridAdminConsole.ChangeDefault

 Enable changes by the user on Search Result Field Selection and Administration Window to reflect to all the other users.

UI:OEP.Navigator.ResultGridAdminConsole.DisabledList.View

 

 Enable the Disabled List section of Search Result Field Selection and Administration Window.

UI:OEP.Navigator.ResultGridAdminConsole.HeaderList.View

 Enable the Selected section of Search Result Field Selection and Administration Window. HeaderList contains all the selected fields for search results.

UI:OEP.Navigator.ResultGridAdminConsole.PropertyEdit.View

 Change the Display name /Caption of a field on Search Result Field Selection and Administration Window.
UI:OEP.Navigator.SearchAdminConsole.AvailableList.View Enable the available fields section of Search Criteria, Field Selection, and Administration.
UI:OEP.Navigator.SearchAdminConsole.DefaultList.View Enable the Selected Section of Search Criteria Field Selection and Administration Window.
UI:OEP.Navigator.SearchAdminConsole.DisabledList.View Enable the Disabled fields section of Search Criteria Field Selection and Administration Window.
UI:OEP.Navigator.SearchAdminConsole.PropertyEdit.View Change the Display name and Make a field as required on Search Criteria Field Selection and Administration Window.
UI:OEP.Navigator.SearchFilterAdminConsole.ChangeDefault Enable the changes made by the user on Search Criteria Field Selection and Administration Window to reflect to all the other users.

UI:OEP.reminder.insert

Add reminders using Navigator Grid Action Menu.

UI:OEP.alert.insert

Add alerts using Navigator Grid Action Menu.

UI:OEP.script.retrieve

View Script as an entity in the Search Type drop-down of the Navigator Search Criteria pane.

UI:OEP.batchupdate.task

Perform a batch update for Task records.

UI:OEP.batchupdate.workticket

Perform a batch update for Work Ticket records.

UI:OEP.appointment.gam

UI:OEP.company.gam

UI:OEP.email.gam

UI:OEP.forecast.gam

UI:OEP.individual.gam

UI:OEP.product.gam

UI:OEP.incident.sales.gam

UI:OEP.script.gam

UI:OEP.incident.service.gam

UI:OEP.incident.support.gam

UI:OEP.task.gam

UI:OEP.workticket.gam

Perform GAM actions for the corresponding entry.

UI:OEP.task.retrieve

View Task as an entity in the Search Type drop-down of the Navigator Search Criteria pane.

UI:OEP.appointment.retrieve

View Appointment as an entity in the Search Type drop-down of the Navigator Search Criteria pane.

Resource used for the results list control

The following table identifies the security resource for the results list control (the table, or list, that displays the results of a search query). This setting affects all instances of the results list control throughout OEP.

This Resource... Function

UI:OEP.RLControl.update

Configure results lists that appear in OEP, such as by reordering columns and changing column widths.
