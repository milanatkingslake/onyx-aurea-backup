Navigator Queries as Mobile Bookmarks

When creating Navigator queries, users can now save them as personal mobile bookmarks. Currently, users can create mobile bookmarks for the following Navigator entities:

  • Companies
  • Individuals
  • Sales Opportunities
  • Service Requests
  • Support Incidents
  • Tasks
  • Products

To save an existing query as a mobile bookmark, users must select the Subscribe as Mobile Bookmark button for the query in the Saved Queries section.
