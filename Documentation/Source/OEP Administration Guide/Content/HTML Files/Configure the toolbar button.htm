Configure the toolbar button to clear all UDFs (optional)

You can configure a toolbar button to clear the values of the specified controls.

This procedure configures the existing toolbar button (previously configured to launch Google.com) to clear the values of the UDFs on the Supplemental tab. This procedure also demonstrates the usefulness of cloning statements.

To configure the toolbar button to clear all UDFs:

  1. Click on the UCW toolbar for the company edit page to display Dynamic Forms Designer.
  2. Deactivate or delete the action statement that configures the toolbar button to launch Google.com:
  1. Expand Control in the Events pane, expand Toolbar_Supplemental (the toolbar's Admin ID property), expand the Item Click (Item ID) (Item Value) event, and then select the action statement identified by the description "Launch Google.com with toolbarItemBtn" (as entered in a previous UI configuration).
  2. Choose an action:
  • To deactivate the statement, clear the Active check box and then click to confirm changes.
  • To delete the statement, click.
  1. Add an action statement to the Item Click (Item ID) (Item Value) event for Toolbar_Supplemental.
  2. In the Description text box, type: Clear UDF_Public
  3. Add a condition to specify the Button toolbar item (toolbarItemBtn):
  1. In the Object drop-down list located in the Conditions section, expand Event Arguments and then select Item ID.
  2. From the Operator drop-down list, select Equal; in the Value text box, type: toolbarItemBtn
  3. Click in the Summary section to add the condition to the action statement.
  1. Configure the action to clear the UDF_Public control as follows:
  1. From the Object (Action) drop-down list, expand Control, expand UDF_Public, and then select Set Value.
  2. From each Source drop-down list, select User Defined; from the Value drop-down list, select Not Checked.
  1. Click to confirm changes.
  2. Clone the statement three times.

The cloned statements appear below the original statement under the Item Click (Item ID) (Item Value) event for Toolbar_Supplemental.
