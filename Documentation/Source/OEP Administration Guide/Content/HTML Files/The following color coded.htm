Design element

Description

Function

Canvas corner (red triangle) located on the upper left of the canvas.

Displays a shortcut menu when right-clicked.

Panel corner (green triangle) located on the upper left of each panel in the canvas.

Displays a shortcut menu when right-clicked.

Section corner (blue triangle) located on the upper left of each section in a panel.

  • Displays a shortcut menu when right-clicked.
  • Relocates the section when dragged.

Control handle (red circle) located on the left of each toolbox control.

  • Displays a shortcut menu when right-clicked.
  • Relocates the control when dragged.  

Control handle (blue circle) located on the left of each intrinsic control.

  • Displays a shortcut menu when right-clicked.
  • Relocates the control when dragged.

See also Configuring layout and Configuring UI controls.
