To change the color scheme of the interface:

  1. Open Brushes.xaml.
  2. Search for HeaderBrush<SolidColorBrush x:Key="HeaderBrush" Color="#E8E8E8"/>.
  3. Set the color of your choose. Example: <SolidColorBrush x:Key="HeaderBrush" Color="Green"/>.
  4. Compile the dll and replace the dll mention in above path.
