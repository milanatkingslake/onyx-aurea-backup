EMF Configuration Notification Queue Cleanup

The EMF process NotificationQueueCleanUp will move records from the queue table to the completed queue table. This will ensure that the master table always has records to be processed, thereby improving performance when the table is queried as well. Technically the record from the tables on_notification_queue and on_notification_queue_extension are moved to on_notification_queue_complete.

It is recommended to schedule all EMF Notification queue clean up processes during non productive hours.

NotificationQueueErrorProcess

The EMF process NotificationQueueErrorProcess will re-execute the notification queue if there is an error record of three times in a given interval.
