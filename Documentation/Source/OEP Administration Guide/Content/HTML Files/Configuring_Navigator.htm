Configuring Navigator Date and Time Formats

You can configure date and time formats in the Navigator using any of the following supported formats.

Supported Date Formats

The following table lists the supported date formats with description and example.

Input Description Example
YYYY Year in a four digit format. 2014
YY Year in a two digit format. 14
Y Year with any number of digits and sign. -25
Q Quarter of year. Sets the month to the first month in quarter. 1..4
M MM Month Number: M refers to one digit month for months below 10. MM refers to two digits months. 1..12
MMM MMMM Month name in locale Jan..December
D DD Day of month 1..31
Do Day of month with ordinal 1st..31st
DDD DDDD Day of year 1..365
X Unix timestamp 1410715640.579

Setting Mask Format

To set the mask format

1. Access the Navigator_entity_field_master table of Persistence database.

2. Set the mask format for any of the date and time fields.

Example

Run the following SQL query on Navigator_entity_field_master table of Persistence database.

Update Navigator_entity_field_master

Set mask_format='CDF||DD/MM/YYYY HH:mm a'

Where field_id=396
