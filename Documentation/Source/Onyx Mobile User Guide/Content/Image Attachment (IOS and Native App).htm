Image Attachment (iOS and Android)

Onyx Mobile users can attach images to various entities, such as Company and Individual if Administrator has configured the option for an entity.

Image Attachment feature is available only on Onyx Mobile App.

To add an image attachment

  1. Open a record of an entity.
  2. In the More section, click the Image Attachment option.

The image attachment page opens.

  1. In the Image Attachment page specify the following information:
  1. Image Title
  2. Description
  3. Image Version. This value can used to upload multiple versions of the same image, such as a photograph of a building under construction.
  1. Browse for the required file to attach.
  • To take a picture and attach, tap Camera.
  1. Click the picture.
  2. Click to go back to the camera mode to take a different picture or click to confirm and attach the current picture.
  3. The picture appears in the Add Attachment page.
  4. Click to save and attach the image.
  • To attach an existing image, click Gallery.
  1. Browse to the location where the image is stored.
  2. Select the required image.
  3. The picture appears in the Add Attachment page.
  4. Click to save and attach the image.

To view a list of attached images

  1. In the More section of a record, click View all image attachment option.

  1. Click on an image to view the same.
