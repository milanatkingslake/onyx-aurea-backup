Add or Edit WorkTicket

To view work tickets for a customer or for an incident, click View all Work Tickets on the record's Details page.

The fields available to add or edit are configured by your System Administrator. The required fields are denoted with * (asterisk mark) next to the field name.

To add or edit work tickets

  1. Open a Company Details, Incident Details, or Individual Details page.
    1. To add a new work ticket:
      1. Click Add work ticket in the More section.
      2. Specify an optional Description.
      3. Assigned To field by default displays the current logged on user name. To change this selection, click . The Internal Contacts page appears where you can select a different internal contact for the incident.
      4. Select values for Priority, Type, Status, Source, and Product by clicking on the down arrow.
      5. Specify other values as required.
    2. To edit an existing work ticket:
      1. To edit an existing work ticket, click View all work tickets.
      2. Open a work ticket from the list.
      3. Click .
      4. In the Edit WorKTicket page, modify the details.

      Alternatively, to add a new WorkTicket, you can click in the View All WorkTicket page.
