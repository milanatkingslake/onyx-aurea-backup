Add or Edit Individual Records

You can edit an existing individual record or add a new record in Onyx. The fields available to add or modify are configured by your administrator. The required fields are denoted with * (asterisk mark) next to the field name.

To add or edit individual records

  1. On the Home page, click Add Individual.
  2. To edit an existing individual, open the individual record and click Edit on the top-right of the Individual Details page.
  3. Type the First Name for the individual. This is a mandatory field.
  4. Enter/Click the respective check boxes/icons to specify the gender (Male or Female) of the individual.
  5. Type the individual's first name, last name, suffix, and URL.
  6. To add a parent company for the individual, click the Link icon . The Search page appears.
  • Type the search criteria and run the search.
  • In the Search Result List page, select a customer record to link to the record you are modifying.
  1. Click a field to type or select values. For drop-down lists or date selection fields, a new page appears. Select the desired value and tap to confirm your selection.
  2. If you have multiple email or phone fields, select one as the primary email and phone.
  3. To save the individual record, click Save at the top-right of the page. The record appears in the My Recent Customers List page.
  • To discard your entries, click Back at the top-left of the page.
  1. To add internal and external contacts to the record, open the Details page of the newly saved record from My Recent Customers List, and add contacts as desired.
