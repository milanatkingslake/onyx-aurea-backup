Add or Edit Incidents

You can add or edit sales, service or support incident in Onyx. The fields available to add or edit are configured by your System Administrator. The required fields are denoted with * (asterisk mark) next to the field name.

To add or edit incidents

  1. Open a Company Details or Individual Details page.
    1. To add a new incident:
      1. Click Add Incidentin the More section. Alternatively, you can also click to add a new incident.
      2. Select an incident category.
      3. In the Add <category> Incident page, specify an incident description.
      4. Assigned To field by default displays the current logged on user name. To change this selection, click . The Internal Contacts page appears where you can select a different internal contact for the incident.
      5. Select values for Priority, Type, Status, Source, and Product by clicking on the down arrow.
      6. Specify other values as required.
    2. To edit an existing incident:
      1. To edit an existing incident, open an incident that is associated with the company or individual.
      2. Click .
      3. In the Edit Support Incident page, modify the details.
  2. Click to save the changes.
