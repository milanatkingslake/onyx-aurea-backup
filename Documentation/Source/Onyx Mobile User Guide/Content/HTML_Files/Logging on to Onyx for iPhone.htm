Logging on to Onyx Mobile App

When you launch Onyx Mobile for the first time on your mobile device, you must enter the configuration settings provided by your administrator before you can log on.

To log on to Onyx Mobile App

  1. Go to the Apple or Google Play app store and download the Onyx Mobile app that corresponds to your device.
  2. On your iOS or Android device, launch Onyx Mobile.

When you launch the app for the first time, the Onyx Configuration page appears.

  1. Enter the configuration settings provided by the administrator as listed in the following table:
Field Description
Server URI Enter the uniform resource identifier (URI) of the OES Server. For example, http://ipaddress: portnumber/servicegateway.
Application Name Type the OEAS application name.
Language Type the installation language 'ENG'.
Site Id Type the Onyx site ID.
  Add Logo URI for Onyx 7.6.1 (optional)
User Name Type your Onyx user name.
Logo URI Type the URL for your company logo. Contact your administrator URL details.
Clear Cache. Tap to clear the existing cache.
  1. Tap Save. The configuration is saved and the Login page appears.
  2. Accept the default user name or type a different one, and then type your Onyx password.
  3. Tap Login to access Onyx Mobile.

To change previously entered configuration settings, tap Settings on the Login screen, enter the new settings on the Onyx Configuration screen, and tap Save.
