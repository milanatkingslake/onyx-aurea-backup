Shortcut Handler

The Shortcut Handler is a beeline link, through which you can launch the scripts and also open the Detail screen of an entity directly by launching Onyx Mobile App or Onyx Mobile Web through a URL.

The Shortcut Handler supports the following entities:

  • Company
  • Individual
  • Incidents
  • Tasks
  • Scripts

To use the Shortcut Handler, you need to construct the required URL. For more details about constructing URL for the Shortcut Handler, see the Shortcut Handler chapter in the Onyx Mobile Administration Guide.

To open Detail screen through the Shortcut Handler

  1. Open a mail or message in which the Shortcut Handler URL is available.
  2. Tap the URL.
  3. The Onyx Mobile Login screen appears.
  4. Log on to the Onyx Mobile.
  5. The respective Detail screen appears.

If the Onyx Mobile is already logged in, it opens the respective Detail screen without displaying the Onyx Mobile Login screen.

You can open multiple Detail screens at the same time through different URLs.
