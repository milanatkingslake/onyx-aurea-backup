Logging on to Onyx Mobile Web

When you log on to Onyx Mobile Web for the first time, enter the details provided by your administrator.

To log on to Onyx Mobile Web

  1. Type the URL provided by your System Administrator to access Onyx Mobile Web.

  1. Specify the following details:
  • Site ID: Click the down arrow to select a Site ID from the Site ID drop-down. If the Onyx Mobile Web contains multiple websites, the sites are listed in the Site ID drop-down.
  • USER ID: Type your Onyx user name.
  • PASSWORD: Type your Onyx password.
  1. Click SIGN IN.
