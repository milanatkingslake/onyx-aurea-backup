Prompted Action

The Prompted Action feature allows you to quickly update certain fields of the entities through the DoMore section on the Detail screen of the Onyx Mobile without going into the Edit screen.

To use the Prompted Action, you need to configure it in the required XML file. For more details about the Prompted Action configuration, see the Prompted Action chapter in the Onyx Mobile Administration Guide.

To use the Prompted Action

  1. Log on to Onyx Mobile App or Onyx Mobile Web.
  2. Navigate to DoMore section on the Detail screen of an entity.
  3. Tap the required DoMore action which is configured as the Prompted Action and update required information on the available fields.
