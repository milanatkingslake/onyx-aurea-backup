Common UI Tasks

Based on whether you are working on Onyx Mobile App or Onyx Mobile Web, the following tasks may differ:

  • Search
  • To search tasks in an iOS device, click the respective tab at the bottom of the page you are currently viewing.
  • To search from an Android device, click the Search icon on the page.
  • To search in Onyx Mobile Web, click the Search icon .
  • About
  • For more information on the Onyx Mobile App, click the About tab at bottom of the iOS page.
  • If you are using Android, tap About from the Flow Menu.
  • In Onyx Mobile Web, click the About icon .
  • Refresh
  • To refresh a page, click the Refresh icon in an Android mobile
  • Swipe down the page to refresh a page in an iOS device.
  • In Onyx Mobile Web, click the Refresh icon .
