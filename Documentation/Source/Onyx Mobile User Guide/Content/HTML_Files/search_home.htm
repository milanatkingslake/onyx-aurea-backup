Search for Company, Individual, Incidents, and WorkTicket

You can make a search for company, individual, incident records meeting the specified search criteria.

To search records

  1. On the Home page to search for:
  1. A company, click Search Company.
  2. An individual, click Search Individual.
  3. An incident, click Search Incidents.
  4. A work ticket, click Search WorkTicket.
  1. On the Search page, specify the required search criteria.
  2. For drop-down lists or date selection fields, a new page with the choices is displayed. Select a value.
  3. Click to run the search.

The Search Result page, lists all the customer records that match the specified criteria.

  1. Click a record to open the corresponding Customer Details page.

We recommend that you enter as many search criteria as possible to narrow down your search and return quick results. On very large databases, search results may not be returned if you perform a wild card search with no specific criteria.
