Scan Utility

The Scan Utility feature allows you to retrieve the following information by scanning the bar code of a product:

  • On the Home screen
  • List of customer records.
  • List of sales / service / support incident records.
  • List of work ticket records.
  • On the Add or Edit screen
  • Product description.

The Scan Utility is supported only on the Onyx Mobile App.

To use the Scan Utility, you need to configure it in the required XML file. For more details about the Scan Utility configuration, see the Scan Utility chapter in the Onyx Mobile Administration Guide.

To use the Scan Utility on Home screen

  1. Log on to Onyx Mobile App.
  2. Tap scan configured tile on the Home screen and scan the product code to retrieve the list of records associated with that product.

To use the Scan Utility on Add or Edit screen

  1. Log on to Onyx Mobile App.
  2. Navigate to Add or Edit screen of an incident or work ticket.
  3. Tap on the Product field and scan the product code to retrieve the product information.
