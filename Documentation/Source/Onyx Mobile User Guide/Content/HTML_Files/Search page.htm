Search for Customer Records

You can search for company and individual records that meet specified criteria.

To search for customer records

  1. Click .
  2. On the Search page, specify the required search criteria.
  3. For drop-down lists or date selection fields, a new page with the choices is displayed. Select a value.
  4. Click to run the search.

The Search Result page, lists all the customer records that match the specified criteria.

  1. Click a record to open the corresponding Customer Details page.

We recommend that you enter as many search criteria as possible to narrow down your search and return quick results. On very large databases, search results may not be returned if you perform a wild card search with no specific criteria.
