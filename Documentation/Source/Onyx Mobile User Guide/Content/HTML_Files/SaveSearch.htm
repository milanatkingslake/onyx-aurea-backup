Save a Search

A search is saved as a combination of the selected search criteria fields and result grid columns. To access search results from Onyx, you can save a search as a bookmark. For information about bookmarks, see Bookmarks.

Note: Unless specifically mentioned, the procedures/details explained in this topic are the same for both Onyx Mobile App and Onyx Mobile Web.

To save a search

  1. In the Search Criteria window, enter values for each search criteria.
  2. Click , then click Result Columns, and in the Search Result Field Selection and Administration window, select the columns, sort order, and maximum rows to display for the search. For more information on doing this, see Select the fields to view.
  3. Click . The Query Explorer window appears.

You can also click after performing an ad-hoc search to save the search criteria and result grid columns to a filter.

  1. Enter a Query Name for your search.
  2. Select whether to save the filter as a public filter available to all users, or as a private filter available only to you.
  3. Select the Is Bookmark check box to bookmark the filter for easy access from the Header bar.

Depending on the filter's search type, the filter is available in the Bookmarks section under the , , , or buttons.

  1. Select the Is HomePage check box to add the filter to the Home page.
  2. Select the Mobile Bookmark check box to add the filter to Onyx Mobile. When you log on to Onyx Mobile, the filter is displayed on the Home page. The filter name is same as the name of the saved Navigator query.
  3. Save the filter and close the window. The last displayed Result Grid configuration for the selected search type is saved as the Result Grid configuration for the filter.

Note: If you change the search criteria for a saved filter and save your changes, the Search Criteria panel is refreshed with the new search criteria and any previously entered values for the search criteria fields are cleared. To view the values you previously entered, reload the saved filter.
