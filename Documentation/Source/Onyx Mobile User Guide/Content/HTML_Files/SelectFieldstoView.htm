Select Fields to View

You can determine which fields display in the Search Results page based on the kind of information you're looking for.

Note: If you plan to save a search as a bookmark, be sure to include the specific columns specified by your System Administrator in the Header or Detail column.

To select fields to view

  1. In the Search Criteria page, select a Search Type. The default search criteria for the selected search type appear.
  2. To open the Search Result Field Selection and Administration page, click , then click Result Columns. The name of the page matches the search type associated with the search.

    3. The Selected column lists the fields that are currently selected in the Search Results page. The Available column lists the fields that can be included in the Search Results page. Fields appear in the Search Results page in the order in which they are listed in the Header and Detail columns.

  3. To view fields in the order that they appear on the Result Grid, click the List View button .
  4. To view fields in the order of their hierarchy, click the Tree View button .
  • Select the fields to view in the Header and Detail columns.
  • To include a field in the Search Results page, select it in the Available column, and then drag it to the Selected column.
  • To change the order in which fields are displayed in the Search Results page, click the List View button, and then drag and drop fields as required within the Selected column.
  • To remove a field from appearing in the Search Results page, drag it to the Available column.
  • To understand how hierarchical fields are automatically moved when you move their parent or dependent fields, see About hierarchical relationships.
  1. In the bottom pane, select the fields to sort the search results by. The Sort By drop-down lists all the fields that you selected as header or detail. You can select up to three fields to sort results by.
  2. In the Maximum Rows field, enter the maximum number of rows that the search should return from the database. For example, if you enter a value of 100, the search will return the first 100 records that match the criteria you specified.

The default value in this field is 300. The maximum value that you can enter depends on the limit set by your system administrator.

  1. To save your changes as the display configuration for the selected search type, click Apply Changes. Click to restore the default search criteria, and close the page.

Note: When you click Apply Changes, the display order of fields is saved to match the last changed List View.

For information on saving your selected display order as part of a search filter, see Saving a search.
