S3 Utility

The S3 Utility feature allows you to perform the following three actions on Sales / Service / Support incidents.

  • Send Email

The Send Email option allows you to send the configured email to the primary contact of an incident. This email is available also under Email tab of the record in OEP.

  • Send Sign Email

The Send Sign Email option allows you to send the configured email with signature to the primary contact of an incident. When you click this option, the signature box appears. You can sign in that signature box. This email is available also under Email tab of the record in OEP.

  • Send Attachment

The Send Attachment option allows you to store an attachment for an incident. This attachment is available also under Attachment tab of the record in OEP.

The S3 Utility is supported only on the Onyx Mobile App.

To use the S3 Utility, you need to configure it in the required XML file. For more details about the S3 Utility configuration, see the S3 Utility chapter in the Onyx Mobile Administration Guide.

To use the S3 Utility

  1. Log on to Onyx Mobile App.
  2. Navigate to DoMore section on the Detail screen of an incident.
  3. Tap the required s3 utility configured option to use the S3 Utility.
