Add Internal Contact

To add yourself as an internal contact, specify your user name.

To view internal contacts for a customer or for an incident

  1. Click on a record to display the details.
  2. Click View Internal Contacts on the record's Details page.

To add an internal contact

  1. On the Internal Contacts page, click . The Users page is displayed.
  1. To search for a user, click . All the available groups are listed.
  2. Click a group to list the users that belong to that group. Click a user name.
  3. Select a contact type from the list. The user is added to the internal contacts.
  4. You can alternatively, type a user name in the Users page and press Enter.
