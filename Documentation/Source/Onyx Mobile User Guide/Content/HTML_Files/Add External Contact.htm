Add External Contact

To view external contacts for a customer or for an incident, click View External Contacts on the record's Details page.

To add an external contact

  1. On the External Contacts List page, click . The Search page appears.
  2. Type the required search criteria, and then click . The Search Result page displays a list of customer records that match the specified criteria.
  3. Select a customer.
  4. Select a contact type for the customer. The selected customer is added as an external contact to the customer record.
