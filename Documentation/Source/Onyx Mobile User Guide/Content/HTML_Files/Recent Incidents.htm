My Recent Incidents

Use this page to view a list of incidents you visited most recently. Icons displayed next to a record depict the following:

  • Sales incident
  • Service incident
  • Support incident

The Recent Incidents screen displays a list of most recently visited incident records and cannot be further configured. The number of records displayed is the same in Onyx Mobile App and Onyx Mobile Web, and depends on the value set in OES. For information on changing the number of records displayed, refer to OEAS Technical Guide.

To View incident details

>>Click an incident record in the list.
