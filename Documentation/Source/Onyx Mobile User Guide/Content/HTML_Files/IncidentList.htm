Incident List

Use this page to view a list of incidents associated with a customer record, based on the filters configured for the list. The information displayed for each incident is configured by your System Administrator. By default, incident ID, description, status, type, and priority are displayed along with the name of the user to whom the incident is assigned.

  • View Incident Details page. Click the incident in the list.
  • Add a new incident. Click Add at the top-right of the page.
