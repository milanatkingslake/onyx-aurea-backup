Cards and Common Actions

Use the following cards and actions to navigate through Onyx.

Cards

  • Home: Opens the Home Page. This page displays a menu that enables you to navigate to other pages of the application.

  • Search: Opens the Search Page where you can enter search criteria for customer records

  • About: Displays Onyx version information

Actions

  • Refresh: Refreshes the information displayed on Onyx Mobile Web screen. Refresh is available on all List screens.
  • Sort : Click on this icon to sort the list pages based on the criteria specified by the administrator.


  • Share: Click on this icon to share the contents of the detail or the list page.

  • Reset Password: User can click this icon to reset the password.
  • Logout: Exits the application


1. Only the users who have UI resource permissions can reset the password. For more enabling this option, contact your System Administrator.
2. Swiping the Onyx Mobile App screen in a top-to-bottom direction refreshes the information displayed on the screen. Refresh is available on all List screens.
3. Share icon is available in iOS and Android apps only,
