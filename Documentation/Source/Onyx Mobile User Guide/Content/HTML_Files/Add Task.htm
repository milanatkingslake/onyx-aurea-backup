Add or Edit Task

You can edit an existing task or add a new task in Onyx. The fields available to add or modify are configured by your System Administrator. The required fields are denoted with * (asterisk mark) next to the field name.

To add or edit tasks

  1. Open an Incident Detailspage.
    1. To add a new task:
      1. Click Add Taskin the More section.
      2. In the Add <category> Task page, specify a task description.
      3. Assigned To field by default displays the current logged on user name. To change this selection, click . The Assigned To page appears where you can select a different internal contact for the incident.
      4. Select values for Priority, Type, and Status by clicking on the down arrow.
      5. Specify other values as required.
    2. To edit an existing task:
      1. To edit the task, open existing tasks for the incident by clicking the View all tasks in the More section.
      2. In the View Tasks page, the existing tasks that are associated with the current incident are listed. Click a task to open the record.
      3. On the <category>Task Details page, click .
      1. In the Edit <category> Task page, modify the details.
  2. Click to save the changes.
