DoMore Groups

The DoMore Groups allow you to select the associated actions of a group quickly, just by clicking the parent action of the group on Detail screen.

The DoMore Groups can be configured on:

  • ID

The DoMore Groups configured on the ID of the Detail screen appears as shown in the image.

  • DoMore actions

The DoMore Groups configured on the DoMore actions of the Detail screen appears as shown in the image.

To customize the DoMore Groups, you need to configure it in the required XML file. For more details about the DoMore Groups configuration, see the DoMore Groups chapter in the Onyx Mobile Administration Guide.
