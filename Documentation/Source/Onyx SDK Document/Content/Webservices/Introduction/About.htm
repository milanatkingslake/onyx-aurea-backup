About this Guide

Scope

This guide describes the different layers of Onyx SDK and also contains information on how to use it for application development using Onyx platform.

Onyx SDK is a surrogate layer to communicate with Onyx Enterprise Application Service application.

Audience

This manual is intended for application programmers. It assumes the following knowledge and experience:

  • proficiency with the programming language you are using
  • familiarity with Onyx features and functions
