Overview

Onyx SDK is a set of assemblies which facilitates interaction with the core business layer of Onyx Enterprise Application Server (OEAS). The communication can be out of process across the network with core onyx security included. The data layer protocol of the Onyx SDK is Json, using NewtonSoft.net (Json.net); which does the serialization & deserialization of xml which is the core communication protocol in use.

Onyx SDK Layers

The following figure depicts the different layers of Onyx SDK.

Onyx SDK Layers

Onyx Core

The Onyx Core layer is the façade layer which the client interacts with. Following are the list of files of the core layer:

  1. OnyxSDK.Public.Core.dll
  2. OnyxSDK.Public.Configuration.dll
  3. OnyxSDK.Public.Communication.dll

This layer exposes the multiple interface to interact with OEAS seamlessly.

Onyx Public Enterprise Service

This layer is a public layer and to be consumed by the clients. Following are the list of interfaces this layer implements. Most of the communication protocol at this layer is using Json.

  1. IBaseESOnyxAuthenticate
  2. IBaseESOnyxAuthenticateNT
  3. IBaseESOnyxDictionaryCacheProxy
  4. IBaseESOnyxEASServices
  5. IBaseESOnyxNatLangProxy
  6. IBaseESOnyxSessionManager
  7. IBaseESOnyxTransactionManager

Onyx Private Enterprise Service

This layer is a public layer and to be consumed by the clients. Following interfaces are implemented by this layer. Most of the communication protocol at this layer is using Json.

  1. IBaseESOnyxAuthenticate
  2. IBaseESOnyxAuthenticateNT
  3. IBaseESOnyxDictionaryCacheProxy
  4. IBaseESOnyxEASServices
  5. IBaseESOnyxNatLangProxy
  6. IBaseESOnyxSessionManager
  7. IBaseESOnyxTransactionManager

Onyx Private Enterprise Service

This layer interacts with the Onyx SDK which is deployed in the web, i.e. on OGS. From the OGS it can communicate to OEAS Proxy/OEAS.

Abbreviations

The following abbreviations are used in this document:

  • OEAS: Onyx Enterprise Application Services
  • OEAS Proxy: Onyx Enterprise Application Services Proxy
  • OGS: Onyx Gateway Services
  • OGS Proxy: Onyx Gateway Services Proxy
  • OEP: Onyx Employee Portal
  • OEDB: Onyx Enterprise Database
  • OED: Onyx Enterprise Data (Metadata)
  • DMZ Network: De-Militarized Zone Network
