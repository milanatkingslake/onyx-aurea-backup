Displaying Internal Users as Customers

Users may also choose this option to either display or hide internal employee records in the Onyx 360 panel through the User Preferences Settings page.  If this option is set to off, the internal email addresses (auto-detected from the domain name) will not be displayed on the panel.

Note: Some customer implementations may choose not to create Onyx individual records for internal employees.  In this case, the internal employee’s information (looked-up based on the From email address) will display as unresolved if the user has opted to Show Resolved Employee records in the user's User Preferences Settings.
