Onyx 360 – Outlook Add-in

Onyx Outlook Add-in is a plug-in for Outlook 2010 and 2013 that allows users to interact with customer and incident records in Onyx CRM from within Microsoft Outlook.

The Onyx Outlook Add-in enables users to easily link emails to Onyx records, create sales opportunities and support tickets from emails, create appointments, synchronize contacts, and reply to emails using Onyx templates without having to switch to the Onyx UI.

The main components of the Onyx Outlook Add-in are – the Search Bar, the Notification Bar and the Contact Panel.

  • Search Bar allows you to perform searches on a Contact’s full name (for example, John Doe) or last name (for example, Doe), an email address (for example, john.doe@email.com), phone number, and full or partial Company names (for example, Earth or Earth Scents).  You can also search using a person’s Onyx ID if known.
  • Notification Bar displays information/feedback about the last action performed in the Onyx Outlook Add-in
  • Contact Panel is the main area of the Onyx Outlook Add-in where the details are displayed based on the current action being performed.  This area is used to display the details for the selected Contact record, various lists (attachments, email templates, and so on), and form fields use can enter as required.
