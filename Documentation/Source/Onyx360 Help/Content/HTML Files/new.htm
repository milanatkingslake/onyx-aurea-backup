New Contact Workflow

To add unresolved contacts into Onyx from Outlook, contacts must first be added to your Outlook Address Book and then synchronized to the Onyx database.  For more information on this topic, see Outlook/Onyx Contact Synchronization

A solitary option to Create/Lookup contact will be available on unresolved contacts on the Onyx pane.  Clicking on this link will perform a lookup on the email address in the user’s Outlook Address Book and load the record, if found.  Otherwise, a New Contact window is loaded with the name and email address pre-loaded from the email message.
