Download/Lookup Outlook Contact

This action allows users to download a contact’s information from the Onyx database to their local Outlook Address Book.  The downloaded contact record is automatically set up to synchronize with Onyx.  For more information about synchronizing Outlook contacts with Onyx, see Outlook/Onyx Contact Synchronization.

If the contact already exists locally on the user’s Outlook Address Book, the contact will be opened window in the Outlook contact form after automatically synchronizing data with Onyx.
