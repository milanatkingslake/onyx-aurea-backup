Add Attachments

Add attachments from the email to the Contact’s record in Onyx.  Alternatively, the attachments may be added to a corresponding incident record created from the action menu.

All attachments found are listed and the user can chose to select the attachment(s) to save to the Contact record in Onyx. 

Note: The Add Attachments action item will only be visible on email messages with embedded file attachments.  If the selected email does not contain file attachments, this menu option will not be seen by the user.

The Outlook User must be able to connect to the OEP File Upload Share and have write access for this feature to work.  If the connectivity to the file share is not available, or if the user does not have access, the Add Attachments option is unavailable for the user.
