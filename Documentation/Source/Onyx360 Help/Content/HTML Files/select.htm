Select Recipients/Save in Onyx

If multiple customer contacts are included in the email message (for example, an email from a Contact, copied to one or more stakeholders from their organization), the Onyx 360 user has the option to save the email message to a different Contact included in the email.  Multiple selections are allowed from this list, so the user can, if needed, save the email message to one or more contacts (or all) from this list.

Just select the Contacts that the email message is required to be saved for and click the Save/Sync selected in Onyx button.

The email message is saved in Onyx, and all the Contacts selected are linked to the email message.  If no contacts are selected, the email is saved to all resolved customers in the list.

Note: The email message is saved only once, but is linked to all the selected Contact records.  This prevents unnecessary duplication of the email message in the Onyx database and allows users to access the email message from the PowerPage of any included Contact in Onyx.
