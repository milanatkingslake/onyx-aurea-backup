Re-synchronize Contacts with Onyx

When a previously synchronized Contact record gets modified in Onyx, or if new information was added to Outlook that has not been synchronized to Onyx – that condition is automatically detected by the Onyx 360 pane and the information is displayed to the user the next time the Contact record is opened. 

The user will view the message that the contact record is out-of-sync.

To re-synchronize a previously synchronized contact, a user must click the Save/Sync selected in Onyx button to refresh the record from the latest information in Onyx.

Note: Onyx 360 implements two-way sync with Outlook.  The most recent information between the contact records in Outlook or Onyx is persisted.

The bi-directional synchronization may be changed, if required, to always persist information from Onyx through User Settings.
