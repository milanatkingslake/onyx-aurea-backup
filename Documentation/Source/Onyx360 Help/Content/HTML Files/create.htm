Create Sales Opportunity / Service Request / Support Ticket

Create a Sales Opportunity, Service Request, or Support Ticket directly from the Outlook window on the Contact’s record. 

Selecting one of the options loads the following screen with the description for the incident record preloaded from the email subject line.  Users will be required to provide additional information for mandatory fields – which may be preset in the User Preferences window in Onyx.

Users may include optional notes that will be saved to the incident record created in Onyx and choose to add the entire email as a work note.
