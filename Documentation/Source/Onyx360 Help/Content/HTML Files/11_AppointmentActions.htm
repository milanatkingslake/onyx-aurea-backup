Appointment Actions

The Onyx 360 panel on the Outlook Calendar view allows users to perform various actions pertaining to the selected calendar appointment.

Note: The list of actions displayed in this menu may vary depending on the configuration and the user’s access permissions as set up by the System Administrator.

The following are some typical actions found in the pop-up menu:

  • Add Appointment
  • Add Attachments
  • Add External Contact
  • Add Me as Internal Contact
  • Create Sales/Service/Support
  • Download/Lookup Outlook Contact
