New Email Workflow

When a user creates a new email, all the actions described earlier are available to be performed, once the email address is successfully resolved to a Contact’s email address in Onyx.

The same actions are available from a New Email window, with the following differences in behavior:

  • Add Email. Sets up the unsent email to be added to the Contact’s record in Onyx – the email will be saved in Onyx when it is sent.  The save action can be canceled (toggle).
  • Add Attachments. The attachments will be added to the Contact’s record
  • Create Sales Opportunity/Service Request/Support Ticket. Creates a corresponding new record in Onyx and the email will be associated when it is sent
  • Subscribe/Unsubscribe Email Conversation. The email will be tracked in Onyx when it is sent.
