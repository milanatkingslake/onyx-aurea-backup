Sales/Support/Service Lists

A list of Sales Opportunities, Support Tickets, or Service Requests associated to the selected Contact record is displayed here.

The properties displayed for each record are configured by the System Administrator and may be tailored to suit your current implementation of Onyx.

Note: For performance reasons, only the top 10 records associated with the Contact will be displayed in the list.  The number of records to be displayed is a configurable value for each list and may be set differently by the System Administrator based on implementation requirements.
