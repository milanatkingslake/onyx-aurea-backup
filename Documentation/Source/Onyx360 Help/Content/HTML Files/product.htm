Product List

A list of products that are associated to the selected Contact record in Onyx is displayed here.

The properties displayed for each record are configured by the System Administrator and may be tailored to suit your current implementation of Onyx.

Note: In addition to the core Onyx objects like incidents externalContact, and so on, custom objects created specific to your Onyx implementation may be included in the lists in the Contact panel as long as the object implements a retrieveList method according to OEAS Development Standards.

For information about creating a custom object and retrieveList method in Onyx, refer to the OEAS Developer’s Guide.
