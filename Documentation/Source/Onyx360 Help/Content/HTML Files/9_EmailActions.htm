Email Actions

Various actions are available to the user to perform from the main Onyx 360 panel.  These actions provide users with more control on managing email message(s) in Onyx.

The following actions are available from the tool bar on the Onyx 360 panel.

  • Subscribe/Unsubscribe Email Conversation
  • Select Recipients/Save in Onyx
  • Refresh
  • New Email Workflow
