Refresh

The Refresh button re-loads the Onyx 360 panel with the Contact’s data from Onyx database.  The button may be used to refresh the panel to display the most recent data – to include data from the Onyx record that may have changed since the panel was loaded.

Note: The Onyx 360 panel always loads the current data from Onyx database whenever an email message is selected.

The Refresh button plays a secondary role of restoring the Onyx 360 panel to its default view from other action views the user may have activated (Attachment List, Search List, and so on).  When clicked from one of the action views, the Onyx 360 panel reverts to displaying information pertaining to the email message currently selected.
