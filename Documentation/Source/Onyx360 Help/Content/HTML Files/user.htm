User Preferences Settings

Users may customize some preferences for Onyx 360 from the Settings window.  These preferences contain the log on information and preferences on saving attachments.

User ID/Password

The user’s Onyx credentials to use when logging on to Onyx.  When Integrated Windows Authentication is implemented, these fields will not be editable.

Note: The password value is encrypted before being persisted to the user’s local computer.

Profile

The OEP Profile to log on with.  The Profile setting is important for the Onyx 360 add-in to download the user preferences that the user may have set in Onyx CRM.

The profiles that the user is associated to are displayed in the drop-down list and the user must select a default profile to use.

Sync Direction

Sets the Contact synchronization between Onyx and Outlook through the Onyx 360 panel.

The following options are available:

Bi-Directional

When set, contacts are synchronized both ways – changes to a contact record in Onyx may be synchronized with the contact record Outlook and vice-versa.  This is the default setting.

Onyx to Outlook

When set, contacts are synchronized one-way. Changes to a contact record in Onyx may be synchronized with the contact record Outlook. However, changes in Outlook may not be synchronized up to Onyx.

Show Resolved Employee Records

Allows users to view internal employee records as customer records (displayed in Blue) if they exist in the Onyx individual table.  If the employee records are not tracked as customer records, this option will have no bearing on the display.

Attachment Preferences

Controls the handling of file attachments that may be found in emails or appointments saved from the Onyx 360 Add-in to the core Onyx database. 

The following settings are available:

Include when saving emails to Onyx

When enabled, automatically saves attachments with the email record in Onyx.  This setting is enabled by default.

Include when creating incidents

When enabled, automatically saves the email attachments to the Sales Opportunity/Service Request or Support Ticket (incident) record in Onyx.  This setting is enabled by default.
