Add Attachment

Allows users to add attachments included in the email message to the specified incident record in Onyx.

A list of all attachments included in the email message is provided for the user to select and the user can choose the attachment(s) to be saved to the incident record in Onyx.

Note: The Outlook user must be able to connect to the OEP File Upload Share and have write access for this feature to work.  Otherwise, the Add Attachments option is unavailable for the user.

The attached files can be viewed or accessed from the Attachments tab on the specified incident record in Onyx.
