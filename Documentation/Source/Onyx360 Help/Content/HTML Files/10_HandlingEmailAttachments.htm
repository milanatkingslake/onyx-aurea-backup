Handling Email Attachments

Attachments in emails can be easily saved to the associated records in Onyx. 

  • For information about how to add an attachment from an email message in Onyx via the Onyx 360 panel, see Contact Actions/Add Attachments and Contact Detail List Actions/Add Attachment.
  • For information on how to add an attachment from an Outlook calendar appointment in Onyx via the Onyx 360 panel, see Appointment Actions/Add Attachments.

By default, the Onyx 360 panel automatically adds all attachments along with the email message when the email is saved in Onyx.  When a Sales Opportunity/Service Request/Support Ticket record is created from an email message, all attachments in the message are associated directly on the incident record.

This feature may be turned off from the Settings page (or globally through configuration) if this is not desired.  For more information about the attachment user preference option, see Onyx 360 Configuration/User Preferences Settings.
