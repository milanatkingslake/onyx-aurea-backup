IMAP Settings

Subscription Inbox Folders

Indicates the path to users’ Inbox folder for IMAP accounts.  This setting is required for using the Email Tracking functionality on IMAP email accounts.

Subscription Outbox Folders

Indicates the path to users’ Outbox folder for IMAP accounts.  This setting is required for using the Email Tracking functionality on IMAP email accounts.

Note: For Microsoft Exchange Server, these settings will remain blank.
