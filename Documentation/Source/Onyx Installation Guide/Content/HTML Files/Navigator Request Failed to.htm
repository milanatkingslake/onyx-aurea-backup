Navigator: Request Failed to Initialize the Module

This error occurs when there are a large number of invalid session records in the database. This can happen when a user exits Onyx without properly closing their Onyx session, such as when the computer loses network connectivity or power supply.

To resolve this error:

  1. Log on to OEP as administrator, and go to Utilities > Diagnostics.
  2. Click Delete Invalid Sessions. This deletes any invalid session records in the database.

A message appears confirming that sessions are successfully deleted.

  1. Click OK, and log out of Onyx.
  2. Log on to Onyx and verify that you are able to successfully use Navigator.
