Running the Extraction Engine

Follow these steps to run the extraction engine.

To run Extraction Engine:

  1. Open command prompt in admin mode.

Example:

  1. Change the directory location to the installed Extraction Engine location. The default location is, ..\Program Files (x86)\ONYX\Extraction Engine. Example:

  1. Keeping pressing the Tab key until see KBengine.exe.

Example:

  1. Type the path in the format - /S[SQLServerName] /D[OnyxDataBaseName] /U[UserName] /P[Password] /I[SiteId] /E1

For Example, /Sdga1dbs04ox76.prod.lclad.com /DOEDB76DEV_MR7 /Usa /PP@ssw0rd /I1 /E1

Where:

  • E1 = Extraction for support incidents.
  • E2 = Extraction for service incidents.
  • E3 = Extraction for sales incidents.
  • E5 = Extraction for Work Tickets incidents.

Example:

You can also use the following switches:

  • /T (Command TimeOut): Specifies the maximum time in seconds the SQL query has to wait before receiving a response from the query engine.
  • /L (Connection Timeout): Specified the maximum time in seconds the query has to wait to before establishing a connection with the SQL server.
  • /M (Minimize logging): Specifies if the application status is logged to a text file or not.
  • /M y (Minimize logging Yes): Specifies that the application status is no longer logged to a text file.
  • /M n (Minimize logging No): Specified that the application status is logged to a text file. This is the default value.
  • /H (Halt <number>): Specifies that the extraction is halted after the specified number of worknotes are extracted.
  • /A (Template code encoding): Encodes the template code. This encoding is done occasionally.
  • /B (HTML document encoding): Encodes the HTML file. This encoding is done occasionally.
  1. Run the script.

The process completed message appears as detailed below.

  • Processing completed successfully
  • Total running time: x seconds
  • Total records extracted: x
  • Total files deleted: x

Where:

X represents numbers.

Example:

See the log file created inside the Extraction Engine folder for more details. The default location is ..\Program Files (x86)\ONYX\Extraction Engine. The log generated displays information about the process execution including number of records executed and deleted.
