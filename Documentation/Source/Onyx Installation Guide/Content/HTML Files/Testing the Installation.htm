Testing the Installation

After Email Accelerator is installed and configured, send at least one test mail to yourself before you start using the application in your production environment.

Before testing the installation:

  • Ensure that you have access to create an Individual record in Onyx and to create a campaign in OES.
  • Create an email account outside of your organization to use during testing, such as an email account with Hotmail, Yahoo, or Gmail.

To test the installation:

  1. In Onyx, create a new Individual Record and note the Individual ID.
  2. Using the OES Campaign Administration tool, create a campaign and set up the following Tracking Code types.
  • A Tracking Code ID for a Viewed email.
  • A Tracking Code ID for the Unsubscribe/Opt-out.
  • A Milestone Action ID for a Viewed Email.
  • A Milestone Action ID for the Unsubscribe/Opt-out.
