Installing Email Accelerator Server

Follow these steps to install external Web server.

To install Email Accelerator Server

  1. Navigate to Platform Components > Web Server > Email Accelerator > Setup > Email Accelerator Server, and then run EmailAcceleratorServer.exe.

The Welcome window appears.

  1. Click Next.

The Logical Application window appears.

  1. Select the required OEAS logical applications and then click Next. The default selection is Onyx.

The Timage Web Site Name window appears.

  1. Retain the name as default, and then click Next.

The Unsubscribe Web Site Name window appears.

  1. Retain the name as default, and then click Next.

The DataBase Information window appears.

  1. Type database and login information as per the examples provided in the following table.

  1. Click Next.

The Destination Folder window appears.

  1. Retain the default folder or click Change to specify the new path, and then click Next.

The Start Copying Files window appears.

  1. Verify the database and login information details, and then click Next.

The Ready to Install the Program window appears.

  1. Click Install.

When the installation is complete, the Installation Wizard Completed window appears.

  1. Click Finish.
  2. Open the Internet Information Services (IIS) Manager from Administrative Tools.
  3. Expand the Web Sites folder, click the Default Web Site, and then make sure that Timage and Unsubscribe Websites are present.
