The New Service anf folder is created.

  1. Open OnyxNotificationServiceWindowsService.exe.config.
  2. Change the base address.
  3. Specify the port number (which was specified at the time of installing the ONS).
  • Default value. <add baseAddress="http://localhost:70/"/>
  • Change to. <add baseAddress="http://localhost:<PortNo>/"/>
  1. Start the service.

To Run On Windows 2012 Server:

  1. Open MultipleSiteConfig.exe.config.

Default value. <add key="FrameworkPath" value="..\Windows\Microsoft.NET\Framework\ v4.0.30319\"/>

  1. Change to :

<add key="FrameworkPath" value="..\Windows\Microsoft.NET\Framework64\v4.0.30319\"/>
