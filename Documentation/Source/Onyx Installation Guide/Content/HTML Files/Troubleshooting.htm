Troubleshooting

This section includes information about errors that may occur when using the Onyx system, and how you can resolve those errors.

Error accessing the following registry value:

Appname = "onyx" keyname = " " valuename = "Dictionary path"

This error may when you try to add a new customer record.

To resolve this issue:

  1. Exit OEP, and then on the OEAS server, shut down the BRM COM component.
  2. Log on to OEP.
  3. Add and save the customer record.
