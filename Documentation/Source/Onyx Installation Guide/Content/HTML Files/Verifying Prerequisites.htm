Verifying Prerequisites

  1. On your OEAS server, make sure that you have a valid SSL Certificate in store.
  2. Configure HTTPS settings for the certificate in IIS and note the port number you are using for the configuration.

If you are using a common Web/App server, then ensure that you configure different ports for OEP and for the HTTPS certificate.

  1. Ensure that each iOS client device also has the same SSL certificate installed.
  2. Ensure all users are logged off from Onyx and stop the OGS service.

Most certificates issues by a trusted authority are usually pre-installed on iOS client devices. If you have a self signed security certificate, you will need to manually install this on the iOS device.
