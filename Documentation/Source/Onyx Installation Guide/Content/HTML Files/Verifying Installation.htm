Verifying Installation

After installing and configuring OEP, log on to the system to verify the installation.

If you are logged onto a web page with Windows Authentication, you cannot use the same browser session to log on to an Onyx Web page using Onyx authentication. To log on to OES (or another Onyx Web page) using Onyx authentication, you must use a new browser instance.

To verify your installation:

  1. In Internet Explorer, type the URL in the format http://<server name>/<website name>, replacing <server name> with the name of your server where you installed the OEP server components and <website name> with the name you provided for the website during installation.

If you are logged onto a web page with Windows Authentication, you cannot use the same browser session. The website name may differ depending on whether you plan to use Onyx authentication, or Windows authentication.

The Onyx Log-in page appears, indicating a successful installation.

  1. Enter the user ID and password, and then click OK.
  2. Select the profile, and then click OK.
  3. To verify Navigator configuration, on the Header bar, click the Navigator icon.

The Navigator page appears with the Search Criteria panel, indicating a successful configuration.
