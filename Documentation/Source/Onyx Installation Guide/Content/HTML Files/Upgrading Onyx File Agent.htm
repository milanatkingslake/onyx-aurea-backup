Upgrading Onyx File Agent

Perform COM+Shutdown before upgrading the Onyx File Agent.

Perform the following steps to upgrade Onyx File Agent.

To Upgrade Onyx File Agent

  1. Navigate to Platform Components > Web Server > File Agent (OFA), and then run OnyxFileAgent.exe.

The Onyx File Agent confirmation window appears.

  1. Click Yes to upgrade Onyx File Agent.

The Resuming the Installation window appears.

  1. Click Next.

After the installation process is complete, the Installation Wizard Completed window appears.

  1. Click Finish.
