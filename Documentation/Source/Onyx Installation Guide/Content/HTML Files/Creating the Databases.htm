Creating the Databases

You can use the \Utilities\Database Setup Script\DbaSetupScriptFreshDB.sql script to create fresh Onyx transaction and Persistence databases, and related database user accounts with required permissions.

To edit the installation script for fresh databases

  1. Browse to \Utilities\Database Setup Script\ and open the DbaSetupScriptFreshDB.sql file in any text editor.

  1. Specify the variable values as following:
  • @database :  Name of the Onyx Transaction database
  • @onyxDBO: Onyx Database owner user name
  • @passwordDBO: Password for the onyx DBO user
  • @onyxEASSql: OEAS user who has privileges to perform data transactions
  • @passwordEAS: Password for OEAS username
  • @onyxLMSql: User who has permissions use the List Manager in OEP
  • @passwordLM: Password for the LM user
  • @persistenceDB: Name of the Onyx Persistence Database
  • @passwordEMFDBO: Password for the Event Management Framework database owner
  1. Save the script.

To run the edited script

  1. Open SQL Server Management Studio and select the master database.
  2. Click File > Open > File.
  3. Browse to \Utilities\Database Setup Script, select DbaSetupScriptFreshDB.sql.
  4. Click Open.
  5. Click Execute to run the script.
