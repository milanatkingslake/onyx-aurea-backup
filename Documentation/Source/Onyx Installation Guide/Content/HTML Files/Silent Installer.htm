Silent Installer

Before running OEAS and OGS Silent Installer Script, create the OnyxEASUser and OGSUser with Administrative privilege.

To install Onyx using Silent Installer you need to follow the steps mentioned below:

  • Copy the respective components to the server on which you want to install Onyx .
  • Open each .vbs file for the respective components and provide required parameters on the top as mentioned later in the tables of this document.
  • Open the Command Prompt as Run as Administrator.
  • Navigate to the path of the .vbs file for respective components and run.

Any parameter that is mentioned as NO in the Required column of the details table is included in case of change of default values in the .vbs file.
