Installing Onyx Employee Portal

OEP, which is the user interface component (presentation tier) of your Onyx system, enables members of your organization to save, view, modify, analyze, and use the customer information stored in the Onyx database.

Perform the OEP (64-bit) server installation after you have installed and configured the Onyx database, OEAS,OEAS Config, OGS, OES, and optionally, OPS.

OEP 32 bit is not supported on Onyx 7.7 or later versions.
