Installing Onyx Email Accelerator Client

Follow these steps to install Onyx Email Accelerator Client.

To install Onyx Email Accelerator Client:

  1. Navigate to Platform Components > Web Server > Email Accelerator > Setup > Email Accelerator Client, and then run EmailAcceleratorClient.exe.

The Welcome window appears.

  1. Click Next.

The Email Accelerator Web Site Name window appears.

  1. Retain the Name as default, and then click Next.

The DataBase Information window appears.

  1. Type database and login information as per the examples provided in the following table.

  1. Click Next.

The Destination Folder window appears.

  1. Retain the default path or click Change to specify the new path, and then click Next.

The Start Copying Files window appears.

  1. Verify the Website name and the Directory path, and then click Next.

The Ready to Install the Program window appears.

  1. Click Install.

When the installation process is complete, the Installation Wizard Completed window appears.

  1. Click Finish.
