Installing OEAS Config

Installing OEAS Config is required for you to configure the transactional database and the logical application. It also helps you to create the users, who have access to middle-tier.

Steps to Install OEAS Config:

  1. Copy the OEASConfig folder to your OEAS server from Platform Components > Application Server (OEAS) from the Onyx installation package.
  2. Open the OEASConfig folder from your OEAS server, and then run OEASConfig setup.

The Microsoft SQL Server 2012 Native Client Welcome window appears.

If the Microsoft SQL Server Native Client is already installed, the OEAS Config Welcome window appears on page 1.

  1. Click Next.

The License Agreement window appears.

  1. Select I accept the terms in the license agreement to continue and click Next.

The Feature Selection window appears.

  1. Select Client Components and click Next.

The Ready to Install the Program window appears.

  1. Click Install.

Installation of the files starts and the Installation Completed window appears after installation is completed.

  1. Click Finish.

The OEAS Config Welcome window appears.

  1. Click Next.

The OEAS Config window appears.

In the OEAS Config window, the Logical application name is displayed as Onyx.

  1. Click Next.

The Database Server window appears.

  1. Type the details to connect to the SQL Server.
  1. Database Server. Type the server name where the Onyx database is located.
  2. Login. Accept the default log on ID or type another logon ID with DBO permissions.
  3. Password. Type the password for the log on ID.
  4. OEDB Name. Type the Onyx Transaction database name.
  1. Click Next.

The SQL Account window appears.

  1. Type the password for OnyxEASSQL.

The OEAS config installer does not validate the password for OnyxEasSql user. If a wrong password is given, it will still install the OEAS config. After that OEP will not be installed properly. If you have given a wrong password and want to change it again, run the Onyx Update Password tool and change the password. For more details see Updating Onyx Password on page 1.

  1. Click Next.

The Ready to Install the Program window appears.

  1. Click Install.

Installation of the files starts and the Installation Wizard Completed window appears after installation is completed.

  1. Click Finish.
