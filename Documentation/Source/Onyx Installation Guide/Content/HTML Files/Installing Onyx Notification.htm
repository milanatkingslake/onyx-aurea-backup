Installing Onyx Notification Service

To install Onyx Notification Service (ONS)

  1. Copy the OnyxNotificationService folder to your OEAS server from Platform Components > Application Server (OEAS) from the Onyx installation package.
  2. Open the OnyxNotificationService folder on your OEAS server, and then run OnyxNotificationService.exe.

The Welcome window appears.

  1. Click Next.

The Logical Application window appears with Onyx as a default logical application name selected.

  1. Click Next.

The DataBase Information window appears.


  1. Do the following:
  1. Server. Type the database server name.
  2. OTDB. Type the Onyx Transactional Database Name.
  3. OPDB. Type the Onyx Persistent Database Name.
  4. Login. Type the SQL user name. The default is Onyx DBO.
  5. Password. Type the SQL password for Onyx DBO.
  6. Port #. Type the value of the port ONS should use.
  1. Click Next.

The Exchange Server Information window appears.

  1. Do the following:
  1. URI.
  1. Enable SSL.
  2. Default Credentials.
  1. Login.
  2. Password.
  3. Domain.
  1. Click Next.

The Logon Information window appears.

  1. Type the required Username in the Domain\Username format and Password for ONS to use.
  2. Click Next.

The Ready to Install the Program window appears.

  1. Click Install.

ONS starts the installation. After completion, the Installation Wizard Completed window appears.

  1. Click Finish.
