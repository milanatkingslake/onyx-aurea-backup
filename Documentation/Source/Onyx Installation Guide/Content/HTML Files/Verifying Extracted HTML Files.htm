Verifying Extracted HTML Files

Follow these steps to verify the extracted HTML files.

To verify extracted HTML files:

  1. After the extractions are completed, go to the machine where Network folders (Sales, Service, Support, and QA) are shared.

For example:

  • ..\OnyxLibraries\Sales
  • ..\OnyxLibraries\Service
  • ..\OnyxLibraries\Support
  • ..\OnyxLibraries\QA
  1. After complete execution of KBengine.exe, make sure that the extracted incident records are stored in HTML format in their respective folders.
