Uninstalling Windows Service for Onyx Notification Service

  1. Open the ONSUnInstall.bat file.
  2. Set the arguments values in the ONSUnInstall.bat file.
Arguments Description Default Value Example

arg1

Windows Service Name

Onyx.ONS1

 

arg2

Windows Service Exe

OnyxNotificationServiceWindowsService.exe

 
  1. Save the ONSUnInstall.bat file.
  2. Run the ONSUnInstall.bat file.
  3. Verify whether the Services.msc OGS service is deleted.
