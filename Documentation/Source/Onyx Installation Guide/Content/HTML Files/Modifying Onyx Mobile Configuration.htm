Modifying Onyx Mobile Configuration

  1. On each iOS /Android client device, go to the Onyx Configuration window, and then set the Use SSL option to On.
  2. Rename the following tag values in the Profiles folder. It is available in the following path:

..\Program Files (x86)\Onyx\AppServer\Applications\Onyx\OnyxGatewayService\App_Data\Profiles.
