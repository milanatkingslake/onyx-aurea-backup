Uninstalling Onyx 360 Server

Uninstall the Onyx 360 Server from Control Panel.

After uninstalling Onyx 360 Server perform the following steps:

  1. Login to OES.
  2. Open Object Designer.
  3. Expand Logical Business Objects.
  4. Disable below mentioned objects:
  • emailConversation
  • outlookSyncContact
  1. Disable below mentioned Properties and Methods under the appointment objects:
  • Property:
  • hashId
  • Methods:
  • findMatch
  1. Disable below mentioned Properties and Methods under the emailMessage objects:
  • Properties:
  • hashId
  • sendEmail
  • Methods:
  • findMatch
