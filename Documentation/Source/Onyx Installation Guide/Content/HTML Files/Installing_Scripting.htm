Installing Onyx Scripting

To install Onyx Scripting

  1. Copy the OnyxScripting folder to your server from Platform Components > Web server from the Onyx installation package.
  2. Run OnyxScripting.exe.

The OnyxScripting Welcome window appears.

  1. Click Next.

The Onyx Authentication Web Site Name window appears.

  1. Type the name of the website that your organization uses to access OnyxScripting or retain the default website name.
  2. Click Next.

The Windows Authentication Web Site Name window appears.

  1. Type the name of the website that your organization uses to access OnyxScripting for Windows authentication or retain the default website name.
  2. Click Next.

The Destination Folder window appears.

  1. Retain the default destination or click Change to select a different destination, and then click Next.

The Start Copying Files window appears.

  1. Click Next.

The Ready to Install the Program window appears.

  1. Click Install.

When the installation process is complete, the Installation Wizard Completed window appears.

  1. Click Finish.

Post Installation Steps

  1. In OES, navigate to System Parameter Admininstration.
  2. Update the parameter value of the ScriptingWebsiteName parameter code. Change localhost to the webserver name in http://localhost/OnyxScripting_Onyx/index.html.
