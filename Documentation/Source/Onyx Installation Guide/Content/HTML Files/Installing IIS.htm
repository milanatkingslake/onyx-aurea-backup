Installing IIS

This topic explains the procedures to install IIS.

Prerequisites:

  • MSXML 4.0 SP2 should be installed.
  • MDAC 2.8 should be installed.
  • Enable the Windows feature IIS by following the steps explained below.
  1. Go to Control Panel > Programs and Features.
  2. In the Control Panel Home pane, click Turn windows features on/off.

The Server Manager window appears.

  1. Under Roles summary, click Add Roles.
  2. In the Add Roles and Features wizard window, select Role-based or feature-based installation and click Next.
  3. Select a server from the server pool and click Next.
  4. Click the Web Server (IIS) drop-down arrow and then select the roles as mentioned in the following table.

 

When you click ASP.net, click Add Required Role Services in the dialog box appearing next.
