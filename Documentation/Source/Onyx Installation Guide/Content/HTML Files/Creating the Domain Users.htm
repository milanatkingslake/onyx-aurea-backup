Creating the Domain Users

  1. Create a Domain User. To do this: Login in to the Exchange server you may have assigned
  2. Double click on the Active Directory Users and Computers shortcut.
  1. If no shortcut in the desktop, go to Start, select Programs/ Microsoft Exchange, select Active
  2. Directory Users and Computers.
  1. On the left pane, under Active Directory Users and Computers root, click on the domain icon (would look like “lab.onyx.com”).


  1. Expand the tree; and right-click on Users folder.
  2. Select New then User.
  3. The New Object – User wizard opens. Fill the fields: First Name, Last name and User Logon name (e.g. chandan) and make a note of this. Click Next. 

  1. Define a password and confirm it, (“onyx “is recommended).
  2. Uncheck User must change password at next logon and check Password never expires.


  1. Click Next.
  2. Verify that Create an Exchange Mailbox is checked. Click Next.

  1. Click Finish.
  2. Restart the App and Web servers.
