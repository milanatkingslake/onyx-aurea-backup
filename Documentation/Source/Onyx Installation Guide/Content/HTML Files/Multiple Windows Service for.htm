Multiple Windows Service for Onyx Gateway Service

  1. Run MultipleSiteConfig.exe.

Source path. ..\Program Files\Onyx\AppServer\ Applications\ Onyx\OnyxGatewayService

Destination path. ..\Program Files\Onyx\AppServer\Applications\Onyx1

  1. Create a folder. For example, OnyxGatewayService.
  2. In Windows Service, click the Onyx Gateway Service option.
