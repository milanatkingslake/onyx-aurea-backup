Verifying OPS Installation

  1. Launch the OPS using the URL that was just created and verify that the service status is Started.
  2. Expand Connections and then expand the <Site Name> (<Site ID>) node. For example, in the following image Onyx (Site 1) is displayed.

  1. Expand <Site Name> (<Site ID>) node and verify the following nodes are listed:
  • Fulfillment Subsystems
  • Fulfillment Queues
  • Batch Subsystems
  • Batch Queues
  • Email Monitor Subsystems
  • Synchronization Subsystems
