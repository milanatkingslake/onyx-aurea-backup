Modifying OEP Configuration File

  1. On your OEP server, back up the config.xml file in case you need to revert to it. The default file location is ..\Program Files (x86) \Onyx\EmployeePortal\CRArchitecture\Configurations\config.xml.
  2. Open the config.xml file, and in the <add baseAddress> node, and change the URI scheme from http:// to https://.
  3. Save and close the config.xml file.
