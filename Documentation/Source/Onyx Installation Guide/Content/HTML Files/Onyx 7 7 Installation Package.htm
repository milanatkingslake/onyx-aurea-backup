Onyx Installation Package

The Onyx release comes as a single installation package containing all updated components. The installation package includes the following folders and files that you can use to install, configure, and customize the Onyx Enterprise Application.

Root Folder:

Onyx  Release Notes. Also includes sub-folders that contain the upgraded demo database, customization tools, documentation, and the installation packages for platform components.

  • Client Components
  • Onyx Email for Outlook: Contains the email add in for Microsoft Outlook.
  • Customization Support
  • Application Server (OEAS): Contains resources to help you customize and localize your Onyx system.
  • LBOAdapterTemplate
  • OGSProxyWeb
  • OGSProxyWindows
  • OnyxGatewayService
  • OnyxNotificationService
  • Database Server
  • Demo Database: Contains upgraded Onyx Demo Databases along with the SQL script to set DBO permissions, if needed.
  • Navigator Custom Entity Scripts: Contains scripts to create custom entities in the Navigator.
  • Mobile Bookmark: Contains SQL scripts to create custom mobile bookmarks.
  • Onyx 360 Client: Contains resource files for English and Japanese languages.
  • Documentation: Contains help and technical documentation updated for this release.
  • Platform Components: Contains setup and installation files to upgrade the Onyx database, application server, and Web server components.
  • Analytics: Contains setup and installation files to install Analytics.
  • App Files: Contains Android and iOS Onyx Mobile Native Application files.
  • Application Server (OEAS)
  • OEASConfig: Contains setup and installation files to install OEAS Config.
  • OGSProxy: Contains setup to install OGSProxy.
  • OnyxGatewayService : Contains setup to install OGS.
  • OnyxNotificationService: Contains setup to install ONS.
  • Setup: Contains setup and installation files to install OEAS.
  • Upgrade OED: Contains setup and installation files to upgrade OED.
  • Database Server: Contains setup and installation files to upgrade the databases.
  • EMF Server: Contains notification process file.
  • Extraction Engine: Contains setup and installation files to install Extraction Engine.
  • Onyx 360
  • Client: Contains setup and installation files to install Onyx 360 Client.
  • Server: Contains setup and installation files to install Onyx 360 Server.
  • Onyx Prerequisite: Contains setup and installation files to install Onyx Prerequisite.
  • Web Server
  • CTI: Contains setup and installation files to install CTI.
  • Email Accelerator: Contains setup and installation files to install Emai Accelerator.
  • File Agent (OFA): Contains setup to install OFA.
  • Onyx Enterprise Studio: Contains setup and installation files to install Onyx Enterprise Studio.
  • Onyx For Desktop: Contains setup and installation files to install Onyx Employee Portal.
  • Onyx Insight: Contains setup and configuration files to install Onyx Insight.
  • Onyx Processing Server: Contains setup and installation files to install Onyx Processing Server.
  • Onyx Web Services: Contains setup to install Onyx Web Services.
  • OnyxMobileWeb: Contains setup and installation files to install Onyx Mobile Web.
  • OnyxScripting: Contains setup to install Onyx Scripting.
