Installing Extraction Engine

To install Extraction Engine in Onyx

  1. Navigate to the Platform Components > Extraction Engine folder.

You can install Extraction Engine on an App server, DB server, or on a separate server machine.

  1. Run ExtractionEngine.exe.

The Welcome window appears.

  1. Click Next.

The Database Server window appears.

  1. Provide the following information:
  • Database server: Type the name of the OEDB server.
  • Login ID: Type the user name having the database owner privilege.
  • Password: Type the password for the login.
  • Name of OEDB database catalog: Type the OEDB database name.
  1. Click Next.

The Destination Folder window appears.

  1. Retain the defalult path or click Change to specify the new path.
  2. Click Next.

The Ready to Install the Program window appears.

  1. Click Install.

When the installation process is complete, the Installation Wizard Completed window appears.

  1. Click Finish.
