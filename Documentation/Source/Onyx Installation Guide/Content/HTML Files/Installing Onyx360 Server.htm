Installing Onyx360 Server

Before running this setup, make sure to install Onyx Web Services and .Net Framework 4.0 or later.

To install Onyx 360 server

  1. Navigate to Platform Components > Onyx 360 > Server, and then run Onyx360Server.exe.

The Welcome window appears.

  1. Click Next.

The Custom Setup window appears.

  1. Select the component to install and click Next.

The Database Server window appears.

  1. Specify the following details and click Next.
  • Database server: Name or IP address of the Onyx Enterprise Database server computer.
  • LoginID: OnyxDBO login name.
  • Password: Password for OnyxDBO.
  • Name of OEDB database catalog: Name of the Onyx Enterprise Database catalog.

The Logical Application window appears.

  1. Select the OEAS Logical Application name for Onyx 360 and click Next.

The Destination Folder for OEP window appears.

  1. Specify a destination folder or accept the default.
  2. Click Next.

The Ready to Install the Program window appears.

  1. Click Install.

When the installation is complete, the Installation Wizard Completed window appears.

  1. Click Finish.
