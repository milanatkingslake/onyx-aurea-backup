Installing IIS in Onyx Email Accelerator Client Server

Follow these steps to install IIS in Onyx Email Accelerator Client Server.

To install IIS in Onyx Email Accelerator Client Server:

  1. Go to Control Panel > Programs and Features.
  2. In the Control Panel Home pane, click Turn windows features on/off.

The Server Manager window appears.

  1. Under Roles summary, click Add Roles.
  2. In the Add Roles and Features wizard, select Role-based or feature-based installation and click Next.
  3. Select a server from the server pool and click Next.
  4. Select the Web Server (IIS) check box and in the dialog box appearing next, click Add Required Features.

  1. Select all the required features.
  2. In the Add Roles and Features wizard, click Role Services under Web Server(IIS), and then select the roles as mentioned in the next page.

When you click ASP.net, click Add Required Role Services in the dialog box appearing next.
