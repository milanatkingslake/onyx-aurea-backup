Running CTI.exe as Administrator

  1. Navigate to the Platform Components > Web Server > CTI folder.
  2. Run CTI.exe.

The Welcome window appears.

  1. Click Next.

The Custom Setup window appears.

Upgrading OEDB to Support CTI

  • If the OEAS and OEP servers are installed in the same server, you can then select all the three features.

 

  • If the OEAS and OEP servers are installed in a different server, copy setup.exe to the OEAS server, and then clear Upgrade OEP as shown in the image above.
  1. Click Next.
  1. If Upgrade OEDB is selected, the Database Server window appears.

Specify the database and log on information as per environment.

  • Database server. Type the database server name. If you are running the setup in the database server, type the name as localhost. Otherwise, if the DB server is in a different machine, type the corresponding machine name or its IP address.
  • Login ID. Type the user that has owner rights to the database.
  • Password. Type the user password.
  • Name of OEDB database catalog. Type the OEAS database name.
  1. If Upgrade OEAS is selected, the Logical Application window appears.

Select the logical application for which you want to install CTI.

  1. If Upgrade OEP is selected, the Destination Folder for OEP window appears.

Leave the default path or click Change to specify the path where the CTI files will be added.

  1. Click Next.

The Ready to Install the Program window appears.

  1. Click Install.

When the installation process is complete, the Installation Wizard Completed window appears.

  1. Click Finish.
