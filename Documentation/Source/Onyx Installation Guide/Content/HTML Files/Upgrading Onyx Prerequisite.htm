Upgrading Onyx Prerequisite

Follow the steps explained below to upgrade Onyx Prerequisite.

To Upgrade Onyx Prerequisite

  1. Navigate to Platform Components > Onyx Prerequisite, and then run OnyxPrerequisite.exe.

The Onyx Prerequisite confirmation window appears.

  1. Click Yes to upgrade Onyx Prerequisite.

The Resuming the Installation window appears.

  1. Click Next.

After the installation is complete, the Installation Wizard Completed window appears.

  1. Click Finish.
