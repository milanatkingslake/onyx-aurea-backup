Email Accelerator Does Not Work

If your Email server is not a local SMTP server, you must modify Email Accelerator settings based on your Email server configuration.

To resolve this:

  1. In your Email Accelerator client installation folder, open the Client.asp file . The default folder path is ..\inetpub\wwwroot\OnyxEmailAccelerator.
  2. Search for objMsg.Configuration.Fields.Item ("http://schemas.microsoft.com/cdo/configuration/sendusing"), and set the value based on your Email server configuration.
  • Set 1 if you are using a local SMTP server.
  • Set 2 if you are using an external SMTP server.
  • Set 3 if you are using an Exchange server.
  1. Save and close the file.
