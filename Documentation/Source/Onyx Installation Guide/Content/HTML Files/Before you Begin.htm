Before you Begin

  • For an overview of the contents of your Onyx installation package, see Onyx Installation Package.
  • For system requirements that have changed for this release, see Onyx System Requirements.
  • For all other system requirements, refer to the Onyx System Requirements Guide.

Ensure that you follow the suggested practices while installing Onyx:

  • Run the installers as an administrator.
  • Before starting the upgrade, it is recommended that you take a backup of your existing Onyx system in case you need to revert to it for any reason.
