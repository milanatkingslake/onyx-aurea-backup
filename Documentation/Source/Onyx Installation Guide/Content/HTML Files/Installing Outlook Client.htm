Installing Outlook Client for Onyx 360

Follow the steps explained below to install Outlook Client for Onyx 360.

To install Outlook Client for Onyx 360

  1. Navigate to the Platform Components > Web Server > Onyx 360 > Client folder.

Onyx 360 is supported for both 32-bit and 64-bit of Outlook.

If Microsoft VSTO 2010 for Office Runtime is not installed, the Onyx360Client.exe will install it.

  1. Run Onyx360Client.exe.

The Prerequisites window appears.

  1. To install the missing prerequisite component, click Install.

The Welcome window appears.

  1. Click Next.

The Selection Type window appears.

  1. Select the Installation mode and click Next.

The Onyx 360 Configuration window appears.

  1. In the Onyx 360 Configuration window, provide the following information.

OEASAuthOnyxURL: Type the OWS URL used to establish the session under Onyx authentication.

OEASOnyxURL: Type the OWS URL used for transaction under Onyx authentication.

OEPLauncherURL: Type the OEP URL used to launch the PowerPage associated record.

OEASAuthWinURL: Type the OWS URL used to establish the session under Windows authentication.

OEASWinURL: Type the OWS URL used for transaction under Windows authentication.

  1. Click Next.

The Onyx 360 Application Configuration window appears.

  1. In the Onyx 360 Application Configuration window, provide the following information.

Application Name: Type the logical application name.

Site ID: Type the Onyx database site ID.

Enable Onyx Authentication: Select the Enable Onyx Authentication check box to run the application under Onyx authentication or clear the Enable Onyx Authentication check box to run the application under windows authentication.

Log To File: Select the Log To File check box to generate the log files or clear the Log To File check box if the generation of log files is not required.

Log Level: Select the required log level.

Log File Retention Days: Type the log file retention days.

Log Onyx Admin Email: Type the admin email ID to send the log files.

  1. Click Next.

The Ready to Install the Program window appears.

  1. Click Install.

When the installation process is complete, the Installation Wizard Completed window appears.

  1. Click Finish.

Customizing Onyx360 for Japanese language (Localization):

By default onyx 360 is configured in English. If we need to change the language from English to Japanese, we need to replace the dll (OnyxOutlookAddin.resources.dll) from the location: ..\Program Files (x86)\Onyx\Outlook360Addin\ with the new dll with the same name built with Japanese language.

Troubleshooting

  • In the actions of an Individual entity, if the Add Attachment option is displayed, but unavailable, do the following:
  • Add the credentials of the Client Machine to the Administrator Group of the Server Machine.
  • When Onyx 360 Ribbon does not appear, do one of the following:
  • Run the Outlook as administrator.
  • Make sure that the registry contains only one entry in the Outlook.
