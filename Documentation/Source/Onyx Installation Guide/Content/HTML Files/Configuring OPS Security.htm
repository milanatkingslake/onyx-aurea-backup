Configuring OPS Security

  1. Launch OES.
  2. Click the Security Administration link.
  3. Expand the Administrators group.


  1. In the Users node, verify that the following information exist for the Onyx Processing Service User:
  • User :  OPSSvcUser
  • User Name: Onyx Processing Service User
  • Allow impersonation: Is selected
  1. In the Onyx Authentication section, verify that the Onyx Authentication check box is cleared.
  2. In the Windows Integrated Authentication section, verify that the Windows Integrated Authentication check box is selected. Verify that the domain Account entered is the domain account created in the Creating the Domain User procedure.
  3. Save details.
  4. In the Users node, create an OPS administrator user, by filling the following information:

  • User ID:  OPSAdmin
  • User Name: OPS Administrator.
  • Email: Email ID of the user created during the Creating the Domain User procedure.
  • Ensure that the Allow impersonation checkbox is selected.
  1. In the Onyx Authentication section, ensure that the Onyx Authentication check box is selected.
  2. In the Windows Integrated Authentication section, esnure that the Windows Integrated Authentication check box is cleared.
  3. Save the details.
  4. Reset IIS on the App and Web servers.
  5. Restart the OnyxEASApp & System Application COM+ Services. To do this:


  1. On the OPS server click Star > Administrative Tools > Component Services.
  2. In the Component Services window, click Computers, click My Computer, and then click COM+ Applications.
  3. Right-click OnyxEASApp, and select Shut Down.
  4. Right-click OnyxEASApp and select Start option.
