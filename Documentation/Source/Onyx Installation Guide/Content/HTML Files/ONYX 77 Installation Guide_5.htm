Installing Onyx Mobile

To install Onyx Mobile for Onyx

  1. From the Apple App Store, download and install the latest version of the Onyx Mobile app for your iOS device.

OR

From the Google Play Store, download and install the latest version of the Onyx Mobile app for your Android device.

The app names are:

Device App Name

iPhone

Onyx Mobile

iPad

Onyx Mobile Plus

Android mobile and Android Tab.
