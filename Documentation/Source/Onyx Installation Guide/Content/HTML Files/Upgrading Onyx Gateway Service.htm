Upgrading Onyx Gateway Service

Perform the following the steps to upgrade Onyx Gateway Service.

To Upgrade Onyx Gateway Service

  1. Navigate to Platform Components > Application Server (OEAS) >OnyxGatewayService, and then run OnyxGatewayService.exe.

The Onyx Gateway Service confirmation window appears.

  1. Click Yes to upgrade Onyx Gateway Service.

The Resuming the Installation window appears.

  1. Click Next.

The Files in Use window appears.

  1. Select Automatically cloxe and attempt to restart applicaton, and click OK.

After the installation is complete, the Installation Wizard Completed window appears.

  1. Click Finish.
  2. Open the OnyxGatewayWindowsService.exe.config file from the \Customization Support\Application Server (OEAS)\OnyxGatewayService folder in release folder and \Program Files\Onyx\AppServer\Applications\Onyx\OnyxGatewayService folder of the machine that has OGS installed.

Take a back up of the OnyxGatewayWindowsService.exe.config file from the \Program Files\Onyx\AppServer\Applications\Onyx\OnyxGatewayService folder.

  1. Compare the OnyxGatewayWindowsService.exe.config file from both directories and merge the required changes from the file present in the \Customization Support\Application Server (OEAS)\OnyxGatewayService folder into the file present in the \Program Files\Onyx\AppServer\Applications\Onyx\OnyxGatewayService folder.

When you merge the changes, it is required to retain the following information in the OnyxGatewayWindowsService.exe.config file, which is present in the \Program Files\Onyx\AppServer\Applications\Onyx\OnyxGatewayService folder.:
• The key="extraData" and key="InstanceName" attributes in the <appSettings> node.
• The <connectionStrings> node.
• Custom <endpoint> node, if any.
