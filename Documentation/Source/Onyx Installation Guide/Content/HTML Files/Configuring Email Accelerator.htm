Configuring Email Accelerator for Sending Mails through LocalHost

Follow these steps to configure Email Accelerator to send mails through LocalHost.

To configure Email Accelerator to send mails through LocalHost:

  1. Right-click the SMTP virtual directory.
  2. Click the Access tab and then click Connection.
  3. Select only the list below.
  4. Add the local host ip address.

Example:

If the Exchange or SMTP sever is available, provide the name in the AcceleratorClient.config in ..\Program Files (x86)\Onyx\Onyx Email Accelerator within the following node.

<SMTPServer>

        SMTP server name

    </SMTPServer>

</Configuration>
