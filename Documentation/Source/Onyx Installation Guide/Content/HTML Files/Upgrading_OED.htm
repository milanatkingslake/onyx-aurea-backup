Upgrading Onyx Enterprise Dictionary

Perform the following steps to upgrade Onyx Enterprise Dictionary.

To Upgrade Onyx Enterprise Dictionary

  1. Navigate to Platform Components > Application Server (OEAS)> Upgrade OED, and run OEDUpgrade.exe.

The OED Upgrade Welcome window appears.

  1. Click Next.

The Logical Application window appears.

  1. Click Next.

The Ready to Install the Program window appears.

  1. Click Install.

When the installation process is complete, the Installation Wizard Completed window appears.

  1. Click Finish.
