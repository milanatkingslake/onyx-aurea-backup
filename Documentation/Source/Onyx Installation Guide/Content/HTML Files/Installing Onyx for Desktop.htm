Installing Onyx for Desktop

Be sure to have the following information ready before you begin:

  • The name of OEAS to connect to.
  • The location of OEAS, whether OEAS is located on the same system that you are installing OEP, or on a remote system.
  • The name of the OEAS logical application. This ensures that the OEP accesses the correct Onyx database.
  • The Onyx database site ID.
  • The SQL account log on ID and password that will be used to access the Onyx database. The user ID can be up to a maximum of 40 characters long, and the password can be up to a maximum of 15 characters.

After performing the setup, you can change the user ID or password to a length exceeding these values. For information on doing this, see "Changing the OEP system user ID or password" in the Onyx Technical Guide.

  • The name of the website that you will use to access OEP using Onyx authentication. This is the name you will use to connect to the UI.
  • If you plan to use Windows authentication, keep the following information ready:
  • The name of the website that you will use to access OEP using Windows authentication. This is the URL you will use to connect to the UI.
  • If you plan to change the name of the default Local Group, the name that you intend to use.
  • The directory location to install the OEP (64-bit) program files. The default location is ..\Program Files\Onyx\Employee Portal.
  • Be sure that you have logged on to your Web server using an account with administrator privileges.

If the Web server is a separate computer from the OEAS, then you must use a domain account that has administrator privileges on both computers. Setup will automatically stop and restart the Internet Information Server as needed during and after installation.

The OEP (64-bit) installer contains updated OEP for .

  1. Close all open applications.
  2. In your Onyx installation package, navigate to Platform Components > Web Server > Onyx for Desktop > OEPx64, and then run EmployeePortal.exe.

The Welcome window appears.

  1. Click Next.

The Logical Applications window appears.

  1. Select the logical application that you want OEP to use.
  2. Click Next.

The Web Site Default Site ID window appears.

  1. Enter the default Onyx Database site ID number for the OEP server to use when connecting to OEAS.
  2. Click Next.

The SQL Account window appears.

  1. Do the following:
  1. Login. Type the Login name or retain the existing login (that is, OEPsa) that will be used to access OEAS at system startup.
  2. Password. Type the password.
  3. Confirm. Repeat the password to confirm.
  1. Click Next.

The Onyx Authentication Web Site Name window appears.

  1. Type the name of the website that your organization uses to access OEP or retain the default website name.
  2. Click Next.

The Windows Authentication Web Site Name window appears.

  1. Type the name of the website that your organization uses to access OEP for Windows authentication or retain the default website name.
  2. Click Next.

The Setup Type window appears.

  1. Select the environment to install OEP.
  • Click the Development option if you are installing OEP to a development environment.
  • Click the Production option if you are installing OEP to a test or a production environment.
  1. Click Next.
  • If you have selected Production in the previous step, the Destination Folder window appears. To continue, go to Step-17.
  • If you have selected Development in the previous step, the Local Group window appears.

  1. Accept the default group name (OnyxUCWUsers) for the local account group. This field cannot be modified.
  2. Click Next.

The Destination Folder window appears where you specify the location to install OEP.

  1. Accept the location displayed in the window, or click Change to select a different location, and then click Next.

The Start Copying Files window appears. This window displays the data entered during setup, along with additional information that the Setup program uses to complete the installation.

  1. Verify that the information displayed is accurate, and then click Next.

To change any of the information displayed, revert to the preceding windows in the wizard and change the information as required.

The Ready to Install the program window appears.

  1. Click Install.

When the installation process is complete, the Installation Wizard Completed window appears.

  1. Click Finish to complete Setup.
  2. Configure server settings to access Navigator. For information, see Configuring Navigator.

Ensure that you provide the following UI resource permissions to the OEPsa user before browsing OEP:
- UI:OEP.logon
