Upgrading OEAS

Onyx OEAS runs on Windows Server 2012, or Windows Server 2012 R2.

To Upgrade OEAS:

  1. Navigate to Platform Components >Application Server (OEAS) > Setup, and then run OnyxEnterpriseApplicationServer.exe.

The OEAS confirmation window appears.

  1. Click Yes to upgrade OEAS.

The Resuming the Installation window appears.

 

  1. Click Next.

After the installation is complete, the Installation Wizard Completed window appears.

  1. Click Finish.

To use the S3 Utility in Onyx Mobile Native Apps, perform the following steps after the OEAS upgrade:
1. Navigate to the \Customization Support\Application Server (OEAS)\Applications in the release folder.
2. Manually copy and paste the Configuration folder to the \Program Files\Onyx\AppServer\Applications\<Logical Application Folder> folder of the machine that has Onyx CRM installed.
3. Perform COM+Shutdown.
