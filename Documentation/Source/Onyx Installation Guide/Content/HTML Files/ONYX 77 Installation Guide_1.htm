Installing OEAS

Onyx OEAS runs on 2012 or 2012 R2 server. Perform a full setup as explained in this topic.

To set up the application server:

  1. In your Onyx installation package, navigate to Platform Components >Application Server (OEAS) > Setup, and then run OnyxEnterpriseApplicationServer.exe.

The Welcome window appears.

  1. Click Next.

The Logon Information window appears.

The Setup program uses the information that you provide in this window to create the account used for the COM+ component’s identity. This account enables the COM+ components to run and is assigned to the Power Users group. If you subsequently uninstall the application server, this local account remains.

  1. In the User name box, retain the default OnyxEASUser user or specify another local or domain account.
  2. In the Password box, type a password that conforms to the security policies of the computer domain.
  3. Click Next.

The Destination Folder window appears.

  1. Specify the location for installation of the application server components and DLLs.

If the folder you specify contains a set of application server DLLs or application server logical application folders, you are prompted to specify a different location.

  1. Accept the location displayed in the window or browse to select a different location, and then click Next.

The Ready to install the Program windows appears.

  1. Click Install.

Installation of the files starts and the Installation Wizard Completed window appears after installation is completed.

  1. Click Finish.

Onyx does not support installing the application server and the enterprise studio to the same directory.
