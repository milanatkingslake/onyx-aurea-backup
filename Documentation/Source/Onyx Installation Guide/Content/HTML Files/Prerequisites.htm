Prerequisites

Ensure that you have the following:

  • Windows 2012 Enterprise/Standard Edition and its Service Packs.

For Windows 2012, apply the Microsoft fix which is available at http://support.microsoft.com/kb/2954953/. If this fix not applied the following error will be thrown “PrintTicket” provider failed to bind to printer. Win32 error: - “2147467231”.

  • OES/OEP (Refer to the OES and OEP Test Procedures). This computer must be in the same or in a different domain as the MS Exchange Server.
  • Full permissions on an MS Exchange server to create the domain user (Administrator/onyx).
  • Clients used for the OPS testing must be in the same domain than the MS Exchange Server.
  • It is recommended (but not mandatory) to have an isolated domain in order to avoid damage to the information of other projects that Exchange Server may contain.
