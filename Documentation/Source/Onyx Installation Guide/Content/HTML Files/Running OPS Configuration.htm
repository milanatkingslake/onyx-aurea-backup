Running OPS Configuration

  1. On the Web Server, run OPSConfig.exe.

Welcome window appears.

  1. Click Next.

The Logical Application window appears.

  1. Select the OEAS logical application for OPS, and click Next.

The Web Site Default Site ID window appears.

  1. Enter the site ID and click Next.

The Log Settings window appears.

  1. Select the type of notification that must be logged. It is recommended that you select Warning.
  2. Click Next.

The Ready to Install the Program window appears.

  1. Click Install.

When the installation process is complete, the Installation Wizard Completed window appears.

  1. Click Finish.
  2. Open the Services window.
  3. Right-click OnyxProcessing and select Properties.
  4. In the In the Properties window, click the Log on tab..
  5. Select This Account option and specify the OPS service user credentials for the user that was created during the Creating the Domain User procedure. Alternatively, click Browse to search for this user account and add the same.

  1. Ensure that the OnyxProcessing service is running.

  1. Perform IISReset.
