Post Installation Steps: OEAS Proxy Creation

  • IIS Settings on the Client Server for OEP & OES:
  • Go to IIS, and then expand the Server.
  • Expand the Sites, and then expand Default Web Site.
  • Click OEP_Onyx.
  • Go to the IIS section, and then double-click the Authentication icon.
  • Right-click Anonymous Authentication, and then click Edit.
  • Click the Specific User option, and then click Set.
  • Type the System Login credentials, and then click OK.
