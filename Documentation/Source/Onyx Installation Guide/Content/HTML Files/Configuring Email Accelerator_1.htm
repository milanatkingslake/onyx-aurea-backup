Configuring Email Accelerator Client Settings

To configure Email Accelerator client settings:

  1. On the Email Accelerator client installation server, navigate to the OnyxEmailAccelerator folder. The default location for this folder is ..\Inetpub\wwwroot\OnyxEmailAccelerator.
  2. In an editor such as Notepad, open the AcceleratorClient.config file, and change the following settings:
  • User ID. DATABASE_USER_NAME (Enter the user ID to log on to the Onyx database. For security reasons, we recommend that you do not use a user with DBO privileges.)
  • Password. DATABASE_PASSWORD (Enter the password corresponding to the user ID.)
  • Initial Catalog. ONYX_DATABASE_NAME (Enter the Onyx database name.)
  • Data Source. DATABASE_SERVER_NAME (Enter the name of your database server.)
  • OptOutText. We respect your privacy. (Enter the default text that will be inserted under each HTML email. You can edit this text in the Email Accelerator client before sending an email.)
  • OptOutURL. http://<virtual machine name>/unsubscribe/default.asp (Enter the location of the Unsubscribe functionality on your external Web server. This value is derived from the Unsubscribe virtual directory alias you entered while installing the external Web server. If this value is blank, Email Accelerator assumes that a different Opt-Out functionality is being used.)
  • TimageURL. http://<virtual machine name>/timage (Enter the location of the Tracking Image HTTP Module installed on your external Web server. This is a required value to track real-time user views of your email. This value is derived from the TImage virtual directory alias you entered while installing the external Web server.
  • iSiteID. 1 (Enter the site ID of your Onyx installation.)
