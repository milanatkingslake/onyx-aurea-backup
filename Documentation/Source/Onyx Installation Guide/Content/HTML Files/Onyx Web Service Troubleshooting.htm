Onyx Web Service Troubleshooting

If the following error is displayed, follow the steps explained.

Could not load type 'System.ServiceModel.Activation.HttpModule' from assembly 'System.ServiceModel, Version=3.0.0.0, Culture=neutral, PublicKeyToken=b77a5c561934e089”

  1. Open the command prompt in administrator mode.
  2. Go to the location where .net Framework 2.0 is installed (for example, cd ..\Windows\ Microsoft.NET\ Framework\v2.0.50727)
  3. Run the aspnet_regiis.exe –iru command.
  4. Go to the location where .net Framework 4.0 or later is installed (for example, cd ..\Windows\ Microsoft.NET\Framework\v4.0.30319
  5. Run again the aspnet_regiis.exe –iru command.
