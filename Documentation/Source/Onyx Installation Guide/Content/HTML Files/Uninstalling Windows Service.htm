Uninstalling Windows Service for OGS

  1. Open the OGSUnInstall.bat file.
  2. Set the arguments values in the OGSUnInstall.bat file.
Arguments Description Default Value Example

arg1

Windows Service Name

Onyx.OGS1

 

arg2

Windows Service Exe

OnyxGatewayWindowsService.exe

 
  1. Save the OGSUnInstall.bat file.
  2. Run the OGSUnInstall.bat file.
  3. Verify whether Services.msc OGS Service is deleted.
