OES: Cannot Open Onyx Enterprise Studio

If after the Setup is complete you cannot open the OES website that you created, try one or more of the following, then try again to open OES:

  • Stop and restart Microsoft Internet Information Server.
  • Use the OEAS Configuration Utility to set a logging level of 7 for the GlobalLogSettings key.
  • Examine the messages logged in the Windows Event Viewer for the application.
  • After you install a new OEAS logical application using the Onyx Configure Enterprise Application Server Setup, you may not be able to open OES due to logon failures. To resolve the logon failures, restart COM+. This issue occurs after a complete install of OEAS followed by an install of additional OEAS logical applications.
