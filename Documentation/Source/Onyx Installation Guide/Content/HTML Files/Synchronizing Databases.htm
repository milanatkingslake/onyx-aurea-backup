Synchronizing Databases

After setting up the Onyx database and the Persistence database, you must synchronize the data between the two databases. To do this:

  • Install the ServiceBroker service on the Onyx database server.
  • Configure Onyx Gateway Service.
  • Use the Onyx Cache Flush utility to run the synchronization process.

For more information on using the Cache Flush utility, refer to the Onyx Technical Guide.
