OEAS Setup: COM+ Runtime Internal Error 0x80041011

When reinstalling OEAS you may encounter the COM+ runtime internal error 0x80041011, if the COM+ application OnyxEASApp or OnyxEASLib was renamed since the last installation of OEAS. If this error occurs, open Component Services and do one of the following:

  • Delete the COM+ application that was renamed and then restart the OEAS Setup program.

OR

  • Rename the COM+ application with its original name and then restart the OEAS Setup program.
