Granting Administration Permissions to the Domain User in the APP/WEB Server

  1. Go to the APP/WEB server, right click on My Computer.
  2. Right click and select Manage. The Server Manager window is displayed.
  3. Click on Configuration node
  4. Click on Windows Firewall with Advanced Security node
  5. Browse through the tree listing to find Local Users and Groups and expand this node.
  6. Click on Groups. The existing groups are displayed on the right
    .

  1. Double click on Administrators. The Administrators Properties window is displayed.
  2. Click Add.
  3. Click Advanced.
  4. Click Locations and verify corresponding domain appears selected. For example,. Lab.onyx.com.

  1. Click OK.
  2. Click Find Now and scroll down the users name list. Select OPS user just created in previous procedure, step 6 (i.e. chandan). It will be added to the group Administrators.


  1. Click OK.
  2. Click OK to close the Administrators Properties window.
  3. Close the Server Manager window. If no error messages were displayed, you successfully configured the Domain User as Administrator of the server.

Troubleshooting

If you are unable to grant permissions using the procedure, do the following:

  • In the MS Exchange Server click the Active Directory Users and Computers.
  • Look for the Domain User created in the Procedure 1, step 6, and double click the user name.
  • Click the Member off tab, and click Add.
  • Click Advanced and Find Now.
  • Select the Domain Administrators group and click OK.
  • Click OK.

The Domain user now is member of the Domain Administrators group.
