Onyx System Requirements

This chapter lists system requirements that are specific to Onyx

Database Server

  • Microsoft SQL Server 2012 Service Pack 3 (Standard Edition)
  • Microsoft SQL Server 2014 Service Pack 1 (Standard Edition)
  • Microsoft SQL Server 2016 (Standard Edition)
  • Microsoft .NET Framework 3.5 or 4.0 or 4.5.1 or 4.6.1

OEAS Server

  • Microsoft .NET Framework 4.6.1
  • Microsoft Windows Server 2012
  • Microsoft Windows Server 2012 R2
  • Microsoft Windows Server 2016

Exchange Server

  • 2007 SP1 or later
  • 2010 SP2 with Rollup 3 or later
  • 2013

OEP client computer

  • Internet Explorer 11.0 or Edge browser (32-bit or 64-bit)

Onyx Mobile

  • iOS 9 or later
  • Android 5.0 or later

Internet Information Service

  • Microsoft Internet Information Services (IIS) 8.0 or later

Onyx 360 (both 32-bit and 64-bit)

  • .Net Framework 4.0 or later
  • Onyx 7.6 or later
  • Microsoft Outlook 2010 or 2013 or 2016

For detailed information about system requirements, refer to the Onyx System Requirements document.
