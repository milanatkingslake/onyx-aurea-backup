Installing Onyx Gateway Service

You must create OGS local account before installing OGS. This newly created account needs to be used during OGS installation.

To Install Onyx Gateway Service:

  1. Copy the Onyx Gateway Service folder to your OEAS server from Platform Components > Application Server (OEAS) from the Onyx installation package.
  2. Open the OnyxGatewayService folder on your OEAS server, and then run OnyxGatewayService.exe.

The Welcome window appears.

  1. Click Next

The Logical application window appears with Onyx as a default logical application name selected.

  1. Click Next.

The DataBase Information window appears.

  1. Do the following:
  1. Server. Type the database server name.
  2. OTDB. Type the Onyx Transactional Database Name.
  3. OPDB. Type the Onyx Persistent Database Name.
  4. Login. Type the SQL user name, who has the DBO privilege. The default is OnyxDBO.
  5. Password. Type the SQL password for Onyx DBO.
  6. Port #. Type the value of the port OGS should use.
  1. Click Next.

The Logon Information window appears.

  1. Type the required user name in the Domain\Username format and Password for OGS to use.
  2. Click Next.

The Ready to install the Program window appears.

  1. Click Install.

OGS starts the installation. After completion, the Installation Wizard Completed window appears.

  1. Click Finish.
