Configuring Database

To install Database:

  1. On your SQL server, open SQL Server Management Studio, and then log on by typing the user name and password.
  2. Do the following:
  1. Expand the Security folder under the DB server.
  2. Expand the Logins folder.
  3. Grant the OnyxLMSql user created during OEAS setup installation, OnyxLMViews and db_datawriter database role memberships at User Mapping.

Example:

  1. Expand the Tables folder and verify that the following tables are present as default:
  1. dbo.Mass_email_batch
  2. dbo.Mass_email_queue
  3. dbo.Mass_email_history
  4. dbo.Mass_email_invalid_records
  1. Expand the Programmability folder and the Stored Procedures folder in the DB name and  verify that the following procedures are present as default:
  1. dbo.SDpcdEmailBatchExecute
  2. dbo.SDpcdEmailBatchInsert
  3. dbo.SDpcdProcessedEmail
  4. dbo.SDpcgDecryptEmailData
  5. dbo.SDpcgEncryptEmailData
  6. dbo.SDpcgGetEmailQueue
  7. dbo.SDpcgValidateEmailQueue
  8. dbo.SDpcuSentEmail
  9. dbo.SDpsiCampaign
  1. In the Web/App server, press Start > All Programs > Administrative Tools, and then click Component Services.
  2. Expand Component Services > Computers > My Computer.
  3. Double-click COM+ Applications.
  4. Right-click OnyxEASApp, and then click Shut Down.

This releases any references to the OED XML file.

  1. Log on to OES Object Designer.
  2. Expand the Physical Model node and the Procedures node to verify the existence of:
  1. SDpcdEmailBatchExecute
  2. SDpcdEmailBatchInsert

The database tables and procedures required for Email Accelerator are created when you install the OEDB. Additionally, OED and Onyx UI code modifications for Email Accelerator are included in the OEAS and Onyx UI setups respectively.
