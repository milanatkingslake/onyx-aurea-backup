Making Changes in Onyx Employee Portal

Follow these steps to make changes in the Onyx Employee Portal.

To make changes in the Onyx Employee Portal:

  1. Open the listmanager/lm_results.js file and look for the function ‘jsDoMassEmailBulkAction()’.

This function has a variable called ‘sClientLocation’. Place the URL of your client installation (for example, client server http:// <Onyx Web Site Name/Computer name>/OnyxEmailAccelerator/ client.asp) into this value between single quotes (‘)
