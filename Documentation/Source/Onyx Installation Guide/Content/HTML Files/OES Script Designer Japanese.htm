OES: Script Designer - Japanese Text is not Displayed in SVG Viewer

If you create a script using Japanese characters, the characters are not displayed correctly when you open the script in SVG viewer.

The following SVG viewer versions are compatible with the respective version of Internet Explorer.

Internet Explorer version Supported SVG viewer version

Internet Explorer 8, 9, 10, or 11

SVG3.03, 3.01

To resolve this:

  1. In your OES installation folder, open the Scripts folder. The default folder path is ..\Program Files(x86)\Onyx\EnterpriseStudio\ScriptDesigner\Scripts.
  2. Open the initialSvg.svg file, search for the text Verdana, and then replace it with the text Arial Unicode MS.
  3. Open the previewSvg.svg, search for the text Verdana, and then replace it with the text Arial Unicode MS.
  4. Save both files.
  5. Ensure that you have installed the SVG viewer version that is compatible with your version of Internet Explorer.
  6. Open the script using SVG viewer and verify that the issue is resolved.
