Installing OPS

  1. Login to the App/Web Server with a user in the domain. Recommended login with the user created in Creating the Domain User procedure.
  2. On the Web Server, run the OnyxProcessingServer.exe.

The Welcome window appears.

  1. Click Next.

The Logical Application window appears.


  1. Select the logical application and click Next.

The Web Site Default Site ID window appears.

  1. Enter the Site ID and click Next.

The Onyx Authentication Web Site Name appears.


  1. Enter Name for Onyx Authentication Web Server, and click Next.

Default name is OnyxPSAdmin_Onyx. Onyx recommends ops_onyx.

The Windows Authentication Web Site Name window appears.


  1. Enter Name for Windows Authentication Web Server, and click Next.

Default name is OnyxPSAdmin_windows. Onyx recommends ops_windows.

The Logon Information window appears.


  1. Specify the user credentials created in Creating the Domain User procedure. Specify the User name in the <domain name>\<user name> format.
  2. Click Next.

The Local Group window appears.

  1. Retain the default name: OPSAdmin.
  2. Click Next.

The Email Configuration window appears.

  1. In order to monitor the activities created by the OPS user, it is recommended that you create a system administrator email account. To know how to create this account, see Creating the Domain User.  Specify the following information and click Next:
  • Server name: Exchange server where you created the email account
  • E-mail from  Email ID of the sender 
  • E-mail to Email ID of the recipient 

The Log Settings window appears.

  1. Select the type of notifications that must be logged.  It is recommended that you log Warning messages.
  2. Click Next.

The Destination Location window appears.

  1. Retain the default path of click Chnage to provide the new path.
  2. Click Next.

The Ready to Install the Program window appears.


  1. Click Install.

When the installation process is complete, the Installation Wizard Completed window appears.

  1. Click Finish.
  2. Restart the App Web Server.
