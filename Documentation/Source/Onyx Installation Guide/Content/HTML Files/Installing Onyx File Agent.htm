Installing Onyx File Agent

To Install Onyx File Agent:

  1. Create an OFA Domain user account on your domain server. For the purposes of this installation, we use the OnyxFAUser user group located on the domain server.
  2. Copy the File Agent (OFA) folder to your server from Platform Components > Web server from the Onyx installation package.
  3. On the server, open the File Agent (OFA) folder, and run OnyxFileAgent.exe.

The Onyx File Agent Welcome window appears.

  1. Click Next.

The Local Group window appears. The OnyxFAUser is displayed as default group.

  1. Click Next.

The Logon Information window appears.

  1. Type the required Username in the Domain\Username format and Password for OFA to use.
  2. Click Next.

The Destination Folder window appears.

  1. If you need to change the location for OFA to be installed, click Change or use the existing location, and then click Next.

The Ready to install the Program window appears.

  1. Click Install.

When the installation process is complete, the Installation Wizard Completed window appears.

  1. Click Finish.
