Configuring Navigator

After installing an instance of Onyx Gateway Service, you must configure settings on the OEP server to enable Navigator to retrieve data by connecting to the appropriate instance of Onyx Gateway Service.

To configure Navigator UI:

  1. On the OEP server, navigate to the CRArchitecture folder within the OEP installation folder. The default path for this folder is ..\Program Files(x86)\Onyx\EmployeePortal\CRArchitecture\Configuration.
  2. Using a text editor such as Notepad, open the Config.xml file.
  3. In the node add baseAddress, enter the name of the computer where you installed Onyx Gateway Service, in the following format, replacing the bracketed text with the appropriate values:

http://<ComputerName>:<PortNumber>/ServiceGateway/" language="<language>

http://<ComputerName>:<ApplicationName>/" language="<language>

Be sure to enter the port number associated with the OGS instance that you intend to use with your installation for OEP.

  1. Save the Config.xml file.
  2. On the OEP client, clear the temporary internet files and browser cache.
