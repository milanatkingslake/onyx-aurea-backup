Configuring Database

Follow these steps to configure database.

To configure database:

  1. Log on to the database machine, which you are pointing for the Extraction Engine.
  2. Start MS SQL Server Management Studio.

Run as administrator before starting MS SQL Server Management Studio.

  1. In the Microsoft SQL Server Management Studio (Administrator) window, expand the Database folder.
  2. Expand the database (that is, OEDB) folder, which you are using for Extraction Engine.
  3. Expand the Table folder.
  4. Right-click dbo.extraction_profile, and then select Edit top 200 row.
  5. In the destination_directory column of the dbo.extraction_profile window, change the value from \\WebServer\OnyxLibraries\Sales to the shared location where you want to extract html files, and then save. For example, <local machine>\OnyxLibraries\sales.
  6. Follow step-7 for Service, Support, and QA.

destination_directory is case-sensitive. Therefore, the names must match with the names you had specified while creating the folders
