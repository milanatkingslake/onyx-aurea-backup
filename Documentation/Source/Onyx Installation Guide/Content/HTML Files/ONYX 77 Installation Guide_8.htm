Prerequisites

  • Make sure to have an App/Web server - for Email Accelerator Server
  • Installed OES and OEP version
  • Windows  2012 or Windows 2012R2
  • Internet Explorer 11.0 or Edge
  • Microsoft .NET Framework 4.5 or later
  • Make sure to have another Server – for Email Accelerator Client
  • Windows  2012 Server image
  • Installed IIS and SMTP Services
  • Internet Explorer 11.0 or Edge
