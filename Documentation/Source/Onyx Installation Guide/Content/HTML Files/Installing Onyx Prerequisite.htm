Installing Onyx Prerequisite

Onyx Prerequisite installs all the files necessary for Onyx Software, such as Visual C++ runtime. This is a pre-requisite before you install OEAS.

To install Onyx prerequisite:

  1. In your Onyx installation package, navigate to Platform Components > Onyx Prerequisite, and then run OnyxPrerequisite.exe.

The Welcome window appears.

  1. Click Next.

The Ready to Install the Program window appears.

  1. Click Install. Installation of the files starts and the Installation Wizard Completed window appears after installation is completed.

  1. Click Finish.
