Server Pre-requisites

Make sure the following pre-requisites are met with before you begin installing Onyx 360 the Outlook Add-in server.

  1. Onyx version Enterprise Application Server (OEAS)
  2. Onyx version Web Services (OWS).
  3. Configure Web-Services to work under Windows Authentication.
  4. .Net Framework 4.0 or later.
  5. Set up the OEPFileShare created in the App server to support the Onyx360 Attachments feature.
  6. Set up Onyx File Agent (OFA) to view attachments in OEP saved from Onyx360.
  7. Grant necessary domain permissions to facilitate attachment file share is accessible by users’ desktop. This is required for email attachment upload - same permission identical to as OEP file attachment.
