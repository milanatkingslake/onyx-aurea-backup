Post Installation Tasks

After set up is completed, perform following post installation tasks.

  1. From Component Services, shutdown OnyxEASApp.
  2. Log on to Onyx Enterprise Studio and then select Object Designer Tool.
  3. Publish OED using a SQL DB log on that has the db_owner permission (that is, Onyx DBO SQL login).
  4. From Component Services, shutdown OnyxEASApp.

Outlook Add-in Server install steps are completed.
