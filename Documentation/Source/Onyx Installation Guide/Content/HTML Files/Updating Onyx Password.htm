Updating Onyx Password

Use the Onyx Update Password tool to encrypt the password existing in the database.

The OnyxUpdatePassword tool must be run from the Web or App server because it requires OSLFEncrypt.dll to encrypt the existing passwords in the database. This tool can be used only for databases that are upgraded from 7.5 or earlier versions to .

To update Onyx password

  1. Copy OnyxUpdatePassword to your server from Utilities provided in the Onyx installation package.
  2. On the server, open the OnyxUpdatePassword folder, and then run OnyxUpdatePassword.

The Onyx Password Update window appears.

  1. Do the following:
  • Onyx Database Server. Type the database server name.
  • Onyx Database Name. Type the Onyx transactional database name.
  • Onyx Database User ID. Type the SQL Server Login name.
  • Onyx Database Password. Type the SQL Server Login password.
  1. Click Connect.

The Set New Password area appears in the Onyx Password Update window.

  1. Do the following:
  • Default Onyx Password. Type the default Onyx password.
  • Confirm Default Password. Repeat the default password to confirm.
  1. Click Next.

The Password Update Progress tab appears in the Onyx Password Update window.

  1. Click Start.

The OnyxUpdatePassword tool starts updating the existing password in the database. After completion, the Onyx passwords update completed successfully message appears in the bottom portion of the Password Update Progress area.

  1. Click Close.
