Installing Onyx Mobile Web

To install Onyx Mobile Web

  1. In your Onyx installation package, navigate to Platform Components > Web Server > OnyxMobileWeb, and then run OnyxMobileWeb.exe.

The Welcome screen appears.

  1. Click Next.

The Logon Information window appears.

  1. Type the required user name in the Domain\Username format and Password for Onyx Mobile Web to use, and then click Next.

The OnyxMobileWeb Authentication Web Site Name window appears.

  1. Specify the name of the Onyx virtual website. Click Next.

The OnyxSystemDetails window appears.

  1. Specify the following details:
  • Base Address: URL of the Onyx Gateway Service.
  • Username: Username with administrative privilege to generate the cache file from OEP
  • Password: Password for the Username
  1. Click Next.

The Destination Folder window appears.

  1. If you need to change the location for Onyx Mobile Web to be installed, click Change or use the existing location, and then click Next

The Start Copying Files widow appears.

  1. Verify the details and click Next to proceed or click Back to change the details.

The Ready to Install the Program window appears.

  1. Click Install.

When the installation process is complete, Installation Wizard Completed window appears.

  1. Click Finish.
