Creating a Network Share

You need to create a network share to enable storing of the output worknote html files in the respective entities.

To create a Network Share:

  1. Create a new folder OnyxLibraries in the desired drive. (For example, ..\)
  2. Create four sub-folders under OnyxLibraries and name them as QA, Sales, Service, and Support.

The folder names provided are as per the destination_directory column in the extraction_profile table in OEDB. Do not modify the sub-folder names.

  1. Right-click the OnyxLibraries folder, point to Share with, and then click Specific people.

The File Sharing dialog box appears.

  1. Do the following:
  • From the list, select Everyone.
  • In Permission Level, select Read/Write.
  1. Click Share.

The folder is shared window appears.

  1. Click Done.
