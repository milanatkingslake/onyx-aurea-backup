Installing SMTP Services in Onyx Email Accelerator Client Server

Follow these steps to install SMTP Services in Onyx Email Accelerator Client Server.

To install SMTP Services in Onyx Email Accelerator Client Server:

  1. Go to Control Panel > Programs and Features.
  2. In the Control Panel Home pane, click Turn windows features on/off.

The Server Manager window appears.

  1. Under Features summary, click Add Features.
  2. Select Role-based or feature-based installation and click Next.
  3. Select a server from the server pool and click Next.
  4. In the Server Roles section Click Next.
  5. In the Features section select SMTP Server.

  1. Click Add Features on the pop-up window and then click Next.
  2. Click Install.
  3. Click Close after the installation.
