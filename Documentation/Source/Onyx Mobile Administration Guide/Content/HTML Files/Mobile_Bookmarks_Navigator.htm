Creating Navigator Queries

When creating Navigator queries, users can save them as personal mobile bookmarks. Currently, users can create mobile bookmarks for the following Navigator entities:

  • Companies
  • Individuals
  • Sales Opportunities
  • Service Requests
  • Support Incidents
  • Tasks
  • Products
  • WorkTickets
  • Forecast
  • Scripts

To save an existing query as a mobile bookmark, users must select Subscribe as Mobile Bookmark button for the query in the Saved Queries section.
