About Mobile Bookmarks

Mobile bookmarks are queries created in Navigator you can view in Onyx Mobile App/Onyx Mobile Web. While creating a Navigator query, users can mark it as a Mobile Bookmark. When users log on to Onyx through their iOS or Android device, these bookmarks are listed as menu items on the Home Page.

  • Creating User-specific Bookmarks
