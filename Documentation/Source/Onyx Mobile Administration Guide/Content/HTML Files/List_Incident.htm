Understanding Onyx Mobile App/Onyx Mobile Web List Screens

The List screens are read-only screens that display key information for a list of customer, incident, or task records. Tapping a record on a List screen opens the Details screen for the record where the user can view additional information and perform tasks on the record.

This screen... Displays a list of...
Customer List Company and individual records that match the specified search criteria.
This screen appears when users run a search on Onyx Mobile App/Onyx Mobile Web or when users tap a mobile bookmark on the Onyx Mobile App/Onyx Mobile Web Home Page.
External Contact List All companies and individuals that are external contacts for the record.
The contact type is displayed below the contact name.This screen appears when users tap the View external contacts navigation menu item on the Company, Individual, or Incident Details screens.
Incident List Incidents that match specified criteria.
This screen appears when users tap an Incident navigation menu item on the Company or Individual Details screen or a mobile bookmark for incidents on the Home Page.
Internal Contact List Internal contacts associated with a record. The contact type is displayed below the contact name.
This screen appears when users tap the View internal contacts navigation menu item on the Company, Individual, or Incident Details screens.
Products List Products that match specified criteria.
This screen appears when users tap a Products navigation menu item on the Company or Individual Details screen or a mobile bookmark for products on the Home Page.
Task List Tasks that match specified criteria.
This screen appears when users tap a Task navigation menu item on the Incident Details screen or a mobile bookmark for tasks on the Home Page.
WorkNote List Work notes associated with an incident or a task.
This screen appears when users tap the View worknotes navigation menu item on the Incident or the Task Details screen.
Image List Images that are attached to an entity.
This screen appears when users tap theView all image attachment option navigation menu item on the Company, Individual, or Incident Details.
Workticket List Workticket that match specified criteria. This screen appears when users run a search on Onyx Mobile/ Web App home screen, tap on View Worktickets on Incident screen, Linked WorkTickets on WorkTicket screen, or a Mobile bookmark for workticket on homepage.
Quote List This screen appears when user taps View Quotes on the Incident screen.
