Configuring Web Links

The Web Links screen contains links to specific Websites from within Onyx Mobile App/Onyx Mobile Web. Tapping a Web link opens the referenced Website within Onyx Mobile App/Onyx Mobile Web. Use this information to configure URI details for the Websites you want users to access from Onyx Mobile App/Onyx Mobile Web.

To configure Web links:

  1. Open the WebLinks.XML file. For a list of XML files and screens it they correspond to, see Understanding XML files.
  2. Configure attributes for the weblink element within the XML file.
  3. Add a row for each additional link you want to create.
  4. Save and close the XML file.
  5. Log on to Onyx Mobile App/Onyx Mobile Web and verify your changes.

Elements on the Web Links Screen

Use the following table to configure properties for each group and field displayed on the Web Links screen:

Element

Attribute

Description
weblink DisplayName Type user-friendly text to display on the Web Links screen for the URI. For example, Avolin.
weblink TargetURL Type the value of the URL to start when the link is tapped.
