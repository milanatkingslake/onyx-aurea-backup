Understanding Onyx Mobile App/Onyx Mobile Web Details Screens

The Details screens are read-only screens and display essential information for a specific record. The screens provide links to view, add, and modify information about the record.

Company Details Screen

Includes name, address, phone, e-mail, and primary contact information for company records. It also provides links to view company contacts and incidents. Users can call or send e-mail to a company, edit the company record, add a company to their favorites, view directions to the company's primary address, and add new incidents for the company.

Individual Details Screen

Includes name, address, phone, e-mail, and primary contact information for individual records. It also provides links to view individual contacts and incidents. Users can call or send e-mail to an individual, edit the individual record, add an individual to their favorites, view directions to the individual's primary address, and add new incidents for the individual.

Incident Details Screen (Sales/Service/Support)

Includes information such as description, ID, owner, priority, status, and type details for incident records. It also displays the last work note added for the incident. Users can edit the incident record, view contacts, tasks, work notes for the incident, and add new work notes or tasks.

Task Details Screen (Sales/Service/Support)

Includes information such as description, ID, owner, priority, status, and type details for task records. It also displays the last work note added for the task. Users can view and add work notes to the task.

Workticket Details Screen

Includes information such as description, ID, type, priority ,and status for workticket records.This screen also displays the last work note added to a workticket. Users can edit the workticket records, and view contacts, linked customers, linked worktickets, linked incidents, and work notes for a workticket. User can also add new work notes from this screen.

Forecast Detail Screen

Includes information such as ID, Primary currency, User Commit, Total Forecast, Total discount, Calculated Forecast, closed date, and booked amount. In this screen, users can view line items for forecasts.

Quotes Details Screen

Includes information such as description, ID, status, primary currency, total cost, and total discount for Quotes. This screen also displays the last work note that is added to a Quote. In this screen, users can view line items for quotes.
