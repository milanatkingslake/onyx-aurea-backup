System Requirements

Onyx Mobile Web is compatible only with Onyx 7.7 or later versions.

Onyx Mobile App is designed to work with the following system requirements:

  • Onyx CRM. Version 7.6 or later
  • iOS. Version 9.0 or later
  • Android. Version 5.0 or later

Android Device Compatibility

Onyx Mobile App/ Onyx Mobile Web for Android is designed for phones and tablet devices. It has been tested with the following devices:

  • Samsung S4
  • Nexus 5
  • Nexus 7

If you use a different Android device other than what is listed above, depending on the screen size of the device, display of certain user interface elements might not be correct.

iOs Device Compatibility

Onyx Mobile App/ Onyx Mobile Web for iOs is designed for iPhones and iPad devices. It has been tested with the following devices:

  • iPhone 6 plus
  • iPad 2

If you use a different iOs device other than what is listed above, depending on the screen size of the device, display of certain user interface elements might not be correct.
