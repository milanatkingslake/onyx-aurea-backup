Configuring Home Page

The Home page contains a menu that enables you to navigate to different screens of the application.

In the Home Page, users can:

  • View lists of recent customers, incidents, companies, and individuals.
  • Add new customer records
  • Access pre-defined Web links
  • Search Company, individual, and incident records.

If you have configured user-specific mobile bookmarks or if users have configured mobile bookmarks for their Navigator queries, these bookmarks are also displayed on the Home page.

For more information about configuring mobile bookmarks, see About Mobile Bookmarks.

For information about configuring web links, see Configuring Web Links.

To configure Home Page

To modify how data is displayed on the Home page, edit the OnyxMobileHomePageMetadata.XML file. To support a profile tile that brings back a specific individual record, add the following code in the OnyxMobileHomePageMetadata.XML file:

<MenuItem Url="../customer/custom.js" Icon="customBookMark.png" DisplayName="<Display option>" Id="<ID>" PageTitle="<Title>" Action="<entity>" DisableNavigation="1">

<Filter ContextVariable="$userId" Property="assignedTo"/>

<Filter Property="statusId" Value="130"/>

</MenuItem>

Example:

-<MenuItem Url="../customer/custom.js" Icon="customBookMark.png" DisplayName="View all incidents assigned to me" Id="12" PageTitle="My Incidents" Action="incident">

<Filter ContextVariable="$userId" Property="assignedTo"/>

<Filter Property="statusId" Value="130"/>

</MenuItem>

You can change the following attributes for each node

You can now create a custom entity list by using the <entityname>Details.xml format. Ensure that you follow the element structure as the OOTB Details XML files.

Element Attribute Description

ResetPassword

 

Display the reset password option on the Home Page.

  • For iOS: It displays under the Settings on the Home Page.
  • For Android: It displays under the menu icon on the Home Page. Reset the password option is displayed when the value is True and not when the value is False.

ResourceId

UI resource who has permission reset the password from the Home Page.

MenuItem

 

MenuItemId

Type a unique value for each item in the navigation menu.

Action

Type a value for the action to be performed when a user taps the menu item. For a list of action values currently available, see Specifying Action Values.

PageTitle

Type a value for the title of the screen that opens on tapping the menu item.

DisplayName

Type the text to use as caption for the item. To avoid cropping, we recommend that captions do not exceed 10 characters, including spaces.

Icon

Do not change this value. In the current implementation, your changes will not be reflected in the application.

Id

Do not change this value. In the current implementation, your changes will not be reflected in the application.

Name

Do not change this value. In the current implementation, your changes will not be reflected in the application.

SequenceIndex

Type a numeric value to specify the sequence that the item will appear in on the menu.

URL

Do not change this value. In the current implementation, your changes will not be reflected in the application.

Visible
  • Set to TRUE to display the item on the menu.
  • Set to FALSE to hide the item.

ResourceId

Contains UI Resource permissions. Type the UI Resource value.
Web Type Web="false" to disable an action on Onyx Mobile Web.

DisableNavigation

Type DisableNavigation="1" for custom entities that does not have detail page.

-

IsCount
  • Set to TRUE to display the count of records for each mobile bookmark.
  • Set to FALSE to hide the record count.
