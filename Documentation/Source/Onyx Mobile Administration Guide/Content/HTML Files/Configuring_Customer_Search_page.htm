Understanding Search Screens

The Search screen allows users to enter specific criteria to search for Company, Individual, and Incident records from their iOS or Android device. Users can perform a search based on several criteria such as ID, name, address, e-mail, location, and type.

The fields and tasks available on the Search screen depend on the configuration made within the XML file corresponding to the screen. Use this information to configure XML files for Customer Search screens.

To configure the fields that are displayed in the Customer Search Result List screen, see Configuring List Screens.
