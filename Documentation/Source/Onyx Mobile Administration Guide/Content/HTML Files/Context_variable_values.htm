Specifying Context Variable Values

Context variables contain values currently available in each record. These values are used to filter for a navigation menu list item. When a user taps a navigation menu list item, the values explained in the table below are returned to OGS.

The following context variable values are currently implemented in Onyx Mobile App/Onyx Mobile Web:

This Context Variable Value... Retrieves Records...
$primaryId That has the primary ID of the displayed record as the value of the property.
$Credential Based on the logged on user's Onyx session ID.
$appName Based on the OEAS application name.
$userId That has the logged on user's ID as the value for the property.
$contentType In the specified content format, for example JSON.
$siteId That matches the site ID specified in the app's configuration.
$lang That matches the language specified in the app's configuration.
