Shortcut Handler

The Shortcut Handler is a beeline link, through which you can launch the scripts and also open the Detail screen of an entity directly by launching Onyx Mobile App or Onyx Mobile Web through a URL.

The Shortcut Handler supports the following entities :

  • Company
  • Individual
  • Incidents
  • Tasks
  • Scripts

URL Construction

Refer the following table to construct the URL for Shortcut Handler:

  • To launch Detail screen
Application URL

Onyx Mobile App

OnyxMobile://?&entityType=<TypeofEntity>
&id=<IdofEntity>

Onyx Mobile Web

http://<servername>/<websitename>/?
&entityType=<TypeofEntity>
&id=<IdofEntity>

Where,

  • <TypeofEntity> is the entity type.
  • <IdofEntity> is the secondary ID number corresponds to the entity type.
  • <servername> is the server name in which Onyx Mobile Web is deployed.
  • <websitename> is the installed website name.

Replace the angular brackets also, when you specify the details.

  • To launch Scripts
Application URL

Onyx Mobile App

Onyxmobile://?&entityType=scriptSession
&id= <IdofEntity>
&scriptSessionId=<scriptSessionId>
&ownerType=<ownerType>
&ownerId=<ownerId>

Onyx Mobile Web

http://<servername>/<iisInstalledMobileWebName>/?
&entityType=scriptSession
&id=<IdofEntity>
&scriptSessionId=<scriptSessionId>
&ownerType=<ownerType>
&ownerId=<ownerId>

Where,

  • <IdofEntity> is the secondary ID number that corresponds to the entityType.
  • <servername> is the server name in which Onyx Mobile Web is deployed.
  • <iisInstalledMobileWebName> is the IIS installed website name.
  • <scriptSessionId> is the scriptSessioin ID to launch a new script.

If you do not provide the <scriptSessionId>, it launches a new script.

  • <ownerType> is the owner type.
  • <ownerId> is the owner ID.

If you do not provide the <ownerType> and <ownerId>, it launches an owner less script.

Replace the angular brackets also, when you specify the details.
