Logging on to Onyx Mobile App/Onyx Mobile Web

When you start Onyx Mobile App/Onyx Mobile Web for the first time, enter the configuration settings provided by your administrator before you log on.

To log on to Onyx Mobile App/Onyx Mobile Web

  1. On your iOS or Android device, start Onyx Mobile App/Onyx Mobile Web.

When you start the application for the first time, the Onyx Configuration screen appears.

  1. Type the following configuration settings:
Field Description
Server URI Enter the uniform resource identifier (URI) of the OES Server. For example, http://ipaddress: portnumber/servicegateway.
Application Name Type the OEAS application name.
Language Type the installation language 'ENG'.
Site Id Type the Onyx site ID.
  Add Logo URI for Onyx 7.6.1 (optional)
User Name Type your Onyx user name.
Logo URI Type the URL for your company logo. Contact your administrator URL details.
Clear Cache. Click to clear the existing cache.
  1. Tap Save.

The configuration is saved and the Login screen appears.

  1. Accept the default user name or type a different one, and then type your Onyx password.
  2. Tap Login to access Onyx Mobile App/Onyx Mobile Web.

To change previously entered configuration settings, tap Settings on the Login screen, enter new settings on the Onyx Configuration screen, and then tap Save.
