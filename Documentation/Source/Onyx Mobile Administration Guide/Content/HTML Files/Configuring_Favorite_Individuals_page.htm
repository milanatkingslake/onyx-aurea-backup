Configuring My Favorite Individuals Screen

The Onyx Mobile App/Onyx Mobile Web Favorite Individuals screen lists the individual records for whom the logged-on user is listed as a favorite contact.

To configure My Favorite Individuals Screen:

To modify how data is displayed on the Favorite Individuals screen, use the FavoriteIndividualsMetaData.xml file.

You can change the following attributes:

Element Attribute Description
sorting isAscending
  • Set to TRUE to list favorite individuals in ascending order.
  • Set to FALSE to list individuals in descending order.
sorting numericSorting
  • Set to TRUE to sort based on numeric values.
  • Set to FALSE to sort based on text.
sorting sortBy Type the field name that you want to sort by.You can sort by:
  • primaryid
  • secondaryid
  • objectType
  • suffix
  • salutation
  • firstName
  • middleName
  • lastName.
