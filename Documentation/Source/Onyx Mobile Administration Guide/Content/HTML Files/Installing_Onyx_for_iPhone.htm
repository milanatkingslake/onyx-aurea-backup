Installing Onyx Mobile App/Onyx Mobile Web

Onyx Mobile App/Onyx Mobile Web and Onyx Mobile Plus are native applications for iOS and Android devices. The free download can be made from the respective App Store.

To install Onyx Mobile App/Onyx Mobile Web:

  • Upgrade your Onyx system to version . For information about upgrading, refer to Onyx Upgrade Guide.

Note the following information for future use:

  • OEAS server IP address
  • OEAS application name
  • Onyx installation language
  • Site ID
  • Port number configured for the OGS instance that will be used with Onyx Mobile App/Onyx Mobile Web.

Inform your users to visit App Store or Play Store to download Onyx Mobile App/Onyx Mobile Web application for their device.

Provide your users with the information you noted in step 2.

To log on to Onyx Mobile App/Onyx Mobile Web, see Logging on to Onyx Mobile App/Onyx Mobile Web.
