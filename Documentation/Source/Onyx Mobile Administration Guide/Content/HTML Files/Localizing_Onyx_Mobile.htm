Localizing Onyx Mobile App/Onyx Mobile Web

Onyx Mobile App/Onyx Mobile Web supports both English and Japanese installations of Onyx, on a single application.

To Use Onyx Mobile App/Onyx Mobile Web in English:

  1. Download the application to a supported iOS or Android device that has the device language set to English.
  2. Configure the application to connect to an English language installation of Onyx.

To use Onyx Mobile App/Onyx Mobile Web in Japanese:

  1. Download the application to a supported iOS or Android device that has the device language set to Japanese.
  2. Configure the application to connect to a Japanese language installation of Onyx.
