Introduction

Onyx Mobile App/Onyx Mobile Web integrates enables users to access Onyx Application by using devices or browsers.
Onyx Mobile is a native application for iOS and Android devices. Users can manage their customer interactions while they are on the move. Onyx Mobile facilitates the users to access their customer and contact information on their iOS or Android devices.

Onyx Web App enables users to access their records through a browser.

Using Onyx Mobile App/ Onyx Mobile Web, you can:

  • View a list of favorite customers.
  • View a list of recent customers and incidents.
  • View details of existing customer, incident, work tickets, and task records.
  • Search for existing customer records.
  • Add new customer, incident, work tickets, and task records.
  • Modify details for existing customer records, such as phone number, e-mail, address, and primary, external, and internal contacts.
  • Modify details for existing incident records, such as contacts, work notes, status, and priority.
  • Send e-mails and call customers from within the Onyx Mobile App/Onyx Mobile Web application.
  • View directions to your customer's location.
  • View lists of incidents, products, work tickets, and tasks associated with a customer.
  • View lists of incidents and tasks assigned to the logged on user.
  • View Navigator queries you had saved as Mobile Bookmarks.
  • Launch Detail screen through a URL without going to the Home screen.
  • Edit records on Detail screen using Prompted Action.
  • Retrieve the list of various records by scanning the bar code of a product.
  • Send email, send signed email, and store attachment on Sales / Service / Support incidents.
  • Launch the scripts from Details screens and Mobile Bookmarks.

Before using Onyx Mobile App/Onyx Mobile Web, you must configure it to match your database and object model schema. For more details, see Configuring Onyx Mobile App/Onyx Mobile Web.
