Configuring DoMore Groups

The DoMore Groups allow you to select the associated actions of a group quickly, just by clicking the parent action of the group on Detail screen.

To configure the DoMore Groups

  1. Open the required XML(s) file in edit mode from the \Onyx\AppServer\Applications\Onyx\OnyxGatewayService\App_Data\Profiles folder. For the list of screens and their XML file names, see Understanding XML files.
  2. Add the following element node in the required XML file(s):
  • ID

<MenuItem MenuItemId="0" DisplayName="Add Actions" Icon="CommonDoMore.png" ResourceId="" Action="quickactions" PageTitle="Quick Actions">

<SubMenuItem MenuItemId="1" DisplayName="Add company" Icon="AddCompanyDoMore.png" ResourceId="UI:OM.company.add" Action="addCompany">

</SubMenuItem>

<SubMenuItem MenuItemId="2" DisplayName="Add incident" Icon="AddIncidentDoMore.png" ResourceId="UI:OM.incident.add" Action="AddIncident">

<Filter Property="ownerId" ContextVariable="$secondaryId" />

</SubMenuItem>

<SubMenuItem MenuItemId="3" DisplayName="Add work ticket" Icon="AddWorkTicketsDoMore.png" ResourceId="" Action="AddWorkTicket">

</SubMenuItem>

<SubMenuItem MenuItemId="4" DisplayName="Add to my favorites" Icon="CommonDoMore.png" ResourceId="" Action="AddMeAsFavorite" ContactTypeDid="67">

</SubMenuItem>

<SubMenuItem MenuItemId="5" DisplayName="Directions" Icon="DirectionsDoMore.png" ResourceId="" Action="map" Web ="False" LinkColumnName="address1,address2,address3,city,regionCode,postCode,countryCode"> </SubMenuItem>

</MenuItem>

You need to add TapType="quickactions" in any Field node of the detail section and Action="quickactions" in the MenuItem node to configure the DoMore Group on detail section.

If multiple MenuItem nodes are configered with Action="quickactions" in an XML, it takes the details from the last MenuItem node to display the sub menu items

  • DoMore actions

<MenuItem MenuItemId="0" DisplayName="Add Actions" Icon="CommonDoMore.png" ResourceId="" Action="" PageTitle="Quick Actions">

<SubMenuItem MenuItemId="1" DisplayName="Add company" Icon="AddCompanyDoMore.png" ResourceId="UI:OM.company.add" Action="addCompany">

</SubMenuItem>

<SubMenuItem MenuItemId="2" DisplayName="Add incident" Icon="AddIncidentDoMore.png" ResourceId="UI:OM.incident.add" Action="AddIncident">

<Filter Property="ownerId" ContextVariable="$secondaryId" />

</SubMenuItem>

<SubMenuItem MenuItemId="3" DisplayName="Add work ticket" Icon="AddWorkTicketsDoMore.png" ResourceId="" Action="AddWorkTicket">

</SubMenuItem>

<SubMenuItem MenuItemId="4" DisplayName="Add to my favorites" Icon="CommonDoMore.png" ResourceId="" Action="AddMeAsFavorite" ContactTypeDid="67">

</SubMenuItem>

<SubMenuItem MenuItemId="5" DisplayName="Directions" Icon="DirectionsDoMore.png" ResourceId="" Action="map" Web ="False" LinkColumnName="address1,address2,address3,city,regionCode,postCode,countryCode">
</SubMenuItem>

</MenuItem>

  1. Save and close the XML file.
  2. Log on to Onyx Mobile and verify the changes.

All sub menu items of a group have to be configured within a single MenuItem node.

It is not possible to configure the DoMore groups on the following actions:
• S3 Utility
• Image Attachment Utility
• Prompted Action
