Configuring Search Screens

  • To modify how data is displayed on the Customer Search screen, use the CustomerSearchMetaData.XML file.
  • To modify how data is displayed on the Company Search screen, use the CompanySearchMetaData.XML file.
  • To modify how data is displayed on the Individual Search screen, use the IndividualSearchMetaData.XML file.
  • To modify how data is displayed on the Incident Search screen, use the IncidentSearchMetaData.XML file.
  • To modify how data is displayed on the Work Ticket Search screen, use the WorkTicketSearchMetaData.XML file.

Use the following table to configure properties for each search criteria field. You cannot add a new group to this screen:

Element Attribute Description
Group Id Contains the value for the group ID. Do not edit this value.
Group Caption Type the text that will appear as the group header caption. The default caption is Search Criteria.
Group lboName Contains the name of LBO that is used to validate search criteria. Do not edit this value.
Group methodName Contains the name of the method that is used to retrieve data based on the search criteria entered. Do not edit this value.
Field FieldName Type the field name you want to use as a search criterion.
Field ResourceId Type the name of the LBO column from which to retrieve data.
Field DisplayName Type the caption for the field that will be displayed on the screen.
Field Value  
Field Type Type the data type for the field. The possible values are CHAR255, BOOLEAN, and INTEGER.
Field Parent If the field is a child field, then type the FieldName value of the parent field.
Field SequenceIndex Type a numeric value to specify the sequence that will appear on the screen.
Field ControlType Type the control type to use for the field. The possible values are TextBox, ComboBox, DropDown, CheckBox, URL, Link, and RadioButton.
Field MaskFormat Type a value to specify if the field validates a mask format for the value typed by the user. The possible values are PostalCode, e-mail, and Phone.
Field IsRequired
  • Type True if a value must be entered in this field to run a successful search.
  • Type False if entering a value in the field is optional.
Field IsEditable
  • Type True to enable users to enter values in the field.
  • Type False to make the field read-only.
Field Visible
  • Type True to enable users to view the field on the screen.
  • Type False to hide the field.
Field CacheName Type the name of the OEAS Reference Lookup table for reference fields, if applicable.
Field CacheParentId Do not change this value. In the current implementation, your changes will not be reflected in the application.
Field PropertyPath Type the XML node path for the field in LBO.
Field LBOObjectName Type the name of LBO the field is associated with.
Field LboProperty Type the name of the LBO property mapped to the FieldName.
Field TypeIdProperty Type the LBO property name for the primary value selected for a collection field.
Field TypeIdValue Type the type ID value to match with OEAS. This attribute is needed only for collection fields.
Field maxLength Type the maximum field length.
Strings   Do not enter any attributes for this node.
