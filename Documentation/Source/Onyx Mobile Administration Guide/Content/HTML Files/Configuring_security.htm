Configuring Security

Using OES Security Administration, you can secure access to Onyx Mobile App/Onyx Mobile Web and its various features. By granting or denying access to UI resources, you control the actions your users can perform.

Set access permissions to UI resources in order to:

  • Grant or deny access to Onyx Mobile App/Onyx Mobile Web.
  • Grant or deny access to navigation menu items on Details screens.

You can use the default UI resources included in your Onyx application, or you can create new resources based on your organization's business requirements. By default, all users are denied permission to the new UI resources created for Onyx Mobile App/Onyx Mobile Web. You must grant access based on your user's needs. It is a good practice to secure UI resources based on roles and then set more restrictive permissions for users within each role.

Resource used for Onyx Mobile App/Onyx Mobile Web log on
This Resource... Provides Access To...
UI.Mobile.logon Onyx Mobile App/Onyx Mobile Web. Users without permission to this resource cannot log on to Onyx Mobile App/Onyx Mobile Web.
Resources used for Onyx Mobile Change Password
This Resource Provides Access To...
Onyx:ChangePassword Change the password - it restricts the call to Change Password API call.
