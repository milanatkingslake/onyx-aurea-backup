S3 Utility

The S3 Utility feature allows you to perform the following three actions on Sales / Service / Support incidents.

  • Send Email

The Send Email option allows you to send the configured email to the primary contact of an incident. This email is available also under Email tab of the record in OEP.

  • Send Sign Email

The Send Sign Email option allows you to send the configured email with signature to the primary contact of an incident. When you click this option, the signature box appears. You can sign in that signature box. This email is available also under Email tab of the record in OEP.

  • Send Attachment

The Send Attachment option allows you to store an attachment for an incident. This attachment is available also under Attachment tab of the record in OEP.

The S3 Utility is supported only on the Onyx Mobile App.

Configuring S3 Utility

To configure the S3 Utility, perform the following steps:

  1. Open the required XML(s) file in edit mode from the \Onyx\AppServer\Applications\Onyx\OnyxGatewayService\App_Data\Profiles folder. For the list of screens and their XML file names, see Understanding XML files.
  2. Add the following element node in the required XML file(s):
  • Send Email

<MenuItem MenuItemId="15" DisplayName="Send Email" Icon="SendEmailSignatureDoMore.png" ResourceId="" Web ="False" Action="SendEmail" TemplateId="1" isSign="false" PageTitle="Sign & Send Email" >
</MenuItem>

Where,

  • Action="SendEmail" is not customizable.
  • TemplateId is the emailTemplate Id from the OnyxDocumentMergeTemplate.config.xml file. To understand more about the OnyxDocumentMergeTemplate.config.xml file, see Details of OnyxDocumentMergeTemplate.config.xml file.
  • isSign="false" is not customizable.
  • Send Sign Email

<MenuItem MenuItemId="15" DisplayName="Sign Send Email" Icon="SendEmailSignatureDoMore.png" ResourceId="" Web ="False" Action="SendSignEmail" TemplateId="1" isSign="true" PageTitle="Sign & Send Email" TextVariables="ownerName;description1;statusId;productId;priorityId" Text="#ownerName, Please acknowledge the product: #productId and #description1 for its Status : #statusId and Priority: #priorityId">
</MenuItem>

Where,

  • Action="SendSignEmail" is not customizable.
  • TemplateId is the emailTemplate Id from the OnyxDocumentMergeTemplate.config.xml file. To understand more about the OnyxDocumentMergeTemplate.config.xml file, see Details of OnyxDocumentMergeTemplate.config.xml file.
  • TextVariables are the object properties to retrieve the corresponding information on the Signature screen.
  • Text is the textual detail of the entity on the Signature screen.
  • isSign="true" is not customizable.
  • Send Attachment

<MenuItem MenuItemId="16" DisplayName="Attach & Send" Icon="SendEmailSignatureDoMore.png" ResourceId="" fileName="OnyxAttachment.pdf" outputFormat="pdf" Web ="False" Action="SendAttach" TemplateId="1" isSign="false" PageTitle="Sign & Send Email">

</MenuItem>

Where,

  • fileName is the name of the file which goes as attachment the email.
  • outputFormat is the format of the attachment in email. Supported formats: pdf, docx, html, jpeg, rtf, txt, and xps.
  • Action="SendAttach" is not customizable.
  • TemplateId is the emailTemplate Id from the OnyxDocumentMergeTemplate.config.xml file. To understand more about the OnyxDocumentMergeTemplate.config.xml file, see Details of OnyxDocumentMergeTemplate.config.xml file.
  • isSign="false" is not customizable.
  1. Save and close the XML file.
  2. Log on to Onyx Mobile App and verify the changes.
