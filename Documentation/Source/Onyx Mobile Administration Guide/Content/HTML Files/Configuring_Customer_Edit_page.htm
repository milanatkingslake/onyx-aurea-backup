Understanding Onyx Mobile App/Onyx Mobile Web Add/Edit Screens

Users can add or modify the following or enter specific values to search for customer records in the Onyx Mobile App/Onyx Mobile Web Add/Edit screens:

  • Customer
  • Task
  • Incident records
  • Worknotes
  • WorkTickets
In this screen... Users can...
Add or Edit Company
  • Add a new company record to their Onyx application using their iOS/Android device.
  • Add details such as company name, address, phone, and e-mail.
  • Link a primary contact to the company.
  • Modify previously entered information for a company record.
Add or Edit Individual
  • Add a new individual record to their Onyx application using their iOS/Android device.
  • Add details such as individual name, address, phone, and e-mail.
  • Link a primary contact to the company.
  • Modify previously entered information for an individual record.
Add or Edit Task
  • Add a new task to their Onyx application using their iOS/Android device.
  • Add details such as description, status, priority, type, recall date, and worknotes.
  • Modify previously entered information for a task.
Add or Edit Incident Add a new incident record to their Onyx application using their iOS/Android device. Add details such as description, status, priority, type, recall date, and worknotes after selecting an incident category to create the incident. Modify previously entered information for an incident record. IncidentCategorList.xml contains the UI resource permission for adding an incident for a particular category. The xml contains the following:
  • Resource ID. Contains the UI Resource permission value.
  • Value. The resource permission of the user. Set always to nil.
  • Key. The ID of the Incident Category.
Add Worknotes Add a new work-note to an incident or task record in their Onyx application using their iOS/Android device.
Add or Edit WorkTicket
  • Add a new workticket to their Onyx application using their iOS/Android/Desktop device.
  • Add details, such as description, status, priority, type, and worknotes.
  • Modify previously entered information for a workticket.
