Step 2: Install Onyx CTI to the OEDB

In this step, you run a Setup file that installs the Onyx CTI database objects to an existing OEDB.

For Onyx CTI upgrades: Your migrated Onyx CTI data is automatically upgraded during this step.

To install Onyx CTI to the OEDB:

  1. Ensure that the existing OEDB is backed up.
  2. Copy the \Setup\Database folder from the Onyx CTI product CD to a local machine (the OEDB server or a separate machine).

For the OEDB Setup to run properly, the Microsoft SQL Server 2008 or 2012 client tools must be installed on the local machine to enable SQL Server authentication.

  1. Within the copied Database folder, run the Setup.hta file, bypassing any security warnings that might appear.
  2. In the Onyx CTI OEDB Setup window, click Next and then type the logon information as follows:

    3. In this box Type this
    Server OEDB server name
    Database OEDB name
    Login User name. Must have database owner (dbo) privileges
    Password Password for the typed user name
  3. Click Install Now.
  4. Click Finish.
  5. Confirm that no errors occurred during the installation process by viewing the oedb_setup.log file in the copied Database folder.

If you find errors in the oedb_setup.log file, restore the previously backed up OEDB and then rerun the Setup file
