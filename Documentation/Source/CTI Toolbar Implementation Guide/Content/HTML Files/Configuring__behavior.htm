Configuring Onyx CTI behavior

You can configure the behavior of Onyx CTI by editing the configuration file for your connector. (Full path: YourOEPwebsite\cti\YourConnector\cti_config.xml, where YourOEPwebsite is your OEP installation directory and YourConnector is the folder containing the code for your connector.) This file enables and disables the following for your connector:

  • Display of the button accessing the test driver interface.
  • Automatic answering of incoming calls.
  • Display of the Log On window when the Logon toolbar button is clicked.
  • Automatic population of the Place Call window with the primary phone number from the currently open customer page.

A separate copy of the configuration file exists in the cti folder. (Full path:
YourOEPwebsite\cti\cti_config.xml, where YourOEPwebsite is your OEP installation directory.) This file is called the main configuration file and exists only to specify the default connector. Other settings are used only when the configuration file for the default connector cannot be found. For a detailed description of the configuration file (including elements and available values), and for information about the test driver and its interface, see the Onyx Computer Telephony Integration Developer’s Reference. For multiple-connector installations, see Assigning connectors to agents.

To specify the default connector:

  1. In the main configuration file, set the defaultConnector element to the ID of your custom connector.

The following example specifies the default CTI connector as the connector that has the ID “54”.

<defaultConnector>54</defaultConnector>

Full path of the main configuration file: YourOEPwebsite\cti\cti_config.xml, where YourOEPwebsite is your OEP installation directory.

  1. On the client machine used to access the toolbar, delete the files in Internet Explorer's Temporary Internet Files folder to ensure that the changes take effect the next time you display the Onyx CTI toolbar.

To enable or disable other Onyx CTI behaviors:

  1. In the configuration file for your connector, specify a value of “0” to disable the behavior or “1” to enable the behavior.

The following example disables display of the CTI toolbar button that accesses the test driver interface.

<debugMode>0</debugMode>

Full path of your connector’s configuration file: YourOEPwebsite\cti\YourConnector\cti_config.xml, where YourOEPwebsite is your OEP installation directory and YourConnector is the folder containing the code for your connector.

  1. On the client machine used to access the toolbar, delete the files in Internet Explorer's Temporary Internet Files folder to ensure that the changes take effect the next time you display the Onyx CTI toolbar.
