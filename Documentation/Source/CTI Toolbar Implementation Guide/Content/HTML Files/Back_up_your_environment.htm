Back up your environment

Before installing or upgrading Onyx CTI, back up, validate, and publish the Onyx Enterprise Dictionary (OED) file on the Onyx Enterprise Application Server (OEAS) machine, back up the Onyx Enterprise Database (OEDB), and back up the following files on the Onyx Employee Portal (OEP) server:

  • YourOEPwebsite\powerpage\application_main.js
  • YourOEPwebsite\powerpage\oep_automate.asp
  • YourOEPwebsite\powerpage\ucf_application_main.asp
  • YourOEPwebsite\QuickSearch\search-client.js

Where:

Your OEPwebsite is the OEP installation directory (default: C:\Program Files\Onyx\Employee Portal).
