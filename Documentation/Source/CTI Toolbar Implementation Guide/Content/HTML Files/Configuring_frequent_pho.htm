Configuring frequent phone numbers for connectors

You can configure a list of frequently used phone numbers for a connector by adding entries to the CTI - Frequent Phones table in the OEDB. This configuration is intended for installations in which these lists are not provided by the CTI systems.

To add entries to a connectors list of frequently used phone numbers:

  1. Using OES Table Administration, select the CTI - Frequent Phones table.

    For information about Table Administration, see the OES Help.
  2. In the Filter list, select the connector ID for your custom connector.
  3. For each phone number entry, click the Add button and then enter the following information:

    4. In this field Enter the following information
    connectorId Connector identifier. Automatically populated with the connector ID selected in the Filter list.
    Changes to the default value do not appear in the <MadCap:variable name="FrontMatter.ProductNameNoVersion" interface.
    phoneNumber Phone number.
    description Description of the phone number.
    active Indicator of this item's active status, which determines the item's visibility in the Onyx CTI interface. Leave the default value of 1 (active).
  4. Click the Save button.
    The OEDB assigns a unique ID (primaryId) to each phone number entry.

For more information about the CTI - Frequent Phones table, see documentation about the ctiFrequentPhones object in the Onyx Computer Telephony Integration Developer's Reference.
