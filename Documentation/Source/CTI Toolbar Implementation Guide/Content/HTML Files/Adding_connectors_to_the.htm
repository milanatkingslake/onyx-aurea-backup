Adding connectors to the database

You must add your connector to the CTI - Connectors table in the OEDB if you want the Onyx CTI toolbar to use your connector. The sample connector is automatically added to this table during installation of Onyx CTI.

Onyx CTI uses this table to access and instantiate your custom connectors. If the entered information is not correct, errors occur when Onyx CTI attempts to load your connector.

To add a connector to the OEDB:

  1. Using OES Table Administration, select the CTI - Connectors table.

For information about Table Administration, see the OES Help.

  1. Add the connector: Click the Add button and then enter the following information:
    In this field Enter the following information
    primaryId (automatically generated)
    className JavaScript class name of the connector. Ensure that this entry exactly matches the case-sensitive value of the custom connector’s JavaScript class.
    description Description of the connector
    connectorFolder Name of the folder (in the cti folder) in which the connector code is located. Ensure that this entry exactly matches the case-sensitive value of the custom connector’s folder
    active Indicator of this item’s active status, which determines the item’s visibility in the Onyx CTI interface. Leave the default value of 1 (active)
  1. Click the Save button.
    The OEDB assigns a unique ID (primaryId) to the connector.
  2. To point the Onyx CTI toolbar to the newly
    added connector, edit the main configuration file by setting the defaultConnector element to the primaryId of your custom connector.
    The full path of the main configuration file is YourOEPwebsite\cti\cti_config.xml, where YourOEPwebsite is your OEP installation directory.
  3. To confirm that the Onyx CTI toolbar appears and that it uses the custom CTI connector:
  1. Stop and restart your OEP Web server, such as by using the Internet Information Services (IIS) Manager tool or by executing the IISReset command (via Command Prompt).
  2. On the client machine used to access the toolbar, delete the files in Internet Explorer's Temporary Internet Files folder.
  3. Log on to OEP with the user name and password of a user that has been granted access to the toolbar. (The system administrator is granted toolbar access by default.)
