Configuring Onyx CTI

This chapter explains how to configure Onyx Computer Telephony Integration (Onyx CTI). Configuration of Onyx CTI typically includes:

  • Adding connectors to the database
  • Configuring access to the toolbar
  • Assigning connectors to agents
  • Configuring frequent phone numbers for connectors
  • Configuring behavior
  • Configuring availability and visibility of toolbar buttons
  • Configuring interface text
