Gather the required Setup information

Before beginning installation or upgrade, collect the following information required by the Setup files:

For this Setup file Collect the following information
OEDB Name of the machine on which you are running the Setup.hta file for the OEDB.
OEDB server name.
OEDB name.
SQL database user. Must have database owner (dbo) privileges to the specified database.
Password for the specified SQL database user.
