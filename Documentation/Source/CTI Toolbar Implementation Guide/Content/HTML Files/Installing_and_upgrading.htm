Installing and upgrading

The installation/upgrade process involves the following steps:

  • Step 1: Clean the migrated data for upgrade
  • Step 2: Install to the OEDB
  • Step 3: Install to OEAS
  • Step 4: Install to OEP
  • Step 5: (Optional) Simulate with the sample connector and test tools
  • Step 6: Install the vendor-specific custom connector
  • Step 7: Configure access to the toolbar
  • Step 8: Point the toolbar to the custom connector
