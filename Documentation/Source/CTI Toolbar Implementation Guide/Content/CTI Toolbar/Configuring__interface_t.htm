Configuring Onyx CTI interface text

You can configure the text that appears in the Onyx CTI interface by editing the Onyx CTI resource string files. Onyx CTI interface text includes window titles, captions, toolbar button labels, image tooltips, and messages (primarily messages that occur outside the database.

For example, you could configure the Onyx CTI interface text to use the phrase “Center ID” instead of “Station ID.” To complete this configuration, you would search the files for the phrase “Station ID” and replace each occurrence with the phrase “Center ID.”

In this example, the line of code that configures the text for the station ID entry message would look like this:

<string name="msgEnterStationId" text="Enter your Center ID."/>

To configure Onyx CTI interface text

  1. Edit the text attributes for the desired strings in the following resource string files, where YourOEPwebsite is your OEP installation directory and language is the language-specific folder (such as "eng"):
  • YourOEPwebsite\language\cti\ctitoolbar.xml
  • YourOEPwebsite\language\cti\ctidialogs.xml 

Some attributes are required to correspond to attribute values in other files, such as the enumeration file. Before editing other attributes in this file, review element and attribute descriptions for the file in the Onyx Computer Telephony Integration Developer’s Reference.

  1. On the client machine used to access the toolbar, delete the files in Internet Explorer's Temporary Internet Files folder to ensure that the changes take effect the next time you display the Onyx CTI toolbar.
