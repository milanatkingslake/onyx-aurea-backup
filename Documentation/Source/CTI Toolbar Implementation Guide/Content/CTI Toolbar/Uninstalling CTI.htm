Uninstalling

Onyx CTI can be uninstalled by performing the following steps:

  1. Browse to Control Panel,and select Programs in the server that has CTI installed . Choose Customer Telephone Interface from the list of programs displayed and right-click to uninstall.
  2. To revert the Onyx Employee Portal Changes, update the mbEnableCTIToolbar attribute value to false in the application_main.js file from the \Program Files\Onyx\EmployeePortal\powerpage folder.
  1. To revert the Onyx Enterprise Dictionary, disable the following objects in the Logical Business Objects in the Onyx Enterprise Studio.
  • ctiAgentAssociation
  • ctiConnector
  • ctiFrequentPhones
  1. Shutdown COM+ and BRM.
  2. Reset IIS.
  1. Publish the OES.
  2. Shutdown COM+ and BRM.
  3. Reset IIS.

CTI is successfully uninstalled.

Shutting down of COM+ and BRM , and resetting of IIS are repeated to ensure that the changes due to OES publish are reflected.
