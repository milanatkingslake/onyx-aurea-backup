Configuring availability and visibility of Onyx CTI toolbar buttons

You can correlate the availability and visibility of Onyx CTI toolbar buttons to the desired agent states and call states by editing the configuration file for your connector. This configuration can reduce agent errors during phone calls.

For example, you may want to prevent agents from changing their states to “Wrap Up” when the agent’s current call is on hold. Agents change their states to “Wrap Up” by clicking the Wrap Up button. For purposes of this example, let’s assume that you want the button to remain visible for the “on hold” call state (even though it’s disabled) and that for all other call states, you want the button both enabled and visible. To complete this configuration, you would edit the Wrap Up button code line for each agent state and call state. You would find the Wrap Up button code line under the “on hold” call state and specify a “0” value for its enabled attribute and a “1” value for its visible attribute. For the Wrap Up button code lines referring to other states, you would specify a “1” value for both attributes.

In this example, the Wrap Up button code line for the “on hold” call state would look like this:

<control controlId="agentWrapUp" enabled="0" visible="1"/>

The Wrap Up button code line for other states would look like this:

<control controlId="agentWrapUp" enabled="1" visible="1"/>

To configure the availability and visibility of an Onyx CTI toolbar button for a particular state

  1. Edit the enabled and visible attributes for the toolbar button and the state (agent state or call state) in the configuration file for your connector.

    The full path of this file is YourOEPwebsite\cti\YourConnector\cti_config.xml, where YourOEPwebsite is your OEP installation directory and YourConnector is the folder containing the code for your custom connector.

Some attributes must correspond to attribute values in other files, such as the enumeration file. Before editing other attributes in this file, review element and attribute descriptions for the file in the Onyx Computer Telephony Integration Developer’s Reference.

  1. On the client machine used to access the toolbar, delete the files in Internet Explorer's Temporary Internet Files folder to ensure that the changes take effect the next time you display the Onyx CTI toolbar.
