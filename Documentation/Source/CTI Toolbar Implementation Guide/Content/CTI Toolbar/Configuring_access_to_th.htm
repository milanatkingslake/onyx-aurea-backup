Configuring access to the Onyx CTI toolbar

You can configure access to the Onyx CTI toolbar by granting and denying the desired Onyx roles and users permission to the UI:OEP.CTI.Toolbar resource. The toolbar appears in OEP for the roles and users to which you grant access only.

To configure access to the Onyx CTI toolbar

  1. Using OES Security Administration, select the UI:OEP.CTI.Toolbar resource (expand Resources and then expand UI Resources).
    For information about Security Administration, see the OES Help.
  2. Click Manage Permissions.
  3. Add the desired roles and users, click Done, and then click the Save button.
  4. On the client machine used to access the toolbar, delete the files in Internet Explorer's Temporary Internet Files folder to ensure that the changes take effect.
