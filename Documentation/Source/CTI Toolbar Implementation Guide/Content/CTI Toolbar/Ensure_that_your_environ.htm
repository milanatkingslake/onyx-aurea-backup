Validate system requirements

Ensure that your environment meets the system requirements for Onyx CTI. If you choose to remotely install files to the Onyx Enterprise Database (OEDB), ensure that the remote machine meets the system requirements and that the Microsoft SQL Server Server 2012 or 2014 or 2016 client tools are installed to enable SQL Server authentication. For details on the system requirements, see the System Requirements for Onyx Computer Telephony Integration.
