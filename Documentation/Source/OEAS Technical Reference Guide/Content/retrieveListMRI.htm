retrieveListMRI

The retrieveListMRI method of the userPreference object searches the database by userId and returns a list of the most-recently-used customer records.

Remarks

The number of records returned depends on the value of the oepNumRowsMRUList system parameter. You can modify this value by using the System Parameter Administration tool.

This method returns records that are sorted first by name (in reverse-alphabetical order) and then by customerPrimaryId (in ascending order).

If you want to sort the records by modified date, you must do so within the presentation tier.

The primaryId comes from the value property of the userPreference instance. It is prefaced by the customer type (1 for individual or 2 for company). The customer type and primaryId are separated by a tilde.
