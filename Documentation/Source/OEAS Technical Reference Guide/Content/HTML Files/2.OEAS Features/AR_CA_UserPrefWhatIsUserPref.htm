User Preferences

User preferences are client settings that users can maintain such as customizing the appearance of the user interface, setting default field values to simplify data entry, and defining search criteria.

The profile feature relates to user preferences and supports customizing the client with a set of preferences for different user roles within an organization. OEP uses this feature for its UI profiles administration. For more information, see "Managing OEP UI profiles" in the Security Administration Help and the OEP Technical Guide.

The User Preference feature is implemented with the userPreference object. Clients use object methods to create, access, and maintain stored preferences. It is the client's responsibility to develop the user interfaces for setting and administering user preferences.

A preference can be specified for many users, but a particular setting for a user preference is unique to each user.
