Outputs

This method call requires you to create an output method call detail that can return a list of products associated with the customer as customer product records, as described in the following table.

Properties that require a method call detail Details

Customer products rowSet

This method call requires an output method call detail that is based on the rowSet returned by the OEAS method, so that the method call can return a list of products.

The following table describes the values to define for the attributes of the output method call detail.

For additional information regarding the method call detail attributes, see Method call detail attributes.

Attribute Value/Description

Name

Provide a logical name for the method call detail.

Description

Provide a logical description for the method call detail.

Type

As this method call detail represents a list of values, select list.

Active

Select this check box to activate the method call detail within the method call.

Element Path

Type or select from the Element Path Helper: rowSet/rows/row

In the Method Call Designer, you can use the Element Path Helper to select the rowSet/rows/row path, although the path is created automatically when you use the Method Call Detail Builder to create the method call detail.

ID Path

Type c0 for this attribute.

The ID Path is relative to the Element Path, so this attribute specifies that the first part of the returned list is the customer product record’s primary ID. The value specified is the column ordinal from the returned rowSet for the primaryId.

Description Path

Type c5 for this attribute.

The Description Path is relative to the Element Path, so this attribute specifies that the second part of the returned list is the description of the customer product. The value specified is the column ordinal from the returned rowSet for the product description.
