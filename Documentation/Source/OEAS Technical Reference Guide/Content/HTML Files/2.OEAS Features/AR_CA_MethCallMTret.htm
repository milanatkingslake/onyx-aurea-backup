retrieve

The following method call example uses the individual.retrieve OEAS method. This method call is designed to:

  • From the OEP Power Page, accept an individual’s primary ID for retrieving that individual’s information.
  • Return the individual’s first name, last name, and email address.

Using Method Call Designer, you can create this method call by defining the method call and method call detail attributes.

Define the method call attributes

The following table describes the values to define for the method call attributes.

For additional information regarding the method call attributes, see Method call attributes.

Attribute Value/description

Name

Provide a logical name for the method call.

Description

Provide a logical description for the method call.

Object

Select individual.

Method

Select retrieve.

Input Object

Select individual.

Active

Select this check box to activate the method call so that it is available for use.

Define the method call detail attributes

You create an input method call detail for each element to be passed to the underlying OEAS method, as well as for each parent element that is also included in the element path. You also create output method call details based on the properties of the individual object that you want the method call to return.
