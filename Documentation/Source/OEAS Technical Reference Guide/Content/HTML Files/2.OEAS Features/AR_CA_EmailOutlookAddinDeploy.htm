Onyx Email for Outlook

Onyx Email for Outlook is a plug-in for Microsoft Outlook that enables OEP users to save email messages from their Outlook mailboxes into OEP. Emails that are saved into OEP are recorded and tracked in the Onyx Enterprise Database.

The Onyx administrator is responsible for validating the setup and installation on a client workstation and for rolling out the Email for Outlook program to OEP users. For more information about the tasks involved, see the following:
