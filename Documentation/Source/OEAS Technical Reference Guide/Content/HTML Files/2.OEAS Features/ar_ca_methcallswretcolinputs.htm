Inputs

This method call requires you to create input method call details for the elements described in the following table.

Element Details

externalContacts

A method call detail representing the parent externalContacts collection node is required by the method call in order to properly construct the XML.

externalContact

A method call detail representing the parent externalContact node is required by the method call in order to properly construct the XML.

primaryId

This method call requires input method call details that are based on all of the required input parameters of the externalContact.retrieveCollection OEAS method. As such, a method call detail representing the primaryId input parameter is required. However, no value needs to be passed to this parameter.

ownerId

This method call requires an input method call detail that is based on the ownerId input parameter for the externalContact.retrieveCollection OEAS method. The value that is passed by this method call detail is the primaryID for the company for which you are looking up the partners.

ownerType

This method call requires an input method call detail that defaults the ownerType to the value that represents the “company” owner type.

The following table describes the values to define for the attributes of the input method call details.

For additional information regarding the method call attributes, see Method call attributes.

Attribute Value/Description

Name

Provide a logical name for each method call detail.

Description

Provide a logical description for each method call detail.

Type

The default and only valid selection for an input method call detail type is scalar.

Hidden

ownerId - For the method call detail representing the ownerId element, do not select this check box. This input must be visible to the script designer since he or she must include this method call detail in the script so that the script end-user can input the company ID.

For all other elements, select this check box so that the input detail is not available to the script designer.

Active

Select this check box for each method call detail to activate it within the method call.

Default to Null

Do not select this check box. A null value is not appropriate for these method call details.

Default Value

ownerType - For the method call detail representing the ownerType element, type the value 2. Valid values for the externalContact ownerType property include 1 (individual), 2 (company), 3 (customerProduct), 6 (incident), 8 (task), and 10 (workTicket). In this case, the default value is 2, because you want to return the external contacts for a company object.

For all other elements, save with the default setting of no default value. A default value is not appropriate for these method call details.

Element Path

For each of the following elements, specify the given path, typing it or selecting it from the Element Path Helper:

externalContacts: externalContacts externalContact: externalContacts/externalContact primaryId: externalContacts/externalContact/primaryId ownerId: externalContacts/externalContact/ownerId ownerType: externalContacts/externalContact/ownerType

In the Method Call Designer, you can use the Element Path Helper to create this path, although the path is created automatically when you use the Method Call Detail Builder to create the method call detail.

Attribute List

Leave this attribute empty.
