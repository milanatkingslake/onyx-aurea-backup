Relationships

Relationships covered in this section include object to table mappings, database table relationships, and logical business object relationships.

Object to table mappings

The Table Administration feature has only one business object. It maps to the following Onyx Enterprise Database (OEDB) table.

Business object OEDB table

tableAdministrationItem

table_administration_item

table_administration_item_ml

table_admin_item_ml_view

Database table relationships

There are no relationships between the table_administration_item table and other OEDB tables, except for its related multiple language extension table. For more information on multiple language support in OEDB, see the OEAS Developer's Reference.

Object relationships

There are no relationships between the tableAdministration object and any other OEAS object.
