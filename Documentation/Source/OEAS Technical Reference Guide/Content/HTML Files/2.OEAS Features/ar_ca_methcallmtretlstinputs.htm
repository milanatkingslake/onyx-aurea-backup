Inputs

This method call requires that you create input method call details for the elements described in the following table.

Element Details

primaryId

This method call requires an input method call detail that is based on the primaryId input parameter for incident.retrieveListQuickSearch so that the method call can accept the ID as the primaryId input parameter to search for the incident’s contact information.

incidentCategory

This method call requires an input method call detail that is based on the incidentCategory input parameter for incident.retrieveListQuickSearch because incidentCategory is a required input parameter.

The following table describes the values to define for the attributes of the input method call details.

For additional information regarding the method call attributes, see Method call attributes.

Attribute Value/Description

Name

Provide a logical name for each method call detail.

Description

Provide a logical description for each method call detail.

Retrieve Using This Input Detail

primaryId - For the method call detail representing the primaryId element, select this check box to ensure that the primaryId value passed by the OEP UI to the method call maps to this input method call detail.

For all other elements, save with the default setting of a non-selected check box.

Active

Select this check box for each method call detail to activate it within the method call.

Default to Null

Do not select this check box. A null value is not appropriate for these method call details.

Default Value

incidentCategory - For the method call detail representing the incidentCategory element, provide a semicolon-separated list of the incident categories you want to include in this retrieve (for example, 1;2;3).

For the primaryId element, save with the default setting of no default value.

Element Path

For each of the following elements, specify the given path, typing it or selecting it from the Element Path Helper:

primaryId: primaryId incidentCategory: incidentCategory

In the Method Call Designer, you can use the Element Path Helper to create this path, although the path is created automatically when you use the Method Call Detail Builder to create the method call detail.

Attribute List

Leave this attribute empty.
