Method call detail attributes

After you have set the attributes for a method call, you can use the Method Call Designer to add the inputs and outputs to that method call. You do this by setting the following method call detail attributes. Setting these attributes maps the method call input or output to the OEAS method parameter or element and further defines its behavior.

  • Name
  • Description
  • Type
  • Hidden
  • Active
  • Retrieve Using This Input Detail
  • Default to Null
  • Default Value
  • Element Path
  • ID Path and Description Path
  • Attribute List

Certain attributes might or might not be presented in Method Call Designer, depending on the type—“merge template” or “scripts”—of method call you are creating, and depending on whether or not you are creating an input or an output method call detail.
