Configuring attachments for emails

For the Onyx Email for Outlook installation:

  • Create an email account on the mail server for the Email for Outlook mailbox. Confirm that the mailbox is accessible from the Outlook client.
  • Create post-processing folders on the mail server to receive archived messages, error messages, and subsystem error notifications.
  • In Security Administration, create an OPS user for the monitored mailbox.
  • Configure OPS Administration to monitor the Email for Outlook mailbox.
  • Edit the Config XML file (located in the Email for Outlook installation folder) with the email account name for the Email for Outlook mailbox.
  • Install the Email for Outlook on a client workstation to validate the installation process.
  • Roll out the Email for Outlook (.exe) to all OEP client computers.
