Match Code

The Match Code feature is OEAS functionality that enables you to create strings that can be used to prevent the proliferation of duplicate individual and company records in OEDB. For example, you can create a match code for a company that consists of a portion of the companyâ€™s name and primary address. Records with equivalent match codes can be merged to remove duplicates.

By default, the match code feature is disabled in OEAS. For information on how to enable and configure match codes, see "Configuring OEAS features," Match code.
