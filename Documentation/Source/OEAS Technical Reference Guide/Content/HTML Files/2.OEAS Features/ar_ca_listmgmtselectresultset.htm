Selecting the search result set

In addition to selecting search criteria, users creating List Manager searches can select the columns they want the search to return. In the default OEAS implementation, a standard result set is provided for the default company and individual result lists, but users can modify these result sets by removing, adding, and re-ordering columns.

Users can select additional columns for the result set from result set tables. The result set tables identify the columns for each table that can be returned in the result set. The result set tables provided by default are:

  • Company
  • Product
  • Product Line
  • Incident
  • Order (Literature)
  • Phone
  • Internal Contact
  • External Contact
  • Individual
  • Incident External Contact
  • Incident Internal Contact
  • Incident Forecast
  • Incident Forecast Line
  • Incident Quote
  • Incident Quote Line
