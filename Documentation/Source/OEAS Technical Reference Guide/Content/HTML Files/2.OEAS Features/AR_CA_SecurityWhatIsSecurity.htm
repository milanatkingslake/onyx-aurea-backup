Security and user authentication

The Security feature of OEAS is involved in several key functions:

  • Control of user access to OEAS business object methods, user-interface resources (including the Web pages of the Onyx Enterprise Studio (OES) administration and design tools), and other resources that you create in Security Administration.
  • User authentication and licensing.

In the following sections, this chapter discusses the concepts and basic elements of the Security feature, as well as Security Administration, the administration tool of OES that you use to administer security details. See Security Administration Help for specific details of how to complete tasks using the administration tool.

  • User security overview for OEAS and the Onyx Enterprise Database (OEDB) — The basic elements of user authentication, session management, and permissions management
  • Elements of security — Defines the differences between users, roles, and resources
  • User authentication: Login and session management — Conceptual information about the process in which a user is identified by his or her account information and password in order to start a session
