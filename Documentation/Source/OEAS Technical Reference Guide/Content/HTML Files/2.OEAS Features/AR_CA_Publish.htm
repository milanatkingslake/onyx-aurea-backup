Publishing

The Publishing feature provides a way for administrators to output domain data to OEP-Offline. Specifically, the Publishing feature enables administrators to:

  • Define the domain data necessary to support the transactional data that end-users are working with in OEP-Offline.
  • Publish all or a subset of the defined domain data so that it is available to OEP-Offline end-users.

Making changes to domain and reference data is not allowed on the offline client and thus uploading the data back to the online client is not supported.
