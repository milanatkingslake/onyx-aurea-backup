saveCollection

The following example uses the internalContact.saveCollection OEAS method. This method call is designed to:

  • Accept the primary ID for an existing internal contact type (New Business Account Manager) for a delete action on that internal contact.
  • Insert a new internal contact for a company with a contact type of Established Business Account Manager.

Using Method Call Designer, you can create this method call by defining the method call and method call detail attributes based on internalContact.saveCollection.
