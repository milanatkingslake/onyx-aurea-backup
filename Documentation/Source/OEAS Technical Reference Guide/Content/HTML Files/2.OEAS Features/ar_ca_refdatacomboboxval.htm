Combo box values

You use Reference Table Administration to define the list of values that appear in a combo box field.

The domain data values for combo-box fields are stored in the ReferenceParameters table and many of the default values of this table are used by multiple OEAS features. For instance, an incident, task, forecast, and quote might each reference the ReferenceParameters value that describes a possible status (such as “Open”). Because the features share this value, when you modify the value (such as to “Active”), you need to make the change only once. The features reference this value by ID, so they automatically inherit the new value. Referencing the ReferenceParameters values by ID ensures consistency and facilitates reporting and maintenance activities.
