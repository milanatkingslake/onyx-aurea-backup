Templates

The Templates feature is the OEAS functionality that enables users to insert pre-written text into the body of an email message. Templates can contain corporate messages, signatures, headers and footers, etc. They are useful for creating reusable messages for commonly sent business communications, such as an introductory paragraph or a corporate message that a sales representative might send to new clients. Templates can also extract platform information such as incident details, customer information, and more, to insert into the text to personalize the template for the intended recipient.

End users can create templates using Template Maintenance, a feature of the Onyx Employee Portal (OEP). Using Template Maintenance they can create either private templates for themselves or, with the proper security permissions, public templates that others can view and use. See “Managing Templates” in the OEP Online Help for further information.

The main elements of the Templates feature include the following:

  • Template groups
  • Templates
  • Merge templates

Template groups: In Template Maintenance, templates are organized in a hierarchy of template groups to provide logical groupings of templates to end users. You create and administer these template groups through Table Administration, an Onyx Enterprise Studio (OES) administration tool. For more information about creating template groups, see Templates.

Templates: Templates, as noted previously, are saved, pre-written text, including such things as corporate messages, signatures, headers and footers, that you can insert into the body of an email message.

Merge templates: Merge templates are templates into whose text you can merge specific data that was extracted from the platform. Merge templates enable users to generate emails with work ticket, incident, or customer summary information in the body of the message, personalizing the template text for the intended recipient. Merge templates require the use of method calls, which you can create using the Method Call Designer, a design tool of OES. For more information about creating method calls for merge templates, see Templates.
