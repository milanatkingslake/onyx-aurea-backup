Overview of tasks to implement the Email feature

The following is a list of tasks you perform to set up and configure the Email feature company-wide.

  • Set up end user security permissions to enable users to insert, update, and delete emails.
  • Set up an SMTP mail server for sending outbound email messages over the Internet. See the Email feature content in the OEAS Installation Guide.
  • Install and configure the Onyx Processing Server.
  • Configure OPS security and OPS user permissions. See Configuring OPS security.
  • Configure EMS and monitored mailboxes for processing inbound mail. See Customizing OPS.
  • Install and deploy Onyx Email for Outlook to OEP users.
  • Configure attachments for emails. See Configuring attachments for emails.
  • Create email address book item entries (queues) for support/service/sales groups. See Configuring attachments for emails.
  • Configure relative dates to enhance searches for emails in OEP. See Configuring attachments for emails.
  • Create template groups for email templates. See Creating template groups for templates.
  • Create OEAS method calls to retrieve customer/incident/work ticket data for merge templates. See Creating template groups for templates.
