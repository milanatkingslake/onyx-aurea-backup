System Parameters

The System Parameters feature enables the system administrator to control the behavior of several features of Onyx Enterprise Application Server (OEAS), including how features are presented to OEP users. Generally you enable or disable a system parameter to control the behavior of a feature, or input a string or integer value for the system parameter that is recognized by the application.

An example of a system parameter is TagIncludeCount. The TagIncludeCount parameter enables you to control the number of times a tag is used before it is returned by retrieveList.
