ReportNet Integration Module

The ReportNet Integration Module includes Onyx Enterprise Database (OEDB) code, domain data, and middle tier business logic with which you can manage the creation and processing of reports on Onyx transactional data using Cognos ReportNet functionality. The integration enables a seamless view of the ReportNet functionality through the Onyx Employee Portal (OEP) interface. For more information on the ReportNet Integration Module, see the ReportNet Integration Module Implementation Guide.

The OEAS elements that support the ReportNet Integration Module are:

  • OEDB views and stored procedures
  • System parameters
  • Security resources
  • reportNet LBO
  • ReportNetStep component
  • Security integration logical methods
