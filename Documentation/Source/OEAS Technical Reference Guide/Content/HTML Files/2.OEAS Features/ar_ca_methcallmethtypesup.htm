OEAS method types that support method calls

The Method Calls feature supports creating method calls based on all of the standard OEAS methods as well as any custom methods you might have created. However, the Templates OEAS feature supports only retrieve, retrieveCollection, and retrieveList method calls.

In general, method calls can call methods that persist collections of objects (saveCollection methods). However, the behavior of these methods (within the context of the method call) is limited. For more information, see Collections.
