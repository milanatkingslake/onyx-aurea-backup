What is calendaring?

Calendaring is a system that enables OEP users to associate their Microsoft Outlook appointments, meetings, and events with customers and incidents. Calendaring consists of the following components:

  • Calendaring for Outlook Add-in
  • Calendar feature

Calendaring for Outlook Add-in

Onyx Calendaring for Outlook is a toolbar add-in for Microsoft Outlook that enables users to create appointments, meetings, and events in Outlook and then share them with OEP by creating an association. An association is a way of connecting or linking an appointment, meeting, or event with OEP records.

The Onyx database manages the relationships between appointments, customers, and incidents, which enables employees to view, share, and act on the same schedule for a particular customer or incident in OEP.

Each OEP user installs the add-in program on his or her machine using the setup program distributed by the administrator (the administrator is responsible for configuring the setup program before distribution to end-users).  For more information, see "Installing Calendaring for Outlook" in the OEAS Installation Guide.

Calendar feature

Microsoft Outlook must be configured and running on the client machine before accessing the Calendar feature in OEP. For requirements and implementation information, see the OEAS Installation Guide and Customer Central (http://customers.onyx.com).

After the appointment is sent through Outlook, users can view Calendar and appointment entries in OEP from the Header Bar, Home Page, Appointments tab, and the History tab.

For the Calendar feature, you can configure the following.

  • Calendar date text
  • Calendar captions
  • System parameter to enable or disable the Calendar feature
