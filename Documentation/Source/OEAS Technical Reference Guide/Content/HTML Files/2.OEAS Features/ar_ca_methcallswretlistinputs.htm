Inputs

This method call requires you to create an input method call detail for the element described in the following table.

Element that requires a method call detail Details

ownerId

This method call requires an input method call detail that is based on the ownerId input parameter for customerProduct.retrieveListCust so that the method call can accept the customer’s primary ID as the ownerId input parameter.

The following table describes the values to define for the attributes of the input method call detail.

For additional information regarding the method call detail attributes, see Method call detail attributes.

Attribute Value/Description

Name

Provide a logical name for the method call detail.

Description

Provide a logical description for the method call detail.

Type

The default and only valid selection for an input method call detail type is scalar.

Hidden

Do not select this check box. This input must be visible to the script designer since he or she must include this detail in the script so that the script end-user can input the company ID.

Active

To activate the method call detail within the method call, select this check box.

Default to Null

Do not select this check box. A null value is not appropriate for this method call detail.

Default Value

Save this method call details with the default setting of no default value. A default value is not appropriate for this method call detail.

Element Path

Type or select from the Element Path Helper: ownerId

 

In the Method Call Designer, you can use the Element Path Helper to create this path, although the path is created automatically when you use the Method Call Detail Builder to create the method call detail.

Attribute List

Leave this attribute empty.
