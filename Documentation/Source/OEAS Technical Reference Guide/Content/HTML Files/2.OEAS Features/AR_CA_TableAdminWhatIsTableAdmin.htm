Table Administration

The Table Administration tool of the Onyx Enterprise Studio enables you to modify the domain data for several Onyx Enterprise Application Server (OEAS) features by editing, adding, and deleting data used by these features. For example, you can use Table Administration to modify the company family domain data for the Customer feature, as well as the currency domain data used by the Multiple Currency feature, and the country data and region data used by the Customer feature.

For complete details regarding the default domain data you can administer using Table Administration, see Table Administration Help in the Onyx Enterprise Studio.

In addition to the default domain data you can administer using Table Administration, you can also use this tool to configure other Onyx tables and your own custom tables. For more information, see Adding a new object to Table Administration.

Domain data for companion products appear in Table Administration when companion products to OEAS are installed.
