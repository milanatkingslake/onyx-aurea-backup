Security integration logical methods

The ReportNet Integration Module includes support for the ReportNet application server to access OEAS authentication infrastructure. An Onyx component installed on the ReportNet server calls logical methods in OEAS to support this integration. OEAS exposes these methods through Onyx Web Services. The ReportNet Integration Module adds or modifies the following logical methods to support this component.

  • group.retrieveListAncestors (new)
  • role.retrieveList (modified)
  • user.retrieveListUsersAndGroups (new)
  • user.searchForOnyxUserGroupRole (new)
