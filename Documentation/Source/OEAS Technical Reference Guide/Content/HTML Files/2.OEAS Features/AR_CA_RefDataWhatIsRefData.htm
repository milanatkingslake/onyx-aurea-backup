Reference Data

Reference data is a specific type of domain (non-transactional) data available to the Onyx Enterprise Application Server (OEAS). Reference data is different from standard domain data in that it enables you to configure certain existing configurable fields, and to insert, modify, and delete user-defined fields that you intend to display on a user interface.

Configuration can include:

  • Modifying field properties such as caption and field type
  • Creating and modifying lookup values for fields
  • Establishing hierarchical relationships between fields

The elements of reference data that enable this level of configuration are field categories and field types.
