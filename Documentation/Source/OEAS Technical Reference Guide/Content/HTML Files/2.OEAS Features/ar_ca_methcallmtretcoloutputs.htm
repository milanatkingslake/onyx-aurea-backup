Outputs

This method call requires that you create an output method call detail for the internalContact object property described in the following table.

Property Details

contactUserName

This method call requires an output method call detail that is based on this property of the internalContact object so that the method call can return the name of the account manager for the incident.

The following table describes the values to define for the attributes of this output method call detail.

For additional information regarding the method call detail attributes, see Method call detail attributes.

Attribute Value/Description

Name

Provide a logical name for the method call detail.

Description

Provide a logical description for the method call detail.

Active

Select this check box for the method call detail to activate it within the method call.

Element Path

Type or select from the Element Path Helper: internalContacts/internalContact[contactTypeId=5]/contactUserName

 

In the Method Call Designer, you can use the Element Path Helper to select the internalContact/contactUserName path, but you must edit the path to include the contactTypeId filter. This example assumes that you have previously defined an “incident” internal contact type of “Account Manager.” It also assumes that the contact type ID for that internal contact type is 5.

Attribute List

Leave this attribute blank.

This example returned only one element, the contactUserName. If it had returned more than one element, each returned element would require the Element Path filter for the contactTypeId.
