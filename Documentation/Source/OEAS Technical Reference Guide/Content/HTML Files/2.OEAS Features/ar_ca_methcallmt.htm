Merge template method calls

Method calls support merge templates, a component of the Templates feature. The purpose of the merge template is to accept an input record (company, customer, incident, individual, or work ticket) and return some preformatted text containing data based on that record. As such, only retrieval methods are used for merge template method calls, as opposed to insert, update, delete, and saveCollection methods. To be successful, they must:

  • Accept an input parameter that is the primary ID of an input record.
  • Return one or more outputs (that will be merged into the template).

The examples provided in the following sections describe creating method calls for merge templates for the retrieve, retrieveCollection, and retrieveList OEAS method types.

For details regarding merge templates, see Templates.
