Element Details

Contact First Name

This method call requires output method call details that are based on these elements of the returned row so that the method call can return the contact’s first name, last name, and phone number.

Contact Last Name

Phone Number

The following table describes the values to define for the attributes of the output method call details.

For additional information regarding the method call attributes, see Method call attributes.

Attribute Value/Description

Name

Provide a logical name for each method call detail.

Description

Provide a logical description for each method call detail.

Active

Select this check box for each method call detail to activate it within the method call.

Element Path

For each of the following elements, specify the given path, typing it or selecting it from the Element Path Helper:

Contact First Name: rowSet/rows/row/c7 Contact Last Name: rowSet/rows/row/c8 Phone Number: rowSet/rows/row/c10

In the Method Call Designer, you can use the Element Path Helper to select the rowSet/rows/row path, although the path is created automatically when you use the Method Call Detail Builder to create the method call detail. In either case, however, you must edit the path to include the column ordinal for the returned value.
