Performing bulk actions

Once the List Manager search has returned the result set that matches the search criteria, users can perform bulk actions against the entire or partial list. The default actions users can perform to the records selected from the result set include the following:

  • Export the selected items to Microsoft Excel
  • Export the selected items to a CSV file
  • Create sales opportunities for selected items
  • Create service requests for selected items
  • Create support incidents for selected items
  • Add an external contact to selected items
  • Add an internal contact to selected items
  • Create a campaign for selected items
  • Create a literature order for selected items
  • Add a survey to selected items
  • Create script sessions for selected items
  • Reassign territory

In addition to performing bulk actions against the result list, users can select and open any record from the result set.
