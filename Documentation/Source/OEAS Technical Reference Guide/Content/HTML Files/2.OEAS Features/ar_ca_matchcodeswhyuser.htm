Why use match codes?

The ability to generate and compare match codes can be used to prevent or clean up duplicate records. For instance, the match code feature can be used to compare records created using an off-line client application with existing records before the new records are inserted into OEDB. Match codes for existing records in OEDB can also be compared on a regular basis. Records with the same match codes can be combined using Merge Customer Records. For more information on Merge Customer Records, see the OEP Help.
