Attachments

The Attachments feature is the Onyx Enterprise Application Server (OEAS) functionality that enables you to attach files (of any file type) to various Onyx entities. For example, you can attach a fax or letter that you sent to a customer regarding a relevant incident or product.

You can use the Attachments feature to:

  • Add file attachments.
  • View, update, and delete your existing file attachments.
