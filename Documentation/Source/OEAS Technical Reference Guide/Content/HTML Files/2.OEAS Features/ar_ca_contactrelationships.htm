Relationships

When reviewing the object and table relationships, note that:

  • Customer, incident, task, customer product, and work ticket records can all have many internal contacts.
  • Customer, incident, task, customer product, and work ticket records can all have many external (customer) contacts.
  • Each contact type must have a contact group and a contact category.
  • Each contact group can be associated with many contact types.
  • Each contact category can be associated with many contact types.
