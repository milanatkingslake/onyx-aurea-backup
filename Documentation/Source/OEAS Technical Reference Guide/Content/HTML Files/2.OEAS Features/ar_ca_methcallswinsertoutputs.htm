Outputs

This method call requires that you create an output method call detail for the individual object property described in the following table.

Property Details

primaryId

This method call requires an output method call detail based on the individual object’s primaryId property so that the method call can return the ID for the new customer.

The following table describes the values to define for the attributes of the output method call detail.

For additional information regarding the method call detail attributes, see Method call detail attributes.

Attribute Value/Description

Name

Provide a logical name for the method call detail.

Description

Provide a logical description for the method call detail.

Type

Because this method call detail represents a single value, select scalar.

Active

Select this check box for the method call detail to activate it within the method call.

Element Path

Type or select from the Element Path Helper: individual/primaryId

 

In the Method Call Designer, you can use the Element Path Helper to create this path, although the path is created automatically when you use the Method Call Detail Builder to create the method call detail.
