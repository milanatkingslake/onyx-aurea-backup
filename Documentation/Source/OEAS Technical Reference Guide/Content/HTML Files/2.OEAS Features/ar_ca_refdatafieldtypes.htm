Field Types

The field type determines how data is captured and displayed in OEP.

  • Check box - Check boxes display in OEP as boxes which users can check or uncheck to activate or deactivate an option. Check boxes appear directly beside their associated field caption. Check box data values are stored in OEDB as a variable character with a value of 1 (checked) or 0 (not checked). You can configure a check box field type for user-defined fields.
  • Combo box - A combo box field presents the user with a drop-down list of values from which to select, but does not enable users to enter new values or text. User-defined combo box fields can be hierarchically related to one another, and can also share the same lookup values. For more information on combo box fields see Combo box field type.
  • Text box - Text boxes are editable fields in which users can enter and modify text. Data entered in a text box is stored in the OEDB as unicode data and is limited to 255 characters. You can configure a text box field type for user-defined fields.
