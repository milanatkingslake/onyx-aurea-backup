Configurable standard fields

Configurable standard fields are combo box fields (drop-down lists) that users select from in the user interface. Configurable standard fields are not fully configurable because they have a fixed field type (combo box) and fixed hierarchical relationships. The only attributes of configurable standard fields that you can configure are the lookup values that appear in the lists.

Examples of configurable standard fields in OEP are the Status and SIC Code (Standard Industry Code) fields found in the Incident and Company forms, respectively.
