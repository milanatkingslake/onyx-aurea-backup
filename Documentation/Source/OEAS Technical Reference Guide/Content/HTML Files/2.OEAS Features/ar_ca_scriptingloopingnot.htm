Looping within a script not supported

Onyx does not support looping within a script to collect multiple answers to the same set of questions. To gather data for multiple instances of a kind, create a different prompt for each instance.

For example, to gather data on an insurance-policy prospectâ€™s children, create as prompts as the number of children you want to get data about: Child1, Child2, Child3, etc.

If you create a loop within a script, remember that a variable stores only the last value entered for any prompt. Usage of the variable in rules, headers, footers, prompts, or method calls is limited to its current value.
