Site

Site ID is an integer specifying the logical database partition of the OEDB. The default value for the site ID is 1.

We recommend using only one site ID, except if you are an application service provider or an Onyx Embedded CRM partner.

The Site feature enables you to retrieve a list of OEAS sites.

You configure OEAS sites during OEAS setup. If you need to add or remove OEAS sites for any reason, you must re-run OEAS setup. If you need to administer the corresponding database table, you must make the changes directly in the database using Microsoft SQL Server.

Note that in order for OEAS to respect a site, you must configure the site during OEAS setup. For details, consult the OEAS Installation Guide.
