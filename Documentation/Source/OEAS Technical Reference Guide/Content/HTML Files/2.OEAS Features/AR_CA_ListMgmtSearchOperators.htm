Search operators

Search operators are the SQL operators that users can apply to the items that they select as criteria for their List Manager searches. For instance, a user can create a search to retrieve all companies associated with a product serial number similar to “NB3” by using the LIKE operator to narrow the returned results.

The default search operators that are installed with OEDB include:

  • operator symbols, such as:

= (equals)

<> (not equal to)

> (greater than)

<= (less than or equal to)

  • string operators, such as BETWEEN, LIKE, IN, NOT IN, IS NULL and IS NOT NULL.
