User-defined fields

User-defined fields give you the flexibility to configure the application to capture additional data specific to your business needs. They are fully configurable fields, in that you can configure field properties such as caption and field type, create lookup values for fields, and establish hierarchical relationships between the fields. In addition, the user-defined fields are fully supported by OEP or OES with no customization of the presentation tier required. You can create and configure a user-defined field when a corresponding user-defined property exists for an object. The user-defined properties are identified by their naming scheme:

  • user1, user2, user3, and so on
