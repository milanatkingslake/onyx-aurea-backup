Why use keywords?

A typical use of keywords is to enable users to create List Manager queries using a keywords as search criteria.

Example: Using keywords as criteria in List Manager queries

Suppose that on a monthly basis, your banking organization wants to track customer requests to cancel credit card accounts. The process that customer service representatives follow is to create a service request (service incident) for each request, including the keyword cancel. At the beginning of each month, account managers can run a List Manager query for service incidents that:

  • were created the previous month
  • are relevant to a particular product (for example, Gold Card Visa)
  • are associated with the keyword cancel

For details regarding the List Manager feature, see List Manager.
