Method call components

There are two main components to a method call: the method call identifying information and the method call details.

The method call identifying information consists of attributes such as the name and description of the method call, the OEAS object and method on which the method call is based, and an active flag for the method call. The method call details are the inputs and outputs that you create for the method call. The method call details themselves consist of attributes that you define to identify and direct the behavior of each detail.
