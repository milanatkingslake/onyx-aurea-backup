Object relationships

Object relationships - products

The object relationships that support the Products feature are illustrated in the following graphic.

The product object provides methods to manage products, and the literatureProduct object provides methods to manage literature products. The productIndex object methods enable users to view the entire family of products and literature products or view selected subsets.

The productGroup object provides hierarchical information that organizes product and literatureProduct objects. The methods of the productGroup object enable users to view the product group hierarchy and modify it. The parentId property of the productGroup object specifies its owner object. The following diagram shows how productGroup objects with hierarchyIds 1 through 4 depict a product line organization for objects with hierarchyIds 5 through 10. The objects at the bottom of the diagram could be product objects, literatureProduct objects, or productGroup objects (whose trees are not shown).
