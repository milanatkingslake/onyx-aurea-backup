reportNet LBO

The reportNet LBO provides an interface for OEAS clients such as OEP to display ReportNet folder contents and create new folders. OEP calls the reportNet methods from PowerPage and Home Page which contain integrations of ReportNet functionality. The LBO includes two methods.

  • The retrieveFolderItems method returns a list of the children of a folder in the ReportNet content store. The children can include subfolders, reports, report views, shortcuts, and queries.
  • The createFolder method creates a new folder in the ReportNet content store. When Home Page report folders are enabled by the ReportNetHomePageFolderName and ReportNetHomePageFolderPath system parameters, OEP creates a Home Page reports folder for each user if one does not exist.
