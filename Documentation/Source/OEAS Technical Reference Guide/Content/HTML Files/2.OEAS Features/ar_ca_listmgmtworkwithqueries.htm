Working with List Manager searches

Using List Manager, users can:

  • create and run a database search
  • save a database search, including any customized result set definitions
  • run a previously saved database search that was saved by themselves or saved by another user as public (the search parameters are loaded with any previously customized result set column definition)
  • modify a previously saved database search
  • delete a previously saved database search

When saving a List Manager search, users can specify it as public so that all users can access and run the search. Otherwise, the search is available only to the user who saved it.
