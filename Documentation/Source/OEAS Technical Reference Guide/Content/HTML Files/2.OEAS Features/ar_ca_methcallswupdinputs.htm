Inputs

This method call requires you to create input method call details for the elements described in the following table.

Element Details

incident

To properly construct the XML, the method call requires a method call detail representing the parent incident node.

primaryId

This method call requires input method call details that are based on the incident.update’s primaryId, statusId, and onyxTimestamp input parameters so that the appropriate values can be passed to the underlying OEAS method.

statusId

onyxTimestamp

The following table describes the values to define for the attributes of the input method call details.

For additional information regarding the method call detail attributes, see Method call detail attributes.

Attribute Value/Description

Name

Provide a logical name for each method call detail.

Description

Provide a logical description for each method call detail.

Type

The default and only valid selection for an input method call detail type is scalar.

Hidden

incident For the method call detail representing the incident element, you can select this check box. It is not necessary that this input be visible to the script designer.

statusId - For the method call detail representing the statusId element, you can select this check box. A default value is provided, so it is not necessary that this input be visible to the script designer.

Note: User input is necessary for all other elements, so do not select this check box.

Active

To activate each method call detail within the method call, select this check box for each method call detail.

Default to Null

Do not select this check box. A null value is not appropriate for these method call details.

Default Value

statusId For the method call detail representing the statusId element, type the value 115. This example assumes that you have previously defined an incident status of “Pending.” It also assumes that the Lookup Value ID for that incident status is 115 (as identified using Reference Table Administration).

For all other elements, save with the default setting of no default value. A default value is not appropriate for these method call details.

Element Path

For each of the following elements, specify the given path, typing it or selecting it from the Element Path Helper:

  • incident: incident
  • primaryId: incident/primaryId
  • statusId: incident/statusId
  • onyxTimestamp: incident/onyxTimestamp

In the Method Call Designer, you can use the Element Path Helper to create this path, although the path is created automatically when you use the Method Call Detail Builder to create the method call detail.

Attribute List

Leave this attribute empty.
