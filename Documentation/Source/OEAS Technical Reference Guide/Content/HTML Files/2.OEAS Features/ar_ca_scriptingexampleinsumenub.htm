MainMenu step

This step presents a menu of the kinds of insurance that the customer can choose. The Life rule takes the script session to the LifeInsuranceQualifications step if the user selects “life” in response to the InsuranceType prompt. If the user selects either of the other responses, however, the AllOther rule takes the script session to the OtherInsurance step.

In the following table, the rules appear in the order in which they occur in the Rules box, which is also the order in which they are evaluated.

InsuranceType
