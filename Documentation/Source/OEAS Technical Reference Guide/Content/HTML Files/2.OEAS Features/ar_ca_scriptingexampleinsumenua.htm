Region step

This step prompts for the customer’s region (state, province, etc.), based on the customer’s country.

Item Type Name Notes

Prompt

Region

region

Default variable: {address/regionCode}

Output variable: {address/regionCode}

Extra Data specifies that the prompt is a drop-down list box, which is necessarily single-select, and that it is populated based on the country code obtained from the {address/CountryCode} variable.

Method call

Pre

Retrieve an individual

Inputs: primaryId, phone/ownerId, address/ownerId (all supplied by customerPrimaryId)

Outputs: all fields in individual record
