The Email Send step component

The Email Send step component

The Email Send step component is used by the emailMessage.send method to manage outbound mail for the Email feature. The Email Send step component uses Microsoft CDO to compose and send mail to a specified mail server, which in turn sends the mail to its destination.

The system parameters EmailSendStepSmtpMode and EmailSendStepSmtpServer are used by the Email Send component to specify how email messages are sent over the Internet. Messages can be sent from either a local or remote email server on which Simple Mail Transfer Protocol (SMTP) is enabled. You use the System Parameter Administration tool to specify values for these system parameters.

Before users can send emails, an SMTP mail server must be specified to send emails over the Internet. For more information about the Email Send step component and about specifying an SMTP mail server, see the following documentation:

  • Email business objects.
  • Email feature documentation in the OEAS Installation Guide.
  • Setting system parameters for the Email feature

Secure SMTP

The Email Send step component supports sending email with user authentication. It supports the following authentication methods:

  • anonymous (not secure)
  • Basic (authentication parameters sent in clear text). This method requires additional configuration on Exchange Server.
  • NTLM (uses integrated windows authentication)

To enable this feature, you must run the OEAS Configuration utility included on the OEAS CD. For more information see, OEAS Configuration utility.
