Inputs

This method call requires that you create input method call details for the elements described in the following table.

Element Details

company

A method call detail representing the parent company node is required by the method call in order to properly construct the XML.

primaryId

This method call requires an input method call detail that is based on the primaryId input parameter for company.retrieve so that the method call can accept the company’s primary ID to search for that company’s information.

The following table describes the values to define for the attributes of the input method call details.

For additional information regarding the method call detail attributes, see Method call detail attributes.

Attribute Value/Description

Name

Provide a logical name for each method call detail.

Description

Provide a logical description for each method call detail.

Type

The default and only valid selection for an input method call detail type is scalar.

Hidden

company - For the method call detail representing the company element, you can select this check box. This input does not need to be visible to the script designer.

  primaryId - For the method call detail representing the primaryId element, do not select this check box. This input must be visible to the script designer since he or she must include this method call detail in the script so that the script end-user can input the company ID.

Active

Select this check box for each method call detail to activate it within the method call.

Default to Null

Do not select this check box. A null value is not appropriate for these method call details.

Default Value

Do not provide a default value. A default is not appropriate for these method call details.

Element Path

For each of the following elements, specify the given path, typing it or selecting it from the Element Path Helper:

company: company primaryId: company/primaryId

 

In the Method Call Designer, you can use the Element Path Helper to create this path, although the path is created automatically when you use the Method Call Detail Builder to create the method call detail.

Attribute List

Leave this attribute empty.
