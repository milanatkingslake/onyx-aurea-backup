Multiple currency

Multiple Currency is an optional sub-feature to the Forecasts and Quotes feature that you can use to generate forecasts and quotes in up to two different currencies other than the base currency of the product chosen. It provides full currency conversion functionality, including meeting Euro conversion requirements.

The Multiple Currency feature includes:

  • Enabling product creation (in Product Administration) with a base currency for the list price and standard cost of a product
  • Enabling currency conversion from a product’s base currency to another currency when a line item is added to a forecast or quote
  • Enabling additional currency conversion of forecast and quotes line items from the primary currency to a secondary currency, so that users can save forecasts and quotes in two separate currencies
