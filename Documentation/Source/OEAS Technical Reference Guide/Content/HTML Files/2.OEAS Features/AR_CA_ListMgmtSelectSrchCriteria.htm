Selecting the search criteria

By default, List Manager searches return a list of either companies or individuals that match the criteria that the users define by selecting search items from the following search tables:

  • Company
  • Product
  • Product Line
  • Incident
  • Order (Literature)
  • Survey
  • Campaign
  • Phone
  • Internal Contact
  • External Contact
  • Individual
  • Incident External Contact
  • Incident Internal Contact
  • Incident Forecast
  • Incident Forecast Line
  • Incident Quote
  • Incident Quote Line

Each search table contains a list of items specific to that table, which users can choose to include as criteria in their search. For example, items of the Company search table can include customer ID, company name, state code, and company family, whereas items of the Order search table can include product number, ship date, and printer name.
