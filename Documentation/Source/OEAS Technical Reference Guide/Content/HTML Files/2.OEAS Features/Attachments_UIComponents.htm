Main user interface components

Attachments user interface components

In OEP, the Attachments node appears on the Navigation Bar under Customers. When a user clicks the Attachments node from within a customer record, all related file links for the customer appear in the Attachments grid. The Attachments grid (and toolbar) enables the user to add, view, edit, and delete attachments.

To retrieve a file in the Attachments grid, the user selects the file’s link in the Title column, and then downloads the file using the File Download dialog box. The downloaded file appears on the client browser. If the browser cannot determine the file type and its associated application, the user must either specify an application to open the file, or cancel the operation.

In the Attachments grid, when a user clicks anywhere in a row other than the Title column, the Details window appears. The Details window enables the user to edit the details about the attachment.

Using the Email feature, end users can browse for new files, literature items, and existing OEP files to attach to email messages through the Attachments option that appears on the Email Message tab. After a file or literature item is selected, the physical file is uploaded to the file store and displayed in the Files to Add text box. For more information on attachments, see “Configuring OEAS features” in Customizing OEAS.
