Publishable objects

Publishable data is the domain data (including reference data) that is available for use in OEP-Offline. To create publishable data, you must make a publishable object of the business object that identifies the data.

Reference data is domain data that is supported by the combined reference field tables (referenceField, referenceDefinition, and referenceParameters). This data is used primarily for populating drop-down fields in the client, as well as for formatting user-defined fields. For more information see Reference Data.
