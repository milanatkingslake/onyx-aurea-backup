CustomersFound step

This step uses a pre method call to provide the user with a list of customers whose records match the data entered in the CustomerSearch step. The user selects one of the customers. If no customer records match the data entered in the CustomerSearch step, the next step is CustomerInfo, for gathering of data for a new-customer record. If there is a match and the user selects a customer, the next step is GetCustomerRecord.
