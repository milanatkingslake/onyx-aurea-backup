Relationships

The Reference Data feature is comprised of the following elements:

  • Reference field elements
  • Reference lookup elements
  • Reference parameter elements

When reviewing the object and table relationships, note that:

  • For every OEDB table that contains configurable fields (standard configurable fields or user-defined fields), for each configurable field there is a relationship to the reference_parameter table based on reference_parameter_did.
  • A reference parameter element can be related to or included in any number of reference lookup elements.
  • Reference field elements must have a different name, but can share a reference ID.
  • A reference field element can be the parent of any number of other reference field elements.
  • Reference field elements can have any number of reference lookup elements.
