Prompts: considerations when using

For country, region, and reference lookup prompts, if you specify that the prompt is a drop-down list box, do not specify a value greater than 1 in the Min box. This avoids the contradiction of requiring more than one selection in a drop-down list box prompt (by design, only a single selection is possible).

For any prompt type in which you can specify minimum and maximum numbers of end-user responses, if you specify a minimum of at least one response (1 or greater in the Min box), the prompt is required whether you select the Required check box or not.
