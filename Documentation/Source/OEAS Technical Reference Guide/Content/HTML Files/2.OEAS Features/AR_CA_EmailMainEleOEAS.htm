The OEAS Email feature

This section describes the main OEAS elements and the tools and applications that support the Email feature company-wide. For additional setup and configuration details, see the Overview of tasks to implement the Email feature.

The main OEAS elements that support the Email feature are:

  • The Email Send step component
  • Email monitor subsystems
  • Attachments for email messages
  • Email address book item entries (queues)
  • Relative dates to enhance date searches for emails
  • Template groups
  • Merged templates
  • Onyx Email for Outlook
