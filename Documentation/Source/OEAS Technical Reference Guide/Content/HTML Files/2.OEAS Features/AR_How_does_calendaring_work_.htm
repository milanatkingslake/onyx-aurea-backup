How Does Calendaring Work?

From Microsoft Outlook, you open an appointment form and, from the Onyx menu option, select Appointment Associations. A window opens displaying all OEP company, individual, and incident records that are currently associated with this appointment.

If the appointment s new (not yet sent out), the Onyx add-in attempts to translate the selected attendees, resources, and organizer to OEP companies and individuals. You can also search for additional company, individual, or incident records from within the Appointment Associations window, and save them as associations by clicking Save.

When you send out an Outlook Appointment for the first time, the appointment record and all its associations are automatically saved in OEP via the execution of an OEAS Web Service call.

When you update an appointment, associations are not updated, you must save the changes directly from the associations form.
