Search views

Search views provide the starting point for building a List Manager search and determine what type of objects are returned by the search. Search views are not actual database views. By default, search views include Companies and Individuals. The Companies search view is configured to return company object instances and the Individuals search view is configured to return individual object instances. By default, a search on a search view returns a subset of data for the object type indicated based on the result set defined by the search result default.
