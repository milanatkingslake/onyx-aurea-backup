What are Messenger messages, notifications, and reminders?

Reminders and messenger messages, although not related to each other, are each notices that OEP users receive; notifications are the periodic searches for reminders and messenger messages.

When an OEP user sets a recall date for a task or incident and specifies that OEP alert him or her with a dialog box, a reminder is the dialog box that notifies the user of the arrival of the date. The reminder includes full details about the task or incident.

Messenger messages are the messages that OEP users create and send to each other with the Messenger feature.

Notifications do not appear in any user interface, although they are called by user interface programs such as Onyx Employee Portal. The periodic checks by these user interface programs for new messages or reminders helps to keep user sessions alive.

For more information on reminders and messenger messages, see the OEP Help.
