Relationships

Relationships

There are no relationships for the System Parameters feature. The settings of the system parameters are interpreted by the application to determine behavior.

Object to table mappings

The System Parameter feature has only one business object. It maps to the following Onyx Enterprise Database (OEDB) table.

Business object OEDB table

systemParameter

system_parameter

system_parameter_ml

system_parameter_ml_view

Database table relationships

There are no relationships between the system_parameter table and other OEDB tables, except for its related multiple language extension table. For more information on multiple language support in OEDB, see the OEAS Developer’s Reference.

Object relationships

There are no relationships between the systemParameter object and any other OEAS object.
