Merged templates

Merged templates enable users to generate emails that include pre-defined text and key pieces of data from the database, such as the incident ID or customer summary information. Merging data in this way personalizes the template text for the intended recipient. Merged templates use method calls, which you create using the Method Call Designer, a design tool of OES. For more information on creating method calls for merged templates, see the material on configuring templates.

In OEP, an error may occur when you attempt to create an email message using the new merge template, specifically when you click either the Insert Template Text or Merge Data Preview button in the Template Selection window. This problem occurs because certain default values in OES Method Call designer are separated by a comma instead of a semi-colon. To resolve this problem, use a semi-colon (;) as a data separator in the Default Value box of Incident Category. For detailed resolution steps, see Onyx Knowledge Base article #487771.
