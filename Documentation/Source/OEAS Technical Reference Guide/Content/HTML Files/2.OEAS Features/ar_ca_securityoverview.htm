Security overview

User security

In OEAS, security services provide user authentication and session management, licensing, and access-permissions management for resources. User accounts are basic to the function of all these services, as follows:

  • Once authenticated and connected to OEAS, users are the basis of session-state information. After a user is authenticated and a session is established, re-authentication is not required for execution of any subsequent requests during that session.
  • Access-permissions management is administered in Security Administration, where access permissions are granted to users.
