retrieve

The following example uses the company.retrieve OEAS method. This method call is designed to:

  • Accept an ID for retrieving company information.
  • Return the company name, the first and last name of the company’s primary contact, the primary phone and address for the company, and the fax number for the company.

If you created this method call for the Scripting feature, and if you further intended to allow company details to be updated within the script, the output details should include all of the company object’s properties, because partial updates are not supported for the company object.

Using Method Call Designer, you can create this method call by defining the method call and method call detail attributes based on company.retrieve.

Define the method call attributes

The following table describes the values to define for the method call attributes.

For additional information regarding the method call attributes, see Method call attributes.

Attribute Value/Description

Name

Provide a logical name for the method call.

Description

Provide a logical description for the method call.

Object

Select company.

Method

Select retrieve.

Active

Select this check box to activate the method call so that it is available for use.

Define the method call detail attributes

You create input method call details for each element to be passed to the underlying OEAS method, as well as for each parent element that is also included in the element path. You also create output method call details based on the properties of the individual object that you want returned by the method call.
