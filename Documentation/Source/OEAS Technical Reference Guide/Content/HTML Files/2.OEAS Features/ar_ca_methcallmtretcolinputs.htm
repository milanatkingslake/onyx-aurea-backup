Inputs

This method call requires that you create input method call details for the elements described in the following table.

Element Details

internalContacts

A method call detail representing the parent internalContacts collection node is required by the method call in order to properly construct the XML.

internalContact

A method call detail representing the parent internalContact node is required by the method call in order to properly construct the XML.

primaryId

This method call requires input method call details that are based on all of the required input parameters of the internalContact.retrieveCollection OEAS method. As such, a method call detail representing the primaryId input parameter is required. However, no value needs to be passed to this parameter.

ownerId

This method call requires an input method call detail that is based on the ownerId input parameter for internalContact.retrieveCollection so that the method call can accept the incident object’s primaryId as the ownerId input parameter.

ownerType

A method call detail representing the internalContact ownerType property is required by the method call. This method call detail must default to the value that represents an “incident” owner type.

The following table describes the values to define for the attributes of the input method call details.

For additional information regarding the method call detail attributes, see Method call detail attributes.

Attribute Value/Description

Name

Provide a logical name for each method call detail.

Description

Provide a logical description for each method call detail.

Retrieve Using This Input Detail

ownerId - For the method call detail representing the ownerId element, select this check box to ensure that the primaryId value passed by the OEP UI to the method call maps to this input method call detail.

For all other elements, save with the default setting of a non-selected check box.

Active

Select this check box for each method call detail to activate it within the method call.

Default to Null

Do not select this check box. A null value is not appropriate for these method call details.

Default Value

ownerType - For the method call detail representing the ownerType element, type the value 6. Valid values for the internalContact ownerType property include 1 (individual), 2 (company), 3 (customerProduct), 6 (incident), 8 (task), and 10 (workTicket). In this case, the default value is 6, because you want to return the internal contact for an incident object.

For all other elements, save with the default setting of no default value. A default value is not appropriate for these method call details.

Element Path

For each of the following elements, specify the given path, typing it or selecting it from the Element Path Helper:

internalContacts: internalContacts internalContact: internalContacts/internalContact primaryId: internalContacts/internalContact/primaryId ownerId: internalContacts/internalContact/ownerId ownerType: internalContacts/internalContact/ownerType

In the Method Call Designer, you can use the Element Path Helper to create this path, although the path is created automatically when you use the Method Call Detail Builder to create the method call detail.

Attribute List

Leave this attribute empty.
