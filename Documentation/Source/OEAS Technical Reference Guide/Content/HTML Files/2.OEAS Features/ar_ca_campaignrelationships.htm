Relationships

When reviewing the object and table relationships of the Campaigns feature, note the following:

  • A tracking code associates a campaign to a customer, customer product, incident, or literature order record.
  • Campaigns can have many tracking codes, but each tracking code can belong to only a single campaign.
  • Tracking code types are not bound to a particular campaign, but can be reused across many different campaigns.
  • Together, the milestone category and milestone action are the components of a tracking code milestone.
  • Milestone categories and milestone actions can be reused to create new tracking code milestones across many tracking codes.
