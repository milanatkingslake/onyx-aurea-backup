Step 4: Sales manager sends a reply to the sales representative

The sales manager (the bcc recipient) has a comment for the sales representative, and does the following:

  • Creates a reply message in Outlook.
  • On the Save in Onyx toolbar, chooses Save on Send so that the email is recorded and tracked in the database.
  • Sends the email to the sales representative.

In OEAS:

  • The email is forwarded to the monitored mailbox created for Onyx Email for Outlook.
  • EMS finds the email message and assimilates it into OEAS.
  • EMS uses the tracking information in the message to create a link to the email thread. The tracking information is also used to create the non-contact associations to the individual, company, and incident records that were created for the previous email.
