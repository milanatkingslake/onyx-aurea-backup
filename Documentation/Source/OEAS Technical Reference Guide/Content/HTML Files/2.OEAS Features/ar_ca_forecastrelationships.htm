Relationships

When reviewing the object and table relationships of the Forecast & Quotes feature, note that:

  • Quotes and forecasts must have a parent incident.
  • An incident can have many quotes, but only one forecast.
  • A forecast can have many forecast line items (details).
  • A quote can have many quote line items (details).
  • Each forecast or quote line item must have a parent forecast or quote (header).
  • Each forecast or quote line item is related to a single product.
