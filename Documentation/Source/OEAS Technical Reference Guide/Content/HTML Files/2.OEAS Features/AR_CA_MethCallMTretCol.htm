retrieveCollection

The following example uses the internalContact.retrieveCollection OEAS method. This method call is designed to:

  • From an OEP incident details window, accept the incident’s primary ID as the owner ID of the internal contact.
  • Set a default value for the ownerType that maps to an incident owner type.
  • Return the name for the Account Manager for the incident, filtering the returned collection by contactTypeId.

The accuracy of this method call depends on the existence of an OEP client rule that allows only one account manager per customer. When the method call detail filter allows the return of more than one object, only the first object encountered is passed to the user interface.

Using Method Call Designer, you can create this method call by defining the method call and method call detail attributes based on internalContact.retrieveCollection.
