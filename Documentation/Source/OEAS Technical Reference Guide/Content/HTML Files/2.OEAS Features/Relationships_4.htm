Relationships

When reviewing the object and table relationships, note that:

A standard product can be associated with any number of forecast or quote line items.

Standard products, including those with literature attributes, can be associated with any number of customer product, and forecast and quote records.

A literature product can be associated with any number of literature order line items.

An incident product can be related to any number of incidents that are of the same incident category as the incident product.

A work ticket (category) incident product can be related to any number of work tickets.

A literature product can be associated with any number of other literature products as a “label product” when its “is label” flag is set to true.
