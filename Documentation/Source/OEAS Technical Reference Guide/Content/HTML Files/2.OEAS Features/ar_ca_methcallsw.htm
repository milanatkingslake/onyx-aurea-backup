Script method calls

Within the Scripting feature, method calls are executed either before the step is loaded (pre method call) or after the step is saved and before its rules are evaluated (post method call). For script method calls, you can use insert, update, delete, retrieve, and saveCollection methods, as well as custom methods. This enables the method call to return values to the workflow, insert, update, or delete data, or perform integration actions or other functionality that is available in OEAS. Method calls created for the Scripting feature can work with single objects as well as with lists or collections.

The examples provided in the following sections describe creating method calls for scripts for the following OEAS method types:

  • insert
  • retrieve
  • retrieveCollection
  • retrieveList
  • saveCollection
  • update

For additional information regarding use of method calls in the Scripting feature, see Method calls, in the Scripting section.
