insert

The following method call example uses the individual.insert OEAS method. It is a complex insert example: The individual object has a coupled relationship to the address and phone objects, and although the primary phone and primary address for the customer are saved as phone and address objects, they are saved using the individual.insert method.

For a simpler insert example see saveCollection. The saveCollection example includes an insert that does not involve a coupled relationship.

This method call is designed to:

  • Insert a new individual record into the OEDB that includes the customer’s first name, middle name, last name, primary phone number, and primary address.
  • Return the primary key for the new customer record.

Using Method Call Designer, you can create this method call by defining the method call and method call detail attributes based on individual.insert.
