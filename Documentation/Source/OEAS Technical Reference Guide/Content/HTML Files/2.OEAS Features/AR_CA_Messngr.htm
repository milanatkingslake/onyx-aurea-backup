Messenger, Reminders, and Notifications

Reminders and messenger messages, although not related to each other, are each notices that OEP users receive; notifications are the periodic searches for reminders and messenger messages.

  • The Messenger feature enables OEP users to exchange OEP messages and do the following:
  • Send a message with an attachment
  • Open messages and any attachments to the messages that were sent to them
  • Update the read status of a message
  • Delete messages
  • The Reminder feature notifies OEP users that a date (such as an appointment or a deadline) associated with a task or incident has arrived.
  • The Notification feature sends messengers and reminders to OEP users. The Notification feature is not visible to the user.

The only configuration possible for the Notifications feature is to change the frequency of notifications. Changing notification frequency requires altering the code of the Onyx Employee Portal (OEP) user interface.
