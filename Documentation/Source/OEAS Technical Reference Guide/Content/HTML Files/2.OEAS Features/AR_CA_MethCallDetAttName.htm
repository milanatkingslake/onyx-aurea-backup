Name

The method call detail Name attribute is the user-friendly name you provide for the method call detail. It is meant to be understood by the method call user (a user of the Templates or Scripting features). There is no requirement for the name to have any particular relation to the underlying OEAS parameter or element. The name can be a maximum of 50 characters.

The following examples demonstrate possible method call detail names with their underlying XML elements:

Sample name Underlying XML element

incident/secondaryId
