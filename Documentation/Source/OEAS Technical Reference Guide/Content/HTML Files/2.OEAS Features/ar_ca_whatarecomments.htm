What are comments?

A comment is associated with a customer record. It consists of user-entered text that enables end-users to record information about customers. For example, a comment could include a customer’s opinion “liked the training seminar” or judgment “voted against the initiative.” Additionally, a comment could include more personal customer information, such as a customer’s preference “prefers to stay at the Waldorf” or their family members’ names.

When working with comments, consider the following:

  • An end-user can create, update, or delete comments.
  • Any type of information can be entered in a comment (preferences, opinions, judgments, evaluations, and so forth).
  • A customer record can contain many comments.

Example: Using comments to record personal interactions with customers

Following are examples of how comments can be used to record customer information.

After interacting with a customer:

  • An airline reservationist could create the following comment for a customer with special needs.

Title: Needs special assistance
Comment: This customer has a broken foot and will need assistance boarding the aircraft and meeting a connecting flight.

  • A sales representative for a cosmetics company could create the following comment for a customer with allergies.

Title: Allergies
Comment: This customer is highly allergic to scented products and products containing lanolin.

  • A personnel manager for a symphony orchestra could create the following comment for a featured artist.

Title: Order Steinway concert grand
Comment: This artist prefers Steinway but will use Bösendorfer if unavailable. Tuning required before each performance: 9/27, 9/28, 9/29.

When an end-user brings up the customer record they can refer to the comments already entered, and they can choose to edit or delete them. Knowing what the previous comments were for a customer enables end-users to make informed decisions and to provide appropriate responses.
