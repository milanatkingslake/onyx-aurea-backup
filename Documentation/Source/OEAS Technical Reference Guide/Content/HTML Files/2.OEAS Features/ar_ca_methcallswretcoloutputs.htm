Outputs

This method call requires that you create an output method call detail for the element representing the returned collection, as described in the following table.

Element Details

externalContacts collection

This method call requires a method call detail that can represent the returned externalContacts collection. This method call detail must support a returned list of partners (in the event that the company has more than one partner).

The following table describes the values to define for the attributes of the output method call detail. Because the Method Call Detail Builder does not provide an externalContact element to select as an output detail, you must create this detail manually.

For additional information regarding the method call detail attributes, see Method call detail attributes.

Attribute Value/Description

Name

Provide a logical name for the method call detail.

Description

Provide a logical description for the method call detail.

Type

As this method call detail represents a list of values, select list.

Active

Select this check box to activate the method call detail within the method call.

Element Path

For the external contacts collection method call detail, type or select from the Element Path Helper: externalContacts/externalContact[contactTypeId=9]

 

You cannot create this method call detail using the Method Call Detail Builder, so there is no default element path. You must provide the path for the output detail. This example assumes that the contact type ID for the “Partners” contact type is 9. You can find this information in Contact Type Administration.

ID Path

Type primaryId in this box.

 

Because the ID Path is relative to the Element Path, this attribute specifies that the first part of the returned list is the contact’s primary ID.

Description Path

Type contactUserName in this box.

 

Because the Description Path is relative to the Element Path, this attribute specifies that the second part of the returned list is the contact’s name.
