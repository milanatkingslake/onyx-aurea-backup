Search tables

Search tables are actually references to the tables containing the data against which the search criteria are evaluated. The reference is made by a view name for the underlying table. Search tables also include an alias for each view.

For instance, when a user creates a List Manager search for a Companies search view that includes search items from the search table Product, the search table items that OEAS uses to build the underlying database query include the views zCompany and zCustomer_Product, using the aliases of “B” and “C” respectively for referencing columns in the query.
