Accessing the Log Messages

Your OEAS log is viewable as a Microsoft Event Log file (.evt file) that is designed to be read using the Microsoft Event Viewer. If your OEAS logical application is running on multiple servers, you must check the Event Viewer on each server for log messages.

To access your log messages:

  1. Open the Microsoft Windows Event Viewer.
  2. In the navigation area of the Event Viewer, double-click your OEAS logical application. The Event Viewer displays only the log messages that pertain to this application.
  3. To read a specific log message, double-click the desired row. The Event Properties dialog box appears.

The Description section displays the content of the log messages in a pipe-delimited format. That is, each data field is separated by a “|” character. For examples of the information returned in the Description section see Reading the log messages.

The application log can also be accessed from a remote machine. To read the application log remotely, verify first that you have the appropriate permissions. Otherwise you cannot read all of the information in the log. For more information, see the Microsoft Event Viewer online Help.
