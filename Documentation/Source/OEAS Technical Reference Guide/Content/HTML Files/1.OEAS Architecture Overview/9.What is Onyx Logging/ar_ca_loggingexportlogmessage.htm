Exporting the Log Messages

You can export event log data from the Microsoft Event Viewer to another application. Onyx recommends exporting the log messages with comma-separated values (as a .csv file), instead of tab-delimited values (as a .txt file). Microsoft Excel readily interprets CSV files.

To export your log messages:

  1. Using the Microsoft Event Viewer, access your log messages.
  2. From the Action menu, select Save Log File As.
  3. Save the log file with a .csv extension.

You can now open your log messages in an application that interprets CSV files, such as Microsoft Excel.
