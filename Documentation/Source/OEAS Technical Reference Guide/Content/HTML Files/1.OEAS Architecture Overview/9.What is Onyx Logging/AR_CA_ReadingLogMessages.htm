Source

Source can represent several different things, depending on the component that logged the message. When the Onyx component that logged the error is the:

  • Onyx Transaction Manager, Source represents the actual source of the error. Example: Microsoft.FreeThreadedXMLDOM.CoCreateInstance:ostComponent
  • Onyx Query Engine, Source describes which process trapped the error and in which internal routine the error occurred. Example: OnyxEBusinessQE.EBusinessQE:ProcessSprocMultiple()
  • Step component, Source provides a description at the logical method level. It describes which process trapped the error and in which logical method the error occurred.
