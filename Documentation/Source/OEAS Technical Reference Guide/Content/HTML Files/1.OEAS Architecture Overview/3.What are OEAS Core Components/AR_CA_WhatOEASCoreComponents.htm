What are OEAS Core Components?

The Onyx Enterprise Application Server is comprised of the following core components:

  • Security (AuthenticateNT, AuthenticateOnyx, and SessionManager)
  • Onyx Transaction Manager (OTM)
  • Onyx Enterprise Dictionary (OED) cache
  • National Language (NatLang) cache
  • Query Engine

These core components provide the foundation for OEAS. Each of the core components is responsible for one or more essential functions that enable all other OEAS components to work together. The business objects depend on the core components to complete transaction and service requests. Because OEAS relies so heavily on these core components, Onyx limits the amount to which you can configure them. For example, using the Table Administration tool, you can configure National Language strings (such as error messages and user interface captions), but you cannot customize the component itself.
