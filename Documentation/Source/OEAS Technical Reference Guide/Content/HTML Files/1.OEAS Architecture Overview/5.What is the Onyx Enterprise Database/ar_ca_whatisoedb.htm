What is Onyx Enterprise Database?

The Onyx Enterprise Database is a relational database that contains your critical customer data, including data representing customer records, service requests, support incidents, sales opportunities, products and literature and so on. OEAS enables you to create or customize database objects including tables, columns, views, indexes, and functions. In addition, you can change the database behavior by creating pre, post, or replace procedures for Onyx logical methods.

For detailed information about customizing the Onyx Enterprise Database, see "Customizing the OEDB" in the OEAS Developer's Reference.
