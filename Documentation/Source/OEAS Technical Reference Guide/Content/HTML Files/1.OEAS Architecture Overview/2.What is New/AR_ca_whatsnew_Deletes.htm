Deletes

The new OEAS and OEDB support cascading deletes for all objects. To support cascading deletes, Onyx Enterprise Dictionary includes information for Relationships and Referential integrity. Based on the logical definition of an object, the SQLGen process creates SQL procedures that include the logic for cascading deletes. OEAS and OEDB continue to implement logical deletes for transactional objects and physical deletes for domain objects. For more information on transactional and domain objects, see "Transactional and non-transactional objects" in Customizing OEAS.
