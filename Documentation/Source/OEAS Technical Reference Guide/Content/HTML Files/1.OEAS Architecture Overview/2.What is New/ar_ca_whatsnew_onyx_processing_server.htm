Onyx Processing Server

With this version of OEAS, Onyx Processing Server (OPS) includes the following:

Fulfillment Subsystems

These subsystems include email and print engines that handle OEP literature orders.

Batch Subsystem

This subsystem processes OEP List Manager actions.

Email Monitoring Subsystem (EMS)

This subsystem creates emailMessage records in OEDB, for inbound email messages received by a corporate email server, and associates those emailMessage records with the appropriate Onyx data.

Synchronization Subsystem

This subsystem manages synchronization of data between OEDB and off-line data stores.

For more information, see What is Onyx Processing Server.
