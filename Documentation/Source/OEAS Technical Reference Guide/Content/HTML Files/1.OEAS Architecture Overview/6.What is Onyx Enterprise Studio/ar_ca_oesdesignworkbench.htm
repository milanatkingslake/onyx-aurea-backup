Design Workbench

The following provides brief descriptions of the design tools included in OES.

Design Tool Description

Method Call Designer

The tool for creating method calls, which are data-driven wrappers for (references to) OEAS methods.

Object Designer

The tool for customizing the OED.

Script Designer

The tool for creating scripts.

Subscription Designer

The tool for creating and managing publishable objects, subscribable objects, subscription relationships, subscription roles, and subscription services.

For complete details of the Method Call Designer, the Script Designer, and the Subscription Designer, see "Using Onyx Enterprise Studio." For details regarding the Object Designer, see the OEAS Developer’s Reference.
