What is Onyx Enterprise Studio?

Onyx Enterprise Studio (OES) contains Onyx Enterprise Application Server (OEAS) administration and design tools. OES is divided into the Administration Workbench and the Design Workbench:

  • The Administration Workbench includes the administration tools with which administrators can configure features of OEAS.
  • The Design Workbench includes tools that require a higher level of technical understanding of OEAS. Technical administrators and developers use these tools to customize OEAS.

All OES tools are Web-based and include HTML-based Help systems. No client-side components are included with these tools.
