Administration Workbench

The Administration Workbench includes the following tools.

In most cases, the administration tools have no concurrency control. Where concurrency control for an administration tool does exist, it is optimistic concurrency control, as is used throughout the OEDB.

Administration Tool Description

Assignment Administration

The tool for assigning leads to customers and incidents based on conditions (business rules) that evaluate to true or false.

Campaign Administration

The tool for adding, deleting, and editing tracking code types, tracking codes, and campaigns.

Contact Type Administration

The tool for adding, deleting, and editing internal and external contact types.

Product Administration

The tool for adding, modifying, and deleting standard products, incident products and literature items.

Reference Table Administration

The tool for configuring reference fields and associated lookup data. Depending on the type of field selected, you can modify field properties such as caption or field type, and associate drop-down values with fields.

Security Administration

The tool for adding and maintaining groups, users, roles, and resources, subscription profiles (OEP-Offline), UI profiles (OEP 6.0 UCW), and for managing access permissions to regulate users’ and roles’ access to resources.

Survey Administration

The tool for adding, deleting, and editing surveys, survey types, questions, and responses.

System ParameterAdministration

The tool for configuring parameter values that define aspects of user interface functionality, and in some cases OEAS functionality and process flow.

Table Administration

The tool for administering the domain data of various objects by editing, adding, and deleting data.
