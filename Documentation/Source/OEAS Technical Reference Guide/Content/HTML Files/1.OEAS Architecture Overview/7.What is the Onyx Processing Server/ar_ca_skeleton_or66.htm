Order Line Failure

When a print, fax, email, or label order line fails during processing, the subsystem updates the order line's status to Error and then processes the remaining order lines in the batch.

To reprocess the order line that had the error:

  1. Fix the source of the error.
  2. Using OPS Administration, update the order line's status to New.

For more information on manually updating status, see OPS Administration.
