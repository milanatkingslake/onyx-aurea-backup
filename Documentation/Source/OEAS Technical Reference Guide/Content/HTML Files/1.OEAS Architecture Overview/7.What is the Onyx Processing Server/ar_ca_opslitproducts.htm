Literature Products

A literature product is any type of item that can be delivered as mail, or email, and that you have configured as a literature product using Product Administration. Literature products can include letters, brochures, white papers, photographs, mailing lists, software programs, or informational packets or promotional pamphlets that you send to customers to promote and supplement your standard products.

A literature product typically refers to a template document for print, or email literature products. The templates contain code that merges data into the output literature product. The templates are:

  • For print literature products, an Open XML Word document with Mail Merge.
  • For email literature products, an XSL stylesheet.

Additionally, physical items and system documents are also literature products.

OPS merges literature order and order line data using templates to produce email and printed documents. You can customize and use sample templates that are installed during OPS setup. For information on customizing template documents, see Templates.

OPS also supports sending files as attachments to electronic documents (email) without performing any data merging.

After a template has been created, you create a literature product in Product Administration that references the file path to the template. The following table describes the literature product attributes that OPS uses during processing. You can modify these attributes using Product Administration.

Attribute How subsystem uses attribute

Active literature

Indicates whether the literature product is active or inactive. If the literature product is inactive, the item cannot be ordered from the Literature feature.

File path

The UNC file path to the template used to create the literature product.

Printer

The literature product’s target printer. The printer path is required for creating print documents.

Is a label

Indicates whether the literature product is a mailing label that you can associate with other literature products.

Label product

Which label, if any, is generated each time an order line is fulfilled for the literature product. The label is automatically generated whenever an order line is fulfilled for the literature product.

Fulfillment types

How the literature product can be fulfilled. You can associate one or more fulfillment types with a product. When a literature order is created in Onyx Employee Portal (OEP), the user selects one of the associated types. For more information, see Fulfillment types.
