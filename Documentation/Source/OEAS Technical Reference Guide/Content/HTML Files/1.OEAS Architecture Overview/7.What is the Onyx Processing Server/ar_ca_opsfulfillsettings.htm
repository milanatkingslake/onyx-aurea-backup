Fulfillment Subsystem Settings

The following table describes the fulfillment subsystem settings available in OPS Administration and how they effect the literature order and order line processing.

Setting What the setting specifies

Batch Size

Approximate number of order lines that are processed in a batch. The valid range is 1 - 2,000.

Start Time

Time for the subsystem to begin processing order lines. This is the local time on the OPS server.

Stop Time

Time for the subsystem to stop processing order lines. This is the local time on the OPS server.

Standard Priority Run Interval

Time interval at which the subsystem periodically checks the order line queue for new order lines. The subsystem checks for both standard and high-priority order lines. The valid range is 1 - 10,080 (1 week).

High Priority Run Interval

Time interval at which the subsystem checks for new, high-priority order lines, ignoring standard-priority order lines. The valid range is 1 - 10,080 (1 week).

Ignore Start/Stop Time For High Priority Items

Whether high-priority order lines are processed outside the operation window.

Mail Server

Email server that sends email messages and subsystem error messages.

Administrator Email Address

Email address to which subsystem error notifications and email system documents are sent.

From Email Address

Email address from which subsystem messages are sent.

From Email Display Name

Display name in the subsystem message.
