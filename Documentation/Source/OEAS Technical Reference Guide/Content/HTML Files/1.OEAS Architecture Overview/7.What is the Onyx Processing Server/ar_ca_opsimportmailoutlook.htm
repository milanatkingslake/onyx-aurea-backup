Saving Email in to OEP Through Onyx Email for Outlook

Onyx Email for Outlook (Email for Outlook) is a plug-in for Microsoft Outlook that enables employees to save emails to the database from personal Outlook inboxes, and track new emails sent from their accounts in OEP.

For instructions on installing, configuring, and distributing Email for Outlook, see the ONYX Installation Guide.
