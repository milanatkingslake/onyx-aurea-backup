Batch Subsystem Settings

When you add a subsystem to an OPS connection, you configure the subsystem settings that determine how the actions will be processed. You can have multiple batch subsystems, with each subsystem configured independently. The following table describes the subsystem settings and how they affect the action processing.

Setting Effect

Start Time

Time that the subsystem’s operation window begins.

Stop Time

Time that the subsystem’s operation window stops.

Run Interval

Time interval (in minutes) between subsystem checks of the batch queue for new action batches. Because the batch subsystem provides continuous processing, this run interval applies only when the batch queue is empty. The valid range is 1 - 10,080 (1 week).

Governor Interval

Number of milliseconds between calls from the subsystem to OEAS. The valid range is 0 - 60,000 (1 minute).

Mail Server

Email server that sends subsystem error notifications.

Administrator Email Address

Email address to which subsystem error notifications will be sent.

From Email Address

Email address from which subsystem error notifications will be sent.

From Email Display Name

Display name in the subsystem error notification.
