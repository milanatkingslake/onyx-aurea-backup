Literature Orders

The Literature Orders feature provides a way for your employees to order literature products for mailing or sending with email messages to your customers. When a user creates a literature order, the user selects:

  • One or more literature products
  • The fulfillment type for each order line

The fulfillment types available consist of the types associated with the product in Product Administration. Each line item, or order line, consists of a product, fulfillment type, and quantity. To understand how a literature order line is processed, see Literature order line process flow.

The following table lists a subset of literature order object properties that OPS uses. The table uses friendly names in place of the literal property names. For more information, Business Objects Library.

Attribute How subsystem uses attribute

Message subject

The subject for an email order.

Email From, From Display Name, To, Cc, Bcc, Reply To

The email message origination and destination information.

OPS system attribute

The type of order. The attribute is set to 0 for regular orders, 1 for system documents, and 2 for labels. This attribute is reserved by OPS.

The following table lists a subset of literature order line object properties and how the fulfillment subsystem uses them. The table uses friendly names in place of the literal property names.  For more information, see Business Objects Library.

Attribute How subsystem uses attribute

Product

Which literature product is ordered. The literature product contains the file path to the template as well as printer information.

Fulfillment type

How the order line is to be fulfilled. The list contains all fulfillment types associated with the product in Product Administration.

Line number

For grouping order lines for email literature orders. Also used for order lines that appear in system documents.

Priority

Whether the order should be processed during a high-priority or standard-priority interval.

Insert date

The date the order line was created.

Order line status

Current status of the order line. For more information, see Literature order line status.

Ship date

The date the order line status is updated to Completed.

Quantity ordered

The number of literature products that were ordered. This value is always 1 for email literature products.

Quantity shipped

The number of literature products that were shipped.

Printer name

The printer to which the print literature products are to be routed.

OPS system attribute

The type of order. The attribute is set to 0 for regular orders, 1 for system documents, and 2 for labels. This attribute is reserved by OPS.

Scheduled process date

Date of processing. If this attribute is blank, then the order line is processed as soon as possible. If this attribute contains a date, then the order line is not processed before the date indicated by the attribute’s value.
