Default OEAS Security Resources Available for OPS

In the default implementation of OEAS, the following OPS resources are available for enabling users to manage OPS.

Security resource Description

OPS.ServiceManagement

Users with access permission for this resource can start, stop, pause, and continue the OPS service.

OPS.SubsystemManagement

Users with access permission for this resource can start, stop, pause, and continue subsystems.

OPS.SubsystemConfiguration

Users with access permission for this resource can add, delete, and configure subsystems.

OPS.OrderQueueManagement

Users with access permission for this resource can manage order lines. For example, they can re-route an order line to another printer or change the status from Error to New.

OPS.OrderQueueView

Users with access permission for this resource can view the fulfillment queues within OPS Administration that contain order lines. This does not include, however, making any changes to order lines.

OPS.BatchQueueView

Users with access permission for this resource can view the batch queues within OPS Administration that contain action batches.
