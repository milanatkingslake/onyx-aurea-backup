Action Batch Failure

As a batch subsystem processes an action batch, one or more calls may be made to OEAS. If all calls complete successfully, the action batch is marked as Completed. However, if even one call fails, the action batch fails and has its status updated to Error. To determine the cause of the error, you can view the action batch XML in OPS Administration. For more information on viewing action batch XML, see the OPS Administration Help.
