Action Batch Status

The action batch status is used to track the status of an action batch and is displayed on the New Batches and Completed Batches pages in OPS Administration. Default statuses are described in the following table.

actionBatchStatusCode name description

0

New

(See Warning following this table)

This is a new action batch; its processing has not begun.

1

In Process

(See Warning following this table)

The action batch is being processed.

2

Completed

(See Warning following this table)

The action batch’s processing is complete.

3

Error

(See Warning following this table)

The action batch failed during processing.

Do not delete the following domain data for action batch status: New, In Process, Completed, or Error. If OPS tries to set the status of an action batch and the domain data doesn't exist, OPS will fail. You can change the value of the name or description properties on the actionBatchStatus object, for example to localize, but Onyx does not recommend changing their meaning, as it might confuse end users.
