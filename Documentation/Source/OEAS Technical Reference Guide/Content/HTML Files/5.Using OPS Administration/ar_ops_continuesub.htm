Restarting a subsystem

When you restart a subsystem, it resumes processing as scheduled. An EMS is unique in that it resumes processing the email messages in the monitored mailbox whose processing was interrupted by the pausing or stopping of the subsystem.

To restart a subsystem:

  1. In the navigation area, select the subsystem.
  2. Click to restart the subsystems.

The subsystem’s status is updated to Started in the navigation area.
