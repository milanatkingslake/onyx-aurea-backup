Sorting batch queues

The following procedure describes how to sort the New Batches queue and Completed Batches.

To sort queues:

  1. In the navigation area, expand the connection node.
  2. Expand the Batch Queues node.
  3. Select New Batches or Completed Batches.

The queue appears in the edit area.

  1. Click the column heading of the column you want to sort by.

OR

To reverse the sort order, click the column heading.
