Pausing or stopping a subsystem

A subsystem completes any processing (the processing of an email or the synchronization of Onyx data) during which you pause or stop the subsystem. After completing the process, it does not initiate a new one until you restart the subsystem.

To pause a subsystem:

  1. In the navigation area, select the subsystem.
  2. Click .

The subsystem’s status is updated to paused in the navigation area.

To stop a subsystem:

  1. In the navigation area, select the subsystem.
  2. Click .

The subsystem’s status is updated to Stopped in the navigation area.
