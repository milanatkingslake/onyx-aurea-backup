Mailing labels

You can configure mailing labels to be generated with Send as Printed Document or Send as Physical Item fulfillment types. You can use customized versions of the sample templates or create new templates to generate the labels. After you have created the label literature product, you can associate it with other literature products.

Onyx recommends dedicating a printer to printing labels.

To create and enable a mailing label:

  1. Create a mailing label template from the sample label template.
  2. Use this template to generate the mailing labels.

For details on creating and customizing OPS templates, see Creating templates.

  1. In Product Administration, create a literature product that references the mailing label template, making the following changes in the edit area:
  • Active Literature: Check this box to indicate that the literature product is active.
  • File Path: Enter the file path to the location where the Open XML Word with Mail Merge template is located.
  • Printer: Select the printer that is to print the labels.
  • Is a Label: Check this box to indicate that this is a mailing label.
  • Label Product: Leave this box empty.
  • Fulfillment Types: Select Send as Printed Document.
  1. To save your changes, click .
  2. In Product Administration, associate a literature product with the new mailing label literature product.

For information on associating a literature product with a mailing label literature product, see the following procedure.

To associate a literature product with a mailing label literature product:

  1. In the navigation area, select the literature product that you want to associate with the mailing label product.
