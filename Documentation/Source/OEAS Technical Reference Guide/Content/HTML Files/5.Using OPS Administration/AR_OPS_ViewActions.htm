Viewing the New Batches queue

All action batches appear in the New Batches queue, except those with a status of completed.

The total number of lines shown depends on the Maximum Rows for New Batches setting in Queue Preferences. For more information on queue settings, see Configuring queue preferences.

To view action batches:

  1. In the navigation area, expand the connection node.
  2. Expand the Batch Queues node.
  3. Select New Batches.

The queue appears in the edit area. All action batches are shown except those with status of Completed.
