Deleting a synchronization subsystem

Before you delete a subsystem, you must update all subscription profiles that specify the subsystem with a different subsystem to use. If any profile uses the subsystem that you attempt to delete, the deletion fails.

To delete a synchronization subsystem:

  1. In the navigation area of OPS Administration, select the mailbox that you want to delete.
  2. To mark a sync subsystem for deletion, click .

In the navigation area, the name of the mailbox is struck through.

  1. Click to carry out the deletion and save changes.
