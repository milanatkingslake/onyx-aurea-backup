Configuring fulfillment subsystems

Working with fulfillment subsystems

After creating OPS connections, you are ready to work with fulfillment subsystems. You can have multiple subsystems for each connection. The fulfillment subsystems, which process literature order lines, are configured and controlled independently of each other. Use OPS Administration to start, pause, continue, or stop a subsystem. For more information about fulfillment subsystems, see Fulfillment subsystems overview.
