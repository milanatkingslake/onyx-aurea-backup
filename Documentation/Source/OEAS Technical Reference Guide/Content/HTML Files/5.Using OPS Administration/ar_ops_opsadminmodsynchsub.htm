Modifying a synchronization subsystem

There are two ways to modify a sync subsystem:

  • By specifying the attributes of the subsystem (in the edit area of OPS Administration, as when you created the subsystem)
  • By configuring event logging for OPS and the sync subsystem

To specify attributes for a sync subsystem:

  1. Open OPS Administration, and in the navigation area, select the sync subsystem whose attributes you want to change.
  2. In the edit area, make desired changes

For information on the attributes that you can change, see the preceding section, Adding and configuring a synchronization subsystem.

  1. To save changes, click on the toolbar.
