Viewing Completed Order Lines

In the edit area, under Completed Order Lines, you can see all order lines with a status of Completed.

The total number of lines shown depends on Maximum Rows for Completed Order Lines setting in Queue Preferences. For more information on queue settings, see Configuring queue preferences.

To view completed order lines:

  1. In the navigation area, expand the connection node.
  2. Expand the Fulfillment Queues node.
  3. Select Completed Order Lines.

The queue appears in the edit area.
