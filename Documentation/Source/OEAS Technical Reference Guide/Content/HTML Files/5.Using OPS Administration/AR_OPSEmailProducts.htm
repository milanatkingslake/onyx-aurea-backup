Email literature products

Email literature products are literature products that have a fulfillment type of Send as Email Body or Send as Email Attachment. You configure these to be ordered from the Literature feature and emailed to a customer. You can create email literature products that merge customer specific data into the email, or you can create email literature products that you send as a file attachment to the email.

To create an email literature product personalized with merged customer data:

  1. Create an XSL template to perform data merging. Create the template by customizing one of the OPS sample XSL templates or by creating a new template. For more information on creating and customizing templates, see Customizing OPS.
  2. In Product Administration, create a literature product that references the template, making the following changes in the edit area:
  • Active Literature: Check this box to indicate that the literature product is active.
  • File Path: Enter the file path to the location where the XSL template is located.
  • Printer: Leave this box empty.
  • Is a Label: Leave this box empty.
  • Label Product: Leave this box empty.
  • Fulfillment Types: Select one or more of the following fulfillment types:
  • Send as Email Body (supports merging)
  • Send as Email Attachment (supports merging)

You can also create an email literature product that does not require data merging and send it as an email attachment.

To create an email literature product to send as a file attachment:

  1. Create the file attachment (pdf, gif, and so on).
  2. In Product Administration, create a literature product that references the file attachment, making the following changes in the edit area:
  • Active Literature: Check this box to indicate that the literature product is active.
  • File Path: Enter the file path to the location where the file is located.
  • Printer: Leave this box empty.
  • Is a Label: Leave this box empty.
  • Label Product: Leave this box empty.
  • Fulfillment Types: Select the following fulfillment type:
  • Send as Email Attachment (no merging)

For details on adding literature products, see Product Administration Help.
