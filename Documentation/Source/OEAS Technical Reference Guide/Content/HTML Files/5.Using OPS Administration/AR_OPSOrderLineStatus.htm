Configure order-line status

Step 7: Configure order-line status

You use order-line status to track the status of literature orders. The status is displayed in the New Order Lines queue and Completed Order Lines in OPS Administration. Configuring order-line status is an optional step.

You can administer the data using Table Administration. With this tool, you can:

  • Create a new order line status
  • Update an existing status
  • Set a status to active or inactive
  • Delete an existing status

The following table describes the default order-line status data installed with the OEDB.

literature order line status code Status Description

0

New

See the Warning following this table.

Processing has not begun yet.

1

In Process

See the Warning following this table.

The order line is being processed.

2

Completed

See the Warning following this table.

Processing is complete.

3

Error

See the Warning following this table.

The processing failed.

4

Data Error

There was an error in the order-line data.

5

Inventory Hold

Hold the order line. Do not process it.

6

Back Order

The order line is back-ordered. Do not process it.

7

Customer Hold

Hold the order line for the customer. Do not process it.

*Do not modify the following data: New, In Process, Completed, or Error. Any changes to this data will result in OPS failure.

You can change the names or descriptions of some statuses, or create new status data, using Table Administration (select Literature Order Line Status in the Table drop-down box. See Table Administration Help for more information.
