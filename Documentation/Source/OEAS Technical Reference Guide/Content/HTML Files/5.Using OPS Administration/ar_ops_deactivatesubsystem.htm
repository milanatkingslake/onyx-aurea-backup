Activating or deactivating an email monitor subsystem or mailbox

The following procedure describes how to activate or deactivate a subsystem or mailbox.

To activate a subsystem or mailbox:

  1. In the navigation area, select the subsystem or mailbox that you want to activate.
  2. In the edit area, check the Activate box.
  3. Click to save your changes.
  4. Click to restart the subsystem if the subsystem was stopped.

To deactivate a subsystem or mailbox:

  1. In the navigation area, select the subsystem or mailbox that you want to deactivate.
  2. Click to stop the subsystem.
  3. In the edit area, clear the Activate box.
  4. Click to save your changes.
