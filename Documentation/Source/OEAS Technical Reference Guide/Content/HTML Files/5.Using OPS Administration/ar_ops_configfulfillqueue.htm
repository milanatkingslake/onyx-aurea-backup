Configuring queue preferences

To configure the queue settings:

  1. Click on the toolbar.
  2. Enter the following information:
  • Maximum Rows for New Order Lines — The number of order lines to show in the New Order Lines queue.
  • Maximum Rows for Completed Order Lines — The number order lines to show in the Completed Order Lines queue.
  • Maximum Rows for New Batches — The number of rows to show in the New Batches queue.
  • Maximum Rows for Completed Batches — The number of rows you want to show in the Completed Batches queue.

In each of the preceding boxes, enter “0” (zero) to show all rows.

  1. Click OK.
