Step 12: Modify system document product group (optional)

By default, the System Document list in OPS Administration returns all active literature products from the System Documents product group in Product Administration. However, you can modify which product group OPS Administration displays as the system document group by configuring a system parameter.

To configure a different system document group:

  1. In Product Administration, create a new product group.

OR

Select an existing product group that you want to use as the system document product group.

For information on using Product Administration and creating product groups, see Product Administration Help.

  1. With the product group selected in the navigation area, in the edit area, locate and record the Group ID number.
  2. In System Parameter Administration, select OPSSysDocProductGroupId.

See System Parameter Administration Help for more information on using the tool.

  1. In the Parameter Value field, type the Group ID you just recorded from Product Administration.
  2. To save your changes, click .

For this change to take effect, you must restart OPS Administration.
