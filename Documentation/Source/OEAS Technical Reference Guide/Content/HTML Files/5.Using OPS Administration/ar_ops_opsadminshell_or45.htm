Viewing action batch XML

View the action batch XML can be useful for developers and for debugging purposes.

To view action batch XML:

  1. In the navigation area, expand the connection node.
  2. Expand the Batch Queues node.
  3. Select New Batches or Completed Batches.

The queue appears in the edit area.

  1. To view the batch XML, do the one of the following:
  • To view the action batch XML, click the action batch Summary.
  • To view the action batch source list status XML, click the action batch Status.
