Onyx Logical Business Object Tester system requirements

Prerequisites for running the Onyx Logical Business Object Tester are:

  • Microsoft Visual Basic 6 runtime files (these files should already be on your machine, if not, they come with OEAS, or any number of other applications)
  • MSXML3 (in either side-by-side or replace mode)
  • Either a local OEAS application server or a COM+ application proxy pointing to any remote OEAS application server

To install a COM+ application proxy:

  1. Using Windows Explorer, browse to the machine that is the OEAS application server (for example \\myappserver), and locate the OnyxEAS proxy file (usually named OnyxEASApp.msi).
  2. Double-click the OnyxEASApp.msi file. Windows Installer installs the COM+ proxy to your machine automatically.
