Using the Onyx Logical Business Object Tester

The following procedure describes how to use the LBO Tester.

To use the Onyx Logical Business Object Tester (LBO Tester) utility:

  1. In Windows Explorer, navigate to the directory where you copied the LBOTester files and double-click LBOTester.exe.
  2. In the Authentication Information group box enter Server, Application, and Site ID. Check the Use Windows Auth. check box to use Windows Authentication. Otherwise enter your Onyx user account User ID and Password.
  3. Press the Establish Session button to connect to the OEAS application server. If you are trying to establish a session with a remote OEAS application server and you get an authentication error, check to make sure you installed the COM+application proxy that points to your remote OEAS application server.
  4. Select the object for the method you want to execute from the Object drop-down box.
  5. Select the method you want to execute from the Method drop-down box.
  6. Fill out the appropriate parameters in the Parameters XML.
  7. Press the Execute button to execute the method.
  8. The return XML for the method is returned in the returnXml box (or the Onyx Logical Business Object Tester viewer window if the Use Viewer check box is checked).
