Onyx Logical Business Object Tester window reference

This table provides a reference for the LBO Tester window.

Box/Button Shortcut keys Function Context

Object

ALT+O

Select object from drop-down list of the current logical business objects available in the Onyx Enterprise Dictionary (OED).

Enabled when connected to an OEAS application server.

Method

ALT+M

Select method to execute from drop-down list of methods available in the OED for the selected object.

Enabled when connected to an OEAS application server.

MRU button

F10

Click button for a pop-up menu listing the Most Recently Used (MRU) methods. This pop-up menu includes the following options:

  • Add current method - adds whatever method is currently selected to the MRU list
  • Clear - clears all methods from the MRU list
  • Add on execute - adds executed methods to the MRU list
  • Add on modify parameters -  when a method's input parameters are modified, adds the method to the MRU list

Enabled when connected to an OEAS application server.

Status bar

 

There are three areas at the bottom of the tester window that show status information:

  • The left area shows textual messages about what the tool is doing, errors returned, etc.
  • The middle area shows a progress bar that indicates what percent of the current execution is finished. This is useful mainly when executing the call more than once so one can see generally what percentage is finished.
  • The right area shows the server/app/site you are connected to, or "(not connected)". This is the same string that appears at the beginning of the title bar.

Display only.

Parameters

ALT+P

Method input XML template generated from the OED.

Enabled when connected to an OEAS application server.

Show customData check box

ALT+U

Displays customData area if checked. Otherwise, customData area is hidden.

Always enabled.

Reset button

ALT+R

Resets the parameters XML to the proper structure based on the logical object model information in the OED.

Enabled when connected to an OEAS application server.

Reformat

ALT+F

Reformats the parameters XML so it is easy to read, without changing the content.

Enabled when connected to an OEAS application server.

Font Size

 

Controls font-size of the text in the Parameters box.

Always enabled.

Execution Count

ALT+X

or

Enter

Specifies the number of times to invoke the selected method when the Execute button is pressed.

Always enabled.

Execution delay

ALT+D

Specifies a delay, in milliseconds, between successive calls to a method.  This is the actual delay time, not the time from start-to-start of successive calls.  If each call takes 500ms and you put a delay of 1000ms, there will be a call occurring every 1500ms.

Enabled only when the execution count is greater than 1.

 

Stop on Failure

ALT+S

E nables you to control behavior when the call is being repeated more than once.  If this is checked, execution will stop whenever a methodStatus of "failure" or a bad HR is returned from OTM.

Enabled only when the execution count is greater than 1.

 

Total Time (ms)

 

Displays the total amount of time in milliseconds taken for the most recent execution.

Display only.

Method Time (ms)

 

Displays the average time taken for one method invocation in the most recent execution.

Display only.

Execute button

ALT+E

Execute selected method.

Enabled when connected to an OEAS application server.

Close button

 

Close LBO Tester application.

Always enabled.

Show Auth Info check box

F8

Displays Authentication Information group box if checked. Otherwise, authentication information is hidden.

Always enabled.

Server

 

Specifies the name of the OEAS application server.

This field can be left blank. When blank, it behaves in the following ways:

  • If a local OEAS is installed, executes locally.
  • If an OEAS proxy is installed, executes on the machine to which the proxy points.

Enabled when not connected to an OEAS application server.

Application

 

Specifies the application name on the OEAS application server.

Enabled when not connected to an OEAS application server.

Site ID

 

Specifies the site ID used for authentication on the OEAS application server.

Enabled when not connected to an OEAS application server.

Use Windows Auth.

 

Check the box to use Windows Authentication. Uses Onyx authentication if not checked.

Enabled when not connected to an OEAS application server.

Autoconnect

 

Check the box to automatically connect to the OEAS application server when LBO Tester starts-up.

Enabled when Use Windows Auth. is checked.

User ID

 

Onyx User ID.

Enabled when not connected to an OEAS application server and Use Windows Auth. is not checked.

Password

 

Onyx user account password.

Enabled when not connected to an OEAS application server and Use Windows Auth. is not checked.

Establish Session button

 

Establish a session with specified authentication information.

Enabled when not connected to an OEAS application server.

End Session button

 

End the current session.

Enabled when connected to an OEAS application server.

returnXml

 

Return XML from executed method.

Display only.

Copy to Clipboard

ALT+C

Copies the return XML from most recently executed method to the Windows clipboard.

Enabled when return XML is displayed in the returnXml area.

Use Viewer

ALT+V

Check the box to use the Onyx Logical Object Tester returnXml Viewer. The return XML pops-up in its own window for enhanced viewing.

Always enabled.
