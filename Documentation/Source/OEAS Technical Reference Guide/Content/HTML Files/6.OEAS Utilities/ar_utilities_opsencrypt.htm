OPSEncrypt utility

Onyx provides the OPSEncrypt utility for encrypting the user password in SMTP Basic authentication. When using OEAS configuration utility, you can set the authentication protocol to Basic. The OPSEncrypt utility enables you to encrypt the password for the OEAS configuration utility setting. You can find OPSEncrypt in the Utilities folder on the OEAS CD ([OEAS CD]\Utilities\OPS\OPSEncrypt).

To run the utility:

  1. On the OEAS CD, copy the files from the OPSEncrypt  directory to a local directory on the OEAS server.
  2. From the local directory, start OPSEncrypt.exe and follow the instructions in the console application.
  3. Enter the password you want to encrypt.
  4. Hit Enter.
  5. Copy and paste the password into the password text box of the OEAS configuration utility.
  6. Hit an key to exit the application.
