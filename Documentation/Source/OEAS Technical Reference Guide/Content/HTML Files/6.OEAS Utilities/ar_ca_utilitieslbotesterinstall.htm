Installing Onyx Logical Business Object Tester

Use the following procedure to install LBO Tester.

To install the Onyx Logical Business Object Tester (LBO Tester) utility:

  1. On the OEAS CD, open the Utilities folder.
  2. Copy the LBOTester folder to a suitable location on your development machine.
  3. Make sure your development machine meets the system requirements.
  4. In Windows Explorer, navigate to the directory where you copied the LBOTester files and double-click LBOTester.exe.
