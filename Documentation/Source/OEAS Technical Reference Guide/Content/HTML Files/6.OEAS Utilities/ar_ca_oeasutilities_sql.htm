OEAS utilities

The OEAS CD includes a Utilities folder which contains utilities for your use. The following table lists the subfolders of the Utilities folder and where to find more information about each.

Subfolder Description More information

Attachments

Contains tools for creating the file attachments directory, migrating file attachments between OEDB versions, and cleaning up deleted attachments.

See OEAS Installation Guide and corresponding readme files with the respective utilities.

Data Migration Tool

Contains the resources for performing the data migration task. The DBA should not attempt data migration before completely reviewing documentation for the OEDB upgrade task.

See OEAS Upgrade documentation and the DataMigrationToolkitGuide.pdf. The PDF can be found in the Data Migration Tool folder.

Database Setup Script

Contains a script that creates SQL login prerequisites for OEAS installation.

See OEAS Installation Guide.

Onyx Logical Business Object Tester

Contains a graphical user interface application for testing logical methods.

See LBO Tester documentation.

MatchCodeCreation utility

Contains a script to create match code values, where none existed before, for all customers and addresses, or if you want to re-create match codes based on a new match code formula.

See MatchCodeCreation utility documentation.

OEAS Configuration utility

Contains a graphical user interface for maintaining certain OEAS settings.

See OEAS Configuration utility documentation.

OEAS Publish

Contains a script for customers who want to customize the OED publish process for their enterprise.

See OED publish process.

OPS

Contains applications for testing OPS templates and encrypting SMTP user password.

See "Creating templates" and OPSEncrypt utility.

Script Transfer

Contains an application for transferring a copy of a script or method call from one application server to another.

See the readme file included with the utility.

Subscription Integrity Scripts

Contains a script for cleaning up the subscription_item table.

See Removing deleted data from subscription_item.
