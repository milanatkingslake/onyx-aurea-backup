Onyx Logical Business Object Tester

The Onyx Logical Business Object Tester (LBO Tester) utility provides an easy way to execute logical methods defined in OEAS without writing code. To use LBO tester you choose an object and the logical method to execute. LBO Tester then constructs a template for the parameters XML needed to call that method. After values are added for required properties and optionally added for non-required properties, you can execute the method and get back the return XML. LBO Tester remembers parameter XML element values entered for each logical method so they do not need to be reentered.
