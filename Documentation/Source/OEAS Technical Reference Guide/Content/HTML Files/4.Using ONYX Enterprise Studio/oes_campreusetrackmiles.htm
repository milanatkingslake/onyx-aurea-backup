Reusing tracking code milestones from a previous campaign

You can reuse tracking code milestones from previous campaigns.

To reuse tracking code milestones from a previous campaign:

  1. Expand the Campaigns node, and select the Tracking Code Milestones node from within a tracking code. The edit area appears, enabling you to select a milestone category and its associated milestone actions from a previous campaign.
  2. In the edit area, define the attributes.
  • Select a Campaign - Select a campaign from the drop-down list. The Campaign list contains all the defined campaigns.
  • Select a Tracking Code - Select a tracking code from the drop-down list to associate with the tracking code milestone. The Tracking Code list contains all the tracking codes defined for the selected campaign.
  • Milestone Categories - Select a milestone category to add it to the current tracking code. The Milestone Categories list contains all the milestone categories used for the selected tracking code.
  • Milestone Actions - The Milestone Actions list provides a list of valid milestone actions (this list in noneditable).
  • Add Milestone Category - Click the Add Milestone Category button to add the tracking code milestone to the current tracking code. This button is enabled only when you have an existing campaign, tracking code, and milestone category selected.
  1. All of the tracking code milestones associated with the selected milestone category are added to the tracking code milestone list for the tracking code.
  2. Click to save your changes to the OEDB.
