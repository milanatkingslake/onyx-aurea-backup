Modifying existing lookup values

Use the following procedure to modify lookup values system wide.

To modify a lookup value system wide:

  1. In the navigation area, select the Lookup Value - All mode option to display the existing lookup values.
  2. Select the lookup value you want to modify. The edit area displays the lookup value attributes.
  3. Enter the new text description for the existing lookup value. You can only modify the Extra Data and Char Code attributes from within the Lookup Value by Field mode. For descriptions of the lookup value attributes, see Defining lookup value attributes.
  4. Click on the toolbar to save your changes.

Any modifications to existing lookup values affects the value system wide. If the lookup value is associated with any other records, then the value on the those records is also updated.
