Toolbar

The toolbar appears under the context area and contains the toolbar buttons that control the tasks related to Assignment Administration.

Icon Tool tip Function Enabled when you...

Add

Adds a new territory, condition, expression, assignment set or user. The item added depends on what is selected in the navigation area.  

Select an item in the navigation area.

Delete

Marks for deletion an item that is selected in the navigation area. The item is not actually deleted until you save changes. When an item is marked for deletion, clicking this button removes the mark.

Select an item in the navigation area.

Onyx recommends that you deactivate items instead of deleting them.

Copy

Copies a selected condition or assignment set to the clipboard.

Select a condition or assignment set in the navigation area.

Paste

Pastes the copied condition or assignment set into a different condition or assignment set node.

Select the condition or assignment set group node that originally contained the copied item.

Move up

Moves a territory, condition, expression, assignment set or user up one position in the navigation area to change the order in which they are processed.

Select an item in the navigation area.

Move down

Moves a territory, condition, expression, assignment set or user down one position in the navigation area to change the order in which they are processed.

Select an item in the navigation area that is not the last item in the list.

Save

Saves changes.

Make a change.

You cannot save a child territory until the parent territory is saved.

Help

Launches Help.

N/A. Always enabled.
