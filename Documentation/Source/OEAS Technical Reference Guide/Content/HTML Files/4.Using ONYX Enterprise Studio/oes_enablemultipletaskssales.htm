EnableMultipleTasksSales

Determines whether the Task tab appears on the Sales Incident window.

  • Allowed Parameter Values

N - The Task tab is not available on the Sales Incident window.

Y - The Task tab is available on the Sales Incident window, and users can enter multiple tasks for Sales Incidents.

  • Default Parameter Value - EnableMultipleTasksSales is installed as enabled (Y) by default.
