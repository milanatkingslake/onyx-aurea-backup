Creating a new campaign

To create a new campaign:

  1. In the navigation area, select the Campaigns node or select an existing campaign and click .

A new campaign titled newCampaign<n> appears at the bottom of the campaigns list in the navigation area, followed by an asterisk (*).

  1. In the edit area, define the attributes for this campaign.
  • Code - Type a unique code to assign to this campaign.
  • Name - Type the name you want to assign to this campaign. You can edit this name after a save.
  • Description - Type a description of this campaign. You can edit this description after a save.
  • Active - Uncheck this box if you want to deactivate this campaign. A checked box indicates an active campaign. The default is active (checked).
  • Click to save your changes to the OEDB.
