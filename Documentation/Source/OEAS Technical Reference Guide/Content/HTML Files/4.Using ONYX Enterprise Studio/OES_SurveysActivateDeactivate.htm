Making items available or unavailable

To make an item available to your users, you must make it active. To make it unavailable, you either delete it or make it inactive.

You cannot delete an item that is used by a survey without removing it from any survey components that are using it, because doing so would violate referential integrity in the database. Therefore, if you try to delete an item that is used by a survey, you cannot save the change.

Question responses are a special case: to delete a question response, you use Delete in the edit area of the question that the question response is associated with. Because the question response is associated only with that question, no other survey component is using the question response when you delete it. If the question response is included in the survey’s question hierarchy, however, the deletion will fail.
