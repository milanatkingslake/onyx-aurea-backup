Working with responses

When you create a response, no matter where you create it (within the edit area for a question or under All Responses in the navigation area), the response appears under All Responses. For more information on creating responses, see Creating responses.

Any changes that you make to a response after creating it are reflected both in All Responses and in any question to which you have added it.

You can also do the following with responses:

  • Make them unavailable.

For more information on making responses unavailable, see Making items available or unavailable.

  • Move them in the order in which they appear as possible answers to a question.

For more information on moving question responses, see Moving responses.

  • Make them editable by the respondent.

For more information on creating editable responses, see Creating editable responses.

  • Link them to the next question in a question hierarchy

For more information on linking question responses with subsequent questions, see Working with question hierarchies.
