Quick overview

The following graphic gives a quick overview of the functional layout common to most tools in the Administration Workbench. The graphic shows the Campaign Administration tool, but the other tools share the same basic functional design.

The administration page is divided into four areas:

  • Context area
  • Toolbar
  • Navigation area
  • Edit area

The content of each area varies depending on the tool. For information specific to each tool, see the quick overview topic for the tool.
