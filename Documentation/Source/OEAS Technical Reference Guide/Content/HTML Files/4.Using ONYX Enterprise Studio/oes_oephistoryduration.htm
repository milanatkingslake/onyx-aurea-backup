OEPHistoryDuration

OEPHistoryDuration specifies the number of days in the past, from the current date, for which OEP displays the history list. In the default installation, this means that OEP lists the history for items updated in the previous 90 days.

  • Allowed Parameter Values - The parameter value stores an integer value. The recommended minimum and maximum parameter values are 0 and 999, respectively (when a parameter value of 0 is set, OEP lists only the history for items updated on the current date).
  • Default Parameter Value - OEPHistoryDuration is installed with a default value of 90.
