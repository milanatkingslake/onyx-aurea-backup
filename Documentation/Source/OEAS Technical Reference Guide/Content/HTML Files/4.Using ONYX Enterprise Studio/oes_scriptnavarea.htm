Navigation area

In Script Designer, the navigation area is labeled Script Information.

You can create and modify scripts, their groups, and their elements by selecting script groups, scripts, or script steps in the navigation area and then modifying them.

Until you create a script group, the navigation area is empty. When you have created a script group, the navigation area contains:

  • A first-level node, which is the individual script group.
  • The script group node contains two second-level nodes, Groups and Scripts.
  • The second-level nodes contain, respectively, any child groups and scripts that you create within the first-level group. (Creating a child group adds another level for scripts and another child group level.)
  • Each script that you create contains a Steps node, which contains the steps of the script.
