Working with survey types

Creating survey types

Survey types are groupings of surveys. In the navigation area, every survey is listed under a survey type.

To create a survey type:

  1. In the navigation area, select the Survey Types node, and then click from the toolbar.
  2. In the Description box, type a description or name for the new survey type, and if you want the survey type to be active, leave the Active box selected.
  3. Click to save your changes.

When you save your changes, the survey type’s new ID appears beside ID.
