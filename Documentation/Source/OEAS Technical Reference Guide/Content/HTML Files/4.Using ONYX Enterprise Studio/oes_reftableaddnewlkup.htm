Adding a new lookup value

Use the following procedure to add a new lookup value within the Lookup Value — All mode.

To add a new lookup value:

  1. In the navigation area, select the Lookup Value — All mode option to display the existing lookup values.
  2. Click and a new lookup value named NewLookupValue<n> appears at the bottom of the lookup values list in the navigation area, followed by an asterisk (*). You can also select an existing lookup value and click , with the same result.
  3. In the edit area, define the attribute for this lookup value.

In the Lookup Value — All mode, the only attribute to define is the Lookup Value attribute. You can only define the Extra Data, Char Code, and Active attributes from within the Lookup Value by Field mode. For descriptions of the lookup value attributes, see Defining lookup value attributes.

  1. Click to save your changes.

When you are creating multiple lookup values, be sure to click before you begin defining a new lookup value. This refreshes the edit area so that you do not overwrite an existing lookup value. Any modifications to existing lookup values affects the value system wide. If the lookup value is associated with any other records, then the value on those records is also updated.
