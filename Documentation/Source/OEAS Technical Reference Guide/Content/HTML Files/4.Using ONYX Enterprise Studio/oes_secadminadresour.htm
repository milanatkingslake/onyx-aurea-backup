Administering UI or Other resource groups

You can add a new resource for a custom business or application functionality to Other Resources or UI Resources.

  • Add a new resource
  • Activate or deactivate a resource
  • Delete a resource

Note: You cannot add new resources for Business Objects; you secure the business object method as a resource and then manage its permissions. See Administering Business Object resources for further details.
