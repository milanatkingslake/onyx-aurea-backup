Adding a new resource

Use the following procedure to add a new resource to Other Resources or UI Resources.

To add a resource:

  1. There are two ways to add a resource in the navigation area:
  • Select either the UI Resource node or Other Resource node and click on the toolbar.

OR

  • Open the UI Resource node or Other Resource node, select a resource from the list, and click on the toolbar.
  1. In the edit area, type the resource's ID in the Resource ID text box.
  2. In the Description text box, type a description of the resource.
  3. The Active box is checked by default, indicating an active resource.

To learn more about the Manage Permissions option, see Managing permissions for resources.

  1. Click on the toolbar to save your changes.
