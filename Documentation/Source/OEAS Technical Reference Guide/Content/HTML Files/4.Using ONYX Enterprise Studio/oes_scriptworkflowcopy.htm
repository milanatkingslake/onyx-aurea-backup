Cloning a script

You can save yourself the work of creating a new script in its entirety by cloning an existing script and then making changes to the clone as needed.

The clone of the original script is placed in the same script group as the original. After cloning the script, you can change the script’s parent group.

To clone a script:

  1. In the navigation area, select the script that you want to clone, and then click .
  2. Make any necessary changes.

To change which script group the script is part of, select a different parent group in the Edit Script area.

  1. Check usage of method calls to make sure that they’re still appropriate and that they still work correctly.
  2. On the toolbar, click to save your changes.
