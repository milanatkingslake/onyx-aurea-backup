Adding a new contact type

To add a new contact type:

  1. In the navigation area, select the contact category to which you want to add a new contact type, and click on the toolbar.

A new contact type titled newContactType<n> appears in the navigation area followed by an asterisk (*).

  1. In the edit area, define the attributes for this contact type.
  • Contact Type ID - This is a non-editable value automatically generated when a new contact type is saved.
  • Contact Type - Type the name you want to assign to this contact type.
  • Active - Uncheck this box to deactivate the contact type. This box is checked, and the contact type active, by default.
  1. Click to save your changes to the OEDB.
