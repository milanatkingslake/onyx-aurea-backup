Reordering tracking code milestones

The order of the tracking codes milestones in the navigation area is the same order that the tracking code milestones appear in the user interface.

To reorder tracking codes milestones:

  1. In the Campaigns node, select the tracking code milestone you want to move.
  2. Click or to reorder the tracking code.

All tracking codes milestone that are moved are marked with an asterisk (*) until the changes are saved.

  1. Click to save your changes to the OEDB.
