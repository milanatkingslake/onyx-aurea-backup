Deleting a lookup value system-wide

Once you mark a lookup value for deletion and save your changes, you cannot undo the deletion.

Once you activate or deactivate the lookup value in this mode, this lookup value is activated or deactivated for all combo box fields to which it is associated.

To delete a lookup value system wide:

  1. In the navigation area, select the Lookup Value — All mode option to display the existing lookup values.
  2. Select the lookup value to be deleted. The edit area will display the lookup value attributes.
  3. Click to mark the value for deletion. The lookup value appears in the navigation area with a strike-through font. For example: Lookup.
  4. Click to save the changes. The lookup value is removed from the navigation area and is deleted from the system.

To undelete a lookup value marked for deletion

  • To “undelete” a lookup value before saving, select the item you want to restore and click . You can restore any item marked for deletion as long as changes have not been saved.
