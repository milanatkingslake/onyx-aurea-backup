Configuring a field mask

Some commonly used symbols for field masks are described in the following table.

Symbol Characters allowed for input

9

Any numeric character ‘0’ – ‘9’, decimal point.

. , : ; - /

Characters used as separators for date, time, and numerical values, depending on your international settings.

/

Escape character: Use this character when you want to insert a reserved character as a literal value within the mask.

A

Alphabetic characters, ‘a’ - ‘z’, ‘A’ - ‘Z’.

D

Any numeric character '0' - '9', or a space.

H

Hexadecimal digit ‘0’ – ‘9’, ‘a’ – ‘f’, ‘A’ – ‘F’

K

Half-width Japanese Katakana characters.

L

Lowercase alphabetic characters only: 'a' - 'z'.

N

Alphanumeric characters.

U

Upper case alphabetic characters ‘A’ - ‘Z’.

X

Any ASCII character.
