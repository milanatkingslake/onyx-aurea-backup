Adding, deleting, and changing the order of fixed responses

You work with fixed responses in the Prompt Responses Web Page Dialog.

To open the Prompt Responses Web Page Dialog:

  • On the Prompts tab of a script step’s Edit Step area, click Responses <n> (“n” is the number of existing responses).

To add a response to a prompt:

  1. In the Prompt Responses Web Page Dialog, click .

A row of input boxes appear in the Code, Text, Editable, and Active columns.

  1. In the Code and Text boxes respectively, type a code string and text.
  2. If the user should be able to add text to the response, select Editable.

If the response is editable and the user’s choice of the response is stored in a variable, the variable does not also store the text that the user adds.

  1. To make the response available to the user, select Active.
  2. Click OK.

To delete a response:

  1. In the Prompt Responses Web Page Dialog, select any box in the row of the response that you want to delete.
  2. On the Prompt Responses toolbar, click to mark the response for deletion.
  3. Click OK.

To move a response:

  1. In the Prompt Responses Web Page Dialog, select the response that you want to move.
  2. On the Prompt Responses toolbar, click or to move the response up or down in the order.
  3. Click OK.
