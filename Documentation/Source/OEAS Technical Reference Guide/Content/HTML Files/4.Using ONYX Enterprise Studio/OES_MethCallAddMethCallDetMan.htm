Adding method call details manually

To add a method call detail manually:

  1. In the navigation area, expand the method call to which you want to add a method call detail.
  2. In the expanded method call, select either the Inputs node, the Outputs node, or an existing method call detail (the type of which you want to add: input or output), and then click .

A new method call detail named newMethodCallDetail<n> appears in the navigation area followed by an asterisk (*).

  1. In the edit area, define the attributes for this method call detail.

For information on the method call detail attributes, see Method call detail attributes.

  1. Click to save your changes.

Notes:

  • How you add method call details determines their order. When you add a method call detail while the Inputs or Outputs node is selected, the new method call detail is added at the bottom of the selected node’s list of existing method call details. When you add a method call detail while an existing method call detail is selected, the new method call detail is added immediately following the selected method call detail.

For information on changing the order of method call details, see Ordering method call details.

  • For assistance in defining the Element Path, ID Path, or Description Path attributes, see About the method call detail helpers.
  • You can also add method call details using the Method Call Detail Builder. See the following section, Adding method call details using the Method Call Detail Builder.
