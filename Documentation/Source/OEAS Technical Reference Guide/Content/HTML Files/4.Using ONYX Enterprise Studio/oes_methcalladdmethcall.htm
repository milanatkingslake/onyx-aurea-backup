Adding a method call

To add a method call:

  1. In the navigation area, either select the method call group node to which you want to add a method call or select an existing method call in that group, and then click .

A new method call named newMethodCall<n> (where “n” is an integer) appears in the navigation area, followed by an asterisk (*).

  1. In the edit area, define the attributes for this method call.

The method call attributes are described in Method call attributes.

  1. Click to save your changes.

Once you have defined the attributes for the new method call, you can add its method call details. See Adding method call details.
