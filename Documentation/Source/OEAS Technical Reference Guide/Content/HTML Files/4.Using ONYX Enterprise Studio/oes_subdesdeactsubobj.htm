Activating or deactivating a subscribable object

Before deactivating a subscribable object, consider the following:

  • An inactive subscribable object does not have subscription controls appearing in OEP. Therefore, OEP end-users can no longer explicitly subscribe to or unsubscribe from the object.
  • If the deactivated subscribable object is created outside of OEP; such as in OEP-Offline, or subscribed to automatically via a subscription cascade or via direct calls to OEAS; the data defined by that object can still be subscribed to and synchronized.
  • Synchronization continues to support subscriptions to the object that existed prior to its being deactivated.

To activate or deactivate a subscribable object:

  1. In the navigation area, locate and select the subscribable object you want to modify.

The attributes of this subscribable object appear in the edit area.

  1. Select or clear the Active check box to activate or deactivate the subscribable object.
  2. Click to save your changes.
