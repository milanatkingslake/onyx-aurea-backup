Modifying user-defined information

User-defined fields are created and maintained in Reference Table Administration. For more information on maintaining user-defined fields, see Reference data in the OEAS feature overview.

To modify user-defined information:

  1. In the navigation area, navigate to and expand the product group of the product you want to modify.
  2. Select the product you want to modify.
  3. Expand the User-Defined Information data cell by clicking anywhere on the cell header.
  4. In the edit area, apply your modifications to the user-defined attributes for this product.
  5. Click on the toolbar to save your changes.
