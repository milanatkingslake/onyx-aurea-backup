Denying a user access to a resource

To deny a user access to a resource:

  1. In the navigation area, select the resource.
  2. Click Manage Permissions.
  3. In the users access-permissions list, select the user and then click either Grant/Deny or, to remove the user from the list, Remove.
  4. When you have finished managing permissions, click Done.
  5. Click on the toolbar to save your changes.
