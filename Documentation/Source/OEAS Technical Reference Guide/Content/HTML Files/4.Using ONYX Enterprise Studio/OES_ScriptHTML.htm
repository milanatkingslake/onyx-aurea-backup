Working with HTML tags in the header, footer, or prompts

You can use any HTML tags in headers and prompts. In a footer, you can use any HTML tags except the <title></title> tag pair.

There are two ways to enter HTML tags in a header, footer, or prompt:

  • Type any tags directly in the Header or Footer text box or the Text box on the Prompts tab.
  • Use the HTML Editor Web Page Dialog for adding many of the HTML formatting tags.

There are two views available in the HTML Editor:

  • Normal View - The view in which the HTML Editor formatting controls are available. This view shows the effect of the HTML tags that you apply, as you apply them.
  • HTML Source View - Shows the HTML tags in plain text.

If you write poorly formed or invalid HTML, undesirable and unpredictable behavior might result in both the Script Designer when you design the script and the end-user application during a script session.

The following table lists the buttons and their functions in the HTML Editor:

Button Tooltip Function

Bold

Makes the selected text bold.

Italic

Makes the selected text italic.

Underline

Makes the selected text underlined.

Cut

Removes the selected text and stores it on the Clipboard.

Copy

Stores a copy of the selected text on the Clipboard.

Paste

Pastes the contents of the Clipboard into the text box

Bullets

Makes the current line or selected lines of text into a bulleted list. Pressing RETURN creates a new bulleted list item.

Numbering

Makes the current line or selected lines of text into a numbered list. Pressing RETURN creates a new numbered list item.

Decrease Indent

Reduces the indentation of the current line or selected lines of text by one level.

Increase Indent

Indents the current line or selected lines of text one level.

Align Left

Aligns the current line or selected lines with the left edge of the header or footer.

Align Center

Centers the current line or selected lines in the header or footer.

Align Right

Aligns the current line or selected lines with the right edge of the header or footer.

Font Color

Opens the Color dialog box, in which you can select an existing color for the selected text or define a custom color and select it for the text.

To create HTML tags using the HTML Editor Web Page Dialog:

  1. In the Edit Step area, on the Step Details tab (if you are working with a header or footer) or the Prompts tab (if you are working with a prompt), click the Editor button for the element in which you want to add HTML tags.
  2. If you have not already typed the header or footer text, type it in the text box.

Pressing RETURN creates a new line.

  1. In the view box, select Normal View.

You can only use the formatting tag buttons and drop-down list boxes in Normal View.

  1. Use the <select a variable> box to insert available variables.
  2. To change the font or font size, make a selection in the appropriate boxes.
  3. Use the formatting buttons to add HTML formatting to the text and variables in the header or footer.

The formatting buttons and list boxes are available only in Normal View.

Buttons and boxes (such as the bold button or the font box) that affect individual characters affect only text that is selected in the text box. Buttons (such as the bulleted list button) that affect paragraph formatting affect the entire paragraph that currently contains the cursor or selection.

  1. To type or edit the HTML tags directly, in the view box, select HTML Source View.
  2. When you have finished formatting the text, click OK.

The text, any variables that you inserted appear and any HTML formatting tags that you added appear.

  1. On the toolbar, click to save your changes to the database.
