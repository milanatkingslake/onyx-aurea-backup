Associating users and roles with an OEP profile

To associate users and roles with an OEP profile:

  1. From the navigation pane, select the profile that you want to associate with a user or role.
  2. Click Manage Membership. The navigation pane displays only the items that can be associated with the profiles (users and roles).
  3. Select the role or user that you want to associate with the profile.
  4. Click Add.
  5. Repeat steps three and four until all the members are added to the profile.
  6. Click Done.
  7. Click to save the profile.
