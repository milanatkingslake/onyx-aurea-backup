Viewing insert and update information

The insert/update information for a product is for display only. It is updated by the Onyx Enterprise Database (OEDB) when a new product is created and each time an existing product is updated.

When you create or modify a product, the insert/update information is not updated until you reload or refresh the Product Administration Web page.

To view the insert/update information for a product:

  1. In the navigation area, navigate to and expand the product group of the product for which you want to view the insert/update information.
  2. Select the product for which you want to view the insert/update information.
  3. Expand the Insert/Update Information data cell by clicking anywhere on the cell header.

This product's insert/update information displays in the edit area.

  • Insert By - The user ID of the Onyx user who created the product.
  • Insert Date (GMT) - The date the product was created.
  • Updated By - The user ID of the Onyx user who last updated the product data. Once you have modified and saved this product, this attribute becomes Previously Updated By.
  • Update Date (GMT) - The date the product data was last updated. Once you have modified and saved this product, this attribute becomes Previous Update Date (GMT).
