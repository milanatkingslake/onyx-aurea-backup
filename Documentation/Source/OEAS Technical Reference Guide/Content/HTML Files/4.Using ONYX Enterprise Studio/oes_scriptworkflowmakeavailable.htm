Making scripts available

You can make scripts available in end-user applications either to all users or only to those users who have access to the UI:OEP:scripting.designer resource. (Use Security Administration to manage access permissions for security resources.)

To restrict the script’s availability to users with access to UI:OEP:scripting.designer security resource:

  1. With the script selected, in the Edit Script area, under Availability, select the Active box and make sure that the Published box is not selected.
  2. Click to save your changes to the database.

To make the script available to all users:

  1. With the script selected, in the Edit Script area, under Availability, select both Active and Published.
  2. Click to save your changes to the database.
