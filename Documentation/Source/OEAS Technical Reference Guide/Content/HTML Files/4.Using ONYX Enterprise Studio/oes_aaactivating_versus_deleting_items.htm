Activating and deactivating items

The following procedure explains how to activate or deactivate an item (territory, condition, expression, assignment set) in the navigation area.

To activate or deactivate an item:

  1. In the navigation area, select the item you want to activate or deactivate.

The attributes for the item display in the edit area.

  1. In the edit area, either select the Active box to activate the item or clear the Active check box to deactivate the item. A checked box indicates an active item.

After you modify an item’s attribute, the item appears in the navigation area with an asterisk (*) until the changes are saved. If the item was deactivated, the item is also italicized and grayed out, but the asterisk (*) disappears once the changes are saved.

  1. Click to save your changes to the OEDB.
