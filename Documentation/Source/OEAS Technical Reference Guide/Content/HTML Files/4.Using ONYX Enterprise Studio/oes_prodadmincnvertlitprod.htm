Converting a standard product to a literature product

Modifying any information on the Literature Information data cell will cause the product to become a literature product. Do not add or modify literature information for the product unless you intend for the product to be a literature product.

To convert a standard product to a literature product:

  1. In the navigation area, navigate to and expand the product group of the product you want to modify.
  2. Select the product you want to modify. The attributes for this product appear in the edit area.
  3. Expand the Literature Information data cell by clicking anywhere on the cell header. This product’s literature information appears in the edit area.
  4. In the edit area, apply your modifications to the literature attributes for this product.
  • Active Literature - Check this box to activate this literature product. A checked box indicates an active literature product that will be returned as an orderable item for Literature Order records.
  • Email attachment - Check this box to specify this literature product as a literature item that can be attached to email messages through the Add Literature Attachment page of the Email feature. For more information, see "Email" in "OEAS feature overview."
  • File Path - Type the file path for the literature product, or click the browse button and navigate to and select the file.
  • Printer - From the dropdown list, select the printer on which to print the literature product, if it is a printable item.
  • Is a Label - Check this box to specify this literature product as a label.
  • Label Product - From the dropdown list, select the type of label to generate for this literature product. When Is a Label is checked, this drop-down is disabled. You cannot choose a label type for a label.
  • Fulfillment Types - The options for how a literature product can be fulfilled by a single literature order or by bulk action in List Manager. You can choose one or more of the available options.
  • Email data in body- Creates a document from an XSL template merged with data to create an individualized message. The message appears in the body of an email.
  • Email data as attachment- Creates a document from an XSL template merged with data to create an individualized message. The message appears as an attachment to an email.
  • Email file as attachment- Specifies a binary document that can be sent as an attachment to an email. Examples include GIFs, PDFs, or EXEs.
  • Print document - Creates a document from a Open XML Word with Mail Merge template merged with data to create an individualized message. The document is automatically routed to a printer queue for printing. Examples include form letters, graphics, and company literature.
  1. Click on the toolbar to save your changes.

Once created, you cannot delete a literature product without deleting the entire product; you can only disable it.
