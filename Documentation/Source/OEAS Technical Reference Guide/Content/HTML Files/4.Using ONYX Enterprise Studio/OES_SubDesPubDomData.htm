Publishing domain data

The Publishable Objects screen provides a list of all the domain data objects that have been created as publishable objects. From this list you can choose the domain data that you want to publish.

Publishing the domain data using Subscription Designer is one step in the process that copies the domain data to OEP-Offline. To ensure that the domain data is available in OEP-Offline, follow the process outlined in Publishing domain data, which includes instructions for running the offline file list generator.

To publish domain data:

  1. In the navigation area, select the Publishable Objects node.

The edit area displays a list of all of the publishable objects.

In the Publishable Objects view, all selected objects are published regardless of their active/inactive state. Setting a publishable object to inactive allows you to take advantage of the Show only active objects option when you are selecting the objects you want to publish. You might choose to deactivate your more static publishable objects to more easily select the more active publishable objects that require re-publishing on a more frequent basis. To activate an inactive publishable object, see Activating or deactivating a publishable object.

  1. In the Publication Directory field, type the location on the OEP-Offline deployment server to which you want the domain data files published. Do not include the file name. Type the path in UNC format if your OES tools are installed on a server other than the OEP-Offline deployment server (for example, \\deploymentservername\clientfiles\website\application\domaindata).

When OEAS is installed on a separate server from the deployment server, the account that you created in the Local Account box of the OEAS setup (whose default name was OnyxEASUser) must have read/write permissions to the publication directory folder.

The location you provide is persisted as a cookie on the client and is automatically populated in the Publication Directory field when you use Subscription Designer again on the same machine.

  1. Select the objects you want to publish.
  2. Click Publish Selected Data Now.
