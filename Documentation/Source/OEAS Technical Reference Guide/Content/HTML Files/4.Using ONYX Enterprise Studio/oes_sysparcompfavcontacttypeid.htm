CompFavoriteContactTypeId

CompFavoriteContactTypeId specifies the contact type ID from the company contact category that is intended as the “Favorite” contact type for companies. OEP uses this system parameter to support the Home Page functionality for “Favorite Companies.”

  • Allowed Parameter Values - The parameter value stores an integer value.
  • Default Parameter Value - The parameter value is system-generated and corresponds to the “Favorite” contact type for the contact category for companies.

The parameter value must be a valid contact type ID from the company contact category for this system parameter to properly support the “Favorite Companies” view in the OEP Home Page.
