Disabling a subscription profile

You can disable a user’s offline privileges for a subscription service using the Profile Active and Sync Active attributes of the user’s subscription profile. These attributes can be used in any combination for a profile. You can disable a profile completely by deactivating both attributes in a profile, prohibiting the user from subscribing or synchronizing data. Or, you can limit the functionality for a profile by deactivating either one of the attributes. The following sections describe how to use these settings.
