Changing the order of rules

When a step has multiple rules to evaluate, the first rule whose condition statement evaluates to true determines the next step in the script session. Therefore, the order in which rules are evaluated can affect which step the user sees next.

To change the order of rules in the step:

  1. In the Rules box, select the rule that you want to move.
  2. Click or to move the rule up or down in the list.

Until you save changes to the database, the rules whose order in the list is affected by the move appear with an asterisk.

  1. On the toolbar, click to save your changes to the database.
