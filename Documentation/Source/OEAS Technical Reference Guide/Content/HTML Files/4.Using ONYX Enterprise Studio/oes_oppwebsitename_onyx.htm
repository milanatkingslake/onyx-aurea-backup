OPPWebSiteName_Onyx

OPPWebSiteName_Onyx specifies the Onyx Partner Portal (OPP) web site name that implements Onyx authentication. This parameter is used for Channel Manager Integration with Onyx Employee Portal.

  • Allowed Parameter Values - a string containing the name of the OPP site that implements Onyx authentication.
  • Default Parameter Value - (empty)
