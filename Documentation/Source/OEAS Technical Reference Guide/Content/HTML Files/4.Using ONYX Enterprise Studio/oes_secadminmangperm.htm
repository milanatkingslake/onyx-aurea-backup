Adding a user or role to a resource

To add a user or role to a resource:

  1. In the navigation area, select the resource.
  2. Click Manage Permissions.
  3. Select the user or role you want to add to the resource and then either click the Add button or drag the user to the access-permissions list in the edit area. If you drag the user, you must drag it either to the heading area of the list or to one of the users already in the list.
  4. Click the Active/Inactive button to set the active status of a user or role within a resource. The default is active. When you set a user or role to inactive within the resource, the inactive status is shown in the Permission column for the user or role.
  5. When you have finished adding users or roles, click Done.
  6. Click on the toolbar to save your changes.
