Modifying an existing record

To modify a record:

  1. In the Table Administration grid, click the field you want to edit. The field's content is highlighted and any text that you type replaces the existing text in the field. If you want to add text without replacing the existing text, click the field again where you want to start adding new text.

User-defined ID fields (primary key fields) cannot be edited once saved to the database. Date fields and system generated fields are also not editable.

  1. Click on the toolbar to save your changes.
