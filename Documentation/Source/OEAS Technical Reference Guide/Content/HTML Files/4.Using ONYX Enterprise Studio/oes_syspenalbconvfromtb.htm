EmailSendStepSmtpServer

When the email server is not local to the SMTP server, this system parameter is used by the Email feature to specify the (DNS) name or IP address of the remote email server from which email messages are sent over the Internet for delivery.

Before you can specify an SMTP server as the value for EmailSendStepSmtpServer, the value “server” must be set for the system parameter EmailSendStepSmtpMode.

  • Allowed Parameter Value - Type the name (DNS) or IP address of the remote email server hosting the SMTP service through which email messages are sent over the Internet. The email server that you name must be accessible from the OEAS server.
  • Default Parameter Value - EnableSendStepSmtpServer is installed as (empty) by default. If the email server is local, leave the value at the default (empty).
