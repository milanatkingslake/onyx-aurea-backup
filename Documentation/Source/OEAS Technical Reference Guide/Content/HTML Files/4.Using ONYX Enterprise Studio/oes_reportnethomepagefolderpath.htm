ReportNetHomePageFolderPath

ReportNetHomePageFolderPath specifies a Cognos Connection folder path where reports exist for OEP to display in the Home Page Reports section.

  • Allowed Parameter Values - a string containing a valid Cognos Connection location. For information on creating this string, see the Cognos Integration Module documentation. To disable this feature, leave this field blank.
  • Default Parameter Value - (empty)
