Adding method call details using the Method Call Detail Builder

For more information on the Method Call Detail Builder, see About the Method Call Detail Builder.

To add a method call detail using the Method Call Detail Builder:

  1. In the navigation area, expand the method call to which you want to add a method call detail.
  2. Select either the Inputs or Outputs node, according to whether you want to add an input or output method call detail.

The Method Call Detail Builder appears in the edit area.

  1. From the Properties list, select the property/parameter that you want to add as an input or output method call detail, and then click Add.

OR

To include all the properties/parameters as method call details, click Add All.

The selected properties/parameters move to the Outputs or Inputs list (depending on whether you are creating output or input method call details).

  1. To remove a method call detail from the Outputs or Inputs list, select it and click Remove.

OR

To remove all the properties/parameters from the Outputs or Inputs list, click Remove All.

The properties/parameters you selected move back to the Properties list.

  1. To create method call details from the properties/parameters you have included in the Outputs or Inputs list, click Build.

Each new method call detail appears in the navigation area, followed by an asterisk (*).

  1. For each new method call detail, modify the method call detail’s attributes as appropriate for your needs.

The Method Call Detail builder automatically provides values for many of the attributes. For information on changing an attribute value, see Modifying a method call detail.

  1. Click to save your changes.

Notes

  • To select multiple items from the Outputs or Inputs list, hold the CTRL key down while selecting the items.
  • The order in which the details appear in the Outputs or Inputs list is the order in which they will appear in the navigation tree when you click Build. To modify the order, see Ordering method call details.
  • You can also add method call details manually. See Adding method call details manually.
