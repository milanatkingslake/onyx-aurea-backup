Deleting an OEP profile

You may delete an OEP profile when there are no users or roles associated with it. When you try to delete a profile that is associated with a user or role, a warning appears and you are unable to delete the profile.

To delete an OEP profile:

  1. In the navigation pane, open the OEP Profile node. Select the profile you want to delete.
  2. Click to mark the profile for deletion. (To un-delete, click again.)

  2. The profile no longer appears in the navigation pane.
