Deleting a variable

When you delete a variable, remember also to remove any references to the variable wherever they might occur in the script.

To delete a variable:

  1. Open the Variables Web Page Dialog.
  2. Select the variable that you want to delete.
  3. To mark the variable for deletion, on the Variables Web Page Dialog toolbar, click .

The variable is marked for deletion.

To remove a mark for deletion, re-select the marked variable, and then click .

  1. Click OK.
  2. On the toolbar, click to save your changes to the database.
