Managing method call details

Using the Method Call Designer, you can manage method call details by:

  • Adding method call details
  • Modifying a method call detail
  • Activating or deactivating a method call detail
  • Deleting a method call detail
  • Copying a method call detail
  • Ordering method call details

Method call detail attributes appear in the Edit Method Call Detail area when you add a new method call detail or when you select an existing method call detail in the navigation area. The method call detail attributes are described in Method call detail attributes. The Method Call Designer provides additional helper menus to help you define certain attributes of each method call detail. For more information on these helper menus, see About the method call detail helpers.
