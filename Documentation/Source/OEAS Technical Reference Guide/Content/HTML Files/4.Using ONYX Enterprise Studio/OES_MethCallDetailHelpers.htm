About the method call detail helpers

There are three pop-up Method Call Detail Helper menus available in the Edit Method Call Detail screen for the Element Path, ID Path, and Description Path attributes. These Helper menus assist you by inserting the proper Xpath for the menu item selected for each of the attributes.

A pop-up menu launcher button, , appears to the right of each of the text fields for the Element Path, ID Path, and Description Path attributes.

Element Path Helper

The Element Path Helper is a menu containing an XML structure that is populated according to the OEAS object you selected for the method call. That object’s method contentIn or contentOut values (depending on whether the method call detail is an input or output detail) determine which properties appear in the Helper menu. For example, if an input method call detail is based on an OEAS object where, in the OED, the contentIn values are set to “keysOnly,” only that OEAS object’s key values appear in the Helper menu. Otherwise, all the properties appear.

If the method call detail is an input detail, the XML structure in the Helper menu is relative to parameters. If the method call detail is an output detail, the XML structure in the Helper menu is relative to parametersReturn. When you select one of the elements from the menu, the Helper automatically populates the Element Path text field with the correct Xpath to the selected element.

ID Path Helper

The ID Path Helper is a menu containing a list of elements relative to the item identified in the Element Path attribute. When you select one of these elements, the Helper populates the ID Path text field with the proper Xpath for the ID component relative to the Element Path. If the Element Path is empty, a dialog box appears. Also, if you subsequently modify the Element Path, a dialog box appears.

Currently, the Builder does not support listing the actual rowSet column names for retrieveList methods.

You can define the ID Path when you create a list-type Scripting output method call detail. For more information on the ID Path, see ID Path and Description Path.

Description Path Helper

The Description Path Helper contains the same list of elements as the ID Path Helper does, but in the Description Path Helper, the selected item is used to find an element’s description component instead of its ID component. The same dialog boxes appear for the same conditions related to the Element Path.

Again, the Builder does not currently support listing the actual rowSet column names for retrieveList methods.

You can define the Description Path when you create a list-type Scripting output method call detail. For more information on the ID Path, see ID Path and Description Path.
