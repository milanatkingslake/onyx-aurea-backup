Changing Edit Layout area information display settings

By default, only the step name and the names of its prompts appear in the Edit Layout area. You can specify that additional information appear there.

To set information display choices for prompts and rules:

  1. To open the Edit Layout Options Web Page Dialog, on the Edit Layout area toolbar, click .
  2. Under Display, beside Rule Information, select one of the following options:
  • Hide - No rule information is displayed.
  • Show Rule Name
  • Show Rule Condition
  • Show Name and Condition
  1. Under Display, beside Prompt Information, select one of the following options:
  • Hide - No prompt information is displayed.
  • Show Prompt Name
  • Show Prompt Text
  • Show Name and Text
  1. Click to save your changes to the database.
