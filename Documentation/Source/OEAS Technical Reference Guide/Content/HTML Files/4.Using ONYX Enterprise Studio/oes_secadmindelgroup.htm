Activating or deactivating a group

There is no functionality to delete a group, you must deactivate it.

To activate or deactivate a group:

  1. In the navigation area, select the group you want to activate or deactivate.

The attributes for this group appear in the edit area.

  1. In the edit area, check or uncheck the Active box to activate or deactivate the group. A checked box indicates an active resource.
  2. Click on the toolbar to save your changes.
