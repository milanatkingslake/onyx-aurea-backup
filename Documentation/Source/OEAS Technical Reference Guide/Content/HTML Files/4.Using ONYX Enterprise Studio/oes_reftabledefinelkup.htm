Defining lookup values for standard incident fields

Incident fields are grouped together by incident category in the incident node in the navigation area. The standard configurable incident fields that are defined as combo box fields display under the specific incident category (default categories are Sales, Product Support, Customer Service, Work Ticket) within the incident node.

To define lookup values for standard incident fields:

  1. In the navigation area, select the Lookup Value by Field mode to display the available nodes.
  2. Click in front of the incident node to expand the node and display the fields associated with it.
  3. Click in front of the incident.category field node to display the incident categories.
  4. Click in front of a specific incident category node to expand the node and display the fields for that incident category.
  5. Click in front of a specific field of that incident category to display any associated lookup values.
  6. Add or modify the lookup values for the incident category field as described in Adding lookup values to a combo box field and Modifying a lookup value for a combo box field, respectively, beginning at step 4.
  7. Click on the toolbar to save your changes.
