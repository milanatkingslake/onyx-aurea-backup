Removing a script step

Remember that you can not delete a script step that has been used in a script session for the script. Therefore, after the script has been delivered to a customer, you must not delete the step. To make the step unavailable in subsequent script sessions, you can change the destination step of any rule that leads to the step that you are removing. For information on changing a rule’s destination steps, see Working with rules.

To delete a script step:

  1. In the navigation or Edit Layout area, select the step.
  2. To mark the step for deletion, on the toolbar, click .
  3. To remove the mark for deletion from a step, highlight the step, and then click .
  4. Click to save your changes to the database.
