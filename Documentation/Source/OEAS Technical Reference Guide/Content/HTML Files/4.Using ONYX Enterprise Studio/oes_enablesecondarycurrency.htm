EnableSecondaryCurrency

EnableSecondaryCurrency enables the system to display forecasts and quotes in primary and secondary currencies. Additionally, the summary data displayed in the bottom right-hand corner of the Forecast/Quotes tab is displayed for both currencies.

  • Allowed Parameter Values

N - Secondary currency is not enabled.

Y - Secondary currency is enabled for all Incident types (Sales, Support and Service).

  • Default Parameter Value - EnableSecondaryCurrency is installed as disabled (N) by default.

For secondary currency values to be visible to the end-user, you must have both EnableSecondaryCurrency and EnableCurrencyConversion set to (Y).
