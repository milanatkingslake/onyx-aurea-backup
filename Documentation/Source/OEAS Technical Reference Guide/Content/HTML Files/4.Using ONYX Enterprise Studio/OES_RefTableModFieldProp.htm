Modifying a field’s attributes

To modify field attributes:

  1. In the navigation area, select the Fields mode option to display the available nodes.
  2. Click to display the fields associated with a particular node.
  3. Select a field to see the attributes displayed in the edit area.
  4. Update the field attributes in the edit area. As you modify the field attributes, the field name in the navigation area displays an asterisk (*). For descriptions of the field attributes, see Defining field attributes.

If you change a field's Field Type property from a combo box (containing lookup values) to a check box or text box, you will lose any lookup values associated with that field. If you change the field's Field Type property back to a combo box, the lookup values that were previously associated with that field will no longer be associated with it. To return the combo box to its previous state, you must manually re-add the lookup values to the field.

  1. Click on the toolbar to save your changes.
