Creating questions

You can create questions for surveys in the following ways:

  • Add the question to the All Questions node of the navigation area and then to the survey.
  • Add the question within the edit area for a survey.

For information on adding a question within the edit area of a survey, see Adding questions within a survey.

To create a question in All Questions:

  1. In the navigation area, highlight All Questions and then click .
  2. In the Text box, type the text of the question.
  3. In the Question Type box, select Single, Multiple, or Free Text.
  • Single specifies that the respondent can pick only one of the available question responses.
  • Multiple allows the respondent to pick more than one of the available question responses.
  • Free Textallows the respondent to write his or her own response.
