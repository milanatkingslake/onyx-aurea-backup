Assignment Administration Help

Assignment Administration enables you to create and modify customer and incident assignment conditions (rules) for customers and incidents. These rules define how users are assigned to incidents and created as internal contacts on company, individual, and incident records.

This documentation describes Assignment Administration's functionality and use. For conceptual information and usage examples, see Assignment Studio.

Assignment capabilities beyond those provided by Assignment Studio can be accomplished in Onyx Process Manager.
