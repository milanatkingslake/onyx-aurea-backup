Adding new milestone actions

You create new milestone actions using the first-level Milestone Actions node. The list of milestone actions that you create are used to populate the Milestone Action drop-down list. This list appears in the edit area when creating a new tracking code milestone for a campaign.

To add a new milestone action:

  1. Select the Milestone Actions node, and click on the toolbar.

OR

Select an existing milestone action, and click on the toolbar.

A new milestone action titled newMilestoneAction<n>, followed by an asterisk (*), appears at the bottom of the milestone actions list.

  • In the edit area, define the attributes for this milestone action.
  • Milestone Action Id - This field is a non-editable field.
  • Name - Type the name you want to assign to this milestone action.
  • Description - Type a description of this milestone action.
  • Active - Check or uncheck this box if you want to activate or deactivate this milestone action.
  1. Click to save your changes to the OEDB.
