Activating or deactivating a record

When you deactivate a record, it no longer appears to end users for active use, but it remains an active element of records previously associated with it.

To modify the active status of an existing record:

  1. Click the Active field to highlight the existing value.
  2. Type the value that corresponds to the status you want for the record.
  • 0 - deactivates the record making it unavailable to end users
  • 1 - activates the record making it available to end users
  1. Click on the toolbar to save your changes.

If any problems occur during the save request, no records will be saved.
