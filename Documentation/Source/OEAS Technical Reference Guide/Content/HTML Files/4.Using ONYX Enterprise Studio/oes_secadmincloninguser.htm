Cloning a user

Cloning a user creates a new user with the same attributes and access permissions as the user you cloned.

To clone a user:

  1. In the navigation area, open the group that the user is a member of and then select the user.
  2. Click on the toolbar.
  3. With the new user selected, in the edit area, change the new user's attributes as needed.

For the list of user attributes you can modify, see Adding a user.

  1. Click on the toolbar to save your changes.
