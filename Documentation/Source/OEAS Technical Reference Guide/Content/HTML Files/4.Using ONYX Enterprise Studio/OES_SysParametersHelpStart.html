System Parameter Administration Help

System Parameter Administration enables you to edit system parameters. You use system parameters to control the behavior of several features of the Onyx Enterprise Application Server (OEAS), including how features are presented to users of the Onyx Employee Portal (OEP) Web client. Generally, you enable or disable a system parameter to control the behavior of a feature, or input a string or integer value for the system parameter that is recognized by the application.

An example of a system parameter is TagIncludeCount. The TagIncludeCount parameter enables you to control the number of times a tag is used before it is returned by retrieveList.

ONYX supports modification of the system parameters that are discussed in this section only. Modifying any other system parameters might adversely affect your system.
