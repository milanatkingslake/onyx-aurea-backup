Presentation of the OES tools

The interfaces for most of the OES tools share a common design that make it easy to move from one tool to another. Most of the interfaces have the same basic layout:

Context area - The context area displays the name of the tool in which you are working and the name of the OEAS application that you are administering or customizing.

Toolbar - The toolbar contains buttons that vary according to the tool you’re using.

Navigation area - In most tools, the navigation area presents trees of elements that you can manage with the tool. To change properties of an element, select the element in the navigation area, then change its properties in the edit area.

Edit area - The edit area provides input boxes through which you can modify an element’s fields and properties.
