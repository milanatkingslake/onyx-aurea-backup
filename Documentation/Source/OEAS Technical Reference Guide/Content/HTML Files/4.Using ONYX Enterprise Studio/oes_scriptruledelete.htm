Deleting a rule

To delete a rule from a step:

  1. On the Rules tab, in the Rules box, highlight the rule that you want to delete.
  2. To mark the rule for deletion, on the Rules toolbar, click .

If the following rule in the list has not already been marked for deletion, the highlight moves to that rule.

  1. To remove the mark for deletion from a rule, highlight the marked rule and, on the Rules toolbar, click .
  2. On the toolbar, click to save your changes to the database and delete the rule.
