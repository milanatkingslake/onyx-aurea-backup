Modifying the display order of lookup values

Modifying the display order of lookup values as they appear in Reference Table Administration also changes the display order of the lookup values as they appear in the user interface.

To change the display order of lookup values:

  1. In the navigation area, select the Lookup Value by Fields mode option to display the available nodes.
  2. Click , or double-click the node to display the fields associated with a particular node.
  3. Click in front of the field you want to view, or double-click the field, to display the lookup values associated with that field.
  4. Select the lookup value you want to change.
  5. Click or to move the lookup value up or down in the navigation area.
  6. Click on the toolbar to save your changes.
