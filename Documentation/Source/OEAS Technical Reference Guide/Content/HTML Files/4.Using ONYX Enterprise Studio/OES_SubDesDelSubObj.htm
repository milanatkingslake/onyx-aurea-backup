Deleting a subscribable object

Before deleting a subscribable object, consider the following:

  • To prevent data loss, all OEP end-users who have subscribed to the object must synchronize their offline changes to OEP and unsubscribe the object.
  • Because of database constraints, you cannot delete a subscribable object if subscriptions exist for the subscribable object.

To delete a subscribable object:

  1. In the navigation area, locate and select the subscribable object that you want to delete.
  2. Click to mark the subscribable object for deletion.

The subscribable object name appears with a strikethrough font in the navigation area (for example: newSubscribableObject).

  1. Click to save your changes and delete the subscribable object.

If you have simply marked the subscribable object for deletion and have not yet saved the changes, you can still “undelete” the subscribable object by unmarking it for deletion.

Use the following procedure to unmark a subscribable object marked for deletion.

To unmark a subscribable object marked for deletion:

  1. In the navigation area, locate and select the subscribable object that you previously marked for deletion. It is distinguished in the navigation area by the strikethrough font of its name.
  2. Click to unmark the subscribable object for deletion.

The strikethrough font is removed from the name, and the subscribable object is unmarked for deletion.

  1. Click to save your changes.
