Overview of tasks to create a script

The general steps you follow when creating a script are as follows:

  • Make sure that the Scripting feature is enabled.

To enable the Scripting feature, contact your Onyx consultant or Onyx Product Support.

  • Plan the script.
  • Create a script group (if necessary).
  • Create the script.
  • Create any method calls needed.
  • Create any variables needed.

When creating variables for a method call, it is often easier to use the Method Call Binding Builder Web Page Dialog to create the variables.

  • Create the script steps, including specifying attributes; creating header and footer text, prompts and responses, and rules; and associating method calls with the step.
  • Test the script, making any needed changes and retesting.
  • Make the script available to end-users.
