Security Administration Help

Security Administration enables you to manage users and their access to the following resources:

  • Methods of the OEAS business objects, as defined in the Onyx Enterprise Dictionary (OED).
  • Onyx Enterprise Studio (OES) and Onyx Employee Portal (OEP) interfaces.
  • Other resources that you create in Security Administration.

In addition to managing user access to objects, security involves user authentication and licensing. For a conceptual background to managing access permissions, see Managing user access with Security Administration.
