Viewing the OEP-Offline subscription service

You can view the OEP-Offline subscription service by clicking on it in the navigation tree. This causes the attributes for the subscription service to display in the edit area.

Onyx recommends that you not modify the default attribute settings for the OEP-Offline subscription service.

ID - The ID attribute is not editable as it is a primary key field. This attribute is a unique application-defined identification number.

Sync Connector ProgID - The Sync Connector ProgID identifies the Exchange sync connector you want to use for the subscription service. OnyxPSExchangeSyncConnector.SyncConnector is the Prog ID of the Exchange sync connector that is provided with OEP-Offline. It is a registered COM component. The Sync Connector ProgID can be a maximum of 255 characters. This attribute is read-only and cannot be edited.

Name - This attribute is the user-friendly name you provide for the subscription service. The name can be a maximum of 50 characters.

Description - The Description attribute is the brief description of the subscription service that you provide. The description can be a maximum of 255 characters.

Active - The Active attribute controls whether or not the subscription service is active.
