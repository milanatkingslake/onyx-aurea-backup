Modifying warranty information

To modify warranty information:

  1. In the navigation area, navigate to and expand the product group of the product you want to modify.
  2. Select the product you want to modify. The attributes for this product appear in the edit area.
  3. Expand the Warranty Information data cell by clicking anywhere on the cell header.
  4. In the edit area, apply your modifications to the warranty attributes for this product.
  • Number of Days - Select this option to require that the warranty for the product be dependent on the number of days since purchase. The default is on. Once selected, type the number of days the warranty is valid for the product. The valid range is 0-32767.
  • Starting With - The warranty start date indicator. Enabled only if Number of Days is selected. Default is First Call.
  • First Call - Select this option to have the warranty begin with the customer's first call regarding the product.
  • Insert Date - Select this option to have the warranty begin with the customer's purchase of the product.
  • Number of Calls - Select this option to require the warranty of the product to be dependent on the number of customer calls made regarding it. The default is off. Once selected, type the maximum number of support calls included in the warranty. The valid range is 0-32767.
  • Maximum Cutoff Date - Type the last date on which the product is under warranty.
  1. Click on the toolbar to save your changes.
