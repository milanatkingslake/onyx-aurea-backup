Work flow for managing user access

The following steps represent a recommended overall work flow for managing user access through Security Administration:

  1. Plan role membership for users, based on the resource-access requirements that users share.
  2. Define groups that users can become members of.
  3. Create users and add them to the groups you have defined.
  4. Define roles to associate with your users.
  5. Add users to the roles you have defined.
  6. Define OEP profiles to associate with your users and roles.
  7. Associate users and roles with the OEP profiles you have defined.
  8. Secure Business Object resources that require restricted access.
  9. Add any custom resources to Other Resources or UI Resources.
  10. Manage permissions for resources.
