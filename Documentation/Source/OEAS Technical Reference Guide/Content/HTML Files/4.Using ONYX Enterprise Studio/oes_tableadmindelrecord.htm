Deleting a record

To delete or undelete a record:

  1. In the Table Administration grid, locate and select the record you want to delete.

Once you mark a record for deletion and save your changes, you cannot undo the deletion. Onyx recommends that you deactivate records within Table Administration instead of deleting them. When you deactivate a record, it no longer appears to end users for active use, but it remains an active element of records previously associated with it.

  1. Click on the toolbar to mark the record for deletion.
  2. Click to save the changes to the Onyx Enterprise Database (OEDB).

If you decide that you do not want to undelete a record marked for deletion, select the record and click on the toolbar.
