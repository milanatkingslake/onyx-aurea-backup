Associating users with assignment sets

To associate a user with an assignment set:

  1. In the navigation area, expand the appropriate individual, company, or incident assignment set.
  2. Either select the Users node or select an existing user and then click .

An asterisk (*) appears next to the user name in the navigation area. If this is an incident assignment set, the Assign Incident attribute appears in the edit area.

  1. In the edit area, define the attributes for this user.
  • User ID - Type a value or select one by clicking . Values are case-sensitive and must be entered accurately. Only users of the same object type as the active assignment set are available.
  • Assign Incident - Incident assignment set only. Enables you to assign incidents to users.
  • Internal Contact Type - Defines the type of internal contact record that is created for the user.  You can type a value or select one by clicking to display the Select a value dialog box. Only internal contacts of the same object type as the active assignment set are available.
  • Click to save your changes.

A new user is created.

  • Next, link the assignment set to a condition.

An assignment set must be saved in the Assignment Tree before it is linked to a condition.
