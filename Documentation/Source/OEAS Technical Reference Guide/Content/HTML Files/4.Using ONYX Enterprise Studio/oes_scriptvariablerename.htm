Renaming a variable

If you rename a variable, be sure also to manually change the name wherever it’s used in headers, footers, prompts, rules, or as method call inputs or outputs in method call bindings.

To rename a variable:

  1. Open the Variables Web Page Dialog.
  2. Select the variable that you want to rename.
  3. Change the name in the Name column.
  4. Click OK.
  5. On the toolbar, click to save your changes to the database.
