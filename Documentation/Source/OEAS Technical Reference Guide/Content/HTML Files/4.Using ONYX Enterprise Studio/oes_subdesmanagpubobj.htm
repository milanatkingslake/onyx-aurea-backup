Managing publishable objects

The Publishable Objects node enables you to configure the publishable objects and publish them, so that the domain data identified by those objects can be copied to OEP-Offline.

You can administer publishable objects by:

  • Publishing domain data.
  • Creating/configuring publishable objects, which includes:
  • Adding a publishable object.
  • Modifying a publishable object.
  • Activating or deactivating a publishable object.
  • Deleting a publishable object.
