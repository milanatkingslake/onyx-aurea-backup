Modifying system parameters

Onyx supports modification of only the system parameters that are discussed in this Help system. Modifying any other system parameters might adversely affect your system.

To modify a system parameter:

  1. In the navigation area, select the system parameter you want to modify.

The Parameter Code becomes highlighted and displays in the edit mode.

  1. Apply your modifications to this system parameter's parameter value.
  2. Parameter Value - Type the value for this system parameter. Allowable data types and suggested values are those listed in the Details area.

The following read-only fields are displayed in the Details area located at the bottom of the page:

  1. Description - Contains a short description of the highlighted system parameter’s function.
  2. Data Type - Displays the data type of the selected system parameter value. The types are:
  • Integer
  • String
  • Y/N
  1. Suggested Value - Displays the suggested value for the selected system parameter. The suggested values are the settings in the SystemParameter table at installation. If improper revisions are made to a particular system parameter, use these values to reset the application back to installation default values.
  2. Minimum Value - Displays the minimum allowed integer value for the selected system parameter. You can enter and save integer values that fall outside this range, but in many cases these excluded values are not supported.
  3. Maximum Value - Displays the maximum allowed integer value for the selected system parameter. You can enter and save integer values that fall outside this range, but in many cases these excluded values are not supported.
  4. Click to save your changes.
