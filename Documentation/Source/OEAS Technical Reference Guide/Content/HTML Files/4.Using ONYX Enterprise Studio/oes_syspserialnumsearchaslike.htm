SerialNumberSearchAsLike

If enabled, any searches done using serial numbers automatically use wild card strings at the end of the serial number search criteria. For example, using the search criteria 123 finds all entries in the database whose serial number begins with a 123. If SerialNumberSearchAsLike is not enabled, only the entries whose serial number is 123 are returned by the search.

  • Allowed Parameter Values

N - Serial number searches look for an exact match.

Y - Serial number searches returns all serial numbers that begin with the entered value.

  • Default Parameter Value - SerialNumberSearchAsLike is installed as disabled (N) by default.
