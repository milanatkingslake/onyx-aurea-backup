Subscription Designer Help

Subscription Designer enables you to manage how users can subscribe to records for use with OEP-Offline. Using Subscription Designer, you can view and configure subscription services, subscribable objects, subscription relationships, and subscription rules, which affects the data that synchronized between OEP (online) and OEP-Offline. You can also use Subscription Designer to publish domain data so that it is available in OEP-Offline.

For an overview of the Subscription Designer interface, see Quick overview of the Subscription Designer.

Common tasks are described in the following sections:

  • Managing subscribable objects
  • Managing subscription services
