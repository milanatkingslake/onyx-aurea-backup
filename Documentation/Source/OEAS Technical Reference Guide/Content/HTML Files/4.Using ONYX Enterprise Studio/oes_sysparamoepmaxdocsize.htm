OEPMaximumDocumentSizeKB

OEPMaximumDocumentSizeKB limits the size of files that can be uploaded to the file share. This is the system-wide maximum size (in kilobytes) of a file that can be uploaded.

  • Allowed Parameter Values - Valid input will be any integer value, 0 or larger. A value of “0” will indicate no file size restriction (not recommended). Any positive integer will indicate the maximum size, in kilobytes.
  • Default Parameter Value - OEPMaximumDocumentSizeKB is installed with a default value of 10,000.
