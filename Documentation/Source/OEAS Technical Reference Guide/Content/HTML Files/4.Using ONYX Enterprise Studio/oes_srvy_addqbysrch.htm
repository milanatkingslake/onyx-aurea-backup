Adding an existing question

Add a question within the edit area of a survey by searching either other surveys or all existing questions.

To add a question from other surveys:

  1. In the navigation area, expand the survey to which you want to add the question and select Survey Questions.
  2. Click the down arrow in the box beside Show questions from survey and then click the survey that contains the question you want to add to this survey.
  1. In the list of questions, select one or more questions and then click Add Selected.

In the navigation area, the selected question is added to the list under the survey’s Survey Questions.

  1. Click to save your changes.

To add a question by searching all questions:

  1. In the navigation area, expand the survey to which you want to add the question and select Survey Questions.
  2. In the edit area of the survey to which you want to add the question, in the Search for a question box, type a string that the question contains and then click Search.

Note that:

  • The search engine is not case-sensitive.
  • If you leave the Search for a question box empty, then clicking Search produces a list of all questions.
  1. In the resulting list of questions, select one or more and then click Add Selected.

In the navigation area, the selected question is added to the list under the survey’s Survey Questions.

  1. Click to save your changes.
