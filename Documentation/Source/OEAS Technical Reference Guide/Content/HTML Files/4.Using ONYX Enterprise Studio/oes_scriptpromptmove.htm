Moving a prompt within the step

During a script session, the order of prompts in the step follows their order in the Prompts box.

To change the order of prompts in the step:

  1. On the Prompts tab, in the Prompts box, select a prompt that you want to move.
  2. Click or to move the prompt within the list.

Until you save changes to the database, the prompts whose order in the list is affected by the move have an asterisk attached.

  1. On the toolbar, click to save your changes to the database.
