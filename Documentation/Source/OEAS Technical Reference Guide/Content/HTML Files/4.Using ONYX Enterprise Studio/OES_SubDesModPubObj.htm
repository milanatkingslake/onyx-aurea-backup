Modifying a publishable object

Use the following procedure to modify a publishable object.

Modifying a publishable object does not automatically publish the data, and it does not alter any data that has previously been published. For example, suppose you create a publishable object with a publication filename of data1.xml and publish the data, and then modify the publishable object to change the publication filename to data2.xml. Future publications will use the data2.xml filename, but there will still be a data1.xml file in the publication directory containing the old data. The old file (data1.xml) will remain until it is deleted by the administrator.

To modify a publishable object:

  1. In the navigation area, locate and select the publishable object that you want to modify.

The attributes for this publishable object display in the edit area.

  1. Apply your modifications to this publishable object's attributes. See Adding a publishable object for details about attributes you can modify.
  2. Click to save your changes.
