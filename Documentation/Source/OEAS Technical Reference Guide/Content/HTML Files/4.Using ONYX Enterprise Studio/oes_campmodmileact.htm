Modifying milestone actions and milestone categories

You modify individual milestone actions and milestone categories through the Milestone Actions node and Milestone Categories node. Changes to individual milestone actions/milestone categories will be reflected in the tracking code milestones that currently use them.

To modify a milestone action:

  1. In the Milestone Actions node, select the milestone action you want to modify.
  2. In the edit area, modify the attributes. See Adding new milestone actions for the list of attributes you can modify.

In the navigation area, an asterisk (*) appears next to the modified milestone action until your changes are saved.

  1. Click to save your changes to the OEDB.

To modify a milestone category:

  1. In the Milestone Categories node, select the milestone category you want to modify.
  2. In the edit area, modify the attributes.
  • Milestone Category Id - This field is a non-editable field.
  • Name - Type the name you want to assign to this milestone category.
  1. In the navigation area, an asterisk (*) appears next to the modified milestone category until your changes are saved.
  2. Click to save your changes to the OEDB.
