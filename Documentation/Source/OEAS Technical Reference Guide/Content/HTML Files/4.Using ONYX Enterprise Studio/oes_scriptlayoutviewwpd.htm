Viewing the script in the Script Layout View Web Page Dialog

The Script Layout View Web Page Dialog allows you to view the layout of the entire script in a resizable Web Page Dialog. This is a viewing-only Web Page Dialog. In it, there is no way to make any changes to the script. You can, however, copy the script’s scalable vector graphics (SVG) representation to the Clipboard from within the Script Layout View Web Page Dialog, then paste the SVG into another program.

Copying the SVG to the Clipboard actually copies two versions of the SVG to the Clipboard, a text version (the XML that represents the script) and a raster version (a pixel-by-pixel bitmap of the actual image in the Script Layout View Web Page Dialog).

With these two copies of the SVG in the Clipboard, you can either paste the XML text version of the SVG into a text editor or paste the raster version of the SVG into a program that can render a bitmap.

When working with the raster image, remember that if the script is very large (comprising many screens), you can not copy it directly into the Clipboard. However, you can use a third-party screen-capture program (such as SnagIt) to concatenate all the screens into a single image and copy and paste that image.

When copying the text (XML) version of the script, remember that a program that interprets the SVG when you paste it might not represent the script in the same way that the Script Designer Edit Layout area does.

To view the script in the Script Layout View Web Page Dialog:

  1. In the navigation area, select either the script, its Steps node, or one of its steps, and then click .

The Script Layout View Web Page Dialog appears, with the script layout represented within it.

If the script layout is larger than the Web Page Dialog, scroll bars appear as necessary.

You can resize the Web Page Dialog by dragging its borders.

  1. To copy the SVG to the Clipboard, right-click anywhere in the main area of the Web Page Dialog.
  2. To return to the Script Designer and continue working with the script, close the Script Layout View Web Page Dialog.
