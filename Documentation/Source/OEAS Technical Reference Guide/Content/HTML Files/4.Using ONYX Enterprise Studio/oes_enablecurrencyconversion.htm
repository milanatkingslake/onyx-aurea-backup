EnableCurrencyConversion
EnableCurrencyConversion

Enables users to perform currency conversions for forecasts and quotes on the Forecast/Quotes tab. With this system parameter enabled, when a user selects a currency code that is different than the base currency of a line item product, the system converts the base currency to determine the new price and totals based on the new currency chosen. The CurrencyConversion table supports the conversion calculation by storing the conversion rate between the two currencies.

If EnableSecondaryCurrency is also enabled, conversions are performed from the primary currency to the secondary currency.

  • Allowed Parameter Values

N - Currency conversion is not enabled.

Y - Currency conversion is enabled for all incident types (Sales, Support and Service).

  • Default Parameter Value - EnableCurrencyConversion is installed as disabled (N) by default.
