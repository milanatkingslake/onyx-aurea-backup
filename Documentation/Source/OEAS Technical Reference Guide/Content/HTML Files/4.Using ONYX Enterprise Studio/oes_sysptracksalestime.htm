TrackSalesTime

Determines whether or not time-tracking is enabled for Sales Incidents. With time-tracking enabled, the system displays the Time Adjustment window every time the user saves an open Sales Incident.

  • Allowed Parameter Values

N - Time is not tracked.

Y - Time is tracked and the time adjustment window is displayed when the incident is saved.

  • Default Parameter Value - TrackSalesTime is installed as enabled (Y) by default.
