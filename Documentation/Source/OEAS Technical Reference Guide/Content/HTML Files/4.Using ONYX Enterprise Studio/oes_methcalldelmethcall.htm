Deleting a method call

After you mark a method call for deletion and save your changes, you can not undo the deletion.

You cannot delete a method call if that method call is being used within OEAS (for instance, if that method call is being used by a script or a merge template). You must remove the association before you can delete the method call. You can remove the association either by deleting the item using the method call or by assigning a new method call to that item. You can also choose to deactivate the method call rather than deleting it, as described in Activating or deactivating a method call.

To delete a method call:

  1. In the navigation area, select the method call that you want to delete.
  2. Click to mark the method call for deletion.

The font of the method call name is struck through n the navigation area (for example: newMethodCall).

  • Click to save your changes and delete the method call.

If you have marked the method call for deletion and have not yet saved the changes, you can still remove the mark for deletion.

To remove a method call’s mark for deletion:

  1. In the navigation area, locate and select the method call that is marked for deletion.
  2. Click to remove the mark for deletion.
  3. Click to save your changes.
