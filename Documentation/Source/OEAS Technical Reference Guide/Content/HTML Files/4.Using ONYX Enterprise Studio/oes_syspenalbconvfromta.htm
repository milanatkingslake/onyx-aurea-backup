EmailSendStepSmtpMode

This system parameter is used by the Email Send step component to specify how email messages are sent over the Internet for delivery. Messages can be sent from either a local or remote email server where Simple Mail Transfer Protocol (SMTP) is installed. For more information about the Email Send step component, see Email Send step component.

  • Allowed Parameter Values

local- This value specifies whether the mail server is local to (on the same machine as) the SMTP server.

server - If the mail server is not local to the SMTP server, change the value to “server.” This value enables you to connect directly to an SMTP service on a remote machine and use SMTP on that machine to send messages. After you have entered the value “server” you must enter the name or IP address of the remote email server through the system parameter EmailSendStepSmtpServer.

  • Default Parameter Value - EnableSendStepSmtpMode is installed as “local” by default.
