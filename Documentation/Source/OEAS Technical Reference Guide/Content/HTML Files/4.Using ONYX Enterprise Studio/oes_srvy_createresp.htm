Creating responses

You can create responses either by creating a response or by creating a question response.

For information on creating a question response directly, see Adding responses to questions.

To create a response in All Responses:

  1. In the navigation area, highlight All Responses and then click .
  2. In the Text box, type the text of the response.
  3. If you want the respondent to be able to select this response but also add his or her own text, select the Editable box.
  4. Click to save your changes.

When you click , an ID value appears beside ID.
