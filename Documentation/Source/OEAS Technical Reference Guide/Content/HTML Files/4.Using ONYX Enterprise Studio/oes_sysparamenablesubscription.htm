EnableSubscription

EnableSubscription supports the OEAS functionality that enables you to take transactional records offline to another data store (subscription service) so that you can access them while you are not connected to the network. When this system parameter is enabled, the OEP UI Subscription feature is active. In other words, once you have enabled EnableSubscription and have restarted the OEP-Online application, OEP UI Subscription controls are available to OEP end-users who have active subscription profiles.

  • Allowed Parameter Values

N - The OEP UI Subscription feature is not enabled.

Y - The OEP UI Subscription feature is enabled.

  • Default Parameter Value - EnableSubscription is installed as enabled (Y) by default.
