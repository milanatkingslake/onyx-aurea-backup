Activating or deactivating a publishable object

Use the following procedure to activate or deactivate a publishable object.

Deactivating a publishable object does not prevent you from publishing it. In the Publishable Objects view, all selected objects are published regardless of their active/inactve state. Setting a publishable object to inactive allows you to take advantage of the Show only active objects option when you are selecting the objects you want to publish. You might choose to deactivate your more static publishable objects to more easily select the more active publishable objects that require re-publishing on a more frequent basis.

To activate or deactivate a publishable object:

  1. In the navigation area, locate and select the publishable object you want to modify. The attributes of this publishable object appear in the edit area.
  2. Select or clear the Active check box to activate or deactivate the publishable object.
  3. Click to save your changes.
