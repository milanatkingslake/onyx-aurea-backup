Working with a script

You work with the script primarily in the edit area, where you can do the following:

  • Specify attributes of the script.
  • Open the Variables Web Page Dialog, in which you define variables.
  • Create and select script steps.

Selecting a script step opens the Edit Step area, in which you:

  • Create the step header and footer.
  • Add prompts and rules.
  • Associate method calls with the step.
  • Incorporate HTML.
  • Define variables.
