Adding new milestone categories

You create new milestone categories using the Milestone Categories node. The list of milestone categories that you create are used to populate the Milestone Category drop-down list. This list appears in the edit area when creating a new tracking code milestone for a campaign.

To add a new milestone category:

  • Select the Milestone Categories node, and click on the toolbar.

OR

Select an existing milestone category and click on the toolbar.

A new milestone action titled newMilestoneCategory<n>, followed by an asterisk (*), appears at the bottom of the milestone actions list.

  • In the edit area, define the attributes for this milestone category.
  • Milestone Category Id - This field is a non-editable field.
  • Name - Type the name you want to assign to this milestone category.
  • Active - Check or uncheck this box if you want to activate or deactivate this milestone category. A checked box indicates an active milestone action.
  1. Click to save your changes to the OEDB.
