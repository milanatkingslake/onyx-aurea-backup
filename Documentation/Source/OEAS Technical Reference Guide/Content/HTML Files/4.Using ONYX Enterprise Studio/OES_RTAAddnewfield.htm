Adding new fields

To add a new configurable field:

  1. In the navigation area, select the Fields mode option to display the available nodes.
  2. Click to display the fields associated with a particular node.
  3. Select a node or one of the fields within a node, and click on the toolbar. A new field appears in the navigation area, and the attributes for this new field display in the edit area.
  4. In the edit area, define the attributes for the new field. For descriptions of the field attributes, see Defining field attributes.
  5. Click on the toolbar to save your changes.
