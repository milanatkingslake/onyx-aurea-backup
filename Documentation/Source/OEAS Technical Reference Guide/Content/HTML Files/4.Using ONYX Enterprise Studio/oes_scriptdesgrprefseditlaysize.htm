Changing Edit Layout area sizing settings

In the Edit Layout area, step boxes begin with a fixed size, then change size to accommodate the lengths and number of prompts. However, you can specify their minimum and maximum width.

To set width limits for step boxes:

  1. To open the Edit Layout Options Web Page Dialog, on the Edit Layout area toolbar, click .
  2. To set the minimum and maximum width (in pixels) for step boxes, under Sizing, in the Minimum Step Width and Maximum Step Width boxes respectively, type an integer.

The maximum value for each box is 99999.

  1. Click to save your changes to the database.
