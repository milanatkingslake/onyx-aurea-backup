Moving responses

You can change the order in which a question’s question responses are presented to users by moving the question responses within the list of question responses in the edit area of the question.

To move a question response in a question:

  1. To move the question response, in the edit area of the question, under Responses, select the question response, and then click either Move Up or Move Down, as appropriate.
  2. Click to save your changes.
