Moving a user

There are two ways to move a user:

  • Using the Group box in the edit area. When you use this method to move a user, Security Administration does not reflect the move until you save the change.
  • Using drag-and-drop.

To move a user using the Group box in the edit area:

  1. In the navigation area, open Users under the group that the user is currently a member of and then select the user.
  2. In the edit area, in the Group box, click the down arrow and then click the group you want to move the user to.
  3. Click on the toolbar to save your changes.

To move a user using drag-and-drop:

  1. In the navigation area, open Users under the group that the user is a member of.
  2. Click the user and drag it either to the group where you want to move the user, to Users under that group, or to one of the users listed under Users in that group.
  3. Click on the toolbar to save your changes.
