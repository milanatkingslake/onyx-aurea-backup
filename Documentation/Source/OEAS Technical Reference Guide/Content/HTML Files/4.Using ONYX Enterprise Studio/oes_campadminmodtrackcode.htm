Modifying a tracking code

To modify a tracking code:

  1. In the Campaigns node, locate and select the tracking code you want to modify.
  2. In the edit area, modify the attributes for this tracking code. See Creating a new tracking code for the list of attributes you can modify.

In the Campaigns node, an asterisk(*), appears next to the modified tracking code until your changes are saved.

  1. Click to save your changes to the OEDB.
