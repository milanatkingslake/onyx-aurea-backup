Using Survey Administration

To successfully work with surveys, it is important to understand the following definitions of the components of a survey. It is particularly important to understand the difference between questions and survey questions, and between responses and question responses.

  • Survey type

A survey type is a grouping of surveys.

  • Survey

A survey is a collection of questions and the responses associated with those questions, sometimes ordered by a question hierarchy.

  • Question

A question is a prompt whose purpose is to obtain information from respondents.

A question is not associated with a survey. Questions are listed in the navigation area, under the All Questions node.

  • Survey question

A survey question is an instance of a question that is associated with a survey.

When you add a question to a survey, then you create a survey question. If you create a question within a survey, you create both a survey question and a question.

Survey questions are listed within the survey with which they are associated, under Survey Questions.

  • Response

A response is a possible answer for a question.

A response is not associated with a question. Responses are listed in the navigation area, under the All Responses node.

  • Question response

A question response is an instance of a response that is associated with a question.

When you add a response to a question, then you create a question response. If you create a response within a question, you create both a response and a question response. Question responses are listed within the question with which they are associated.

  • Question hierarchy

A question hierarchy is a structure that links question responses to subsequent survey questions, thus determining which survey questions are asked and in what order.
