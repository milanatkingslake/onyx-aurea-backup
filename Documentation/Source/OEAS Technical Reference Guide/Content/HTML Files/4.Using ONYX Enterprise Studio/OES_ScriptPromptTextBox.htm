Configuring a text box prompt

For text box prompts, you can specify the following:

  • Data type: string, datetime (sent to the script engine in UTC format), date, integer, or float
  • Minimum and maximum number of characters for the response
  • The character width of the text box

In the following procedure, all the steps except the last (in which you save changes) take place on the step’s Prompts tab.

To specify attributes for text box prompts:

  1. In the Data Type box, select the desired data type.

The default data type is string.

  1. In the Min and Max boxes respectively, set the inclusive minimum and maximum values for the following aspects of the response:
  • If the data type is string: the character length
  • If the data type is datetime or date: valid dates
  • If the data type is float or integer: valid values
  1. In the Size box, specify the width of the text box in characters.
  2. On the toolbar, click to save your changes to the database.
