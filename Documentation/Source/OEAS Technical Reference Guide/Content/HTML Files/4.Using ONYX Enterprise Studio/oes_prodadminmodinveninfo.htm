Modifying inventory information

To modify inventory information:

  1. In the navigation area, navigate to and expand the product group of the product you want to modify.
  2. Select the product you want to modify. The attributes for this product appear in the edit area.
  3. Expand the Inventory Information data cell by clicking anywhere on the cell header.
  4. In the edit area, apply your modifications to the inventory attributes for this product.
  • Stock Level - Type the number of available units for the product. This is not a required field. The valid range is 0-2,147,483,647.
  • List Price - Type the customer price for this product. This is not a required field. The valid range is 0-99999999.99.
  • Standard Cost - Type the per unit product cost. This is not a required field. The valid range is 0-99999999.99.
  • Currency Code - From the dropdown list of currency codes, select the currency for this product. This is not a required field. The default is null, or the value defined by the DefaultCurrencyCode system parameter.
  • First Ship Date - Type the earliest date that the product can be shipped. This is not a required field.
  • Obsolete Date - Type the date of obsolescence for the product. This is not a required field.
  • Product Status - From the dropdown list, select the product status for this product.
  • Orderable - Uncheck this box to disallow this product from being shipped. A checked box implies an orderable product.
  1. Click on the toolbar to save your changes.
