Configuring lookup values by field

The Lookup Values by Fields mode enables you to configure lookup values for all configurable fields defined as combo box fields in the Fields mode.

Within the Lookup Values by Fields mode, you locate and select the field for which you want to configure lookup values and configure the values specifically for that field, and any field that has been related to that field (see Relating existing fields).

  • Add lookup values to a combo box field
  • Modify a lookup value for a combo box field
  • Delete lookup values from a combo box field
  • Modify the display order of lookup values
  • Create resolution codes
  • Define lookup values for hierarchically related combo box fields
  • Define lookup values for standard incident fields
  • Define lookup value attributes
