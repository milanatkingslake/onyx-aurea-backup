Configuring a product prompt

For product prompts, you can specify the type of products in the prompt. The user can choose just one item from the list.

To specify attributes for a product prompt:

  1. On the Prompts tab, in the Extra Data box, append a value to “type=”.

The values that you can add include “customer”, “literature”, or “forecastquote”.

  1. On the toolbar, click to save your changes to the database.
