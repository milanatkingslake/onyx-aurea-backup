Creating a Survey

To create a survey, you must first create or expand a survey type.

To expand a survey type:

  • In the navigation area, click beside the Survey Types node, and then click beside the survey type that you want to expand.

To create a survey:

  • With the desired survey type expanded, select Surveys, and then click from the toolbar.

The edit area changes to appear as follows:

  1. Fill in the Name and Description boxes.
  2. If you want the survey to be available to respondents, leave the Active box checked.
  3. Click to save your changes.

When you save your changes, an ID value appears beside ID.
