Creating editable responses

An editable response presents an answer for the respondent to select, but also allows the respondent to supply his or her own answer.

A typical use of an editable response is to let the respondent specify some other response than the set of responses available for a question. For instance, in the survey example described in Using surveys: an example we ask GlobalTechnologies’ customers, “What features would you like to see added to Product (A)(B)?” Among the possible responses to this question, we provide “Other” and make the response editable. This allows respondents to describe a desired feature that is not included in the question’s available responses.

You can also create free-text questions, in which the respondent writes his or her answer in its entirety. For information on creating a free-text question, see Working with questions.

To make a response editable:

  1. In the navigation area, under All Responses, select the response.
  1. Check the Editable box.
  2. Click to save your changes.
