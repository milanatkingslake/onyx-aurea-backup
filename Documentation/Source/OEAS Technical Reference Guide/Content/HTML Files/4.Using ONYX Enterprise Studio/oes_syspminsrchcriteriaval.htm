MinSearchCriteriaValue
MinSearchCriteriaValue

Specifies the total number of characters that a user needs to input in the Quick Search window to perform a search. If you set this parameter too low, too many records are returned when a search is performed, which can impact system performance.

  • Allowed Parameter Values - The parameter value stores an integer value. The minimum parameter value is 1, and the maximum parameter value is 10.
  • Default Parameter Value - MinSearchCriteriaValue is installed with a default value of 1 - 3.
