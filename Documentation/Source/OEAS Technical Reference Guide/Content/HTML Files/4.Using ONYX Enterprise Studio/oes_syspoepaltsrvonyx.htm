OEPAltUploadServer_Onyx

OEPAltUploadServer_Onyx determines whether an alternate server has been set up to handle uploading/downloading files on the file server when Onyx authentication is in use.

  • Allowed Parameter Values - The parameter value stores the HTTP address of the alternate Upload/Download File Server.
  • Default Parameter Value - OEPAltUploadServer_Onyx is installed with a default value of NULL, which indicates that no alternate upload/download file server exists.
