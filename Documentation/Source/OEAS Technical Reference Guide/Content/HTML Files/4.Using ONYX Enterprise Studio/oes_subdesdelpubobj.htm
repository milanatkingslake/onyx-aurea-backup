Deleting a publishable object

Use caution when deleting publishable objects and consider deactivating objects rather than deleting them as there might be transactional data that references the publishable object’s domain data.

Onyx recommends that you not delete any of the default publishable objects that are created during installation of OEAS.

Use the following procedure to delete a publishable object.

To delete a publishable object:

  1. In the navigation area, locate and select the publishable object that you want to delete.
  2. Click to mark the publishable object for deletion.

The publishable object name appears with a strikethrough font in the navigation area (for example: newPublishableObject).

  1. Click to save your changes and delete the publishable object.

Deleting a publishable object that has already been published does not delete the object’s data file from the publication directory. That file will remain in the publication directory until deleted by the administrator.

If you have simply marked the publishable object for deletion and have not yet saved the changes, you can still “undelete” the publishable object by unmarking it for deletion.

Use the following procedure to unmark a publishable object marked for deletion.

To unmark a publishable object marked for deletion:

  1. In the navigation area, locate and select the publishable object that you previously marked for deletion. It is distinguished in the navigation area by the strikethrough font of its name.
  2. Click to unmark the publishable object for deletion.

The strikethrough font is removed from the name, and the publishable object is unmarked for deletion.

  1. Click to save your changes.
