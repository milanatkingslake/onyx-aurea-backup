Defining subscribable objects

Subscribable data is the transactional data that is available for subscription by OEP end-users so that it can be synchronized to OEP-Offline. To create subscribable data, you must make a subscribable object of the business object that identifies that data.

You can manage subscribable objects by:

  • Adding a subscribable object
  • Modifying a subscribable object
  • Activating or deactivating a subscribable object
  • Deleting a subscribable object
