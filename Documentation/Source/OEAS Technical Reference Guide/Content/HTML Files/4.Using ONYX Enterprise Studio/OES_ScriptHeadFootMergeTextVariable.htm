Merging text with variables

Scalar variables are useful for merging text. Before you use scalar variables, remember to define them and specify whether they are global or local to the step in which you will use them. For information on creating a scalar variable, see Working with variables.

To merge text in a text header or footer, using a variable:

  1. In the step’s Edit Step area, click the Step Details tab.
  2. In the Header or Footer text box, amidst the text, type the variable name directly, including the opening and closing braces.

For example, if the variable name is LastName, you type:

{LastName}

If the variable code and text are identical, the ‘$’ is optional.

OR

Select a variable from the Available Variables box.

OR

Add a variable from within the HTML Editor Web Page Dialog. For information on working in the HTML Editor, see Working with HTML tags in the header, footer, or prompts.

  1. To remove a variable from the text, delete it as you normally delete text.
  2. Click to save your changes to the database.
