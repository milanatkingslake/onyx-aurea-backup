Adding an OEP profile

Use the following procedure to create, and activate or deactivate an OEP profile.

To add a new OEP profile:

  1. In the navigation pane, select the OEP Profiles node.
  2. Click on the toolbar. A new OEP profile titled newProfile(n) appears in the navigation pane, followed by an asterisk (*). The edit pane appears to the right of the navigation pane.
  3. In the edit pane, specify values for the new profile's attributes.
  • Primary ID - You may edit the default Primary ID. The Primary ID must be a number.
  • Profile Name - Enter a name for the profile.  
  • Description - Enter a description for the profile.  
  • Active - Clear the check box to deactivate the profile.

Deactivating an OEP profile does not deactivate users who belong to that profile. If, however, you deactivate all OEP profiles to which one or more users belong, those users will not be able to log in to OEP. Users must belong to at least one active OEP profile.

  1. Click to save the profile.

To activate or deactivate an OEP profile:

Deactivated profiles appear in the navigation pane grayed out and in italics.

  1. In the navigation pane, select the OEP profile you want to activate or deactivate.

The attributes for this profile appear in the edit pane.

  1. In the edit area, select the Active check box to activate the profile; clear it to deactivate the profile.
  2. Click to save your changes.
