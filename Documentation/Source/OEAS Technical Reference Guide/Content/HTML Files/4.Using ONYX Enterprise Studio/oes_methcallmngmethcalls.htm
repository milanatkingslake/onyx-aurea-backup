Managing method calls

Using Method Call Designer, you can manage method calls by:

  • Adding a method call
  • Modifying a method call
  • Activating or deactivating a method call
  • Deleting a method call
  • Copying a method call

When you add or modify a method call, you assign values to its attributes. These attributes appear in the Edit Method Call area when you add a new method call or select an existing method call in the navigation area. The method call attributes are described in Method call attributes.
