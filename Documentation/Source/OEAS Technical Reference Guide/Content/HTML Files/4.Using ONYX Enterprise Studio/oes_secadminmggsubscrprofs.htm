Managing subscription profiles

A subscription profile is the enrollment of an Onyx user in a subscription service, which enables the user to use OEP-Offline. Each user has only one subscription profile for the subscription service.

For more information on subscription services, see Subscription Designer Help.

Managing subscription profiles involves the following tasks:

  • Adding and deleting a user’s subscription profile
  • Modifying a user’s subscription profile
  • Resetting a user’s subscribed data
  • Viewing and refreshing a subscription profile’s subscription rules
  • Disabling a subscription profile
  • Creating an advanced subscriber
  • Creating a proxy subscriber
