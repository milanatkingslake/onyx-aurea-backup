Configuring a country prompt

For country prompts, you can specify the following:

  • Whether the prompt appears as a list box or as a drop-down list box
  • Minimum and maximum number of responses that the user can select when the prompt is a list box
  • Whether the countries listed are sorted alphabetically in the user’s interface
  • The number of lines to display when the prompt appears as a list box

In the following procedure, all the steps except the last (in which you save changes) take place on the step’s Prompts tab.

To specify attributes for a country prompt:

  1. In the Min box, set the minimum required number of responses.

If you specify 0 in the Min box, the prompt is optional. If, however, you specify 0 and also select Required, selecting Required box overrides Min, so that the minimum number of responses is effectively 1, even though you specified 0.

  1. In the Max box, set the maximum allowed number of responses.

By default, the value in the box is 1. If you specify 1 in the Max box, the prompt is a single-select prompt, and you can also specify an output variable. For information on specifying an output variable, see To add a prompt to a step.

To specify an unlimited number of responses, delete the value in the box.

  1. To make the prompt a drop-down list box, in the Extra Data box, specify “dropdown=1” (the default value).

If you specify a value greater than 1 in Max, specifying that the prompt is a drop-down list box overrides the Max value, and the prompt is a single-select prompt by virtue of being a drop-down list box. If you specify a Min value greater than 1 and specify that the prompt is a drop-down list box, then in an end-user interface session using this script, the step will return an error and the session will not continue beyond the step.

  1. In the Size box, type the number of lines to display in the list box.

If you type dropdown=1 in the Extra Data box, the value in the Size box has no effect.

  1. On the toolbar, click to save your changes to the database.
