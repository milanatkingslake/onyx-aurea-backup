EnableMultipleTasksSupport

Determines whether the Task tab appears on the Support Incident window.

  • Allowed Parameter Values

N - The Task tab is not available in the Support Incident window.

Y - The Task tab is available in the Support Incident window, and users can enter multiple tasks for Support incidents.

  • Default Parameter Value - EnableMultipleTasksSupport is installed as enabled (Y) by default.
