Activating or deactivating a contact type

To activate a contact type:

  1. In the navigation area, locate and select the contact type you want to activate or deactivate.

The attributes for this contact type display in the edit area.

  1. In the edit area, check or uncheck the Active box to activate or deactivate the contact type. A checked box indicates an active contact type.

Once you modify a contact type attribute, the contact type appears in the navigation area with an asterisk (*) until the changes are saved. If you deactivated the contact type, it is also italicized and grayed out, but the asterisk (*) disappears once the changes are saved.

  1. Click on the toolbar to save your changes to the OEDB.

New contact types are active by default.
