Toolbar

In addition to the Script Designer toolbar, there are also the Edit Layout area toolbar and other toolbars that are specific to certain tabs in the Edit Step area. For information on these toolbars, see Working with script steps.

The following table lists the buttons on the Designer toolbar, what they do, and when they are available.

Icon Tool tip Function Availability

Add

Adds a new script group, script, or script step. The item added depends on which item is selected in the navigation area.

Always enabled.

Delete

Marks an item for deletion. The item marked for deletion depends on which item is selected in the navigation area. If you select an item that is marked for deletion and then click this icon again before saving changes, the mark for deletion is removed.

Enabled when an item is selected in the navigation area. (All deletes are physical deletes.)

Clone

Makes a copy of a script within a script group.

Enabled when a script is selected in the navigation area.

Copy

Copies a script step that is selected in either the navigation area or the Edit Layout area.

Enabled when a script step is selected.

Paste

Pastes a copy of a script step into a script.

Enabled when a script Steps node or step is selected.

Script Layout View

Opens the Script Layout Web Page Dialog, which displays the layout of the selected script.

Enabled when a script is selected.

Refresh Selected Script in OEAS

Flushes the script cache for the selected script.

Enabled when a script or a step within a script is selected.

Refresh All Scripts in OEAS

Flushes the cache for all scripts.

Always enabled.

Save

Saves to the database all changes made since the last save or since the Script Designer was started.

Enabled after any changes have been made.

Show/Hide Context Area

Shows or hides the context area.

The upper image shows the button when the context area is expanded and clicking the button will contract it.

The lower image shows the button when the context area is contracted and clicking the button will expand it.

Always enabled.

User Preferences

Makes user preferences available in the edit area for changes.

Always enabled.

Help

Opens the Script Designer Help system.

Always enabled.
