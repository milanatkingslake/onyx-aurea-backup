Configuring a multiline text box prompt

For multiline text box prompts, you can specify the following:

  • Minimum and maximum number of characters in the user’s response
  • How many rows tall the box can be

In the following procedure, all the steps except the last (in which you save changes) take place on the step’s Prompts tab.

To specify attributes for a multiline text prompt:

  1. In the Min box, set the minimum required number of characters in the response.
  2. In the Max box, set the maximum allowed number of characters in the response.
  3. In the Size box, specify how many rows tall the box can be.
  4. On the toolbar, click to save your changes to the database.
