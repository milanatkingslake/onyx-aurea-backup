Setting behavior and appearance within the Script Designer

You can specify behavior of certain elements of the Script Designer and, for each script, the appearance of elements in the Edit Layout area.

In the User Preferences area, you can specify the following behaviors in Script Designer. These settings apply no matter which script you work in.

  • Whether the navigation area and script layout always appear
  • The effect of clicking the canvas in the Edit Layout area

In the Edit Layout Options Web Page Dialog, you can specify the following for the Edit Layout area. You specify these settings when a script is selected, and the settings are specific to that script.

  • Sizing of elements
  • Information display
