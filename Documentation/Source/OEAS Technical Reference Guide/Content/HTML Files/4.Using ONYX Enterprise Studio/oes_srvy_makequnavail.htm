Making questions unavailable

You can make a question unavailable either by deleting it or making it inactive. The consequences that follow from which you choose to do include the following:

  • If you make a question inactive, you can not add that question to any more surveys.
  • If you mark for deletion a question for which there is a survey question (an instance associated with a survey), you cannot save that change.

If you make a survey question inactive that is included in a question hierarchy, the survey questions might not appear to the user in the order that you intend. To prevent this, when you make a survey question inactive, be sure to modify the question hierarchy so that the inactive survey question is no longer part of it. For more information on changing a question hierarchy, see Working with question hierarchies.

To make a question inactive:

  1. In the navigation area, under All Questions, select the question.
  2. In the edit area, remove the checkmark from the Active box.
  3. Click to save your changes.

You can also make a question inactive by selecting a survey question that is an instance of that question and, in the survey question’s edit area, removing the checkmark from the Active box.

  1. In the navigation area, under All Questions, select the question.

  3. Click to save your changes.

To delete a survey question:

  1. In the navigation area, expand the survey with which the survey question is associated.
  2. Under Survey Questions, select the question.
  3. From the toolbar, click .
  4. Click to save your changes.
