Working with questions

After creating the survey, you create new questions and/or add existing questions to it, as well as:

  • Adding new or existing responses
  • Specifying whether it’s permissible to answer the question with a single response, with multiple responses, or with free text

After adding a question, you can:

  • Edit the question.

For information on editing the question, see Editing questions.

  • Move a survey question in the survey’s list of survey questions.

For information on moving survey questions, see Moving survey questions.

  • Add responses to or remove them from the question.

For information on adding and removing responses, see Working with responses.

  • If you create a question hierarchy for the survey, define a survey question’s place in the hierarchy.

For information on structuring the question hierarchy, see Working with question hierarchies.

  • Change the question’s availability.

For information on making questions available or unavailable, see Making questions unavailable.
