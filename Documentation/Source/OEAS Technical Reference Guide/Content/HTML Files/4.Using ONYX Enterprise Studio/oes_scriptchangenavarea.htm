Changing the navigation area size

You can resize the navigation area or hide it altogether.

To change the navigation area size:

  • Click and drag the right-hand border of the navigation area.

For information on hiding the navigation area, see Hiding and showing the navigation area.

To reveal the navigation area when it is hidden:

  1. To change the edit area to the User Preferences area, click .
  2. In the User Preferences area, unselect the Auto-Hide check box, and then click OK.

You can also reveal the navigation area using just the mouse: On the left side of the OES window, pass the mouse over the vertical bar to which the navigation area shrank.
