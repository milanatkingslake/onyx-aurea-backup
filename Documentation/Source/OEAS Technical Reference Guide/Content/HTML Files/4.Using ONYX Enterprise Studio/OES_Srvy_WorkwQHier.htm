Working with question hierarchies

Question hierarchies add structure to surveys by defining which survey question follows another. Using question hierarchies, you can create different branches in the survey. In the example given in Surveys, the question hierarchy splits the survey according to whether customers use Product A or Product B.

Any survey questions for which you don’t define a question hierarchy appear in the survey in the same order that they appear under the survey in the navigation area.

Question hierarchies can include free-text questions and questions that have multiple responses. For questions with multiple responses and for free-text questions, Survey Administration automatically specifies “(all responses)” in the Response box. The survey question that you select as the next question is the next question regardless of the respondent’s answer.
