Activating or deactivating a lookup value system wide

From within the Lookup Values — All node, you can activate or deactivate a lookup value system wide.

Once you activate or deactivate the lookup value in this mode, this lookup value is activated or deactivated for all combo box fields to which it is associated.

To activate or deactivate a lookup value system wide:

  1. In the navigation area, select the Lookup Value - All mode to display the existing lookup values.
  2. Select the lookup value you want to modify. The attributes for this lookup value are displayed in the edit area.
  3. In the edit area, check or uncheck the Active box to activate or deactivate the lookup value. A checked box indicates an active lookup value. Once you modify a lookup value property, the lookup value appears in the navigation area with an asterisk (*) until the changes are saved. If the lookup value was deactivated, it is also italicized and grayed out, but the asterisk (*) disappears once the changes are saved
  4. Click on the toolbar to save your changes.
