Using Security Administration

Security Administration enables you to:

  • Add and change the attributes of Onyx users.
  • Define users as offline users by adding a subscription profile for them.
  • Create and populate profiles for OEP users
  • Add roles, which are collections of users defined by common access permissions.
  • Manage user membership in roles.
  • Create resources for customizations.
  • Secure and unsecure resources.
  • Add users and roles to a resource's access-permission list. Adding a user or role to a resource's access-permission list gives the user or role access to that resource.
  • Deny access to users that currently have access to a resource.

The procedures to complete these tasks are divided into the following sections:

  • Managing groups
  • Managing users
  • Managing roles
  • Managing resources
  • Managing OEP profiles
