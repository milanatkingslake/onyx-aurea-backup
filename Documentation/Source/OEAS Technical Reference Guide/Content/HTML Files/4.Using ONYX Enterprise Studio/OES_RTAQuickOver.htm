Quick overview of Reference Table Administration

The user interface of Reference Table Administration is divided into four areas:

  • Context area
  • Toolbar
  • Navigation area
  • Edit area

Context area - Lists the name of the administration tool, the application, and the site being maintained.

Toolbar - Appears under the context area and contains the toolbar buttons that control the tasks related to Reference Table Administration. The following table lists the toolbar buttons.

Icon Tool tip Function Context

Add

The Add button is used to add a new field, add a new lookup value, or to associate a lookup value with a combo box field.

Enabled when you have selected the Fields mode (radio button) and then selected a node.

For the Lookup Value by Field mode, it is enabled when you open the navigation tree, and then select a field.

For the Lookup Value - All mode, it is enabled when you select this mode.

Delete

Marks lookup values for deletion. The item is not deleted until you click Save.

Enabled in Lookup Value by Field mode when a lookup value is selected.

For the Lookup Value - All mode, it is enabled when you select a lookup value.

The Delete function is not enabled in Fields mode except for new fields that have not been saved.

Move up

Moves a lookup value up one position in the navigation area, changing the order that the lookup value appears in both the Onyx Employee Portal and in Reference Table Administration.

Enabled only when a lookup value is selected in Lookup Value by Field mode.

Move down

Moves a lookup value down one position in the navigation area, changing the order that the lookup value appears in both the Onyx Employee Portal and in Reference Table Administration.

Enabled only when a lookup value is selected in Lookup Value by Field mode.

Save

Saves all modifications.

Enabled after any changes have been made.

Help

Launches Help.

Enabled at all times.

Navigation area - Depending on which of three modes that you select, the navigation area displays either reference fields alone; the fields and, under each field, its associated lookup values; or a list of all the lookup values.

Edit area - In the edit area, you can view and modify field attributes, define relationships between fields and lookup values, and define lookup values. The options available for configuration in the edit area are controlled by the mode you select in the navigation area. For descriptions of the attributes in the edit area for each mode, refer to the following:

  • Fields mode, see Defining field attributes.
  • Lookup Values by Fields mode, see Defining lookup value attributes.
  • Lookup Value — All mode, see Defining lookup value attributes.
