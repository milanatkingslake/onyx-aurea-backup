Adding lookup values to a combo box field

You can add both new and existing lookup values to a combo box field.

To add a value to a combo field:

  1. In the navigation area, select the Lookup Value by Fields mode option to display the available nodes.
  2. Click to display the fields associated with a particular node. The node expands and displays its associated fields.
  3. Click in front of the field to which you want to add a lookup value. The field node expands and displays its existing lookup values.
  4. Select either the field node or an existing lookup value within the field node and click . A new lookup value named NewLookupValue<n> appears at the bottom of the lookup values list in the navigation area, followed by an asterisk (*).
  5. In the edit area, define the attributes for the lookup value, either creating a new lookup value or adding an existing lookup value (note that entries are case-sensitive). For descriptions of the lookup value attributes, see Defining lookup value attributes.
  • To create a new lookup value, type the lookup value in the Lookup Value box and define the remaining attributes of Extra Data and Char Code, and Active. When creating a lookup value, note that you cannot enter a duplicate name that differs only by case; it will not be supported in the Lookup Value box.
  • To add an exiting lookup value, type the first character of the lookup value in the Lookup Value box. Existing lookup values that begin with that character are displayed in the Existing Lookup Values box. Select the lookup value you want from the Existing Lookup Values box, and it is displayed in then displayed in the Lookup Value box.
  1. If you are adding multiple lookup values to a field, click Add Lookup Value. The lookup value you just defined is displayed with the field in the navigation area and a new lookup value named NewLookupValue<n> appears at the bottom of the lookup values list in the navigation area, followed by an asterisk (*). The edit area is cleared so that you can define the attributes for the new lookup value.
  2. When you have finished adding new lookup values, click on the toolbar to save your changes.

You can change the position of how these lookup values appear in the user interface by clicking or to move them up or down one position at a time. Click on the toolbar to save your changes.
