IndivFavoriteContactTypeId

IndivFavoriteContactTypeId specifies the contact type ID from the individual contact category that is intended as the “Favorite” contact type for individuals. OEP uses this system parameter to support the Home Page functionality for “Favorite Individuals.”

  • Allowed Parameter Values - The parameter value stores an integer value.
  • Default Parameter Value - The parameter value is system-generated and corresponds to the “Favorite” contact type for the contact category for individuals.

The parameter value must be a valid contact type ID from the individual contact category for this system parameter to properly support the “Favorite Individuals” view in the OEP Home Page.
