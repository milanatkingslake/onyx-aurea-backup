Adding a step

When you add a step to a script, you can also set the step’s attributes and create its header and footer.

To add a step to a script:

  1. Do one of the following:
  • Right-click the Edit Layout area canvas, and select Add Step from the context menu that appears. (The new step appears at the cursor’s location.)
  • In the navigation area, select the Steps node under the script (or an individual step) and, in the Script Designer toolbar, click .
  • In the Edit Layout area toolbar, click .
  1. On the Step Details tab of the Edit Step area, specify the following attributes:
  • Name - The name of the new step.
  • Description - A description of the step. The description should be meaningful to anyone who works with the script.
  1. Do any of the following that the script design requires:
  • Create a header or footer. For information on creating a header or footer for the step, see Working with headers and footers.
  • Create one or more prompts. For information on creating a prompt, see Working with prompts and responses.
  • Create one or more rules. For information on creating a rule, see Working with rules.
  • Associate one or more method calls with the step. For information on associating a method call with the step, see Working with method calls.
  1. Click to save your changes to the database.
