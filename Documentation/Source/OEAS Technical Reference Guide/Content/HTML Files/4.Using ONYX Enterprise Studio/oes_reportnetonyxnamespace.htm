ReportNetOnyxNamespace

ReportNetOnyxNamespace specifies the security namespace in Cognos used for OEP-Cognos single sign on functionality.

  • Allowed Parameter Values - a string containing a valid Cognos security namespace created for Onyx Security Provider. For more information on Onyx Security Provider, see the Cognos Integration Module documentation.
  • Default Parameter Value - (empty)
