Quick overview of the Subscription Designer

The user interface of Subscription Designer is divided into four areas:

  • Context area
  • Toolbar
  • Navigation area
  • Edit area

Context area - The context area lists the tool name and the name of the application.

Toolbar - The toolbar appears under the context area and contains the toolbar buttons that support the Subscription Designer tasks.

The following toolbar buttons are available:

Icon Tool tip Function Availability

Add

Adds a new publishable object, subscribable object, subscription relationship, or subscription role. The item added depends on what is selected in the navigation area.

Enabled when you select a node in the navigation area for which an additional item is available.

Delete

Used to delete a publishable object, subscribable object, subscription relationship, or subscription role. When clicked, the item selected in the navigation area is marked for deletion. Can be used to unmark an item previously marked for deletion when that deletion has not yet been saved.

Enabled when you select a publishable object, subscribable object, subscription relationship, or subscription role in the navigation area.

Save

Used to save all new and modified items. Removes all dirty flags.

Enabled after any changes have been made.

Show/Hide Context Area

Shows or hides the context area.

 

The upper image shows the button that is displayed when the context area is expanded. Clicking the button will contract it.

 

The lower image shows the button that is displayed when the context area is contracted. Clicking the button will expand it.

Always enabled.

Help

Launches Help.

Always enabled.

Navigation area - The navigation area is where you begin when working with publishable objects, subscribable objects, and subscription services. To view or modify any of these items, navigate to and select it in the navigation area, then make any changes in the edit area.

There are two first-level nodes in the navigation area: Publishable Objects and Subscription Services. You can expand or collapse an item in the navigation area by clicking or beside it.

  • Publishable Objects node - This node contains all the domain objects that are published to the offline data store.
  • Subscription Services node - This node contains a node for the OEP-Offline subscription service. The OEP-Offline node contains the subscribable objects of that subscription service, and each subscribable object node can be expanded to reveal its subscription relationships and any roles associated with those relationships.

Edit area - In the edit area, you view and modify the item that you have created or selected in the navigation area. The appearance of the edit area changes according to what is selected in the navigation area.
