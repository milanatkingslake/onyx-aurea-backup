Edit Layout area

In the edit area, the Edit Layout area is a graphical interface with the current script. You can use the Edit Layout area to:

  • Insert, remove, and change the rules between script steps.
  • View information about prompts and rules.

The following illustration shows the Edit Layout area after several steps and their prompts and rules have been created in a script:

In the illustration, note the following:

  • Canvas (the blank area on which step and other elements are placed)
  • Step boxes, which represent steps and contain the step name

In Edit Layout Options, you can specify that the step boxes also show the prompt text.

  • Rule lines, with arrows indicating the direction to the destination step
  • Boxes with information about rules

In Edit Layout Options, you specify whether rule information boxes are displayed and what information appears in them.

  • Toolbar buttons
  • Pan bar, which appears on any side that the cursor approaches and which you click to move your view of the canvas and script

The canvas context menu (which you open by right-clicking the canvas) contains the following commands:

  • Add Step
  • Zoom In
  • Zoom Out
  • Zoom To Fit

When the cursor is over a step, the context menu includes the Add Prompt and Add Rule commands.

Following are some of the techniques with which you can manage the script on the canvas:

  • Drag step boxes on the canvas.
  • Drag rules to new destination steps.
  • To define an area to zoom into, hold down the CONTROL key, and click and drag the cursor.
  • To move the canvas in any direction, hold down the ALT key, and click and drag the cursor.
