Modifying a tracking code type

To modify a tracking code type:

  1. In the Tracking Code Types node, locate and select the tracking code type you want to modify.

The attributes for this tracking code type display in the edit area.

  1. In the edit area, modify the tracking code type's attributes. See Creating a new tracking code type for the list of attributes in the edit area.

In the navigation area, an asterisk (*) appear next to the modified tracking code type until the changes are saved.

  1. Click to save your changes to the OEDB.
