Viewing a script step

The Edit Layout area provides a graphical view of steps and rules within the entire script.

In the Edit Layout area, you can interact with steps and rules as follows:

  • Visually rearrange steps and rules in the area by dragging a step.

When you drag a step, any rule attached to the step moves with it. When steps and rules overlap, steps always cover rules. A larger step can hide a smaller step as well as its rules.

  • Bring a hidden step to the front by selecting it in the navigation area.

Any hidden rules remain hidden.

  • Click a step to open the Edit Step area for the step.
  • Click a rule to open the Rules tab in the Edit Step area for the step in which the rule was created.
