Working with headers and footers

Header and footer text is rendered with HTML in a script session. In both headers and footers, you can use variables to merge text and HTML tags to improve the visual impact and clarity of the text. This section includes information on merging text with variables. For information on working with HTML tags, see Working with HTML tags in the header, footer, or prompts.

To add text to a header or footer:

  1. Type the text in the Header or Footer text box, as appropriate, including variable names or HTML tags.
  • To add text formatting, you can either type the HTML tags directly into the text box or use the HTML Editor Web Page Dialog to format the text. For more information on using the HTML Editor, see Working with HTML tags in the header, footer, or prompts.
  • To use a variable to merge text, you can either type the variable name (enclosed with braces) directly into the text box or select the variable from the box that shows <select a variable>. For more information, see Merging text with variables.
  1. Click to save your changes to the database.
