Adding a new tracking code milestone for a tracking code

Creating a new tracking code milestone involves selecting a milestone category and milestone action to associate with a tracking code.

To add a new tracking code milestone for a tracking code:

  1. In a tracking code, select the Tracking Code Milestones node, and click on the toolbar.

OR

Select an existing tracking code milestone from the tracking code milestone list, and click on the toolbar.

A new milestone category/milestone action titled (empty category)/(empty action)appears at the bottom of the tracking code milestone list.  

  • In the edit area, define the attributes for this tracking code milestone.
  • Milestone Category - Select a milestone category from the drop-down list. This list contains all milestone categories defined in the Milestone Categories node.
  • Milestone Action - Select a milestone action from the drop-down list. This list contains all milestone actions defined in the Milestone Actions node.
  • Capacity - This field exists for product growth and customizations.
  • Maximum Allowable - This field exists for product growth and customizations.
  • Active - Check or uncheck this box if you want to activate or deactivate this tracking code milestone.
  1. Click to save your changes to the OEDB.

You modify individual milestone actions and milestone categories through the Milestone Actions node and Milestone Categories node. Changes to individual milestone actions/milestone categories will be reflected in the tracking code milestones that currently use them.
