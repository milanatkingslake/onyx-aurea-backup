Edit area

In the edit area, you view and modify a script group, a script, and the script’s elements. The appearance of the edit area changes according to what is selected in the navigation area.

For illustrations of the edit area when one of the following is selected, see the appropriate section.

  • Script group (Working with a script group)
  • Script (Working with a script)
  • Script step (Working with script steps)

When you select the Steps node within a script in the navigation area, the Edit Layout area occupies the entire edit area. Groups nodes and Scripts nodes have no attributes to modify, so when they are selected, the edit area is blank.
