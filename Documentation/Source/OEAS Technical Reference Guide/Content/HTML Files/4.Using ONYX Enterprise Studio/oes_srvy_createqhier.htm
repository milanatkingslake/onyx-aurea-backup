Creating question hierarchies

It’s a good idea to diagram your survey’s question hierarchy before creating it.

To create a question hierarchy:

  1. In the navigation area, expand the survey for which you want to create a question hierarchy.
  1. Select Question Hierarchy.
  1. In the edit area, click the Questions box and select the survey question that you want to be the first question in the survey.
  2. In the Responses box, select the question response to which you want to link the next survey question.

In the Responses box, free-text questions and questions that allow multiple answers are given the response “(all responses).” This prevents such questions from interrupting question hierarchies.

  1. In the Next Question box, select the survey question that should follow the question response that you just selected.
  2. Click Add Link.

With the questions used in the example, the edit area now looks like the following. Note that the Hierarchy summaryarea updates the current hierarchy each time you add a link.

  1. To create the rest of the question hierarchy, repeat steps 3 through 6 for each combination of survey question, question response, and next survey question.
  2. Click to save your changes.
