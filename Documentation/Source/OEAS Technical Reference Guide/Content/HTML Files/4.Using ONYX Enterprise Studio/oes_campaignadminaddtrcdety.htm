Adding a tracking code of a particular tracking code type to a campaign

To add a tracking code of a particular tracking code type to a campaign:

  • In the Campaigns node, either:
  • Add a new tracking code that is of the tracking code type you want to associate with the campaign. See Creating a new tracking code for further information.

OR

  • Modify an existing tracking code within the campaign and change its tracking code type to the type that you want to associate with the campaign. See Creating a new tracking code type for further information.

The tracking code type is now associated with the campaign. In the navigation area, the tracking code type is displayed under the appropriate campaign with the new or modified tracking code listed under it.
