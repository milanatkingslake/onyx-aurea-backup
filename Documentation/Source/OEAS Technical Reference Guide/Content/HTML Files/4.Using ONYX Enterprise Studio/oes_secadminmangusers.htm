Managing users

In Security Administration, a user is an Onyx user account. When an individual logs on, he or she either does so with a Windows domain account that is mapped to an Onyx user account or logs on directly to the Onyx user account. Users are pivotal to access-permissions managementâ€”you grant access permissions to users or to roles that users are members of. See Managing roles for further details.

Managing users involves the following tasks:

  • Adding a user
  • Activating or deactivating a user
  • Modifying a user
  • Cloning a user
  • Moving a user
  • Viewing only active users and groups
