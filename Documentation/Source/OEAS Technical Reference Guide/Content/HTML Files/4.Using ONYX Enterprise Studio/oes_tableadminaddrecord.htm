Adding a new record

To add a new record:

  1. Click on the toolbar.

A new row is added at the bottom of the table (all fields are editable except date fields).

  1. In the newly created row, type field values for all non-application defined fields.
  2. The Active field requires a valid value. Type the value that corresponds to the status you want for the record:
  • 0 - deactivates the record making it unavailable to end users
  • 1 - activates the record making it available to end users
  1. Click on the toolbar to save your changes.

In addition to using the Save button, you can press the tab key in the last cell of a newly created row to create another row at the bottom of the table.
