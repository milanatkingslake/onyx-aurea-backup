Working with script steps

To create the script steps and their elements, such as prompts and rules, use the Edit Step area.

When a single step is selected, the edit area is divided into the Edit Step area and the Edit Layout area. The Step Details, Prompts, Rules, and Method Calls tabs in the Edit Step area are where you do much of the work associated with steps:

  • Specifying the step’s attributes (Step Detail)
  • Adding and modifying step elements (Prompts, Rules, and Method Calls)
