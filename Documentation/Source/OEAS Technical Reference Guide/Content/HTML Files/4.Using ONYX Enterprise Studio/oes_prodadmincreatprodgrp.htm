Creating a product group

To create a new product group:

  1. There are several ways to create a product group in the navigation area:
  • Select the Products and Literature node, and click on the toolbar. A new product group titled newGroup<n> appears in the navigation area, followed by an asterisk (*).

OR

  • Locate and expand the product group to which you want to add a new sub-product group, select the existing Groups node, and click . A new product group titled newGroup<n> appears under the selected Groups node, followed by an asterisk (*).
  1. In the edit area, define the attributes for this product group:
  • Group Name - Type the name for this product group. The default is newGroup<n>.
  • Parent Group - Select the parent group that you want for this new product group, or accept the default. Choose the blank entry in the list if this product group is to be a top-level group.
  • Active Group - A checked box implies an active product group. The default is active (checked). Uncheck this box if you want to disable this product group.

When you deactivate a product group, that product group and its underlying products are no longer available to areas of the end-user interface where standard products are returned for inclusion in customer product, literature order, and forecast/quote records.

The following attribute is automatically assigned a value and is read only:

  • Group Id - A unique ID assigned by the database during a save.
  • Click on the toolbar to save your changes.

You must save a new product group before you will be able to add a new sub- product group within it.
