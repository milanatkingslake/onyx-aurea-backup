User interface

The user interface of Assignment Administration is divided into four areas:

  • Context area: Lists the name of the application and the site being maintained.
  • Toolbar: Contains the toolbar buttons that control the tasks related to Assignment Administration.
  • Navigation area: Lists territories and assignment sets. To view or modify any of these items, navigate to and select it in the navigation area, then make any desired changes in the edit area.  
  • Edit area: Lists attributes of the item selected in the navigation area. You can view and modify the item that you have created or selected in the navigation area. The appearance of the edit area changes according to what is selected in the navigation area.
