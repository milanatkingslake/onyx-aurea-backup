Activating or deactivating a user

To remove a user account, you deactivate it rather than delete it. There is no functionality to delete a user.

To activate or deactivate a user:

  1. If the user that you deactivate is subscribed to and has Onyx data synchronized, deactivate the user’s subscription profile.

For information on deactivating the user’s subscription profile, see Disabling the subscription profile.

  1. In the navigation area, select the user you want to activate or deactivate.
  2. The attributes for this user appear in the edit area.
  3. In the edit area, check or uncheck the Active box to activate or deactivate the user.

A checked box indicates an active resource.

  1. Click on the toolbar to save your changes.
