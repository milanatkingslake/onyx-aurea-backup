Viewing only active users and groups

The Show Only Active Users and Groups checkbox is at the top of the tree in the navigation area. This checkbox toggles between showing and hiding inactive users and groups. If the checkbox is selected and you cannot find a user or group in the navigation tree, the user or group is probably marked inactive.

For information on activating or deactivating a user or group, see Adding a user or Adding a group.

To view only users and groups that are active:

  • In the navigation area, check the Show Only Active Users and Groups checkbox.
