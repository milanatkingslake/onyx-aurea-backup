Linking assignment sets to conditions

Assignment sets are linked to conditions of the same object type and they are reusable throughout the Assignment Tree.

To link an assignment set to a condition:

  1. In the navigation area, expand the appropriate condition.
  2. Select the Linked Assignment Sets node or select an existing assignment set.

In the edit area, the currently linked assignment sets appear in the upper box and the available assignments sets appear in the lower box.

  1. In the Available Assignment Sets box, select an assignment set and then click the Add button.

An asterisk (*) appears next to the assignment set name in the navigation area. The assignment set is linked to the condition. The assignment moves into the linked assignment sets box in the edit area.

You can change the processing order for evaluation of assignment sets by using the Move Up and Move Down buttons. Processing order is important for incidents. The first assignment in the list gets assigned to the incident. Use the Remove button to unlink the assignment set from the condition.
