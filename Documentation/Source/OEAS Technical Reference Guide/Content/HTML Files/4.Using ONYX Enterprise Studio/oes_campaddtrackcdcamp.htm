Creating a new tracking code

To create a new tracking code:

  1. There are three ways to create a new tracking code:
  • Select the Tracking Codesnode (within the appropriate campaign) that is of the desired tracking code type, and click .

OR

  • Select an existing tracking code (within the appropriate campaign), and click .

OR

  • select the Tracking Code Types node of the appropriate campaign. In the edit area, select the tracking code type for the new tracking code from the drop-down box. Click Add Tracking Code.

A new tracking code titled newTrackingCode<n> appears at the bottom of the tracking codes list, followed by an asterisk (*).

  • In the edit area, define the attributes for this tracking code.
  • Id - This field is non-editable. It remains empty until the tracking code is saved.
  • Code - Type the code you want to assign to this tracking code. After the tracking code is saved, this field is not editable. (Each code is unique and cannot be reused.)
  • Name - Type the name you want to assign to this tracking code.
  • Description - Type a description of this tracking code.
  • Type - Select the tracking code type from the Type box.
  • Start Date - Enter the start date for this tracking code in the format specified in your Windows Regional Options window, which you access through your Windows Control Panel.

The date entered is for informational purposes only. It does not affect how or when users assign a tracking code to an incident. However, if desired, you can write a custom step component that enforces a custom business rule using this date.

  • End Date - Enter the end date for this tracking code in the format specified in your Windows Regional Options window which you access through your Windows Control Panel.
  • Active - Uncheck this box if you want to deactivate this tracking code.
  • Click to save your changes to the OEDB.
