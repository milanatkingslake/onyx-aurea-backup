Quick overview of the System Parameter Administration tool

Overview of the System Parameter Administration tool

The user interface of System Parameter Administration is divided into five main areas:

  • Context area
  • Toolbar
  • Navigation area
  • Edit area
  • Details area

Context area - The context area lists the name of the application and the site being maintained.

Toolbar - The toolbar appears under the context area and contains the toolbar buttons that control the tasks related to System Parameter Administration.

The following toolbar buttons are available:

Icon Tool tip Function Context

Save

Saves all modifications.

Enabled after any changes have been made.

Help

Launches Help.

Enabled at all times.

Navigation area - Contains the Parameter Code column that lists the system parameters available for modification.

Edit area - Contains the Parameter Value column that displays the current value, and allows editing of the value when the system parameter is selected.

Details area - Displays additional read-only data for the selected system parameter, including a description of the system parameter's function. For descriptions of the attributes in the Details area, see Modifying system parameters.
