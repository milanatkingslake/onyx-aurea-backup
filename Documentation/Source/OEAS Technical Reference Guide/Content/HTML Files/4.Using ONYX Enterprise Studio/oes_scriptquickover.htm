Quick overview of the Script Designer

The user interface of Script Designer is divided into four areas:

  • Context area
  • Toolbar
  • Navigation area
  • Edit area

In addition, secondary Web Page Dialogs provide the user interfaces for:

  • Creating, defining, or modifying variables, responses, and method calls.
  • Adding Hypertext Markup Language (HTML) tags to headers, footers, and prompts.

Buttons available in the Edit Script and Edit Step areas launch the dialogs. For more information about using them, see the following sections:

  • Variables (Working with variables)
  • Prompt Responses (Working with prompts and responses)
  • HTML Editor (Working with headers and footers)
  • Layout Options (Changing Edit Layout area sizing settings and Changing Edit Layout area information display settings)
  • Method Call Bindings (Working with method calls)
  • Method Call Binding Builder (Working with method calls)
