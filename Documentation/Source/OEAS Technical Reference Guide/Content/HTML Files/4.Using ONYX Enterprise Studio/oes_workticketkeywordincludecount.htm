WorkTicketKeywordIncludeCount
WorkTicketKeywordIncludeCount

Specifies the number of times a tag must be associated with a work ticket record before it appears as a retrievable value in the tags list for work tickets.

Allowed Parameter Values - The parameter value stores an integer value.

Default Parameter Value - WorkTicketKeywordIncludeCount is installed with a default value of 10. (No maximum; minimum of 10.) If upgrading from QA Incidents, the default value is the value previously indicated for QAKeyWordIncludeCount.
