Creating a standard product

You create a product by entering data into the required general product fields. Optionally, you can modify the inventory information and warranty information for a product and add details to any user-defined fields configured for products on your application.

Modifying any information on the Literature Information data cell will cause the product to become a literature product. Do not add or modify literature information for the product unless you intend for the product to be a literature product.

To create a standard product:

  1. In the navigation area, navigate to and expand the product group to which you want to add the new product.
  2. Select either the Products node under the product group you have expanded, or any one of the existing products within the product group, and click on the toolbar.
  3. In the edit area, define the general attributes for this product.
  • Product Number - Type a unique product number for the product. The default is newProduct<n>.
  • Description - Type the description or name for this product.
  • Incident Product - From the dropdown list select a related incident product. This is not a required field.
  • Alternate Product - Type the alternate product number. This is not a required field.
  • Parent Group - From the dropdown list of product group names and IDs, select the parent product group for this product or accept the default.
  • Active Product - A checked box implies an active product. The default is active (checked). Uncheck this box if you want to deactivate this product.

If you deactivate a product, that product is no longer available to the areas of the application where products are returned for inclusion in customer product, literature order, and forecast/quote records.

  1. Click on the toolbar to save your changes.
