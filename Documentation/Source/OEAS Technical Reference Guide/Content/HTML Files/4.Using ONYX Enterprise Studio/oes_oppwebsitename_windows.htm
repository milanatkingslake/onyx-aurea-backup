OPPWebSiteName_Windows

OPPWebSiteName_Windows specifies the Onyx Partner Portal (OPP) web site name that implements Windows authentication. This parameter is used for Channel Manager Integration with Onyx Employee Portal.

  • Allowed Parameter Values - a string containing the name of the OPP site that implements Windows authentication.
  • Default Parameter Value - (empty)
