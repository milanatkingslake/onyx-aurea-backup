Modifying question hierarchies

You change the structure of question hierarchies by removing and/or adding links between question responses and next-questions.

For example, to remove the multiple-answer survey question “What features would you like to see in Product A?” from the sample question hierarchy (see Using surveys: an example), you would:

  • Remove the link between the previous survey question’s responses “Centrally” and “In multiple locations” and the multiple-answer survey question “What features would you like to see in Product A?”.
  • For the survey question “What features would you like to see in Product A?”, remove the link question response “(all responses)” and the next survey question in the hierarchy, “In which markets do you use Product A?”
  • Add a link between the question responses “Centrally” and “In multiple locations” and the survey question “In which markets do you use Product A?”

To add the new link, use the same procedure with which you originally created the links.

To remove a link between a question response and the next survey question in the question hierarchy:

  1. Under Hierarchy summary, select the link that you want to break.
  2. Click Remove Link.

In the Hierarchy Summary, the link description is crossed out.

  1. Click to save your changes.
