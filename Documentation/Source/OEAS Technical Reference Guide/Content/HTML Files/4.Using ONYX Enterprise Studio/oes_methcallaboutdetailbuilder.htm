Modifying a method call detail

To modify a method call detail:

  1. In the navigation area, select the method call detail that you want to modify.
  2. In the edit area, make any changes to the method call detail’s attributes.

The method call detail attributes are described in Method call detail attributes.

  • Click to save your changes.

If your modifications include changing the Element Path, ID Path, or Description Path attributes, see About the method call detail helpers.
