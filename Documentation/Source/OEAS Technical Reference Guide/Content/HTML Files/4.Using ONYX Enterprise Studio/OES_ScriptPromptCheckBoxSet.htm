Configuring a check box set prompt

For check box set prompts, you can specify the following:

  • Whether available responses are fixed or are determined by a list variable and method call
  • A fixed set of possible responses
  • Whether responses are sorted alphabetically in the client user interface

When you localize a script, the sorting in certain Unicode language sets that takes place in the database can produce incorrect sorting in the client user interface.

  • Minimum and maximum number of responses that the user can select
  • Whether the user can provide additional text to any of the responses

In the following procedure, all the steps except the last (in which you save changes) take place on the step’s Prompts tab.

To specify attributes for check box set prompts:

  1. Under Response Options:
  • To create a fixed set of responses:
  • Select Fixed.
  • Click Responses to open the Prompt Responses Web Page Dialog, and then add responses.
  • For procedures for adding responses, see Adding, deleting, and changing the order of fixed responses.
  • To obtain the set of responses from a method call:
  • Select Variable.
  • Select a list variable either from the box beside Variable or, to select the variable from or define it in the Variables Web Page Dialog, click ....
  • The list variable that you use to populate a set of responses must be populated by a method call. For information on working with method calls, see Method calls and Working with method calls.
  1. To sort the responses alphabetically in the user’s interface, select Sort In UI.
  2. In the Min box, set the minimum required number of responses.

If you specify 0 in the Min box, the prompt is in effect optional. If, however, you specify 0 and also select Required, selecting Required overrides Min, so that the minimum number of responses is effectively 1, even though you specified 0.

  1. In the Max box, set the maximum allowed number of responses.

To specify an unlimited number of responses, delete the value in the Max box.

If the prompt is a single-select prompt (1 is specified in the Max box), you can specify an output variable in the Output Variable box. For information on specifying an output variable, see the procedure To add a prompt to a step:.

By default, the value in the box is 1.

  1. On the toolbar, click to save your changes to the database.
