Moving a group

There are two ways to move a group:

  • Using the Group box in the edit area

With this method, the move is not reflected in the navigation area until you save your changes by clicking .

  • Using drag-and-drop

To move a group using the Group box in the edit area:

  1. In the navigation area, select the group.
  2. From the Group box in the edit area, click the down arrow and then click the group you want to move the group to.
  3. Click to save your changes.

To move a group using drag-and-drop:

  1. In the navigation area, click the group you want to move and drag it to the new group.
  2. Click on the toolbar to save your changes.
