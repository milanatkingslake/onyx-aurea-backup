Deleting a prompt

To delete a prompt from a step:

  1. On the Prompts tab, in the Prompts box, highlight the prompt that you want to delete.
  2. To mark the prompt for deletion, on the Prompts toolbar, click .

If the following prompt in the list has not already been marked for deletion, the highlight moves to that prompt.

  1. To remove the mark for deletion from a prompt, highlight the prompt and then, on the Prompts toolbar, click .
  2. On the toolbar, click to save your changes to the database.
