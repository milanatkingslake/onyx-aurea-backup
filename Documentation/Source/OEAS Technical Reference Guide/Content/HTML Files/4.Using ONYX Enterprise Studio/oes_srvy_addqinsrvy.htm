Adding questions within a survey

You can add questions within a survey by:

  • Creating a new question
  • Adding an existing question

For information on searching for a question, see Adding an existing question.

To create a new question within a survey:

  1. In the navigation area, expand the survey.
  2. Select Survey Questions and then click .

The default question type is free text.

  1. In the edit area, type the text of the question in the Text box.
  2. In the Question Type box, select Single, Multiple, or Free Text.
  • Single specifies that the respondent can pick only one of the available question responses.
  • Multiple allows the respondent to pick more than one of the available question responses.
  • Free Text allows the respondent to write his or her own response.
  1. To save your changes, click .

The question is ready to add responses. For information on adding responses to the question, see Working with responses.
