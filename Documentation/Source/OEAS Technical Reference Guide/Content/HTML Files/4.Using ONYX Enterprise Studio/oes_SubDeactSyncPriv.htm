Deactivating and re-activating synchronization

When you deactivate a user’s synchronization for a subscription service, synchronization between the staging server data store and the OEDB is turned off. The user can still, however, subscribe to additional records and continue to work with records previously taken offline.

To deactivate synchronization for the subscription profile:

  1. In the Security Administration navigation area, select the subscription profile.
  2. In the edit area, clear the selection for the Sync Active attribute.
  3. Click on the toolbar to save your change. Data will no longer synchronize for the profile.

To re-activate synchronization for the subscription profile:

  1. In the Security Administration navigation area, select the subscription profile.
  2. In the edit area, select the Sync Active attribute.
  3. Click on the toolbar to save your changes. Data will no longer synchronize for the profile.
