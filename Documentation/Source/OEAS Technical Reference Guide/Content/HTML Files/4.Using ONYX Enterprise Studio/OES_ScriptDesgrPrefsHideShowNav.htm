Hiding and showing the navigation area

When Auto-Hide is selected, the navigation area shrinks to a vertical bar on the left side of the Script Designer window.

To hide the navigation area:

  1. To open the User Preferences area, on the toolbar, click .
  2. Under Navigation Area, select the Auto-Hide check box.
  3. Click OK.

To show the navigation area when it is hidden:

  • Move the cursor over the vertical bar that is the compressed navigation area.
