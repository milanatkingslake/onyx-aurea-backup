Activating or deactivating a resource

By deactivating a resource all users are allowed access to the resource.

To activate or deactivate a resource:

  1. In the navigation area, select the resource you want to activate or deactivate.
  2. The attributes for this resource display in the edit area.
  3. In the edit area, check or uncheck the Active box to activate or deactivate the resource. A checked box indicates an active resource.
  4. Click on the toolbar to save your changes.
