Adding territories
Adding new territories

To add a parent territory:

  1. In the navigation area, select the Territories node and click .

A new territory named newTerritory<n> appears at the bottom of the territory list in the navigation area, followed by an asterisk (*).

  1. In the Edit Territory area, define the attributes for this territory.
  • Name - Type the name you want to assign to this territory. You can edit this name after a save.
  • Description - Type a description of this territory. You can edit this description after a save.
  • Parent - Type the name of this territory or select one from the list.
  • Active - Clear this check box if you want to deactivate this territory. The default is active (selected).
  1. Click to save your changes.

A parent territory has been created.

You cannot create a child territory until the parent territory is saved.

To add a child territory:

  1. In the navigation area, select an existing territory and click .

A new territory named newTerritory<n> appears under at the bottom of the territory list in the navigation area, followed by an asterisk (*).

  1. In the Edit Territory area, define the attributes for this territory:
  • Name - Type the name you want to assign to this territory. You can edit this name after a save.
  • Description - Type a description of this territory. You can edit this description after a save.
  • Parent - Select a parent territory from the drop-down list.
  • Active - Clear this check box if you want to deactivate this territory. The default is active (selected).
  1. Click to save your changes.

A child territory has been created.
