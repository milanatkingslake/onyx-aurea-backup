Deleting a field

Reference Table Administration does not enable you to delete configurable fields. If you need to delete a field, you must delete the field from the database manually.

Fields definitions are stored in the ReferenceFields table.

Before deleting any fields in the database, make sure you have made a backup of your database. As a general rule, perform any modifications to your database on a test database in order to insure that you do not adversely impact your production database.
