Reciprocal relationships

Customer records in CCM can have reciprocal relationships through their external contacts.

For example, after automatic reciprocation is enabled, adding John’s person record as a ‘Spouse’ contact to Sue’s person record in CCM automatically creates a two-way relationship between the John’s and Sue’s records, with ‘Spouse’ selected as the external contact type for both records in relation to each other. Similarly, adding a restaurant’s property record as an ‘Owned By’ contact to Sue’s person record results in the automatic addition of Sue as an ‘Owner Occupier’ contact to the restaurant’s property record

Setting up reciprocal contacts involves adding external contact types and then enabling automatic reciprocation.

Enable automatic reciprocation

You can enable automatic reciprocation for company and individual external contact types.

For example, you could enable automatic reciprocation between identical ‘Spouse’ contact types under individual, or you could enable automatic reciprocation between an ‘Owner Occupier’ contact type under individual and an ‘Owned By’ contact type under company.

To enable automatic reciprocation:

  1. In OES, open the Contact Type Administration tool.
  2. If needed, add external contact types and refresh your browser..
  3. Select the node for the first desired external contact.

For example, select ‘Owner Occupier’ under Individual.

  1. In the displayed Edit Contact Type window, select the second contact type from the relevant drop-down list.

For example, select ‘Owned By’ from the Company reciprocal contact type drop-down list.

  1. Note the name of the selected contact type.

In this example, the name of the selected contact type is ‘Owned By'.

  1. Select the node for the noted contact type (the second, reciprocated external contact).

For example, select ‘Owned By’ under Company.

  1. In the displayed Edit Contact Type window, select the first contact type from the relevant drop-down list.

For example, select ‘Owner Occupier’ from the Individual reciprocal contact type drop-down list.

  1. Click the Save button.
