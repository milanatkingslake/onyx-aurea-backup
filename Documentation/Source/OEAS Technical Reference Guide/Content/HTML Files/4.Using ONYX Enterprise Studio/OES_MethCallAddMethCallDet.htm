Adding method call details

There are two ways to add method call details: manually, and with the Method Call Detail Builder. The benefits to using the Method Call Detail Builder include the following:

  • In most cases, the Builder provides you a list of likely inputs and outputs for the details based on the OEAS method you have selected for the method call and indicates which of the elements are required as an input. For more information about the input and output lists provided by the Builder, see About the Method Call Detail Builder.
  • In most cases, the Builder automatically pre-populates the Element Path attribute in the Edit Method Call Detail frame with the appropriate standard XML for each method call detail that you add using the Builder.
  • The Builder allows you to create multiple details at a single time.
