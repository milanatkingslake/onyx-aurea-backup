Linking standard products to an incident product

To link standard products to an incident product:

  1. In the navigation area, locate and select the product that you want to modify. The attributes for this product appear in the edit area.
  2. From the Incident Product dropdown list, select the incident product to which you want to link the product.
  3. Click on the toolbar to save your changes.

You can link an unlimited number of products to a given incident product.
