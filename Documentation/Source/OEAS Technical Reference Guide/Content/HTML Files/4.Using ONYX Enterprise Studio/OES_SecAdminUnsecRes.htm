Unsecuring a business object resource

Unsecuring a resource removes all access restrictions from the resource, enabling all users access to the resource. You can unsecure business object resources that you previously secured.

To unsecure a resource:

  1. From the Business Objects node, expand the appropriate business object group, and select the method you want to remove access permissions from.
  2. Click Unsecure Resource in the edit area.
  3. Click on the toolbar to save your changes.
