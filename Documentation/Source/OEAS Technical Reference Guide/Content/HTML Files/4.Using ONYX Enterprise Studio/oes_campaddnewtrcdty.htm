Creating a new tracking code type

To create a new tracking code type:

  1. In the navigation area, select the first-level Tracking Code Types node, and click on the toolbar.

A new tracking code type titled newTrackingCodeType<n> appears at the bottom of the tracking code type list in the navigation area, followed by an asterisk (*).

  1. In the edit area, define the tracking code type's attributes.
  • Id - This is a non-editable field that is assigned by the database.
  • Name - Type the name you want to assign to this tracking code type.
  • Description - Type a description of this tracking code type.
  • Active - Uncheck this box if you want to deactivate this tracking code type.
  1. Click to save your changes to the OEDB.
