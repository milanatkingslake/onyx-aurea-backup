Quick overview of the Survey Administration tool

The Survey Administration tool is part of the Administration Workbench of OES. The Survey Administration Web page is divided into four areas:

  • Context area
  • Toolbar
  • Navigation area
  • Edit area

Context area - The context area lists the name of the application and the site being maintained.

Toolbar - The toolbar appears under the context area and contains the toolbar buttons that perform the tasks related to surveys.

Icon Tool tip Function Context

Add

Adds a new survey type, survey, question, or response. The item added depends on what is selected in the navigation area.

Enabled when any item (except Question Hierarchy) is selected in the navigation area.

Delete

Marks for deletion an item that is selected in the navigation area. The item is not actually deleted until you save changes. When an item is marked for deletion, clicking this button removes the mark.

Note: You can mark for deletion an item that is in use, such as a question that is used in a survey, but you cannot delete it (which you would ordinarily do by clicking Save).

Enabled when an item is selected in the navigation area.

Move up

Moves a question up in a survey’s list of questions.

Enabled when a survey question is selected.

Move down

Moves a question down in a survey’s list of questions.

Enabled when a survey question is selected.

Save

Saves changes that have not been saved yet.

Enabled after you make any changes.

Help

Launches Help.

Always enabled.

Navigation area - The navigation area is where you begin when working with survey types, surveys, questions, responses, and question hierarchies. To view or modify any of these items, navigate to and select it in the navigation area, then make any desired changes in the edit area.

There are three first-level nodes in the navigation area: Survey Types, All Questions, and All.  You can expand or collapse an item in the navigation area by clicking or beside it.

  • Survey Types node - This node contains all the survey types, and all the surveys within each survey type.
  • All Questions node - This node contains a global list of all the questions that have been created. When you create a survey question from within a survey, a corresponding question is created and listed in this node.
  • All Responses node - This node contains a global list of all the responses that have been created. When you create a question response from within a question, a corresponding response is created and listed in this node.

Edit area - In the edit area, you view and modify the item that you have created or selected in the navigation area. The appearance of the edit area changes according to what is selected in the navigation area.
