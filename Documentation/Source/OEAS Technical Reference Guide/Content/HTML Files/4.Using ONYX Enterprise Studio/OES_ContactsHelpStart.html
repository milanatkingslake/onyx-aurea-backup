Contact Type Administration Help

The Contact Type Administration tool enables you to add, modify, and delete contact types. Contact types are used to show employee relationships to your customers and to classify contacts within the Customer and Internal contact groups.

Configuring contact categories and contact group types is not supported. The default categories and types are required for use by Onyx Employee Portal (OEP).
