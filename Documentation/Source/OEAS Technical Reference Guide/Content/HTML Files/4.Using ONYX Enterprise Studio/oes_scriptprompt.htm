Working with prompts and responses

The Prompts tab in the Edit Step area is where you specify the attributes of a prompt and its responses.

The Prompts tab appears as follows:

The availability and meaning of various fields on the tab depends on the control type that you specify for the prompt. For example:

  • You can specify a data type in the Data Type box only for text box prompts.
  • The behavior you define with the Min, Max, and Extra Data boxes changes according to the prompt type. For more information, see the appropriate subsection following the procedure for adding a prompt.

For several prompt types, you use the Prompt Responses Web Page Dialog to add fixed responses. For procedures on working with fixed responses, see Adding, deleting, and changing the order of fixed responses.

You can also create a custom control type for a prompt. For information on how to do so, see the adding a custom scriptControlType scenario.
