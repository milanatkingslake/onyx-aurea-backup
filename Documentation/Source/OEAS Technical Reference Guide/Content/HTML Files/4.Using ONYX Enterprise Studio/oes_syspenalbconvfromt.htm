EmailCreateFileLinkforLit

This system parameter is used by the Email feature to determine whether or not to create a fileLink record when an OEP user attaches a literature item to an email message.

  • Allowed Parameter Values

Y - When this system parameter is set to Y, a fileLink object is created for the literature item file and the file is copied to the file share. It is this copy on the file server that is opened when viewing saved attachments.

  • Default Parameter Value - EmailCreateFileLinkforLit is installed as (N) by default. When this system parameter is set to N, no fileLink object is created. When a literature item file is attached to the email, the file is opened directly.
