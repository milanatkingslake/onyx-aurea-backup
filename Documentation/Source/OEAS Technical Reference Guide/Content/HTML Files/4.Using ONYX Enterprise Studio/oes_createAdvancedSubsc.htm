Creating an advanced subscriber

An advanced subscriber is an OEP user who can subscribe other users to transactional data. An advanced subscriber gains this capability for a particular subscription service by having:

  • Access permission for the UI:OEP.subscription.proxy.subscribe resource.

By default, the OEP.administrator role has access permission for the UI:OEP.subscription.proxy.subscribe resource.

You can also grant access to the UI:OEP.subscription.proxy.subscribe resource to any other role whose users you want to make advanced subscribers for a particular subscription service, or grant a specific user access to the resource.

  • A subscription profile for the subscription service.

To create an advanced subscriber:

  1. For the UI:OEP.subscription.proxy.subscribe resource, add the user or role to those with access permission to the resource and then save your changes.

For more information on granting access permissions for a resource, see Managing permissions for resources.

  1. For the user that you want to be a advanced subscriber, add a subscription profile, and then save your changes.

For more information on adding a subscription profile, see Adding and deleting a user’s subscription profile.
