Adding a user to a role

Use the following procedure to add a user to a role.

To add a user to a role:

  1. In the navigation area, select the role you want to add a user to.
  2. In the edit area, click Add/Remove Users. Note that in the navigation area, only the users and groups appear. This is because only users can be added to roles.
  3. In the navigation area, select the user you want to add to the role, the page changes to the following (the Add button is enabled).
  4. Click the Add button or drag the user to the user list at the bottom of the edit area. If you drag the user, you must drag it either to the heading area of the user list or to one of the users already in the list.
  5. Click Done.
  6. Click on the toolbar to save your changes.
