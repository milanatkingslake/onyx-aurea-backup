EnableConversionFromTotal
EnableConversionFromTotal

When this parameter is enabled, the total value in a secondary currency for a list of line items in a quote is determined by converting the total value of the quote in the primary currency. If not enabled, the total value for the quote in the secondary currency is calculated by summing each of the line items from the individual prices in the secondary currency. Since values might be rounded when the system converts a currency, these methods might produce different results.

  • Allowed Parameter Values

N - The secondary currency total is calculated by summing the secondary currency line items.

Y - The secondary currency total is calculated by converting the total price of the primary currency line items.

  • Default Parameter Value - EnableConversionFromTotal is installed as disabled (N) by default.
