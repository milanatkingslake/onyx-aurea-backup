Removing a user from a role

Use the following procedure to remove a user from a role.

To remove a user from a role:

  1. In the navigation area, select the role you want to remove a user from.
  2. In the edit area, click Add/Remove Users.
  3. In the navigation area, on the members list, select the user you want to remove from the role and then click Remove.
  4. Click Done.
  5. Click on the toolbar to save your changes.
