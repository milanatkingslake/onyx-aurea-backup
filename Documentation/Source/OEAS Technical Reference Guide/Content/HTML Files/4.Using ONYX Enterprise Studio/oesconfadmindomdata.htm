Configuring and administering domain data

You can configure and administer OEAS domain data using the Onyx Enterprise Studio (OES) administration tools. The topics Global domain data and Local domain data explain OEAS domain data and list the administration tool you use to administer this data.

Typical activities you perform using the administration tools include adding, deleting, and modifying items. For an overview of each administrative tool, see What is the Onyx Enterprise Studio?.
