Modifying a campaign

To modify a campaign:

  1. In the Campaigns node, locate and select the campaign you want to modify.
  2. In the edit area, modify the attributes for this campaign. See Creating a new campaign for the list of attributes in the edit area.

In the navigation area, an asterisk (*) appears next to the modified campaign until your changes are saved.

  • Click to save your changes to the OEDB.
