Deleting a script group

You do not have to delete all the scripts in a script group before deleting the group. Deleting a script group also deletes all of the group’s scripts.

If the script group contains child script group, however, you must delete the child script group before deleting the parent script group.

To delete a script group:

  1. In the navigation area, select the script group.
  2. To mark the script group for deletion, on the toolbar, click .
  3. Click to make the deletion in the database.
