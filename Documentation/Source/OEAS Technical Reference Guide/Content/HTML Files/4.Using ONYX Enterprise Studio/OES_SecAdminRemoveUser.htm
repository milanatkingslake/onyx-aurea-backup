Removing a user or role from a resource

Removing a user or role from a resource automatically denies the user or role access to the resource.

To remove a user or role from a resource:

  1. In the navigation area, select the resource.
  2. In the edit area, click Manage Permissions.
  3. In the appropriate access-permissions list, select the user or role you want to remove from the resource and then click Remove.
  4. When you have finished removing users or roles, click Done.
  5. Click on the toolbar to save your changes.
