Viewing and changing the script in the Edit Layout area

When either a script or a script step is selected, the Edit Layout area shares the edit area with the Edit Script or Edit Step area. The Edit Layout area can also occupy the entire edit area.

In the Edit Layout area, you can:

  • Move steps and their rules.
  • Select a step or rule.
  • Add a step, rule, or prompt.
  • Change the view of the script, including zooming into or out of it.
  • Open the Edit Layout Options area, to change the Edit Layout area according to your preferences.

For more information on these actions, see the following procedures or the sections on working with steps, rules, and prompts.

To expand the Edit Layout area to the entire edit area:

  1. To open User Preferences, on the toolbar, click .
  2. In the User Preferences area, under Layout Area, make sure that the selection in the Clicking on Canvas box is Shows Layout Area Only.
  3. Click the Edit Layout area canvas.

OR

In the navigation area, select the script’s Steps node.

  1. To restore the Edit Script area to view, select the script in the navigation area.

To zoom into and out of the script:

  • On the Edit Layout toolbar, either click to zoom into the script or click to zoom out.

OR

  • Right-click the Edit Layout area canvas and, from the context menu, choose Zoom In or Zoom Out.

To show all the script steps in the Edit Layout area:

  • On the Edit Layout toolbar, click .

OR

  • Right-click the Edit Layout area canvas and, from the context menu, choose Zoom To Fit.

To change Edit Layout area display options for a script:

  • On the Edit Layout toolbar, click , and then make the desired changes.

The changes you make affect the Edit Layout area only in the script in which you make them.

For more information on changing the default width of steps, see Changing Edit Layout area sizing settings.

For more information on changing the information that appears in the Edit Layout area, see Changing Edit Layout area information display settings.
