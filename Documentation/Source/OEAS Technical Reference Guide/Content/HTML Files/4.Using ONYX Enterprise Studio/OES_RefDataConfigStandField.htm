Implementing a standard configurable field

Standard configurable fields are defined in Reference Table Administration by default, and because they are supported by OEAS and OEP by default, no additional customization is required.

Using Reference Table Administration, you can configure the lookup values for those standard configurable fields that are defined as combo box field types. Changing the field name, field type, or hierarchical structure of standard configurable fields is not supported.

Captions for standard configurable fields are not modified using Reference Table Administration. Instead, use OEP's UI Configuration Workbench.
