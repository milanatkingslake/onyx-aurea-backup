Deleting or undeleting a contact type

After you mark a contact type for deletion and save your changes, you cannot undo the deletion. Additionally, you cannot delete a contact type if there is a record that has been associated with that contact type. That association must be deleted before you can delete the contact type.

Onyx recommends that you deactivate contact types instead of deleting them. When you deactivate a contact type, the contact type and its associated records no longer appear to users for active use, but they remain active elements of records previously associated with them. See Activating or deactivating a contact type.

To delete a contact type:

  1. Select the contact type you would like to delete.
  2. Click to mark the contact type for deletion.

The contact type name appears with a strike-through font in the navigation area (for example: newContactType), and an asterisk (*) appears on the contact type.

  • Click on the toolbar to save your changes to the OEDB.

To undelete a contact type marked for deletion:

  1. Locate and select the contact type that you previously marked for deletion. It is distinguished in the navigation area by the strike-through font of its name.
  2. Click to unmark the contact type for deletion.
  3. Click to save your changes to the OEDB.
