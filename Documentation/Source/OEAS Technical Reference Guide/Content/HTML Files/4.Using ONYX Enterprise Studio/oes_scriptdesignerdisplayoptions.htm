Changing layout display options for a script

You can specify the following for each script:

  • Whether the script layout displays automatically when the script is selected in the navigation area

If the Onyx Enterprise Studio machine is slow and/or a script is very large, displaying the script in the Edit Layout area can take more time than you want it to. You can specify that the script layout is not displayed until you click to display it.

When a script is selected and the Always Show Script Layout box has not been selected, the script layout does not appear until the user clicks Show Script Layout in the Edit Layout area. Once the script layout has appeared, it continues to appear when the script is selected, until the user closes Script Designer.

  • How the Edit Layout area is affected when you click the canvas in the layout

To specify whether the script layout appears whenever the script is selected:

  1. To open the User Preferences area, on the toolbar, click .
  2. To delay drawing of the script layout until the user takes action, make sure that the Always Show Script Layout box is not selected.

OR

To specify that the script layout always appears when the script is selected, make sure that the Always Show Script Layout box is selected

  1. Click OK.

To change behavior when you click the Edit Layout area canvas:

  1. To open the User Preferences area, on the toolbar, click .
  2. In the Clicking on Canvas box, select one of the following:
  • Does Nothing - Clicking the canvas has no effect.
  • Shows Layout Area Only - Clicking the canvas expands the Edit Layout area to fill the entire edit area.
  • Shows Script and Layout Area - Clicking the canvas shows the Edit Layout area and the Edit Script area.
  1. Click OK.
