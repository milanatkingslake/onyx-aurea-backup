Modifying a user’s subscription profile

Modifying a user’s subscription profile follows the model for modifying most other objects in OES administration and design tools.

To modify a subscription profile:

  1. In the navigation area, select the subscription profile.
  2. In the edit area, make any desired changes.

For more information on the attributes that you can change for the subscription profile, see Adding and deleting a user’s subscription profile.

  1. Click on the toolbar to save your changes.

Changing a user’s staging server

Complete the following series of tasks if you need to move a user’s mailbox to another staging server:

To change a user’s staging server:

  1. Coordinate with the specific users and have them upsync any unsynchronized changes to their current staging server. You must wait for the sync server to process all records, and ensure that all data was successfully synchronized.
  2. Complete all necessary steps to move the user’s mailbox. See the Microsoft Exchange product documentation for detailed instructions.
  3. In Security Administration, change the Staging Server and, if the user has a different ID on the two servers, the Staging Server User ID attributes for the user’s subscription profile.

For information on changing these attributes, see Adding and deleting a user’s subscription profile.

  1. Reset the data for the user whose mailbox you are moving. See Resetting a user’s subscribed data for more information.
  2. After the next synchronization cycle is complete, manually delete the user’s synchronization folders in the original staging server.
  3. Instruct the user to reconfigure his or her Outlook profile and favorites for the new staging server. See the OEP-Offline Getting Started Guide for detailed instructions.

Follow the same steps if the user is remaining on the same staging server, but the staging server ID has been changed.
