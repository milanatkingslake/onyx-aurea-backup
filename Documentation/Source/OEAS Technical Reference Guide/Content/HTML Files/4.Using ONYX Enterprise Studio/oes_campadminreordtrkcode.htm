Reordering tracking codes

The order of the tracking codes in the navigation area is the same order that the tracking codes appear in the end-user user interface.

To reorder tracking codes:

  1. In the Campaigns node, locate and select the tracking code you want to move.
  2. Click or to reorder the tracking code.

All tracking codes that are moved are marked with an asterisk (*) until the changes are saved.

  1. Click to save your changes to the OEDB.
