Managing OEP profiles

OEP profiles are distinct customized OEP user interfaces that fit the different user roles within your organization. For example, you can configure the OEP UI so that your external sales team uses a different Sales Opportunity and Products page than your internal sales team, and your software technical support team uses a different Support Incident page than your hardware support team.

Users within each group can be given access to one or more profiles. Users with multiple roles within the company may have access to multiple profile-specific UIs, using one OEP profile to complete certain tasks, and another OEP profile to complete other tasks. All the profiles users can access are available at log on, where they can open the default profile or select a different profile.  

Managing Profiles includes the following tasks:

  • Adding an OEP profile
  • Associating users and roles with an OEP profile
  • Modifying an OEP profile
  • Deleting an OEP profile
