Exiting or refreshing the Web tool page

If you navigate off the current page, or refresh the page via the Microsoft Internet Explorer Refresh button prior to saving any changes you have made, a warning message alerts you of the potential loss of your updates. You have the option of discarding the changes or cancelling the refresh action and returning to the current page to either continue editing the unsaved data or to save your changes.
