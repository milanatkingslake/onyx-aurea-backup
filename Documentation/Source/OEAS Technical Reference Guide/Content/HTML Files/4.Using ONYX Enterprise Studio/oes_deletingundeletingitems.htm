Deleting items

Use the following procedures to delete items or to clear items marked for deletion.

To delete an item:

  1. In the navigation area, select the item that you want to delete.

After you mark an item for deletion and save your changes to the OEDB, you cannot undo your changes or undelete it. (All deletes are physical deletes.)

  1. Click to mark the item for deletion.

The item’s name appears with a strike-through font in the navigation area. For example: newCampaign.

  1. Click to save your changes to the OEDB.

To clear an item marked for deletion:

  1. In the navigation area, select the item that you previously marked for deletion. It is distinguished in the navigation area by the strike-through font of its name. For example: newCampaign.
  2. Click to unmark the item for deletion.
  3. Click to save your changes to the OEDB.
