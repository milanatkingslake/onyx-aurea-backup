Adding responses to questions

You can add responses to questions by:

  • Creating a new question response within a question.
  • Adding an existing response.

For information on searching for existing responses, see Adding existing responses to a question.

To create a new response within a question:

  1. In the navigation area, highlight the question to which you want to add the response.

You can highlight the question either under a specific survey or under All Questions.  

  1. In the Add a new response box, type the text of the response and then click Add.

The response is added both to the question and to the list of all responses.

  1. Click to save your changes.
  2. If you want the respondent to be able supply his or her own text to this response:
  • In the navigation area under All Responses, highlight the response.
  • In the edit area, select the Editable box and then click to save your changes.
