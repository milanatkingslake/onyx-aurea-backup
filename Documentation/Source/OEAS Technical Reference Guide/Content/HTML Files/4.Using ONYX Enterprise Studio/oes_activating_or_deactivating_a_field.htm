Activating or deactivating a field

To activate or deactivate a field:

  1. In the navigation area, select the Fields mode.
  2. Click to display the fields associated with a particular node.
  3. Select the field you want to modify. The attributes for the field display in the edit area.
  4. In the edit area, check or uncheck the Active box to activate or deactivate the field. A checked box indicates an active field.
  5. Once you modify a field attribute, the field appears in the navigation area with an asterisk (*) until you save your changes. If you deactivate the field, the field name is also italicized and grayed out, but the asterisk (*) disappears once the changes are saved.
  6. Click on the toolbar to save your changes.

Script errors may appear on a page if you have deactivated UDFs (User Defined Fields) on the page. To resolve this issue, either activate the UDFs using Reference Table Administration or remove the UDFs from the page using UCW.
