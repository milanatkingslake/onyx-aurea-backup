Modifying a user

You modify a user’s attributes in the edit area, as you specified attribute values when you created the user.

To modify a user:

  1. In the navigation area, open the Group that the user is a member of and then select the user.
  2. In the edit area, apply the modifications to the user's attributes.

For the list of user attributes you can modify, see Adding a user.

  1. Click on the toolbar to save your changes.
