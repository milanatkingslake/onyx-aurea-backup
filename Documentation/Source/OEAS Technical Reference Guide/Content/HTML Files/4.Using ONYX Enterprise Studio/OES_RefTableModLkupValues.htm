Modifying a lookup value for a combo box field

Within the Lookup Values by Fields mode, you can modify a lookup value for a specific field without modifying that value where it is associated with other fields, unless the other field has been related to the field containing the lookup value that you are modifying (see Relating existing fields).

For example, suppose that fields a, b, and c are combo box fields. When you created lookup values for field a, one of those values was “Billing,” and when you created lookup values for field b, you created one of them from the Existing Lookup Values list, adding “Billing” as a lookup value for field b. However, when you configured field b, you used the Related to Existing Field box to relate it to field a. Subsequently, when you use the Lookup Value by Field mode in Reference Table Administration to modify the lookup value “Billing” for field a, changing it to “Business,” the lookup value “Billing” for field c is also modified because field c is related to field a, meaning that they share the same lookup values. The lookup value for field b remains “Billing.”

To modify a lookup value for a combo box field:

  1. In the navigation area, select the Lookup Value by Fields mode option to display the available nodes.
  2. Click to display the fields associated with a particular node. The node expands and displays its associated fields.
  3. Click in front of the field to which you want to add a lookup value. The field node expands and displays its existing lookup values.
  4. Select the lookup value you want to modify. The attributes for that lookup value appear in the edit area.
  5. In the edit area, modify the lookup value attributes. For descriptions of the lookup value attributes, see Defining lookup value attributes.
  6. Click on the toolbar to save your changes.

You can change the position of how these lookup values appear in the user interface by clicking or to move them up or down one position at a time and saving your changes.
