Selecting a table

To select a table:

  1. In the navigation area, select a table from the Table list.

A description of the table appears in the Descriptionfield.

  1. If the table you selected has filters associated with it, you must select a filter from the Filter list before you can view or edit any records.

In the edit area, the Table Administration griddisplays the records contained in the table.

Keyboard navigation tips

  • Use the tab key to move forward in a row.
  • Use shift-tab to move backward in a row.
  • Use the up and down arrows to move between rows.
  • Use shift and the left/right arrows to de-select text in a record's field.
  • To sort the data in a column, click the column's grid heading. The sort order of the data changes from ascending to descending, and vice versa.
