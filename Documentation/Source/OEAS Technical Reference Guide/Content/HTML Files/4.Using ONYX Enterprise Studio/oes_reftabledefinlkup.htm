Defining lookup values for hierarchically related combo box fields

You can hierarchically relate combo box fields when you configure fields in the Fields mode of Reference Table Administration (see Hierarchically relating combo box fields). Once the hierarchy has been created, you can define lookup values for the hierarchically related combo boxes in the Lookup Value by Fields mode.

For hierarchically related combo box fields, you must first define the drop-down values for the parent field and then define the drop-down values of the child field(s).

To define lookup values for hierarchically related combo box fields:

  1. In the navigation area, select the Lookup Value by Fields mode option to display the available nodes.
  2. Click in front of the node that includes the fields to which you want to add lookup values. The node expands and displays its associated fields.
  3. Click in front of the parent field to select the parent field and display any associated lookup values.
  4. Select either the parent field node or an existing lookup value within the parent field node and click . A new lookup value named NewLookupValue<n> appears at the bottom of the lookup values list in the navigation area, followed by an asterisk (*).
  5. In the edit area, define the attributes for the lookup value, either by adding an existing lookup value or creating a new lookup value (see Adding lookup values to a combo box field).
  6. If you are adding multiple lookup values to the field, click Add Lookup Value to associate the lookup value you just defined with the field. The lookup value displays with an asterisk(*) in the navigation area, and a new lookup value named NewLookupValue<n> appears at the bottom of the lookup values list in the navigation area, followed by an asterisk (*). The edit area refreshes enabling you to define the attributes for another lookup value for this field.
  7. Click on the toolbar to save the new lookup values.
  8. In the navigation area, click in front of the lookup value that you just created to expose the child combo box field below it.
  9. Select the child field and click . A new lookup value named NewLookupValue<n> appears at the bottom of the lookup values list in the navigation area, followed by an asterisk (*).
  10. As described previously, in the edit area, define the attributes for the lookup value, either adding an existing lookup value or creating a new lookup value.
  11. Again, if you are adding multiple lookup values to the child field, select Add Lookup Value to associate the lookup value you just defined with the field, and define the next lookup value.
  12. Click on the toolbar to save your changes.
