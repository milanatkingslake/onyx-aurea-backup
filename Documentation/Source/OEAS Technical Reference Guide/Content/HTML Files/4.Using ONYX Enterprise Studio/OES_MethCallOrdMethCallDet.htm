Ordering method call details

To move a method call detail within a list:

  1. In the navigation area, locate and select the method call detail that you want to move.
  2. Click or to move the method call detail up or down in the navigation tree.
  3. Click to save your changes.

You can also control the order of method call details at the time that you add them. For more information, see Adding method call details manually and Adding method call details using the Method Call Detail Builder.
