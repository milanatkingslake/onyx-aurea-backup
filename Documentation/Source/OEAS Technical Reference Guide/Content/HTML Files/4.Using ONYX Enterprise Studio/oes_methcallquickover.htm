Quick overview of the Method Call Designer

The user interface of Method Call Designer is divided into four areas:

  • Context area
  • Toolbar
  • Navigation area
  • Edit area

Context area - The context area lists the tool name and the name of the application.

Toolbar - The toolbar appears under the context area and contains the toolbar buttons that control the tasks related to Method Call Designer.

The following toolbar buttons are available in the Method Call Designer:

Icon Tool tip Function Availability

Add

Used to add a new method call or method call detail.

Enabled when any node or item is selected in the navigation area.

Delete

Used to delete a method call or method call detail. When clicked, the item selected in the navigation area is marked for deletion. Can be used to undelete a previously deleted item if that deletion has not yet been saved.

Enabled when you select a method call or method call detail in the navigation area.

Copy

Copies a selected method call or method call detail.

Enabled when you select a method call or method call detail in the navigation area.

Paste

Pastes a copied method call or method call detail.

For method calls, enabled when you select either the method call group node from which the method call was copied or when you select another existing method call from that group.

  For method call details, enabled when you select the appropriate detail node (input or output), based on whether you copied an input or output detail or when you select an existing detail under the appropriate detail node.

Up

Moves a method call detail up one position in the navigation tree.

Enabled when a method call detail is selected in the navigation area.

Down

Moves a method call detail down one position in the navigation tree.

Enabled when a method call detail is selected in the navigation area.

Save

Used to save all modified method call and method call detail data. Removes all asterisks. (An asterisk indicates an unsaved change for an element.)

Enabled after any changes have been made.

Show/Hide Context Area

Shows or hides the context area.

The upper image shows the button when the context area is expanded and clicking the button will contract it.

The lower image shows the button when the context area is contracted and clicking the button will expand it.

Always enabled.

Help

Launches Help.

Always enabled.

Navigation area - The navigation area displays the available method call groups and existing method calls and method call details.

Edit area - The edit area displays the attributes for either a method call or method call detail that you have selected in the navigation area. The edit area enables you to define the method call and method call detail attributes.
