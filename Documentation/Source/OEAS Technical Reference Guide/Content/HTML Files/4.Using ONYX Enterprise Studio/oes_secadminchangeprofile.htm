Modifying an OEP profile

To modify an existing OEP profile:

  1. Open the OEP Profiles in the navigation pane.
  2. Select the profile that you want to change. The Edit OEP Profile pane opens.

From the edit pane, you may change the profile name or description; change the active status; or add or delete members.

  1. Click to save your changes.
