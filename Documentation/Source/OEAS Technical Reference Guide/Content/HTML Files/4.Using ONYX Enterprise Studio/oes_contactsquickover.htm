Quick overview of the Contact Type Administration tool

The user interface of Contact Type Administration is divided into four areas:

  • Context area
  • Toolbar
  • Navigation area
  • Edit area

Context area - The context area lists the name of the application and the site being maintained.

Toolbar - The toolbar appears under the context area and contains the toolbar buttons that control the tasks related to Contact Type Administration.

The following toolbar buttons are available:

Icon Tool tip Function Context

Add

Used to add a new contact type. The contact type is added to the category selected in the navigation area.

Enabled only when a contact type category is selected in the navigation area.

Delete

Used to delete a contact type. The contact type selected in the navigation area is marked for deletion. Can be used to undelete a previously deleted contact type if that deletion has not yet been saved.

Enabled only when you select a contact type. Contact Groups and Categories are not modifiable.

Save

Used to save all modified contact type data. Removes all dirty flags.

Enabled after any changes have been made.

Help

Launches Help.

Enabled at all times.

Navigation area - The navigation area displays the contact types available, organized by group and then by category.

Edit area - The edit area displays the attributes for the contact type that you have selected in the navigation area. You can edit this information to add new or modify existing contact type data.
