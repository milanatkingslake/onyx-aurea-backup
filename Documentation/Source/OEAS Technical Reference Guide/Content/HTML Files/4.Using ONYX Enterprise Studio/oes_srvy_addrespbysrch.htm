Adding existing responses to a question

You can add a response within the edit area of a question by searching either other questions or all existing responses. The following two procedures explain each of these search methods. Each procedure assumes that you have already specified either single or multiple for the question type.

To add a response from another question:

  1. In the navigation area, either within a survey or in the All Questions node, select the question to which you want to add the response.
  2. In the box beside Show responses from question, click the down arrow and then select the question that contains the response or responses that you want to add to this question.
  1. In the responses listed, select one or more and then click Add Selected.

The selected response is added to the list under Responses.

  1. Click to save your changes.

To add a response by searching all responses:

  1. In the navigation area, either within a survey or in the All Questions node, select the question to which you want to add the response.

The edit area changes to look like the following:

  1. To search all responses, under Add responses, in the Search for a response box, either:
  • Type a string that the response contains.

OR

  • To obtain a list of all responses, leave the box blank.
  1. Click Search.

The list of relevant responses appears in the box under Add Selected.

Note that:

  • The search engine is not case-sensitive.
  • If you leave the Search for a response box empty, then clicking Search produces a list of all responses.
  1. In the responses listed, select one or more and then click Add Selected.

The selected response is added to the list under Responses.

  1. Click to save your changes.
