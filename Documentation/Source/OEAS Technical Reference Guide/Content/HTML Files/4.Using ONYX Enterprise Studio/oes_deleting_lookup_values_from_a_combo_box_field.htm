Deleting lookup values from a combo box field

Once you mark a lookup value for deletion and save your changes, you cannot undo the deletion. Additionally, you can not delete a lookup value if there are any records in the system that are associated with that lookup value. Onyx recommends that you deactivate lookup values instead of deleting them. When you deactivate a lookup value, the value no longer appears to users for active use, but it remains an active element of records previously associated with it.

To delete a lookup value from a combo box field:

  1. In the navigation area, select the Lookup Value by Fields mode option to display the available nodes.
  2. Click in front of the node to display the fields associated with that node.
  3. Click in front of the field to display the lookup values associated with that field.
  4. Select the lookup value to be deleted. The edit area displays the lookup value attributes.
  5. Click to mark the value for deletion. The lookup value appears with a strike-through font. For example: Lookup.
  6. Click on the toolbar to save your changes. The lookup value is deleted and no longer appears in the navigation area.

To undelete a lookup value marked for deletion:

  • To "undelete" a lookup value before saving, select the item you want to restore and click . You can restore any item marked for deletion as long as changes have not been saved.
