Copying a method call

Remember the following when copying method calls:

  • Copying method calls is supported only within method call group nodes. You cannot copy a method call from the Scripting group node into the Merge Template group node, or vice versa.
  • You can copy and paste method calls between browser instances. These browser instances might even be pointed at different databases.

To copy a method call:

  1. In the navigation area, select the method call that you want to copy, and then click .
  2. Select the method call group node (the same from which the method call was copied) or an existing method call within that group, and click to paste the copied method call.

A copy of the original method call named <originalName><n> appears in the navigation area, followed by an asterisk (*). The attributes for the new method call appear in the edit area.
