Modifying an incident product

To modify an incident product:

  1. In the navigation area, expand the Incident Products node.
  2. Expand the Incident Category node for the incident product you want to modify.
  3. Select the incident product you want to modify.
  4. In the edit area, apply your modifications to the attributes for this incident product.
  5. Click on the toolbar to save your changes.
