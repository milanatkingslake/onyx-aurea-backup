EnableSpellCheck

EnableSpellCheck determines whether the spell checker functionality is enabled for OEP for all features that support spell checking. Spell checking is supported for incident, task, and work ticket work notes, customer comments, messenger messages, literature order comments, and alerts.

  • Allowed Parameter Values

N - Spell checking is not enabled.

Y - Spell checking is enabled.

  • Default Parameter Value - EnableSpellCheck is installed as enabled (Y) by default.

Spell checking is enabled when EnableSpellCheck is set to Y and Microsoft Word is installed on the client machine.
