Activating or deactivating items

The following procedure explains how to activate or deactivate items.

If you deactivate an item such as a Product Group, you can still configure any of its subordinate items (individual products) through the appropriate administration tool (Product Administration). End users, however, cannot apply those subordinate items to a record once the item is deactivated.

To activate or deactivate an item:

  1. In the navigation area, select the item you want to activate or deactivate.

The attributes for the item display in the edit area.

  1. In the edit area, check or uncheck the Active box to activate or deactivate the item. A checked box indicates an active item.

After you modify an item’s attribute, the item appears in the navigation area with an asterisk (*) until the changes are saved. If the item was deactivated, the item is also italicized and grayed out, but the asterisk (*) disappears once the changes are saved.

  1. Click to save your changes to the OEDB.

In both Survey Administration and Security Administration, you must select the item in the edit area, and then click the Active/Inactive button.
