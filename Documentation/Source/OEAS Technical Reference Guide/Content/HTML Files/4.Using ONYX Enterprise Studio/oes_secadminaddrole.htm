Adding a role

Use the following procedure to create a role and activate the members list which enables you to add or remove users.

To add a role:

  1. In the navigation area, select the Roles node or a specific role.
  2. Click on the toolbar. A new group titled newRole1 appears at the bottom of the list in the navigation area, followed by an asterisk (*).  
  3. In the edit area, define the attributes for this role as follows.
  • Role ID - Enter a unique string for the Role ID. You can only change the role's ID in this required field when you create the role.
  • Description - In this box, type a description of the role. This field is required. When you create a role, the box is blank by default.
  • Active - Clear this check box to disable the role. A checked box implies an active role. The default is active (checked).
