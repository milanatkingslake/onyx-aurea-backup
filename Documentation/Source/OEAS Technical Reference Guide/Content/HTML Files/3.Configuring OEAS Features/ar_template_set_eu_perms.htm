Setting end-user permissions

For end users of the Onyx Enterprise Portable (OEP), functionality of the Templates feature is governed by permissions that you set using Security Administration, an administration tool of OES. The following table lists the Templates permissions you can set using Security Administration.

Permissions Description

Delete

For users with this permission, the Delete button in Template Maintenance is enabled.

Insert

For users with this permission, the Add button in Template Maintenance is enabled.

Retrieve

End users with this permission can access the Templates feature area.

Update

For users with this permission, the Save button in Template Maintenance is enabled.

Update Public

For users with this permission, the Save as Public Template check box in Template Maintenance is enabled.

Permissions

Description

Security Administration includes a section for administering resources. You can set the templates permissions in the UI Resources. For more information about UI Resources and OEAS security, see Security and user authentication. For information about using Security Administration see Security Administration Help.
