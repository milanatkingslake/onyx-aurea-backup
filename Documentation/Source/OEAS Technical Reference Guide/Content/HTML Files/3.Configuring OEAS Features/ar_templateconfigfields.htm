Configuring configurable fields

You can configure the textFormatTypeId field of the Templates feature to meet the specific requirements of your organization. Configuring this field includes creating lookup values.

Configurable fields for the Templates feature are supported by the Reference Data feature, as described in Reference Data. You can administer this field with Reference Table Administration, as described in the following table.

Configurable field type Administration tasks

Standard

With standard configurable fields, you can create, delete, and modify lookup values.

For additional information regarding these tasks, see Reference Table Administration Help.

The object and associated elements that support configurable fields for the Templates feature are listed in the following section.

mergeTemplate object
