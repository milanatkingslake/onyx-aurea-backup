Configuring configurable fields

For the Attachments feature, there is only one configurable field: Attachments.Category. This combo box field contains the attachment file type categories (advertisements, contracts, investor reports for example), which are displayed and used on the Attachment Detail window. This field is a standard configurable field supported by the Reference Data feature, as described in Reference Data.

You can configure this field by creating the lookup values for the field using Reference Table Administration. For more information, see Reference Table Administration Help.

The object and associated elements that support this configurable field are listed in the following table.

There are no user-defined fields for the Attachments feature.
