Configuring quote templates

You can configure custom quote templates in addition to the default templates included with OEAS. Using Table Administration, you can add records to the template table to make custom templates selectable in the user interface. The default templates are either an ASP page or an XSL style sheet depending on whether they are intended for use in the online or the offline versions of OEP.

The following table describes the fields and valid values that you can configure for quote templates in Table Administration.
