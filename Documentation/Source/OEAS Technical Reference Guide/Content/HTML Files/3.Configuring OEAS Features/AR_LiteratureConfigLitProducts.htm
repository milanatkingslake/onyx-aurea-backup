Creating literature products

Literature products are a primary component of the Literature Orders feature. End users can order one or more literature products for a customer by creating a literature order for the customer and associating the literature products to that order.

You can administer literature products using Product Administration. Using this tool, you can:

  • Create new literature products and product groups
  • Modify existing literature products and product groups
  • Organize literature products by product groups
  • Set literature products and product groups to active or inactive
  • Delete literature products and product groups

For information about using Product Administration, see Product Administration Help.

When you are creating and modifying literature products in Product Administration, there are several literature product attributes that you can also configure. These attributes include:

  • Printer
  • Label Product

For information on configuring these attributes of the literature products, see Products.

The Literature feature depends on the Onyx Processing Server (OPS). For literature product configuration guidelines, refer to Using OPS Administration.
