Task Manager

Apart from customizing the filterHeader and filterDetail objects, the only option for configuring the Task Manager feature is to configure end-user error messages and some column headings for data that is retrieved from the database.

Configuring end-user error messages or column headings can be useful when:

  • You customize a step or stored procedure in order to perform additional validation and require a message for cases when the validation fails.
  • Your customization returns data that is not consistent with a default column heading.

These end-user error messages and column headings are maintained in the national language tables. You can administer them using Table Administration.
