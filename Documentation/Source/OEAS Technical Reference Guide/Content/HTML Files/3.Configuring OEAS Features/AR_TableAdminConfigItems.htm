Configuring table administration items

Table administration items described

Table administration item domain data, retrieved by the tableAdministrationItem object, specifies the domain data areas of OEAS that can be administered with the Table Administration tool.

The table administration item data is installed by default when you install OEDB. Using the Table Administration tool, you can create and administer table administration item data by:

  • Creating new table administration items
  • Modifying existing table administration items
  • Deleting a table administration item

For further details about this tool, see Table Administration Help.

The following table lists table administration item elements displayed by Table Administration.

Element Description

objectName

The name of the object called to retrieve and save data.

filterObjectName

The object to call the retrieveCollection method on, to return the list of filters for this entry.

filterKeyPropertyName

The name of the property returned by the call to the retrieveCollection on the filterObjectName that represents the key for the filter. It is used to set the value of the property specified in filteredObjectPropertyName.

filterDescriptionPropertyName

The name of the property returned by the call to retrieveCollection on the filterObjectName that represents the description for the filter. It is displayed in the drop-down list representing the list of filters.

filteredObjectPropertyName

The name of the property that any filtering is based on. It is used to construct the input XML passed to the object specified in objectName in the retrieveCollection call to return all the entries that match the filter.

menuName

The string that appears in the Table list within Table Administration.

description

The string that appears in the Table Administration user interface when the user makes a selection from the Table list.
