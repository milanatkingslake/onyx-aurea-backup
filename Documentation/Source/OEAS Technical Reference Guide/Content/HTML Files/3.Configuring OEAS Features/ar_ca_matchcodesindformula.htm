Individual match code formula

The individual match code formula is made up of the five sections defined below. The individual match code is always 25 characters long. If the individual record contains no data corresponding to a particular character position in the code, the stored procedure substitutes an asterisk (*).

Company name

This section is composed of up to three parts of the company™s name, using the following sequence:

  • The first two characters of the entire company name
  • The first character of the second string in company name
  • The first character of the third string in company name
