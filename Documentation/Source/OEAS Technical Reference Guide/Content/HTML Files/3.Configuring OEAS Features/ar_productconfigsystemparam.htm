Setting system parameters

You can set the following system parameters to modify the default behavior of the OEDB and OEP user interface for the Products feature. The following table briefly describes these system parameters.

System Parameter Description

AllowSpaceInProdNum

Indicates if spaces are allowed in product numbers.

DefaultCurrencyCode

The default currency code assigned to new products entered into the database through Product Administration.

You can administer these system parameters using System Parameter Administration. Using this tool, you can set the parameter value that controls each system parameter. For details regarding System Parameter Administration and these system parameters, see System Parameter Administration Help.
