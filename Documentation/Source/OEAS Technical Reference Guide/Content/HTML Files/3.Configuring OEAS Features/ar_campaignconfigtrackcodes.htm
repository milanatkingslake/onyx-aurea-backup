Creating campaigns and tracking code data

You can configure the tracking code data for the Campaigns feature so that end users can choose a tracking code and milestones to associate with a customer, customer product, incident, or literature order record.

There is no default data installed for campaigns, tracking code types, tracking codes, or tracking code milestones. However, you can create and administer data for these tracking code elements using the Campaign Administration tool.

The Campaign Administration tool enables you to:

  • Create and update campaigns, tracking code types, tracking codes, and tracking code milestones.
  • Deactivate or delete campaigns, tracking code types, tracking codes, and tracking code milestones.

For details on configuring tracking code data, see Campaign Administration Help.
