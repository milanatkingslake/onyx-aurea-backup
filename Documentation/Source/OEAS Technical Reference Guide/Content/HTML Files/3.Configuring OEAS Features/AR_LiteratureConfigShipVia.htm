Configuring ship methods

When users create a literature order, they can select one of several pre-defined shipping methods. A number of ship methods are installed by default during installation of the OEDB, but you can configure them to meet the needs of your organization.

You can configure the domain data for ship methods using Table Administration. With this tool, you can:

  • Create new ship methods
  • Modify existing ship methods
  • Set ship methods to active or inactive
  • Delete existing ship methods

The following table lists ship via fields displayed by Table Administration.

Element Description

description

The name of the shipping method.

active

Specifies whether the data is active or inactive: 1 is active and 0 is inactive.
