Method Calls

For the Method Call feature, you can configure end-user error messages. This can be useful in the following situations:

  • You customize an OEAS step in order to perform additional validation and require a message in case the validation fails.
  • You want to provide the end user with additional information about the process.

These end-user error messages are maintained in the national language tables. You can administer them using Table Administration. For additional information on configuring error messages, see National Language.

For information on designing and creating method calls, see Method calls.
