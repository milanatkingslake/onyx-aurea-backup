Modifying match code functionality

Modifying match code functionality involves editing the existing stored procedures and functions to return different results. The topics in this book describe the existing match code formulas and the procedures and functions which implement them. To implement a different match code formula, you can copy and rename an existing procedure or function, change it appropriately, and then specify the renamed copy as a post procedure for the relevant logical methods in OED. Alternatively, you can create new procedures and functions to be called by the appropriate logical methods. For details on creating or customizing stored procedures, see Customizing OEDB.
