Configuring user-defined fields for the new category

You can also configure user-defined fields for the new task category. The following table lists guidelines for configuring a user-defined field in Reference Table Administration. Use the Fields option to add a user-defined field to the task group. For more information on adding reference fields, see Reference Table Administration Help.

Edit area attribute Valid value

Field Name

task.<category>.<userX> where <category> is the categoryCode configured in Table Administration and X corresponds to the user-defined property number (1-10)

Caption

Any string suitable for display

Sequence #

Any integer

Active

1 (active), or

0 (inactive)

Field Type

According to business requirements
