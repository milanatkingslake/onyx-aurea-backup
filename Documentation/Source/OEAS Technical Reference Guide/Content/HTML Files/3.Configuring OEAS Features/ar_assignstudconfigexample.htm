Configuration example

This example modifies the "Assign Companies" business process to automatically subscribe the members of the assignment set. Subscribed members can download the company record to OEP-Offline.

The example demonstrates how to:

  • Create OPM packages for use within business rules
  • Add packages to, and create rules in, activities
  • Prepare input data for submittal to an OEAS business object method

For more in-depth information about OPM and its features, refer to the OPM documentation set.
