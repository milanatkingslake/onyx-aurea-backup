Add package to activity

In this step, the subscriptionItemSubscribe package is added to the activity. Only packages that are added to an activity can be processed by the activity's rules.

To add the package to the activity:

  1. Right-click the Company Assignment activity, and then select Content from the shortcut menu.
  • To add the package to the list of packages used in this activity:
  • Select the subscriptionItemSubscribe package in the Global Packages/Attributes list.
  • Select the oeasBusinessObjects package in the Activity Packages/Attributes list.

The subscriptionItemSubscribe package is now a child package of the oeasBusinessObjects package.

  • Click to include the package in the Activity Packages/Attributes list

The Activity Package Wizard appears with the package selected in the list.

  1. Ensure that the default settings Not Persist and Single are selected, as shown below.

The Not Persist option is acceptable because this process contains only one activity. (The Persist option is needed when the process contains multiple activities.) The Single option is acceptable because there is only one instance of the package in this group for the duration of the activity.

  1. Click Next to view the Relationship settings. (There is no need to change any of the settings on this portion of the wizard.
  2. Click Finish to exit the wizard.

The Activity Packages/Attributes list now includes the subscriptionItemSubscribe package as one that will be used in this activity (and the entire process).
