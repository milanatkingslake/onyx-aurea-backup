Configuring currency conversion data

OEAS uses currency conversion data to calculate currency conversions between two different currency codes. How this calculation is performed is explained in detail in Currency conversion processing.

The installation CD includes the file CurrencyConversion.dat. This is a tab-delimited text file that contains all combinations of currency conversions for the entire list of currencies that are included in the currency domain data by default on installation of the OEDB. You can use the data of this file as a reference point to help you understand how currency conversion is performed, and to give you a starting point for downloading current conversion rates.

You can use the SQL Server BCP utility to copy this file into the currency_conversion table.

In the CurrencyConversion.dat file, all conversion rates are set to 1.0000000, and can be replaced with accurate data prior to implementing OEAS in a production environment.
