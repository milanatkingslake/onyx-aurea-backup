Disabling the OEP UI Subscription feature

You can disable the OEP UI Subscription feature using the system parameter EnableSubscription. By setting EnableSubscription to disabled, you turn off the OEP UI Subscription feature system-wide. In other words, once you have disabled EnableSubscription and have restarted the OEP-Online application, OEP UI Subscription controls are no longer available to OEP end-users, even if those end users have active subscription profiles.

You can administer this system parameter using System Parameter Administration. For details regarding System Parameter Administration and specific details regarding EnableSubscription, see System Parameter Administration Help.

The EnableSubscription system parameter has no effect on an offline user’s ability to synchronize their offline data with OEP.

To disable synchronization for a specific user, see Disabling the subscription profile.
