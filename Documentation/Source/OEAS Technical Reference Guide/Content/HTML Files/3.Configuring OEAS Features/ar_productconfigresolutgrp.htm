Configuring resolution groups for incident products

When resolution group data exists, it is displayed in the Resolution Group drop-down list box in the Product Administration edit area when you have selected an incident product from the navigation area. By associating a resolution group to an incident product in Product Administration, you configure the lookup values that an end-user selects for the resolution field of an incident when they have associated that particular incident product to the incident.
