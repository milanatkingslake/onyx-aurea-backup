Setting system parameters

There are several system parameters that you can use to modify the default behavior of the OEDB and OEP for the Customers feature.

You administer system parameters using System Parameter Administration. Using this tool, you can set the parameter value that controls each system parameter. For details regarding System Parameter Administration and specific details regarding each system parameter, see System Parameter Administration Help.

The following table briefly describes these system parameters.

System parameter Description

CompFavoriteContactTypeId

Specifies the contact type ID from the company contact category that is intended as the Favorite contact type for companies. OEP uses this system parameter to support the Home Page functionality for Favorite Companies.

EmailLowerCase

Converts e-mail addresses to lower case for inserts and updates.

IndivFavoriteContactTypeId

Specifies the contact type ID from the individual contact category that is intended as the Favorite contact type for individuals. OEP uses this system parameter to support the Home Page functionality for Favorite Individuals.

OEPHistoryDuration

Specifies the number of days in the past, from the current date, for which OEP displays the history list.

SerialNumberSearchAsLike

If enabled, any searches performed using serial numbers automatically use wild card strings at the end of the serial number search criteria. For example, a search for 123 returns all entries in the database with a serial number beginning123. If SerialNumberSearchAsLike is not enabled, only the entries with the exact serial number 123 are returned by the search.
