Messenger, Reminders, and Notification

You can configure the Messengers and Reminders features by:

  • Configuring end-user error messages and column headings

You can additionally configure the Messenger feature by:

  • Setting system parameters

The only configuration possible for the Notifications feature is to change the frequency of notifications. Changing notification frequency requires altering the code of the Onyx Employee Portal (OEP) user interface.
