Setting system parameters

You can set several system parameters to modify the default behavior of the OEDB and OEP user interface for the Tasks feature.

You can administer these system parameters using System Parameter Administration. Using this tool, you can set the parameter value that controls each system parameter. For details regarding System Parameter Administration and these system parameters, see System Parameter Administration Help.

The following table briefly describes these system parameters.

System Parameter Description

EnableMultipleTasksSales

Determines whether the Task tab appears on the Sales Incident window.

EnableMultipleTasksService

Determines whether the Task tab appears on the Service Incidentwindow.

EnableMultipleTasksSupport

Determines whether the Task tab appears on the Support Incident window.

EnableSpellCheck

Enables spell checking for all features that support spell checking, including task work notes.
