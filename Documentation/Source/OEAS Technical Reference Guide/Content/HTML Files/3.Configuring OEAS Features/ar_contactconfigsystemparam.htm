Setting system parameters

There are two system parameters that you can use to modify the default behavior of the OEDB and OEP for the Contacts feature.

You can administer these system parameters using System Parameter Administration. Using this tool, you can set the parameter value that controls each system parameter. For details regarding System Parameter Administration and the system parameters of the Contacts feature, see System Parameter Administration Help.

The following table briefly describes these system parameters.

System Parameter Description

CompFavoriteContactTypeId

Specifies the contact type ID from the company contact category that is intended as the Favorite contact type for companies. OEP uses this system parameter to support the Home Page functionality for Favorite Companies.

IndivFavoriteContactTypeId

Specifies the contact type ID from the individual contact category that is intended as the Favorite contact type for individuals. OEP uses this system parameter to support the Home Page functionality for Favorite Individuals.
