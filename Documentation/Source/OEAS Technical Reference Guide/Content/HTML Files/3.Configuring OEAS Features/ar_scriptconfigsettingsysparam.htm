Setting the system parameter OEPScriptCacheLife

OEPScriptCacheLife is the only system parameter for the Scripting feature. By setting the value for OEPScriptCacheLife, you can modify how frequently the OEP cache of scripts checks for updates.

The frequency with which the cache of scripts checks for updates determines how quickly a new script becomes available to OEP users. The default value is 300 seconds (5 minutes). If you set the OEPScriptCacheLife value to 0, then scripts are not cached and are immediately visible in OEP.

Setting the OEPScriptCacheLife value to 0 slows system performance.

You administer system parameters by using System Parameter Administration to set the parameter value that controls OEPScriptCacheLife. For details regarding System Parameter Administration and specific details regarding OEPScriptCacheLife, see System Parameter Administration Help.
