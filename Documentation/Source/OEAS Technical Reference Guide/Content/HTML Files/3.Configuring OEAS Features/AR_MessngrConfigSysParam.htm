Setting system parameters

There is a single system parameter that you can set to modify the default behavior of OEP for the Messenger feature.

You administer system parameters using System Parameter Administration. Using this tool, you can set the parameter value that controls the system parameter. For details regarding System Parameter Administration and specific details regarding this system parameter, see System Parameter Administration Help.

The following table briefly describes the Messenger system parameter.

System parameter Description

EnableSpellCheck

Enables spell checking for all features that support spell checking, including messenger messages.
