Configuring email address book items (queues)

Email address book items, referred to in OEP as queues, populate the From box of an email message. Queues make it easy for inbound reply messages from individuals or companies to be sent directly to a monitored corporate mailbox, such as a sales or support mailbox, for processing by an email monitor subsystem. For more information, see Configuring email monitor subsystems to process inbound mail.

In Table Administration, you use the email_address_book_item table to create email address book item entries (queues) for your company.

The following table lists the editable fields displayed in Table Administration.

Field Description

emailAddress (required)

Type the email address
Example: support@yourcompany.com

displayName

Type the name that is displayed in the user interface for the email address
Example: HelpDesk

allowAsFrom

Set to one of the following values:

1 - allows a user to send an email from this address

0 - user is not allowed to send an email from this address

active

Set to 1.
