Configuring product status data

Product status domain data is used by the statusId property of the product object. It is installed by default during installation of the OEDB. However, as noted previously, you can use Reference Table Administration to modify the lookup values or create new lookup values.

The product status lookup values are displayed in the Product Status drop-down list box within the Inventory Information edit area when you have selected a product from the navigation area.
