ENUM_I

The ENUM_I data type denotes enumeration integers. Search items of this type represent lookup values that are stored by an ID value in the table to which they have been associated.

When a user selects a search item of type ENUM_I, all the values possible for that search item appear, including both the string values and their associated IDs. ENUM_S

The ENUM_S data type denotes enumeration strings. Search items of this type represent lookup values that are stored by an character codes in the table to which they have been associated. For example, the country code is stored in the Company table by its two-character country code, while the full name of the country is stored in the Country table.

When a user selects a search item of type ENUM_S, all the values possible for that search item appear, including both the descriptive string values and their associated character codes.
