Forecasts and Quotes

You can configure the Forecasts and Quotes feature by:

  • Configuring configurable fields
  • Configuring currency codes
  • Configuring currency conversion data
  • Configuring error messages and column headings
  • Creating products and product groups
  • Setting system parameters

Before you configure a forecast or quote, Onyx recommends that you specify a default currency code (DefaultCurrencyCode) in System Parameter Administration for the forecast/quote (it is optional). Otherwise, no currency code will be assigned when you save the forecast or quote.
