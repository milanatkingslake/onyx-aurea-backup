Adding a new filter to an object

Filters reduce the number of items returned when selecting an object. Use the following procedure to add a filter to the object you created in the procedure Adding a new object to Table Administration. For an example of an object with a filter, look at the Region object in Table Administration Item to see how it uses Country as its filter.

To add a filter to an object in Table Administration:

  1. If you haven’t already done so, use the Object Designer to create the object. The object must have the retrieveCollection and saveCollection methods. For more information about creating new OEAS objects, see Using the Object Designer.
  2. In Table Administration Item, locate the object that you created in Adding a new object to Table Administration, and enter a name for the new filter in the filterObjectName field.
  3. In the filterKeyPropertyName field, enter the name of the property that is the key for the object specified in filterObjectName.
  4. In the filterDescriptionPropertyName field, enter the name of the property that is the description for the filter specified in filterObjectName.
  5. In the filteredObjectPropertyName field, enter the name of the property of the object specified in the objectName field, that the filtering is based on.
  6. Click on the toolbar to save your changes.
  1. Refresh the Table Administration tool. When the object is selected in the Table list, the filter that is associated with that object now appears in the filter list.
