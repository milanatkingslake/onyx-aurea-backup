Create subscribe method package

The subscriptionItemSubscribe package calls the OEAS business object method that subscribes the user to the company record. This package contains attributes that represent input parameters to the method.

To create the retrieveMatchcodeCount package:

  1. In the Process Explorer window, right-click Library and then select Maintain Library from the shortcut menu.

The Library Maintenance dialog appears.

  1. In the Library Maintenance dialog, click Create under the Packages list window.
  2. Enter the settings shown in the following table, and then click Next to proceed.
Field Setting

Package Name

subscriptionItemSubscribe

Package Type

Main Package (default setting when the Parent Package field is empty)

Parent Package

If this field is not empty, click No Parent to clear it

Allow multiple

Cleared (not checked).
  1. Click Add to add the following attributes to the new package:
Attribute name Data type

subscribedUserId

Text

subscriptionServiceId

Number

objectType

Number

objectPrimaryKey

Text

syncAction

Number

_statusType

Text
  1. Click Finish to save the new package.
  2. Click Close to complete the package creation.
