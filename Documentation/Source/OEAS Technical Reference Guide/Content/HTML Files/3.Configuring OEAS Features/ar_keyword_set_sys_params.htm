Setting system parameters

There are two system parameters that you can use to modify the default behavior of the Keywords feature. The following table describes these system parameters.

System parameter Description

KeyWordIncludeCount

Specifies how many times a keyword must be associated with an incident or a task, of a specific category, before it appears in the keywords list for that incident or task category

WorkTicketKeywordIncludeCount

Specifies how many times a keyword must be associated with a work ticket before it appears as a lookup value in the keywords list for work tickets

You can administer these system parameters using System Parameter Administration. Using this tool, you can set the parameter value that controls each system parameter. For details regarding System Parameter Administration and the system parameters of the Keywords feature, see System Parameter Administration Help.
