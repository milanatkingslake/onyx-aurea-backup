Configuring attachments for emails

End users can browse for new files, literature items, and existing files to attach to their email messages. After a file or literature item is selected, the physical file is uploaded to the file store and displayed in the end-user interface.

Configuring these file and literature item attachments for use in OEP involves the following:

  • Using Table Administration to specify file shares for storing files for email attachments.

For more information, see Configuring file paths for attachments.

  • Using Product Administration to identify literature items that end users can attach to emails.

For more information, see Creating literature products and "Product Administration Help."

  • Using System Parameter Administration to set the default behavior for OEAS system parameters that are used by the Email feature.

For more information, see Setting system parameters for the Email feature.
