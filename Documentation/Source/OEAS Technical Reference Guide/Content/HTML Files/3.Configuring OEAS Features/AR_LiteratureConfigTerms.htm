Configuring terms

Terms for literature orders represent the payment terms. When users create a literature order, they can select one of several pre-defined payment terms, such as Cash (full amount is due on delivery) or Net 30 (full amount is due within 30 days of delivery). A number of terms are installed by default during installation of the OEDB, but you can configure them to meet the needs of your organization.

You can configure the domain data for literature terms using Table Administration.

With Table Administration, you can:

  • Create new terms
  • Modify existing terms
  • Set terms to active or inactive
  • Delete existing terms

The Terms fields that are displayed by Table Administration are listed in the following table.

Field Description

description

A description of the payment terms available for literature processing.

active

Specifies whether the field is active or inactive. 1 is active and 0 is inactive.
