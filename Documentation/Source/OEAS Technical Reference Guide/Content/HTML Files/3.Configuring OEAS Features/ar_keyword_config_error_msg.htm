Configuring error messages

For the Keywords feature, you can configure end-user error messages. These messages can be used when you customize the application to perform additional validation or processing and require a message when the validation fails. You can also use these messages if you want to provide the user with additional information about the process.

These end-user error messages are maintained in the national language tables, and you can administer them using Table Administration. For additional information on configuring error messages, see National Language.
