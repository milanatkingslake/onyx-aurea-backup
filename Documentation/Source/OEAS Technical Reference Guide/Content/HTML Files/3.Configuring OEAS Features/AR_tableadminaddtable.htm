Adding a new object to Table Administration

Following is the procedure to add a new object to Table Administration.

To add a new object to Table Administration:

  1. If you haven’t already done so, use the Object Designer to create the object. The object must have the retrieveCollection and saveCollection methods. For more information about creating new OEAS objects, see Using the Object Designer.
  2. In Table Administration, select Table Administration Item from the Table list in the navigation area.
  3. Click on the toolbar. A new row appears at the bottom of the page.
  4. In the newly created row, enter a name for the new object in the objectName field.
  5. Tab (or scroll) over to the menuName field and enter the text string that you want to appear in the Table list for the new object.
  6. In the description field, enter the text string that you want to appear in the Table Administration description area for the object.
  7. Click on the toolbar to save your changes.
  8. Refresh the Table Administration tool. The new object has been added to the Table list.
