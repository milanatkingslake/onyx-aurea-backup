Creating label product data

You use the domain data for label products within the Product Administration tool when you are creating or modifying products. Label product data is displayed in the Label Product drop-down list box within the Literature Information data cell of the Product Administration edit area when you have selected a product from the navigation area.

Label product data is not installed by default. Rather, you create label product data for the Product Administration Label Product attribute within Product Administration itself, when you assign a product a literature type of “mailing label document” within the Literature Information data cell. Once you have assigned a product a literature type of “mailing label document,” that product is subsequently displayed in the Label Product drop-down list box.
