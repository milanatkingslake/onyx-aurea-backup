Scripting

When you work with the Script Designer, you are designing, creating, and specifying the behavior of a specific script. The following material describes how you can configure the behavior of all scripts in Onyx Enterprise Application Server (OEAS) and end-user applications such as Onyx Employee Portal (OEP).

You can do the following to configure behavior for the Scripting feature:

  • Configure the scriptsession.status standard configurable field
  • Configure user-defined fields to maintain different data from those maintained by steps created with the Script Designer
  • Configure error messages and column headings
  • Set the system parameter OEPScriptCacheLife

For information on designing, creating, and publishing a script, see Help for the Script Designer.
