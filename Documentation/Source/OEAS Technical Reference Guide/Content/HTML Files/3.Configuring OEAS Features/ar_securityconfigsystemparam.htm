Setting system parameters

One way you can modify the default behavior of the Security feature is to configure the system parameter GlobalSessionTimeOut. The following table briefly describes this system parameter.

System Parameter Description

GlobalSessionTimeout

Determines when a user session times out.

You can administer this system parameter using System Parameter Administration. Using this tool, you can set the parameter value that controls this system parameter. For details regarding System Parameter Administration and GlobalSessionTimeOut, see System Parameter Administration Help.
