Products

Although products themselves are domain data to several other features of the application, you can configure certain attributes of the Products feature to assist you in creating them. You can configure the Product feature by:

  • Configuring configurable fields
  • Configuring currency codes
  • Configuring error messages and column headings
  • Creating label product data
  • Configuring printers
  • Setting system parameters

You can additionally configure domain data for the Customer Products feature. See Customer products.
