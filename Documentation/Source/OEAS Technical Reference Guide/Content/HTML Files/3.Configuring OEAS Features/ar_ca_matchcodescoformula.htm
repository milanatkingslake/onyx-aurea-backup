Company match code formula

The company match code formula is made up of the three sections defined in the following table. The company match code is always 25 characters long. If the company record contains no data corresponding to a particular character position in the code, the stored procedure substitutes an asterisk (*).

Section Length (characters) Component

Postal code

5

The first five characters of a company's postal code

Company name

8

This section is composed of up to three parts of the company™s name, using the following sequence:

  • The first six characters of the entire company name
  • The first character of the second string in a company name
  • The first character of the third string in a company name

Address

12

The first 12 characters of the address1 property for the company
