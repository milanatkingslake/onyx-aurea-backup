retrieveCollection (fileLink)

retrieveCollection method

The retrieveCollection method of the fileLink object returns a collection of fileLink objects.

retrieveCollection standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieves a collection of fileLink objects

Example

      </fileLink>
