lockedBy property

The lockedBy property of an object is not used in this version of OEAS.

This is a generic topic that applies to all properties with the same name. The remarks section may contain details specific to a given object.

Data type

string

Access

Read-only; for more information,  see Read Only attribute.

Remarks

This property exists for compatibility with previous versions of the OEAS system. The lockedBy property has no physical representation in the database.

When retrieved, the lockedBy property should always return NULL (null="1").

When performing an insert or update method, you never need to set a value for the lockedBy property.
