delete (company)

delete method

The delete method of the company object logically deletes an object from the database. Secondary objects, as defined in the relationship section of the Onyx Enterprise Dictionary, may also be logically deleted or updated depending on the logicalCascadeType elements of its relationships.

delete standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Deletes the object instance
