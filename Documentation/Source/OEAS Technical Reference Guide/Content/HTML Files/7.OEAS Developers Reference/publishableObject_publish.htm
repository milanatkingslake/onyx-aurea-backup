publish (publishableObject)

publish method

The publish method of the publishableObject object retrieves domain data and writes it to an XML file.

Parameters

Parameter

Direction

Data type

Content type

Description

primaryIdList

in

string

n/a

A semi-colon delimited list of primary IDs. When you specify NULL, all active publishableObjects are published.

filePath

in

string

n/a

The location in which to publish the domain data files. This location can be a path on the local server or a UNC on a remote server. Typically, domain data is written to the machine that serves the files for the OEP-Offline client. The path you specify must allow read/write operations by the OnyxEASUser.

Steps

Step data

Step type

Description

OnyxPublishableObjectDAC.PublishableObject

com

Calls the extract method and persists the results

Remarks

When the publish method succeeds, it returns an empty parametersReturn node.

</parameters>
