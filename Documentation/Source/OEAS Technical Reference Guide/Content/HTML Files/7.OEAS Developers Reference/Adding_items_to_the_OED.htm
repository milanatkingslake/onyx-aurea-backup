Adding items to the OED

To add an item:

  1. In the navigation area, select an item of the type that you want to add and click .

    The new item appears in the navigation area followed by an asterisk ( * ) (For example: newProperty1*). An asterisk appears next to any item in the navigation area that has not yet been saved.

  2. In the edit area, define the object attributes or physical model attributes.

    Once you modify an attribute, the item, as well as its parent object, appears in the navigation area with an asterisk ( * ) until the changes are saved.

  3. Click to save your changes.

To add an item using the Add Item button:

  • When you are editing an item that has an Add Item button in the lower right corner of the edit area, press this button to add another item of the same element type.
