saveCollection (group)

saveCollection method

The saveCollection method of the group object performs multiple actions at once: inserting and updating object instances, as appropriate.

saveCollection standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Inserts and updates object instances

Remarks

The delete action is not supported by the saveCollection method on the group object. However, you can deactivate a group with an update action by setting the active property to zero.

Example

Parameters

<parameters>

   <groups objectType="group" collection="1">

      <group objectType="group" action="update">

         <groupId>PR</groupId>

         <managerId>PhilD</managerId>

         <escalationId>PhilD</escalationId>

      </group>

      <group objectType="group" content="all" action="insert">

         <groupId>R&amp;DV</groupId>

         <parentGroupId>dev</parentGroupId>

         <description>R&amp;D Vendors</description>

         <authorityId>2</authorityId>

         <detailAccess>2</detailAccess>

         <managerId>AliceS</managerId>

         <escalationId>AliceS</escalationId>

         <emailAlias>R&amp;DV@global.com</emailAlias>

         <updateBy>sa</updateBy>

         <updateDate>2001-12-07 13:02:46</updateDate>

   </groups>

</parameters>
