searchQuery Object

searchQuery object

The searchQuery object is a transactional object representing a user's SQL query persisted in the database. A searchQuery belongs to a specific view in the database identified by the searchViewId.

The searchQuery object has one key property, primaryId, that uniquely identifies the searchQuery in the database. Once the object is created, the value of this property cannot be changed.

Methods

Method

Description
