enumerationParametersXML (searchViewDetail)

enumerationParametersXML property

The enumerationParametersXML property of the searchViewDetail object contains the method parameters required for the method call to the enumerationMethod.

Data type

string (maximum length)

Access

Read/write

Remarks

This property contains the XML input to the enumerationMethod.

When the XML is a string it needs to be escaped properly. For instance for region.retrieveList, use the following.

<enumerationParametersXML>

   <parameters>

     <countryCode>US</countryCode>

   </parameters>

</enumerationParametersXML>
