retrieveList (role)

retrieveList method

Remarks

This method supports paging of the result set with the pageSize and startRowIndex attributes on the rowSet parameter.

--------------------------------------------------------------------------------

This method might retrieve only a limited list of data, and not always the entire list of data available. You can use the Onyx Configuration Utility to configure the maximum rows returned. For more information, see Paging functionality.
