familyId (company)

familyId property

The familyId property of the company object reflects a unique numerical ID number assigned to a family of company objects.

Data type

i4

Access

Read/write

Remarks

The familyId property is used to indicate the unique numerical ID assigned to a family of company records as defined in the company table of the Onyx Enterprise Database. A familyId identifies a unique identification number given to each kind of family created and is linked to the company_family table. This table allows the user to get information such as the family name and family code.

For example, the familyId of 10 could identify the companyFamily object that ties a familyCode of "GLOBAL" to a familyDetail of "Global Enterprises Demo Corporation".
