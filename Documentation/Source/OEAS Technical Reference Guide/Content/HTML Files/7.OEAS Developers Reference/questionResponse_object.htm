questionResponse Object

questionResponse object

The questionResponse object represents a valid question-response pair. In order to constrain the possible answers to a question, you create a questionResponse object for each of the possible answers. The questionId and responseId, together with the sideId from the context XML, form the primary key for this object.

Methods

Method

Description

retrieveCollection

Returns a collection of questionResponse objects based on the search criteria supplied in the parameters XML

retrieveListBySurvey

Returns the list of responses to each question in a survey

saveCollection

Performs multiple actions at once: inserting, updating, and deleting questionResponse objects, as appropriate
