incidentProduct object

The incidentProduct object enables users to choose from a predefined list of valid product names to associate with a given incident object. You configure product names according to your organization's needs using the Product Administration tool.

The incidentProduct object has two key properties, primaryId and incidentCategory, that uniquely identify the incidentProduct in the database. Once the object is created, the values of these properties cannot be changed.
