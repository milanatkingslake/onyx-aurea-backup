alternateId (incidentProduct)

alternateId property

The alternateId property of the incidentProduct object relates an incidentProduct to an object in a foreign database or custom table.

Data type

string (maximum length)

Access

Read/write

Remarks

OEAS saves and retrieves this property, but it does not reference any other data in the Onyx Enterprise Database. To relate this property to another database or custom table, you must create a custom component.
