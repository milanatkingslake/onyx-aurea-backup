Deleting items

Deleting an item is much different than disabling an item. Disabling an item simply makes the item unavailable for use. You can make the item available for use again by reactivating the item. However, when you delete an item it is physically deleted from the OED. You cannot reactivate a deleted item. Instead, you must re-create the item or copy it from an earlier saved version of the OED. Be very careful when deleting items from the OED.

Note: If you delete an item and then add it back later, the Object Designer will generate a different ID (GUID) for the item. As far as the metadata is concerned, this is a different item even though it might have the same name as the original.
