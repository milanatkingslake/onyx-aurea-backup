primaryDiscount (forecastLine)

primaryDiscount property

The primaryDiscount property of the forecastLine object contains the discount amount in percent or absolute primary currency units depending on discountType.

Data type

float

Access

Read/write

Remarks

The discount is either an absolute amount in the primary currency or a percent, depending on the setting of the discountType property.

  • Percent values range from 0.0 to 100.0.
  • Amounts can be any number in the chosen currency.

confidenceFactor x quantity ( primaryUnitPrice - primaryDiscount ) = forecastLine total

1.00 x 1 ( $2560.00 - $640.00 ) = $1920.00
