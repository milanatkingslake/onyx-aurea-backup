actionBatchXml collectionItemPath
collectionItemPath

The collectionItemPath actionBatchXml element specifies the XPath that identifies the set of source list item nodes in the object collection list. The path is relative to the sourceList node. This element is only required if you are using a source list. For an example of an actionBatchXml document using a source list, see Processing an actionBatchXml document.

Cardinality

1 (required if sourceList is present)

Default value

none
