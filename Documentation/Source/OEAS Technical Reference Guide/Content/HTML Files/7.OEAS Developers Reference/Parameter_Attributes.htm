Parameter attributes

When you select a parameter in the Object Designer, the attributes for that parameter display in the edit area.

The attributes are defined as follows:

  • ID: The globally unique identifier(GUID) assigned by the Onyx Enterprise Dictionary. This attribute is read-only.
  • Name: Type the name you want to assign to this parameter. Parameter names must be conform to XML naming conventions.
  • Description: Type the description for this parameter.
  • Data Type: Select the type of data to be stored in this parameter:
  • For simple types, select a standard XML-data type.
  • For objects, select object.
  • Object Type: Select the name of the object referenced in the Data Type. This list includes objects marked for deletion and objects that are disabled, both which are marked as such.
  • For strings, select string.
  • Direction: Select the direction in which the method uses the parameter.
  • For parameters used on input only, select in.
  • Content In: Select the content type for the input parameter. NOTE: This field will appear below the Disabled field.

If Content In is Partial, then select property to be used for the object key in the Available Properties box. Then click the right arrow to move the property to the Selected Properties box. Repeat this step for each property you want to add to this key.

To remove a selected property, highlight the property in the Selected Properties box. Then click the left arrow to move the property to the Available Properties box.

  • For parameters used on output only, select out.
  • Content Out: Select the content type for the content out parameter. NOTE: These fields will appear below the Disabled field.

If Content Out is Partial, then select the property to be used for the object key in the Available Properties box. Then click the right arrow to move the property to the Selected Properties box. Repeat this step for each property you want to add to this key.

To remove a selected property, highlight the property in the Selected Properties box. Then click the left arrow to move the property to the Available Properties box.

For retrieveList methods, set the Object Type to rowSet, the Content Out to All, and enter Column Names be used in the result set. Click the Add button to move the column to the Selected Columns box. Repeat this step for each column you want to add to the result set.

Use the Move Up, Move Down, and Delete buttons to structure the result set.

  • For parameters used on input and output, select inOut.
  • Content In: Select the content type for the input parameter. NOTE: This field will appear below the Disabled field.

If Content In is Partial, then select property to be used for the object key in the Available Properties box. Then click the right arrow to move the property to the Selected Properties box. Repeat this step for each property you want to add to this key.

To remove a selected property, highlight the property in the Selected Properties box. Then click the left arrow to move the property to the Available Properties box.

  • Content Out: Select the content type for the content out parameter. NOTE: These fields will appear below the Disabled field.

If Content Out is Partial, then select property to be used for the object key in the Available Properties box. Then click the right arrow to move the property to the Selected Properties box. Repeat this step for each property you want to add to this key.

To remove a selected property, highlight the property in the Selected Properties box. Then click the left arrow to move the property to the Available Properties box.

  • Default Value: Enter the default value for this parameter using proper syntax. This value is optional.
  • Installed By: Type the name of the vendor or application adding this parameter.
  • Collection: Check this box to define this parameter as a collection.
  • Disabled: Check this box to disable this parameter. Clear the box to enable the object.
