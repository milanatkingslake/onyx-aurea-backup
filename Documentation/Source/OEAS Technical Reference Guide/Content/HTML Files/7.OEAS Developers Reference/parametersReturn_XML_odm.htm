parametersReturn XML (data manager)

parametersReturn XML

The parametersReturn is a part of the returnXml top-level node and contains the results of the call to the Query Engine. The returnXml node also contains the methodStatus, success or failure, and customData. The client application can optionally use the customData node; Onyx step components ignore it.

The ODM passes the returnXml node back to the Onyx Transaction Manager, which returns it to the client that submitted the request. The contents of parametersReturn depends on the type of method that the ODM processes.

parametersReturn contents
Method type parametersReturn content

custom

Any set of scalar inOut or out parameters

retrieve

One content="all" object

retrieveCollection

A collection of content="all" objects

insert, update, and delete

One content="keysOnly" object

saveCollection

A collection of content="keysOnly" objects

retrieveList

A rowSet object
