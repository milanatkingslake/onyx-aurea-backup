locked

locked property

The locked property of an object is not implemented by this version of OEAS.

This is a generic topic that applies to all properties with the same name. The remarks section may contain details specific to a given object.

Data type

boolean

Access

Read-only; for more information,  see Read Only attribute.

Remarks

The locked property will always return NULL. This property exists for compatibility with previous versions of the OEAS system. The locked property has no physical representation in the database. You never need to set a value for it when performing an insert or an update.
