retrieveCollection (fileLocation)

retrieveCollection method

The retrieveCollection method of the fileLocation object returns a collection of fileLocation objects.

retrieveCollection standard method information

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Retrieves a collection of fileLocation objects

Example

Input Parameter:

<parameters>

   <fileLocations objectType="fileLocation" collection="1">

      <fileLocation objectType="fileLocation"/>

   </fileLocations>

</parameters>

Output Parameter:

<parametersReturn>

   <fileLocations objectType="fileLocation" collection="1" content="all">

      <fileLocation objectType="fileLocation" content="all">

         <primaryId>1000160</primaryId>

         <description>This is the physical location of attachment files</description>

         <pathName>\\docappserver\OEPFileShare</pathName>

         <insertBy>sa</insertBy>

         <insertDate>2002-01-25 22:30:25</insertDate>

         <updateBy>sa</updateBy>

         <updateDate>2002-01-25 22:30:25</updateDate>

         <active>1</active>

      </fileLocation>

   </fileLocations>

</parametersReturn>
