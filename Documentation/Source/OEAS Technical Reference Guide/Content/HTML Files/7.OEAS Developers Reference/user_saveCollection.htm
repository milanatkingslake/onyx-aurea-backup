saveCollection (user)

saveCollection method

The saveCollection method of the user object performs multiple actions at once: inserting, updating, and deleting object instances, as appropriate.

saveCollection standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters.

OnyxOSLFStep.OSLF

com

Encrypts the value supplied for the user's password property

OnyxDM.DataManager

com

For update actions, calls a pre-procedure to check the input data and ensure that the system administrator's user account is not being deactivated

For delete actions, calls a pre-procedure to delete the user's corresponding networkUser to aviod violating a foreign key

Inserts, updates, and deletes object instances

For inserts and updates, calls a post-procedure to manage the data stored for the user's corresponding networkUser, which is stored separately

Example

The following example shows the content of the input parameters and output parametersReturn used to save a collection of user object instances.
