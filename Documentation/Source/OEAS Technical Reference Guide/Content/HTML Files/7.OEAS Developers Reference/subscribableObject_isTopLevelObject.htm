isTopLevelObject (subscribableObject)

isTopLevelObject property

The isTopLevelObject property of the subscribableObject object indicates whether an object type is a top-level object in the subscription tree.

Data type

boolean

Access

Read/write

Remarks

When a object is a top-level object, such as a company or individual, the synchronization server does not cascade subscriptions from the parent of the object.

Valid values of this property are:

false (0): The instance is not a top-level object.

true (1): The instance is a top-level object.
