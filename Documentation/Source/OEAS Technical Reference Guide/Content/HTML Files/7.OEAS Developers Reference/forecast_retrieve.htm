retrieve (forecast)

retrieve method

The retrieve method of the forecast object searches the database based on a given primaryId and returns an object instance.

retrieve standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxMasterDetailRetrievalStep.MasterDetailRetrieval com Propagates the ownerId into the forecastLines detail collection and creates the collection node if necessary
OnyxDM.DataManager com Retrieves the forecast header
forecastLine.retrieveCollection method Retrieve a collection of forecastLine objects
