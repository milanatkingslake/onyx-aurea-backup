type (methodCallDetail)

type property

The type property of the methodCallDetail object specifies whether the detail represents a single value or a list of values.

Data type

string

Access

Read/write

Remarks

The following table summarizes the valid values for this property.

Value

Description

scalar (input or output)

single text value, or

single ID/description pair (output only)

list (output only)

multiple text values, or

multiple ID/description pairs

For details on ID/description pairs see the idPath and descriptionPath topics.
