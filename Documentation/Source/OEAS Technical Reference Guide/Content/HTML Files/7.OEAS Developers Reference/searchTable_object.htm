searchTable Object

searchTable object

The searchTable object is a domain object representing a table or view that exists in the database schema. The Onyx database schema includes several default views. For views, the name ends with "_LM_VIEW" such as company_lm_view. Views facilitate queries on multiple tables simultaneously. A database administrator must set up the database schema for a custom searchTable object to work with list management.

The searchTable object has one key property, primaryId, that uniquely identifies the searchTable in the database. Once the object is created, the value of this property cannot be changed.
