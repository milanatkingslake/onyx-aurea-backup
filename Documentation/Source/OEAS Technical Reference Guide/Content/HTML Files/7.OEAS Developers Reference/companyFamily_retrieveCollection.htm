retrieveCollection (companyFamily)

retrieveCollection method

The retrieveCollection method of the companyFamily object returns a collection of objects based on the search criteria supplied in the parameters XML.

retrieveCollection standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieves a collection of companyFamily objects

Example

The following example shows the content of the input parameters and output parametersReturn used to retrieve a companyFamily object instance with the code GBLD.

Parameters

<parameters>

   <companyFamilies objectType="companyFamily" collection="1">

      <companyFamily objectType="companyFamily">

         <familyCode>GBLD</familyCode>

      </companyFamily>

   </companyFamilies>

</parameters>

parametersReturn

<parametersReturn>

   <companyFamilies objectType="companyFamily" collection="1" content="all">

      <companyFamily objectType="companyFamily" content="all">

         <primaryId>1000342</primaryId>

         <familyCode>GBLD</familyCode>

         <familyName>Global Distribution</familyName>

         <familyDetail null="1"/>

         <insertBy>sa</insertBy>

         <insertDate>2002-04-01 21:47:16</insertDate>

         <updateBy>sa</updateBy>

         <updateDate>2002-04-01 21:47:16</updateDate>

         <onyxTimestamp>0000000024C04A42</onyxTimestamp>

         <active>1</active>

      </companyFamily>

   </companyFamilies>

</parametersReturn>
