appointment object

The appointment object enables users to insert, update, retrieve, and delete appointment object instances. Appointments are based on calendar items stored in a central server and manipulated in a client application.

The appointment object has one key property, primaryId, that uniquely identifies object instances in the database. Once the object is created, the value of this property cannot be changed.

Methods

Method Description
delete Deletes a specified appointment object instance
insert Saves a new appointment object instance
retrieve Returns an existing appointment object instance
retrieveListByAssociation Returns a list of appointment records that match the supplied association data
update Updates an appointment object instance
