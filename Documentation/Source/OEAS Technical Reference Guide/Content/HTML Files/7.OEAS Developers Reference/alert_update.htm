update (alert)

update method

The update method of the alert object saves modifications to an existing alert object instance.

update standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com

Calls a pre-procedure to determine if update would create multiple alerts for a single owner; if true, prevents the update and returns an error message

Modifies the specified alert record in the database

Example

The following example shows the content of the input parameters and output parametersReturn used to update the expirationDate property of an alert object instance.

Parameters

<parameters>

   <alert objectType="alert" content="all">

      <expired>0</expired>

      <onyxTimestamp>00001FBE56065336</onyxTimestamp>

      <primaryId>FF85D479-B5F0-4159-8553-30DCEAFADCA8</primaryId>

      <ownerId>18675D72-C6E1-4A35-85D7-19F961AD1F10</ownerId>

      <ownerType>2</ownerType>

      <text><![CDATA[Global headquarters has > 2500 employees.]]></text>

      <expirationDate>2002-05-30 00:00:00</expirationDate>

      <insertBy></insertBy>

      <insertDate></insertDate>

      <updateBy></updateBy>

      <updateDate></updateDate>

   </alert>

</parameters>

parametersReturn
