retrieveCollection (fulfillmentType)

retrieveCollection method

The retrieveCollection method of the fulfillmentType object returns a collection of fulfillmentType objects.

retrieveCollection standard method information

Steps

Step Data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieves a collection of object instances
