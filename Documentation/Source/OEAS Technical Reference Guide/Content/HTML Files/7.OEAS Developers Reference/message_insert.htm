insert (message)

insert method

The insert method of the message object saves a message object instance.

insert standard method information

Step type

Description

com

OnyxDM.DataManager

Adds a message object instance to the database

Remarks

The method fails when the input XML contains no user or group recipient, or contains no active user recipients.
