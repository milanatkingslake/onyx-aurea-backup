cascadeToPrimary (subscriptionRelationshipRole)

cascadeToPrimary property

The cascadeToPrimary property of the subscriptionRelationshipRole object indicates whether the role specifies a cascading subscription to the primary object of the relationship when the secondary object of the relationship is subscribed.

Data type

boolean

Access

Read/write

Remarks

When the user does not belong to a role that has a subscriptionRelationshipRole defined for the subscriptionRelationship to which a subscribed object belongs, the cascade method uses the value of the defaultCascadeToPrimary property of the subscriptionRelationship object.

Valid values of this property are:

false (0): Do not subscribe the primary object of the relationship.

true (1): Subscribe the primary object of the relationship.
