retrieveListByPartner (incident)

retrieveListByPartner method

The retrieveListByPartner method of the incident object returns the list of incidents assigned to a partner depending on the input parameters.

Parameters

Parameter Direction Data type Content type Description

partnerId

in

string

n/a

Filters by the incident records for a partner

incidentSecondaryIdList

in

string

n/a

The list of incident secondaryIds to search

userIdList

in

string

n/a

The list of partner users that may have access to the customer

expandUserList

in

boolean

n/a

The flag if the partner groups should be retrieved for the users in the list

partnerGroupIdList

in

string

n/a

The list of partner groups that may have access to the customer

expandPartnerGroupList

in

boolean

n/a

The flag the if partner groups should be expanded using parentId

assignedToList

in

string

n/a

A semicolon-delimited list of assignedTo values

categoryIdList

in

string

n/a

A semicolon-delimited list of categoryId values

statusIdList

in

string

n/a

A semicolon-delimited list of statusId values

sourceIdList

in

string

n/a

A semicolon-delimited list of sourceId values

priorityIdList

in

string

n/a

A semicolon-delimited list of priorityId values

includeMask

in

string

n/a

Identifies whether the phone number mask is displayed on the output or not

rowSet

out

object

all

Contains the rowSet column definitions and the incident table rows matching with the owner and category criteria

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates the input parameters

OnyxDM.DataManager

com

Retrieves the list of incident objects based on supplied criteria
