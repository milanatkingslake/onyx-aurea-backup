usageObjectType (methodCall)

usageObjectType property

The usageObjectType property of the methodCall object contains a valid object type enumeration.

Data type

i4

Access

Read/write

Remarks

You can use this property to associate methodCall objects with the feature for which it was intended. The purpose of this property is to enable method call consumer applications to filter method calls by feature. The following table lists the valid values for this property.
