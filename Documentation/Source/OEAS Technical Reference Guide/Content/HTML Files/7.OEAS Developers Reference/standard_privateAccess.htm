privateAccess

privateAccess property

The privateAccess property of an object indicates whether the object is visible to all users or only to the user who last updated it.

This is a generic topic that applies to all properties with the same name. The remarks section may contain details specific to a given object.

Data type

boolean

Access

Read/write

Remarks

The privateAccess property works in conjunction with the updateBy property. If privateAccess is set to true, the object will not be returned by a retrieve, retrieveList, or retrieveCollection method, unless the user calling the method is the last user to update the instance of the object. The method checks for a match between the current userId in the contextXml and the userId in the updateBy property.

Valid values for the privateAccess property are:

false (0): (Default) The object is visible to the all users.

true (1): The object is only visible to the last user to update it as recorded in the updateBy property of the object.
