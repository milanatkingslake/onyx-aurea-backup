userSubTypeId (individual)

userSubTypeId property

The userSubTypeId property of the individual object contains the numerical ID which identifies the subcategory to which an individual belongs.

Data type

i4

Access

Read/write

Remarks

The userSubTypeId property is an integer value that can be configured using the Reference Table administration tool. An example of a userSubTypeId might be 327, which could have a description of "Distributor" in the reference objects. There is a parent child relationship between a userTypeId and a userSubTypeId.
