actionBatchXml_sourceListMap_sourceElementPath
sourceListMap/sourceElementPath

The sourceListMap/sourceElementPath actionBatchXml element specifies the path to the element from the sourceList object that is copied into the template provided in methodInputs, before executing the OEAS method call. This path must be relative to the node pointed by the collectionItemPath.

Cardinality

1 (required)

Default value

none
