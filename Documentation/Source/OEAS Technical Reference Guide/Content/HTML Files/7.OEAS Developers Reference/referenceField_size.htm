size (referenceField)

size property

The size property of the referenceField object describes the length, in characters, of a text field.

Data type

ui1

Access

Read/write

Remarks

This property is valid only for referenceField objects whose type property is "Freetext."

For referenceField objects provided by Onyx, this property is valid only for user-defined text fields, such as customer.user1 - customer.user10.
