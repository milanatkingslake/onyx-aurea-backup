response Object

response object

The response object contains an answer that users can attach to a question. Responses are reusable in multiple questions. You attach a response to a question using the questionResponse object. To specify a response users provide the response text and response type.

Methods

Method

Description

retrieveCollection

Returns a collection of response objects based on the search criteria supplied in the parameters XML

saveCollection

Performs multiple actions at once: inserting, updating, and deleting response objects, as appropriate

Properties

Property

Description

active

Specifies whether the domain data the object provides is available to the user interface

insertBy

Contains the user ID of the Onyx user who initially saves the object

insertDate

Contains the time and date on which the object is initially saved

primaryId

Contains the unique ID assigned to the object when it is first saved to the database

responseType

Indicates whether a response can have free text associated with it

text

Contains the wording of the response if not a free text question

updateBy

Contains the user ID of the Onyx user who last saved modifications to the object

updateDate

Indicates the time and date on which modifications to the object were last saved
