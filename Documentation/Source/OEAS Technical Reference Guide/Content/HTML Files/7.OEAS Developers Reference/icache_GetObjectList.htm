GetObjectList (ICache3)
GetObjectList method

The GetObjectList method of the ICache3 interface retrieves an XML string from the Onyx Enterprise Dictionary (OED) cache for a given logical application. The string is composed of the businessObject collection. The returned XML structure is abbreviated and does not include child collection elements (properties and methods) of the businessObject elements.

Syntax

IDL

HRESULT GetObjectList(

    [in] BSTR ApplicationName, 

    [out] BSTR* ObjectList, 

    [out, retval] OEDCacheStatusEnum* Status);

IDL parameter descriptions

ApplicationName

[in] A string containing the logical application name that determines which subset of the cache to use for a query.

ObjectList

[out] A string containing a partial OED XML structure including all the business objects.

Status

[out, retval] An enumeration of type OEDCacheStatusEnum that contains the status of the method call.

public OEDCacheStatusEnum GetObjectInstance(

    string ApplicationName,

    out string ObjectList)

C# parameter descriptions

ApplicationName

[in] A string containing the logical application name that determines which subset of the cache to use for a query.

ObjectList

[out] A string containing a partial OED XML structure including all the business objects.

Return values

This method returns an enumeration of type OEDCacheStatusEnum that contains the status of the method call. See the OEDCacheStatusEnum topic for a description of individual enumeration values.

Example

The following Visual Basic code demonstrates how to use the GetObjectList method to retrieve the OED businessObjects XML structure.

Note: This code is taken from an IStep_Execute function which provides the ApplicationName parameter.

Dim oCache As New OnyxDictionaryCacheLib.DictionaryCache

Dim status As OnyxDictionaryCacheLib.OEDCacheStatusEnum

Dim strObjectList As String

Set oCache = New OnyxDictionaryCacheLib.DictionaryCache

status = oCache.GetObjectList(ApplicationName, strObjectList)

The XML is returned in the strObjectList parameter. The following snippet illustrates the structure returned.

      <description>actionBatch</description>

      <system/>

      <installedBy>Onyx</installedBy>

      <version>5.00.000.0.0.0</version>

      <disabled/>

</businessObjects>
