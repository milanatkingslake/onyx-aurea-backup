retrieve (incident)

retrieve method

The retrieve method of the incident object searches the database based on the properties defined by the object's primary objectKey entry and returns an object instance.

retrieve standard method information

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Retrieve the specified incident object from the database

Uses a database function mapped to the alertId property to return an alertId value

Uses a database function mapped to the ownerName property to return an ownerName value based on ownerId

Uses a database function mapped to the tags property to return a delimited list of tag values
