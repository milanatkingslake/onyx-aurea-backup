Overview (Add Method)

Adding a method

You can add custom methods to the OED that are not currently defined. By adding methods to a business object you can persist or retrieve information from the database in new ways.

Scenario

This sample adds a retrieveListByUser method to the keyWord object. By adding this method, you can retrieve tag records from the database that were entered by the current user. If a user does not remember what tag they used for an incident, they can use this method to retrieve a list of all the tags that they entered. This represents a new way of returning tag information that was not previously defined.

Guidelines

Naming Custom Stored Procedures

To guarantee that Onyx does not overwrite your customization during a schema upgrade, use the following naming convention.

c?_name

where:

c

Identifies the package as a customization

?

Represents a one character abbreviation for a database object

For packages, use 'p'

name

Describes the stored procedure

For example, you might name a custom stored procedure for this custom retrieveList method: cpKeyWordRetrieveListByUser.
