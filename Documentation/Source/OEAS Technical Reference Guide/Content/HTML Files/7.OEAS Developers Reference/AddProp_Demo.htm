Demo (Add Property)
Demonstration

This demonstration assumes that you have made the modifications to the OED as outlined in the previous Steps topic for Adding a property. To demonstrate the success of our modifications, we insert a new company record into the database including a stock ticker symbol and retrieve it again to verify that the record includes the new property.

We call the Onyx Transaction Manager (OTM) with the following XML parameters.

Input parameters for the company insert method

<parameters>

   <company objectType="company">

      <primaryId/>

      <secondaryId/>

      <locked/>

      <url/>

      <marketSector/>

      <dunnsNumber/>

      <privateAccess/>

      <insertBy/>

      <lockedBy/>

      <phones objectType="phone" collection="1"/>

      <addresses objectType="address" collection="1"/>

      <stockTickerSymbol>TTS</stockTickerSymbol>

   </company>

</parameters>

The OTM returns a key for the new record.

View the returnXml from the company insert method

<returnXml>

      <otmId>2558B103-6B21-40B1-91BF-3640D519AEFB</otmId>

      <methodStatus>

            <statusType>success</statusType>

      </methodStatus>

      <parametersReturn>

            <company objectType="company" content="keysOnly">

                  <primaryId>8DB2EEE2-D2EF-4BC0-8054-6354A46D4889</primaryId>

                  <phones objectType="phone" collection="1"></phones>

                  <addresses objectType="address" collection="1"></addresses>

            </company>

      </parametersReturn>

      <customData></customData>

</returnXml>

We use the key to retrieve the record we just created.

Input parameters for the company retrieve method

<parameters>

   <company objectType="company">

      <primaryId>8DB2EEE2-D2EF-4BC0-8054-6354A46D4889</primaryId>

      <phones objectType="phone" collection="1"/>

      <addresses objectType="address" collection="1"/>

   </company>

</parameters>
