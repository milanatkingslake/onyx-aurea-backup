profileId (userPreference)

profileId property

The profileId property of the userPreference object contains the primaryId of a UI profile that is associated with a preferenceName and userId.

Data type

i4

Access

Read/write

Remarks

The profileId is an object key for the userPreference object. This property identifies an OEP UI Configuration Framework (UCF) profile. OEP users can have multiple profiles, therefore, unique preferences with the same preferenceName must exist among a user's preferences.
