IEBusQE interface

This interface provides step components with access to the Onyx Enterprise Database. The interface exposes four methods: two for stored procedure calls and two for ad hoc SQL commands.

This interface has the following methods:

Method

Description

CallStoredProcedure

Executes a stored procedure in the application database.

CallStoredProcedureReturnExtendedInfo

Executes a stored procedure in the application database and returns column information.

Execute

Executes an ad hoc SQL command against the application database.

ExecuteReturnExtendedInfo

Executes an ad hoc SQL command against the application database and returns column information.

This interface has the following enumerations:

Enumeration

Description

epDirection

Indicates the direction (input, output) for a parameter of a stored procedure.

epType

Indicates the data type of a parameter.
