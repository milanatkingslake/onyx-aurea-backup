retrieveListByIdList (incident)

retrieveListByIdList method

The retrieveListByIdList method of the incident object returns the list of incidents identified by the incidentIdList method parameter.

Parameters

Parameter Direction Data type Content type Description

incidentIdList

in

string

n/a

A semicolon-delimited list of incident IDs

includeMask

in

string

n/a

Identifies whether the phone number mask is displayed on the output or not

rowSet

out

object

all

Contains the rowSet column definitions and the incident table rows matching with the owner and category criteria

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates the input parameters

OnyxDM.DataManager

com

Retrieves the list of incident objects based on the list of IDs

<includeMask></includeMask>
