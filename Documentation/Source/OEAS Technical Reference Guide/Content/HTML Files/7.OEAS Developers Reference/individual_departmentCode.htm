departmentCode (individual)

departmentCode property

The departmentCode property of the individual object reflects the company's department code associated with the individual.

Data type

string (maximum length)

Access

Read/write

Remarks

The departmentCode property is used to indicate the departments that can be assigned to an individual. The departmentCode is the ID for the department that an ndividual can be assigned. The departmentCode property is used by the database to retrieve a department description. A department code, such as ENGR may correlate to a department description of Engineering.

The departmentCode property is created and maintained with the Reference Table administration tool.

A department Code XML would be:

 <departmentCode>Marketing</departmentCode>
