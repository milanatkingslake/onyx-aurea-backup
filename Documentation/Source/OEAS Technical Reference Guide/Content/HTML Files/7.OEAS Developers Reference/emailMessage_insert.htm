insert (emailMessage)

insert method

The insert method of the emailMessage object creates an instance.

insert method standard information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com

Calls a pre-procedure that generates an integer value for the secondaryId when the element is null in the parameters XML

Calls a pre-procedure that generates an integer value for the threadId when the element is null in the parameters XML

Inserts an object instance
emailContact.saveCollection method Inserts a collection of emailContact objects
emailAttachment.saveCollection method Inserts a collection of emailAttachment objects

Remarks

Resolve all emailContacts before inserting an email message because email addresses for all emailContacts are required for an insert.
