parentId (referenceLookup)

parentId property

The parentId property of the referenceLookup object is used in conjunction with the parentId property in a referenceField object instance. These two parent properties are used to find the list of child referenceLookup rows for a parent referenceParameter row chosen from a parent referenceField list.

Data type

i4

Access

Read/write

Remarks

The parentId property is used to establish hierarchical relationships for referenceLookup objects.
