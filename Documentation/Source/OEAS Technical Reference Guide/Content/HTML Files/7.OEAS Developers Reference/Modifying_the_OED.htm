Modifying the OED

To modify the Onyx Enterprise Dictionary (OED), you must do so using Object Designer or other such Onyx tools that are designed specifically for this purpose. Onyx does not support customizing the OED directly.

Before you use the Object Designer, familiarize yourself with the design tool itself, how it works, and how it is intended to be used.

The customization process usually involves more than simply making modifications to the OED using the Object Designer. In many cases, when you customize OEAS business objects, customize the physical model, and write custom code (such as building your own step components) to take full advantage of the changes you made.
