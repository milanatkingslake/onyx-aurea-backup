OnyxTransformStep

The OnyxTransformStep transforms the step package XML before or after the Onyx Transaction Manager (OTM) processes another step of a method. While you can use it to run an XSL transform against any part of the step package, OnyxTransformStep is most useful when it transforms an actionBatchXml node within the parameters XML of an actionBatch.execute call. You can configure the actionBatch.executeForEMS method to call OnyxTransformStep to add automated functionality when the email monitoring system detects an incoming email message.

The OnyxTransformStep allows you to use validation code, business logic, and constants in the XSL template to create, update, and delete action nodes and produce functionality well beyond what possible using actionBatch.execute alone.

The programmatic identifier (ProgId) for the this component is OnyxTransformStep.TransformStep.
