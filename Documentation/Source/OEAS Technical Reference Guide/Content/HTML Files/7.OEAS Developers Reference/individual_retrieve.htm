retrieve (individual)

retrieve method

The retrieve method of the individual object searches the database based on the properties defined by the object's primary objectKey entry and returns an object instance.

retrieve standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxMasterDetailRetrievalStep.MasterDetailRetrieval

com

Propagates the primaryId value of the master object into the ownerId properties of the phones collection

OnyxMasterDetailRetrievalStep.MasterDetailRetrieval

com

Propagates the primaryId value of the master object into the ownerId properties of the addresses collection

OnyxDM.DataManager

com

Retrieves a single individual object

When companyId is non-NULL, returns companyName from the specified company object

When companyId is NULL, returns companyName from the individual object

Uses the database function mapped to the alertId property to return an alertId value

phone.retrieveCollection

method

Retrieves a phone collection

address.retrieveCollection

method

Retrieves an address collection

emailaddress.retrieveCollection

method

Retrieves an email address collection
