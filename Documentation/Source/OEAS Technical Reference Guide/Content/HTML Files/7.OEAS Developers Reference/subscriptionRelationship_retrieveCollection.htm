retrieveCollection (subscriptionRelationship)

retrieveCollection method

The retrieveCollection method of the subscriptionRelationship object retrieves a collection of object instances.

retrieveCollection standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Retrieves a collection of subscriptionRelationship objects

Example

This example retrieves the subscription relationships in which the parent object is an incident, the object type enumeration is 6.
