RetrieveStringList (NatLangProxy)
RetrieveStringList method

The RetrieveStringList method of the NatLangProxy interface obtains a range of National Language strings from the in-memory cache.

Syntax

Visual Basic/VBScript

Function RetrieveStringList(

AppName As String,

SiteID As Long,

LanguageCode As String,

StartID As Long,

EndID As Long) As Variant

Visual Basic/VBScript parameter descriptions

AppName

[in] A string that identifies the OEAS application to query.

SiteId

[in] A long integer that identifies the specific site to query within the application.

LanguageCode

[in] A string that identifies the spoken language of the string to be accessed.

StartID

[in] The integer ID of the start of the range to retrieve.

EndID

[in] The integer ID of the end of the range to retrieve.

Return Values

This method returns the list of strings in a two-dimensional array. The first dimension contains the integer ID of each string, and the second dimension contains the string itself. Unlike the RetrieveString method of this interface, RetrieveListString does not provide for automatic substitution of placeholders.

The C# version of this method returns a two-dimensional array as a System.Object type. To access the individual array elements the Object must be cast to Object[,] (an object array) and then either assign each element to a string or copy the value into a new string. For an example of this coding technique, see the C# code sample in the Example section of DictionaryProxy GetPropertyConstraints method.

Remarks

Except in cases of severe COM-related errors, the method does not utilize the error object. If the method fails to retrieve information from the cache for any reason (for example, the cache is unavailable, or if the search contained a non-existent ID range, language code, or application name), it returns an empty string instead of an array.

Example

The following example demonstrates how to use the RetrieveStringList method to obtain a consecutive series of strings from the National Language cache.

View the VBScript code sample

' Declare some variables

dim str_appname, str_lang, site_id, start_id, end_id, var_ResultArray

str_appname = "appname"

str_lang = "eng"

site_id = 1

start_id = 1415

end_id = 1419

dim oNatLangProxy

set oNatLangProxy = CreateObject("OnyxNatLangProxy.NatLangProxy")

' Call the method

var_ResultArray = oNatLangProxy.RetrieveStringList(str_appname, site_id, str_lang, start_id, end_id)

' Check to see if the method returned an array

if isArray(var_ResultArray) then

  dim size, x

  size = UBound(var_ResultArray, 1)

  for x = 0 to size

      ' Do something with the contents of the array

  next

end if

set oNatLangProxy = Nothing
