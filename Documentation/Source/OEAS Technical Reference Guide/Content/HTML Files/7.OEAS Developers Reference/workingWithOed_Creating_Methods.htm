Creating methods

When creating methods for custom objects, you can choose from eight valid types:

  • insert
  • update
  • delete
  • retrieve
  • retrieveCollection
  • saveCollection
  • retrieveList
  • custom

You typically describe the first six of these (insert, update, delete, retrieve, retrieveCollection, and saveCollection) as the standard SQL generation methods.

You always implement the retrieveList and custom method types through a stored procedure. These methods always use a "replace" type of stored procedure map.
