scheduledProcessDate property

The scheduledProcessDate property of actionBatch object contains the date on which the batch process is expected to begin.

Data type

dateTime

Access

Read/Write

Remarks

The scheduledProcessedDate property is set only once when the batch is scheduled to process. Attempts to change this value are acceptable only before the batch is processed. The scheduledProcessedDate value is created by the database during the initial scheduled for the process.

If this attribute is NULL, then the action batch is processed as soon as possible. If this attribute is NOT NULL, then the action batch is processed on or after the scheduled process date.
