retrieveListByUser (subscriptionRelationship)

retrieveListByUser method

The retrieveListByUser method of the subscriptionRelationship object retrieves subscription rules that have been applied to a particular relationship between two subscribable objects. When the specified user is a member of one or more roles for which role-based rules are defined for the relationship, the role-based rules override the default rules of the relationship. The role-based rules are specified by subscriptionRelationshipRole objects. Role-based rules, if they exist for the relationship and user, can effect the contents of the cascadeToPrimary, cascadeToSecondary, and filterExpression columns of the result set.

Remarks

The results of the retrieveListByUser method specify the subscription relationship rules that dictate whether related records are automatically subscribed or unsubscribed. The rules data in the parametersReturn are specific to a single relationship between two subscribable objects.
