saveCollection (surveyQuestion)

saveCollection method

The saveCollection method of the surveyQuestion object performs multiple actions at once: inserting, updating, and deleting surveyQuestion objects, as appropriate.

saveCollection standard method information

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

Inserts, updates, and/or deletes a collection of object instances
