insert method or action

The following table describes the behavior to expect for insert methods or insert actions on saveCollection methods.

Behavior based on the stored procedure return code value defines the entire result set. In this case, the return code column is listed as not applicable (N/A) because it is not included in the result set.

Behavior only applies to the single row when the return code column is applicable. This is an important distinction, because multiple rows with different meanings might be passed back when methods return a status of success or success with information.
