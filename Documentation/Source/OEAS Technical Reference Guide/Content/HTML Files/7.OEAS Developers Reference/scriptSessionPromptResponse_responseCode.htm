responseCode (scriptSessionPromptResponse)

responseCode property

The responseCode property of the scriptSessionPromptResponse object contains the code component of a response selected by the user.

Data type

string

Access

Read/write

Remarks

A response includes a code component and a value component. The code component is intended for script step branching rule logic and is not necessarily intended for display purposes in the user interface. The script engine populates output variables with the code component of the response.

For the masked edit control, the responseCode contains the raw, unmasked data.
