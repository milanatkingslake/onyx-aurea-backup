assignedId (task)

assignedId property

The assignedId property of the task object contains an ID from a foreign database or custom table.

Data type

string (maximum length)

Access

Read/write

Remarks

The assignedId property is used for integration with another database or a custom table. This value can be any string representing a value stored in an integrated database or a table that you created. The Onyx Enterprise Application Server does not use this property in any of its logic. To use this property meaningfully, you must create a custom step component and a corresponding custom method or step.
