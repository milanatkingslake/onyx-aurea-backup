userActive (subscriptionProfile)

userActive property

The userActive property of the subscriptionProfile object indicates whether the user object instance is active for a OEAS user who has a subscription profile.

Data type

string

Access

Read-only; for more information, see Read Only attribute

Remarks

Valid values of this property are:

false (0): The user is a deactivated OEAS user.

true (1): (Default) The user is an active (valid) OEAS user.
