retrieveList

The retrieveList method of the literatureShipVia object searches the database and returns the complete list of shipping types.

Parameters

This method has one output parameter:

Element

Data Type

Direction

Content Type

Description

rowSet

object

out

all

A rowSet that contains all shipping options for literature products.

Steps

The default step for this method is:

Step

Step Type

Description

OnyxDomainDataDAC.DomainData

com

Uses the wbospsgShipVia stored procedure to obtain the complete list of shipping options for literature products.

Remarks

This method has no input parameters.

The columns returned in the rowSet object are:

Column Ordinal

Column Heading

Description

1

Ship Via ID

The unique identifier for the shipping method. Use this value in the shipViaId property of the literatureOrder object.

2

Description

The user-readable description for the shipping method.
