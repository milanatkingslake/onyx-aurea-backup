SetImpersonation (ISession)

SetImpersonation method

The SetImpersonation method of the ISession interface enables impersonation for a user. You set impersonation for the user that you want to do impersonations not the user you want to impersonate. Impersonation enables the user identity of a system service or component such as OPS to perform actions on behalf of other users. This ensures that records appear as updated or inserted by the original user and not by the system user.

Syntax

IDL

HRESULT SetImpersonation(

    [in] const BSTR strSessionID,

    [in] const BSTR strAppName,

    [in] const BSTR strUserToImpersonate,

    [in] VARIANT_BOOL bImpersonate);

IDL parameter descriptions

strSessionID

[in] A string that contains a session ID returned by the authentication component.

strAppName

[in] A string that contains the application name.

strUserToImpersonate

[in] A string containing id of the user to impersonate. If the bImpersonate parameter is set to false, this parameter is ignored

bImpersonate

[in] A value that specifies whether to disable (0) impersonation for the session or to enable (1) the session to impersonate the user specified in the strUserToImpersonate parameter.
