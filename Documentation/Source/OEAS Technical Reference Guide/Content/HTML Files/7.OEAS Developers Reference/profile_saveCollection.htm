saveCollection (profile)

saveCollection method

The saveCollection method of the profile object performs multiple actions at once: inserting, updating, and deleting objects, as appropriate.

saveCollection standard method information

Steps

Step type

Description

com

Validates input parameters

OnyxDM.DataManager

Inserts, updates, and/or deletes a collection of profile objects
