secondaryDiscount (forecastLine)

secondaryDiscount property

The secondaryDiscount property of the forecastLine object contains the discount amount in percent or absolute secondary currency units depending on discountType.

Data type

float

Access

Read/write

Remarks

The discount is either an absolute amount in the secondary currency or a percent, depending on the setting of the discountType property.

  • Percent values range from 0.0 to 100.0, inclusive.
  • Amounts can be any number in the chosen currency.
