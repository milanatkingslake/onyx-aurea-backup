retrieveCollection (questionResponse)

retrieveCollection method

The retrieveCollection method of the questionResponse object returns a collection of objects based on the search criteria supplied in the parameters XML.

retrieveCollection standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Retrieves a collection of object instances

Example

Parameters

<parameters>

   <questionResponses objectType="questionResponse" collection="1">

      <questionResponse objectType="questionResponse">

         <questionId/>

         <responseId/>

         <responseText/>

         <sequenceNumber/>

         <insertBy/>

         <insertDate/>

         <updateBy/>

         <updateDate/>

         <active>1</active>

      </questionResponse>

   </questionResponses>

</parameters>
