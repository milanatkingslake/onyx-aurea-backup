action attribute

The action attribute is required only when calling the OTM to execute a saveCollection method. The action attribute instructs the saveCollection method to perform any of the following actions (methods) on a specified object: delete, insert, and/or update an object.

Valid values for the action attribute are:

Attribute value Action performed

(none)

Only the saveCollection method requires the action attribute.

delete

Invokes the delete method of the step component.

insert

Invokes the insert method of the step component.

update

Invokes the update method of the step component.

Most saveCollection methods support all three actions. A few of them, however, perform only one or two of the possible three actions.
