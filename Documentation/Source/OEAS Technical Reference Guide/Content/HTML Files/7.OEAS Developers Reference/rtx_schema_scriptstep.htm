scriptStep

The scriptStep element contains a single script step that has actually been viewed in the current script session.

XPath

/root/script/scriptSteps/scriptStep

Cardinality

1..* (required)

Default value

none

Comments

The cardinality of this element depends on the script engine method. See the method specific schema tables (advanceStep, loadStep, resumeSession) for specific requirements. The script cache holds an in-memory copy of this element and all its children for each step in an active script session.
