Installing the Sample (OTM Interface Sample)
Installing the sample

Before you can run this sample code, you need to install the code as a web application on the server running OEAS.

To create the web application:

  1. Copy the sample code directory to the server running OEAS.
  2. Run MicrosoftÂ® Internet Services Manager administrative tool.
  3. Connect to the server running OEAS.
  4. Select the Default Web Site on that server.
  5. From the Action menu choose New, then Virtual Directory.
  6. In the Virtual Directory Creation Wizard, name your new directory whatever you want. When it asks for the source folder for the directory, enter the path to the directory where you copied the sample code.
  7. Make sure to enable ASP script execution for the web application.
  8. Finish the wizard.

Accessing the installed sample

Once you install the sample, you can run it and follow the rest of the OTM Interface Sample topics within the web application.

To run the sample:

  1. Enter the following URL in a browser:

http://SERVER_NAME/VIRTUAL_DIRECTORY_NAME/defaultframes.htm

  1. Replace the SERVER_NAME and VIRTUAL_DIRECTORY_NAME with the appropriate names from your OEAS server and OTM Interface Sample virtual directory, respectively.
