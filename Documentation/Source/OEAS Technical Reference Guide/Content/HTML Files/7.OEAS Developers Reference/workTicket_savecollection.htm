saveCollection (workTicket)

saveCollection method

The saveCollection method of the workTicket object inserts, updates, and/or deletes a collection of specified objects.

Parameters

Parameter

Direction

Data type

Content type

Description

workTickets

inOut

object

any(in) keysOnly(out)

Contains the workTicket information to be inserted, updated, or deleted

Steps

Step data

Step type

Description

OnyxCollectionManager.CollectionManager

com

Forms individual standard method calls for each object instance in the parameters and executes them through the internal Onyx Transaction Manager (OTM)

Remarks

Unlike the saveCollection standard method, this saveCollection uses the OnyxCollectionManager component. The main difference between the two steps is that OnyxCollectionManager wraps each object instance action in a separate transaction.
