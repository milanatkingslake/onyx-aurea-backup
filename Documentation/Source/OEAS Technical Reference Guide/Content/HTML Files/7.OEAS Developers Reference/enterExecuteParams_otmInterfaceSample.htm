enterExecuteParams (OTM Interface Sample)
Run execute code

In the installed sample Web application, use this form to input the valid parameters for your environment and execute a logical object method call. ApplicationName, LogicalObjectName, LogicalMethodname, and ParamXML are required and validated. The values you use here must comply with the Onyx Enterprise Dictionary for the ApplicationName you supply. CustomDataXML is a required parameter but is not validated. It can be any value including an empty string.
