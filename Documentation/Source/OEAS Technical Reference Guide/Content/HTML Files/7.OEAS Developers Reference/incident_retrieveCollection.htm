retrieveCollection (incident)

retrieveCollection method

The retrieveCollection method of the incident object returns a collection of object instances based on the search criteria supplied in the parameters XML.

retrieveCollection standard method information

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Retrieves object instances that match search criteria

Example

Input Parameter:

<parameters>

   <incidents objectType="incident" collection="1">

      <incident objectType="incident">

         <assignedTo>samg</assignedTo>

      </incident>

   </incidents>

</parameters>
