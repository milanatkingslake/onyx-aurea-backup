insertReplace (assignmentAssociation)

insertReplace method

The insertReplace method of the assignmentAssociation object creates or replaces an assignmentAssociation object for a specified owner object.

Parameters

Parameter Direction Data type Content type Description
assignmentAssociation in/out object all/keysOnly A new assignmentAssociation object on input.

Steps

The default steps for the insertReplace method of the assignmentAssociation object are:

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Calls a pre-procedure to determine if an assignmentAssociation already exists for specified owner. If so, the existing association is deleted before the new one is inserted.
