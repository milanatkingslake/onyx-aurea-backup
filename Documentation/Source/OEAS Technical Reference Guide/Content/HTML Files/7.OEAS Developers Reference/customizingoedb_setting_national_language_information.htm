Returning national language errors

National language errors may be returned using the message_code and database_message columns described in result set intperpretation by setting the appropriate variables. The following sample shows how a national language error might be raised from a replace procedure:

-- If the search type (table_id) is NOT an individual, company

-- or customer or if it is null then raise an invalid object error.

IF @table_id IS NULL OR @table_id NOT IN (@INDIVIDUAL_OBJ_TBL_ID,

                                          @COMPANY_OBJ_TBL_ID,

                                          @CUSTOMER_OBJ_TBL_ID)

BEGIN

   SET @return_code  = 2

   SET @message_code = 709031

END

These variables will be used with returning the error result set described in behavioral rules for replace procedures.
