Adding relationships

Before adding relationships, see adding object keys. You define a relationship for the routingPrinter object to restrict the countryCode property to existing country objects. This prevents a user from creating a new routingPrinter object with a countryCode that may not exist in any literatureOrder objects.

The following table shows the required attributes for the relationship between country and routingPrinter objects. Using Object Designer, add a relationship with the following attributes.
