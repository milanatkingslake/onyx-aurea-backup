literatureProduct Object

literatureProduct object

A literatureProduct object represents an individual literature item that you can distribute to a customer. You can add literature products to the system through the Product Administration tool in the Onyx Enterprise Studio. Client applications can then retrieve literatureProduct objects from the database and use them to create Literature orders.

Literature products also exist in the system as standard products (they can also be retrieved using the objects in the product feature), but the literature feature objects provide functionality specific to the handling of literature items. For example, the literatureProduct object has properties that identify electronic source documents that can be printed or emailed as part of the fulfillment of a literature order.

The literatureProduct object has one key property, productNumber, that uniquely identifies the literature product in the database. Once the object is created, the value of this property cannot be changed.
