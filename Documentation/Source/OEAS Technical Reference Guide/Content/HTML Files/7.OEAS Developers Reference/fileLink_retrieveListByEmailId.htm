retrieveListByEmailId (fileLink)

retrieveListByEmailId method

The retrieveListByEmailId method of the fileLink object returns a list of fileLink objects owned by emailAttachment objects for a particular email message.

Parameters

Parameter Direction Data type Content type Description
emailMessageId
(required)		 in string n/a The ownerId value of the emailAttachment object
rowSet out object all(out) The row set containing information about the fileLink instances of the specified emailAttachment object

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieves the fileLink objects owned by emailAttachment object for a particular email message

Remarks

This method retrieves a collection of fileLink objects. The records returned are based on a match against the emailAttachment.ownerId property. The emailAttachment instances are based on the relationship between emailAttachment.primaryId and fileLink.ownerId.
