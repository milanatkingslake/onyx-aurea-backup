productNumber (literatureProduct)

productNumber property

The productNumber property of the literatureProduct object uniquely identifies a given product within the system.

Data type

string (maximum length)

Access

Read/write

Remarks

The literature product is a special case of a product. You create literatureProduct records by using the ID of an existing product record and adding the literature specific properties. This value cannot be changed after the literature product has been added to the database.
