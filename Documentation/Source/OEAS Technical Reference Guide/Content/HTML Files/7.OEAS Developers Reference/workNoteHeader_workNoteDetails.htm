workNoteDetails (workNoteHeader)

workNoteDetails property

The workNoteDetails property of the workNoteHeader object contains a collection of workNoteDetail object instances.

Data type

object

Access

Read-only; for more information, see Read Only attribute

Remarks

The workNoteDetails property contains all workNoteDetail object instances associated with a workNoteHeader object instance. All object instances in the workNoteDetails collection have the same ownerId.
