name (referenceField)

name property

The name property of the referenceField object contains the name for a reference field.

Data type

string (maximum length)

Access

Read/write

Remarks

The name property does not typically display on a user interface.

The naming convention for the name property is <table>.<field>, which identifies the database table and column for each field. For example, a referenceField object with a name of company.status indicates that the data is stored in the company table, within the status column.
