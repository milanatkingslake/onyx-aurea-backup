retrieveCollection (internalContact)

retrieveCollection method

The retrieveCollection method of the internalContact object returns a collection of objects based on the search criteria supplied in the parameters XML.

retrieveCollection standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Retrieves a collection of internalContacts

Uses a database function mapped to the contactUserName property to retrieve a user name based on the value of the contactUserId property

Example

Parameters

In the following example, a search is completed for internalContact records with a contactUserId of "default".

<parameters>

   <internalContacts objectType="internalContact" collection="1">

      <internalContact objectType="internalContact">

         <primaryId/>

         <ownerId/>

         <ownerType/>

         <contactTypeId/>

         <contactUserId>default</contactUserId>

         <contactUserName/>

         <comment/>

      </internalContact>

   </internalContacts>

</parameters>
