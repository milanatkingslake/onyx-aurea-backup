subscriptionService object

subscriptionService object

The subscriptionService object contains information about the available subscription services in a system.

Methods

Method

Description

retrieveCollection

Returns objects that match search criteria

saveCollection

Performs multiple actions at once: inserting, updating, and deleting objects, as appropriate

Properties

Property

Description

active

Specifies whether the domain data the object provides is available to the user interface

connectorProgId

Contains the programmatic identifier of the COM component that connects the synchronization server to the staging server

description

Contains text that describes the subscription service

insertBy

Contains the user ID of the Onyx user who initially saves the object

insertDate

Contains the time and date on which the object is initially saved

name

Contains the name of the subscription service

primaryId

Contains the unique ID assigned to the object when it is first saved to the database

updateBy

Contains the user ID of the Onyx user who last saved modifications to the object

updateDate

Contains the time and date on which modifications to the object were last saved

user10

Contains an optional user-defined field
