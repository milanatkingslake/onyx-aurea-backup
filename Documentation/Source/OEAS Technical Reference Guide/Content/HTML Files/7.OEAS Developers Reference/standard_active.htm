active

active property

The active property of an object specifies whether the domain data the object represents is displayed to the end-user.

This is a generic topic that applies to all properties with the same name. The remarks section might contain details specific to a given object.

Data type

boolean

Access

Read/write

Remarks

Administration tools, such as those in the Onyx Enterprise Studio suite, enable users to manage both active and inactive domain data.

Rather than deleting unused domain data objects, Onyx recommends deactivating these objects. By default, a deactivated domain data object is not displayed to the end-user. The reason for keeping deactivated domain data is to maintain referential integrity for any transactional objects that reference deactivated domain data. Use the Onyx Enterprise Studio Administration Workbench to activate and deactivate domain data objects. For more information see the Deactivating versus deleting items topic under Administration Workbench.

Valid values of the active property are:

  • 0: The domain data object is deactivated.
  • 1: (Default) The domain data object is active.
