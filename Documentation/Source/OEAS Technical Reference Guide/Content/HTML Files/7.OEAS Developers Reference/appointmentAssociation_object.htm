appointmentAssociation object

The appointmentAssociation object enables users to insert, update, retrieve, and delete appointmentAssociation object instances. An instance of this object links an appointment to a company, individual, or incident.

The appointmentAssociation object has one key property, primaryId, that uniquely identifies object instances in the database. Once the object is created, the value of this property cannot be changed.

Methods

Method Description
retrieveCollection Returns a collection of objects based on the search criteria supplied in the parameters XML
retrieveListByExternalAppointment Returns a list of appointmentAssociation records based on the supplied appointment criteria
saveCollection Performs multiple actions at once: inserting, updating, and deleting objects, as appropriate

Properties

Property Description
appointmentId The unique identifier of the associated appointment object
associatedWithId The unique identifier of the associated company, individual, or incident object
associatedWithType An enumerated value that identifies the type of object identified in the associatedWithId
insertBy The Onyx user who initially saved the object instance
insertDate The time and date on which the object instance was initially saved
onyxTimestamp Facilitates concurrency control with the Onyx Enterprise Database
primaryId A unique value that OEAS assigns to the appointmentAssociation object instance
updateBy The user ID of the person who last updated the object instance
updateDate The time and date of the last object instance update
