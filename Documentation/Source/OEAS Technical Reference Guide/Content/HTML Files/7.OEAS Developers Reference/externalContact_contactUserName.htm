contactUserName (externalContact)

contactUserName property

The contactUserName property of the externalContact object contains the name of the individual or company that servers as an external contact.

Data type

string (maximum length)

Access

Read-only; for more information, see Read Only attribute

Remarks

You cannot use this property in the retrieveCollection method of the externalContact object to filter the returned collection by names of contacts. The OEAS ignores this property on input to the retrieveCollection method.

If the external contact is an individual, the contactUserName is a concatenation of the individual's firstName, middleName, and lastName.

<contactUserName>George Lee Anyone</contactUserName>

If the external contact is a company, the contactUserName is identical to the companyName property of the company object.

For example, the following is returned when the contactId is mapped to a company:

<contactUserName>Widgets Galore</contactUserName>
