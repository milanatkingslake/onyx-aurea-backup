attachmentCategory (message)

attachmentCategory property

The attachmentCategory property of the message object contains an ID representing the type of incident record, such as Sales, Customer Service, or Product Support.

Data type

i4

Access

Read/write

The following table lists the default attachmentCategory property values and description strings for each supported object type. The values for incident objects correspond to the categoryId property.
