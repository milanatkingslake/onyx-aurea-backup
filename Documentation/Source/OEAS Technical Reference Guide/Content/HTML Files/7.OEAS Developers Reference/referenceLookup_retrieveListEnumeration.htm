Remarks

This method, unlike the retrieveList method for the referenceLookup object, returns all referenceLookup values for the specified field, regardless of what parentId they have or even if they have a NULL parentId. It returns a distinct list of reference parameter IDs, so if a parameter is used with different parents, the parameter is returned only once.

The name parameter is not case-sensitive and supports wild card substitution. For more information on wild card substitution, see the retrieveList method (standard behavior) topic.
