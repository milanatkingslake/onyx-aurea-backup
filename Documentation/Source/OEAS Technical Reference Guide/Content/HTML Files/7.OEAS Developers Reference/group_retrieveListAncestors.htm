retrieveListAncestors

The retrieveListAncestors method of the group object returns a list of groups hierarchically above the specified child group. The data is returned as a rowSet object in the parametersReturn XML.

Step

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieves information about group object instances

Remarks

This method supports the Onyx security component for Cognos security integration.

The stored procedure sorts the result set by proximity. It returns the parent group before parent's parent and so on.

If the object ID does not exist or represents a root item, the stored procedure returns an empty rowSet object.

If either input parameter is null or the objectType is not a valid value, the stored procedure returns an error.
