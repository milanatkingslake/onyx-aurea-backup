processedDate (emailMessage)

processedDate property

The processedDate property of the emailMessage object contains the date to use for managing email messages.

Data type

dateTime

Access

Read/write

Remarks

The processedDate provides a single date filter criterion to use as a retrieveList input parameter. By using processedDate, users will not need to decide among insertDate, sentDate, receivedDate, or updateDate for filter criteria.

This property is populated with the current date (UTC) when an emailMessage instance is inserted or when an existing instance (draft) is updated. The time portion of the dateTime value is 1:00:00 regardless of the actual time that the object instance was inserted or updated.

For inbound email messages processed by the Email Monitoring Subsystem, processedDate is the date that the mail server receives the message.
