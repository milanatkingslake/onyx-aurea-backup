individual object

individual object

An individual object can represent an individual person. This person can be a lead, an end user, a prospect, or any other type of contact.

Use the individual object when you want to do something specific with an individual but not necessarily with a company.

Methods

Method

Description

delete

Either logically or physically deletes an object from the database. Secondary objects, as defined in the relationship section of the Onyx Enterprise Dictionary, may also be logically deleted or updated depending on the logicalCascadeType elements of its relationships.

insert

Persists an object to the database.

retrieve

Returns a collection of objects based on the search criteria supplied in the parameters XML.

retrieveListByCompany

Returns data on all the individuals that belong to a specified company, typically used with the customer.retrieveListHierarchy method to display an organization chart.

update

Persists modifications to an existing object instance.
