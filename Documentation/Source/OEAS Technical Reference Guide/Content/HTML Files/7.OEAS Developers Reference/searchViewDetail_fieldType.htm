fieldType (searchViewDetail)

fieldType property

The fieldType property of the searchViewDetail object contains the database column data type of the fieldName.

Data type

string (maximum length)

Access

Read/write

Remarks

This property is used only for items with a nodeType of FIELD. Valid values for this include the following.

  • BIT
  • CHAR
  • DATETIME
  • ENUM_I
  • ENUM_S
  • FLOAT
  • INTEGER
  • INTEGER_EQ
  • TINYINT

For more information on the behaviors associated with these field type values, see Configuring search view details.
