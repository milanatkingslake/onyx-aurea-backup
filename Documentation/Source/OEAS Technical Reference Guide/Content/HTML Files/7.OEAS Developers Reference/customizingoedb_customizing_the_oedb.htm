Customizing the OEDB

The following topics are provided to assist you in customizing the Onyx Enterprise Database (OEDB), referred to as the physical model in Object Designer:

  • Creating and customizing tables, which includes guidelines for creating custom database tables.
  • Creating and customizing procedures, which includes a definition of the procedures that are recognized by OEAS, the recommended approach for customizing existing procedures, and reference code that you can use in your customizations.

Additionally, within the "Creating and customizing procedures" topic, there is information about:

  • Error handling
  • Row-level filtering for procedures
  • Working with GUIDs
  • Database security
  • Reference code including some useful functions and procedures.
