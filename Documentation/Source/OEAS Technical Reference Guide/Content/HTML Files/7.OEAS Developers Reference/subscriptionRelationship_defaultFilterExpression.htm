defaultFilterExpression (subscriptionRelationship)

defaultFilterExpression property

The defaultFilterExpression property of the subscriptionRelationship object contains an expression used in a WHERE clause that limits the subscribed records.

Data type

string

Access

Read/write

Remarks

When a subscriptionRelationshipRole exists for the relationship and the user who is subscribing belongs to that role, the filter of the cascadeToSecondary property, if it exists, overrides the contents of the defaultFilterExpression property.
