objectName (subscribableObject)

objectName property

The objectName property of the subscribableObject object contains the name of the object identified by the objectType property.

Data type

string

Access

Read-only; for more information, see Read Only attribute

Remarks

On input for insert and update actions on the saveCollection method, a value for the objectName property is not required. The objectName value is retrieved from the database using the object type enumeration.
