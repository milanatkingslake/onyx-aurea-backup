name (publishableObject)

name property

The name property of the publishableObject object contains text that identifies the published domain data.

Data type

string

Access

Read/write

Remarks

If you publish one file from a particular domain object, the name property typically contains the name of the object. However, you can create multiple files drawing from a single domain object. For example, you could create two productGroup domain data files by filtering on the parentId property of the productGroup object. The values you specify for the name properties identify the contents of the files, for instance, productGroupRetail and productGroupWholesale.
