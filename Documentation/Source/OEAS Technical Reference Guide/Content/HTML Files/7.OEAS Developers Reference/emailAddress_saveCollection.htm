saveCollection (phone)
saveCollection method

The saveCollection method of the emailAddress object performs multiple actions at once: inserting, updating, and deleting objects, as appropriate.

saveCollection standard method information

Description

OnyxDataValidationStep.DataValidation

com

Data validation step
OnyxNotification.LBOAdapter com This step is added for notification feature. It captures data before updating the object

OnyxDM.DataManager

com

Perform multiple actions with emailAddress objects

Uses a literal persistence map that persists zero (0) in the restricted column of the email_address table

Calls a post-procedure to check for two or more email addresses a) marked as primary and having same owner or b) with same email address type and same owner; if true, returns an error message and performs a rollback of all actions
OnyxNotification.LBOAdapter com This step is added for notification feature. It captures data after saving the object
