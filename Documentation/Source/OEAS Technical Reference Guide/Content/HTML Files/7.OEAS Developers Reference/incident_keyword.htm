keyWords property

The keyWords property of the incident object contains the key words associated with the incident record.

Data type

string (maximum length)

Access

Read/write

Remarks

The key words are important words that the users store with an incident record. The users can perform database searches or generate reports based on the key words.

The individual keyWords have maximum of 25 characters limit, but you can have multiple keyWords of 25 characters or less up to a limit of 255 characters. You can seperate the keyWords with semicolon.
