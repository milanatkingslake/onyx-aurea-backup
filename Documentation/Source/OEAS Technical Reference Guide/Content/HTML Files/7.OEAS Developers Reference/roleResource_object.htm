roleResource object

The roleResource object associates resources with roles.

Method

Method

Description

saveCollection

Inserts, updates, and deletes roleResource object instances

Properties

Property

Description

active

A boolean value that indicates whether the role is active or inactive

insertBy

The ID of the end-user who created the object instance

insertDate

The date on which the end-user created the object instance

permissionType

An integer that specifies the access permissions of the roleId to the resource

resourceId

The ID of the resource

roleId

The ID of the role

updateBy

The user ID of the person who last updated the object instance

updateDate

The date of the last object instance update
