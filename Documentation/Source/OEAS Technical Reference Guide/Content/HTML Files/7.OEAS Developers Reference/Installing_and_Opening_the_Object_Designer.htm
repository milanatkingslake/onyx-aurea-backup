Installing and opening the Object Designer

The Onyx Setup program installs the Object Designer as part of the Onyx Enterprise Studio (OES) installation. If Object Designer is not installed you must run the OES setup program from your CD

By default, when the Object Designer is installed, security is set to allow access only to a user in the administrator role. Security for the Object Designer can be configured using Security Administration. For more details on configuring security, see the Administrator's Reference.

To open the Object Designer directly:

  1. In the Address box of Internet Explorer, type the URL that points to the Object Designer Web site, or click on a pre-defined link that points to the Object Designer Web site.

  2. If you are authenticated with Onyx authentication, the Onyx Enterprise Studio login screen appears. Type your Onyx user account name in the User ID box and your password in the Password box and then click Login.

    - OR -

    If you are authenticated with integrated Windows authentication, the Object Designer Web site opens.

To open Object Designer through the Onyx Enterprise Studio Home Page:

  1. In the Address box of Internet Explorer, type the URL that points to the Onyx Enterprise Studio Web Home page and then press Enter, or click on a pre-defined link that points to the Onyx Enterprise Studio Home Page.

    The Onyx Enterprise Studio Home Page appears. The Onyx Enterprise Studio Home Page is the virtual directory that was assigned during setup.

  1. If you are authenticated with Onyx authentication, the Onyx Enterprise Studio login screen appears. Type your Onyx user account name in the User ID box and your password in the Password box and then click Login.

    - OR -

    If you are authenticated with integrated Windows authentication, the Object Designer Web site opens.

  2. After you are authenticated as a valid user of the Object Designer, the Object Designer Web page is displayed in the browser window.
