saveCollection (publishableObject)

saveCollection method

The saveCollection method of the publishableObject object performs multiple actions at once: inserting, updating, and deleting objects, as appropriate.

saveCollection method standard behavior

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Insert, update, and/or delete a collection of publishableObject object instances
