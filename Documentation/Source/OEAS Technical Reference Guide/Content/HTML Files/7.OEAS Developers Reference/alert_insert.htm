insert (alert)

insert method

The insert method of the alert object adds an alert record to the database.

insert standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com

Calls a pre-procedure to determine if an alert already exists for specified owner; if true, prevents insert and returns error message

Adds an alert object instance to the database

Remarks

If an alert object exists for the parent object identified by ownerId and ownerType, the insert fails.

Example

The following example shows the content of the input parameters and output parametersReturn used to insert an alert object.

Parameters

<parameters>

   <alert objectType="alert">

      <primaryId/>

      <ownerId>A623DFA2-475C-439A-8410-1A68E2A89BDC</ownerId>

      <ownerType>2</ownerType>

      <text><![CDATA[Global headquarters will move to the new address effective 3/15.]]></text>

      <expirationDate>03/30/2002</expirationDate>

      <expired/>

      <insertBy/>

      <insertDate/>

      <updateBy/>

      <updateDate/>

      <onyxTimestamp/>

   </alert>

</parameters>

parametersReturn

<parametersReturn>

   <alert objectType="alert" content="keysOnly">

      <primaryId>7508ACBA-4F1F-4849-AF9B-EF29DA68BA1E</primaryId>

   </alert>

</parametersReturn>
