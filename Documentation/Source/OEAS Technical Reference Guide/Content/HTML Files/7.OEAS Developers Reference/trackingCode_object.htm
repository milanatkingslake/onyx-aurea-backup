trackingCode Object

trackingCode object

A trackingCode object identifies a specific event or activity to be measured or tracked within a campiagn. A system administrator associates a campaign with a trackingCode which enables users to associate incidents, products, and literature orders with the campaign. The administrator must create the code using the Campaign Administration tool before users can access it in the user interface. A tracking code belongs to one campaign.

Methods

Method

Description

retrieve

Returns a specific trackingCode object

retrieveCollection

Returns a collection of trackingCode objects based on filter criteria

retrieveListEnumeration

Returns a list of tracking codes displayed in List Manager

saveCollection

Performs multiple actions on a collection of trackingCode objects
