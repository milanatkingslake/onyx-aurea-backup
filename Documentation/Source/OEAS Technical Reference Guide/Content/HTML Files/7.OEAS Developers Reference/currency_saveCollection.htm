saveCollection (currency)

saveCollection method

The saveCollection method of the currency object performs multiple actions at once: inserting, updating, and deleting objects, as appropriate.

saveCollection standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Saves a collection of object instances
