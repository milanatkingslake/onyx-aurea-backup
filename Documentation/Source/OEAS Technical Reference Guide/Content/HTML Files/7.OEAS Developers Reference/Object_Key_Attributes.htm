Object Key attributes

When you select an object key in the Object Designer, the attributes for that key display in the edit area.

The attributes are defined as follows:

Attribute Name

Description

ID

The globally unique identifier (GUID) assigned by the Onyx Enterprise Dictionary. This attribute is read-only.

Name

Type the name you want to assign to this object key. Key names must be conform to XML naming conventions.

Description

Type the description for this object key.

Object Key Type

Select the key type for this object key:

  • For an object key that uniquely identifies each record, select primary.
  • For an object key that uniquely identifies a record but is not the primary key, select alternate.
  • For an object key that uniquely identifies a record and is easily comprehended by users, select friendly.

Available Properties

In the Available Properties box, highlight the property to be used as the object key. Then click the right arrow to move the property to the Selected Properties box. Repeat this step for each property you want to add to this key.

To remove a selected property, highlight the property in the Selected Properties box. Then click the left arrow to move the property to the Available Properties box.
