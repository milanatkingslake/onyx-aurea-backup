primary (address)

primary property

The primary property of the address object identifies whether the address is the customer's primary address or an alternate address.

Data type

boolean

Access

Read/write

Remarks

A customer has only one primary address and can have as many alternate addresses as needed.

A primary property value is required in the input parameters XML on insert and update methods.

Valid values for the primary property are:

  • 0 : The address is an alternate address for the customer.
  • 1 : The address is the primary address for the customer.
