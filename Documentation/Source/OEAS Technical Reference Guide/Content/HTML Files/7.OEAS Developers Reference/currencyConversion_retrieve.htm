retrieve (currencyConversion)

retrieve method

The retrieve method of the currencyConversion object searches the database based on the properties defined by the object's primary objectKey entry and returns an object instance.

retrieve standard method information

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Retrieve one currency conversion object

Example

Input Parameter:

<parameters>

   <currencyConversion objectType="currencyConversion">

      <currencyCodeFrom>ARP</currencyCodeFrom>

      <currencyCodeTo>ATS</currencyCodeTo>

      <effectiveDate>2002-02-22 08:00:00</effectiveDate>

   </currencyConversion>

</parameters>

Output Parameter:

<parametersReturn>

   <currencyConversion objectType="currencyConversion" content="all">

      <currencyCodeFrom>ARP</currencyCodeFrom>

      <currencyCodeTo>ATS</currencyCodeTo>

      <effectiveDate>2002-02-22 08:00:00</effectiveDate>

      <rate>1.25</rate>

      <triangulationCode null="1"/>

      <active>1</active>

   </currencyConversion>

</parametersReturn>
