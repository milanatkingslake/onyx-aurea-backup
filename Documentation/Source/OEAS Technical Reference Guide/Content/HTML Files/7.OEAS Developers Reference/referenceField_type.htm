type (referenceField)

type property

The type property of the referenceField object specifies how the field displays on the user interface.

Data type

string (maximum length)

Access

Read/write

Remarks

Valid values for the type property include:

  • CHECKBOX
  • COMBO
  • FREETEXT
  • HIERCOMBO

For consistency, input these values in uppercase.

To create a hiercombo referenceField object, pass in both a type of "HIERCOMBO" and the parentName property of the referenceField object that you want to assign as a parent for this field.

For example, to make the field Company.User2 a child of Company.User1, call the saveCollection method (with the action attribute set to "update") and pass in the type property with a value of "HIERCOMBO" and the parentName property with a value of "Company.User1".
