Concurrency control

The following properties support the OEAS record locking scheme.

OEAS uses these properties to support pessimistic record locking:

  • locked
  • lockedBy

OEAS uses these properties to support optimistic record locking for all of its Web applications, such as with the OEP-Web client:

  • onyxTimestamp
  • updateDate (used indirectly, when OEAS compares it with the onyxTimestamp property)
