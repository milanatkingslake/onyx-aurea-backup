OEAS properties

Many OEAS objects contain a set of properties that are defined consistently in the OEAS object model, regardless of which object owns the property.

OEAS implements the following standard properties, categorized here according to their general function:

  • Identifying objects and records (key properties)
  • Locking records
  • Accessing records
  • Maintaining a history of updated records
  • Modifying the user interface
