shipDate (forecastLine)

shipDate property

The shipDate property of the forecastLine object contains the date on which the products identified in the forecastLine object are expected to be shipped to the customer.

Data type

dateTime

Access

Read/write

Remarks

This date is entered by the end-user.

The dateTime data type has the following format:

YYYY-MM-DD hh:mm:ss

where:

YYYY = year

MM = month

DD = day

hh = hour

mm = minute

ss = second

Valid values for the dateTime data type range from 1900-01-01 00:00:00 to 9999-12-30 23:59:59.
