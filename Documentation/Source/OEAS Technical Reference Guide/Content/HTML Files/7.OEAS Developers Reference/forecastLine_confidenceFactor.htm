confidenceFactor (forecastLine)

confidenceFactor property

The confidenceFactor property of the forecastLine object contains the level of confidence for the eventual sale of the forecastLine object. This value is expressed as the decimal equivalent of a percentage.

Data type

float

Access

Read/write

Remarks

The confidenceFactor enables the user to specifiy the probability that the forecastLine projection will be realized. The factor applies a weighting to the forecastLine amount before it is used to calculate the total forecast amount.

confidenceFactor x quantity ( primaryUnitPrice - primaryDiscount ) = forecastLine total

.75 x 1 ( $2560.00 - 0 ) = $1920.00

The valid values for this property range from 0 to 100 percent inclusive written in decimal format (0.0 <= confidence factor <= 1.0).
