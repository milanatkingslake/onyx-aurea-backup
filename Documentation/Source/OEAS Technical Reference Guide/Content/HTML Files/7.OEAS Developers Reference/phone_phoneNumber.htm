phoneNumber (phone)

phoneNumber property

The phoneNumber property of the phone object identifies the phone number that is associated with the phone object.

Data type

string (maximum length)

Access

Read/write

Remarks

The phoneNumber in the database is without spaces or masks.

The phoneNumber is displayed with the masking property applied as specified in the calling method. The mask is maintained in the country table.

Valid values for includeMask are (in this context the mask is for the United States):

  • none: phoneNumber output might appear as 2535551235
  • append: phoneNumber output might appear as 2535551235**MASK:(999) 999-9999 /x 99999
  • apply: phoneNumber output might appear as (253) 555-1235

The phoneNumber property is used to indicate a phone number to a specific phone object.
