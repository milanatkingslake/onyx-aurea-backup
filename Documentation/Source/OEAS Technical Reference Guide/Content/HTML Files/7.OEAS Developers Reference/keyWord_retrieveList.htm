retrieveList (keyWord)

retrieveList method

The retrieveList method of the keyWord object searches the database by categoryId and returns all keyWord objects associated with the specified incident, task, or workticket category.

Remarks

Valid values of categoryId are:

  • If objectTypeEnum = 6 or 8, the categoryId is a key to the incidentCategory and taskCategory tables
  • If objectTypeEnum = 10 (workTicket) the categoryId = 4

Valid values of objectTypeEnum are:

  • 6 = incident
  • 8 = task
  • 10 = work ticket
