saveCollection (methodCall)

saveCollection method

The saveCollection method of the methodCall object performs multiple actions at once: inserting, updating, and deleting methodCall objects, as appropriate.

saveCollection method standard behavior

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Insert, update, and/or delete a collection of methodCall objects

Remarks

On an insert action, the methodCall object must contain name, objectName, methodName, usageObjectType, and active elements, or the insert fails.

Example

The following example uses saveCollection to create a new company insert methodCall object.

Parameters

<parameters>

   <methodCalls objectType="methodCall" collection="1">

      <methodCall objectType="methodCall" content="any" action="insert">

         <name>company_insert</name>

         <objectName>company</objectName>

         <methodName>insert</methodName>

         <usageObjectType>192</usageObjectType>

         <active>0</active>

      </methodCall>

   </methodCalls>

</parameters>

parametersReturn

<parametersReturn>

   <methodCalls collection="1" objectType="methodCall">

      <methodCall objectType="methodCall" content="keysOnly" action="insert">

         <primaryId>100007</primaryId>

      </methodCall>

   </methodCalls>

</parametersReturn>

The following example uses saveCollection to update the active property on the methodCall object created in the previous example.

Parameters

<parameters>

   <methodCalls objectType="methodCall" collection="1">

      <methodCall objectType="methodCall" content="any" action="update">

         <primaryId>100007</primaryId>

         <active>1</active>

      </methodCall>

   </methodCalls>

</parameters>
