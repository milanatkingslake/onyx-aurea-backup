retrieveCollection (region)

retrieveCollection method

The retrieveCollection method of the region object returns a collection of object instances based on the search criteria supplied in the parameters XML.

retrieveCollection standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Retrieves object instances that match search criteria

Example

The following example shows the content of the input parameters and output parametersReturn used to retrieve a collection of region object instances associated with the CA country code.

Parameters

<parameters>

   <regions objectType="region" collection="1">

      <region objectType="region">

         <regionCode/>

         <countryCode>ca</countryCode>

         <regionName/>

         <insertBy/>

         <insertDate/>

         <updateBy/>

         <updateDate/>

         <onyxTimestamp/>

         <active/>

      </region>

   </regions>

</parameters>
