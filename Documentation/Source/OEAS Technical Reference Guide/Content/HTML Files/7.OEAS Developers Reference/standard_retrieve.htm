retrieve method

A retrieve method returns an object instance based on its primary key.

Parameters

This method accepts an object whose content="keysOnly" as the input parameter, and returns an "all" object as an output parameter. You pass the input XML within the root element named <parameters>.

Parameter Direction Data type Content in type Content out type Description

Object name

inOut

object

keysOnly

all

An object whose content="keysOnly" on input.
Steps

All standard methods call the Onyx Data Validation Step for input data validation and either directly or indirectly call Onyx Data Manager to access the data tier. Many methods perform additional steps. For a table that lists the default steps of a specific business object method, refer to its method topic under the business object.

Remarks

An error occurs if an object instance is not found with the specified primary key.
