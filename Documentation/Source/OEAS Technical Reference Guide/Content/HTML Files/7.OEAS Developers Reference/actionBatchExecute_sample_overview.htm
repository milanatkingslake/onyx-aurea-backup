(ActionBatchExecute) Sample Overview

Using actionBatch to execute a series of OEAS method calls

Read the following sections in this reference before reading the actionBatch.execute sample topics in this book:

  • OnyxActionBatchExecute step component
  • actionBatch.execute method
  • actionBatchXml schema
  • actionBatch statusXml schema

The topics in this book cover using the actionBatch.execute method to execute a series of OEAS method calls. The topics include the following:

  • Providing a sample actionBatchXml document. The actionBatchXml used in this sample is setup for inserting three new companies in a database with incidents, workNotes and workTickets.
