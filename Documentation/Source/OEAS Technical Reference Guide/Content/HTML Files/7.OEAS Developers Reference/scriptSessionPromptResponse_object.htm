scriptSessionPromptResponse object

scriptSessionPromptResponse object

The scriptSessionPromptResponse object identifies a prompt and the associated response for a given step in a script session.

Methods

Method

Description

retrieveCollection

Returns a collection of zero or more scriptSessionPromptResponse objects that match the search criteria

retrieveListBySession

Returns domain and transactional data related to a scriptSession

retrieveMergedListBySession

Calls retrieveListBySession and merges the value of the scriptSession stateXml variables with the prompt text returned by the retrieveList

saveCollection

Performs multiple actions at once: inserting, updating, and deleting objects, as appropriate

Properties

Property

Description

additionalText

Contains user input text that modifies the response value or serves as an alternate response value

insertBy

Contains the user ID of the Onyx user who initially saves the object.

insertDate

Contains the time and date on which the object is initially saved.

primaryId

Contains the unique ID assigned to the object when it is first saved to the database.

responseCode

Contains the code portion of a response selected by the user

scriptPromptId

Contains a foreign key to the prompt domain data associated with a response

scriptSessionStepId

Contains a foreign key to the scriptSessionStep object that owns the response

updateBy

Contains the user ID of the Onyx user who last saved modifications to the object.

updateDate

Contains the time and date on which modifications to the object were last saved.

value

Contains readable response text suitable for display in the user interface
