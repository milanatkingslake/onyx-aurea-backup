parameters XML (OnyxTransformStep)

parameters XML

The only requirement of OnyxTransformStep for input is that the step package be valid and that the stepAdditionalData be correct. OnyxTransformStep does the following processing:

  1. Unescapes the encoded node in the step package that is specified by the escapedNode name/value pair in the stepAdditionalData.
  2. Transforms the unescaped node using the XSLT template specified by the templatePath name/value pair in the stepAdditionalData.
  3. Escapes the transformed node.

The XSL transform processing must result in a valid step package that the Onyx Transaction Manager (OTM) passes to the next step of a method or, if OnyxTransformStep is the last step, to the calling client application.

For an example of OnyxTransformStep, see the actionBatch.executeForEMS method.
