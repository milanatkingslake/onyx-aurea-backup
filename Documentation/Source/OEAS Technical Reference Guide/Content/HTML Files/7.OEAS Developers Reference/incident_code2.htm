code2 (incident)

code2 property

The code2 property of the incident object contains one of four possible resolution code fields for an incident.

Data type

i4

Access

Read/write

Remarks

Resolution codes enable searches by categorizing solutions to incidents. It is used to provide choices to the end-user.

The code2 property references an existing resolution code type in the database. Incident categories own resolution code types, therefore the value in the code2 property must reference a child value of the incidentCategory for a given incident.
