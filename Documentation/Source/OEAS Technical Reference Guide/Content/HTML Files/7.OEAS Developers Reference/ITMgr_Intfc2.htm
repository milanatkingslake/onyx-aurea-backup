ITransactionManager2 interface

The Transaction Manager2 interface serves as an entry point to the OEAS. The ITransactionManager2 interface inherits from the ITransactionManager interface and exposes one additional method, Execute2.  

The ITransactionManager2 interface operates in the same manner as older versions of ITransactionManager. Although Onyx supports both interfaces, all ITransactionManager calls pass through ITransactionManager2 for execution.

The methods of the ITransactionManager2 interface are listed below:

Method

Description

Execute

Calls a method of a logical business object.

Execute2

Uses an additional session persistence parameter while calling a logical business object.
