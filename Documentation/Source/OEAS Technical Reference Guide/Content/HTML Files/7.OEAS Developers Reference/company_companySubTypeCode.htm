companySubTypeCode (company)

companySubTypeCode property

The companySubTypeCode property of the company object identifies the sub category to which a company belongs.

Data type

i4

Access

Read/write

Remarks

The companySubTypeCode property is a unique ID for the subtype or subcategory to which a company belongs and can be configured using the Reference Table administration tool. An example of a companySubTypeCode might be "101", which could have a description of "Direct" in the reference objects. There is a parent child relationship between a companyTypeCode and a companySubTypeCode.
