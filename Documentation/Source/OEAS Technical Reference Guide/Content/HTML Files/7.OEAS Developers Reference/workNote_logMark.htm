logMark (workNote)

logMark property

The logMark property of the workNote object indicates whether this particular workNote starts a new workNoteDetail sequence or is part of a multi-part workNoteDetail.

Data type

boolean

Access

Read/write

Remarks

Valid values of the active property are:

  • 0: The workNote does not start a new workNoteDetail sequence.
  • 1: The workNote starts a new workNoteDetail sequence.
