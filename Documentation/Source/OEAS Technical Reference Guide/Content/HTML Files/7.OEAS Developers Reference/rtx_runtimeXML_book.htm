Run-time XML book

Run-time XML

This book contains a description of the run-time XML structure used in the Scripting feature. The run-time XML represents the interface to the script engine. This documentation assumes the reader is already familiar with XML, OEAS, and the Scripting feature. This book divides the documentation into three parts.

Run-time XML overview

The overview explains many of the design principles underpinning the run-time XML schema.

Run-time XML schema tables

This section contains graphic tables of the master run-time XML structure and the individual script engine method run-time XML including input and output structures for each method.

Run-time XML details

The details contain individual topics for each element and attribute in the master run-time XML structure.

Document conventions

This document uses whitespace (tabs, spaces, and carriage returns) to present XML structures in a more humanly readable form. In practice, the XML typically contains no whitespace. For a script engine method to succeed, the run-time XML is always presented to OEAS as an entity encoded string. This document shows both entity encoding and structural markup examples. You can use the structural markup to better visualize the run-time XML schema, but use the entity encoding examples if you want to try it out in actual code.
