delete (forecast)

delete method

The delete method of the forecast object logically deletes an object from the database. Secondary objects may also be logically deleted depending on their relationship to the forecast object.

delete standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Deletes a forecast header and each associated forecastLine of detail
