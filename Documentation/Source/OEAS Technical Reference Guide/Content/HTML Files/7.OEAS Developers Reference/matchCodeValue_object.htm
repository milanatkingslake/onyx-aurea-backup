matchCodeValue object

The matchCodeValue object is used for custom match code implementations.

Properties

Property Description

insertBy

Contains the user ID of the Onyx user who initially saves the object

insertDate

Contains the time and date on which the object is initially saved

matchCodeValue

Contains a value that is generated for the match code

ownerId

Uniquely identifies the parent object of the child object

ownerType

Contains a unique numerical value that identifies the type of parent object that owns the child object

primaryId

Contains the unique ID assigned to the object when it is first saved to the database

updateBy

Contains the user ID of the Onyx user who last saved modifications to the object

updateDate

Indicates the time and date on which modifications to the object were last saved
