effectiveDate (currencyConversion)

effectiveDate property

The effectiveDate property of the currencyConversion object is the date when the currency exchange rate takes effect and is required when the rate is entered in the database.

Data type

dateTime

Access

Read/write

Remarks

The currencyConvert method uses a currency exchange rate from the database.

If there are multiple rates with different effectiveDates, the method uses the rate with the effective date closest to the current date, but earlier than or equal to the current day's date.

The effectiveDate property is user input. Typically it is entered for several rates at a time as part of an SQL script that loads the latest rates.

The dateTime data type has the following format:

YYYY-MM-DD hh:mm:ss

where:

YYYY = year

MM = month

DD = day

hh = hour

mm = minute

ss = second

Valid values for the dateTime data type range from 1900-01-01 00:00:00 to 9999-12-30 23:59:59.
