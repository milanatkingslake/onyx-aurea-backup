saveCollection (scriptSessionStep)

saveCollection method

The saveCollection method of the scriptSessionStep object performs multiple actions at once: inserting, updating, and deleting objects, as appropriate.

saveCollection standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Insert, update, and/or delete a collection of scriptSessionStep objects

Remarks

The saveCollection method enables you to perform actions on multiple active scriptSessionStep objects. The scriptSessionStep object also supports the standard insert, update, and delete methods. The standard insert, update, and delete methods offer a performance advantage over the saveCollection method.

Example

The following example uses saveCollection to update the completed field on a given scriptSession object.

Parameters

<parameters>

   <scriptSessionSteps objectType="scriptSessionStep" collection="1">

      <scriptSessionStep objectType="scriptSessionStep" content="any" action="update">

         <primaryId>83302BE8-753F-40EB-8BEB-8C99BDF86142</primaryId>

         <scriptSessionId/>

         <scriptStepId/>

         <sequenceNumber/>

         <completed>1</completed>

         <insertBy/>

         <insertDate/>

         <updateBy/>

         <updateDate/>

      </scriptSessionStep>

   </scriptSessionSteps>

</parameters>
