update (task)
update method

The update method of the task object persists modifications to an existing object instance.

update standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters
notificationAdapter.preUpdateEvent com This step is added for notification feature. It captures data before updating the object

OnyxGuidGeneratorStep.GuidGenerator

com

Calls a component that generates GUID primary key values used for inserting the tags identified in the task's tags property

OnyxDM.DataManager

com

Saves modifications to the specified task and calls a post-procedure that inserts or updates the tag object with the terms listed in the task's tags property
notificationAdapter.postUpdateEvent com This step is added for notification feature. It captures data after saving the object
