contactTypeId (externalContact)

contactTypeId property

The contactTypeId property of the externalContact object is the unique numerical value associated with a specific contactType.

Data type

i4

Access

Read/write

Remarks

You can classify externalContact objects by contactType in the OEAS. In order to classify an externalContact object you assign a contactType object to the externalContact object. The contactTypeId property contains the primaryId of the contactType object.
