retrieve (emailMessage)

retrieve method

The retrieve method of the emailMessage object searches the database by primaryId and returns an emailMessage object instance.

retrieve method standard information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxMasterDetailRetrievalStep.MasterDetailRetrieval com Propagates the ownerId into the emailContacts detail collection and creates the collection node if necessary
OnyxMasterDetailRetrievalStep.MasterDetailRetrieval com Propagates the ownerId into the emailAttachments detail collection and creates the collection node if necessary
OnyxDM.DataManager com Retrieves an object instance
emailContact.retrieveCollection method Retrieves an emailContacts collection that contains all emailContacts associated with an emailMessage
emailAttachment.retrieveCollection method Retrieves an emailAttachments collection that contains all attachments associated with an emailMessage
