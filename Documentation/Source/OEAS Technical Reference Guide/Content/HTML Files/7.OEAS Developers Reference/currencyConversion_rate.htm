rate (currencyConversion)

rate property

The rate property of the currencyConversion object contains the ratio for currency conversion.

Data type

float

Access

Read/write

Remarks

The conversion from the input parameter, fromValue, to the output parameter, toValue, is accomplished using the rate property's value.

The rate property is user input. Users typically use an SQL script to load currency conversion rates from an external source. These scripts also typically set the effectiveDate date.
