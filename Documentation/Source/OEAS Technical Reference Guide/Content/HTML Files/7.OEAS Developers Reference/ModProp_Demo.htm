Demo (Modify Property)
Demonstration

This demonstration assumes that you have made the modifications to the OED as outlined in the previous Steps topic for Modifying a property. To demonstrate the success of the modifications, you attempt to insert a new company record into the database without providing the required email address property and observe that the insert fails. You then perform the same transaction with the email address property set and observe that the insert succeeds.

You call the Onyx Transaction Manager (OTM) with the following sets of input XML parameters and observe the results returned by the OTM.

The following set of input and output XML demonstrates how the insert fails when the email address is not provided.
