NatLangProxy Interface

The NatLangProxy interface provides a mechanism for out-of-process client applications, including those based on scripting languages, to retrieve strings from the OnyxNatLangCache. These methods are supported through Visual Basic, VBScript, and JScript.

The methods of the NatLangProxy interface are listed below:

Method

Description

RetrieveString

Obtains a single National Language string from the OnyxNatLangCache component.

RetrieveStringList

Obtains a range of National Language strings from the OnyxNatLangCache component.
