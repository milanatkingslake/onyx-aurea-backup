Security error values

When called from a C++ client application, methods return a standard COM HRESULT or one of the custom HRESULTS listed below. When a Visual Basic client application calls this method and an error occurs, the Visual Basic Err object specifies one of the decimal values listed below and provides a brief description of the error.

The session ID is invalid. The Session Manager component returns this error when it detects that the supplied session ID is invalid. Either the session has expired, ended, or the client tried to send in a session ID that was not returned by the authentication components.

OSLF_E_INVALIDSESSIONVAR
0x80040605
-2147219963

The session variable name is invalid. The Session Manager component returns this error when requesting the value of a session variable and the variable name supplied is invalid.

OSLF_E_INVALIDUSERID

The user ID is invalid, or no such user exists in the system.

OSLF_E_LOGINFAILED
0x80040601
-2147219967

The user login failed. The authentication components return this error when a request is made to establish a session and the user credentials and the credentials are invalid. Either one or both of the following conditions cause this error: the user does not exist, or the password is incorrect. This error can also occur when a user tries to change their password.

OSLF_E_NOLICENSE
0x80040603
-2147219965

No licenses are available to service the request. The authentication components return this error when the user credentials are valid, but all licenses for the customer are in use.

OSLF_E_RESOURCECACHE_NOTAVAIL
0x80040615
-2147219947

The resource cache is not available. The Session Manager component returns this error when a method requires the resource cache and it is not initialized or it failed to load.

OSLF_E_SECURITYERROR
0x80040617
-2147219945

This error is a security call context error. The OEAS security components return this error when a component cannot determine the security call context within a method. This error usually occurs when the OEAS security components are not running in a COM+ environment.

OSLF_E_XMLPARSERERROR
0x80040614
-2147219948

An error occurred while using the Microsoft XML parser. Either the parser is not available or the XML format is incorrect.

OSLF_E_UNEXPECTED
0x80040600
-2147219968

An unexpected or unknown error occurred within the OEAS security components. Inspect the application logs to troubleshoot the problem. Possibly, a downstream component such as the Query Engine logged a message identifying the problem.

OSLF_S_UPDATE_LICENSE
0x00040618
263704

This is a success HRESULT. All available licenses have been consumed. Access is allowed, but the license agreement must be updated.
