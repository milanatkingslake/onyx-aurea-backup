postCode (address)

postCode property

The postCode property of the address object identifies the postal code for the address object.

Data type

string (maximum length)

Access

Read/write

Remarks

The postCode property is used to indicate the postal code for the address and indicates the postal code for the individual or company that owns the record. The postCode should be set in the database without any masking characters.

The postCode is displayed with the masking property applied as specified in the calling method. The mask is maintained in the COUNTRY table.

Valid values for includeMask are (in this context the mask is for the United States):

  • none: postalCode output might appear as 980054091
  • append: postalCode output might appear as 980054091**MASK:99999-9999
  • apply: postalCode output might appear as 98005-4091
