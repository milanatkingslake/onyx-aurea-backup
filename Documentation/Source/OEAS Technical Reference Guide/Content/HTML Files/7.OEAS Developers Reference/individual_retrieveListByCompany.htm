retrieveListByCompany method

The retrieveListByCompany method of the individual object returns data on all the individuals that belong to a specified company. The data is returned as a rowSet object in the XML. This data can be used by the user interface to display information on user accessible individuals associated with the specified company. The retrieveListByCompany method is typically used with the customer.retrieveListHierarchy method to display an organization chart.
