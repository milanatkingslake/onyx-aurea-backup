IAuthenticateOnyx interface

Use the AuthenticateOnyx component to verify users that do not log in to a Microsoft Windows account to access the Onyx Enterprise Application Server (OEAS). Users accessing the system using a browser either through an intranet or the Internet can be authenticated with this component. The browser client needs to collect an Onyx user name and password and pass those items to AuthenticateOnyx. The AuthenticateOnyx component exposes a single interface, IAuthenticateOnyx. The methods of this interface are listed below:

Method

Description

ChangePassword

Changes the password of a user

EstablishSession

Authenticates and maps a user, and initiates a session
