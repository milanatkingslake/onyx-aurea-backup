Adding parameters

Before adding parameters, see adding methods. You add parameters to specify what the method uses for input and/or output. Both the retrieveCollection and saveCollection methods use a routingPrinters collection as the input and output parameter. Both methods' parameter attributes are the same except for the ContentOut.

The following table shows the required attributes for the retrieveCollection and saveCollection parameters. Using Object Designer, add a parameter to each method with the following attributes.

Attribute

Value

Name

routingPrinters

Data type

object

Object type

routingPrinter

Direction

inOut

Collection

checked

ContentIn

any

ContentOut

all (retrieveCollection)

any (saveCollection)
