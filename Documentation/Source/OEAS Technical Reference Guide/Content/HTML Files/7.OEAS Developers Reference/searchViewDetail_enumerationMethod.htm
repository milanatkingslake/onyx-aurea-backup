enumerationMethod (searchViewDetail)

enumerationMethod property

The enumerationMethod property of the searchViewDetail object contains the method name of the enumeration object with which the detail item is associated.

Data type

string (maximum length)

Access

Read/write

Remarks

If the detail item is a region code for instance, you should retrieve a list of valid region codes to display to the user as a selection option. You specify the retrieveListEnumeration method of the region object as the method used to populate the interface options.

Note: The typical method types used for enumeration methods are retrieveList and retrieveCollection. However, other method types may be used as long as they return a collection of nodes that can be selected using the enumerationIdPath and enumerationDescriptionPath XPaths.
