inputObjectType (methodCall)

inputObjectType property

The inputObjectType property of the methodCall object contains a valid object type enumeration representing the type of an input object parameter.

Data type

i4

Access

Read/write

Remarks

You use this property to indicate the object type for methods that accept an object ID as input such as retrieve, retrieveCollection, and retrieveList. This property is optional.

The following table summarizes valid values for inputObjectType depending on the method and/or feature.

Feature/method

inputObjectType value

Methods that do not accept an input object ID

null

Scripted workflow feature

null

Merge template feature

valid object type enumeration (e.g. 1 for individual; 6 for incident)
