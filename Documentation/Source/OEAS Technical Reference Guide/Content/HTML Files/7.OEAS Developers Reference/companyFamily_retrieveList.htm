retrieveList (companyFamily)

retrieveList method

The retrieveList method of the companyFamily object searches the database and returns the complete list of company families for the site.

Parameters

Parameter Direction Data Type Content type Description
rowSet out object all(out) Information from the companyFamily

Columns returned in the rowSet object

columnOrdinal columnName Physical column
1 iFamilyId company_family.company_family_did
2 vchFamilyName company_family.family_name

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters.
OnyxDM.DataManager com Retrieves company families.

Remarks

The returned rowSet object contains the columnDefinition for each of the columns and a list of company families for a given site. The company_family table is populated at system setup and can be administered with the Table Administration tool.
