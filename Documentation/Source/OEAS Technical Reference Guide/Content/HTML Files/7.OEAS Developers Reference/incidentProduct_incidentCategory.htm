incidentCategory (incidentProduct)

incidentCategory property

The incidentCategory property of the incidentProduct object contains the incident categoryId of the incident object to which a product is related. End-users relate an incidentProduct to an incident record with the productId property of the incident object.

Data type

i4

Access

Read/write

Remarks

The incidentCategory property references an existing incidentCategory in the database. For more information on configuring incidentCategory, consult the OES Reference Table Administration online Help.
