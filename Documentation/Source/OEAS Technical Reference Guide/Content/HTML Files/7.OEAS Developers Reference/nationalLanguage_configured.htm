configured (nationalLanguage)

configured property

The configured property of the nationalLanguage object indicates whether the message came from the national_language table or the nat_lang_config table.

Data type

boolean

Access

Read-only; for more information, see Read Only attribute

Remarks

Valid values for this property include zero and one.

  • 0 = from national_language table
  • 1 = from nat_lang_config table
