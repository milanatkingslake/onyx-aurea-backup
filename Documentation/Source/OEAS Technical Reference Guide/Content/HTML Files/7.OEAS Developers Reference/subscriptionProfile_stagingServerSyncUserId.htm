stagingServerSyncUserId (subscriptionProfile)

stagingServerSyncUserId property

The stagingServerSyncUserId property of the stagingServer object contains the user ID that the synchronization server uses to connect to the staging server.

Data type

string

Access

Read/write

Remarks

This property contains the mailbox user name for the super user under which the synchronization server accesses the staging server, such as an Exchange server. A mailbox for this user is created manually during installation.
