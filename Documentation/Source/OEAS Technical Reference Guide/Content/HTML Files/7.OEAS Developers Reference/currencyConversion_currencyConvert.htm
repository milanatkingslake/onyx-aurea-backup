currencyConvert (currencyConversion)

currencyConvert method

The currencyConvert method of the currencyConversion object changes the value of a monetary quantity in one currency to its equivalent monetary quantity in another currency.

Parameters

Parameter Direction Data type Content type Description

currencyCodeFrom

in

string

n/a

The code of the currency from which the conversion takes place

currencyCodeTo

in

string

n/a

The code of the currency to which the conversion takes place

fromValue

in

float

n/a

The monetary value to be converted

effectiveDate

in

dateTime

n/a

The date when the currency exchange rate takes effect

toValue

out

float

n/a

The monetary value in new currency

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Perform the currency conversion from the fromValue to the toValue

Remarks

This method uses the rate property value to perform the currency conversion. Each rate is associated with an effectiveDate property that indicates when the rate goes into effect. The method uses the currency exchange rate with the most current effectiveDate. For the currencyConvert method to be successful, the effectiveDate must be equal to or earlier than the current day's date.

If the currency can not be converted directly from the available rates, you can use the triangulationCode, if applicable, to perform the conversion.
