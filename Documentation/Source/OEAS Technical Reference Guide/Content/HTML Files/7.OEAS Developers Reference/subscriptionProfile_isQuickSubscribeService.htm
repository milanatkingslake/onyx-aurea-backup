isQuickSubscribeService (subscriptionProfile)

isQuickSubscribeService property

The isQuickSubscribeService property of the subscriptionProfile object indicates whether the profile is used for the one-click subscription feature in the OEP online client.

Data type

boolean

Access

Read/write

Remarks

Valid values of this property are:

false (0): The profile does not include one-click subscription.

true (1): The profile includes one-click subscription.
