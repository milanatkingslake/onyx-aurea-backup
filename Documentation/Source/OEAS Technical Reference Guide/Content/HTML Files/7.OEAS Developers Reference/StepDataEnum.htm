StepDataEnum

StepDataEnum enumerates the columns of the Steps array, as returned, of the GetSteps method of the ICache interface. The possible values of this enumeration are as follows:

Name (value)

Description

osdStepType (0)

The type of step. Valid values are com, method, and transaction.

osdStepData (1)

The ProgID of the associated step component if the step type is com.

osdStepAdditionalData (2)

Information, if any, to be passed to the instantiated component.
