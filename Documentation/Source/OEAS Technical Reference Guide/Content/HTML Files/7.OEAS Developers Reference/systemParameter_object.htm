systemParameter object

The systemParameter object enables applications to update and retrieve parameters that are used throughout the system. You use a systemParameter to configure an application. An example of a systemParameter is TagIncludeCount. The TagIncludeCount parameter enables you to control the number of times a tag is used before it is returned by retrieveList.

The systemParameter object has one key property, code, that uniquely identifies the parameter in the database.

Methods

Method

Description

retrieve

Returns a specific systemParameter object based on the primary key supplied in the parameters XML

retrieveCollection

Returns a collection of objects based on the search criteria supplied in the parameters XML

saveCollection

Performs update actions only on a collection of systemParameter objects
