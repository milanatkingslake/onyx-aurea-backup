retrieveListByFilterNoDefault
retrieveListByFilterNoDefault method

The retrieveListByFilterNoDefault method of the incident object searches the database by the filter criteria you pass in and returns a rowSet of incident data. The method does not use defaults for parameter values that are not provided.

Remarks

When searching the database by group (by passing the group parameter in the input XML), you must specify each group and subgroup that you want to search. Subgroups are not automatically returned with their parent group.

If you use both the assignedToList parameter and the groupIdList parameter as search criteria for this method, you might receive duplicate results in the returnXML. This happens when a user identified by the assignedToList parameter is a member of the group represented by the groupIdList parameter.

This method supports paging functionality. For more information, see the Paging functionality.
