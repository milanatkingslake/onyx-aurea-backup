emailCc (literatureOrder)

emailCc property

The emailCc property of the literatureOrder object contains the email address of a carbon-copy recipient.

Data type

string (maximum length)

Access

Read/write

Remarks

The emailCc property is used for specifying addresses of carbon-copy recipients of email messages. This property supports multiple email addresses separated by semi-colons.
