country object

The country object represents domain data to identify the nationality of an address and provides functionality to perform phone and postal code masking. The database setup installs a default set of country objects including most countries.

The country object has one key property, countryCode, that uniquely identifies the country in the database. Once the object is created, the value of this property cannot be changed.

Methods

Method Description
retrieve Returns a specific country object based on the search criteria
retrieveCollection Returns a collection of country objects based on the search criteria
retrieveList Retrieves a rowSet of country information
saveCollection Performs multiple actions at once: inserting, updating, and deleting objects, as appropriate

Properties

Property Description
active Specifies whether the domain data the object provides is displayed in the user interface
addressFormat Specific address format required for mailings sent to the country
countryCode Two letter code designation for the country
countryName Contains the display string for the country
insertBy Contains the user ID of the Onyx user who initially saved the object
insertDate Contains the time and date on which the object was initially saved
phoneMask Mask used for the country's telephone code
postCodeMask Mask used for the country's postal code
updateBy Contains the user ID of the Onyx user who last modified the object
updateDate Contains the time and date on which the object was last modified
