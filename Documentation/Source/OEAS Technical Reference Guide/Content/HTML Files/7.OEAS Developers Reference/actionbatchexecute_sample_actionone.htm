Process first action: company.insert

The first action creates a company using the company name from the source list item. For each sourceListMap in the actionBatchXml the value pointed to by the sourceElementPath is copied to the targetElementPath. In this case, each source list item holds the name of the inserted company.

<sourceListMaps>

  <sourceListMap>

    <sourceElementPath>companyName</sourceElementPath>

    <targetElementPath>parameters/company/companyName</targetElementPath>

  </sourceListMap>

</sourceListMaps>

Using the updated methodInputs parameters, the method call is made on the object thatlogicalObjectName and logicalMethodName point to. In this case, company.insert is called to insert a new company in the database with phone and address objects. Error-handling behavior is determined by the value of onFailure.
