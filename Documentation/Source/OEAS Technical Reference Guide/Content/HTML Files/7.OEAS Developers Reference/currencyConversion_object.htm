currencyConversion object

The currencyConversion object enables users to access the currency conversion rate information from the database and provides the method for currency conversion. By default, OEAS does not provide a method for inserting currencyConversion objects. You must use a custom method to persist currencyConversion objects. Once you persist the objects, you can use them to perform conversions.

The currencyConversion object has three key properties, currencyCodeFrom, currencyCodeTo, and effectiveDate, that store conversion information in the database. The values of currencyCodeFrom, currencyCodeTo, and effectiveDate cannot be changed once the object is created.

Methods

Method Description

currencyConvert

Changes the value of a monetary quantity in one currency to its equivalent monetary quantity in another currency

retrieve

Returns a collection of objects based on the search criteria supplied in the parameters XML

saveCollection

Performs multiple actions in a single OEAS method call such as inserting, updating, and deleting objects, as appropriate

Properties

Property Description

currencyCodeFrom

The currency from which the conversion takes place.

currencyCodeTo

The currency to which the conversion takes place.

effectiveDate

The date when the currency exchange rate takes effect.

rate

The ratio for currency conversion.

triangulationCode

The third currency code, if it exists.
