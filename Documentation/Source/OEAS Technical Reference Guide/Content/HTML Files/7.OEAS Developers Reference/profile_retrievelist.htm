retrieveList (profile)
retrieveList method

The retrieveList method of the profile object returns information about OEP profiles that are associated with the specified user.

Remarks

When the userId parameter is 'sa', this method returns all OEP profiles.

Users are associated with OEP profiles in two ways:

  • Directly through profileUser instances that exist for a user.
  • Indirectly through profileRole instances associated to roles that a user is a member of.
