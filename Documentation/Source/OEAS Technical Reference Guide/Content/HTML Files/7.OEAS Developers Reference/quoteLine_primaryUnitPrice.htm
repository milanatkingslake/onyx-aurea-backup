primaryUnitPrice (quoteLine)

primaryUnitPrice property

The primaryUnitPrice property of the quoteLine object contains the price in the primary currency for the given product on the quoteLine.

Data type

float

Access

Read/write

Remarks

confidenceFactor x quantity ( primaryUnitPrice - primaryDiscount ) = quoteLine total

1.00 x 1 ( $2560.00 - 0 ) = $2560.00
