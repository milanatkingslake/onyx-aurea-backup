customerProduct object

The customerProduct object enables you to track customer product purchases and warranty information. With its methods, you can insert, update, retrieve, and delete customerProduct objects and customerProductDetail objects. You can also perform a filtered search on customerProduct and customerProductDetail objects.

The customerProduct object has one key property, primaryId, that uniquely identifies the customer product data in the database. Once the object is created, the value of this property cannot be changed.
