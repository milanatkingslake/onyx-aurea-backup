referenceId (referenceField)

referenceId property

The referenceId property of the referenceField object is an identifier that points to a collection of referenceLookup object instances.

Data type

i4

Access

Read/write

Remarks

OEAS generates the value of this property upon creation of a referenceField object.

You can update the referenceId property so that it contains the same value as another referenceField. If you do this, the referenceField will inherit the same referenceParameter objects as the other referenceField object. Typically, however, referenceFields have unique referenceId property values.
