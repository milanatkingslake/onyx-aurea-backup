Creating the custom literatureProduct (OPS)

Creating the custom literatureProduct

Creating the custom literatureProduct includes the following steps.

  • Creating a literatureProduct instance
  • Writing an XSL stylesheet for the boilerplate text and data merge

Creating a literatureProduct instance

Creating the literatureProduct instance associates an XSL stylesheet with a product number, specifies how the email content is sent, and how the product displays in the user interface. For more information on the steps to create a new literatureProduct, see Creating a literature product. You will need the information in the following table to create the product.

Writing an XSL stylesheet

You can write an XSL stylesheet to include whatever boilerplate and data merge you want to send to the customer. This example uses a brief form letter with a customized salutation and profile information. The email subsystem uses the stylesheet to transform the orderBatch XML. To use this sample, save the following code using the 'File Path' of the literatureProduct you just created.

<?xml version="1.0" encoding="utf-8" ?>

<!-- Copyright (c) 2002 Onyx Software Corporation. All rights reserved. -->

<xsl:stylesheet xmlns:xsl="http://www.w3.org/1999/XSL/Transform" version="1.0">

<xsl:output method="text" encoding="utf-8" indent="no" />

<xsl:template match="literatureOrderLine">

<xsl:text>Dear </xsl:text>

<xsl:value-of select="ancestor::literatureOrder/salutation"/>

<xsl:text> </xsl:text>

<xsl:value-of select="ancestor::literatureOrder/lastName"/>

<xsl:text>,

 Thank you for using Acme. We show the following profile information for you:

  First Name: </xsl:text><xsl:value-of select="ancestor::literatureOrder/firstName"/><xsl:text>

  Last Name: </xsl:text><xsl:value-of select="ancestor::literatureOrder/lastName"/><xsl:text>

 We at Acme appreciate your business and look forward to being of service to you.

 Please feel free to contact us at any time.

Respectfully,

Acme Customer Service

 </xsl:text>

</xsl:template>

</xsl:stylesheet>
