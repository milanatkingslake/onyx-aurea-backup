appointmentAttendee object

The appointmentAttendee object enables users to insert, update, retrieve, and delete appointmentAttendee object instances. These objects represent the persons or resources participating in the appointment.

The appointmentAttendee object has one key property, primaryId, that uniquely identifies object instances in the database. Once the object is created, the value of this property cannot be changed.

Methods

Method Description
retrieveCollection Returns a collection of objects based on the search criteria supplied in the parameters XML
saveCollection Performs multiple actions at once: inserting, updating, and deleting objects, as appropriate

Properties

Property Description
attendeeTypeCode Identifies the attendee as the meeting organizer, an invitee, or a resource
displayName The name of the attendee to display with the appointment
emailAddress The email address of the attendee
insertBy The Onyx user who initially saved the object instance
insertDate The time and date on which the object instance was initially saved
onyxTimestamp Facilitates concurrency control with the Onyx Enterprise Database
ownerId The unique identifier of the appointment object that owns this attendee
primaryId The unique identifier of the object
updateBy The user ID of the person who last updated the object instance
updateDate The time and date of the last object instance update
