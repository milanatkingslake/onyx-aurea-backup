update (company)

update method

The update method of the company object persists modifications to an existing object instance.

update standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
notificationAdapter.preUpdateEvent com This step is added for notification feature. It captures the data before updating the object
OnyxOptionalForeignKeyStep.OptForeignKey com

When primaryContactId is NULL, allows method to persist contactFirstName and contactLastName

When primaryContactId is non-NULL, sets contactFirstName and contactLastName to NULL
OnyxGuidGeneratorStep.GuidGenerator com Generates guid, which will be used by opKeyWordPutKeyWordsPost stored procedure defined in core step
OnyxDM.DataManager com Updates the specified company object
phone.saveCollection method Insert/update/delete existing phone collection with the new collection
address.saveCollection method Insert/update/delete existing address collection with the new collection
emailaddress.saveCollection method Insert/update/delete existing email address collection with the new collection
ConsonaCRM.Onyx.BusinessRuleManager business rule manager Associate an assignmentTerritory with the company.
ConsonaCRM.Onyx.BusinessRuleManager business rule manager Assign an assignmentTerritory with the company.
notificationAdapter.postInsertEvent com This step is added for notification feature. It captures the data after saving the object.
