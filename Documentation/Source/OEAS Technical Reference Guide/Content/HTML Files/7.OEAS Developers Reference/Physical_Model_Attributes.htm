Physical model attributes

This book describes the physical model attributes, as displayed in the Object Designer.

Type of OED Metadata

Description

Tables

Identifies a table used by OEAS in the database schema.

Columns

Defines table columns.

Database Actions

Defines the actions (retrieve, update, and delete) performed on each table.

Filter Rules

Defines rules for row-level filtering and security for each database action.

Procedures

Defines a stored procedure.

Procedure Parameters

Defines the parameters for a stored procedure.
