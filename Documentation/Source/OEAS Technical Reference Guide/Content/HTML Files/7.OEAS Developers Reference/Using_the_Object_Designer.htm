Using the Object Designer

This book provides instructions for the following tasks that apply to all nodes of the OED: general tasks, such as adding, modifying, and deleting items; and maintenance tasks, such as validating, saving, and publishing OED versions.

  1. Moving items in the navigation tree
  2. Adding items to the OED
  3. Modifying items in the OED
  4. Enabling or disabling items in the OED
  5. Copying items in the navigation tree
  6. Pasting items in the navigation tree
  7. Deleting items in the OED
  8. Validating changes to the OED
  9. Publishing your changes to the OED
  10. Saving your changes

Information about specific nodes of the OED are described in the Object attributes and Physical Model attributes sections.

The instructions contained within this section are intended for users who have already reviewed the Onyx Enterprise Dictionary Concepts and Managing the OED topics.
