retrieveListByOwner (fileLink)

retrieveListByOwner method

The retrieveListByOwner method of the fileLink object searches the database by ownerId and returns a list of file attachment records.

retrieveList standard method information

Parameters

Parameter Direction Data type Content Type Description
ownerId (required) in string The unique ID for the customer whose file attachment records are to be retrieved
rowSet out object all(out) A list of rows that contain data about the file attachments that belong to the customer
