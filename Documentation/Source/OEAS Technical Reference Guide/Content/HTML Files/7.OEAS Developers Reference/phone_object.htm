phone object

The phone object contains all the information about primary and alternate phones that are associated with a customer record. Phone objects may be saved with the customer or may be saved separately. The phones collection of the company or individual objects contains phone objects.

The phone object has one key property, primaryId, that uniquely identifies the phone in the database. Once the object is created, the value of this property cannot be changed.
