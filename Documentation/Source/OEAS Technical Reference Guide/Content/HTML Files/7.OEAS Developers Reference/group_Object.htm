group object

The group object enables users to delete, insert, retrieve, and update user groups.

Methods

Method Description
retrieve Returns an existing group object instance
retrieveList Returns a list of all groups
retrieveListAncestors Returns a list of groups hierarchically above the specified child group
retrieveListGroups Returns group object instances owned by the specified parent group
retrieveListUsers Returns all user object instances that belong to the specified group
saveCollection Saves a collection of group object instances

Properties

Property Description
active An integer that indicates whether the group is active or inactive
authorityId A bitmask that specifies group access to functions of Onyx Employee Portal (OEP)
description The name of the group
detailAccess A bitmask that specifies group access to functions of OEP
emailAlias The email alias of the group
escalationId The user ID of a point person for the group
groupId A string that uniquely identifies the group
insertBy The ID of the end-user who created the object instance
insertDate The date on which the end-user created the object instance
managerId The user ID of the group's manager
parentGroupId The group that owns the group object instance
updateBy The user ID of the person who last updated the object instance
updateDate The date of the last object instance update
