Mapping standard maintenance properties

Each object for which you want to enable the SQL generation engine should include the standard maintenance properties. Never specify persistence maps for a number of the maintenance properties, because SQL generation automatically supplies values on insert and update methods. The "active" property is the only maintenance property that should ever have a persistence map. The following table summarizes the maintenance properties and their associated map types.
