updateDate

updateDate property

The updateDate property of an object indicates the time and date on which modifications to the object were last saved.

This is a generic topic that applies to all properties with the same name. The remarks section may contain details specific to a given object.

Data type

dateTime

Access

Read-only; for more information,  see Read Only attribute.

Remarks

The value of the updateDate property of an object is set numerous times: when the object is initially saved and each time that it is updated. When a new object is created (saved for the first time), the updateDate property is set with the same value as the insertDate property.

The database obtains the date value from the database server clock. The database converts the date value to a UTC date before saving it.

The dateTime data type is formatted as follows:

YYYY-MM-DD hh:mm:ss

where:

YYYY = year

MM = month

DD = day

hh = hour

mm = minute

ss = second
