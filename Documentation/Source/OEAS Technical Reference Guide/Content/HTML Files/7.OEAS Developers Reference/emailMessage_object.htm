emailMessage object

The emailMessage object records information for each email drafted, sent, or received into the system.

Methods

Method Description
delete Logically deletes a specified object instance
insert Saves a new object instance
retrieve Returns a single instance of an object
retrieveListAssociation Returns information about the object instances that are associated with a specified emailMessage instance
retrieveListByAssociation Returns information about emailMessage instances that are associated with a specified object instance
retrieveListByFilter Returns information about emailMessage instances based on the input parameters
retrieveListByThreadId Returns information about all emailMessage instances in a thread
retrieveListQuickSearch Returns information about emailMessage instances based on the input parameters
send Sends an email message
update Saves modifications to a specified instance of an object

Properties

Property Description
