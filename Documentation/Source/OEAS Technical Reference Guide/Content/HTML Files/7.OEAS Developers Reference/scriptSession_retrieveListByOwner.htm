retrieveListByOwner (scriptSession)

retrieveListByOwner method

The retrieveListByOwner method of the scriptSession object returns summary information for all scriptSession objects belonging to the specified owner. The data is returned as a rowSet object in the parametersReturn XML.

Remarks

If the ownerType is individual, this method returns all scriptSessions for which the individual is the owner and all scriptSessions for which the individual is the contact. For company and incident ownerTypes, this method only returns scriptSessions for which the company or incident is the owner.

Example

The following example uses retrieveListByOwner to return specific information on scriptSession objects for a customer with an ID of '4'. The owner is an individual, so the method returns two rows. The first row is a session owned by the individual. The second row is a session for which the individual has been made the contact.
