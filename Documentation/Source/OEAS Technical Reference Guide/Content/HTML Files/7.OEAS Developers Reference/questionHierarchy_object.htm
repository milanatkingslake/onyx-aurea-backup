questionHierarchy Object

questionHierarchy object

The questionHierarchy object enables you to configure the sequence of questions in a survey. If you want to present the next question in a survey based on the customer's response to the current question, you create a questionHierarchy object. Each questionHierarchy object specifies the next question to follow a specific question-response pair.

Methods

Method

Description

retrieveCollection

Returns a collection of questionHierarchy objects based on the search criteria supplied in the parameters XML

saveCollection

Performs multiple actions at once: inserting, updating, and deleting questionHierarchy objects, as appropriate

Properties

Property

Description

active

Specifies whether the domain data the object provides is available to the user interface

insertBy

Contains the user ID of the Onyx user who initially saves the object

insertDate

Contains the time and date on which the object is initially saved

Contains the primaryId of the question to follow the current question-response pair

nextQuestionText

Contains the text from the question object corresponding to the nextQuestionId

Contains the primaryId of the current question in the hierarchy

questionText

Contains the text from the question object corresponding to the questionId

responseId

Contains the primaryId of the response to the current question

Contains the text from the response object corresponding to the responseId

Contains the primaryId of the survey containing the question hierarchy

updateBy

Contains the user ID of the Onyx user who last saved modifications to the object

updateDate

Indicates the time and date on which modifications to the object were last saved
