saveCollection (externalContact)

saveCollection method

The saveCollection method of the externalContact object performs multiple actions at once: inserting, updating, and deleting objects, as appropriate.

saveCollection standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Performs multiple actions with externalContact objects

Example

Parameters

In this example, the saveCollection method sets the action attribute of the externalContact instance to logically delete the record.

<parameters>

   <externalContacts objectType="externalContact" collection="1">

      <externalContact objectType="externalContact" action="delete">

         <primaryId>92F6837B-F0EA-478E-A4D9-AFFFA36579E8</primaryId>

         <ownerId/>

         <ownerType/>

         <contactId/>

         <contactTypeId/>

         <contactUserName/>

         <comment></comment>

      </externalContact>

   </externalContacts>

</parameters>
