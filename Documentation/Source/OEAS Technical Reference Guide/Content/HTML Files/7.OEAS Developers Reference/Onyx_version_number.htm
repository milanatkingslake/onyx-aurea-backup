Onyx version number

The Onyx version number is comprised of three numbers (major, minor, and revision) that are separated by periods. This number is incremented by Onyx and is noneditable.

Example: 7.00.000

Where:

  • The first number (7) is the major version number, incremented only for major content changes
  • The second number (00) is the minor version number, incremented only for minor content changes
  • The third number (000) is the revision version number, incremented only for very small content changes
