workNote object

The workNote object enables users to create and view notes related to incident or task objects.

The workNote object is a legacy interface. A step component of the insert method transforms workNote object instances to workNoteHeader and workNoteDetail object instances.

The workNote object has two key properties, primaryId and seqNum, that uniquely identify the workNote in the database. Once the object is created, the values of these properties cannot be changed.

Methods

Method

Description

Properties

Property

Description

Uniquely identifies the parent object of the child object

A bitmask that determines which portions of a workNoteText are displayed in an web-based user interface

seqNum

The ordinal number of a workNote object in a series of workNote objects comprising a compound workNote object

Contains the user input free-text for a seqNum and primaryId combination
