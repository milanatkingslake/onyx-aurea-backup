categoryId (task)

categoryId property

The categoryId property of the task object contains an ID representing the type of task.

Data type

i4

Access

Read/write

Remarks

The categoryId of a task object corresponds to the categoryId of its parent object.

This property references an existing category. The default task categoryIds are:

  • 11: Support
  • 12: Service
  • 13: Sales

You can also create custom task types.
