literatureShipVia object

The literatureShipVia object exposes a single method that returns the list of current shipping methods available for distributing literature products to customers. The values available through this object are used in the shipViaId property of the literatureOrder object. To edit the list of values, use the Table Administration tool available in the Onyx Enterprise Studio.
