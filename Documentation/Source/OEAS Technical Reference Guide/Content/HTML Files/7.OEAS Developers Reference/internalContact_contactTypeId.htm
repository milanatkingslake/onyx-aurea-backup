contactTypeId (internalContact)

contactTypeId property

The contactTypeId property of the internalContact object is the unique numerical value associated with a specific contactType.

Data type

i4

Access

Read/write

Remarks

The contactTypeId property is a required field and must be included with an insert action of the saveCollection method.

The contactTypeId identifies the type of owner of the internalContact object.

The contactTypeId is a unique identifier that is used to obtain the contactType object and determine information about the contact. The contactTypeId is the primaryId for the associated contactType object.
