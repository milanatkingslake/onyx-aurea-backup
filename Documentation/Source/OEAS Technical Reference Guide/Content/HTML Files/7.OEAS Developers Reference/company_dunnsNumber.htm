dunnsNumber property

The dunnsNumber property of the company object reflects the company's D-U-N-S Number.

Data type

string (maximum length)

Access

Read/write

Remarks

The D&B (Dun & Bradstreet) D-U-N-S Number is D&B's distinctive nine-digit identification sequence, which links you to a wealth of quality information products and services originating exclusively from D&B. The D&B D-U-N-S Number is an internationally recognized common company identifier in EDI and global electronic commerce transactions.

An example of a dunnsNumber is 83-557-0359, which is the D-U-N-S Number for Onyx Software Corporation.
