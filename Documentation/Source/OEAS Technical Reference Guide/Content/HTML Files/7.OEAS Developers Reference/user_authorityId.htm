authorityId (user)

authorityId property

The authorityId property of the user object is a bitmask that specifies user access to functions of the Onyx Employee Portal—Windows client.

Data type

i4

Access

Read/write

Remarks

The role and resource objects are used in OEAS to support user access control; therefore, OEAS does not use the authorityId property. It is used to maintain compatibility with Onyx Employee Portal—Windows client.
