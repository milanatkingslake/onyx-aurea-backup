Copying items in the navigation tree

To copy an item to the clipboard:

  • In the navigation area, select an item of the type that you want to copy and click .

    Some items cannot be copied: key maps, input and output maps for procedure parameters, the business objects collection, the tables collection, and the procedures collection.

Warning: When renaming, or copying and pasting business objects or methods, some fields must be changed manually. You must check the object names on parameters. In each method's stepAdditionalData field the stepObject, stepMethod, stepName, and dataPath elements of the field may need to be changed. If you do not check these fields and make the necessary corrections, the copied or renamed object or method will not work properly.
