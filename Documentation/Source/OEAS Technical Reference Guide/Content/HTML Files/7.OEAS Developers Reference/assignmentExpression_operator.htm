operator (assignmentExpression)

operator property

The operator property of the assignmentExpression object contains the operator portion of the expression.

Data type

string

Access

Read/write

Remarks

Supported operators:

Operator Description
= Is equal to
!= Is not equal to
between Is between two values
> Is greater than
< Is less than
startsWith Evaluates to true if the property identified by propertyName begins with the supplied string
hasAValue Evaluates to true if the property identified by propertyName contains data
isInList Evaluates to true if the value of propertyName can be found in a semi-colon delimited list
