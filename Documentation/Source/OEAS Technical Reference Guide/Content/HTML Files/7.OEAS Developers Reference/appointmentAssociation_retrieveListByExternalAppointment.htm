retrieveListByExternalAppointment (appointmentAssociation)

retrieveListByExternalAppointment method

The retrieveListByExternalAppointment method of the appointmentAssociation object returns a list of appointmentAssociation records that match the supplied input criteria.

Parameters

Parameter Direction Data type Content type Description
externalAppointmentId in string n/a The unique identifier of the external appointment
rowSet out object all Contains rowSet column definitions and the appointmentAssociation records in the database that match the specified search criteria

Columns returned in the rowSet object

The contents of each row in result set are dependent on the type of object whose data is returned. If the object does not have the property referenced, the column is empty.

columnOrdinal columnName Physical column
1 primaryId appointment_association.appointment_association_id
2 appointmentPrimaryId appointment.appointment_id
3 associatedWithPrimaryId company.company_id or individual.individual_id or incident.incident_id
4 associatedWithSecondaryId company.secondary_id or individual.secondary_id or incident.secondary_id
5 associatedWithType appointment_association.associated_with_type_enum
6 individualPrimaryId individual.individual_id
7 individualSecondaryId individual.secondary_id
8 companyPrimaryId company.company_id
9 companySecondaryId company.secondary_id
10 salutation individual.salutation
11 firstname individual.first_name or company.contact_first_name
12 middleName individual.first_name
13 lastName individual.last_name or company.contact_last_name
14 suffix individual.suffix
15 companyName company.company_name
16 incidentDescription incident.incident_desc1

Step data Step type Description
