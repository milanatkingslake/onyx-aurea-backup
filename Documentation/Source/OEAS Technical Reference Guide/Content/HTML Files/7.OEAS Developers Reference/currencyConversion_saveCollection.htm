saveCollection method

The saveCollection method of the currencyConversion object performs multiple actions in a single OEAS method call such as inserting, updating, and deleting objects, as appropriate.

Parameters

Parameter Direction Data type Content type Description

currencyConversions

inOut

object

any

currencyConversion object whose content="all" or "partial"

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates the input parameters

OnyxDM.DataManager

com

Inserts/updates/deletes the currencyConversion objects
