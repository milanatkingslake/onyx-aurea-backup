batchSizeOrderLines (subsystem)

batchSizeOrderLines property

The batchSizeOrderLines property of the subsystem object contains the approximate number of literatureOrderLine objects that the subsystem attempts to process at a given batch interval time.

Data type

i4

Access

Read/write

Remarks

The batchSizeOrderLines property is approximate because the subsystem component may retrieve more or less order lines based on internal subsystem component logic. This number is intended as a guideline for the system and is not meant to indicate an exact quantity.
