stepAdditionalData (optional foreign key)

stepAdditionalData

The stepAdditionalData contains the design time information for the OnyxOptionalForeignKey step to determine on which fields to perform preprocessing. To do this you add the OnyxOptionalForeignKey step to the method and provide the necessary stepAdditionalData for the step.

Example

stepAdditionalData contains...

   stepObject=company;

   stepMethod=insert;

   dataPath=;

   foreignKeyPropertyName=primaryContactId;

   denormFields=primaryContactFirstName,primaryContactLastName;

This stepAdditionalData tells the OnyxOptionalForeignKey step to check the foreignKey field in the parameters XML for a valid value. If the foreignKey field in the parameters XML contains a valid value, the step sets the value of the denormFields to null. In this case, the step checks the parameters XML for a valid value in the primaryContactId. If it finds a valid value in primaryContactId, it sets the primaryContactFirstName and primaryContactLastName values to null.
