retrievePartnerListOfNewLeads (incident)

retrievePartnerListOfNewLeads method

The retrievePartnerListOfNewLeads method of the incident object returns the list of incidents assigned to a partner depending on the input parameters. These incidents have not been assigned to partner users or partner groups.
