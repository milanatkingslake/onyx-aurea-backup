methodParameterXml (publishableObject)

methodParameterXml property

The methodParameterXml property of the publishableObject object contains XML used by the method that retrieves the domain data to publish.

Data type

string

Access

Read/write

Remarks

When the publish method uses a retrieveCollection method, the methodParameterXml property contains the <parameters> input node for the retrieveCollection method. In the case of the campaign object, the default implementation OEAS uses the following XML to retrieve all active campaign object instances.

<parameters>

   <campaigns objectType="campaign" collection="1">

      <campaign objectType="campaign">

         <active>1</active>

      </campaign>

   </campaigns>

</parameters>

When the publish method uses the extract method, the methodParameterXml property contains the input parameters for the extract method. In the case of the currencyConversion object, the default implementation OEAS uses the following XML, which specifies a database view, to retrieve all active currencyConversion object instances.

<parameters>

   <viewName>currencyConversion</viewName>

   <objectName>currencyConversion</objectName>

   <objectNamePlural>currencyConversions</objectNamePlural>

</parameters>

The extract method appends "_DP_VIEW" to the value of the <viewName> element; therefore, the name of the view is currencyConversion_DP_VIEW. Because extract method formats its results to match the structure of a retrieveCollection <parametersReturn> element, the <objectNamePlural> element is required to specify the name of collection node in the retrieveCollection results.
