saveCollection (internalContact)

saveCollection method

The saveCollection method of the internalContact object performs multiple actions at once: inserting, updating, and deleting objects, as appropriate.

saveCollection standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Saves a collection of internal contacts

Example

Parameters

In this example, we delete an internalContact record using the "delete" action.

      <internalContact objectType="internalContact" action="delete">

         <ownerId/>

         <ownerType/>

         <contactTypeId/>

         <contactUserId/>

         <contactUserName/>

         <comment/>

      </internalContact>

   </internalContacts>

</parameters>
