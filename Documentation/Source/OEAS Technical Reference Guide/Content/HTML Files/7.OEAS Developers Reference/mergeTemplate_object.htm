mergeTemplate object

mergeTemplate object

The mergeTemplate object holds a text string that can include tags that can be used to merge data from customer, incident, workTicket, and other objects.

The mergeTemplate object has one key property, primaryId, that uniquely identifies the object instance. Once the object is created, the value of this property cannot be changed.
