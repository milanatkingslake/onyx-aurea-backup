keyWord object

The keyWord object enables users to retrieve a list of frequently used words or phrases to use as search criteria when retrieving incident object instances. Users enter a new tag or choose from a list of commonly used tags. OEAS compiles a list of tags per incidentCategory based on a count of the times that a tag has been used.

The keyWord object does not have any properties.
