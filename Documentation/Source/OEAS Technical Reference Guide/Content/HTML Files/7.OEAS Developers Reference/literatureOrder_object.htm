literatureOrder Object

literatureOrder object

The literatureOrder object stores information about the recipient of print or electronic communications from your company. Onyx users or some business logic can enter the order in the database. Other manual or automated processes, such as the Onyx Processing Server, handle fulfillment of the order at a later time.

The literatureOrder object has a single collection, literatureOrderLines, that contains information about the items ordered in literatureOrderLine objects. The literatureOrder object participates in a master detail relationship with the literatureOrderLine objects.

The literatureOrder object has one key property, primaryId, that uniquely identifies the literature order in the database. The primaryId is a system generated GUID and cannot be modified.
