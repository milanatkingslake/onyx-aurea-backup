Error constants

Unlike the methods of the ICache interface, which return a status value after each call, successful or otherwise, the methods of the DictionaryProxy interface only indicate status after a run-time failure. They do this by modifying the Windows error object. This object contains a unique number and description that explain the cause behind the method failure. The following table lists the error number values and their corresponding descriptions.
