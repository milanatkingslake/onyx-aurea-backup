update (customerProduct)

update method

The update method of the customerProduct object saves modifications to an existing customerProduct object and inserts, updates, or deletes customerProductDetail objects.

update standard method information

Steps

Step data Step type Description

OnyxGuidGeneratorStep.GuidGenerator

com

Calls a component that generates GUID primary key values used for inserting the tags identified in the customerProduct’s Tags property

OnyxDataValidationStep.DataValidation

com

Data validation step

OnyxDM.DataManager

com

Update the customerProduct object and modifies each associated customerProductDetail object

Saves modifications to the specified task and calls a post-procedure that inserts or updates the tag object with the terms listed in the task's tags property
customerProductDetail.saveCollection method Insert/update/delete existing customerProductDetail collection with the new collection
