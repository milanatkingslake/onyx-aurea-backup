purge (workNote)

purge method

The purge method of the workNote object deletes one or more workNote object instances having the specified primaryId.

Parameters

Parameter

Direction

Data type

Content type

Description

primaryId

in

string

n/a

The ID of the workNote to be purged

ownerId

OnyxDM.DataManager

com

Purges the workNote objects

Remarks

A typical scenario for using the purge method occurs when a user saves a lengthy work note before completing it. On the next work note save by the user, the user interface calls the purge method on the previously saved records and inserts the latest version of the work note.

During the purge process, the seqNum property is used along with the primaryId property to uniquely identify workNote object instances. Therefore, specifying a primaryId in the purge method input parameters results in the deletion of a block of work notes that share the specified primaryId.
