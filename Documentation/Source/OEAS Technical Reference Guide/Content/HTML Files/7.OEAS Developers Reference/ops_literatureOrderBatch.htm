literatureOrderBatch (OPS)

literatureOrderBatch

The literatureOrderBatch node of the orderBatch XML contains the details that the fulfillment engine uses to process a given batch.

Note: For a complete orderBatch example, you can look in the Utilities directory of the product CD for orderBatchEmailSample.xml and orderBatchPrintSample.xml.

The following is an example of the literatureOrderBatch node of the orderBatch XML.

<literatureOrderBatch objectType="literatureOrderBatch">

   <primaryId/>

   <secondaryId/>

   <subsystemId/>

   <fulfillmentTypeList/>

   <batchOrderLineSize/>

   <immediatePriority/>

   <batchCompleted/>

   <insertBy/>

   <insertDate/>

   <updateBy/>

   <updateDate/>

   <onyxTimestamp/>

</literatureOrderBatch>
