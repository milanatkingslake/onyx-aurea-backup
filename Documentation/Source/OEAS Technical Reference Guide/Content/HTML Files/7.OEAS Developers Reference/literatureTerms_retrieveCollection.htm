retrieveCollection method

A retrieveCollection method of the literatureTerms object returns the collection of object instances based on the search criteria.

Parameters

Parameter Direction Data type Content type Description

literatureTermsSets

inOut

object

any

A collection of literatureTermsSets

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates the input parameters

OnyxDM.DataManager

com

Retrieves the collection of the literatureTerms objects
