getNotificationCounts (notification)

getNotificationCounts method

The getNotificationCounts method of the notification object returns the number of due reminders and unread messages for the specified user.

Parameters

Parameter

Data type

Direction

Description

userId (required)

string

in

A string that identifies an Onyx user for whom reminders and unread mail message are checked.

reminderStartDate

dateTime

in

The time and date that starts the block of time to check for due reminders. The default value is the current date and time.

reminderEndDate

dateTime

in

The time and date that ends the block of time to check for due reminders. The default value is the current date and time plus 15 minutes.

messageCount

i4

out

The number of unread messages for the user.

reminderCount

i4

out

The number of reminders for the user that have a dueBy value that falls within the specified date and time range.

newestMessageDate

dateTime

out

The time and date of the most recent unread message sent to the user. If there are no unread messages, this parameter is set to NULL.

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Retrieves message and reminder counts

Remarks

The userId parameter is required; its default value is not the current user. Although a user interface typically calls this method to get counts for the current user, the userId parameter can be any valid userId.
