immediatePriorityAlways (subsystem)

immediatePriorityAlways property

The immediatePriorityAlways property of the subsystem object indicates whether the subsystem processes immediate priority orders regardless of operationStartTime and operationStopTime values.

Data type

boolean

Access

Read/write

Remarks

Valid values for the immediatePriorityAlways property include the following.

Value

Description

1

Process immediate priority jobs regardless of subsystem start and stop times

0

Process immediate priority jobs only between subsystem start and stop times
