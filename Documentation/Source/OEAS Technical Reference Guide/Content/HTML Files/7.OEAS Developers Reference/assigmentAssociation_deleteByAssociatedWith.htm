deleteByAssociatedWith (assigmentAssociation)

deleteByAssociatedWith method

The deleteByAssociatedWith method of the assignmentAssociation object deletes all assigmentAssociations for an OEAS object.

Parameter

Parameter Direction Data type Content type Description
associatedWithId in string n/a The unique identifier of the object associated with the appointments

Steps

The default steps for the deleteByAssociatedWith method of the assignmentAssociation object are:

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Calls a replace procedure to collect all assignmentAssociation records for the associated object and delete them.
