parametersReturn XML

For the createFolder method, the parametersReturn XML is empty.

For the retrieveFolderItems method, the parametersReturn XML contains a single child node called folderItemsXml.

<parametersReturn>

   <folderItemsXml/>

</parametersReturn>

The folderItemsXml contains a root node with a collection of one or more items. The following table describes the schema for the folderItemsXml.

If type is shortcut, this specifies the path to the target folder, report, query, or reportView.

For report, query, or reportView items, valid values include 0 or 1 depending on whether or not the item has a default output.

For other types of items, this is always empty.
