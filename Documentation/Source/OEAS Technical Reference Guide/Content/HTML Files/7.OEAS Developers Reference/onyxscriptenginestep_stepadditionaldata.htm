stepAdditionalData

The OnyxScriptEngineStep uses the StepAdditionalData parameter of the IStep Execute method to determine which of its internal functions to call. The following code shows the expected StepAdditionalData where METHOD_NAME is replaced by the appropriate logical method name advanceStep, loadStep, or resumeSession. If you use any other logical method name, it returns an error.

stepObject=scriptSession;

stepMethod=METHOD_NAME;

dataPath=;
