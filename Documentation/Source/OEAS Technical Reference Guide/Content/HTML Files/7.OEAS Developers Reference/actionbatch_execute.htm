execute (actionBatch)

execute method

The execute method of the actionBatch object allows a caller to request multiple method calls to various OEAS business objects in a single call to the Onyx Transaction Manager (OTM).

Parameters

Parameter Direction Data type Content type Description

actionBatchXml

in

string

n/a

XML that defines the actions of the batch processing. This XML must be encoded.

statusXml

out

string

n/a

The status of all the calls processed by the execute method.

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates the input parameters

OnyxActionBatchExecute.ActionBatchExecute

com

Executes a series of method calls using the OTM.

Remarks

The contents of the actionBatchXml parameter must conform to the actionBatchXml schema.

For more information on how to use actionBatch.execute, see the following topics in this reference:

  • OnyxActionBatchExecute step component
  • actionBatchXml schema
  • actionBatch statusXml schema

For an example of using actionBatch.execute, see How to use actionBatch to execute a series of OEAS method calls.
