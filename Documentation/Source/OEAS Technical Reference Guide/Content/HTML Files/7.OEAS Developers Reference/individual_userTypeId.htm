userTypeId (individual)

userTypeId property

The userTypeId property of the individual object contains the numerical ID that identifies the type or category to which an individual belongs.

Data type

i4

Access

Read/write

Remarks

The userTypeId property is an integer value that can be configured using the Reference Table administration tool. An example of a userTypeId might be 186, which could have a description of "Reseller" in the reference objects. There is a parent child relationship between a userTypeId and a userSubTypeId.
