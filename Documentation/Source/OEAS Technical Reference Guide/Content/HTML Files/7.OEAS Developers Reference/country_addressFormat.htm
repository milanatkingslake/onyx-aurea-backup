addressFormat (country)

addressFormat property

The addressFormat property of the country object identifies the specific address format required for mailings sent to the country.

Data type

string (maximum length)

Access

Read/write

Remarks

The address format is a specific format required for mailings sent to the country. You can configure the addressFormat value in the country table using the Table Administration tool.

If no address format is known, the addressFomat property can be left as NULL. Currently, during setup, the addressFormat is set to NULL for all country objects.
