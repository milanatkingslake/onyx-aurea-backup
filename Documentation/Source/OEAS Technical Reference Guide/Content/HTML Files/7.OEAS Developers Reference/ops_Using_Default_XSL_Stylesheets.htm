Using sample email templates

Onyx Processing Server (OPS) includes sample email templates that you can use as is or modify for your customization. The OPS setup installs the templates on the OPS server with the file names given in the following table. The email engine uses XSL stylesheets to transform the input orderBatch XML into email messages that it sends to customers or employees.

By default, OPS Setup places the following templates in C:\Program Files\Onyx\ProcessingServer\Document Templates.

Sample email templates

Default email content

Creates SMTP headers and default email body content  from orderBatch literatureOrder data. One template is configured per email subsystem in OPS Administration.

OPSEmailDefaultSmtpHeaders.xsl

Plain text email

Merges orderBatch literatureOrderLine data with plain text to create optional SMTP headers and email body content. One template is configured per literature product in Product Administration.

OPSTextEmailBasic.xsl

HTML email

Merges orderBatch literatureOrderLine data with HTML content to create optional SMTP headers and email body content. One template is configured per literature product in Product Administration.

OPSHTMLEmailBasic.xsl

HTML order batch report

Merges orderBatch data into tabular listing of order lines per batch using HTML format. One template is configured per subsystem in OPS Administration.

OPSHTMLEmailBatchSysDoc.xsl

HTML order report

Merges orderBatch data into tabular listing of order lines per order using HTML format. One template is configured per subsystem in OPS Administration.

OPSHTMLEmailOrderSysDoc.xsl

Note: The last two sample templates are examples of system documents.
