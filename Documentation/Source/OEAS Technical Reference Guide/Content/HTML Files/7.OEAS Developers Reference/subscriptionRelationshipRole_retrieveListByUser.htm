retrieveListByUser (subscriptionRelationshipRole)

retrieveListByUser method

The retrieveListByUser method of the subscriptionRelationshipRole object retrieves subscription rules applied to roles that the specified user is a member of.

Remarks

To get all the subscriptionRelationshipRole object instances for the specified user and subscription service, specify NULLs for the primaryObjectType and secondaryObjectType. To filter the result set, specify any combination of primaryObjectType, secondaryObjectType, and active.

Specifying true for the active parameter can return active subscriptionRelationshipRole instances that override active subscriptionRelationship instances. Specifying false can return an inactive subscriptionRelationship instance, and inactive and active subscriptionRelationshipRole instances for that relationship.

If active is empty or NULL, all active and inactive subsc_relationship and subsc_relationship_role will be returned.
