saveCollection (referenceField)

saveCollection method

The saveCollection method of the referenceField object performs multiple actions at once: inserting, updating, and deleting objects, as appropriate.

saveCollection standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters.

OnyxDM.DataManager

com

Inserts, update, and deletes a collection of referenceField objects.

For insert actions, stores the 1 value in the rank column of the reference_field table for the record inserted for a referenceField object instance. This action is based on a literal logical-to-physical map, which specifies a non-null column that has no corresponding property on the logical object.
