retrieve (message)

retrieve method

The retrieve method of the message object searches the database by primaryId and returns a message object instance.

retrieve standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Obtains the specified message record from the database

Uses a database function mapped to the attachmentSecondaryId property to retrieve a value for this property

Remarks

The recipientUserIdList and recipientGroupIdList elements are empty in the parametersReturn because these properties are write-only on inserts. OEAS creates messageDestination object instances for managing message distribution and read status. Onyx Employee Portal (OEP) reproduces the recipient list at the top of the message body.

The database function that obtains the attachmentSecondaryId property uses the values of the message object's attachmentType and attachmentId properties to obtain the value of the secondaryId property from the attached company, individual, incident, or workTicket object instance.
