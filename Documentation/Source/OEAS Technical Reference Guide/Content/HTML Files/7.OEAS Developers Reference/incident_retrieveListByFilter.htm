retrieveListByFilter (incident)

retrieveListByFilter method

The retrieveListByFilter method of the incident object searches the database by the filter criteria you pass in and returns a rowSet of incident records.

retrieveList standard method information

Remarks

The retrieveListByFilter search is not case sensitive for the keyWords input parameter.

This method supports wild card substitution in search criteria. For more information on wild card substitution, see the retrieveList method (standard behavior) topic.

When searching the database by group (by passing the group parameter in the input XML), you must specify each group and subgroup that you want to search. Subgroups are not automatically returned with their parent group.

If you use both the assignedTo parameter and the group parameter as search criteria for this method, you might receive duplicate results in the returnXML. This happens when the user identified by the assignedTo parameter is a member of the group represented by the group parameter.

This method supports paging functionality. For more information, see the Paging functionality.
