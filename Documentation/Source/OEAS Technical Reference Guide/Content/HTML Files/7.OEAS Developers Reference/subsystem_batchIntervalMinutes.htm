batchIntervalMinutes (subsystem)

batchIntervalMinutes property

The batchIntervalMinutes property of the subsystem object contains the time in minutes that the subsystem waits before processing a new group of orders.

Data type

i4

Access

Read/write

Remarks

The subsystem satisfies immediate priority orders before retrieving batch priority orders. For this reason, a given batch interval may contain only immediate priority orders, or both immediate and batch priority orders.

The time interval is absolute based on the operationStartTime. If you set the interval to five (5) minutes, for example, the subsystem attempts to start processing a new batch every five minutes starting with the operationStartTime. If the subsystem is still processing a previous batch, it continues processing without starting a new batch.
