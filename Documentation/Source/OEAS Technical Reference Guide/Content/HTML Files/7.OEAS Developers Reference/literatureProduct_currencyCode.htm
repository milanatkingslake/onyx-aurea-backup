currencyCode (literatureProduct)

currencyCode property

The currencyCode property of the literatureProduct object contains one-to-three-letter alphabetic code that represents the unit of currency referenced by the cost and listPrice properties. Currency information is available through the currency object.

Data type

string (maximum length)

Access

Read/write

Remarks

An example of a currencyCode could be an AUD to indicate an Australian Dollar, a PTE to indicate a Portugese Escudo, or a VEB to indicate a Venezualan Bolivar.

To make changes to, or add, a currencyCode, use the Table Administration tool.
