Integrating a Custom Step Component

Integrating a custom step component

To add a custom step component to the OEAS, you must modify the system in a few places before your integration is complete. The DLL that hosts the step component must be added to the COM+ application on the server, and the business object methods that use the step must be updated using the Onyx Object Designer. If the change affects the way client applications collect or display data, then they must also be updated.

Add the component to a COM+ application

Add the new step component to the OnyxEASLib COM+ library application using the Windows 2000 Component Services manager.

To add a new component to a COM+ application:

  1. Open the Component Services application and locate the OnyxEASLib library application that will house the new component.
  2. Expand the application to reveal the Components folder.
  3. Right-click the Components folder and select New, Component from the pop-up menu.
  4. Click Install New Components.
  5. Navigate to the location of the component and click Open.
  6. Click Next and then Finish to exit the wizard.

Apply the necessary Roles to the component to enforce COM+ security. Verify that the component has the proper settings for transaction support.

Add the component to a business object method

Using the Object Designer, edit the Onyx Enterprise Dictionary (OED) to add the new component to the business object methods that will use it. See Object Designer Help for more information about adding a step to a method.

After the changes have been made to the OED, restart the OnyxEASApp COM+ application for the changes to take effect.
