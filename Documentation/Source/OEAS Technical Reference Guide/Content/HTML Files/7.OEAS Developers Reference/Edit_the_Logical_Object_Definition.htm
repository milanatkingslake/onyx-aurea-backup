Step 5: Modify the Company Object
Modify the company object

The Company object must be modified to:

  1. Add the stock ticker symbol property. This property is where the step component returns the stock ticker symbol data.
  2. Add a call to the step component into the Company object's retrieve method.

Step 1 - Adding the stock ticker symbol property

Follow the procedure described in the Adding a Property example topic to add the stock ticker symbol property to the company object.

Step 2 - Adding the step component to the retrieve method

Perform the following procedure to add the step component call to the Company object retrieve method.

  1. Open Onyx Enterprise Studio Object Designer.
  2. Navigate to Logical Business Objects\company\Methods\retrieve\Steps.
  3. Click the Add button to create a new step.
  4. In the Name: field, enter a user friendly name for the step such as retrieveStockSymbol.
  5. Enter the following fields:

    Description: Custom Step Component Call

    Step Type: com

    Step Data (ProgID): CustomStep.StepComponent

    Additional Data: stepName=400;stepObject=company;stepMethod=retrieve;dataPath=

    Override Template: (leave blank)

    Installed By: Customization

    Disabled: (leave unchecked)

  6. Click the Save button to save your changes.
  7. Optionally, you can click the Validate button to verify that the modifications you made are acceptable. When you publish, the validation is also performed, so this is an optional step.
  8. Click the Publish button. Answer Yes to the prompt that asks if you want to overwrite the current OED.
  9. In the next step, you will shut down the Onyx App COM+ to cause the OED to reload.
