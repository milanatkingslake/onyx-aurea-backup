retrieveListHistory (customer)

retrieveListHistory method

The retrieveListHistory method of the customer object retrieves a list, sorted by last update date, of recent customer activity consolidated from: Attachments, Campaigns, Comments, Incidents, Literature Orders, Products, Surveys, Tasks, Scripts, and Work Tickets data. The data is returned as a rowSet object in the XML. The rowSet object has the same overall structure for all activities, but certain column definitions change depending upon which activity's data is returned. See the Remarks section.

Remarks

The returned rowSet object contains the columnDefinition for each of the columns and a collection of customer activity objects that satisfy the search criteria. The returned rows are sorted by update date not by object type. Customers that are either marked as private or as deleted are not returned.
