emailAddress object

The emailAddress object contains all the information about primary and alternate emailAddresss that are associated with a customer record. emailAddress objects may be saved with the customer or may be saved separately. The email addresses collection of the company or individual objects contains emailAddress objects.

The emailAddress object has one key property, primaryId, that uniquely identifies the emailAddress in the database. Once the object is created, the value of this property cannot be changed.
