retrieveList (region)

retrieveList method

The retrieveList method of the region object searches the database and returns the complete list of regions for a specific country.

retrieveList standard method information

Parameters

Parameter

Direction

Data type

Content type

Description

countryCode (required)

in

string

n/a

Contains the country code for the given list of regions.

rowSet

out

object

all(out)

A list of rows that contain data about the regions that belong to the given country.

Columns returned in the rowSet object

columnOrdinal

columnName

region_ml_view.region_code
