actionBatchXml_sourceListMap_targetElementPath
sourceListMap/targetElementPath

The sourceListMap/targetElementPath actionBatchXml element specifies the path of the data target node in methodInputs into which the source list value pointed to by the sourceElementPath is copied. The path is used to select the target node from the methodInputs node. This path must be relative to the methodInputs node.

Note: targetElementPath takes precedence over targetCollection if both are present.

Cardinality

0..1 (optional)

Default value

none
