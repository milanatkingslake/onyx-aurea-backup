update (scriptSession)

update method

The update method of the scriptSession object saves modifications to an existing scriptSession object.

update standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Updates the object instance

Example

The following example uses the update method to change the statusId from 'new' (1) to 'inprogress' (2).

Parameters

<parameters>

   <scriptSession objectType="scriptSession" content="any">

      <primaryId>88515B97-9961-4DD9-8104-18F8B4FD7F7A</primaryId>

      <scriptId>100000</scriptId>

      <scriptName>Insurance</scriptName>

      <ownerId>A9635BEF-1F4B-4C1A-89E4-00B1440BAABA</ownerId>

      <ownerType>1</ownerType>

      <ownerSecondaryId>1</ownerSecondaryId>

      <contactId null="1"/>

      <contactObjectType null="1"/>

      <duration null="1"/>

      <statusId>2</statusId>

      <assignedTo>sa</assignedTo>

      <stateXml null="1"/>

      <insertBy>sa</insertBy>

      <insertDate>2003-05-15 07:30:45</insertDate>

      <updateBy>sa</updateBy>

      <updateDate>2003-05-15 07:30:45</updateDate>

   </scriptSession>

</parameters>
