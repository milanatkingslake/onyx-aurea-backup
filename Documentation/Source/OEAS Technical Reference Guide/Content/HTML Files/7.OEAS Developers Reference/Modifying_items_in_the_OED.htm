Modifying items in the OED

To modify an item:

  1. In the navigation area, locate and select the item that you want to modify.

    The attributes for this item display in the edit area.

  2. Apply your modifications to the object attributes or physical model attributes.
  3. Once you modify an attribute, the item, as well as its parent object, appears in the navigation area with an asterisk ( * ) until the changes are saved. If you disabled the item, it additionally appears italicized and grayed out, but the asterisk disappears once the changes are saved.
  4. Click to save your changes.
