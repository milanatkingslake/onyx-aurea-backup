Steps (Modify Method)
Method modification steps

This sample demonstrates how to add a custom step to an existing method. The following steps will demonstrate this sample.

  1. A custom COM component that implements the IStep interface is created using Microsoft Visual C#.Net.
  2. Code is added to the step component that returns data for a property.
  3. The step component is tested standalone to verify that it is operating properly.
  4. The step component is installed on the server.
  5. The new property is added to the company business object and the company retrieve method is modified, adding a step that calls the custom step component.
  6. OEAS is restarted to load the modifications to the customer retrieve method.
