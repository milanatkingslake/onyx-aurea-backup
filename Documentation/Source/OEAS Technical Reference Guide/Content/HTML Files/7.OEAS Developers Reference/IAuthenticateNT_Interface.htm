IAuthenticateNT interface

Use the AuthenticateNT component to verify users and establish sessions when the user has a local network account and can be mapped to an Onyx user. The AuthenticateNT component implements a single interface, IAuthenticateNT. The method of the this interface is listed below:

Method

Description

EstablishSession

Authenticates a user, maps a local network user to an OEAS user, and initiates a session
