ofMatchCodeCompanyNameParse

The ofMatchCodeCompanyNameParse function parses the company name to return the portion that is used in the match code. The opMatchCodeCompanyInsert and opMatchCodeCompanyUpdate stored procedures call it when generating a match code for a company.

Input Parameters

The following table describes the input parameters for ofMatchCodeCompanyNameParse.

Column

Parameter Name

Parameter Description

Data Type

Default Values and Comments

1

@company_name

The name of the company

NVARCHAR(255)

 

Return Value

  • The company name portion of a company’s match code

Remarks

Company name match code examples:

  • Onyx Software Corporation returns a value of “Onyx SSC”
  • Microsoft Corporation returns a value of “MicrosC*”

For more information, see the comments in the header of the function code.
