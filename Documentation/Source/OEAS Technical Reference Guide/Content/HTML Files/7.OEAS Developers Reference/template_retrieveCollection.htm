retrieveCollection (template)

retrieveCollection method

The retrieveCollection method of the template object returns a collection of objects based on the search criteria supplied in the parameters XML.

retrieveCollection standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validate input parameters

OnyxDM.DataManager

com
