governorIntervalMilliseconds (subsystem)

governorIntervalMilliseconds property

The governorIntervalMilliseconds property of the subsystem object contains the time in milliseconds between calls to Onyx Transaction Manager (OTM) for list management processing.

Data type

i4

Access

Read/write

Remarks

The governorIntervalMilliseconds property acts as a throttling mechanism to prevent the Onyx Processing Server from flooding the Onyx Enterprise Application Server with calls to the OTM. This property is used by the actionBatch subsystems that process groups of method calls for any business object.
