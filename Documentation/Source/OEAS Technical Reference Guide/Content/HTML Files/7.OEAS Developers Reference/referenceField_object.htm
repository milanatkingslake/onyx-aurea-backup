referenceField object

A referenceField object defines a reference field and the way in which it displays on a UI, including its internal name, the caption that displays on the UI, and whether the reference field is a text box, a check box, or a combo box.

If a referenceField object has a type property of "combo" or "hiercombo", it is associated with one or more referenceParameter objects via a referenceLookup object. The referenceLookup object, in essence, associates a referenceField object with the referenceParameter objects that display as possible values for the specified reference field.

The referenceField object has two key properties (primaryId and name) that uniquely identify the referenceField data in the database.
