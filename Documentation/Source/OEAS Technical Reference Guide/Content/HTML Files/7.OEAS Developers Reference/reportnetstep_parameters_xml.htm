parameters XML

The ReportNetStep component uses the input parameters to form a call to the Cognos server using the Cognos SDK. The step component does not change or filter the parameter values in any way. For retrieveFolderItems, the searchPathList can contain multiple values delimited by colons. When the searchPathList contains multiple values, the component creates a separate call to the Cognos server for each path value.

Example

createFolder

<parameters>

   <onyxNamespace/>

   <folderPath/>

   <folderName/>

   <reportNetServer/>

</parameters>

retrieveFolderItems

<parameters>

   <onyxNamespace/>

   <searchPathList/>

   <reportNetServer/>

</parameters>
