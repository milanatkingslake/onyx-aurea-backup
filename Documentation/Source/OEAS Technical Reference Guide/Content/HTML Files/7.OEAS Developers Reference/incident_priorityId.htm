priorityId (incident)

priorityId property

The priorityId property of the incident object contains the priority assigned to the incident.

Data type

i4

Access

Read/write

Remarks

The priorityId property references an existing priority type in the database. Incident categories own priority types, therefore the value in the priorityId property must reference a child value of the incidentCategory for a given incident.

For example, if the incidentCategory is Sales (categoryId), your priority types (priorityId) might be any of the following: Immediate, High, Medium, Low. These are default priority types. You can add your own using OES Reference Table Administration.
