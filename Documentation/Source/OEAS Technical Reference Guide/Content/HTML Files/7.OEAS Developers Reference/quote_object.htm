quote object

quote object

The Forecast/Quotes feature's quote object enables users to delete, insert, retrieve, and update quotes associated with incidents, usually sales incidents.

The quote object has one key property, primaryId, that uniquely identifies the quote in the database. Once the object is created, the value of this property cannot be changed.

Methods

Method

Description

delete

Deletes a specified quote object instance

insert

Persists an object to the database

retrieve

Returns a collection of objects based on the search criteria supplied in the parameters XML

retrieveList

Returns a list of quotes matching the specified search criteria

update

Persists modifications to an existing object instance
