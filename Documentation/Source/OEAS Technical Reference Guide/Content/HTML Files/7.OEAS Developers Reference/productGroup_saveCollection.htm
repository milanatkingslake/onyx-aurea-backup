saveCollection (productGroup)

saveCollection method

The saveCollection method of the productGroup object performs multiple actions at once: inserting, updating, and deleting objects, as appropriate.

saveCollection standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters.

OnyxDM.DataManager

com

Saves a collection of productGroup object instances.

For insert actions, stores the 1 value in the rank column of the product_group table for the record inserted for a productGroup object instance. This action is based on a literal logical-to-physical map, which specifies a non-null column that has no corresponding property on the logical object.
