parametersReturn XML (OnyxCollectionManager)

parametersReturn XML

The OnyxCollectionManager step component sets the overall status of the method based on the following conditions.

Condition Overall method status

If any one method fails

failure

If any one call returns successWithInfo and no calls return failure

successWithInfo

If all calls return success

success

Example

OnyxCollectionManager assembles any errors in the returnXML. The associationId attribute of the error node identifies each object instance action listed in the parametersReturn node that failed.
