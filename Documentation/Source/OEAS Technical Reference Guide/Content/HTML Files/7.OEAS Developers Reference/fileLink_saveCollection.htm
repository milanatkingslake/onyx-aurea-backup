saveCollection (fileLink)

saveCollection method

The saveCollection method of the fileLink object performs multiple actions at once: inserting, updating, and deleting object instances, as appropriate.

saveCollection method standard behavior

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Inserts, updates, and deletes a collection of object instances
