context element

The context element contains information that identifies the context from which the client application is calling.

Client applications that use the ITransactionManager interface submit the session ID that is obtained from an authentication component. For more information on authentication components, see Authentication overview. The Session Manager passes the OTM the context information, based on the user's session ID, and adds it to the step package.

Custom step components that call the IInternalTransactionManager interface must build the contents of the context element manually. In most cases, your step component can copy the data from from the step package that it receives from the OTM.

The elements of the context element are:

Element Description

applicationName

The name of the OEAS application.

logName

The name of the Onyx log to use when logging messages.

logSettings

An integer value that identifies the current level of logging.

originalUser

The Windows 2000 domain account that initiated the call to the OTM.

otmId

A GUID that is used to identify the current method call. This is useful when tracking a method call in the application log.

permissionMask

A bitmask that contains information about the resources this user is allowed to access.

preferredLanguage

A code that identifies the national language table used by the client application.

sessionId

A GUID that identifies the current session. The client gets this value from an authentication component.

sessionSecondsRemaining

An integer value representing the number of seconds remaining in the client application's session for the user as determined by OEAS using the GlobalSessionTimeOut system parameter.

siteId

The site ID value.

source

A code that identifies the client application that called the OTM.

userId

The username of the client calling the OTM.

Example

The contents of a sample context element look like this:

   <context>

      <sessionId>343B353F-87AD-4BCA-94E1-8CC6DC1EA9AE</sessionId>

      <userId>rogerw</userId>

      <partnerUser>0</partnerUser>

      <siteId>1</siteId>

      <applicationName>Onyx</applicationName>

      <source>OEP</source>

      <preferredLanguage>ENG</preferredLanguage>

      <originalUser>ISSAQUAH\RogerW</originalUser>

      <permissionMask>fffffffffffff...0000000000000000000</permissionMask><!-- abbreviated: not a valid mask -->

      <logName>Onyx</logName>

      <logSettings>7</logSettings>

      <impersonatingUser/>

      <sessionSecondsRemaining>600</sessionSecondsRemaining>

      <otmId>075DEE49-4DAD-426C-9475-BE2B59F80402</otmId>

   </context>
