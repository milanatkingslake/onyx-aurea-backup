Using sample fax templates

Onyx Processing Server (OPS) includes sample fax templates that you can use as is or modify for your customization. The OPS setup installs the templates on the OPS server with the file names given in the following table. The fax engine uses XSL stylesheets to transform the input orderBatch XML into an SMTP message that it sends to a Fax server which converts it into a fax. OPS is not dependent on any particular Fax software. Using XSL templates, you can configure the SMTP to suit any Fax server that supports the protocol. For more information on SMTP message format required by the Fax server, consult the Fax server documentation.

By default, OPS Setup places the following templates in C:\Program Files\Onyx\ProcessingServer\Document Templates.
