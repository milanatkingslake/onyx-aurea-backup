gridHeadingStart (taskCategory)

gridHeadingStart property

The gridHeadingStart property of the taskCategory object contains the national language ID corresponding to the first grid heading returned by a task retrieval of this type of task category.

Data type

i4

Access

Read/write

Remarks

This property is used only by the retrieveListByOwner method.

By default, the gridHeadingStart stores NationalLanguage IDs shipped with OEAS.
