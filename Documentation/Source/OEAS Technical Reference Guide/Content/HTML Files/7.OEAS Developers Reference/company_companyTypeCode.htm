companyTypeCode (company)

companyTypeCode property

The companyTypeCode property of the company object identifies the type or category to which a company belongs.

Data type

i4

Access

Read/write

Remarks

The companyTypeCode property is a unique ID for the type or category to which a company belongs and can be configured using the Reference Table administration tool. An example of a companyTypeCode might be "102", which could have a description of "Lead" in the referenceParameters object. There is a parent child relationship between a companyTypeCode and a companySubTypeCode.
