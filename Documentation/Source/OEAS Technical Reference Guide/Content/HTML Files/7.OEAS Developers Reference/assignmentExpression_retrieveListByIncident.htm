retrieveListByIncident (assignmentExpression)

retrieveListByIncident method

The retrieveListByIncident method of the assignmentExpression object returns a list of assignmentExpression records that match the supplied input criteria.

Parameters

Parameter Direction Data type Content type Description
ownerId in string n/a Incident owner unique identifier
ownerType in i4 n/a Incident owner type
categoryId in i4 n/a Incident category
typeId in i4 n/a Incident type
trackingId in i4 n/a Incident tracking identifier
trackingCode in string n/a Incident tracking code
sourceId in i4 n/a Incident source identifier
statusId in i4 n/a Incident status identifier
priorityId in i4 n/a Incident priority
productId in string n/a Incident product identifier
assignedTo in string n/a Incident assigned to username
rowSet out object all Contains rowSet column definitions and the assignmentExpression records in the database that match the specified search criteria

10 conditionPriority assignment_condition_ml_view.sequence_number
