retrieveListAssociation (emailMessage)

retrieveListAssociation method

The retrieveListAssociation method of the emailMessage object returns information about the object instances that are associated with a specified emailMessage instance.

retrieveList standard method information

Parameters

emailMessageId in string — The primary key of the emailMessageId associated with to the objects being returned by this method
rowSet out object all(out) Contains rowSet column definitions for the rows associated to the emailMessage

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieves a list of objects associated with an emailMessage

Remarks

An email message that is marked private is not included in the result set unless the current user is the person who set the record to private.
