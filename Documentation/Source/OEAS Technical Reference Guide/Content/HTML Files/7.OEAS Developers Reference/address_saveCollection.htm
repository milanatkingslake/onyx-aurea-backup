saveCollection (address)

saveCollection method

The saveCollection method of the address object performs multiple actions at once: inserting, updating, and deleting objects, as appropriate.

saveCollection standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxNotification.LBOAdapter com This step is added for notification feature. It captures data before updating the object
OnyxDM.DataManager com

Perform multiple actions with address objects

Calls a post-procedure to check for two or more addresses a) marked as primary and having same owner or b) with same address type and same owner; if true, returns an error message and performs a rollback of all actions
OnyxNotification.LBOAdapter com This step is added for notification feature. It captures data after saving the object
