connectorProgId (subscriptionService)

connectorProgId property

The connectorProgId property of the subscriptionService object contains the programmatic identifier of the COM component that connects the synchronization server to the staging server.

Data type

string

Access

Read/write

Remarks

The default implementation of OEP Offline requires an Exchange server as the staging server, and the programmatic identifier of the corresponding connector is OnyxPSExchangeSyncConnector.SyncConnector.
