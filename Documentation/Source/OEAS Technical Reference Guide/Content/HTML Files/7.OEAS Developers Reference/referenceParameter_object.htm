referenceParameter object

A referenceParameter object defines a single value that can be associated by one or more referenceField objects.

A referenceLookup object creates a many-to-many relationship between referenceField objects and referenceParameter objects.

The referenceParameter object has one key property, parameterId, that uniquely identifies the value in the database. Once the object is created, the value of this property cannot be changed.
