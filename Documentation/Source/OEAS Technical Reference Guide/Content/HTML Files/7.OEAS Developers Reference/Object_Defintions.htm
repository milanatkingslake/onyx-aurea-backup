Object Definitions
Object definitions

Object definitions are sets of XML elements that define a business object. Each business object implemented by OEAS, such as customer, incident, and contacts, is represented in the OED by an object definition. Object definitions contain detailed information about the business object properties and methods including the data types of the properties, any constraints placed on a particular property (property constraints), and descriptions for each of the elements.

Step components and client applications can request an object definition to determine information such as the data types of properties and the constraints on a property.

To obtain an object's definition or information about an object's parameters, list of methods, or any other information not available with the GetPropertyConstraints method, call the GetObjectDefinition method.
