retrieveCollection (country)

retrieveCollection method

The retrieveCollection method of the country object returns a collection of country objects based on the search criteria.

retrieveCollection standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieves a collection of country object instances

Example

The following example shows the input parameters and output parametersReturn for the retrieveCollection method. Notice the pageSize attribute on the countries collection that limits the maximum size of the returned collection to three objects. The example also illustrates the successWithInfo status when you limit the returned collection.
