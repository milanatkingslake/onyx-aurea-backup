retrieveCollection (surveyType)

retrieveCollection method

The retrieveCollection method of the surveyType object returns a collection of objects based on the search criteria supplied in the parameters XML.

retrieveCollection standard method information

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Retrieves a collection of object instances

Example

Input Parameter:

<parameters>

   <surveyTypes objectType="surveyType" collection="1">

      <surveyType objectType="surveyType">

         <primaryId></primaryId>

         <description></description>

         <insertBy></insertBy>

         <insertDate></insertDate>

         <updateBy></updateBy>

         <updateDate></updateDate>

         <active></active>

      </surveyType>

   </surveyTypes>

</parameters>
