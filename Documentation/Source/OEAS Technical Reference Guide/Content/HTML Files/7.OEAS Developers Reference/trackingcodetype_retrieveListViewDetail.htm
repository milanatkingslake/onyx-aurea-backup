retrieveListViewDetail method

retrieveListViewDetail method

The retrieveListViewDetail method of the trackingCodeType object returns the list of tracking code types to display in the List Manager.

retrieveList standard method information

Parameters

Parameter Direction Data type Content type Description

customerType

In

i4

n/a

The customerType is 1 for the individual or 2 for the company

campaignCode

in

string

n/a

The campaign in which trackingCodeTypes are returned

trackingCodeType

in

i4

n/a

The trackingCodeType in which trackingCodes are returned

rowSet

out

object

all

Contains the rowSet column definitions and the incident table rows matching with the owner and category criteria
