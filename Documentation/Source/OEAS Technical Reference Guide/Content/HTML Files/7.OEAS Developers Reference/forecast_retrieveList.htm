retrieveList (forecast)

retrieveList method

The retrieveList method of the forecast object searches the database and returns a rowSet containing the list of forecast items that match the input criteria.

retrieveList standard method information

Parameters

Parameter Direction Data type Content type Description
ownerId (required) in string n/a The unique identifier for the object that owns the forecast
ownerType (required) in i4 n/a Specifies the type of object for the given ownerId
rowSet out object all(out) List of rows containing forecast information

Columns returned in the rowSet object

columnOrdinal columnName Physical column
1 primaryId forecast_header.forecast_header_id
2 secondaryId forecast_header.secondary_id
3 vchName forecast_header.name

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieves each forecastLine of detail from the database
