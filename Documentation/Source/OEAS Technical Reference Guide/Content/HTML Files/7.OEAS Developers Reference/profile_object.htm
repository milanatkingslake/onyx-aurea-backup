profile object

The profile object identifies a set of UI Configuration Workbench (UCW) data that can be associated with OEP users and roles.

Methods

Method

Description

retrieveCollection

Returns objects that match search criteria

saveCollection

Performs multiple actions at once: inserting, updating, and deleting objects, as appropriate

Properties

Property

Description

active

Specifies whether the profile is available for association with users and roles

description

Text that describes the profile

insertBy

The user ID of the Onyx user who initially saves the object

insertDate

The time and date on which the object is initially saved

name

The name of the profile

primaryId

The unique ID assigned to the object when it is first saved to the database

updateBy

The user ID of the Onyx user who last saved modifications to the object

updateDate

The time and date on which modifications to the object were last saved
