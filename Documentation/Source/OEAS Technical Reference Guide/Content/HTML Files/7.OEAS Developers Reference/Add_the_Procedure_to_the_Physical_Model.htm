Step 2: Add the procedure to the physical model

In order for the OEAS to generate the correct Query Engine call for your custom stored procedure, you must give it a definition in the physical model section of the Object Designer. Use the following steps to create the physical model.

  1. Open Onyx Enterprise Studio Object Designer.
  2. Navigate to Physical Model\Procedures and select Procedures.
  3. Click the Add button to add a new procedure to the physical model.
  4. In the Name field enter the name of the new procedure, in this case cpKeyWordRetrieveListByUser.
  5. Click the box to expand the new procedure. This exposes the Parameters node. Select the Parameters node and click the Add button to add a new parameter. Enter the name, data type, direction, default value, and size for each parameter in the appropriate text box or drop-down list box.
