alertId (individual)

alertId property

The alertId property of the individual object identifies the numerical ID of the alert record that is associated with an individual record.

Data type

string (maximum length)

Access

Read-only; for more information, see Read Only attribute

Remarks

The numerical ID is the foreign key value that represents the association between the individual table and the alert table.

The alertId property replaces the alertNumber property, which was used in OEAS prior to version 4.0.
