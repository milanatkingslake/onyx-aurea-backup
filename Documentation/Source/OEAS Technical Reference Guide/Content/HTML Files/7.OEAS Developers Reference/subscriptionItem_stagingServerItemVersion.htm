stagingServerItemVersion (subscriptionItem)

stagingServerItemVersion property

The stagingServerItemVersion property of the subscriptionItem object contains a value that identifies the version of a subscribed object while it is offline.

Data type

string

Access

Read/write

Remarks

The synchronization server uses the value of this property to determine whether the OEDB contains the latest version of the object instance. When the offline client has the latest version, an upwards synchronization with the OEDB occurs.
