retrieveListByIdList (customer)

retrieveListByIdList method

The retrieveListByIdList method of customer object returns the customer data for the list of customer IDs.

Parameters

Parameter Direction Data type Content type Description

customerIdList

in

string

n/a

A list of customer primaryId values

customerType

in

i4

n/a

Type of customer: 1=individual;2=company;9=customer

includeMask

in

string

n/a

Identifies whether the phone number mask is displayed on the output or not

rowSet

out

object

all

Contains the rowSet column definitions and the incident table rows matching with the owner and category criteria

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates the input parameters

OnyxDM.DataManager

com

Retrieves the information for a customer including customer type

<customerType>2</customerType>
