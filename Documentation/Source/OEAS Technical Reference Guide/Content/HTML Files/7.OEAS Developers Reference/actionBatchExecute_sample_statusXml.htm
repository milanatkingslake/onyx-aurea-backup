(actionBatchExecute) sample statusXml

Receiving returned statusXML

After executing all actions in the batch, all the parameters and returnXml for all the executed methods are collected and returned in the statusXml out parameter. The statusXml output parameter contains an XML document as text, which needs to be loaded into a separate DOM for processing. This is demonstrated in actionBatch tester.js script.

The following statusXml sample is the actual returnXml from an actionBatch.execute call using the sample actionBatchXml.
