retrieveListByIndividual (assignmentExpression)

retrieveListByIndividual method

The retrieveListByIndividual method of the assignmentExpression object returns a list of assignmentExpression records that match the supplied input criteria.

Parameters

Parameter Direction Data type Content type Description
userTypeId in i4 n/a User type identifier
userSubTypeId in i4 n/a User subtype identifier
companyId in uuid n/a Owner company identifier
sourceId in i4 n/a User record creation source
statusId in i4 n/a User record status
phoneNumber in string n/a User's phone number
city in string n/a User's city
regionCode in string n/a User's region
countryCode in string n/a User's country
postCode in string n/a User's postal code
rowSet out object all Contains rowSet column definitions and the assignmentExpression records in the database that match the specified search criteria

Columns returned in the rowSet object

columnOrdinal columnName Physical column
1 expressionId assignment_expression.assignment_expression_did
2 conditionId assignment_expression.assignment_condition_did

5 propertyValue2 assignment_expression.property_value2
6 operator assignment_expression.operator
7 conditionName assignment_condition_ml_view.assignment_condition_name
8 conditionDescription assignment_condition_ml_view.assignment_condition_desc
9 territoryId assignment_condition_ml_view.assignment_territory_did
10 conditionPriority assignment_condition_ml_view.sequence_number
