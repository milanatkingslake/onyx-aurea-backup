retrieveList (workTicketAuditLog)

retrieveList method

The retrieveList method of the workTicketAuditLog object returns all workTicketAuditLog object instances owned by a given work ticket.

retrieveList standard method information

Parameters

This method requires one input parameter, and returns a rowSet object as an output parameter. You pass the input XML within the root element named <parameters>.
