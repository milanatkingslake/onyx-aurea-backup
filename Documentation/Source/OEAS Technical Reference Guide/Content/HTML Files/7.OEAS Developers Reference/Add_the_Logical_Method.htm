Step 3: Add the logical method

In order for the OEAS to find the new method of the keyWord object, you need to add a new method definition in the Logical Model section of the Object Designer.

  1. Open Onyx Enterprise Studio Object Designer.
  2. Navigate to Logical Business Objects\keyWord\Methods and select Methods.
  3. Click the Add button to add a new method. Enter the name, description, method type, and installed by information for the new method as described in the following table.
