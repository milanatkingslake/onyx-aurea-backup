evaluateIndividual (assignmentAssociation)

evaluateIndividual method

The evaluateIndividual method of the assignmentAssociation object causes the assignment of the individual record to be evaluated for territory assignment. This method can be called to allow for user-initiated reassignments of individual records.

Parameter

Parameter Direction Data type Content type Description
individual in object keysOnly A company object with a valid primaryId property

Step

The default step for the evaluateIndividual method of the assignmentAssociation object is:

Step data Step type Description
OnyxBPExecuteStep.BPExecuteStep business process Launches the business process that evaluates the individual record for assignment to a territory
