productGroup object

The productGroup object provides hierarchical information that organizes product and literatureProduct objects and enables users to view and make changes to the product group hierarchy.

The productGroup object has one key property, hierarchyId, that uniquely identifies the structure of product groups and relationships of products and product groups in the database. Once the object is created, the value of this property cannot be changed.

Methods

Method

Description

retrieve

Searches the database by hierarchyId and returns a productGroup record

retrieveCollection

Returns a collection of product groups

saveCollection

Processes a group of productGroup object instances for multiple inserts, updates, and deletes using a single call

Properties

Property

Description
