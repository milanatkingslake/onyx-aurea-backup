Column attributes

When you select a column in the Object Designer, the attributes for that column display in the edit area.

The attributes are defined as follows:

Attribute Name

Description

ID

The globally unique identifier (GUID) assigned by the Onyx Enterprise Dictionary. This attribute is read-only.

Name

Type the name you want to assign to this column.

Description

Type the description for this column.

Column Type

Select the data type for this column from the list.

Length

Type the length, in bytes, of the column. For Unicode characters, specify three bytes per character. For example, to accommodate a string with a maximum of 85 Unicode characters, specify a length of 255.

Not Null

Check this box if NULL is not allowed in this column. This indicates that a value is required in the column for each row.

Installed By

Type the name of the vendor or application adding this column.
