Steps (Modify Property)
Property modification steps

This sample demonstrates how to change the email address of a company business object from an optional to a required property.

  1. Open Onyx Enterprise Studio Object Designer.
  2. Navigate to Logical Business Objects\company\properties\emailAddress\Property Constraints\required.
  3. Set the Type selection to Required.
  4. Check the Value box to enable the Required constraint.
  5. Click the Save button to save your changes.
  6. Optionally, you can click the Validate button to verify that the modifications you made are acceptable. When you publish, the validation is also performed, so this is an optional step.
  7. Click the Publish button. Answer Yes to the prompt that asks if you want to overwrite the current OED.
  8. Shut down the Onyx App COM+ to cause the OED to reload.
