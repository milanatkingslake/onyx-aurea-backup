allDay (appointment)

allDay property

The allDay property of the appointment object indicates whether the appointment is scheduled to take place for the entire scheduled day.

Data type

boolean

Access

Read/write

Remarks

When set to true, the allDay property affects the startDate and endDate properties of an appointment object as follows: Outlook sets the start to 12:00am of the starting day and sets the end to 12:00am of the day after the ending day. For example, with an all day event for the 15th the startDate property is set to 12:00 am on the 15th and the endDate property is set to 12:00 am on the 16th.
