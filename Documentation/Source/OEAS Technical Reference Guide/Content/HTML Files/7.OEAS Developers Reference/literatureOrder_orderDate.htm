orderDate (literatureOrder)

orderDate property

The orderDate property of the literatureOrder object contains the date the order was placed.

Data type

dateTime

Access

Read/write

Remarks

This value cannot be changed after the literature order has been created.

The dateTime data type is formatted as follows:

YYYY-MM-DD hh:mm:ss

where:

YYYY = year

MM = month

DD = day

hh = hour

mm = minute

ss = second

Valid values for the dateTime data type range from 1900-01-01 00:00:00 to 9999-12-30 23:59:59.
