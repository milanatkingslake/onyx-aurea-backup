retrieve (group)

retrieve method

The retrieve method of the group object searches the database by groupId and returns a group object instance.

retrieve standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieves a specific group based on the groupId

   <group objectType="group">

      <groupId>R&amp;DV</groupId>

    </group>
