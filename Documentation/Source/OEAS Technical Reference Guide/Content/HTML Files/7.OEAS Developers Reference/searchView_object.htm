searchView Object

searchView object

The searchView object is a domain object representing a view against which queries can be built in List Manager.

The searchView object has one key property, primaryId, that uniquely identifies the searchView in the database. Once the object is created, the value of this property cannot be changed.

Methods

Method

Description

retrieveCollection

Returns a collection of objects based on the search criteria supplied in the parameters XML

Performs multiple actions at once: inserting, updating, and deleting objects, as appropriate
