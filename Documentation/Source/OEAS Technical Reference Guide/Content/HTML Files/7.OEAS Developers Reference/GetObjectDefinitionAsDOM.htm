GetObjectDefinitionAsDOM (ICache2)
GetObjectDefinitionAsDOM method

The GetObjectDefinitionAsDOM method of the ICache2 interface retrieves an XML DOM pointer to the object definition of a business object.

Syntax

IDL

HRESULT GetObjectDefinitionAsDOM(

    [in] BSTR ApplicationName, 

    [in] BSTR ObjectName, 

    [out] VARIANT* ObjectDefinition, 

    [out, retval] OEDCacheStatusEnum* Status);

IDL parameter descriptions

ApplicationName

Input. A string containing the logical application name that determines which subset of the cache to use for a query.

ObjectName

Input. A string that contains the name of the specified object.

ObjectDefinition

Output. A variant that contains an XML DOM pointer to the business object definition.

Status

Output return value. An enumeration of type OEDCacheStatusEnum that contains the status of the method call.

Visual Basic

Function GetObjectDefinitionAsDOM(

    ApplicationName As String, 

    ObjectName As String, 

    ObjectDefinition As Variant)

    As OEDCacheStatusEnum

Visual Basic parameter descriptions

ApplicationName

Input. A string containing the logical application name that determines which subset of the cache to use for a query.

ObjectName

Input. A string that contains the name of the specified object.

ObjectDefinition

Output. A variant of type VT_UNKNOWN that contains an XML DOM pointer to the business object definition.

Status

Output return value. An enumeration of type OEDCacheStatusEnum that contains the status of the method call.

Example

The following Visual Basic code demonstrates how to use the GetObjectDefinitionAsDOM method to return an incident object definition:

Dim oCache As New OnyxDictionaryCacheLib.DictionaryCache

Dim status As OnyxDictionaryCacheLib.OEDCacheStatusEnum

Dim ptrObjectDefinition As Variant

Set oCache = New OnyxDictionaryCacheLib.DictionaryCache 

status = oCache.GetObjectInstanceAsDOM("Onyx", "incident", ptrObjectDefinition)
