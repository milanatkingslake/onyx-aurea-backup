retrieveCollection (workNoteHeader)

retrieveCollection method

The retrieveCollection method of the workNoteHeader object returns a collection of workNoteHeader object instances that match search criteria supplied in the parameters XML.

retrieveCollection standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Retrieves a collection of workNoteHeader objects instances

OnyxWorkNoteHeaderStep.WorkNoteHeader

com

Retrieves a collection of workNoteDetail object instances owned by the workNoteHeader retrieved in the previous step

Remarks

OEAS only supports a single workNoteHeader object per parent object. If retrieveCollection returns more than one workNoteHeader, the system uses the primaryId of the first workNoteHeader returned.
