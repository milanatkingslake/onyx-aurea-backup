The text attribute of the list item element contains the value specified by the descriptionPath property of the methodCallDetail object.

XPath

/root/variables/variable/item/@text

Cardinality

1 (required)

Default value

none

Comments

Because this represents a list variable, you can only use this as the input for a list control.

For more information on using this attribute, see the Variables topic.
