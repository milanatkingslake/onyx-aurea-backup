Session Management Overview

Session management

The Session Manager component provides the following services for OEAS security:

  • Session creation
  • Session state
  • Session lifetime
  • Impersonation
  • Permissions checking
  • License checking

The Session Manager maintains information about each user session, such as a the time that the user last accessed OEAS. This time and a time-out system parameter determine when OEAS ends a session because of user inactivity.
