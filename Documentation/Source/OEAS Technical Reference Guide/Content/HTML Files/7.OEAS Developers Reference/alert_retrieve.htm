retrieve (alert)

retrieve method

The retrieve method of the alert object searches the database by primaryId and returns an alert object instance.

retrieve standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com

Retrieves a specific alert based on the primaryId

Uses database function mapped to expired property to return expired value based on expirationDate value
