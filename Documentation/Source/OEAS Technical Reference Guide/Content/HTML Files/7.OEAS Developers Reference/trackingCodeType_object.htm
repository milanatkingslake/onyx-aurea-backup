trackingCodeType Object

trackingCodeType object

The trackingCodeType object provides a way of grouping related tracking codes. With tracking code types, you can logically organize the tracking codes used in a campaign. A trackingCodeType object can be assigned to any number of trackingCode objects.

Methods

Method Description

retrieveCollection

Returns the collection of trackingCodeType objects

retrieveListByCampaign

Returns the list of tracking code types where at least one tracking code of the type exists on a specified campaign

retrieveListViewDetail

Returns the list of tracking code types to display in the List Manager

saveCollection

Performs multiple actions on a collection of trackingCode objects

Properties

Property Description

active

Indicates the status of a trackingCodeType

description

A text summary of the trackingCodeType

name

A name for display in the UI

primaryId

The unique identifier that OEAS assigns to the object
