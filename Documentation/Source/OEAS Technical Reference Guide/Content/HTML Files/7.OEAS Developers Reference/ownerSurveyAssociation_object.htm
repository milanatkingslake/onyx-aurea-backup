ownerSurveyAssociation Object

ownerSurveyAssociation object

The ownerSurveyAssociation object associates a survey with a customer. The ownerSurveyAssociation object is a transaction object that represents an instance of the survey given to a customer. You can give the same survey to a customer multiple times or give the same survey to multiple customers. Each time you give a survey to a customer, you create a new ownerSurveyAssociation object.
