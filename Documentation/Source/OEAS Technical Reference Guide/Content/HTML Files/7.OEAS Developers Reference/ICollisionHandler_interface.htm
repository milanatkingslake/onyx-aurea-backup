ICollisionHandler interface

The ICollisionHandler interface is called by both the OPS synchronization service and the OEP-Offline client.

The OPS synchronization subsystem calls a component that implements ICollisionHandler to resolve data collisions that occur during the upsync from the Exchange server to the OEDB.

The OEP-Offline client (specifically, the Onyx Data Connector) calls a component that implements ICollisionHandler to resolve data collisions for records that cannot upsync from the Outlook data store to the Exchange server.

This interface exposes a single method:

Method

Description

Resolve

Attempts to resolve a data collision.
