profileUser object

The profileUser object contains associates an Onyx user with an OEP profile.

Methods

Method

Description

retrieveCollection

Returns objects that match search criteria

saveCollection

Performs multiple actions at once: inserting, updating, and deleting objects, as appropriate

Properties

Property

Description

insertBy

The user ID of the Onyx user who initially saves the object

insertDate

The time and date on which the object is initially saved

profileId

The ID of an OEP profile

updateBy

The user ID of the Onyx user who last saved modifications to the object

updateDate

The time and date on which modifications to the object were last saved

userId

The ID of an Onyx user
