saveCollection (fulfillmentType)

saveCollection method

The saveCollection method of the fulfillmentType object performs multiple actions at once: the insert, update, and delete of fulfillmentType objects, as appropriate.

saveCollection standard method information

Steps

OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Inserts, updates, and deletes a collection of object instances
