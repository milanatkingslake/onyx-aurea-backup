totalTime (incident)

totalTime property

The totalTime property of the incident object contains the elapsed time, in minutes, that an incident is open (and not paused) in the user interface.

Data type

i4

Access

Read/write

Remarks

The client user interface application must implement the time tracking function. OEAS only stores the value calculated by the client application.

The totalTime property is an addend in the sum that comprises the totalLabor property.
