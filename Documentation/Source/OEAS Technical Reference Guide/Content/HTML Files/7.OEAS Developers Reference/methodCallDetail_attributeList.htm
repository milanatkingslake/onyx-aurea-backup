attributeList (methodCallDetail)

attributeList property

The attributeList property of the methodCallDetail object contains a semi-colon delimited list of name=value pairs representing attributes of the encapsulated element.

Data type

string

Access

Read/write

Remarks

The name=value pair corresponds to the local-name of an attribute node and its text node, respectively. The script engine or OnyxMethodCallExecute step components apply this attribute to the element described by the methodCallDetail object. The following table shows an example of an attributeList specification and the resulting element with the applied attributes.
