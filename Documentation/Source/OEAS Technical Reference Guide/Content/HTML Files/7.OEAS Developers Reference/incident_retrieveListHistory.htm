retrieveListHistory (incident)

retrieveListHistory method

The retrieveListHistory method of the incident object retrieves a list, sorted by last update date, of recent incident activity consolidated from: Attachments, Forecasts, Quotes, Tasks, linked Work Notes, and Work Tickets data. The data is returned as a rowSet object in the XML. The rowSet object has the same overall structure for all activities, but certain column definitions change depending upon which activity's data is returned.

Remarks

The returned rowSet object contains the columnDefinition for each of the columns and a collection of customer incident activity objects that satisfy the search criteria. The returned rows are sorted by update date not by object type. Customers that are either marked as private or as deleted are not returned.
