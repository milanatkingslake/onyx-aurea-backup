update (incident)

update method

The update method of the incident object persists modifications to an existing object instance.

update standard method information

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

notificationAdapter.preUpdateEvent

com

This step is added for notification feature. It captures data before updating the object

OnyxGuidGeneratorStep.GuidGenerator

com

Calls a component that generates GUID primary key values used for inserting the tags identified in the incident's keyWords property

OnyxDM.DataManager

com

 Updates the specified incident object and calls to two post-procedures
  • Insert or update keyWords
  • Copy the incident instance into a record in the incident audit log table
notificationAdapter.postUpdateEvent com This step is added for notification feature. It captures data after saving the object
