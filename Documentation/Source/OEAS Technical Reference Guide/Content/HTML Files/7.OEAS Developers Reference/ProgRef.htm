Programmer's Reference

Core component library

This book contains the topics that explain how to use the COM interfaces to the OEAS core components. The interfaces are:

Interface

Description

DictionaryProxy

Provides out-of-process client applications access to object metadata information.

IAuthenticateNT

Authenticates users through integrated Windows authentication.

IAuthenticateOnyx

Authenticates users based on an OEAS user account.

ICache

Provides access to the Onyx Enterprise Dictionary run-time cache.

ICache2

Inherits ICache and adds GetObjectDefinitionAsDOM method.

ICache3

Inherits ICache2 and adds GetObjectList method.

IEBusQE

Provides access to the Onyx Enterprise Database through the Query Engine.

IInternalStringTransactionManager

Provides a mechanism for .NET step components to initiate business object method calls.

IInternalTransactionManager

Allows step components to call methods of OEAS business objects.

ISession

Manages client session information.

ISession2

Inherits ISession and adds GetContextXML2 method to support session time outs.

IStep

Used in the creation of custom step components.

IStepString

Provides a mechanism for step components created using the Microsoft .NET Framework to enable a call from the Onyx Transaction Manager.

ITransactionManager

Acts as the primary entry point to the OEAS for client applications.

ITransactionManager2

Inherits ITransactionManager and adds Execute2 method to support session time outs.

INatLangCache

Provides access to resource strings available in the OnyxNatLangCache.

NatLangProxy

Provides out-of-process client applications access to resource strings available in the OnyxNatLangCache.
