resource object

resource object

The resource object enables users to delete, insert, retrieve, and update resources.

Methods

Method

Description

retrieveList

Returns a list of resources in the specified resource group

retrieveListRoles

Returns a list of roles that are have permissions to the specified resource

retrieveListUsers

Returns the a list of users associated with a specified resource

saveCollection

Saves a collection of resource object instances

Properties

Property

Description

active

A boolean that indicates whether the role is active or inactive

description

The name of the resource

group

The resource category

insertBy

The ID of the end-user who created the object instance

insertDate

The date on which the end-user created the object instance

maskBit

The offset value of the bit that represents the resource in the permissions mask.

resourceId

The unique identifier of the resource

updateBy

The user ID of the person who last updated the object instance

updateDate

The date of the last object instance update
