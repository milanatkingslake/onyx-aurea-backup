retrieveListBySession (scriptSessionPromptResponse)

retrieveListBySession method

The retrieveListBySession method of the scriptSessionPromptResponse object returns domain and transactional data related to a scriptSession. The purpose of this method is to support printing or displaying a summary of responses for a given script session. The data is returned as a rowSet object in the parametersReturn XML.

Remarks

The result set is ordered by the following sequence:

  1. stepSequenceNumber
  2. promptSequenceNumber
  3. responseSequenceNumber

This method returns prompt text without any merging of variables. For prompt text merged with variable information, call the retrieveMergedListBySession method.

Example

The following example uses retrieveListBySession to show domain and transactional information about all the responses to a script in a given session. Notice in two of the rows returned that the node c5 text still contains the variable template.
