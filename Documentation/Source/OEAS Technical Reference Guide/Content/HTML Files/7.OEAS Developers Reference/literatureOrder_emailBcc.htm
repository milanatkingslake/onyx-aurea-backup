emailBcc (literatureOrder)

emailBcc property

The emailBcc property of the literatureOrder object contains the email address of a blind-carbon-copy recipient.

Data type

string (maximum length)

Access

Read/write

Remarks

The emailBcc property is used for specifying addresses of blind-carbon-copy recipients of email messages. This property supports multiple email addresses separated by semi-colons.
