OTM interface sample

The Onyx TransactionManager (OTM) interface is the only entry point for client applications to access the business logic of the OEAS middle tier. This interface consists of a single method (Execute). All object methods use the same call but with different input XML. See the specific documentation for an object method to determine the correct input XML.

The OTM Interface Sample shows how a company insert call works.

This sample inserts a company record in the Onyx Enterprise Database. To use it, do the following:

  1. Install the sample.
  2. Make sure you have a valid Onyx user ID and password.
  3. Construct the input XML.
  4. Get authenticated by the OEAS. This requires an existing Onyx user in the OEAS system.
  5. Call the OTM Execute method.
  6. Verify a successful transaction.
