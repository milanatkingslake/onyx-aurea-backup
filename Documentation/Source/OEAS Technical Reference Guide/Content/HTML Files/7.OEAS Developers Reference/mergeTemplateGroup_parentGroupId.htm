parentGroupId (mergeTemplateGroup)

parentGroupId property

The parentGroupId property of the mergeTemplateGroup object contains the primaryId of the parent mergeTemplateGroup that the mergeTemplateGroup instance belongs to.

Data type

i4

Access

Read/write

Remarks

The parentGroupId property defines the hierarchy of template groups. When this property is NULL, the mergeTemplateGroup is a top-level template.
