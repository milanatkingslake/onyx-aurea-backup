Adding a step to the individual insert method

Perform the following procedure to add the step component call to the individual object insert method.

  1. Open Onyx Enterprise Studio Object Designer.
  2. Navigate to Logical Business Objects\individual\Methods\retrieve\Steps.
  3. Click the Add button to create a new step.
  4. Enter the data from the following table in the appropriate fields.
Field Value

Name

welcomeEmail

Description

send email greeting to new customers

Step Type

COM

Step Data

WelcomeEmailStep.WelcomeEmail

Additional Data

stepObject=individual;

stepMethod=insert;

dataPath=;

Override Template

[leave blank]

Installed By

customization

Disabled

unchecked
  1. Click the Save button to save your changes.
  2. Optionally, you can click the Validate button to verify that the modifications you made are acceptable. When you publish, the validation is also performed, so this is an optional step.
  3. Click the Publish button. Click 'OK' at the prompt that asks if you want to overwrite the current OED.
  4. In the next step, you will shut down the Onyx App COM+ to cause the OED to reload.
