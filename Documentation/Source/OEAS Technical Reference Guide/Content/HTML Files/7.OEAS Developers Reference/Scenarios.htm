Authentication scenarios

The following authentication scenarios describe the use of OEAS security under various circumstances. Each scenario describes the authentication process beginning with a client call and ending where the Onyx Transaction Manager (OTM) is called. The OTM behaves the same regardless of the type of authentication.

 

This section describes the following scenarios:

  • Browser access for Windows domain users
  • Browser access for users without domain accounts
  • Win32 application access
