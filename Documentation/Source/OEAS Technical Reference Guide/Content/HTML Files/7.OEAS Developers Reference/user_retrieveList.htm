retrieveList (user)

retrieveList method

The retrieveList method of the user object returns a list of Onyx users.

Parameters

Parameter

Direction

Data type

Content type

Description

active

in

boolean

—

Specifies whether to return active users or inactive users. If the active element of the input parameters XML is empty or NULL, all users are returned, active and inactive. When active contains 1, all active users are returned. If the element contains 0, inactive users are returned. If active contains an invalid value, such as 2, the method succeeds, but no users are returned.

rowSet

out

object

all

A list of Onyx users.

Remarks

This method retrieves only a limited list of data, and not always the entire list of data available. You can use the OEAS Configuration utility to configure the maximum rows returned. For more information, see Paging functionality.
