Object Physical Map version attributes

When you select the object physical map node in the Object Designer, the versioning attributes display in the edit area.

The attributes are defined as follows:

Attribute Name

Description

ID

The globally unique identifier (GUID) assigned by the Onyx Enterprise Dictionary. This attribute is read-only.

Installed By

Type the name of the vendor or application installing this object.

Onyx Version

The Onyx version number of this object. This attribute is read-only.

Custom Version

Type the custom version number you want to assign to this object.

Released

This attribute is for documentation purposes only. Check this box to document that the object physical maps are no longer under development.

The Regenerate Maps button creates object physical maps for the object's properties. This utility is useful when adding a business object to the OED. After adding properties, generate the maps, and edit them to provide the necessary attributes for each map.

Caution: Regenerating maps deletes the attributes of existing object physical maps for the currently selected business object.
