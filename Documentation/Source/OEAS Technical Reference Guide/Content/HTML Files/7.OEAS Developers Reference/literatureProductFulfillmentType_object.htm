literatureProductFulfillmentType Object

literatureProductFulfillmentType object

The literatureProductFulfillmentType object provides a means to assign fulfillmentType domain data to a literatureProduct. The fulfillment type determines how the Onyx Processing Server handles the product. Typical fulfillment types include email and print.

Methods

Method

Description

retrieveCollection

Retrieves a collection of literatureProductFulfillmentTypes

saveCollection

Performs multiple actions at once: the insert, update, and delete of literatureProductFulfillmentType objects, as appropriate
