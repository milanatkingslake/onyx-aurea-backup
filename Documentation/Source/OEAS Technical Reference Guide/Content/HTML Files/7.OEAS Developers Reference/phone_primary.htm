primary (phone)

primary property

The primary property of the phone object identifies whether the phone number is the customer's primary phone number or an alternate phone number.

Data type

boolean

Access

Read/write

Remarks

A customer has only one primary phone and can have as many alternate phones as needed.

Valid values for the primary property are:

  • false (0): The phone is an alternate phone for the customer.
  • true (1): The phone is the primary phone for the customer.
