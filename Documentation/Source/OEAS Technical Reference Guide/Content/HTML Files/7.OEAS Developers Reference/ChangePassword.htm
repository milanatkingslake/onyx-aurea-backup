ChangePassword (IAuthenticateOnyx)

ChangePassword method

The ChangePassword method of the IAuthenticateOnyx interface changes the password of a user.

Syntax

IDL

HRESULT ChangePassword(

    [in] const BSTR strOnyxUserName, 

    [in] const BSTR strOldPassword, 

    [in] const BSTR strNewPassword, 

    [in] const BSTR strApplicationName, 

    [in] const BSTR strSiteID);

Parameter descriptions

strOnyxUserName

[in] A string that contains the user's login name.

strOldPassword

[in] A string that contains the user's old password.

strNewPassword

[in] A string that contains the user's new password.

strApplicationName

[in] A string that contains the application name.

strSiteID

[in] A string that contains the site ID.

Return values

When called from a C++ client application, this method returns a standard COM HRESULT or one of the custom HRESULTS listed below. When a Visual Basic client application calls this method and an error occurs, the Err object specifies one of the decimal values listed below and provides a brief description of the error.
