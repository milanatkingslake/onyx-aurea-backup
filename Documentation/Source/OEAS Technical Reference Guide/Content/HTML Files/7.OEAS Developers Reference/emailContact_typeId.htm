typeId (emailContact)

typeId property

The typeId property of the emailContact object identifies the type of email contact.

Data type

i4

Access

Read/write

Remarks

Valid values of the typeId property include:

Value Description
1 To
2 Cc (courtesy copy)
3 Bcc (blind courtesy copy)
4 From

When the collection of emailContact instances are inserted with an emailMessage instance, one—and only one—emailContact must have a From typeId.
