Pasting items in the navigation tree

To paste an item from the clipboard:

  1. To place the copied item in a specific location, select the item above that location and click . To append the item to a collection, select the collection and click .

    The new item appears in the navigation area having the same name as the original item with a number and an asterisk ( * ) appended to it, for example, company1* or secondaryId1*. An asterisk appears next to any item in the navigation area that has not yet been saved.

  2. Modify the pasted item as needed. The Object Designer generates GUIDs for the copied items and updates the items' references with the GUIDs.
