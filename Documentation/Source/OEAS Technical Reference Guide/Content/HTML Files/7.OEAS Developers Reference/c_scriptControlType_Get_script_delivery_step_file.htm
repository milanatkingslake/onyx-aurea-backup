To locate and back up the script_delivery_step.xsl file, use the procedure for locating and backing up script_controls.js. This procedure is in the topic "Locating and backing up script_controls.js."

The scripting source directory contains three files that begin with 'script_delivery_', so make sure that you back up the .xsl file and not the .asp or the .js file.
