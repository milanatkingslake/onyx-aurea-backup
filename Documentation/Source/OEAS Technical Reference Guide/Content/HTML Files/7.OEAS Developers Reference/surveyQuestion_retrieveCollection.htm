retrieveCollection (surveyQuestion)

retrieveCollection method

The retrieveCollection method of the surveyQuestion object returns a collection of objects based on the search criteria supplied in the parameters XML.

retrieveCollection standard method information

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Retrieves a collection of object instances

<parameters>

   <surveyQuestions objectType="surveyQuestion" collection="1">

         <questionText/>

         <sequenceNumber/>

         <insertBy/>

         <insertDate/>

         <updateDate/>

      </surveyQuestion>
