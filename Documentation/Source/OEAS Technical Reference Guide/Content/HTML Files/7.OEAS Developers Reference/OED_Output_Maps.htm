Output Maps

The output maps collection contains information of how data is returned from a stored procedure. The collection may be empty, or may contain one or more output map entries. Each map has a map type, which is determined by whether the stored procedure is designed to communicate through several "out" parameters or via a single result set.

The following map types are available for output maps:

Map Type

Description

parameter

The parameter map type allows values returned by stored procedure output parameters to be transferred to method output parameters. With this map type, you must select a procedure output parameter and the method parameter that should receive the value of the procedure parameter.

resultSet

The resultSet map type allows a result set returned by the stored procedure to be transferred into a rowSet object that exists as a method parameter. For this type of map, no procedure parameter need be specified.
