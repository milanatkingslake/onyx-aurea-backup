Verify the required nodes

When reading the step package, your step component must ensure that all required XML nodes are present in the XMLDOMDocument interface. If all XML nodes are present, your step component can complete its task successfully. If any of the XML nodes are missing, exit the function.

This example requires the following XML nodes:

  • xmlCompanyName
  • xmlSymbol

' The companyName is in the parametersReturn element.

    Set xmlCompanyName = XMLParser.selectSingleNode("//parametersReturn/company/customerName")

' The stockSymbol is in the parametersReturn element.

    Set xmlSymbol = XMLParser.selectSingleNode("//parametersReturn/company/stockSymbol")

' Check to make sure that we found the nodes we need,

' otherwise don't bother with execution.

    If Not xmlCompanyId Is Nothing And Not xmlSymbol Is Nothing Then

'Do the processing

    End If
