insert (literatureOrder)

insert method

The insert method of the literatureOrder object adds a literatureOrder object and its associated literatureOrderLine objects to the database.

insert standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Calls a pre-procedure that generates an integer value for the secondaryId when the element is null in the parameters XML

Calls a pre-procedure that inserts the current date when the orderDate element is null

Inserts a literature order record

Calls a post-procedure that inserts customer information from the owner object when those elements are null

Calls a post-procedure that inserts shipping address information from the owner object when those elements are null

literatureOrderLine.saveCollection

method

Inserts a collection of literatureOrderLine objects

Remarks

Include the complete collection of literatureOrderLine objects with their parent object on the insert. Each child object in the collection must have the action attribute set to "insert."
