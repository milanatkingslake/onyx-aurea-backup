categoryId (incident)

categoryId property

The categoryId property of the incident object contains an ID representing the top-level type of incident record, such as Sales, Service, or Support.

Data type

i4

Access

Read/write

Remarks

You can create your own unique categoryIds using the Reference Table Administration tool in the Onyx Enterprise Studio. The value of this property references an existing incidentCategory. This includes the default Sales, Service, and Support categories, or the categories that you create.
