Encoding standards

The XML in the OEAS makes use of both character encoding and data encoding standards. Data encoding standards define how elements and attributes are used in OEAS. Element encoding consists of a discrete set of element tags that bracket the value of the specified element. Attribute encoding consists of elements that contain additional attribute qualifiers, allowing them to transfer information by attributes alone.
