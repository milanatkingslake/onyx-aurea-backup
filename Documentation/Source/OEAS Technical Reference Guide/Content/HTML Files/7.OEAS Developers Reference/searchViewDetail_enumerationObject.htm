enumerationObject (searchViewDetail)

enumerationObject property

The enumerationObject property of the searchViewDetail object contains the name of the object that retrieves an enumerated set of items.

Data type

string (maximum length)

Access

Read/write

Remarks

If the detail item is a region code, for instance, you should retrieve a list of valid region codes to display to the user as a selection option. You specify the region object here.
