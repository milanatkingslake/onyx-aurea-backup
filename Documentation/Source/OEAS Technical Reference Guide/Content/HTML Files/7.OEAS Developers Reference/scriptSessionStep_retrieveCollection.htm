retrieveCollection (scriptSessionStep)

retrieveCollection method

The retrieveCollection method of the scriptSessionStep object searches the database and returns a collection of zero or more scriptSessionStep objects that match the search criteria.

retrieveCollection standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Returns a collection of scriptSessionStep objects

Remarks

The retrieveCollection method returns the steps of a session in ascending order of sequenceNumber.

Example

The following example shows a retrieveCollection that searches for all scriptSessionStep objects belonging to a given scriptSession.

Parameters

<parameters>

   <scriptSessionSteps objectType="scriptSessionStep" collection="1">

      <scriptSessionStep objectType="scriptSessionStep">

         <scriptSessionId>100003</scriptSessionId>

      </scriptSessionStep>

   </scriptSessionSteps>

</parameters>
