retrieve (task)

retrieve method

The retrieve method of the task object searches the database based on the properties defined by the object's primary objectKey entry and returns an object instance.

retrieve standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters.

OnyxDM.DataManager

com

Obtains the specified task from the database.

Uses a database function mapped to the keyWords property that retrieves all tags associated with the task and concatenates them into a delimited list.

Uses a database function mapped to the ownerName property to retrieve a customer name based on the ownerId, which identifies the incident owning the task. The function returns the name of the customer owning the incident that owns the task.
