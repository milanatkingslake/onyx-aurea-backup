retrieveListByAssociation (appointment)

retrieveListByAssociation method

The retrieveListByAssociation method of the appointment object returns a list of appointment records that match the supplied input criteria.

Parameters

Parameter Direction Data type Content type Description
associatedWithId in string n/a The unique identifier of the object associated with the appointments
associatedWithType in i4 n/a The type of object associated with the appointments (by default this is company, individual, or incident)
startDate in dateTime n/a The beginning date and time to search for appointments
endDate in dateTime n/a The ending date and time to search for appointments
rowSet out object all Contains rowSet column definitions and the appointment records in the database that match the specified search criteria

Columns returned in the rowSet object

columnOrdinal columnName Physical column
1 appointmentId appointment.appointment_id, appointmentAssociation.appointment_id
2 externalAppointmentId appointment.external_appointment_id
3 attendeeOrganizer appointment_attendee.display_name or appointment_attendee.email_address
4 attendeeRequired appointment_attendee.display_name or appointment_attendee.email_address
5 attendeeOptional appointment_attendee.display_name or appointment_attendee.email_address
6 attendeeResources appointment_attendee.display_name or appointment_attendee.email_address
7 subject appointment.subject
8 location appointment.location
9 startDate appointment.start_date
10 endDate appointment.end_date
11 allDay appointment.all_day
12 private appointment.private
13 calendarReminder appointment.calendar_reminder
14 reminderMinutes appointment.reminder_minutes
15 cancelled appointment.cancelled
16 recurText none - generated in the stored procedure

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieve list of appointment objects based on the supplied criteria
