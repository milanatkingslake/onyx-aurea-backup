Developing the client user interface

You typically communicate with the Onyx TransactionManager by a client user interface application. The client application typically handles user authentication, input validation, construction of XML input, and so on. The client application exposes a form. Users fill in the form and click a save button to persist the form data. When users click save, the client application creates an xml string containing the data to be persisted.
