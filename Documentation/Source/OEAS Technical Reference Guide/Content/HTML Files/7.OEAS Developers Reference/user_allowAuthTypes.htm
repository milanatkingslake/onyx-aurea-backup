allowAuthTypes (user)

allowAuthTypes property

The allowAuthTypes property of the user object contains bitmask that indicates what types of authentication are available to the user.

Data type

ui1

Access

Read/write

Remarks

Valid values of the allowAuthTypes property are:

  • 0: The user has no login.
  • 1: The user has Windows authentication.
  • 2: The user has Onyx authentication.
  • 3: The user has both Windows and Onyx authentication.
