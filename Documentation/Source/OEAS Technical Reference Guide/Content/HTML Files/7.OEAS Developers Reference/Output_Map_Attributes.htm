Output Map attributes

When you select an output map in the Object Designer, the attributes for that output map display in the edit area.

The attributes are defined as follows:

  • Map Type: Select the type of procedure map for this output map:
  • To map procedure out parameters to method parameters, select parameter.
    • Procedure Parameter: Select the procedure parameter to include in this map.
  • To map the result set created by the procedure method parameter, select resultSet.
  • Method Parameter:Select the name of the method parameter. Generally, if the Map Type is resultSet, choose rowSet.
