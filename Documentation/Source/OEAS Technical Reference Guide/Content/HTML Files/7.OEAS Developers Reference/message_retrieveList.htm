retrieveList (message)

retrieveList method

The retrieveList method of the message object returns a message list for the specified user.

retrieveList standard method information

Parameters

Parameter

Data type

Direction

Description

userId (required)

string

in

A string that identifies an Onyx user

rowSet

object

out

A rowSet of information about each message in the user's queue
