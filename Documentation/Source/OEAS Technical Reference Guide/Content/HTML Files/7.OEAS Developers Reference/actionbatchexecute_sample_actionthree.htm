(ActionBatchExecute) process third action
Process third action: worknote.insert

The third action creates a work note for the incident created in the second action, incident.insert. For each sourceListMap in the actionBatchXml the value pointed to by the sourceElementPath value is applied to targetElementPath. In this case the companyName element from the sourceList is mapped to the methodInputs/parameters/workNote/workNoteText element.

<sourceListMaps>

  <sourceListMap>

     <sourceElementPath>companyName</sourceElementPath>

     <targetElementPath>parameters/workNote/workNoteText</targetElementPath>

  </sourceListMap>

</sourceListMaps>

For each actionReturnMap the value pointed to by the sourceElementPath value is applied to the targetElementPath. In this case the incident.primaryId returned from the incident.insert action is mapped to the workNote.ownerId to be used in the workNote.insert action.

<actionReturnMaps>

    <actionReturnMap>

       <sourceActionId>a</sourceActionId>

       <sourceElementPath>parametersReturn/incident/primaryId</sourceElementPath>

       <targetElementPath>parameters/workNote/ownerId</targetElementPath>

    </actionReturnMap>

</actionReturnMaps>

Using the updated methodInputs parameters, the method call is made on the object thatlogicalObjectName and logicalMethodName point to. In this case worknote.insert is called to insert a new worknote using the incident ID of the newly inserted incident from the previous action. Error-handling behavior is determined by the value of onFailure.

XML for workNote.insert action

<action>

   <onFailure>stop</onFailure>

   <id>b</id>

   <logicalObjectName>workNote</logicalObjectName>

   <logicalMethodName>insert</logicalMethodName>

   <methodInputs>

      <parameters>

         <workNote objectType="workNote">

            <primaryId/>

            <seqNum/>

            <ownerId/>

            <ownerType>6</ownerType>

            <logMark>1</logMark>

            <enteredBy/>

            <entryDate/>

            <publishBitmask>1</publishBitmask>

            <workNoteText>This incident was created using actionBatch.execute!</workNoteText>

            <user1/>

            <user2/>

            <user3/>

            <user4/>

            <user5/>

            <user6/>

            <user7/>

            <user8/>

            <user9/>

            <user10/>

            <insertBy/>

            <insertDate/>

         </workNote>

      </parameters>

      <customData/>

   </methodInputs>

   <actionReturnMaps>

      <actionReturnMap>

         <sourceActionId>a</sourceActionId>

         <sourceElementPath>parametersReturn/incident/primaryId</sourceElementPath>

         <targetElementPath>parameters/workNote/ownerId</targetElementPath>

      </actionReturnMap>

   </actionReturnMaps>

</action>
