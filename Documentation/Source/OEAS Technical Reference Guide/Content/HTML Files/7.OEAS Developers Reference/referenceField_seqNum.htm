seqNum (referenceField)

seqNum property

The seqNum property of the referenceField object contains the ordinal position of a referenceField object on a user interface.

Data type

i2

Access

Read/write

Remarks

The ordinal position of a referenceField object is determined by its relationship to other referenceField objects in the same group.

For example, the seqNum property can be used to order a referenceField object named individual.user1 before or after a referenceField object named individual.user7.

The presentation tier must respect the value of the seqNum property in order for it to be meaningful.
