Procedure Parameter Attributes

When you select a procedure parameter in the Object Designer, the attributes for that procedure parameter display in the edit area.

The attributes are defined as follows:

Attribute Name

Description

ID

The globally unique identifier (GUID) assigned by the Onyx Enterprise Dictionary. This attribute is read-only.

Name

Type the name you want to assign to this property. Property names must conform to SQL Server and XML naming conventions.

Data Type

Select the data type for this parameter.

Direction

Select the direction that the data should pass through this parameter:

  • If the parameter is required on input only, select in.
  • If the parameter is required on output only, select out.
  • If the parameter is required on input and returned on output, select inOut. The parameter may change during the process

Size

For data types that are variable in length, such as varchar, type the number of bytes for the parameter. Use the size that is defined for the parameter in the procedure declaration. Size is not needed for fixed data types like integer.
