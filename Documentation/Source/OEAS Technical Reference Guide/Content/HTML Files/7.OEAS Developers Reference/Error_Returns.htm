OEDCacheStatusEnum

OEDCacheStatusEnum enumerates the return status for each of the ICache interface methods. The possible values of this enumeration are as follows:

Name (value)

Description

ocsSuccess (0)

The operation completed successfully.

ocsOEDNotFound (1)

The requested Onyx Enterprise Dictionary (OED) name cannot be found.

ocsObjectNotFound (2)

The requested object cannot be found in the OED cache.

ocsMethodNotFound (3)

The requested method for the given object cannot be found.

ocsReadError (4)

An error occurred when attempting to read from the OED cache.

ocsWriteError (5)

An error occurred when attempting to write to the OED cache.

ocsOEDLoadError (6)

The OED cache cannot load the metadata from disk.

ocsPropertyNotFound (7)

The requested property cannot be found in the OED cache.
