retrieveCollection (emailAssociation)

retrieveCollection method

The retrieveCollection method of the emailAssociation returns a collection of objects based on the search criteria supplied in the parameters XML.

retrieveCollection method standard behavior

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieves a collection of emailAssociation objects

Example

Parameters

<parameters>

   <emailAssociations objectType="emailAssociation" collection="1">

      <emailAssociation objectType="emailAssociation">

         <primaryId/>

         <ownerId>3AE95902-21F5-45F3-B3EA-5767A4AFFC45</ownerId>

         <emailContactId/>

         <associatedWithId>1</associatedWithId>

         <associatedWithType>1</associatedWithType>

         <showAssociationOnObject>1</showAssociationOnObject>

         <typeCode>2</typeCode>

      </emailAssociation>

   </emailAssociations>

</parameters>
