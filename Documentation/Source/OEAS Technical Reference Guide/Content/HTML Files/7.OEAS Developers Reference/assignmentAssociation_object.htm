assignmentAssociation object

The assignmentAssociation object contains an association between a customer or an incident and a territory.

The assignmentAssociation object has one key property, primaryId, that uniquely identifies object instances in the database. Once the object is created, the value of this property cannot be changed.

Methods

Method Description
deleteByAssociatedWith Deletes all assignmentAssociation objects associated with an OEAS object
evaluateCompany Initiates the evaluation of an OEAS company record for assignment
evaluateIndividual Initiates the evaluation of an OEAS individual record for assignment
insertReplace Inserts or updates an association between a territory and a customer or incident
retrieveCollection Returns a collection of objects based on the search criteria supplied in the parameters XML

Properties

Property Description
assignmentTerritoryId The unique identifier of the associated assignmentTerritory object
associatedWithId The unique identifier of the associated company, individual, or incident object
associatedWithType An enumerated value that identifies the type of object identified in the associatedWithId
insertBy The user ID of the Onyx user who initially saves the object
insertDate The time and date on which the object is initially saved
primaryId The unique identifier of the assignmentAssociation object
updateBy The user ID of the Onyx user who last saved modifications to the object
updateDate The time and date on which modifications to the object were last saved
