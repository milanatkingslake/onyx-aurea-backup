showAssociationOnObject (emailAssociation)

showAssociationOnObject property

The showAssociationOnObject property of the emailAssociation object specifies whether the association causes the email message to display when users view the associated object instance.

Data type

boolean

Access

Read/write

Remarks

Valid values of this property are:

0: The email message does not appear when viewing the associated object.

1: The email message appears when viewing the associated object.
