saveCollection (userPreference)

saveCollection method

The saveCollection method of the userPreference object performs multiple actions at once: inserting, updating, and deleting userPreference objects, as appropriate.

saveCollection method standard behavior

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Inserts, updates, and/or deletes a collection of userPreference objects
