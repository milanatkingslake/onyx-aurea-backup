emailAssociation object

emailAssociation object

The emailAssociation object tracks which transactional objects are associated with an email message.

Methods

Method Description
retrieveCollection Retrieves a collection of specified objects
saveCollection Saves a collection of specified objects

Properties

Property Description
associatedWithId The ID of the object instance associated with the emailMessage
associatedWithType The type of object associated with the email
emailContactId The ID of the contact that relates to the association
insertBy The user ID of the Onyx user who initially saves the object
insertDate The time and date on which the object is initially saved
onyxTimestamp Facilitates record locking with the Onyx Enterprise Database
ownerId Identifies the parent object instance of the child object
primaryId The ID assigned to the object when it is first saved to the database
showAssociationOnObject Specifies whether the association causes the email message to display when users view the associated object instance
typeCode The type of email association
updateBy The user ID of the Onyx user who last saved modifications to the object
updateDate The time and date on which modifications to the object were last saved
