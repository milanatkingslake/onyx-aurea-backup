insert (task)

insert method

The insert method of the task object persists an object to the database.

insert standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxGuidGeneratorStep.GuidGenerator

com

Calls a component that generates GUID primary key values used for inserting the tags identified in the task's tags property

OnyxDM.DataManager

com

Calls a pre-procedure that generates an integer value for the secondaryId when the element is null in the parameters XML

Adds the specified incident object to the database

Calls a post-procedure that inserts or updates the tag object with the terms listed in the task's tags property
notificationAdapter.postInsertEvent com This step is added for notification feature. It captures data after saving the object
