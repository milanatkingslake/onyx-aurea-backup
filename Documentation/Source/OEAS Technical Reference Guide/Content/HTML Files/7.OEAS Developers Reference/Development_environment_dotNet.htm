Configure your .NET step component development environment

Before developing a .NET step component for the OEAS platform, ensure that you have the necessary tools and a strong understanding of the following technologies:

  • Custom COM+ step component development for OEAS
  • Microsoft .NET Framework
  • Microsoft Visual Studio
  • Microsoft COM+

You can use C#, C++, or VB.NET to create .NET step components for use with OEAS.
