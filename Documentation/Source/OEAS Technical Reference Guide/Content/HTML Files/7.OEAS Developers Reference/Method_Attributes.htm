Method attributes

When you select a method in the Object Designer, the attributes for that method display in the edit area.

The attributes are defined as follows:

  • ID: The globally unique identifier (GUID) assigned by the Onyx Enterprise Dictionary. This attribute is read-only.
  • Name: Type the name you want to assign to this method. Method names must be conform to XML naming conventions.

If you change the name of an existing object or method, use the Security Administration tool to establish the access privileges as if the object/method were new. Any security settings previously established for the renamed object/method are not maintained.

Important: When you change the name of a method, you must also make changes to the step Additional Data field "stepMethod" on each step owned by the method. Otherwise, errors occur in the COM objects called from the steps.
  • Description: Type the description for this method.
  • Method Type:Select the type of method you want to create. This selection determines the SQL generation process for the method.
  • Installed By: Type the name of the vendor or application adding this method.
  • Disabled: Check this box to disable this method. Clear the box to enable the method.
