retrieveCollection (methodCallDetail)

retrieveCollection method

The retrieveCollection method of the methodCallDetail object returns a collection of methodCallDetail objects based on the search criteria supplied in the parameters XML.

retrieveCollection method standard behavior

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Retrieves a collection of methodCallDetail objects

Remarks

The methodCallDetails collection must contain at least one methodCallDetail element in the parameters XML, although the methodCallDetail element can be empty.

Example

The following example shows a retrieveCollection that filters on the methodCallId property.

<parameters>

   <methodCallDetails objectType="methodCallDetail" collection="1">

      <methodCallDetail objectType="methodCallDetail">

         <methodCallId>100014</methodCallId>

      </methodCallDetail>

   </methodCallDetails>

</parameters>
