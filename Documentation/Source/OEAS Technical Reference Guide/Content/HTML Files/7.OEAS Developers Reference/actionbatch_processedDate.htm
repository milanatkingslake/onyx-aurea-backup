processedDate property

The processedDate property of actionBatch object contains the date on which the batch was processed.

Data type

dateTime

Access

Read/Write

Remarks

The processedDate property is set only once when the batch is first processed. Attempts to change this value are ignored. The processedDate value is created by the database during the initial process. The database obtains the date value from the database server clock. The database converts the date value to UTC date before saving it.
