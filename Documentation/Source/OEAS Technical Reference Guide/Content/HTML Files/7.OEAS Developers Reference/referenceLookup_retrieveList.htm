Remarks

This method supports wild card substitution in search criteria. For more information on wild card substitution, see the retrieveList method (standard behavior) topic.

This method returns a rowSet with the contents of a referenceField object. If this method retrieves a referenceLookup object whose type property contains freetext, then it returns an empty result set.

To access fields in hierarchical combo boxes, the parentId property identifies the parent referenceParameter object instance of the data to be accessed.
