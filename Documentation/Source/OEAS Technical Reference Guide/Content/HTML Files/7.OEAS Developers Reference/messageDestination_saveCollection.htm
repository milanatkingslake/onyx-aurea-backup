saveCollection (messageDestination)

saveCollection method

The saveCollection method of the messageDestination object performs multiple actions at once: inserting, updating, and deleting object instances, as appropriate.

saveCollection standard method information

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Saves, updates, and deletes a collection of messages as indicated by the input parameters

Remarks

A saveCollection update action is typically used to mark messages as read when recipients open their messages.
