saveCollection (customerProductDetail)

saveCollection method

The saveCollection method of the customerProductDetail object performs multiple actions at once: the insert, update, and delete of customerProductDetail objects. The parent object of the customerProductDetail object calls this method. It is not intended to be called directly.
