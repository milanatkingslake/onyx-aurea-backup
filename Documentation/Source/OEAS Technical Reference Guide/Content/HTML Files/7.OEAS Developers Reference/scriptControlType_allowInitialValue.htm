allowInitialValue (scriptControlType)

allowInitialValue property

The allowInitialValue property of the scriptControlType object specifies whether a control can be bound to a literal or variable input value.

Data type

Boolean

Access

Read/write

Remarks

For some controls such as checkBox, the script designer might want to assign a default value to save the user some time on input. The designer does this using an input value or input variable whose value is bound to the control when it is rendered in the script step. Suppose you have a prompt that asks whether the customer wants to receive email updates about products. The designer wants the checkBox associated with this prompt to be checked by default, because most customers want to receive such emails. He can set the initial value of the checkBox to 1 (literal value), and the checkbox appears checked when it is rendered in the user interface. For a scriptControlType to support such input binding, the allowInitialValue property must be set to true (1).
