phones (company)

phones property

The phones property of the company object represents a collection of phone objects for the company.

Data type

object

Access

Read/write

Remarks

A phones object contains a collection of one or more phone objects. A customer can have multiple phones that are accessed via the phones collection. The phones property, however, does not correspond to any object in the Onyx Enterprise Dictionary (OED). Instead, the phones property is defined as a collection of phone objects.
