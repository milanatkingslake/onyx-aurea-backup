currency object

The currency object represents domain data for the various global monetary units. It associates a standard ANSI code with a display name.

The currency object has one key property, currencyCode, that uniquely identifies the currency in the database. Once the object is created, the value of this property cannot be changed.

Methods

Method Description
retrieveCollection Returns a collection of objects based on the search criteria
retrieveList Returns a rowSet of currency information
saveCollection Performs multiple actions at once: inserting, updating, and deleting objects, as appropriate

Properties

Property Description
active Specifies whether the domain data the object provides is displayed in the user interface
currencyCode The one-to-four-letter alphabetic code that identifies the currency
description Contains a text string that identifies the currency
insertBy Contains the user ID of the Onyx user who initially saved the object
insertDate Contains the time and date on which the object was initially saved
localeId Contains a standard international language identifier associated with the currency code
updateBy Contains the user ID of the Onyx user who last modified the object
updateDate Contains the time and date on which the object was last modified
