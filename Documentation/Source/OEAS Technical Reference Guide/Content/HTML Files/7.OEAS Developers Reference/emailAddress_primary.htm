primary (phone)
primary property

The primary property of the emailAddress object identifies whether the email address is the customer's primary email address or an alternate email address.

Data type

boolean

Access

Read/write

Remarks

A customer has only one primary email address and can have as many alternate email addresses as needed.

Valid values for the primary property are:

  • false (0): The email address is an alternate email address for the customer.
  • true (1): The email address is the primary email address for the customer.
