retrieveListByDate (reminder)

retrieveListByDate method

The retrieveListByDate method of the reminder object returns a list of reminder objects based on date criteria that you pass in.

retrieveList standard method information

Parameters

Parameter

Direction

Data type

Content type

Description

userId (required)

in

string

--

The ID of the OEAS user for whom reminders are set. This parameter is required because the method does not use the current user as the default.

startDate (required)

in

dateTime

--

Specifies the beginning date and time in a range to search for reminders.

endDate

in

dateTime

--

Specifies the ending date and time in a range to search for reminders. The default value is the current time plus 15 minutes.

alarmActive

in

i1

--

Indicates whether to retrieve reminders with the alarm set.

rowSet

out

object

all

Contains rowSet column definitions and the reminder table rows matching the date range and alarm set criteria.
