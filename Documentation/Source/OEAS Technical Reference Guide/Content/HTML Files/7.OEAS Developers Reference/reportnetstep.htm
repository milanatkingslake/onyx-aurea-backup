ReportNetStep

The ReportNetStep component enables an OEAS client application to retrieve content metadata for a folder or to create a new folder in the Cognos content store. The step implements the functionality for the reportNet logical business object (LBO). The retrieveFolderItems and createFolder methods of the reportNet LBO include a step to call the ReportNetStep component. The ReportNetStep component determines which method to call from the StepAdditionalData parameter to the IStep Execute method. This step component is not intended for use by other standard OEAS methods or custom methods. If the StepAdditionalData contains anything other than one of these methods, the component returns an error.

In addition to validating the StepAdditionalData, the component validates the other IStep Execute parameters as outlined in the following list.

  • The application name must be present, or an error is returned.
  • The component ignores LogicalObjectName.
  • The component ignores LogicalMethodName.
  • The XMLParser (DOM) must be present. If the parser is missing, the step raises a COM error. The step cannot return XML or log errors.
