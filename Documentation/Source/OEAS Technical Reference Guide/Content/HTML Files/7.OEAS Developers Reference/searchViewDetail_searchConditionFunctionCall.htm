searchConditionFunctionCall (searchViewDetail)

searchConditionFunctionCall property

The searchConditionFunctionCall property of the searchViewDetail object contains the function applied to the searchTable alias and fieldName on the left hand side of the searchViewDetail WHERE clause.

Data type

string

Access

Read/write

Remarks

The format of the searchConditionFunctionCall is f(~0), where "f" is some database function. The UI replaces the ~0 with the searchTable alias and fieldName values separated by a period when constructing the SQL Query.
