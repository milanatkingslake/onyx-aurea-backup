Step 4: Install the Step Component on the Server
Install the step component on the server

Before the step component can be called, it must be copied to, and registered on, the Onyx Enterprise Application Server.

To install the step component on the application server:

  1. Copy the library file to a directory on the application server. For this sample, copy the CustomStep.dll to the desired location on your machine.
  2. Using the Microsoft Component Services tool, open the OnyxEASLib library application.
  3. Using Microsoft Windows Explorer, select the directory that contains the CustomStep.dll file that you just copied. Drag and drop the CustomStep.dll file to the Components folder of the OnyxEASLib library application. Doing this automatically registers the file.
  4. Make a note of the ProgId of the step component. The ProgId is the name of the DLL in the Component Services folder. When you add a new step to the insert method of the company object, you must supply this ProgId.
