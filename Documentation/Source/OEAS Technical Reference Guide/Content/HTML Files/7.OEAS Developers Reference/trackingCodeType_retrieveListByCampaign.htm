retrieveListByCampaign (trackingCodeType)

retrieveListByCampaign method

The retrieveListByCampaign method of the trackingCodeType object returns a list of tracking code types where at least one tracking code of the type exists on a specified campaign. If no campaignCode is specified, all tracking code types are returned.

Remarks

The retrieveListByCampaign search is case sensitive.

The following table shows the valid values and their behavior for the active input parameter.

Valid Value Description

1

Return only active tracking code types with one or more active tracking codes

0

Return only inactive tracking code types containing one or more inactive tracking codes
