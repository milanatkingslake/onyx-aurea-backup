retrieveCollection (externalContact)

retrieveCollection method

The retrieveCollection method of the externalContact object returns a collection of objects based on the search criteria supplied in the parameters XML.

retrieveCollection standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com

Retrieves a collection of external contacts

Uses a database function mapped to the contactUserName property to retrieve a user name based on the value of the contactId property

Remarks

For this retrieveCollection method, the OEAS ignores the contactUserName property for filter purposes. Any value you provide as input to this property has no affect on the collection returned.

Example

Parameters

In the following example, a search is completed for subsidiary (contactType=39) externalContact types.

<parameters>

   <externalContacts objectType="externalContact" collection="1">

      <externalContact objectType="externalContact">

         <primaryId/>

         <ownerId/>

         <ownerType/>

         <contactId/>

         <contactTypeId>39</contactTypeId>

         <contactUserName/>

         <comment/>

      </externalContact>

   </externalContacts>

</parameters>
