surveyType Object

surveyType object

The surveyType object provides for organizing surveys into groups. The user interface renders the survey selection tree by group and then by name, so all surveys of the same type appear together in alphabetical order.

Methods

Method Description

retrieveCollection

Returns a collection of surveyType objects based on the search criteria supplied in the parameters XML

retrieveListViewDetail

Returns information about the surveyTypes of the specified customer type

saveCollection

Performs multiple actions at once: inserting, updating, and deleting surveyType objects, as appropriate

Properties

Property Description

active

Specifies whether the domain data the object provides is available to the user interface

description

Contains a brief statement of the purpose of the type

insertBy

Contains the user ID of the Onyx user who initially saves the object

insertDate

Contains the time and date on which the object is initially saved

primaryId

Contains the unique ID assigned to the object when it is first saved to the database

updateBy

Contains the user ID of the Onyx user who last saved modifications to the object

updateDate

Indicates the time and date on which modifications to the object were last saved
