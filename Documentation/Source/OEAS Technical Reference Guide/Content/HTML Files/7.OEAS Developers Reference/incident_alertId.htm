alertId (incident)

alertId property

The alertId property of the incident object contains the ID of the alert record that is associated with the incident record.

Data type

string (maximum length)

Access

Read-only; for more information, see Read Only attribute

Remarks

The ID is the foreign key value that represents the association between the incident table and the alert table.

The alertId property replaces the alertNumber property, which was used in OEAS prior to version 4.0.
