warrantyExpDate (literatureProduct)

warrantyExpDate property

The warrantyExpDate property of the literatureProduct object contains the date when the warranty of the associated product expires.

Data type

dateTime

Access

Read/write

Remarks

This is a denormalized property and can only be modified through the product object.

The dateTime data type is formatted as follows:

YYYY-MM-DD hh:mm:ss

where:

YYYY = year

MM = month

DD = day

hh = hour

mm = minute

ss = second

Valid values for the dateTime data type range from 1900-01-01 00:00:00 to 9999-12-30 23:59:59.
