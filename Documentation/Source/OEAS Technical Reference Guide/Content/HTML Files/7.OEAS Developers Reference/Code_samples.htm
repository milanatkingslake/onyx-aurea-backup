Code samples

How to samples

This book contains several samples to illustrate common customization scenarios. The following table lists the samples and gives a brief description of each.

Sample

Description

Add a control type to scripting

Demonstrates how to add a custom script control type to the Onyx Scripting feature.

Add custom collision handling

Demonstrates how to add custom collision handling to manage subscribable objects, when the default collision handling is not sufficient.

Modify business objects

Demonstrates how to save custom data to the Onyx Enterprise Database via the OEAS. In the sample, custom data is saved to the Onyx Enterprise Database by creating a physical column in a database table, adding a property to the logical model, and mapping the logical property to the physical column using the OED. In addition, this sample shows how to modify a property constraint, how to modify a method by calling a custom step component, and how to add a method using a custom stored procedure.

Communicate with OTM

Demonstrates how to communicate with the Onyx Transaction Manager (OTM), the main entry-point into the OEAS, using Active Server Pages. In the sample, Active Server Pages serve as a foundation for a Web user interface that sends customer data to the OEAS and the Onyx Enterprise Database.

Use actionBatch to execute a series of OEAS method calles

Demonstrates how to insert three new companies in a database with incidents, workNotes and workTickets
