sicCode property

The sicCode property of the company object identifies the SIC code of the company.

Data type

i4

Access

Read/write

Remarks

The sicCode property is used to indicate the Standard Industrial Classification (SIC) code. The unique four digit number is assigned to industries by the US Government. Companies may have more than one SIC code. The primary code reflects their main line of business, while secondary codes can exist to reflect other products or services that a company provides. For example, the SIC Code for Onyx Software is 7372, indicating a Computer Programming and Data Processing company.
