totalTime (workTicket)

totalTime property

The totalTime property of the workTicket object contains the elapsed time, in minutes, that a work ticket is opened and not paused in the user interface.

Data type

i4

Access

Read/write

Remarks

OEP does not currently use this property. OEAS will save the value passed in to this property.

The totalTime property is intended to be added in the sum that comprises the totalLabor property.
