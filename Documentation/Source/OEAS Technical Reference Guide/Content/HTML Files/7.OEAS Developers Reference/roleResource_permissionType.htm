permissionType (roleResource)

permissionType property

The permissionType property of the roleResource object specifies the access permissions of the roleId to the resourceId.

Data type

ui1

Access

Read/write

Remarks

The valid value of the permissionType property is 1, the role has access permissions to the resource.

You can grant access permissions to a role resource, but you cannot deny access permissions to a role resource. You can deny individual users access permissions to a role resource on a user-by-user basis via the userResource object.
