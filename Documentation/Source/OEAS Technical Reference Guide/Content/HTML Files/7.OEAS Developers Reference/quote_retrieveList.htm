retrieveList (quote)

retrieveList method

The retrieveList method of the quote object searches the database and returns a rowSet containing the list of quote items that match the input criteria.

Parameters

Parameter

Direction

Data type

Content type

Description

ownerId (required)

in

string

n/a

The unique number of the object that owns the quote object

ownerType

in

i4

n/a

Specifies the type of object for the given ownerId

rowSet

out

object

all(out)

List of rows containing quote information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Data validation step

OnyxDM.DataManager

com

Retrieves each quoteLine of detail from the database

Remarks

The retrieveList search is case sensitive.

Regardless of what you put in the ownerType parameter, the OEAS ignores it.
