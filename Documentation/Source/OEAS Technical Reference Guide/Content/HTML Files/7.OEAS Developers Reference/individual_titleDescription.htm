titleDescription (individual)

titleDescription property

The titleDescription property of the individual object reflects titles that can be assigned to an individual.

Data type

string (maximum length)

Access

Read/write

Remarks

The titleDescription property that can be configured using the Reference Table administration tool. An example of a titleDescription might be Vice-President, which could have a titleCode of VP in the reference objects. There is a parent child relationship between a titleCode and a titleDescription.
