reminder Object

reminder object

The reminder object enables users to create and modify text messages that prompt action related to task or incident objects.

The reminder object has one key property, primaryId, that uniquely identifies the reminder in the database. Once the object is created, the value of this property cannot be changed.

Methods

Method

Description

delete

Logically deletes a specified reminder object

insert

Saves a new reminder object

retrieve

Returns an existing reminder object

retrieveCollection

Returns a collection of reminder objects

retrieveItemInfo

Returns information about a specific reminder and its parent incident

retrieveListByDate

Returns information for all reminders within the specified start and end date

saveCollection

Performs multiple actions at once: the insert, update, and delete of objects as appropriate

update

Saves modifications to an existing reminder object
