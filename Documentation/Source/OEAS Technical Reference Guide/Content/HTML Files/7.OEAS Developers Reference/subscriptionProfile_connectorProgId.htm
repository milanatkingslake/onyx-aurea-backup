subscriptionServiceConnectorProgId (subscriptionProfile)

subscriptionServiceConnectorProgId property

The subscriptionServiceConnectorProgId property of the subscriptionProfile object contains the programmatic identifier of the COM component that connects the synchronization server to the staging server.

Data type

string

Access

Read-only; for more information, see Read Only attribute

Remarks

The default implementation of OEP-Offline requires an Exchange server as the staging server, and the programmatic identifier of the corresponding connector is OnyxPSExchangeSyncConnector.SyncConnector.
