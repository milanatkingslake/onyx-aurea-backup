expired (alert)

expired property

The expired property of the alert object contains a boolean that indicates whether the alert has expired.

Data type

boolean

Access

Read-only; for more information, see Read Only attribute

Remarks

The retrieve method sets the expired flag when the expirationDate is reached. The parametersReturn XML includes the expired flag, but it is not persisted in the database. The user interface must call the delete method if the expired alert is to be removed from the database.

Valid values of the expired property are:

  • 0: The alert has not expired.
  • 1: The alert has expired.
