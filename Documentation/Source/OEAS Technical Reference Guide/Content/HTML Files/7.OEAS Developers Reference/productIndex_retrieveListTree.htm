retrieveListTree (productIndex)

retrieveListTree method

The retrieveListTree method of the productIndex object returns a list of products and product groups.

retrieveList standard method information

Parameters

Parameter

Direction

Data type

Content type

Description

parentId

in

i4

n/a

A number that is the hierarchyId of the product tree's parent product group.

orderable

in

i1

n/a

A number that indicates what type of products to return. Non-orderable products are those products that are not available through standard shipping methods. Possible values for this parameter are:

NULL: return all orderable and non-orderable products and literature products.

1: return all orderable literature products.

0: return all orderable and non-orderable literature products (excluding non-literature products).

active

in

boolean

n/a

A value that indicates whether to return inactive products and product groups:

1: return only active products.

0: return both active and inactive products.

rowSet

out

object

all

An object whose content="all."
