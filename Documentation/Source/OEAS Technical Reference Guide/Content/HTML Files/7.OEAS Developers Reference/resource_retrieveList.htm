retrieveList (resource)

retrieveList method

The retrieveList method of the resource object returns a list of resources for the specified resource group.

retrieveList standard method information

Parameters

Parameter

Data type

Direction

Description

group

string

in

A string that indicates the resource group:

  • UI — user interface resource
  • OM — business object method
  • other — resources created for an installation

When the group element is empty, NULL, or not present, all resources are returned.

rowSet

object

out

Row set of group information.

Columns returned in the rowSet object

columnOrdinal

columnName

Physical column

1

vchResourceId

onyx_resource_ml_view.resource_name

2

vchResourceDesc

onyx_resource_ml_view.resource_desc

3

active

onyx_resource_ml_view.record_status

Steps

Step data

Step type

Description

com

Validates input parameters

OnyxDM.DataManager

com

Retrieves a rowSet object listing all resources in the specified resource group

Remarks

The retrieveList search is case sensitive for the group input parameter.
