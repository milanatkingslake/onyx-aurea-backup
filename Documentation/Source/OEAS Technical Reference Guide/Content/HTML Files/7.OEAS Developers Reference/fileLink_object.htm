fileLink object

The fileLink object enables users to insert, retrieve, update, and delete file attachment link information for an owner object, such an incident, company, individual, workTicket, customerProduct, or literatureOrder.

The fileLink object has one key property, primaryId, that uniquely identifies the object instance. Once the object is created, the value of this property cannot be changed.

Methods

Method Description
delete Logically deletes a specified fileLink object instance.
insert Saves a new object instance.
retrieveCollection Returns a collection of object instances belonging to a specified owner object.
retrievelistbyEmailId Returns a list of fileLink objects owned by emailAttachment objects for a particular email message.
retrievelistbyOwner Searches the database by ownerId and returns a list of file attachment records.
retrieveListByOwnerExtMobile Retrieves a list of fileLink object based on owner, for mobiles
saveCollection Performs multiple actions at once: inserting, updating, and deleting objects, as appropriate.
update Saves modifications to an existing object.
