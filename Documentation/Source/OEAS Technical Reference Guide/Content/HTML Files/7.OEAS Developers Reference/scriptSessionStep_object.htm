scriptSessionStep object

scriptSessionStep object

The scriptSessionStep object identifies a script step accessed for a given script session.

The scriptSessionStep object has one key property, primaryId, that uniquely identifies the object instance. Once the object is created, the value of this property cannot be changed.

Methods

Method

Description

delete

Deletes the specified scriptSessionStep object

insert

Creates a new scriptSessionStep object

retrieve

Returns the specified scriptSessionStep object

retrieveCollection

Returns a collection of zero or more scriptSessionStep objects that match the search criteria

saveCollection

Performs multiple actions at once: inserting, updating, and deleting objects, as appropriate

update

Saves modifications to an existing scriptSessionStep object

Properties

Property

Description

completed

Specifies whether a step was completed by the user during the script session

insertBy

Contains the user ID of the Onyx user who initially saves the object

insertDate

Contains the time and date on which the object is initially saved

primaryId

Contains the unique ID assigned to the object when it is first saved to the database

scriptSessionId

Contains a foreign key to the scriptSession object that owns the scriptSessionStep

scriptStepId

Contains a foreign key to the step domain data associated with the scriptSessionStep object

sequenceNumber

Contains a number representing the sequential position of a step in a session

updateBy

Contains the user ID of the Onyx user who last saved modifications to the object

updateDate

Contains the time and date on which modifications to the object were last saved
