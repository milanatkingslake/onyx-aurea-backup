subscribableObject object

The subscribableObject object holds information about each transactional object type that can be subscribed to. This object has a compound primary key, consisting of the objectType and subscriptionServiceId properties. These keys allow the subscribableObject to have a set of instances for each subscription service supported by an implementation.

Description
