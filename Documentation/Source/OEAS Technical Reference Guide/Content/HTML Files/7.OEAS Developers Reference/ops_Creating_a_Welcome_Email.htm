Creating a welcome email

The creating a welcome email scenario contains the steps you must take to send an email welcoming new customers. This scenario explains how to modify the individual insert method to include a step that sends email. The scenario includes the following customizations:

  • Create a new literatureProduct that references a custom XSL stylesheet containing your welcome greeting boilerplate
  • Write a COM custom step component to retrieve an individual and insert a literatureOrder
  • Add a logical COM step to the individual insert method in the OED
  • Test the customization and install the component on the server
  • Restart the COM+ application

The Modifying a Method topic explains how to create a custom step component. This book focuses on the specifics of modifying a method for this customization.
