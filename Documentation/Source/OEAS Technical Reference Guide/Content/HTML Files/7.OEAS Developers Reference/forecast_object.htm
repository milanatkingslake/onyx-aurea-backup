forecast object

The forecast object enables users to delete, insert, retrieve, and update the forecast associated with a single incident, typically a sales incident.

The forecast object has one key property, primaryId, that uniquely identifies the forcast product in the database. Once the object is created, the value of this property cannot be changed.

Methods

Method Description
delete Deletes a specified forecast object instance
insert Persists an object to the database
retrieve Returns a collection of objects based on the search criteria supplied in the parameters XML
retrieveList Returns a list of forecasts matching the specified search criteria
update Persists modifications to an existing object instance
