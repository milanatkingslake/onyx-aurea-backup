multiSelect (scriptControlType)

multiSelect property

The multiSelect property of the scriptControlType object specifies whether the control supports multiple response selections at run time.

Data type

Boolean

Access

Read/write

Remarks

Some controls need specific design-time parameters in order to activate the multiSelect support. For instance, list controls that support drop-down functionality must have the string "dropdown=false" (without quotes) included in the defaultExtraData.
