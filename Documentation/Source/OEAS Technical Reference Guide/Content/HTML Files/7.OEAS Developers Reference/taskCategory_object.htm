taskCategory object

The Task feature's taskCategory object enables users to choose from a predefined list of valid categories when classifying a task object. Examples of taskCategory objects include Sales, Service, and Support. These categories are provided by default with OEAS, but you can customize OEAS to include other categories that suit your business needs.

The taskCategory object has one key property, primaryId, that uniquely identifies the taskCategory in the database. Once the object is created, the value of this property cannot be changed.
