insertReplace (assignmentAssociation)

insertReplace method

The insertReplace method of the ownerSurveyAssociation object creates or replaces an ownerSurveyAssociation object  for a specified owner object.

Parameters

Parameter

Direction

Data type

Content type

Description

ownerSurveyAssociation

in/out

object

all/keysOnly

A new ownerSurveyAssociation object on input.

Steps

The default steps for the insertReplace method of the ownerSurveyAssociation object are:

Step data

Step type

Description

com

Validates input parameters

OnyxDM.DataManager

com

Calls a pre-procedure to determine if an ownerSurveyAssociation object already exists for specified owner. If so, the existing association is deleted before the new one is inserted.
