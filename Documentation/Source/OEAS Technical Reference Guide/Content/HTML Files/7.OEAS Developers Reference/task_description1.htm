description1 (task)

description1 property

The description1 property of the task object contains user input text information for a single task object.

Data type

string (maximum length)

Access

Read/write

Remarks

If the user input text exceeds the limits of the description1 property, use the description2 property to store the remainder of the text. It is the client application's responsibility to place overflow from the description1 property into the description2 property.

The value of the description1 property is typically provided by end-users.
