reusable (literatureMessage)

reusable property

The reusable property of the literatureMessage object indicates whether the literatureMessage is displayed for reuse by other users.

Data type

boolean

Access

Read/write

Remarks

This property defaults to zero if not supplied on an insert. If true, the object can be displayed in the user interface for reuse by other users. The reusable property works in conjunction with the privateAccess property. If reusable is set to 1 (true) and privateAccess=1, then it is only reusable by the user who created it. If this is set to 1 (true) and privateAccess=0, then it is reusable by all users.
