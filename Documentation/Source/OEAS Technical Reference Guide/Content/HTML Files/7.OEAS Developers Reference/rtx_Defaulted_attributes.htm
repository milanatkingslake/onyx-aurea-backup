Defaulted attributes

Defaulted attributes are optional attributes for which the run time XML schema defines a default value. If the attribute is missing in the run-time XML, the consumer application should assume the default value.

For instance, if a scriptPrompt does not have a min attribute indicating a minimum value, a value of NULL is assumed for the min property.

This reduces the size of the run-time XML, but makes the consumer responsible for creating the attribute and assigning the default value.
