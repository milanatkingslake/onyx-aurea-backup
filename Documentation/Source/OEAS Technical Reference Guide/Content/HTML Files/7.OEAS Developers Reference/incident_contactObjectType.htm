contactObjectType (incident)

contactObjectType property

The contactObjectType property of the incident object contains a unique numerical value that identifies the type of object that the contactId property specifies.

Data type

i4

Access

Read/write

Remarks

When the caller supplies a contactId in the input parameters for the insert method, a value in the contactObjectType property is required. If there is no contactId, the value of this property can be NULL.

Valid values of the contactObjectType property include:

Valid Value

Object Type

1

individual

2

company
