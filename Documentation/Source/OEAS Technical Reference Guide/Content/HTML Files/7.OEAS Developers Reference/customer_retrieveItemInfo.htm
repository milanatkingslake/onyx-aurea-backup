retrieveItemInfo (customer)

retrieveItemInfo method

The retrieveItemInfo method of the customer object returns the customer type (individual or company) and customer information based on the primaryId. The method returns a rowSet object in the XML.

retrieveList standard method information

Retrieves information for one customer including customer type

Remarks

The returned rowSet object contains the columnDefinition for each of the columns that satisfy the search criteria. Customers that are marked as private are not returned, unless the service request is from the user who marked the customer object instance as private.

</parameters>
