Adding steps

Before adding steps, see adding parameters. You add steps to define the behavior of a method. In this case, you add two steps for each method. Both of the methods for this customization call the Onyx data validation step component followed by the Onyx data manager step component. The step attributes are the same for both methods except for the additional data attributes. The order that you create the steps in the Object Designer is important. Onyx recommends that you create the validate step first to make sure the input is correctly formed before calling other steps.
