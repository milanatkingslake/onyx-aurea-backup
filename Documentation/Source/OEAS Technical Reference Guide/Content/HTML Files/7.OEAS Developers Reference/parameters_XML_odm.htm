parameters XML (data manager)

parameters XML

The ODM performs the following processing steps on the parameters XML that it receives from the OTM:

  1. Extracts the application name, step object, step method, and step additional data and verifies that each is correct.
  2. Prepares the parameters XML for SQL generation. For instance, on an insert, ODM identifies the top-level object in each step call and creates a GUID for that object. It propagates the GUID and foreign keys to child objects and determines the object type of the parent object.
  3. Retrieves an XSL transformation and applies it to the parameters XML to create parameters that the ODM uses to call a generic stored procedure. An XSL transformation is a style sheet that describes how to change the structure of an XML document into an XML document with a different structure.
  4. ODM validates and parses the generated SQL before passing it as a stored procedure string, with parameters, to the Query Engine.
