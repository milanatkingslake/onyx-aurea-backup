Procedure Parameters
Parameters

The parameters collection is owned by the procedure entity. The collection contains metadata describing each parameter of the stored procedure, in the order of their appearance in the procedure call signature. Each parameter has the following attributes:

  • name - The name attribute contains the parameter name as specified in the physical stored procedure signature.
  • data type - The value for the data type is specified in the metadata as the eptType that is used by the Onyx Query Engine.

  • direction - The direction attribute indicates the direction of data flow into or out of the stored procedure. Valid values are in, out, and inOut.
  • default value - The default value attribute contains a vlue that is used when values are missing from the input XML stream.
  • size - The size attribute is only required for string data types and indicates the length of the procedure parameter in characters.
