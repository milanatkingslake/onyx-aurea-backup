incidentCategory object

The incidentCategory object contains domain data used for classifying an incident object. Examples of incidentCategory objects include Sales, Service, and Support. These categories are provided by default with OEAS, but you can customize OEAS to include other categories that suit your business needs.

The incidentCategory object has one key property, primaryId, that uniquely identifies the incidentCategory in the database. Once the object is created, the value of this property cannot be changed.
