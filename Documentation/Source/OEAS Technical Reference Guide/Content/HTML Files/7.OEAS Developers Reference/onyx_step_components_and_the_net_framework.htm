Onyx step components and the .NET Framework

The Onyx Transaction Manager (OTM) recognizes a step component that implements the IStepString interface as a step that accepts a string of XML rather than an instance of the IXMLDOMDocument interface. IStepString was created to facilitate integration with custom step components developed for the .NET Framework.

Likewise, the OTM exposes the IInternalStringTransactionManager interface, which accepts a string of XML from the step component instead of an instance of the IXMLDOMDocument interface. With the exception of how the XML is passed between step components and the OTM, these two interfaces function exactly as their counterparts, IStep and IInternalTransactionManager.
