retrieveListFavoritesByUser (customer)

retrieveListFavoritesByUser method

The retrieveListFavoritesByUser method of the customer object returns a list of favorite customers, which are individuals and companies for which the userId is set up as an internal contact of type "Favorite." The data is returned as a rowSet object in the XML.

Remarks

Only the firstName and lastName of the primary contact are returned for companies. All of the columns are returned for individuals.

This method uses the IndivFavoriteContactTypeId and CompFavoriteContactTypeId system parameters. You can modify these values using the OES Administration Workbench System Parameter Administration tool. The IndivFavoriteContactTypeId specifies which contact type is the favorite for the individual contact category. If the objectType input parameter is 1, the method returns all individuals where the userId is an internal contact and the contact type matches the IndivFavoriteContactTypeId. The CompFavoriteContactTypeId specifies which company contact type is the favorite for the company contact category. If the objectType input parameter is 2, the method returns all companies where the userId is an internal contact and the contact type matches the CompFavoriteContactTypeId. If the objectTypeinput parameter is 9, both sets are returned.

This method supports paging functionality. For more information, see the Paging functionality.
