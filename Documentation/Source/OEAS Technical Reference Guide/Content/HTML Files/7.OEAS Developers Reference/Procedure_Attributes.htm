Procedure attributes

When you select a procedure in the Object Designer, the attributes for that procedure display in the edit area.

The attributes are defined as follows:

Attribute Name

Description

ID

The globally unique identifier (GUID) assigned by the Onyx Enterprise Dictionary. This attribute is read-only.

Name

Type the name you want to assign to this procedure.

Description

Type the description for this procedure.

Installed By

Type the name of the vendor or application installing this object.

Onyx Version

The Onyx version number of this object. This attribute is read-only.

Custom Version

Type the custom version number you want to assign to this object.

Released

This attribute is for documentation purposes only. Check this box to document that the procedure is no longer under development.
