Embedded Objects
Embedded objects

An embedded object is an object that is a property of another object. It has a dataType of "object" and is identified, by its name, in the objectType element.

The following example shows the string returned by the DictionaryProxy GetObjectDefinition method call for the company object. The highlighted metadata calls out the phone object, which is embedded in the phones property of the company object.
