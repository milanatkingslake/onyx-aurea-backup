searchViewDetail Object

searchViewDetail object

The searchViewDetail object is a domain object representing searchable fields in a search view. The searchViewDetail belongs to a specific searchTable and searchView identified by the searchTableId and searchViewId properties, respectively.

The searchViewDetail object has one key property, primaryId, that uniquely identifies the searchViewDetail in the database. Once the object is created, the value of this property cannot be changed.
