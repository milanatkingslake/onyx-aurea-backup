resultFunctionCall (searchViewDetail)

resultFunctionCall property

The resultFunctionCall property of the searchViewDetail object contains the function call included in the select clause.

Data type

string

Access

Read/write

Remarks

The format of the resultFunctionCall is f(~0), where "f" is a database function. When constructing the SQL query, the calling client replaces the ~0 with the searchTable alias and fieldName values separated by a period.

For instance, suppose you want to return secondary_id instead of a uniquidentifier for a customer record. To do this, you can specify a function to format the GUID before returning it in the result.

If, for example:

  • fieldName is owner_id
  • searchTableAlias is B
  • resultFunctionCall is ofCustomerGetSecondaryFromPrimary(~0)

Then the resulting select clause would be:

SELECT ofCustomerGetSecondaryFromPrimary(B.owner_id)
