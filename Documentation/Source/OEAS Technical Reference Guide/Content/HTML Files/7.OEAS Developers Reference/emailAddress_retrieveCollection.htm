emailAddress retrieveCollection Method

The retrieveCollection method of the emailAddress object returns a collection of objects based on the search criteria supplied in the parameters XML.

retrieveCollection standard method information

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com
