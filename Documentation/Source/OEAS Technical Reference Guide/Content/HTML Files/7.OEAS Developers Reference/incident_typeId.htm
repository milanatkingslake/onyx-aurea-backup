typeId (incident)

typeId property

The typeId property of the incident object contains a predefined code that identifies the incident sub-type.

Data type

i4

Access

Read/write

Remarks

The typeID property references an existing incident type in the database. Incident categories own incident types, therefore the value in the typeId property must reference a child value of the incidentCategory for a given incident.

For example, if the incidentCategory is Sales (categoryId), your incident types (typeId) might be any of the following: General, Product Inquiry, Pricing, Small Account, Medium Account, and Large Account. These are default incident types. You can add your own using the Reference Table Administration tool.
