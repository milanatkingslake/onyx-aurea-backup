@loadedScriptSessionStepId

The loadedScriptSessionStepId attribute contains the primaryId of the transactional scriptSessionStep object loaded by the loadStep method.

XPath

/root/script/@loadedScriptSessionStepId

Cardinality

0..1 (optional)

Default value

NULL

Comments

This attribute identifies the scriptSessionStep returned by calling loadStep when there are multiple scriptSteps returned in the run-time XML. Both advanceStep and resumeSession also call the loadStep method.
