ITransactionManager interface

The Transaction Manager interface serves as the entry point to the OEAS. All client calls to the system must be made using the Execute method of this object.

Note: Onyx recommends that you use the Transaction Manager2 interface, because this newer interface offers control over session persistence.

This interface exposes a single method:

Method

Description

Execute

Makes a call to a method of a logical business object
