retrieveList (group)

retrieveList method

The retrieveList method of the group object returns a list of all group information.

Parameters

retrieveList standard method information

Parameter Data type Direction Description
active i4 in An integer that specifies whether to return active groups or inactive groups. If the active element of the input parameters XML is empty, NULL, or not present, all groups are returned, active and inactive. When active contains 1, all active groups are returned. If the element contains 0, inactive groups are returned. If active contains an invalid value, such as 2, the method succeeds, but no groups are returned.
rowSet object out Row set of group information.

Columns returned in the rowSet object

columnOrdinal columnName Physical column
1 groupId user_group_ml_view.group_code
2 description user_group_ml_view.group_desc
3 parentGroupId user_group_ml_view.parent_group_id
4 active user_group_ml_view.record_status

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieves information about groups
