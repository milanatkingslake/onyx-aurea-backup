networkUser (user)

networkUser property

The networkUser property of the user object is the user's network domain name and user name.

Data type

string (maximum length)

Access

Read/write

Remarks

The networkUser property is necessary for integrated Windows authentication, in which the user has a domain account on a local network and that account is mapped to an Onyx user account.

The format of this string is:

<domain name>\<user name>

The networkUser property must be unique for the site. You cannot have two users with the same networkUser string.
