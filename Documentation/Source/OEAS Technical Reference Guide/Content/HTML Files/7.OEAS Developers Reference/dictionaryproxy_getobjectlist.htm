GetObjectList (DictionaryProxy)
GetObjectList method

The GetObjectList method of the DictionaryProxy interface retrieves an XML string from the Onyx Enterprise Dictionary (OED) cache for a given logical application.

Syntax

    ApplicationName As String)

    As String

C#

Use with the .NET primary interop assembly version of this component, OnyxDictionaryCacheProxyPIA.

public string GetObjectList(

    string ApplicationName)

C# parameter descriptions

ApplicationName

[in] A string containing the logical application name that determines which subset of the cache to use for a query.

Return Value

This method returns an XML string from the OED cache for the given logical application. The string is composed of the businessObject collection. The returned XML structure is abbreviated and does not include child collection elements (properties and methods) of the businessObject elements.

Example

The following Visual Basic code demonstrates how to use the GetObjectList method to retrieve the OED businessObjects XML structure.

Visual Basic code sample

Dim strAppName, strObjectList

strAppName = "Onyx"

Dim proxy

Set proxy = CreateObject("OnyxDictionaryCacheProxy.Dictionary")

strObjectList = proxy.GetObjectList(strAppName)

If Err.Number <> 0 Then

End If

Set proxy = Nothing
