retrieveCollection (contactType)

retrieveCollection method

The retrieveCollection method of the contactType object returns a collection of objects based on the search criteria supplied in the parameters XML.

retrieveCollection standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieves a collection of contactType object instances

Example

Parameters

<parameters>

   <contactTypes objectType="contactType" collection="1">

      <contactType objectType="contactType">

         <primaryId/>

         <description/>

         <categoryId>1</categoryId>

         <groupTypeId/>

         <attributeId/>

         <active/>

      </contactType>

   </contactTypes>

</parameters>

parametersReturn

<parametersReturn>

   <contactTypes objectType="contactType" collection="1" content="all">

         <categoryId>1</categoryId>

         <groupTypeId>1</groupTypeId>

         <attributeId>1</attributeId>

         <active>1</active>

      </contactType>

   </contactTypes>

</parametersReturn>
