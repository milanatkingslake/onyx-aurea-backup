Customization tools

Several tools are available in the Onyx Enterprise Studio to help you customize the OEAS. The Object Designer is a design tool that helps you edit the data and metadata that define the logical business objects that are stored in the Onyx Enterprise Dictionary (OED). ONYX supports modifications to the OED only if they are made using the Object Designer.

This guide identifies several possible customization opportunities that you may want to consider. It also contains several scenarios, complete with sample code, to assist you in learning how to customize OEAS.

Before you attempt to customize OEAS, familiarize yourself with the tools in the Onyx Enterprise Studio, particularly the Object Designer. You should also be familiar with the following technologies and programming languages:

  • COM/COM+
  • XML
  • VB, VB.NET, C#, or C++
  • Microsoft XML DOM
  • MDAC
  • SQL
  • ActiveX Data Objects (ADO)
  • OLE DB
  • ASP
