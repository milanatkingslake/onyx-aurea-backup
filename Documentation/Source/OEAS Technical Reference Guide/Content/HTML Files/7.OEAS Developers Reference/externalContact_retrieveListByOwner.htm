retrieveListByOwner (externalContact)

retrieveListByOwner method

The retrieveListByOwner method of the externalContact object searches the database by ownerId and ownerType and returns a list of externalContact records.

Parameters

Parameter Direction Data type Content type Description
ownerId (required) in string n/a ID of the owner that owns the list of contacts
ownerType
(required)		 in i4 n/a Type of object that owns this list of contacts
includeMask in string n/a Identifies whether the phone number mask is displayed on the output or not
rowSet out object all(out) A rowSet structure containing the external contacts information

Columns returned in the rowSet object

columnOrdinal columnName Physical column Comments
1 contactPrimaryId external_contact.contact_id
2 contactSecondaryId individual.secondary_id or company.secondary_id
3 vchName individual.first_name, individual.last_name, or company.company_name
4 vchPhoneNumber phone.phone_number
5 vchFaxNumber phone.phone_number
6 vchCompanyName individual.company_name or company.company_name
7 vchTitleDesc individual.title_desc Null for company rows
8 vchEmailAddress individual.email_address or company.email_address
9 vchAddress1 address.address1
10 vchComment external_contact.comments
11 TypeOfContact external_contact.contact_object_id
12 iContactTypeId external_contact.contact_type_did
13 iSysContactId external_contact.external_contact_id
14 chContactTypeDesc contact_type_ml_view.contact_type_desc
15 customerType contact_type_ml.contact_type_desc Either 1 (individual) or 2 (company).

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieves a list of external contacts information

Remarks

The retrieveListByOwner search is case sensitive.

Owners include company and individual objects.
