primaryUnitPrice (forecastLine)

primaryUnitPrice property

The primaryUnitPrice property of the forecastLine object contains the price in the primary currency for the given product on the forecastLine.

Data type

float

Access

Read/write

Remarks

confidenceFactor x quantity ( primaryUnitPrice - primaryDiscount ) = forecastLine total

1.00 x 1 ( $2560.00 - 0 ) = $2560.00
