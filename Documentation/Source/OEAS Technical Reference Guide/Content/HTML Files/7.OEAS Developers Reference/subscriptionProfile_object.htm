subscriptionProfile object

The subscriptionProfile object relates an Onyx user to a subscriptionService, staging server, and sync server subsystem. This object has a composite primary key, the userId and subscriptionServiceId properties, that allows users to have multiple profiles when there is more than one subscription service available.

Methods

Method

Description

retrieveCollection

Returns objects that match search criteria

retrieveList

Returns a list of information about active subscription profiles

saveCollection

Performs multiple actions at once: inserting, updating, and deleting objects, as appropriate
