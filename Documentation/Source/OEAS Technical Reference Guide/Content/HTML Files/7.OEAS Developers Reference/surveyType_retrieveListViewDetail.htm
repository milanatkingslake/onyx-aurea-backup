retrieveListViewDetail (surveyType)

retrieveListViewDetail method

The retrieveListViewDetail method of the surveyType object returns information about the surveyTypes of the specified customer type. The result set mimics the searchViewDetail object.

retrieveList standard method information

Parameters

Parameter Direction Data type Content type Description

customerType

in

i4

-

The customerType is 1 for the individual or 2 for the company

rowSet

out

object

all

Contains the rowSet column definitions and the incident table rows matching with the owner and category criteria

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates the input parameters

OnyxDM.DataManager

com

Retrieves the collection of the surveyType object
