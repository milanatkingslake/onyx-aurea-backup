retrieveList

retrieveList method

The retrieveList method of the literatureTerms object searches the database and returns the complete list of payment terms for literature product orders.

Parameters

This method has one output parameter:

Element Data Type Direction Content Type Description

rowSet

object

out

all

A rowSet that contains all payment terms options for literature product orders.

Steps

The default step for this method is:

Step Step Type Description

OnyxDomainDataDAC.DomainData

com

Uses the wbospsgTerms stored procedure to obtain the complete list of payment terms for literature product orders.

Remarks

This method has no input parameters.

The columns returned in the rowSet object are:

Column Ordinal Column Heading Description

1

Terms ID

The unique identifier for the payment terms. Use this value in the terms property of a literatureOrder object.

2

Description

The user-readable description for the payment terms.
