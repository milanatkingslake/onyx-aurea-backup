retrieveListForStatus (subscriptionItem)

retrieveListForStatus method

The retrieveListForStatus method of the subscriptionItem object returns status information about a specific record that is subscribed by a specific user.

Remarks

The value of the isSubscribed column is calculated based on the syncAction:

  • When the syncAction is 0, 1, 3, or 5, isSubscribed contains 1.
  • When the syncAction is 2, 4, 6, 7 or 8, isSubscribed contains 0.
