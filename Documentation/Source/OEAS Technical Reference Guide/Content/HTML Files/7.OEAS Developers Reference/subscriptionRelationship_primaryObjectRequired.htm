primaryObjectRequired (subscriptionRelationship)

primaryObjectRequired property

The primaryObjectRequired property of the subscriptionRelationship object indicates whether the primary object in the relationship is automatically subscribed when the secondary object is subscribed.

Data type

boolean

Access

Read/write

Remarks

Unlike the defaultCascadeToPrimary property, this flag does not cause the sync server to call the cascade method on the primary object instance. Only the single primary instance is automatically subscribed.
