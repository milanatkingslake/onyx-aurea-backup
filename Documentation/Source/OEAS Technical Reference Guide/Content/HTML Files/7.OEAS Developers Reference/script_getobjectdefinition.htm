GetObjectDefinition (DictionaryProxy)
GetObjectDefinition method

The GetObjectDefinition method of the DictionaryProxy interface retrieves an object definition of a business object.

Syntax

Visual Basic/VBScript

Function GetObjectDefinition(

    ApplicationName As String,

    ObjectName As String)

    As String

Visual Basic/VBScript parameter descriptions

ApplicationName

[in] A string containing the logical application name that determines which subset of the cache to use for a query.

ObjectName

[in] A string that contains the name of the object.

C#

Use with the .NET primary interop assembly version of this component, OnyxDictionaryCacheProxyPIA.

public string GetObjectDefinition(

    string ApplicationName,

    string ObjectName)

C# parameter descriptions

ApplicationName

[in] A string containing the logical application name that determines which subset of the cache to use for a query.

ObjectName

[in] A string that contains the name of the object.

Return value

This method returns a string that contains the XML definition of the object.

Remarks

If the method fails, the number property of the error object contains a value equal to one of the predefined error constants that indicates the status of the method call.

Example

The following example demonstrates how to use the GetObjectDefinition method to return an object definition.

View the VBScript code sample

' Declare some variables

dim str_app, str_obj, str_objDef

str_app = "appname"

str_obj = "object"

' Call the cache through the proxy

dim proxy

set proxy = CreateObject("OnyxDictionaryCacheProxy.Dictionary")

str_objDef = proxy.GetObjectDefinition(str_app, str_obj)

if err.number <> 0 then

    ' Do something with the error object

    ' Do something with the contents of the string

end if
