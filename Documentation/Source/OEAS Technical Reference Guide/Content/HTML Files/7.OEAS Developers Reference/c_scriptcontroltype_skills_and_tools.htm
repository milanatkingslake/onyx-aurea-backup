Required skills and tools

Before working with this sample, familiarize yourself with the following technologies:

Technology

Rationale

SQL (optional)

You need to know this if you use a database management tool to add domain data to the OEDB.

Javascript

The sample creates a custom scriptControlType class in Javascript.

XML

You need to understand XML to use Onyx LBO Tester and to modify the XSL stylesheets

XSL

OEP renders the HTML control at runtime using an XSL stylesheet template.

You should also familiarize yourself with the following tools:

Tool

Rationale

Onyx LBO Tester (optional)

This is one of the ways to add the domain data to the OEDB.

Onyx Scripting designer (SWD)

Using this tool, you create a workflow script that tests your custom scriptControlType.

Onyx employee portal (OEP)

Using this application, you run the script and test the custom scriptControlType.
