scriptSession object

scriptSession object

The scriptSession object identifies the script being delivered, the customer being interviewed, and the user delivering the script. It gathers all the step, prompt, and response transactions for a given script session and persists the script state information.

The scriptSession object has one key property, primaryId, which uniquely identifies the object instance. Once the object is created, the value of this property cannot be changed.

Methods

Method

Description

Persists the transactional data for the current step and returns the domain data for the next step

Retrieves and restarts a previously completed or suspended script session

Retrieves a list of scriptSession instances for one or more specified agents delivering scripts

retrieveListByOwner

Returns summary information for all scriptSession objects belonging to the specified owner

saveCollection

Performs multiple actions at once: inserting, updating, and deleting objects, as appropriate

update

Saves modifications to an existing scriptSession object
