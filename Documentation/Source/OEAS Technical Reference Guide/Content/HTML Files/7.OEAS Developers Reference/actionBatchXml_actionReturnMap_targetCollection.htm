actionBatchXml_actionReturnMap_targetCollection
actionReturnMap/targetCollection

The actionReturnMap/targetCollection actionBatchXml element specifies a list data targets. Instead of a single object in the action, there can be a collection of objects in which case the XPath is broken down into object location pointed to by the itemPath and the name of the element in the elementPath.

Note: targetElementPath takes precedence over targetCollection if both are present.

Cardinality

0..1 (optional)

Default value

none
