image (incident)

image property

The image property of the incident object indicates whether or not an image is associated with a particular incident.

Data type

i1

Access

Read/write

Remarks

The OEAS does not support images by default. You need to provide additional customization for this feature to work properly.

Valid values for this property include:

0 = No image has been recorded for this incident.

1 = At least one image has been recorded for this incident.
