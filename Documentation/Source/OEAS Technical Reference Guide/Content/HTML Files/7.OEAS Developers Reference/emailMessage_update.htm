update (emailMessage)

update method

The update method of the emailMessage object saves modifications to a specified instance of an object.

update method standard information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com

Calls a pre-procedure that generates an integer value for the threadId when the element is null in the parameters XML

Updates an object instance
emailContact.saveCollection method Updates a collection of emailContact objects
emailAttachment.saveCollection method Updates a collection of emailAttachment objects

Remarks

If the emailMessage to be updated already has a threadId and the threadId element in the parameters XML is absent or null, the pre-procedure does not generate a new threadId.
