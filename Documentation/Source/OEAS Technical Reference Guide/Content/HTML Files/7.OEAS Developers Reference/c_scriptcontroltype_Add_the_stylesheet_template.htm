Add the stylesheet template

Every script control type must have an XSL template that defines the rendering of the control in the user interface. The template typically includes the following definitions.

  • A variable in which to store an actual or default response
  • The HTML to render the control on the page

For the Scripting feature, the script_delivery_step.xsl file in the Onyx Employee Portal (OEP) source files contains all XSL templates for script control types. In this chapter, the following topics describe how to modify this file for a custom script control type.

  • Locating and backing up_script_delivery_step.xsl file
  • Adding the template
  • Adding the template binding
