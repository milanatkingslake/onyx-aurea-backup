actionBatchXml sourceList
sourceList

The sourceList actionBatchXml element specifies a list of objects for which actions are performed. A sourceList object does not have to be a complete object. It only needs to contain the data that is used by an action. For an example of an actionBatchXml document using sourceList, see Processing an actionBatchXml document.

Note: A source list is not required to create an actionBatch. If your actionBatch contains a single sequence of actions that only need to be executed once, the sourceList element should be omitted.
