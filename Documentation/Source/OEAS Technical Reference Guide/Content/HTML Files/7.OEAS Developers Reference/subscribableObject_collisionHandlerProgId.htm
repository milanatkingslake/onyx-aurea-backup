collisionHandlerProgId (subscribableObject)

collisionHandlerProgId property

The collisionHandlerProgId property of the subscribableObject object contains the programmatic identifier of the COM component that resolves data collisions that occur when offline clients attempt to update records in the OEDB.

Data type

string

Access

Read/write

Remarks

OEAS supports multiple collision handler components. For any subscribable object, you can specify a specific collision handler to use.
