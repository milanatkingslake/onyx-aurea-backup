GetConfiguredProperties (DictionaryProxy)
GetConfiguredProperties method

The GetConfiguredProperties method of the DictionaryProxy interface retrieves the configured properties for a logical object. This method has been deprecated.

Syntax

Visual Basic/VBScript

ApplicationName As String,

ObjectName As String)

As Variant

Visual Basic/VBScript parameter descriptions

ApplicationName

[in] A string containing the logical application name that determines which subset of the cache to use for a query.

ObjectName

[in] A string that contains the name of the object.

C#

Use with the .NET primary interop assembly version of this component, OnyxDictionaryCacheProxyPIA.

public object GetConfiguredProperties(

    string ApplicationName,

C# parameter descriptions

ApplicationName

[in] A string containing the logical application name that determines which subset of the cache to use for a query.

ObjectName

[in] A string that contains the name of the object.

Return value

This method returns a two-dimensional array of variants that contains the configured properties for the specified object.

The C# version of this method returns a two-dimensional array as a System.Object type. To access the individual array elements the Object must be cast to Object[,] (an object array) and then either assign each element to a string or copy the value into a new string. For an example of this coding technique, see the C# code sample in the Example section of DictionaryProxy GetPropertyConstraints method.

Remarks

If the method fails, the number property of the error object contains a value equal to one of the predefined error constants that indicates the status of the method call.

Example
