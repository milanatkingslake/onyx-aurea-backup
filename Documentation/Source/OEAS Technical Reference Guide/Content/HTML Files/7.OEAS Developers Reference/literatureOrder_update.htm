update (literatureOrder)

update method

The update method of the literatureOrder object saves modifications to an existing literatureOrder object and its child literatureOrderLine objects in the database.

update standard method information

Description

com

Data validation step

OnyxDM.DataManager

Calls a pre-procedure to determine if action will orphan any non-reusable literatureMessage objects; if true, procedure deletes the given literatureMessage objects

Updates the literature order record

method

Insert/update/delete existing literatureOrderLine collection with the new collection
