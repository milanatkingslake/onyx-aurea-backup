retrieve (survey)

retrieve method

The retrieve method of the survey object returns a collection of objects based on the search criteria supplied in the parameters XML.

retrieve standard method information

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Retrieves an object instance

Example

Input Parameter:

<parameters>

   <survey objectType="survey">

      <primaryId>100000</primaryId>

   </survey>

</parameters>

Output Parameter:

<parametersReturn>

   <survey objectType="survey" content="all">

      <primaryId>100000</primaryId>

      <surveyTypeId>100000</surveyTypeId>

      <surveyTypeDescription>Lead Profiles</surveyTypeDescription>

      <name>Initial Contact</name>

      <description>Survey for new potential customers</description>

      <insertBy>sa</insertBy>

      <insertDate>2005-12-29 21:50:00</insertDate>

      <updateBy>sa</updateBy>

      <updateDate>2005-12-29 21:50:00</updateDate>

      <active>1</active>

      <surveyTypeActive>1</surveyTypeActive>

   </survey>

</parametersReturn>
