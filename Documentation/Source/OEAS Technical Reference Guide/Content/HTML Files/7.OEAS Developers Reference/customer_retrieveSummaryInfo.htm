retrieveSummaryInfo (customer)

retrieveSummaryInfo method

The retrieveSummaryInfo method of the customer object retrieves customer information based on the primaryId. The data is returned as a rowSet object in the XML.

retrieveList standard method information

Parameters

Parameter Direction Data type Content type Description

primaryId (required)

in

string

n/a

The primaryId of the customer

includeMask

in

string

n/a

Specifies how the mask data should be applied to the phone number and fax number (append, apply, none)

rowSet

out

object

all(out)

Customer information summary

Remarks

Customers that are either marked as private or as deleted are not returned.

Customers that are marked as private are not returned, unless the service request is from the user who marked the customer object instance as private.
