updateBy

updateBy property

The updateBy property of an object contains the user ID of the Onyx user who last saved modifications to the object.

This is a generic topic that applies to all properties with the same name. The remarks section may contain details specific to a given object.

Data type

string

Access

Read-only; for more information,  see Read Only attribute.

Remarks

The OEAS sets the updateBy property to the userId contained in the context XML when you update an object.
