Locating and backing up the script_controls.js file

If you do not know where the OEP source files are, use the following steps to locate script_controls.js.

To locate the OEP source files:

  1. Find the local path on the Properties page for the OEP Web site in the IIS administrative tool.
  2. Using Windows Explorer or a command prompt, navigate to the target file

To back up the original script_controls.js file:

  1. Make a copy of the original file
  2. Rename the copy to something like script_controls.old

To add a JavaScript class definition to the file, see Adding Javacript class definition to script_controls.js.
