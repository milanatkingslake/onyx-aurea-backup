retrieveListByWorkTicket (customer)

retrieveListByWorkTicket method

The retrieveListByWorkTicket method of the customer object returns a set of customer (individual and/or company) data associated with a particular work ticket. The data is returned as a rowSet object in the XML.

Remarks

The mask is maintained in the country table. For more information, see "Masking." Valid values for includeMask are:

  • none: phone output might appear as 8185551000
  • append: phone output might appear as 8185551000**MASK:(999) 999-9999 /x 99999
  • apply: phone output might appear as (818) 855-1000

Some restrictions apply to postal code masking. For more information, see "Creating Masks."
