url (individual)

url property

The url property of the individual object identifies the Uniform Resource Locator (URL) for the individual.

Data type

string (maximum length)

Access

Read/write

Remarks

The URL Address for the individual's Web site. The URL is the Internet address. URLs are short strings that identify resources in the web: documents, images, downloadable files, services, electronic mailboxes, and other resources. They make resources available under a variety of naming schemes and access methods such as HTTP, FTP, and Internet mail addressable in the same simple way.
