immediatePriority (literatureOrderLine)

immediatePriority property

The immediatePriority property of the literatureOrderLine object indicates that this literatureOrderLine object should be handled as Immediate priority.

Data type

boolean

Access

Read/write

Remarks

The immediatePriority property indicates whether the order line should be processed within the standard or high priority interval of the OPS subsystem. A value of 0 (zero) indicates the orderline is processed within the standard priority interval while a value of 1 indicates the orderline is processed within the high priority interval.
