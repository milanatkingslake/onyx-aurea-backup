Moving items in the navigation tree

You can rearrange the order of the following items in the navigation tree as desired:

  • Parameters
  • Methods
  • Object Keys
  • Properties
  • Sort Properties
  • Steps

Important: Steps are executed in the order displayed, top-to-bottom. Moving a step in the Object Designer affects the order in which it is executed by OEAS.

To move an item:

  1. In the navigation area, locate and select the item you want to move.

    Verify that you have the correct item highlighted before proceeding.

  2. Click or to move the item up or down.

    Once you move an item, its parent object appears in the navigation area with an asterisk ( * ) until the changes are saved.

  3. Click to save your changes.
