stepAdditionalData

The ReportNetStep uses the StepAdditionalData parameter of the IStep Execute method to determine which of its internal functions to call. The following code shows the StepAdditionalData where METHOD_NAME is replaced by the appropriate logical method name retrieveFolderItems or createFolder. If you use any other logical method name, it returns an error.

stepObject=reportNet;

stepMethod=METHOD_NAME;
