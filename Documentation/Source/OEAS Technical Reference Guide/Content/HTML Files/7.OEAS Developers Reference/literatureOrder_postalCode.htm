postalCode (literatureOrder)

postalCode property

The postalCode property of the literatureOrder object contains the postal code of the recipient of the literature order.

Data type

string (maximum length)

Access

Read/write

Remarks

This value is taken from the company or individual that placed the order referenced by the ownerId property.

The postCode property is used to indicate the postal code for the address and indicates the postal code for the individual or company that owns the record. The postCode should be set in the database without any masking characters.
