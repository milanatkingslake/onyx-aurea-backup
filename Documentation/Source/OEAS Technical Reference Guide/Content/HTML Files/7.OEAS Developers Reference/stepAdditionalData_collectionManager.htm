stepAdditionalData (OnyxCollectionManager)

stepAdditionalData

The stepAdditionalData contains the design time information for the OnyxCollectionManager step to determine the object methods and collection node on which to operate. If you add the OnyxCollectionManager step to a method, provide the necessary stepAdditionalData for the step.

Example

stepAdditionalData contains...

stepObject=workTicket;

stepMethod=saveCollection;

dataPath=workTickets

The stepObject value indicates to use the workTicket object methods (insert, update, and delete) to perform the saveCollection. The dataPath value indicates to use the workTickets collection node as input to the saveCollection. This node contains the object instances to be operated on. The dataPath also indicates that the results of the method are in the workTickets node in the parametersReturn output XML.
