Obtain data from the step package

After you verify that all expected XML nodes exist, your step component can begin its processing. For this example, we want to obtain the company name from the step package. The step component then sends this data to a hypothetical external database to look up the company's stock ticker symbol, which is then returned to the application in the XML DOM.
