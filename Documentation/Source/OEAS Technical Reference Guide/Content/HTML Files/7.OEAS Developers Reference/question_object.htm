question Object

question object

The question object contains a question that users can attach to a survey. Questions are reusable in multiple surveys. You attach a question to a survey using the surveyQuestion object. To specify a question, administrators provide the text and type of the question.

Methods

Method

Description

retrieveCollection

Returns a collection of question objects based on the search criteria supplied in the parameters XML

saveCollection

Performs multiple actions at once: inserting, updating, and deleting question objects, as appropriate

Properties

Property

Description

active

Specifies whether the domain data the object provides is available to the user interface

insertBy

Contains the user ID of the Onyx user who initially saves the object

insertDate

Contains the time and date on which the object is initially saved

primaryId

Contains the unique ID assigned to the object when it is first saved to the database

questionType

Contains an enumeration value that characterizes the valid response type for the question

text

Contains the words that form the question

updateBy

Contains the user ID of the Onyx user who last saved modifications to the object

updateDate

Indicates the time and date on which modifications to the object were last saved
