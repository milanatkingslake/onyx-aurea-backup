shipDate (literatureOrderLine)

shipDate property

The shipDate property of the literatureOrderLine object contains the date when the order items were shipped to the customer.

Data type

dateTime

Access

Read/write

Remarks

The Onyx Processing Server updates this property after it processes a given literatureOrderLine.

The dateTime data type is formatted as follows:

YYYY-MM-DD hh:mm:ss

where:

YYYY = year

MM = month

DD = day

hh = hour

mm = minute

ss = second

Valid values for the dateTime data type range from 1900-01-01 00:00:00 to 9999-12-30 23:59:59.
