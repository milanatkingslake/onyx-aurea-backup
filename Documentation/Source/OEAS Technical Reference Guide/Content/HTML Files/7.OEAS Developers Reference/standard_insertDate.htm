insertDate property

The insertDate property of an object contains the time and date on which the object is initially saved.

This is a generic topic that applies to all properties with the same name. The remarks section may contain details specific to a given object.

Data type

dateTime

Access

Read-only; for more information,  see Read Only attribute.

Remarks

The insertDate property is set only once: when the object is first saved. Attempts to change this value are ignored. The insertDate value is created by the database during the initial insert. The database obtains the date value from the database server clock. The database converts the date value to a UTC date before saving it.

The dateTime data type is formatted as follows:

YYYY-MM-DD hh:mm:ss

where:

YYYY = year

MM = month

DD = day

hh = hour

mm = minute

ss = second
