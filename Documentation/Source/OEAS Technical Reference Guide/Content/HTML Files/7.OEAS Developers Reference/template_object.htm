template object

template object

The template object is a domain object used to retrieve, insert, update and delete quote templates from the database. Quote templates are used for generating hard copy quotes that can be given to customers.

Methods

Method

Description

retrieveCollection

Returns a collection of objects based on the search criteria supplied in the parameters XML

saveCollection

Performs multiple actions at once: inserting, updating, and deleting objects, as appropriate

Properties

Property

Description

active

Indicates whether the domain data the object provides is available to the user interface

description

The description for the given quote template document

destinationAppCode

The type of application that is used to generate the physical quote document

fileName

The file name for the physical quote document

insertBy

The user ID of the Onyx user who initially saves the object

insertDate

The time and date on which the object is initially saved

name

The user-friendly template name

onyxAppName

The name of the Onyx user interface that can generate quotes using the quote template

onyxFormName

The name of the form in the Onyx user interface that uses the quote template

onyxTimestamp

Facilitates concurrency control with the Onyx Enterprise Database

primaryId

A unique ID assigned to the object when it is first saved to the database

templateTypeCode

The type of the quote template

updateBy

The user ID of the Onyx user who last saved modifications to the object

updateDate

The time and date on which modifications to the object were last saved
