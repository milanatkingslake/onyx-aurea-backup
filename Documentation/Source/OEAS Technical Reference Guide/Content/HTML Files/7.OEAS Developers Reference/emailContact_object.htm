emailContact object

The emailContact object records the email addresses that are inserted with the owning emailMessage object instance.

Methods

Method Description
retrieveCollection Retrieves a collection of specified objects
saveCollection Saves a collection of specified objects

Properties

Property Description
contactIsResolved A value that indicates whether the contact is resovled
displayName The friendly name displayed for the email address
emailAddress The email address for the contact recorded when the email was sent
insertBy The user ID of the Onyx user who initially saves the object
insertDate The time and date on which the object is initially saved
onyxTimestamp Facilitates record locking with the Onyx Enterprise Database
ownerId Identifies the parent object of the child object
primaryId The ID assigned to the object when it is first saved to the database.
privateAccess Indicates whether the object is visible to the public
readOnlyAccess Indicates that a user has marked the record as read-only
typeId The type of email contact
updateBy The user ID of the Onyx user who last saved modifications to the object
updateDate The time and date on which modifications to the object were last saved
