keyWords (incident)

tags property

The tags property of the incident object contains the tags associated with the incident record.

Data type

string (25)

Access

Read/write

Remarks

Tags are terms that users use to classify records so that they can later find all records that share that classification. Users can perform database searches or generate reports based on tags.

Each tag has a 25 character limit, but you can have multiple tags of 25 characters or less up to a limit of 255 characters. The list of tags is semicolon separated.
