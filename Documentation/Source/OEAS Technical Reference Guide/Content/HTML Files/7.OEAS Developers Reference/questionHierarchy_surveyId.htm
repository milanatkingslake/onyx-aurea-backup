surveyId (questionHierarchy)

surveyId property

The surveyId property of the questionHierarchy object contains the primaryId of the survey for the questionHierarchy object.

Data type

i4

Access

Read/write

Remarks

The questionHierarchy object derives its relationship to the survey object from the surveyQuestion object. The surveyQuestion object contains the survey primaryId in its surveyId property which is related to the questionHierarchy surveyId property. This derivation leverages the existing constraints defined by relationships on the surveyQuestion object.
