regionName (region)

regionName property

The regionName property of the region object returns the display string for the country.

Data type

string (maximum length)

Access

Read/write

Remarks

The regionName is the commonly referred to name for an area of a country. There is a unique mapping between a regionCode and a regionName. You can configure the regionName value in the REGION table using the Table administration tool.

An example of a regionCode and a regionName for British Columbia is: 

<regionCode>BC</regionCode>

<regionName>British Columbia</regionName>
