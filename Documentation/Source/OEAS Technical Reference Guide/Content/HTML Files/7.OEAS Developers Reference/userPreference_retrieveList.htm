retrieveList (userPreference)

retrieveList method

The retrieveList method of the userPreference object searches the database by userId and returns a list of the user's preference records.

Retrieves a list of user preferences

Remarks

When you supply only a userId, this method returns a complete list of that user's preferences. When you supply a preference name along with a userId, the method returns information only for the specified preference for that user. Because a user can have multiple UI profiles, including a preference name in the method call can return multiple rows.
