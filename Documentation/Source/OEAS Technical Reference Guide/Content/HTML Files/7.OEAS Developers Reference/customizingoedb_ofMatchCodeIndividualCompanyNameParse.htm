ofMatchCodeIndividualCompanyNameParse

The ofMatchCodeIndividualCompanyNameParse function parses an individual record’s company name for the portion that is used in the match code. The opMatchCodeIndividualInsert and opMatchCodeIndividualUpdate stored procedures call it when generating a match code for an individual.

Input Parameters

The following table describes the input parameters for ofMatchCodeIndividualCompanyNameParse.

Column

Parameter Name

Parameter Description

Data Type

Default Values and Comments

1

@company_name

The name of the individual’s company

NVARCHAR(255)

 

Return Value

  • The company name portion of an individual’s match code

Remarks

Company name match code examples:

  • Onyx Software Corporation returns a value of “OnSC”
  • Microsoft Corporation returns a value of “MiC*”
