sourceId (customerProduct)

sourceId property

The sourceId property of the customerProduct object contains a domain data ID for the source of the product.

Data type

i4

Access

Read/write

Remarks

The sourceId identifies the channel through which the customer acquired the product. The database enforces an integrity constraint to use a valid ID in the database. You can configure sourceId data using the Onyx Enterprise Studio Reference Table Administration tool. For more information on Reference Table Administration, see the Adminstrator's Reference.
