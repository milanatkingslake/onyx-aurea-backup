detailAccess (group)

detailAccess property

The detailAccess property of the group object is a bitmask that specifies group access to functions of the Onyx Employee Portal (OEP).

Data type

i4

Access

Read/write

Remarks

The role and resource objects are used in OEAS to support group access control; therefore, OEAS does not use the detailAccess property. It is used to maintain compatibility with Onyx Employee Portal—Windows client.
