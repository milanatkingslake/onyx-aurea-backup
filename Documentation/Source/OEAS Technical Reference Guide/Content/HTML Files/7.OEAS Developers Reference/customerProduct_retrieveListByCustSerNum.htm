retrieveListByCustSerNum (customerProduct)

retrieveListByCustSerNum method

The retrieveListByCustSerNum method of the customerProduct object returns a partial list of the customerProduct object's productIds based on ownerId and serialNumber.

Remarks

The retrieveListByCustSerNum search is not case sensitive.

The SerialNumberSearchAsLike system parameter enables this method to return a list of customer products that match a partial serialNumber. When this system parameter is set to 'Y', this method supports wild card substitution for the serialNumber parameter. For more information on wild card substitution, see the retrieveList method (standard behavior) topic.
