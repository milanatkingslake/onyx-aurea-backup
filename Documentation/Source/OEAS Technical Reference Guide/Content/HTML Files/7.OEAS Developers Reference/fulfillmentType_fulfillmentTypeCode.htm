fulfillmentTypeCode (fulfillmentType)

fulfillmentTypeCode property

The fulfillmentTypeCode property of the fulfillmentType object identifies the fulfillmentType object.

Data type

ui1

Access

Read/write

Remarks

The administrator that creates the fulfillmentType object supplies this code and is responsible for ensuring its uniqueness. The administrator sets the fulfillmentTypeCode when she first creates the object with the OPS Administration tool. This code must be unique and must not be changed after the object has been created.
