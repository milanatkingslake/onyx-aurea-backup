stepAdditionalData (OnyxTransformStep)

stepAdditionalData

The stepAdditionalData contains design time information for OnyxTransformStep that specifies the location of the XSL template to use and the node within the parameters XML to use as input for the XSL transform.

Example

The following stepAdditionalData is defined specifically for OnyxTransformStep as a com step in the actionBatch.executeForEMS method.

templatePath="c:\program files\onyx\appserver\onyx\templates\OnyxEMSIncidentInsert.xsl";

escapedNode="stepPackage/parameters/actionBatchXml"

The templatePath is either the absolute path of the template (as shown above) or a relative path to the location of the dllhost.exe file on the application server (typically, c:\windows\system32).

The escapedNode specifies the parameters XML node that the XSL transform modifies. The contents of this node must be escaped XML so that it can be embedded in the step package XML. OnyxTransformStep converts the specified node to regular XML. After the node is transformed, OnyxTransformStep converts the XML reserved characters (<,>, and /) of the node back to escaped characters. When the escapedNode is blank, nothing is unescaped.
