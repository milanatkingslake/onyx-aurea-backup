code4 (incident)

code4 property

The code4 property of the incident object contains one of four possible resolution code fields for an incident.

Data type

i4

Access

Read/write

Remarks

Resolution codes enable searches by categorizing solutions to incidents. It is used to provide choices to the end-user.

The code4 property references an existing resolution code type in the database. Incident categories own resolution code types, therefore the value in the code4 property must reference a child value of the incidentCategory for a given incident.
