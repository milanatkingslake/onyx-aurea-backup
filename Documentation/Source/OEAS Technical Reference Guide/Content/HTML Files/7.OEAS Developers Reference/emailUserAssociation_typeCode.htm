typeCode (emailUserAssociation)

typeCode property

The typeCode property of the emailUserAssociation object specifies the type of email association.

Data type

i4

Access

Read/write

Remarks

The values of the typeCode property, listed below, indicate the relationship of the associated user object instance to an emailMessage object instance.

Value Description
1 Related to the email, but not a recipient or sender
2 Recipient or sender of the email
3 A user who sent the email from an emailAddressBookItem address
