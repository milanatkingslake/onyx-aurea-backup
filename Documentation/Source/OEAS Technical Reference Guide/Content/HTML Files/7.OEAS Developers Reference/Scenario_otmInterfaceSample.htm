Scenario (OTM Interface Sample)

Scenario

This sample illustrates how to use the business logic in the OEAS by calling its single entry point--the Execute method of the ITransactionManager interface. If you are developing a client application to access the OEAS business logic, you will have to use the OTM Execute method. In this example, we insert a customer record in the database using this method. To do so, you must understand Onyx authentication and XML input parameters.

This code sample explains how to get Onyx authentication, construct XML input parameters, and verify a successful transaction to insert a company record in the database. The sample illustrates the communication from the perspective of the OEAS and does not attempt to explain client application development.
