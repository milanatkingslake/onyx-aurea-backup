enumerationItemPath (searchViewDetail)

enumerationItemPath property

The enumerationItemPath property of the searchViewDetail object contains the XPath to the enumeration item.

Data type

string (maximum length)

Access

Read/write

Remarks

This property specifies the XPath of the enumerated item(s) within the returnXML of the enumerationMethod call. For example, the region retrieveListEnumeration method returns a rowSet where each enumeration item is contained within a row. The following shows the XPath expression that identifies each row.

//parametersReturn/rowSet/rows/row
