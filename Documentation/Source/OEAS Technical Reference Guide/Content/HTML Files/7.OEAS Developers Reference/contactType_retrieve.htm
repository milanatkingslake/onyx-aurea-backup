retrieve (contactType)

retrieve method

The retrieve method of the contactType object returns a specific object based on the primary key.

retrieve standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieves a specified contactType object

<parameters>
