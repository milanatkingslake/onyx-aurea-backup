opMatchCodeAddressUpdatePost

The opMatchCodeAddressUpdatePost procedure updates the address portion of an individual or company match code. This procedure should be enabled on the address.saveCollection method for the insert and update actions when implementing the match code feature.

Remarks

The address information for the match code will be retrieved in the stored procedure to support partial updates.

This procedure will update out the address portions of the match code when the primary address is updated. A match code will exist even if called during an address insert.
