Deleting items in the OED

Warning: Prior to deleting an item, determine how the delete will impact other metadata or the user interface. Deleting an item with critical dependencies can have adverse effects.

To delete an item:

  1. In the navigation area, locate and select the item you want to delete.

    Verify you have the correct item selected before proceeding.

  2. Click to mark the item for deletion.

    The item name appears with a strike-through font in the navigation area (For example: newProperty1), and an asterisk ( * ) appears on the business object on which the item appears. An asterisk appears next to any item that has been modified until it has been saved.

  3. Click to save your changes.

To undelete an item:

  1. In the navigation area, locate and select the item you want to undelete. It is distinguished in the navigation area by the strike-through font of its name. For example: newProperty1.
  2. Click to unmark the item for deletion.

    The item name returns to normal.

You can undelete any item marked for deletion so long as it has not yet been saved.
