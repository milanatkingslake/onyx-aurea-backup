retrieveCollectionBatch (literatureOrder)

retrieveCollectionBatch method

The retrieveCollectionBatch method of the literatureOrder object returns a collection of literatureOrder objects. It searches the database for literatureOrder objects that have one or more literatureOrderLines with the status of "In Process" and that have the same literatureOrderBatchId.

retrieveCollection standard method information

Parameters

On input, the parameter is a collection object consisting of one object. This object must contain a GUID value in the literatureOrderBatchId element. All other element values are ignored.

On return, the collection object consists of objects that satisfy the search criteria. If no objects meet the search criteria, this method returns an empty collection object. No error is returned in this case.

Remarks

The OnyxPSSysDocStep.SysDoc step is a custom step that automatically generates system documents as Onyx Processing Server (OPS) processes orders. When batch processing of orders starts, OPS calls the retrieveCollectionBatch method. This step inserts additional literatureOrders and literatureOrderLines for system documents. A batch system document is added to the batch as a new order that contains one order line per batch system document. An order system document is added as an order line to each existing order in the batch.
