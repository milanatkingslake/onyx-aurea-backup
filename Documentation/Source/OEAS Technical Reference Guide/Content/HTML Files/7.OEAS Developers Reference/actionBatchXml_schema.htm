Element

This column contains the local name of each XML node in actionBatchXml. Indentation signifies hierarchy in the XML structure.

Cardinality

The value in this column indicates how many of the given nodes are expected in actionBatchXml. One (1) indicates that the node is required. One-to-many (1..*) indicates that at least one occurrence of the node is required. Zero-to-one (0..1) and zero-to-many (0..*) indicate optional nodes. Required nodes can be children of optional nodes; in this case, the child is required only if the parent is present.

Comments

This column contains notable information about the node. For a general description of the node, see the detailed topic by clicking on the local name in the first column.
