delete (alert)

delete method

The delete method of the alert object deletes the specified alert object instance.

delete standard method information

Steps

The default steps for the delete method of the alert object are:

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Logically deletes an alert object instance from the database
