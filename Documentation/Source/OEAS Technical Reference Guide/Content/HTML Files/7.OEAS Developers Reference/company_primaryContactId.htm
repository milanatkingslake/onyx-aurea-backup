primaryContactId (company)

primaryContactId property

The primaryContactId property of the company object reflects the GUID of the primary contact of the company.

Data type

string (maximum length)

Access

Read/write

Remarks

The primaryContactId property is used to indicate the GUID assigned to the individual who is considered the primary contact for the company. The primaryContactId is the same value that is the primaryId for the individual if the person is someone other than the owner. If no contact exists this value is 0 (zero). The primary contact is the single individual who is most closely associated with the company.
