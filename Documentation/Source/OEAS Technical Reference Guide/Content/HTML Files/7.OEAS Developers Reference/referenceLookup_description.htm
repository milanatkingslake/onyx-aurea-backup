description (referenceLookup)

description property

The description property of the referenceLookup object contains a description of the reference parameter used by the instance of the referenceLookup object.

Data type

string (maximum length)

Access

Read-only; for more information, see Read Only attribute

Remarks

This is a denormalized property and can only be updated through the referenceParameter object.

It also supports valid value lists created using the Reference Table Administration tool that contain code values instead of ID values. An example of this use is the list created for individual title codes, which includes PRES, VP, and CFO.
