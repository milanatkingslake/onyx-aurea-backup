Creating a User and Password (OTM Interface Sample)
Creating a user and password

Before you can run the sample code, you need to create a new Onyx user in the Onyx Enterprise Application Database. Use OES Security Administration to create a new user.

To access the Administration Workbench, substitute the name of the server running OEAS for <serverName> in the following URL:

http://<serverName>/onyxenterprisestudio_onyx/
