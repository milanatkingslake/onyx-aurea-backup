Demonstration

This demonstration assumes that you have made the modifications to the OED as outlined in the previous Steps topic for Adding a Method. To demonstrate the success of our modifications, we call the keyWord retrieveListByUser method to verify that it returns the keyWord records that the current user entered previously. You may need to insert some incident records to add key words for the current user. For testing purposes, you may want to adjust the TagIncludeCount system parameter. You can use Onyx Enterprise Studio System Parameter Administration to adjust this value. The stored procedure in this sample only returns keyWord records that have a use count equal to or greater than the system parameter value.

We call the Onyx Transaction Manager (OTM) with the following XML parameters.

<parameters></parameters>

The returnXml of the retrieveListByUser method includes the key words entered by the current user.
