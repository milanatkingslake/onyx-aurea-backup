actionBatchXml_sourceListMap_targetCollection
sourceListMap/targetCollection

The sourceListMap/targetCollection actionBatchXml element specifies a list of data targets. Instead of a single object in the action, there can be a collection of objects in which case the XPath is broken down into the object location that itemPath and the name of the element in the elementPath point to.

Note: The targetElementPath element takes precedence over targetCollection if both are present.

Cardinality

0..1 (optional)

Default value

none
