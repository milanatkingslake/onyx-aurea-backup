workTicket object

The workTicket object enables users to track customer support and product quality control issues. A workticket object instance can be associated with incident, company, and individual object instances, as well as other workTicket object instances.

The workTicket object has one key property, primaryId, that uniquely identifies object instances in the database. Once the object is created, the value of this property cannot be changed.
