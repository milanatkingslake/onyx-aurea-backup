logMarkSequenceNumber (workNoteDetail)

logMarkSequenceNumber property

The logMarkSequenceNumber property of the workNoteDetail object contains an ordinal number for the set of workNoteDetail items that share the same log mark.

Data type

i4

Access

Read/write

Remarks

The system generates the logMarkSequenceNumber based on the data sent in and existing records in the database. The logMarkSequenceNumber and sequenceNumber combination is unique among workNoteDetail objects belonging to a given workNoteHeader master object but is not unique across workNoteHeader objects.

This property allows for sorting items within a log mark in ascending order while sorting the log marks themselves in descending order.

The caller can pass in a value for logMarkSequenceNumber or leave it blank (NULL), in which case OEAS defaults the value to the next sequence number within the logmark.

If you pass in a logMarkSequenceNumber value, OEAS increments the sequenceNumber from the maximum sequenceNumber already present for workNote Detail object instances that have the specified logMarkSequenceNumber.
