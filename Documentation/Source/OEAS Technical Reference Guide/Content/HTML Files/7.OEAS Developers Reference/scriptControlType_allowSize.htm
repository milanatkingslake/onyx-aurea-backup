allowSize (scriptControlType)

allowSize property

The allowSize property of the scriptControlType object specifies whether the control supports custom sizing.

Data type

Boolean

Access

Read/write

Remarks

Use this property to specify whether a control supports a size attribute. The meaning of size depends on and is implemented by the script control. The following table lists some typical uses of the size attribute.

Typical use

Setting the width of a textBox control

Setting the default number of choices displayed in a listBox

Setting the default number of choices displayed in a list control (country, region, and so on)

Setting the default number of lines displayed in a multiLineTextBox

Note: The effectiveness of this property can depend on other control parameters such as the dropdown parameter (defaultExtraData) for certain list controls.
