retrievePartnerListByAccess (customer)

retrievePartnerListByAccess method

The retrievePartnerListByAccess method of the customer object returns the customer data for a partner by the list of user or partner group access IDs.

Parameters

Parameter Direction Data Type Content type Description

partnerGroupAccessIdList

in

string

n/a

A semicolon-delimited list of partnerGroupAccess.primaryId values

userAccessIdList

in

string

n/a

A semicolon-delimited list of userAccess.primaryId values

includeMask

in

string

n/a

Identifies whether the phone number mask is displayed on the output or not

rowSet

out

object

all

Contains the rowSet column definitions and the incident table rows matching with the owner and category criteria

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates the input parameters

OnyxDM.DataManager

com

Retrieves the information for customers
