ownerQuestionResponse Object

ownerQuestionResponse object

The ownerQuestionResponse object records a customer's response to a survey question. Literally, it relates a questionId and responseId to a customer. A question can have more than one response associated with it. Customers can give different responses to the same question at different times, however, a duplicate response to the same question fails in the database. The ownerQuestionResponse object constrains you to using a valid questionResponse pair.
