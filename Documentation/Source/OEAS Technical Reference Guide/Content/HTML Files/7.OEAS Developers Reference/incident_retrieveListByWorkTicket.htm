retrieveListByWorkTicket (incident)

retrieveListByWorkTicket method

The retrieveListByWorkTicket method of the incident object returns a rowSet of incident information for each incident object instance that is associated with the workTicket object instance specified on input.

retrieveList standard method information

Parameters

Parameter Direction Data type Content type Description

primaryId (required)

in

string

n/a

The unique identifier of a workTicket object instance that is associated with one or more incident object instances

categoryIdList (required)

in

string

n/a

A required semicolon-delimited integer list that identifies the top-level type of incident, such as Sales, Service, or Support, that you want to return

rowSet

out

object

all(out)

Row set of information about incident object instances
