enterSessionParams (OTM Interface Sample)
Enter your parameters

In the installed sample Web application, use this form to input the valid parameters for your environment and get a session ID. OnyxUserName, Password, ApplicationName, and SiteID are required and validated. The values you use here must exist already in the database. SourceName is a required parameter but is not validated. It can be any value including an empty string.
