subscriptionRelationshipRole object

The subscriptionRelationshipRole object provides role-based subscription rules for a specific relationship between two subscribable objects. User roles are created using the Security Administration tool.

The contents of the cascadeToPrimary, cascadeToSecondary, and filterExpression properties override the default values provided by the subscriptionRelationship object for the users who belong to a role associated with a subscriptionRelationshipRole object instance.

Methods

Method

Description

retrieveCollection

Returns objects that match search criteria

retrieveListByUser

Retrieves subscription rules applied to roles that the specified user is a member of

saveCollection

Performs multiple actions at once: inserting, updating, and deleting objects, as appropriate

Properties

Property

Description

active

Specifies whether the domain data the object provides is available to the user interface

cascadeToPrimary

Indicates whether the role specifies a cascading subscription to the primary object of the relationship when the secondary object of the relationship is subscribed

cascadeToSecondary

Indicates whether the role specifies a cascading subscription to the secondary object of the relationship when the primary object of the relationship is subscribed

description

Contains text that describes the subscriptionRelationshipRole object instance

filterExpression

Contains an expression used in a WHERE clause that limits the automatically subscribed secondary object instances according to the roles that the subscribing user is a member of

insertBy

Contains the user ID of the Onyx user who initially saves the object

insertDate

Contains the time and date on which the object is initially saved

name

Contains the name of the subscriptionRelationshipRole

primaryId

Contains the unique ID assigned to the object when it is first saved to the database

roleId

Contains the identifier of the role that is associated with the subscriptionRelationship object

subscriptionRelationshipId

Contains the identifier of the subscriptionRelationship object instance that the subscriptionRelationshipRole is associated with

updateBy

Contains the user ID of the Onyx user who last saved modifications to the object

updateDate

Contains the time and date on which modifications to the object were last saved
