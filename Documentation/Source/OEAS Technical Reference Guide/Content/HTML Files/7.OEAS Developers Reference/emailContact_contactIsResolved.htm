contactIsResolved (emailContact)

contactIsResolved property

The contactIsResolved property of the emailContact object specifies whether a specific email address of the contact has been determined.

Data type

boolean

Access

Read/write

Remarks

Valid values for this property are:

false (0): The email address for the contact is not resolved.

true (1): The email address for the contact is resolved.

The client application can call the retrieveListResolvedAddresses method to get a list of possible recipients for each of the recipient strings specified by an end user. When an end user chooses a contact from the list or manually enters a valid email address, the client application can use the contactIsResolved property to mark the contact resolved. When an end user saves an email message as a draft and subsequently adds more recipients later, the client can call retrieveListResolvedAddresses and input only the newly-supplied recipients.

When you insert a collection of emailContacts with an emailMessage object instance, the client application must set the contactIsResolved property for each emailContact instance.
