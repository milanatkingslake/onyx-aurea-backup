insert (searchQuery)

insert method

The insert method of the searchQuery object persists an object to the database.

insert standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Inserts an object instance

Remarks

The insert method automatically inserts a zero (0) in the read_only_access column of the database record inserted for a searchQuery object instance. This action is based on a literal type logical-to-physical map, which specifies a non-null database column that has no corresponding property on the logical object.
