saveCollection (reminder)

saveCollection method

The saveCollection method of the reminder object Inserts, updates, and/or deletes a collection of specified objects.

Parameters

Parameter

Direction

Data type

Content type

Description

reminders

inOut

object

any(in) keysOnly(out)

Contains the reminder information to be inserted, updated, or deleted

Steps

Step data

Step type

Description

OnyxCollectionManager.CollectionManager

com

Forms individual standard method calls for each object instance in the parameters and executes them through the internal OTM

Remarks

Unlike the saveCollection standard method, this saveCollection uses the OnyxCollectionManager component. The main difference is that OnyxCollectionManager wraps each object action in the parameters in a separate transaction.

Example

This example shows a saveCollection that inserts reminder objects for two different incidents.
