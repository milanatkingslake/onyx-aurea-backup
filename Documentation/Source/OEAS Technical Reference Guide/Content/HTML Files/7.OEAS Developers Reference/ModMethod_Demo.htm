Demo (Modify Method)
Demonstration

This demonstration assumes that you have made the modifications to the OED as outlined in the previous Steps topic for Modifying a method. To demonstrate the success of the modifications, you retrieve a company record from the database that does not have a value already set in its stock ticker property. Observe that the step component inserts it's own stock ticker value into the stock ticker property.

The Onyx Transaction Manager (OTM) is called with the following XML parameters.
