methodCall object

The methodCall object identifies the object name, object type, and method name for an OEAS method wrapper. You can use this object to enable high level reuse of OEAS methods. For instance, a script designer application can use this object to give non-technical users the ability to include an incident insert in a script design. For conceptual information on wrapped method calls, see the Method Calls topic in the feature overview section.

The methodCall object has one key property, primaryId, that uniquely identifies the object instance. Once the object is created, the value of this property cannot be changed.
