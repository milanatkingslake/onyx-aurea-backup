RetrieveList (INatLangCache)
RetrieveList method

The RetrieveList method of the INatLangCache interface enables a calling client to access a list of resource strings from the OnyxNatLangCache. The list of strings is identified by the combination of site ID, language code, and a range of message IDs.

Syntax

IDL

HRESULT RetrieveList(

    [in] const BSTR bstrAppName, 

    [in] const long lSiteID, 

    [in] const BSTR bstrLangCode, 

    [in] const long lStartMsgID, 

    [in] const long lEndMsgID, 

    [out] VARIANT*  pvID, 

    [out] VARIANT*  pvMsg, 

    [out, retval] NatLangCacheStatusEnum *peStatus);

IDL parameter descriptions

bstrAppName

[in] Case insensitive. Corresponds to a unique Onyx database name in the application server system registry. bstrAppName identifies which part of the cache to access. This parameter supports multiple customers in an application service provider environment. Each customer has a separate database.

lSiteId

[in] Corresponds to a key field in the site table of a database. Typically, sites represent departments or divisions within a company. This parameter supports partitioning a database among divisions in an organization.

bstrLangCode

[in] Corresponds to a key field in the language table of a database. The language code identifies the spoken language of the string being accessed.

lStartMsgID

[in] Corresponds to a key field in the NationalLanguage and NatLangConfig tables. This parameter combined with lSiteID and bstrLangCode uniquely identifies a string in the cache. This parameter tells the cache component what is the first string in the range of strings to return.

lEndMsgID

[in] Corresponds to a key field in the NationalLanguage and NatLangConfig tables. This parameter combined with lSiteID and bstrLangCode uniquely identifies a string in the cache. This parameter tells the cache component what is the last string in the range of strings to return.

pvID

[out] Contains a variant array of long integers representing returned MsgId's from the NationalLanguage table.

pvMsg

[out] Contains a variant array of variants representing returned resource strings from the NationalLanguage table.

peStatus

[out, retval] tput return value. peStatus contains a pointer to a success or failure code for the method. See NatLangCacheStatus Enumeration topic for a description of individual codes.
