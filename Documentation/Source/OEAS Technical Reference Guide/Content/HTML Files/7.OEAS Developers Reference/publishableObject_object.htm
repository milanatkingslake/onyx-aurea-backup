publishableObject object

publishableObject object

The publishableObject object assembles sets of domain data and formats them into XML files. These files are then transferred to offline clients for use in their user interface.

Methods

Method

Description

extract

Filters and retrieves domain data using a database view

publish

Retrieves domain data and writes it to an XML file

retrieveCollection

Returns objects that match search criteria

saveCollection

Performs multiple actions at once: inserting, updating, and deleting objects, as appropriate

Properties

Property

Description

active

Specifies whether the domain data the object provides is available to the user interface

description

Contains text about the domain object that is displayed to the end user

filename

Contains the name of the file that the object's domain data is written to

insertBy

Contains the user ID of the Onyx user who initially saves the object

insertDate

Contains the time and date on which the object is initially saved

methodName

Contains the method used to retrieve the object's domain data

methodParameterXml

Contains XML used by the method that retrieves the domain data to publish

name

Contains text that identifies the published domain data

primaryId

Contains the unique ID assigned to the object when it is first saved to the database

retrieveObjectName

Contains the name of the object that has the method specified by the methodName property

updateBy

Contains the user ID of the Onyx user who last saved modifications to the object

updateDate

Contains the time and date on which modifications to the object were last saved
