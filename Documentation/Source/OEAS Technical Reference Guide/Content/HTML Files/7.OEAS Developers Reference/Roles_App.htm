OnyxEASApp roles

The OnyxEASApp COM+ application defines two roles that control access to OEAS components: OnyxEASRestricted and OnyxEASPublic.

OnyxEASRestricted

This role protects all the private interfaces of various OEAS components from unauthorized access. By default only OnyxEASUser belongs to this role. This role ensures that:

  • The Onyx Transaction Manager (OTM) can use the data access and query engine components.
  • Only the authentication components can access the IEstablishSession interface.
  • Only the step components can access the IInternalTransactionManager interface.

OnyxEASPublic

This role ensures that all the public interfaces for the OTM, Session Manager, and Authentication components are accessible to the clients directly.

The domain group "Everyone" is included in this role.
