survey Object

survey object

The survey object provides domain data for administering customer surveys. You can use it compile a list of question, questionHierarchy, and response objects into a single domain object that users can select to give to a customer.

Methods

Method Description

retrieve

Returns a collection of survey objects based on the search criteria supplied in the parameters XML

retrieveCollection

Returns a collection of survey objects based on the search criteria supplied in the parameters XML

retrieveListViewDetail

Returns information about the surveys of the specified type

saveCollection

Performs multiple actions at once: inserting, updating, and deleting survey objects, as appropriate

Properties

Property Description

active

Specifies whether the domain data the object provides is available to the user interface

description

Contains a brief statement of the purpose of the survey

insertBy

Contains the user ID of the Onyx user who initially saves the object

insertDate

Contains the time and date on which the object is initially saved

name

Contains a word or phrase that constitutes the distinctive designation of the survey

primaryId

Contains the unique ID assigned to the object when it is first saved to the database

surveyTypeActive

Contains the active status of the related surveyType object instance

surveyTypeDescription

Contains the description from surveyType object

surveyTypeId

Contains the primaryId of the related surveyType object

updateBy

Contains the user ID of the Onyx user who last saved modifications to the object

updateDate

Indicates the time and date on which modifications to the object were last saved
