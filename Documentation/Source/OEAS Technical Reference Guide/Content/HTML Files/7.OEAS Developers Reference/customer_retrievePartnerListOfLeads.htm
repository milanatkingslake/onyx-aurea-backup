retrievePartnerListOfLeads (customer)

retrievePartnerListOfLeads method

The retrievePartnerListOfLeads method of the customer object returns the list of existing customer leads for a partner.

Parameters

Parameter Direction Data type Content type Description

partnerId

in

string

n/a

The partner.primaryId value

customerSecondaryIdList

in

i4

n/a

The list of customer secondaryIds to search for

customerType

in

i4

n/a

Type of customer: 1=individual;2=company;9=customer

userIdList

in

string

n/a

The list of partner users that may have access to the customer

expandUserList

In

boolean

n/a

A flag if the partner groups should be retrieved for the users in the list

partnerGroupIdList

in

string

n/a

The list of partner groups that may have access to the customer

expandPartnerGroupList

in

boolean

n/a

A flag if the partner groups should be expanded using parentId

companyName

in

string

n/a

Name of the company or individual

firstName

in

string

n/a

First name/contact of the individual or company

lastName

in

string

n/a

Last name/contact of the individual or company

postCode

in

string

n/a

Post code of the customer's primary address

emailAddress

in

string

n/a

The customer's email address

phoneNumber

in

string

n/a

Phone number of the customer's primary phone

includeMask

in

string

n/a

Identifies whether the phone number mask is displayed on the output or not

out

object

all
