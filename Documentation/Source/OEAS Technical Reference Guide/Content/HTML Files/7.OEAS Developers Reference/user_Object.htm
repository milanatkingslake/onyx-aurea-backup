user Object

user object

The user object enables users to delete, insert, retrieve, and update Onyx users.

Methods

Method

Description

retrieve

Returns an existing user object instance

retrieveList

Returns a list of users

retrieveListEnumeration

Returns a list of users with their group membership

retrieveListRoles

Returns the roles associated with a specified user object instance

retrieveListUsersAndGroups

Returns a list of Onyx user and group data

saveCollection

Saves a collection of user object instances

searchForOnyxUserGroupRole

Returns a list of Onyx user, group, and role data corresponding to IDs, names, or descriptions contained in the search criteria

Properties

Property

Description

active

Indicates whether the user is active or inactive

allowAuthTypes

A bitmask that indicates what types of authentication are available to the user

authorityId

A bitmask that specifies user access to functions of the Onyx Employee Portal—Windows client

allowImpersonation

A boolean value that specifies whether the user can impersonate other Onyx users.

computerName

The network identification of the user's computer

detailAdd

A bitmask that specifies user access to functions of the Onyx Employee Portal—Windows client

detailDelete

A bitmask that specifies user access to functions of the Onyx Employee Portal—Windows client

detailView

A bitmask that specifies user access to functions of the Onyx Employee Portal—Windows client

emailAlias

The email address of the user

groupId

A string that identifies the group that the user belongs to

insertBy

The ID of the end-user who created the object instance

insertDate

The date on which the end-user created the object instance

languageCode

A string that indicates the user's language

lastLoginDate

The date and time that the user last logged on

managerId

The userId of the user's manager

name

The user's name

networkUser

The user's network domain name and user name

password

A string that the user specifies to access the system

securityId

A bitmask that specifies user access to functions of the Onyx Employee Portal—Windows client

updateBy

The user ID of the person who last updated the object instance

updateDate

The date of the last object instance update

userId

A string that identifies the user on the system
