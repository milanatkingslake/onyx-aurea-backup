What to expect back from the OTM

Client applications that call the OTM should expect some or all of the following items in the output XML, as returned in the step package.

-<returnXml>

-<otmId>

-<context>

-<sessionSecondsRemaining>

-<methodStatus>

-<statusType>

-<message> or -<error>

-<associationId> (only if the value of statusType = "failure" or "successWithInfo")

-<parametersReturn>

-<rowSet> (only for retrieveList methods)

-<customData> (only if it contains data)
