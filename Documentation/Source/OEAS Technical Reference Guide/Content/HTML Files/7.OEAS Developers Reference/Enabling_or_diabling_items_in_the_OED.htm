Enabling or disabling items in the OED

Warning: Do not disable default OEAS business objects, since doing so will severely impact OEAS functionality. You should enable and disable only custom objects and methods.

To enable or disable an item:

  1. In the navigation area, locate and select the item you want to enable or disable.

    The details for this item display in the edit area.

  2. Check or uncheck the Disabled box to enable or disable this item. A checked box implies a disabled item.

    When modifying an item, the item and its parent object appear in the navigation area marked with an asterisk ( * ), until the modifications are saved. If you disable the item it also appears italicized and grayed out, but the asterisk disappears once the modifications are saved.

  3. Click to save your changes.
