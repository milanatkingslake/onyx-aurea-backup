ownerType property

The ownerType property of a child object is a unique numerical value that identifies the type of parent object that owns the child object.

This is a generic topic that applies to all properties with the same name. The remarks section may contain details specific to a given object.

Data type

i4

Access

Read/write

Remarks

The ownerType property uses an enumeration value specified by the "Object Type" attribute in the Object Designer.

Valid values of the ownerType property depend on the object and its relationships defined in the OED. The following table shows the valid ownerType objects for each child object type.
