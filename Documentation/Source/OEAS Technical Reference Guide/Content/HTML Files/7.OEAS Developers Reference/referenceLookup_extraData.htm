extraData (referenceLookup)

extraData property

The extraData property of the referenceLookup object can be either a text description for a reference parameter value or a mechanism used to group similar data values.

Data type

string (maximum length)

Access

Read/write

Remarks

When used as a text description, the extraData property attaches full descriptions to short or abbreviated lookup data values.

When used as a grouping mechanism, the extraData property provides additional detail, such as levels of granularity, for reference parameter values. For example, for a Sales incident reference parameter value Open, you could associate types or degrees of Open, such as Active and Waiting. To capture these degrees of the value, specify Open as an extra data value to Active and Waiting. These extra data values enable users to run more concise queries for reference data. If no extraData value is assigned to a field, it defaults to the initial value, which in the preceding example is Open.
