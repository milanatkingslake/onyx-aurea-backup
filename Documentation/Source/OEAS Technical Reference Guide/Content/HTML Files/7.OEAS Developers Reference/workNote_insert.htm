insert (workNote)

insert method

The insert method of the workNote object saves a workNote object.

Parameters

Parameter

Direction

Data type

Content type

Description

workNote

inOut

object

all(in) keysOnly(out)

An object whose content="all".

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxWorkNoteStep.WorkNote

com

Calls the WorkNote COM object to add the workNote object to the database

Remarks

The primaryId is NULL for the input XML if this is the first workNote for an incident. Subsequent workNotes for the same incident have the same primaryId which is used in the insert. When you insert a workNote with a primaryId and an ownerId that match another workNote, OEAS increments the seqNum property.
