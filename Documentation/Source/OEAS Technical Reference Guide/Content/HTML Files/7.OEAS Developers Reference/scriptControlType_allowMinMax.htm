allowMinMax (scriptControlType)

allowMinMax property

The allowMinMax property of the scriptControlType object specifies whether the control supports minimum and maximum value constraints.

Data type

Boolean

Access

Read/write

Remarks

Use this property to specify whether your control supports validation of minimum and maximum constraints. The meaning of minimum and maximum depends on and is implemented by the script control. The following table lists some typical uses of minimum and maximum value constraints.

Typical use

Example constraint

Requiring the user to make a selection in a listBox

minimum = 1

Limiting the number of selections a user can make in a checkBoxSet

maximum = N

Specifying a range for the number of characters a user must enter in a textBox

minimum = 3, maximum = 10

Limiting the amount of text a user can enter in a multiLineTextBox

maximum = 256

Note: The effectiveness of this property can depend on other control parameters such as the dropdown parameter (defaultExtraData) for certain list controls.
