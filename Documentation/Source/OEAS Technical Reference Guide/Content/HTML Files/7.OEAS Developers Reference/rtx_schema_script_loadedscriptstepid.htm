@loadedScriptStepId

The loadedScriptStepId attribute contains the primaryId of the domain scriptStep object loaded by the loadStep method.The loadedScriptStepId attribute contains the primaryId of the domain scriptStep object loaded by the loadStep method.

XPath

/root/script/@loadedScriptStepId

Cardinality

1 (optional)

Default value

NULL

Comments

The cardinality of this element depends on the script engine method. See the method specific schema tables (advanceStep, loadStep, resumeSession) for specific requirements.

This attribute identifies the scriptStep returned by loadStep when there are multiple scriptSteps returned in the run-time XML such as when calling resumeSession. Both advanceStep and resumeSession can call loadStep method. This attribute is optional since advanceStep will not call loadStep after persisting the last step of the script.
