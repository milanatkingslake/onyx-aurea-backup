(ActionBatchExecute) parametersReturn XML

parametersReturn XML

The OnyxActionBatchExecute step component sets the overall method status of the actionBatch.execute method call based on the following conditions.

Condition Overall method status

If any method call returns failure

failure

If any method call returns successWithInfo and no calls return failure

successWithInfo

If all calls return success

success

After performing all actions for each sourceList item, OnyxActionBatchExecute collects all the parameters and returnXml for all executed methods, regardless of success or failure and returns it in the statusXml returned in the parametersReturn XML.

See Receiving returned statusXml for a sample statusXml document.
