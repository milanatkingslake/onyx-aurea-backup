retrieve (company)

retrieve method

The retrieve method of the company object searches the database based on the properties defined by the object's primary objectKey entry and returns an object instance.

retrieve standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxMasterDetailRetrievalStep.MasterDetailRetrieval com Propagates primaryId value of master object into the ownerId properties of the phones collection
OnyxMasterDetailRetrievalStep.MasterDetailRetrieval com Propagates primaryId value of master object into the ownerId properties of the addresses collection
masterDetailEmailAddresses com Propagates primaryId value of master object into the ownerId properties of the email addresses collection
OnyxDM.DataManager com

Retrieve a single company object

When primaryContactId is non-NULL, returns contactFirstName and contactLastName from the specified contact object

When primaryContactId is NULL, returns contactFirstName and contactLastName from the company object

Uses database function mapped to alertId property to return alertId value
phone.retrieveCollection method Retrieve a phone collection which contains all phones associated with a company
address.retrieveCollection method Retrieve an address collection which contains all addresses associated with a company
emailaddress.retrieveCollection method Retrieve an address collection which contains all addresses associated with a company
