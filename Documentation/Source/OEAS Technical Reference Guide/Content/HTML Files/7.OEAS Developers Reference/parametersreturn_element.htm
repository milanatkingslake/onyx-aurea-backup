parametersReturn element

The parametersReturn element contains most of the output data for the method.

When Onyx step components finish their work, they insert their result data into the parametersReturn element. The contents of this element vary from one method to the next. When step components invoke retrieveList methods, they typically insert rowSet data into the parametersReturn element.

Example

The following example shows a sample parametersReturn element for the insert method of the company object:

<parametersReturn>

 <company objectType="company" content="keysOnly">

  <primaryId>351</primaryId>

  <phones collection="1" objectType="phone">

   <phone objectType="phone" content="keysOnly" action="insert">

    <primaryId null="1"></primaryId>

   </phone>

   <phone objectType="phone" content="keysOnly" action="insert">

    <primaryId>651</primaryId>

   </phone>

  </phones>

  <addresses collection="1" objectType="address">

   <address objectType="address" content="keysOnly" action="insert">

    <primaryId null="1"></primaryId>

   </address>

   <address objectType="address" content="keysOnly" action="insert">

    <primaryId>400</primaryId>

  </addresses>

 </company>

</parametersReturn>
