retrieveListByFilterNoDefault method

The retrieveListByFilterNoDefault method of the incident object searches the database by the filter criteria you pass in and returns a rowSet of incident data. This method does not use defaults for parameter values that are not provided.

Parameters

Parameter Direction Data type Content type Description

groupIdList

in

string

n/a

A semicolon-delimited list of groups whose users are added to the assignedToList values

assignedToList

in

string

n/a

A semicolon-delimited list of assignedTo values used for filtering

statusIdList

in

string

n/a

A semicolon-delimited list of statusId values used for filtering
priorityIdList in string n/a A semicolon-delimited list of priorityId values used for filtering
sourceIdList in string n/a A semicolon-delimited list of sourceId values used for filtering
typeIdList in string n/a A semicolon-delimited list of typeId values used for filtering
categoryIdList in string n/a A semicolon-delimited list of categoryId values used for filtering
keyWordList in string n/a A semicolon-delimited list of keyword values used for filtering
dueByStart in dateTime n/a The starting due-by date for incident reminders
dueByEnd in dateTime n/a The ending due-by date for incident reminders
insertStart in dateTime n/a The start date for incident insertions
insertEnd in dateTime n/a The end date for incident insertions
updateEnd in dateTime n/a The end date for incident updates
updateStart in dateTime n/a The start date for incident updates
includeContact in boolean n/a Switches to include the individual ID for the person listed as contact for the incident if the person is someone other than the owner
rowSet out object all Contains the rowSet column definitions and the incident table rows matching with the owner and category criteria
