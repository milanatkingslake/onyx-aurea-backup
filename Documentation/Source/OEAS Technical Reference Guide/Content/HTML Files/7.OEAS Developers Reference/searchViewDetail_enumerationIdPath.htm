enumerationIdPath (searchViewDetail)

enumerationIdPath property

The enumerationIdPath property of the searchViewDetail object contains the XPath to the enumerated id.

Data type

string (maximum length)

Access

Read/write

Remarks

This property identifies the element within the enumerationItemPath that contains the enumeration item. If the detail item is a region code, for instance, you should retrieve a list of valid region codes to display to the user as selection options. The region code is found in the c0 child element of the enumerationItemPath element (//parametersReturn/rowSet/rows/row).
