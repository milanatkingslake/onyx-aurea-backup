retrievePartnerUserList (incident)

retrievePartnerUserList method

The retrievePartnerUserList method of the incident object returns the data for a partner user - optionality filtered by the user assigned to an incident. This method always implements partner security for the user.

Parameters

Parameter Direction Data type Content type Description

assignedTo

in

string

n/a

Filters by the user assigned to the incident

categoryId

in

i4

n/a

Filters by an ID representing the top level type of Incident such as Sales, Service, or Support

includeMask

in

string

n/a

Identifies whether the phone number mask is displayed on the output or not

rowSet

out

object

all

Contains the rowSet column definitions and the incident table rows matching with the owner and category criteria

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates the input parameters

OnyxDM.DataManager

com

Retrieves the list of incident objects based on the optional assignedTo criteria
