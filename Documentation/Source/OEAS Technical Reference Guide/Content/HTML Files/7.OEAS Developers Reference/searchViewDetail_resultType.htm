resultType (searchViewDetail)

resultType property

The resultType property of the searchViewDetail object contains a code that specifies what type of data is contained in a column result.

Data type

string

Access

Read/write

Remarks

By default, OEAS uses the following resultType codes:

  • companyPrimaryId
  • individualPrimaryId

The UI uses this property to enable it to link to another UI page from a column result. For instance, a search for companies returns a list of companies with a column for primaryId, where the UI renders each primaryId as a hyperlink to the company detail page. When the user clicks the hyperlink, the UI recognizes companyPrimaryId resultType and returns the company information with the corresponding primaryId. To customize this field, you must configure the UI to recognize the new resultType and define a function in the UI to handle that resultType.
