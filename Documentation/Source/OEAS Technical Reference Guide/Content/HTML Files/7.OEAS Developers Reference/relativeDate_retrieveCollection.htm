retrieveCollection (relativeDate)

retrieveCollection method

The retrieveCollection method of the relativeDate object returns a collection of objects based on the search criteria supplied in the parameters XML.

retrieveCollection method standard behavior

Parameters

Parameter

Direction

Data type

Content type

Description

relativeDates

inOut

object

any(in) all(out)

A collection of relativeDate objects

Steps

Step data

Step type

Description

OnyxGenericDAC.GenericDAC

com

Retrieves a collection of relativeDate objects

Remarks

When the relativeDate input object does not specify any values on which to search, all relativeDate instances are returned.
