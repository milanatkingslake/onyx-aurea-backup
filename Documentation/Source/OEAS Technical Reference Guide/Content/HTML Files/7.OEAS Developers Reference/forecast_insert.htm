insert (forecast)

insert method

The insert method of the forecast object persists an object to the database.

insert standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com

Calls a pre-procedure that generates an integer value for the secondaryId when the element is null in the parameters XML

Inserts a forecast object instance

Calls a post-procedure to enforce a business rule which allows only one forecast instance per owner incident
forecastLine.saveCollection method

Insert a collection of forecastLine objects

On insert actions, calls a database function mapped to the forecastLine lineId property to calculate the next sequential line number
