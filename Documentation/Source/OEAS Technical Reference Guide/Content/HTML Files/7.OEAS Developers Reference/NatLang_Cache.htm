NatLang Cache

INatLangCache Interface

The OnyxNatLangCache component implements the INatLangCache interface. INatLangCache exposes two methods:

Method

Description

Retrieve

Retrieves a single resource string from the OnyxNatLangCache.

RetrieveList

Retrieves a list of resource strings from the OnyxNatLangCache.

You can use these methods for retrieving resource strings from the in-memory copy of the tables containing default and customized strings, either one string at a time (Retrieve) or many strings simultaneously (RetrieveList). Onyx business objects, custom business objects, and step components use this interface to retrieve resource strings such as user interface captions, logging messages, and error messages.

The OnyxNatLangCache component uses the following enumeration:

Enumeration

Description

NatLangCacheStatusEnum

Indicates the return status of the INatLangCache methods.
