retrieveCollection (publishableObject)

retrieveCollection method

The retrieveCollection method of the publishableObject object searches the database and returns a collection of zero or more publishableObject objects that match the search criteria.

retrieveCollection method standard behavior

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Retrieves a collection of publishableObjects objects

      </publishableObject>
