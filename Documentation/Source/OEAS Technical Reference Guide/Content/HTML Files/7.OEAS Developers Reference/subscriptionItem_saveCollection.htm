saveCollection (subscriptionItem)

saveCollection method

The saveCollection method of the subscriptionItem object performs multiple actions at once: inserting, updating, and deleting objects, as appropriate.

Important: Do not call this method from the user interface. Instead, use the subscribe method to insert, delete, and update subscriptionItem object instances.
