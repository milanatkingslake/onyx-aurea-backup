retrieve (customerProduct)

retrieve method

The retrieve method of the customerProduct object searches the database by primary ID and returns a customerProduct object and its associated customerProductDetail objects.

retrieve standard method information

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxMasterDetailRetrievalStep.MasterDetailRetrieval

com

Propagates primaryId value of master object into the ownerId properties of the customerProductDetails collection

OnyxDM.DataManager

com

Retrieve a customer record and its collection of associated CustomerProductDetail records

customerProductDetail.retrieveCollection

method

Retrieve a customerProductDetail collection which contains all customerProductDetails associated with a customerProduct
