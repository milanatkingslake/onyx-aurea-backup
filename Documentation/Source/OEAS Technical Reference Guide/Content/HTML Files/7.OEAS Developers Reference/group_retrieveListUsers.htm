retrieveListUsers (group)

retrieveListUsers method

The retrieveListUsers method of the group object returns information about users that belong to the specified group.

retrieveList standard method information

Parameters

Parameter Data type Direction Description
groupId string in A collection of users usually defined by organizational structure.
active i4 in An integer that specifies whether to return active users or inactive users. If the active element of the input parameters XML is empty, NULL, or not present, all users are returned, active and inactive. When active contains 1, all active users are returned. If the element contains 0, inactive users are returned. If active contains an invalid value, such as 2, the method succeeds, but no users are returned.
rowSet object out Row set of group information.

Columns returned in the rowSet object

columnOrdinal columnName Physical column
1 chUserId users.user_id
2 chUserName users.user_name
3 tiRecordStatus users.record_status

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieves a rowSet object listing users who are members specified group
Step data Step type Description

Remarks

The retrieveListUsers search is case sensitive.
