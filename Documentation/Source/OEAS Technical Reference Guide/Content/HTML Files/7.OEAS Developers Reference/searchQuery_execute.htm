execute (searchQuery)

execute method

The execute method of the searchQuery object executes an SQL statement.

Parameter

Direction

Data type

Content type

Description

in

i4

in

string

queryContext

in

out

object

Step data

Step type

Description

com

Remarks

The OED does not contain a definition of the result set returned in the rowSet parameter. The contents of the rowSet parameter are determined at runtime.
