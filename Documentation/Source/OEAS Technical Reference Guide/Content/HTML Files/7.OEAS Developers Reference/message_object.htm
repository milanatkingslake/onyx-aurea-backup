message object

The message object enables users to insert and retrieve message object instances. A user interface can call methods on the message object to create, send, and read messages. Users can attach company, individual, incident, and workTicket objects to messages.

The messageDestination object associates message object instances with the users specified in message recipient lists. OEAS creates an instance of the messageDestination object for each userId in a message recipient list. These object instances track the read status of the message for each recipient.

The message object has one key property, primaryId, that uniquely identifies object instances in the database. Once the object is created, the value of this property cannot be changed.
