retrieveCollection (assignmentExpression)

retrieveCollection method

The retrieveCollection method of the assignmentExpression object returns a collection of objects based on the search criteria supplied in the parameters XML.

retrieveCollection standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieves a collection of assignmentExpression objects

Example

Parameters

<parameters>

 <assignmentExpressions objectType="assignmentExpression" collection="1">

  <assignmentExpression objectType="assignmentExpression">

   <primaryId>100000</primaryId>

   <conditionId></conditionId>

   <propertyName></propertyName>

   <propertyValue1></propertyValue1>

   <propertyValue2></propertyValue2>

   <operator></operator>

   <insertBy></insertBy>

   <insertDate></insertDate>

   <updateBy></updateBy>

   <updateDate></updateDate>

  </assignmentExpression>

 </assignmentExpressions>

</parameters>
