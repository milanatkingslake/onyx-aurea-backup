onyxFormName (template)

onyxFormName property

The onyxFormName property of the template object specifies the name of the form in the Onyx user interface that uses this quote template.

Data type

string (maximum length)

Access

Read/write

Remarks

This property is used in conjunction with onyxAppName when determining the list of quotes that appear in a given user interface.

For example, an onyxAppName of 'OEP' and an onyxFormName of 'Incident - Sales' indicates that the quote template will appear in the list of available templates for the sales incident form in the OEP interface.

This property is required on a saveCollection insert and must reference an existing form instance in the onyx_form database table.
