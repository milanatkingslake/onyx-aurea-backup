supportType (product)

supportType property

The supportType property of the product object contains The warranty schedule in use for the product.

Data type

string (maximum length)

Access

Read/write

Remarks

The System Parameters Administration tool's WarrantyDateMethod field contains the value for this property. Valid values include:

Property Value

Description

BYPRODUCT

Allows you to set different warranties for different products.

FIRSTCALL

Warranty schedule defined by a number of days starting after the first support call is received.

INSERTDATE

Warranty schedule defined by a number of days beginning with the insert of a product order.

CALLCOUNT

Warranty schedule set by number of support calls allowed.

The default value is INSERTDATE.
