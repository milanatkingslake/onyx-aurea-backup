Property attributes

Property attributes

When you select a property in the Object Designer, the attributes for that property display in the edit area.

The attributes are defined as follows:

Attribute Name

Description

ID

The globally unique identifier (GUID) assigned by the Onyx Enterprise Dictionary. This attribute is read-only.

Name

Type the name you want to assign to this property. Property names must be conform to XML naming conventions.

Description

Type the description for this property.

Data Type

Select the type of data to be stored in this property:

  • For simple types, select an XML-data type.
  • For objects, select object.

Object Type

Select the object you want to use. This list includes objects marked for deletion and objects that are disabled, both which are marked as such. Invalid objects are not listed. Default OEAS step components ignore the Required constraint for properties whose data type is object.

Content

Select the content type for this object:

  • For normal strings, select string.

Generate ID

Select the type of ID generation:

  • For globally unique unique identifier (GUID) generation by the ODM, select odmGuid.
  • When ID generation is performed by the database, select db.

The underlying table uses an identity column to automatically generate the primary key.

  • For no ID generation, select empty space at top of drop-down list.

Installed By

Type the name of the vendor or application adding this property.

Collection

Check this box to define this property as a collection of items.

Read Only

Check this box to define this property as read-only.

Disabled

Check this box to disable this property.

Denormalized Definition

Check this box to indicate whether the property is denormalized at the object level, meaning the property is not persisted in the relevant table for the object that the property belongs to.

Join Relationship

Select the name of an existing relationship that enables the denormalized definition. The target object must be the primary object in the relationship definition.

Target Property

Select the property of the target object that you want to use.
