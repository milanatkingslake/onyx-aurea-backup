sequenceNumber (workNoteDetail)

sequenceNumber property

The sequenceNumber property of the workNoteDetail object contains the ordinal value of the workNoteDetail in the workNoteDetails collection of the workNoteHeader object.

Data type

i4

Access

Read-only; for more information, see Read Only attribute

Remarks

The system increments the sequenceNumber property for every workNoteDetail object with the same logMarkSequenceNumber. The logMarkSequenceNumber and sequenceNumber combination is unique among workNoteDetail objects belonging to a given workNoteHeader master object but is not unique across workNoteHeader objects.
