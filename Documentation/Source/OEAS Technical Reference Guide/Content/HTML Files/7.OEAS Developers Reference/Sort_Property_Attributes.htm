Sort Property attributes

When you select a sort property in the Object Designer, the attributes for that sort property display in the edit area.

The attributes are defined as follows:

  • Property: Select the name of the property you want to use for ordering result set records from the retrieveCollection method. This list includes all properties for the current object defined in the OED that are not denormalized properties. If you enter more than one property, the order of properties listed in the navigation area determines their position in the order by clause for the method.
  • Sort Direction: Select the direction you want to sort this result set:
  • To sort the results in ascending order, select asc.
  • To sort the results in descending order, select desc.
