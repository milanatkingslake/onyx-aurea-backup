surveyQuestion Object

surveyQuestion object

The surveyQuestion object enables you to group questions into a survey to present to customers. This provides for reuse of questions in multiple surveys. By containing properties for surveyId and questionId this object relates a question to a specific survey. It also supports sequencing of questions in the survey.

Methods

Method Description

retrieveCollection

Returns a collection of surveyQuestion objects based on the search criteria supplied in the parameters XML

retrieveListViewDetail

Returns information about the surveyQuestion object of the specified type

saveCollection

Performs multiple actions at once: inserting, updating, and deleting surveyQuestion objects, as appropriate

Properties

Property Description

active

Specifies whether the domain data the object provides is available to the user interface

insertBy

Contains the user ID of the Onyx user who initially saves the object

insertDate

Contains the time and date on which the object is initially saved

questionId

Contains the primaryId of the related question object

questionText

Contains the text of the related question

sequenceNumber

Contains an ordinal number for the question

surveyId

Contains the primaryId of the related survey object

updateBy

Contains the user ID of the Onyx user who last saved modifications to the object

updateDate

Indicates the time and date on which modifications to the object were last saved
