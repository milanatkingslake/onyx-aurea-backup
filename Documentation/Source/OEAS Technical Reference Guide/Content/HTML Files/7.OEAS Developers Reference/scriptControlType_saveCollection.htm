saveCollection (scriptControlType)

saveCollection method

The saveCollection method of the scriptControlType object performs multiple actions at once: inserting, updating, and deleting objects, as appropriate.

saveCollection standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Insert, update, and/or delete a collection of scriptControlType objects

Remarks

You cannot delete a control that a scriptPrompt references. However, you can mark it as inactive by setting the active property to zero.

Example

The following example shows a saveCollection method that updates the active field on the customerSearch (custom) control.

Parameters

<parameters>

   <scriptControlTypes objectType="scriptControlType" collection="1">

      <scriptControlType objectType="scriptControlType" content="any" action="update">

         <controlTypeCode>customerSearch</controlTypeCode>

         <name/>

         <description/>

         <allowUserInput/>

         <requiresResponseList/>

         <allowSort/>

         <multiSelect/>

         <allowEditableResponses/>

         <requiresDataType/>

         <allowText/>

         <allowInitialValue/>

         <allowMinMax/>

         <allowSize/>

         <allowExtraData/>

         <defaultExtraData/>

         <insertBy/>

         <insertDate/>

         <updateBy/>

         <updateDate/>

         <active>0</active>

      </scriptControlType>

   </scriptControlTypes>

</parameters>

parametersReturn

<parametersReturn>

   <scriptControlTypes collection="1" objectType="scriptControlType">

      <scriptControlType objectType="scriptControlType" content="keysOnly" action="update">

         <controlTypeCode>customerSearch</controlTypeCode>

      </scriptControlType>

   </scriptControlTypes>

</parametersReturn>
