Testing the customization

Before testing the customization make sure you configured and populated the routing printer table. If you followed all the steps so far, everything should work perfectly the first time, but it's always a good idea to test your customization. If you created two printer records with different country codes in the previous topic, you can test this customization by adding a couple of customers with corresponding country codes. You also must have a print literature product defined. Insert a new order for each customer using the print literature product. The orders should be sent to the two different printers.
