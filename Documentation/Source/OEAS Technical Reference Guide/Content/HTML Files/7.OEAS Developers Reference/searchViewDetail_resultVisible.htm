resultVisible (searchViewDetail)

resultVisible property

The resultVisible property of the searchViewDetail object indicates whether the field is visible in the result set.

Data type

boolean

Access

Read/write

Remarks

The resultVisible property combines with searchResultDefaultRequired and resultAllowed properties to define the behavior of the UI with respect to the searchViewDetail. The following table lists the valid combinations for resultAllowed, searchResultDefaultRequired, and resultVisible.
