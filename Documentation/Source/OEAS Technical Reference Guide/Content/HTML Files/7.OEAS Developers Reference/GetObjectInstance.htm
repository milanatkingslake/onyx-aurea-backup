GetObjectInstance (ICache)
GetObjectInstance method

The GetObjectInstance method of the ICache interface retrieves an empty object instance of a business object.

Syntax

IDL

HRESULT GetObjectInstance(

    [in] BSTR ApplicationName, 

    [in] BSTR ObjectName, 

    [out] BSTR* ObjectInstance, 

    [out, retval] OEDCacheStatusEnum* Status);

IDL parameter descriptions

ApplicationName

[in] A string containing the logical application name that determines which subset of the cache to use for a query.

ObjectName

[in] A string that contains the name of the specified object.

ObjectInstance

[out] A string that contains an XML stream containing the instance of the object.

Status

[out, retval] An enumeration of type OEDCacheStatusEnum that contains the status of the method call.

C#

Return values

This method returns an enumeration of type OEDCacheStatusEnum that contains the status of the method call. See the OEDCacheStatusEnum topic for a description of individual enumeration value

Example

The following Visual Basic code demonstrates how to use the GetObjectInstance method to return an empty incident object:

Dim oCache As New OnyxDictionaryCacheLib.DictionaryCache

Dim status As OnyxDictionaryCacheLib.OEDCacheStatusEnum

Set oCache = New OnyxDictionaryCacheLib.DictionaryCache 

status = oCache.GetObjectInstance("Onyx", "incident", strXML)
