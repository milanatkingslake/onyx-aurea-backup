Custom version number

Custom version numbers can be used when you customize OED entities or create custom ones. Similar to the Onyx version number, the custom version number is comprised of the major, minor, and revision version numbers. This number is not incremented by Onyx and is editable. Characteristics of the number are:

  • By default, the numbers are zero.
  • Each number allows up to four digits.
  • Only digits 0-9 are allowed.
  • Leading zeros entered for a number are preserved.
  • Blanking out a number results in a zero value.
