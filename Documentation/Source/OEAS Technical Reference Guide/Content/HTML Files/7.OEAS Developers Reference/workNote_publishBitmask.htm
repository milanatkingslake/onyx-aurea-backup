publishBitmask (workNote)

publishBitmask property

The publishBitmask property of the workNote object contains a bitmask that determines which portions of a workNoteText are displayed in a web-based user interface.

Data type

i4

Access

Read/write

Remarks

Valid values of the publishBitmask property include:

Value

Description

0

The work note is not published to the Web.

1

The work note is published to the Web.

2

The work note was entered via the Web and is published to the Web.

4

The work note was previously published to the Web and reset to not published.

8

This value publishes the work note to the Web and resets all previous Web-published work notes from 1 to 4. Work notes with the 2 value are not reset.
