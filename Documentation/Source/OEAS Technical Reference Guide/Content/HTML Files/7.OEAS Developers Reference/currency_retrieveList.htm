retrieveList (currency)

retrieveList method

The retrieveList method of the currency object returns a rowSet of currency information.

retrieveList standard method information

Parameters

Parameter Direction Data Type Content type Description
rowSet out object all(out) A table of the currency codes, their descriptions, and localeIds

Columns returned in the rowSet object

columnOrdinal columnName Physical column
1 chCurrencyCode currency_ml_view.currency_code
2 vchCurrencyDesc currency_ml_view.currency_desc
3 chLocaleId currency_ml_view.locale_code

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieve a rowSet of currency information

Example

The following example shows the input parameters and output parametersReturn for the retrieveList method. Notice the pageSize attribute on the rowSet element. This limits the maximum rows returned to three. The example includes the returnXml content to illustrate the associated successWithInfo information when you limit the maximum rows returned.
