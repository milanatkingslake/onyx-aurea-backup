contactUserName (internalContact)

contactUserName property

The contactUserName property of the internalContact object contains the name of the user that serves as an internal contact for another object.

Data type

string (maximum length)

Access

Read-only; for more information, see Read Only attribute

Remarks

The contactUserName is the name associated with an userId when it is created in the OEAS. This property displays the name associated with the userId for this contact. You manage user names with Security Administration.
