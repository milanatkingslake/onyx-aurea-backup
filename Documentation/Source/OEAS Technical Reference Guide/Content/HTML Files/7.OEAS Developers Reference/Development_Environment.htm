Configure your OEAS Development Environment

Before developing on the OEAS platform, ensure that you have the necessary tools and understand the technologies that OEAS implements. Make sure that you install, in your development environment, some or all of the following software (depending on what you plan to do):

  • All software required for installing OEAS - refer to the System Requirements Guide.
  • Microsoft SQL Server
  • Microsoft Visual Studio
  • Onyx Enterprise Studio (which contains the Object Designer and the administration tools)

You should also be comfortable with the following tools and technologies:

  • COM, COM+, DCOM
  • T-SQL
  • XML
  • Microsoft Active Server Pages (ASP)
  • Microsoft C#, VB.Net, or C++
  • Microsoft Internet Information Server (IIS)
