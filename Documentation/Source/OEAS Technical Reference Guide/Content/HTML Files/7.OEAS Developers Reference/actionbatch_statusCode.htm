statusCode property

The statusCode property of actionBatch object is the status code of the batch such as 0 (Not Processed), 1 (Processing Jobs), 2 (Error) and 3 (Complete) processed by Onyx Processing Service.

Data type

i2

Access

Read/Write

Remarks

The statusCode is used to track the status of the actionBatch and is displayed on the new batches and completed batches pages in the OPS Administration.
