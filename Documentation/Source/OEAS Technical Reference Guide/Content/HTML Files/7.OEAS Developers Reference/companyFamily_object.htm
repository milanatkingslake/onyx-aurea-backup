companyFamily object

The companyFamily object is the logical representation of the domain data stored in the company_family table and represents groupings of customer companies by families. The Table Administration tool uses the companyFamily object to maintain company family data.

Each company object can be related to no more than one companyFamily object.

Methods

Method Description
retrieveCollection Returns a collection of objects based on the search criteria supplied in the parameters XML
retrieveList Retrieves a list of all company families for the site
saveCollection Performs multiple actions at once: inserting, updating, and deleting objects, as appropriate

Properties

Property Description
active Specifies whether the domain data the object provides is available for use by the caller
familyCode A character code for the company family
familyDetail User-defined text information for a companyFamily object
familyName The full name of the company family
insertBy The user ID of the Onyx user who initially saves the object
insertDate The time and date on which the object is initially saved
onyxTimestamp Facilitates record locking with the Onyx Enterprise Database
primaryId The unique ID assigned to the object when it is first saved to the database
updateBy The user ID of the Onyx user who last saved modifications to the object
updateDate The time and date on which modifications to the object were last saved
