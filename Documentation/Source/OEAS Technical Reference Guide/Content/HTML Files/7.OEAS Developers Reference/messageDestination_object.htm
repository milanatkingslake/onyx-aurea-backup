messageDestination object

The messageDestination object enables users to insert, update, and delete messageDestination object instances. OEAS creates an instance of the messageDestination object for each userId in a message object recipient list. The messageDestination object instances track the read status of the message for each recipient.

The message object has one key property, primaryId, that uniquely identifies object instances. OEAS generates the primaryId by concatenating the userId and messageId properties. The primaryId is not persisted in the database.

Method

Method Description

retrieveCollection

Returns the collection of object instances based on the search criteria

saveCollection

Inserts, updates, and deletes messageDestination object instances

Properties

Property Description

insertBy

The Onyx user who initially saved the object instance

insertDate

The time and date on which the object instance was initially saved

messageId

The primaryId of the message sent to a user

primaryId

A unique value that OEAS assigns to the message object

read

A boolean that specifies whether the recipient has read the message

recipientId

The userId of the person who receives the message

updateBy

The user ID of the person who last updated the object instance

updateDate

The time and date of the last object instance update
