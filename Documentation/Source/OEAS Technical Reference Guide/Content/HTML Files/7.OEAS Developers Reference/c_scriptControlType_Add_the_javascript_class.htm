Add the Javascript class

Every script control type must have a Javascript class that defines the run-time behavior of the control in the user interface. The class typically defines behaviors such as how the control:

  • validates input.
  • handles DHTML events.
  • sets its response.

For the Scripting feature, the script_controls.js file in the OEP source files contains all Javascript class definitions for script control types. The following topics describe how to add a Javascript class definition to the script_controls.js file for a custom script control type.
