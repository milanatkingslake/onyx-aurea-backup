literatureTerms object

The literatureTerms object exposes a single method that returns the list of current payment terms available for literature product orders. The values available through this object are used in the terms property of the literatureOrder object. To edit the list of values, use the Table Administration tool available in the Onyx Enterprise Studio.

Methods

Method Description

retrieveCollection

Returns the collection of object instances based on the search criteria

retrieveList

Obtains the list of active payment terms for a literature product order

saveCollection

Inserts, updates, and/or deletes the collection of the specified objects
