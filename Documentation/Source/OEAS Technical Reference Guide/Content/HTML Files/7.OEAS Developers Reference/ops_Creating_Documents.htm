Creating templates

Before you can use the Onyx Processing Server, you must create the templates that the subsystems use to process literature orders. The templates contain the logic necessary to merge information from orderBatch XML with a print or email template. They can be XSL stylesheets or Open XML Word documents with Mail Merge. Onyx provides sample stylesheets and templates that you can customize, or you can create your own. Both types use the orderBatch XML as the input source for data merging.

This book contains information on using XSL and Word documents and on the structure of the orderBatch XML. For more information on how these templates fit into OPS, see the OPS Concepts section of this reference. For more information on configuring literatureProduct objects, see the OES Product Administration online Help.
