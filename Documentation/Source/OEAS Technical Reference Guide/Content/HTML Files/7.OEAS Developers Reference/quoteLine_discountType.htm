discountType (quoteLine)

discountType property

The discountType property of the quoteLine object contains the type (percent or amount) of the discount for the quoteLine object.

Data type

i4

Access

Read/write

Remarks

The discountType can be either amount or percent. This value identifies how to use the primaryDiscount and secondaryDiscount values to determine the cost to the customer. If the discountType is "amount", the primaryDiscount and secondaryDiscount values are subtracted from the unit price. If the discountType is "percent", the primaryDiscount and secondaryDiscount percents of the price are subtracted from the unit price.

To configure the values for this property, use OES Reference Table Administration. Using Reference Table Administration, you create a referenceLookup object that includes an extraData property.

  • extraData = "amount" ... indicates that the discount is an absolute currency amount.
  • extraData = "percent" ... indicates that the discount is a percentage discount.
