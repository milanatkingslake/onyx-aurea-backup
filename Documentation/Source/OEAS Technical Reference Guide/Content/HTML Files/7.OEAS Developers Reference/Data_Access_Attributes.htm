Data access attributes

The following attributes are defined for use on the root element for any object or object collection in an object instance. The possible values and where each attribute is used are shown in the table below.

Note: The presence of the content and objectType attributes is entirely optional. Client applications are not required to pass it, and therefore step components should not assume that their values are set nor set correctly. The only use of these attributes is to make the XML easier to read for debugging or development purposes.
