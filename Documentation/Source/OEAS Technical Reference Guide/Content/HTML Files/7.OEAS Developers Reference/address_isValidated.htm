isValidated Property

The isValidated property of the address object reflects whether it is QAS validated.

Data type

Integer

Access

Read/write

Remarks

This property is added to determine accuracy of the addresses recorded in Onyx database. Address is validated through Experian QAS Address validation APIs.

This property can contain any of the following three values:

  • 0: Address record is not verified by QAS
  • 1: Address record is verified by QAS and is considered a valid address
  • -1: Address record is verified by QAS. However, it is considered an invalid address
