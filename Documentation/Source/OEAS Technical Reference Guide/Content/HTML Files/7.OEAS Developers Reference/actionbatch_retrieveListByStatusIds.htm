retrieveListByStatusIds method

The retrieveListByStatusIds method of the actionBatch object returns the list of objects according to their status IDs.

Parameters

Parameter Direction Data type Content type Description

actionBatchStatusCodeList

string

object

n/a

A semicolon-delimited list of status codes

rowSet

out

object

n/a

An actionBatch rowSet

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates the input parameters

OnyxDM.DataManager

com

Retrieves the list of objects based on the status IDs
