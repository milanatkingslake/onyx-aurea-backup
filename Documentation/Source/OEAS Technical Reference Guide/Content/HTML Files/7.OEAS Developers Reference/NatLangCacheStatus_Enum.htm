NatLangCacheStatusEnum
NatLangCacheStatusEnum enumeration

The following table lists the error values used by the peStatus parameter in the INatLangCache interface methods.

Name (value)

Description

NL_CACHE_SUCCESS (0)

The operation completed successfully.

NL_CACHE_CACHE_NOT_FOUND (1)

The requested cache name is not located in the cache list. This refers to partitioning the cache by appName.

NL_CACHE_ITEM_NOT_FOUND (2)

The requested item was not found in the given cache.

NL_CACHE_READ_ERROR (3)

An error occurred attempting to read from the cache.

NL_CACHE_WRITE_ERROR (4)

An error occurred attempting to write to the cache.

NL_CACHE_INVALID_ID_RANGE (5)

The requested start and end ID range is not valid.

NL_CACHE_LOADING_ERROR (6)

An error occurred attempting to load the cache.

NL_CACHE_INVALID_PARAM (7)

Invalid params were passed in to the method.
