retrieveCollection (currency)

retrieveCollection method

The retrieveCollection method of the currency object returns a collection of objects based on the search criteria.

retrieveCollection standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Return a collection of currency objects

Example

The following sample shows the input parameters and output parametersReturn for the currency retrieveCollection method. Notice the pageSize attribute on the currencies collection element. Without the pageSize attribute this example would return all currency objects, because the parameters do not specify any filter criteria. Instead, the pageSize attribute limits the returned collection to two objects. The example includes the returnXml content to illustrate the successWithInfo information returned when you limit the maximum collection size.

Parameters

<parameters>

   <currencies objectType="currency" collection="1" pageSize="2">

      <currency objectType="currency">

         <currencyCode/>

         <description/>

         <localeId/>

         <insertBy/>

         <insertDate/>

         <updateBy/>

         <updateDate/>

         <onyxTimestamp/>

         <active/>

      </currency>

   </currencies>

</parameters>
