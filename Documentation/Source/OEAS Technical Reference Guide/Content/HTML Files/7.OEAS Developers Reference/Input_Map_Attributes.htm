Input Map attributes

When you select an input map in the Object Designer, the attributes for that input map display in the edit area.

The attributes are defined as follows:

  • Procedure Parameter: Select the name of the procedure input parameter.
  • Map Type: Select the type of map that should be mapped to the procedure:
    • To map to a property, select property.
      • Property: Select the property to map the parameter to.
    • To map to a method parameter, select methodParameter.
      • Method Parameter: Select the method parameter to map the procedure parameter to.

Important: Never map a procedure parameter to a method parameter of data type "object"â€”map to the object properties instead.
    • To map to a literal value that will be used as the parameter value, select literal.
      • Map Data: Type the literal value to be mapped to the procedure.
    • To map to any legitimate XML, XSL, or SQL construct that can be resolved in the SQL Gen stylesheet, select XSL literal.
      • Map Data: Type the XML, XSL, or SQL construct to be mapped to the procedure.
    • To map to run-time context XML, select context variable.
      • Map Data: Type one of the following:
        • To map to the <siteId>, type vSiteId.
        • To map to the <userId>, type vUserId.
        • To map to the <preferredLanguage>, type vLanguage.
