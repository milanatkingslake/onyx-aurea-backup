expirationDate (alert)

expirationDate property

The expirationDate property of the alert object contains the time and date that the alert expires.

Data type

dateTime

Access

Read/write

Remarks

The insert and update methods accept any dateTime value for the expirationDate property. It is the responsibility of the user interface to determine the validity of expirationDate.

When the retrieve method gets the alert for the user interface and the expirationDate has been reached, the retrieve method sets the expired flag for the alert. The user interface must call the delete method if the expired alert is to be removed from the database.

It is the client application's responsibility to convert the expirationDate property value to UTC if that is how the value is to be saved.
