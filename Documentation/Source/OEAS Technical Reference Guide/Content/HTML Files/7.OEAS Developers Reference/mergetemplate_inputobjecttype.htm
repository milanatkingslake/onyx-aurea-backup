inputObjectType (mergeTemplate)

inputObjectType property

The inputObjectType property of the mergeTemplate object contains a value that specifies the type of object whose data will be merged into the template defined by the contents of the text property.

Data type

i4

Access

Read-only; for more information, see Read Only attribute

Remarks

OEAS determines the value of this property based on the methodCall object instance identified by the methodCallId property.
