retrieveList (country)

retrieveList method

The retrieveList method of the country object returns a rowSet of country information.

retrieveList standard method information

Parameters

Parameter Direction Data Type Content type Description
rowSet out object all(out) Information for countries

Columns returned in the rowSet object

columnOrdinal columnName
1 chCountryCode
2 chCountryDesc
3 chPhoneMask
4 chPostCodeMask
5 vchAddressFormat

Steps

Step data Step type
OnyxDataValidationStep.DataValidation com
OnyxDM.DataManager com

Example

The following example shows the input parameters and output parametersReturn for the retrieveList method. Notice the pageSize attribute on the rowSet element that limits the maximum rows returned to five. This example includes the returnXml to illustrate the successWithInfo status when you limit the maximum rows returned.
