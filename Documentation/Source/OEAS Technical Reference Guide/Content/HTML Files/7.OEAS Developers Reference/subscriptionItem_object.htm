subscriptionItem object

subscriptionItem object

The subscriptionItem object tracks all transactional records that are subscribed by offline users.

Methods

Method

Description

cascade

Creates and updates subscriptionItem objects based on relationship rules and filters that are applied to user-initiated subscription requests and to previously subscribed records

existenceCheck

Checks for the existence of records in the OEDB and enables ID tracking and assignment when IDs change because of customer merges

retrieveCollection

Returns objects that match search criteria

retrieveListForReview

Retrieve subscriptionItem information that match the passed in offline user and other filter criteria

retrieveListForStatus

Retrieve subscriptionItem information for the specified record and offline user

saveCollection

Performs multiple actions at once: inserting, updating, and deleting objects, as appropriate

subscribe

Creates and updates subscriptionItem object instances based on user-initiated requests

Properties

Property

Description

insertBy

Contains the user ID of the Onyx user who initially saves the object

insertDate

Contains the time and date on which the object is initially saved

objectPrimaryKey

Contains the identifier of the object instance to which a user is subscribed

objectType

Contains the type enumeration of the object instance to which the user is subscribed

onyxTimestamp

Facilitates record locking with the Onyx Enterprise Database

primaryId

The ID assigned to the object when it is first saved to the database

stagingServerItemId

Contains the identifier of the subscribed object on the staging server

stagingServerItemVersion

Contains the timestamp of an object while it is offline

subscriptionServiceId

Contains the identifier of the subscription service that was used to take the object instance offline

syncAction

Contains a code that the synchronization server uses to track subscribed records

updateBy

Contains the user ID of the Onyx user who last saved modifications to the object

updateDate

Contains the time and date on which modifications to the object were last saved
