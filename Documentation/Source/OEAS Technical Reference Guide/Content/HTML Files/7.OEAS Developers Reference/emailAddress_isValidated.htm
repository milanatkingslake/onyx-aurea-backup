isValidated Property

The isValidated property of the emailAddress object reflects whether it is QAS validated.

Data type

Integer

Access

Read/write

Remarks

This property is added to determine accuracy of the email addresses recorded in Onyx database. Email address is validated through Experian QAS Address validation APIs.

This property can contain any of the following three values:

  • 0: Email address record is not verified by QAS
  • 1: Email address record is verified by QAS and is considered a valid email address.
  • -1: Email address record is verified by QAS. However, it is considered an invalid email address
