sourceId (incident)

sourceId property

The sourceId property of the incident object contains the code identifying the activity such as letter, phone call, or RFP that initiated the incident.

Data type

i4

Access

Read/write

Remarks

The sourceId property references an existing source type in the database. Incident categories own source types, therefore the value in the sourceId property must reference a child value of the incidentCategory for a given incident.

For example, if the incidentCategory is Sales (categoryId), your source types (sourceId) might be any of the following: Mail, Phone, E-Mail, Fax. These are default source types. You can add your own using the Reference Table Administration tool.
