alias (searchTable)

alias property

The alias property of the searchTable object contains a string used as an alias in the query.

Data type

string (maximum length)

Access

Read/write

Remarks

This is typically a single or double letter abbreviation that facilitates usability and readability. For example, in the default "Companies" view, the alias for the company_lm_view searchTable is "B". The UI can reference this value when constructing a SQL query for a searchViewDetail.

You must use unique aliases for all searchTable objects in a given searchView. To enable reusability of searchTable objects in multiple searchView objects, you should make the alias unique.
