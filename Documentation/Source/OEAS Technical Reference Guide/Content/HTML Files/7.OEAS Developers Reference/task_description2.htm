description2 (task)

description2 property

The description2 property of the task object contains user input text information for a single incident object.

Data type

string (maximum length)

Access

Read/write

Remarks

Contains the continuation of user input text that exceeds the limit of the description1 property. It is the client application's responsibility to place overflow from the description1 property into the description2 property.

The value of the description2 property is typically provided by end-users.
