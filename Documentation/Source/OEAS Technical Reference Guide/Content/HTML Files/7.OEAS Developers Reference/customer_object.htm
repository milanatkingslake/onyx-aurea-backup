Customer object

The Customer Feature's customer object includes methods that are common to a Customer (both a Company and an Individual). These methods enable users to merge two existing customer records into a single customer record, retrieve a list of customer records, and retrieve specific customer information based upon specific search criteria.

A customer object is a logical object that is more fully defined by the company, individual, phone, email address, and address objects. You can use the customer object when you want to perform an action that is generic to both the company and individual record. For example, by using the customer.retrieveList method to retrieve a list of customer records, OEAS returns both company and individual records.

The customer object is comprised of the following transactional objects: individual, company, phone, emailAddress, and address objects. Non-transactional objects that provide other important customer data include the companyFamily, country, and region objects.

Methods

Method Description

merge

Merges two company or two individual records into a single record of that type

retrieveItemInfo

Retrieves the customer type (individual or company) and other information based on the primaryId

retrieveList

Retrieves a set of customer (individual and/or company) data

retrieveListByIdList

Returns the customer data for the list of customer IDs

retrieveListByPartnerAccess

Retrieves the customers by partnerAccessIds

retrieveListByWorkTicket

Retrieves information for a customer that is associated with a particular work ticket

retrieveListEMF

Retrieves a set of customer (individual and/or company) data that satisfies specific search criteria

retrieveListFavoritesByUser

Retrieves a list of favorite customers, which are individuals and companies for which the userId is set up as an internal contact of type "Favorite"

retrieveListHierarchy

Retrieves the entire organization chart structure for a company

retrieveListHistory

Retrieves a list, sorted by last update date, of recent customer activity consolidated from: Attachments, Campaigns, Comments, Incidents, Literature Orders, Products, Surveys, Tasks, and Work Tickets data

retrieveListTopNByUser

Retrieves a list of customers recently updated by a specified user

retrievePartnerListByAccess

Returns the customer data for a partner by the list of users or partner group access IDs

retrievePartnerListOfLeads

Returns the list of existing customer leads for a partner

retrievePartnerListOfNewLeads

Returns the list of new customer leads for a partner

retrievePartnerUserList

Allows a partner user to view their customers as in the
"My Customer" selection

retrieveSummaryInfo

Retrieves information of the customer, based on the primaryId

retrieveusersByFavouriteOwnerAnd Users

Retrieves a list of users of favorite customer which is filtered by the list of user id
