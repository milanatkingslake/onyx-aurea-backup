ConstraintDataEnum

ConstraintDataEnum enumerates the columns of the Constraints array, as returned, of the GetPropertyConstraints method of the ICache interface. The possible values of this enumeration are as follows:

Name (value)

Description

ocdPropertyName (0)

The name of the property.

ocdPropertyType (1)

The property type. See property attributes in the Object Designer for valid values.

ocdConstraintName (2)

The name of the constraint. The valid values are "required," "min," "max," and "default."

ocdConstraintValue (3)

The constraint value.
