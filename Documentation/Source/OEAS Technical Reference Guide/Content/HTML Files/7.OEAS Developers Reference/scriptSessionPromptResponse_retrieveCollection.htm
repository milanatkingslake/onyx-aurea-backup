retrieveCollection (scriptSessionPromptResponse)

retrieveCollection method

The retrieveCollection method of the scriptSessionPromptResponse object searches the database and returns a collection of zero or more scriptSessionPromptResponse objects that match the search criteria.

retrieveCollection standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Returns a collection of scriptSessionPromptResponse objects

Example
