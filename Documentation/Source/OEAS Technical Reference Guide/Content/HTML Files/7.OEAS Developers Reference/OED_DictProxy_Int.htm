DictionaryProxy Interface

DictionaryProxy interface

The DictionaryProxy interface provides a mechanism for out-of-process client applications, including those based on scripting languages, to obtain object metadata information from the Onyx Enterprise Dictionary (OED) cache. These methods are supported through Visual Basic, VBScript, and JScript.

The methods of the DictionaryProxy interface are listed below:

Method

Description

GetConfiguredProperties

Deprecated

GetObjectDefinition

Retrieves the object definition of a specified object

GetObjectInstance

Retrieves an empty object instance of a specified object

GetObjectList

Retrieves an XML string from the OED cache for a given logical application

GetPropertyConstraints

Retrieves the property constraints of a specified property
