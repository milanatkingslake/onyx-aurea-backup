saveCollection (emailAttachment)

saveCollection method

The saveCollection method of the emailAttachment object performs multiple actions at once: inserting, updating, and deleting object instances, as appropriate.

saveCollection method standard behavior

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Inserts, updates, and deletes a collection of emailAttachments objects

Remarks

Client applications should not call this method directly. Only the insert and update methods of the parent object, emailMessage, should call this method.
