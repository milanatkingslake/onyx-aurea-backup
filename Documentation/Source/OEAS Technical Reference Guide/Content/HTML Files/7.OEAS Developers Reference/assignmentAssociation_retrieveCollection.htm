retrieveCollection (assignmentAssociation)

retrieveCollection method

The retrieveCollection method of the assignmentAssociation object returns a collection of objects based on the search criteria supplied in the parameters XML.

retrieveCollection standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieves a collection of assignmentAssociation objects
