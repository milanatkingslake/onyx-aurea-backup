Set up the development environment

Before you begin coding, set up your development environment carefully to ensure that the step component that you create is compatible with OEAS.

Using the Visual Basic IDE, set up the development environment accordingly:

  1. Create a new ActiveX DLL project.
  2. Name your project and class. For this sample, name your project "CustomStep.vbp" and name the class "StepComponent.cls".
  3. Reference the appropriate libraries:
  • Onyx IStep 1.0 Type Library (which includes a reference to the Microsoft XML DOM, version 4.0)
  • OnyxEBusinessQE Type Library 1.0
