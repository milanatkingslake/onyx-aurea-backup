fileLocation object

The fileLocation object enables users to retrieve path names to resources used for saving and retrieving file attachments.

The fileLocation object has one key property, primaryId, that uniquely identifies the object instance. Once the object is created, the value of this property cannot be changed.

Method

Method Description

retrieveCollection

Returns a collection of all fileLocation object instances for the site

saveCollection

Performs multiple actions in a single OEAS method call such as inserting, updating, and deleting objects, as appropriate

Properties

Property Description

active

Indicates whether the resource is active or inactive

description

A free-text description of the resource used for saving attachments

insertBy

The ID of the end-user who created the object instance

insertDate

The date on which the end-user created the object instance

pathName

A path of the resource used for saving and retrieving attachment files

primaryId

A unique numerical value that OEAS assigns to a fileLocation object instance

updateBy

The user ID of the person who last updated the object instance

updateDate

The date of the last object instance update
