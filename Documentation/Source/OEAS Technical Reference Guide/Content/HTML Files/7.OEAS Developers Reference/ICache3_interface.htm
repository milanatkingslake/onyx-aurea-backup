ICache3 interface

The ICache3 interface inherits all of the methods of the ICache and ICache2 interfaces and exposes one additional method, GetObjectList. These methods enable a caller to retrieve information for the business objects that are stored in the Onyx Enterprise Dictionary (OED) cache.

Note: This interface is intended for use by step components that run in-process to the OEAS COM+ application. See the DictionaryProxy interface for information on accessing the OED cache through an out-of-process client application.

The methods of the ICache3 interface are listed below:

Method

Description

GetConfiguredProperties

Deprecated

GetObjectDefinition

Retrieves a logical object definition for a specified object

GetObjectDefinitionAsDOM

Retrieves a pointer to a logical object definition for a specified object

GetObjectInstance

Retrieves an empty object instance of a specified object

GetObjectList

Retrieves an XML string from the OED cache for a given logical application.

GetPropertyConstraints

Retrieves the property constraints for a specified property

GetSteps

Retrieves the steps for a specified object and method

The enumerations of the ICache interface are listed below:

Enumeration

Description

OEDCacheStatusEnum

Indicates the return status for each of the methods.

ConfiguredPropertyDataEnum

Identifies the columns of the ConfiguredProperties array.

ConstraintDataEnum

Identifies the columns of the Constraints array.

StepDataEnum

Identifies the columns of the Steps array.
