addresses (company)

addresses property

The addresses property of the company object represents a collection of address objects for the customer.

Data type

object

Access

Read/write

Remarks

An addresses object contains a collection of one or more address objects. A customer can have multiple addresses that are accessed via the addresses collection. The addresses property, however, does not correspond to any object in the Onyx Enterprise Dictionary (OED). Instead, the addresses property is defined as a collection of address objects.
