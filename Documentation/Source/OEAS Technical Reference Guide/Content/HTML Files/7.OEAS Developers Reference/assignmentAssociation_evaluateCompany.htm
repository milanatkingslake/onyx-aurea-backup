evaluateCompany (assignmentAssociation)

evaluateCompany method

The evaluateCompany method of the assignmentAssociation object causes the assignment of the company record to be evaluated for territory assignment. This method can be called to allow for user-initiated reassignments of company records.

Parameter

Parameter Direction Data type Content type Description
company in object keysOnly A company object with a valid primaryId property

Step

The default step for the evaluateCompany method of the assignmentAssociation object is:

Step data Step type Description
OnyxBPExecuteStep.BPExecuteStep business process Launches the business process that evaluates the company record for assignment to a territory
