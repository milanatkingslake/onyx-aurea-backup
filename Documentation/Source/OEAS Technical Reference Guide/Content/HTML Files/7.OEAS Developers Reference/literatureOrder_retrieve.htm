retrieve (literatureOrder)

retrieve method

The retrieve method of the literatureOrder object searches the database by primaryId and returns a literatureOrder object instance and all its literatureOrderLine detail objects.

retrieve standard method information

com

Propagates primaryId value of master object into the ownerId properties of the literatureOrderLines collection

OnyxDM.DataManager

Retrieves a literature order record

method

Retrieves a literatureOrderLine collection which contains all literatureOrderLines associated with a literatureOrder
