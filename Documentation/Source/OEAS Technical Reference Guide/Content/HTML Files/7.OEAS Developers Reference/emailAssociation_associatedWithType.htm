associatedWithType (emailAssociation)

associatedWithType property

The associatedWithType property of the emailAssociation object contains a value that specifies the type of object associated with the email and the circumstance of the association.

Data type

i4

Access

Read/write

Remarks

The default implementation of OEP supports the following objects for email associations:

Value Object type
1 Individual
2 Company
6 Incident
