ISession2 interface

The ISession2 interface inherits all of the methods of the ISession interface and exposes one additional method, GetContextXML2.  These methods enable a caller to retrieve user information and control the session.

The ISession2 interface operates in the same manner as older versions of ISession.  Although Onyx supports both interfaces, all ISession calls pass to ISession2 for execution.

The methods of the ISession2 interface are listed below:

Method

Description

ClearSessionItems

Clears the values of all session variables.

EndSession

Ends a session.

GetContextXML

Retrieves session data such as user ID, preferred language code, and site ID.

GetContextXML2

Retrieves session data with an additional flag that indicates whether the session access time should be updated. The default value is True.

GetSessionItem

Retrieves the value of session variable.

GetSessionItems

Retrieves all session variables and their values.

HasPermission

Determines whether a user has permission to use a resource.

HasPermissions

Determines whether a user has permission to use several specified resources.

RemoveSessionItem

Removes the value of a session variable.

SetImpersonation

Enables a system service or component identity user to impersonate end-users.

SetSessionItem

Saves the value of a session variable.
