sequenceNumber (methodCallDetail)

sequenceNumber property

The sequenceNumber property of the methodCallDetail object contains the ordinal value of the methodCallDetail relative to details of the same methodCall.

Data type

i4

Access

Read/write

Remarks

The methodCallDetails for a given methodCall are sorted by sequenceNumber which controls the creation order of nodes in the input XML. Onyx recommends that the sequenceNumber be unique for each methodCallDetail of a given methodCall object, but this is not a constraint. If the sequenceNumber is not unique, your method call might fail.
