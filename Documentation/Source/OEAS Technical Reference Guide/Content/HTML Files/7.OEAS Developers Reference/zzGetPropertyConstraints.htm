GetPropertyConstraints (ICache)
GetPropertyConstraints method

The GetPropertyConstraints method of the ICache interface retrieves the constraints for a property of a business object.

Syntax

IDL

HRESULT GetPropertyConstraints(

    [in] BSTR ApplicationName, 

    [in] BSTR ObjectName, 

    [in] BSTR PropertyName, 

    [out] SAFEARRAY(VARIANT)* Constraints, 

    [out, retval] OEDCacheStatusEnum* Status);

IDL parameter descriptions

ApplicationName

[in] A string containing the logical application name that determines which subset of the cache to use for a query.

ObjectName

[in] A string that contains the name of the specified object.

PropertyName

[in] A string that contains the specified property name.

Constraints

[out] A two-dimensional array of variants (SAFEARRAY) that contains the property constraints for the business object.

Status

[out, retval] An enumeration of type OEDCacheStatusEnum that contains the status of the method call.

Return values

This method returns an enumeration of type OEDCacheStatusEnum that contains the status of the method call. See the OEDCacheStatusEnum topic for a description of individual enumeration values.

Example

The following Visual Basic code demonstrates how to use the GetPropertyConstraints method to determine the restraints, if any, on the incident object's primaryId property.

Dim oCache As OnyxDictionaryCacheLib.DictionaryCache

Dim status As OnyxDictionaryCacheLib.OEDCacheStatusEnum

Dim constraints() As Variant

Dim sPropertyName As String

Dim sPropertyType As String

Dim sConstraintName As String

Dim sConstraintValue As String

'Set the cache and identify the object and property in the method

Set oCache = New OnyxDictionaryCacheLib.DictionaryCache 

status = oCache.GetPropertyConstraints("Onyx", "incident", "primaryId", constraints())

'Set the variables to the content of the array

If status = ocsSuccess Then

  sPropertyName = constraints(0, ocpPropertyName)

  sPropertyType = constraints(0, ocpPropertyType)

  sConstraintName = constraints(0, ocpConstraintName)

  sConstraintValue = constraints(0, ocpConstraintValue)

End If 

Note: For an invalid object name or property name, the error message is "Property not found" because of how items are keyed in the cache.
