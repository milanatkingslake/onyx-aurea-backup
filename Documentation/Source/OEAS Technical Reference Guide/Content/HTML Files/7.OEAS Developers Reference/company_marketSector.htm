marketSector (company)

marketSector property

The marketSector property of the company object reflects the company's sector of the market, such as finance, health care, technology, and so on.

Data type

i4

Access

Read/write

Remarks

The marketSector property is a unique ID for a category to which a company belongs and can be configured using the Reference Table administration tool. An example of a marketSector might be "103", which could have a description of "Hi Tech" in the referenceParameter object.
