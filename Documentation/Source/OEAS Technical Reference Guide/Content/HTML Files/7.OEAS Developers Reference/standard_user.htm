user1-userN property

The user1-userN properties can contain any kind of textual information, as you define for your customization.

Data type

string

Access

Read/write

Remarks

The user1-userN properties are optional properties that you can customize on a per-site basis.

These fields are sometimes called "user-defined fields" or UDFs. You can modify the display of existing UDFs by using OES Reference Table Administration and UI Configuration Workbench (UCW).
