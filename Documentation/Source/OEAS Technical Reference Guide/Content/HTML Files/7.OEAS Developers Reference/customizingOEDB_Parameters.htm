Parameters

The following sections describe parameters that are important to understand when customizing or creating procedures for the OEDB. They include:

  • Parameters for communicating error and message information, including:
  • @message_code
  • @error_code
  • @message_parameter
  • Parameters that support context values, including:
  • @context_user_id
  • @context_site_id
  • @context_language_code
