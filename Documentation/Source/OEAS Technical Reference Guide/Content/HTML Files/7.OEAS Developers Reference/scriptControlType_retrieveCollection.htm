retrieveCollection (scriptControlType)

retrieveCollection method

The retrieveCollection method of the scriptControlType object searches the database and returns a collection of zero or more scriptControlType objects that match the search criteria.

retrieveCollection standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Returns a collection of scriptControlType objects

Example

The following example shows a retrieve collection in which no criteria are specified. As a result, OEAS returns all scriptControlType objects in the database.

Parameters

<parameters>

   <scriptControlTypes objectType="scriptControlType" collection="1">

      <scriptControlType objectType="scriptControlType"/>

   </scriptControlTypes>

</parameters>
