Table attributes

When you select a table in the Object Designer, the attributes for that table display in the edit area.

The attributes are defined as follows:

Attribute Name

Description

ID

The globally unique identifier (GUID) assigned by the Onyx Enterprise Dictionary. This attribute is read-only.

Name

Type the name you want to assign to this table.

Description

Type the description for this table.

Extends Table

Select the name of the table you want to extend. This list includes all active tables in the OEDB. Use this field if you need to add properties to an object. You may create an extension table rather than add columns to the object's existing table. The extension table has a one-to-one relationship with the table it extends, except in the case of multi-language tables where a one-to-many relationship occurs.

Multi Language Table

Check this box if the table is an extension table or a view for providing multi-language support. This informs SQL generation that WHERE clauses used against this table must filter on the language_code column using the user's preferred language from the context XML.

Released

This attribute is for documentation purposes only. Check this box to document that the table is no longer under development.

Installed By

Type the name of the vendor or application installing this table.

Onyx Version

The Onyx version number of this table. This attribute is read-only.

Custom Version

Type the custom version number you want to assign to this table.
