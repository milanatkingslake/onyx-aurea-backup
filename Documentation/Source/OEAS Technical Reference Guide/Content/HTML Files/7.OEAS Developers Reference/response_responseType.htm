responseType (response)

responseType property

The responseType property of the response object indicates whether a response can have free text associated with it

Data type

i4

Access

Read/write

Remarks

This property only applies to responses to single-answer and multiple-choice questions. It allows the user to include free text with a response that you have specified.

Type

Description

1

Normal--no free text allowed

2

Free text allowed in addition to specified response text
