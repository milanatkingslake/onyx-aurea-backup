retrieveListMobileSearch method

The retrieveListMobileSearch method of the incident object returns the list of objects that meets the specified criteria.

Parameters

Parameter Direction Data type Content type Description

primaryId

in

string

n/a

The incident ID from the incident table

secondaryId

in

i4

n/a

Mirrors the primaryId of the incident object

incidentCategory

in

string

n/a

comma-delimited string consisting of the category of the incident you want to return

ownerId

in

string

n/a

The company or individual who owns the incident

companyName

in

string

n/a

Name of the company who is the contact of the incident

firstName

in

string

n/a

First name of the contact of the incident

lastName

in

string

n/a

Last name of the contact of the incident

description1

in

string

n/a

The description of the incident

typeId

in

string

n/a

The integer reference to the type of the incident

statusId

in

string

n/a

The integer reference to the status of the incident

sourceId

in

string

n/a

The integer reference to the source of the incident

priorityId

in

string

n/a

The integer reference to the priority of the incident

assignedTo

in

string

n/a

A user-defined field which can be used as a key to an external database

incidentProductCode

in

string

n/a

The code for an incident product

includeMask

in

string

n/a

Switches to determine masking behavior of the incident component for a given retrieveListByFilter call.

rowSet

out

object

all

Contains the rowSet column definitions and the incident table rows matching with the owner and category criteria
