parentId (company)

parentId property

The parentId property of the company object contains the primaryId of the parent company.

Data type

string (maximum length)

Access

Read/write

Remarks

To enable compatibility with other applications, the data type for this property is a string. However, OEAS typically uses this property as a GUID value.

An empty, null, or a missing node for the parentId indicates that the company has no parent.
