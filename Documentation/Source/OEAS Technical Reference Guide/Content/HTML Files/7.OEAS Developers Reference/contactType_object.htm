contactType object

The contactType object classifies the various types of contacts within the system. It also enables contact types to be organized within a hierarchy of contact categories and groups for controlling display in the user interface. By default, this object is organized by categoryId (contacts that are associated with an individual, company, product, work ticket, or incident) and groupTypeId (a customer contact or an internal contact to a userl).

The contactType object has one key property, primaryId, which uniquely identifies the contactType in the database. Once the object is created, the value of this property cannot be changed.

Methods

Method Description

Properties

Property Description

attributeId Reserved for backward compatibility

groupTypeId Identifies the contact group type to which the contact type belongs
primaryId A unique ID assigned to the object when it is first saved to the database
