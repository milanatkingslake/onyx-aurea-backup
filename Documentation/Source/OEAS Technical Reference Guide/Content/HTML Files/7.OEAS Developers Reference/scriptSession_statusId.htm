statusId (scriptSession)

statusId property

The statusId property of the scriptSession object contains a constant signifying the stage of completion for the script.

Data type

i4

Access

Read/write

Remarks

The following table shows the default valid values for the statusId property. These values are stored as extra data on the scriptSession status reference parameters and must not be changed.

Value

Enumeration

1

New

2

In progress

3

Suspended
4 Completed
