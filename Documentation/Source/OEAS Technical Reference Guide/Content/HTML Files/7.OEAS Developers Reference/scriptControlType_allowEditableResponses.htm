allowEditableResponses (scriptControlType)

allowEditableResponses property

The allowEditableResponses property of the scriptControlType object specifies whether a response supports additional text in a textbox.

Data type

Boolean

Access

Read/write

Remarks

A value of 1 specifies that the control can have a textbox in which a user can enter an additionalText value for a scriptSessionPromptResponse. For example, you might have a dropDownListBox with choices of red, green, or blue. If the script designer specifies the responses as editable, then the user can enter a modifier such as light or dark in the associated textbox, and it is persisted as additionalText along with the color value on the scriptSessionPromptResponse. Alternatively, you might have choices of red, green, blue, and other, among which only the 'other' choice is marked as editable. In this case, the user can select the 'other' value and enter purple as an alternate color choice in the textbox. The scriptSessionPromptResponse is persisted with a value of 'other' and additionalText of 'purple'. The following table summarizes this scenario.

Note: Even if allowEditableResponses is set to true for a control type, no textbox is rendered in the user interface unless a response associated with a control instance is marked as editable.
