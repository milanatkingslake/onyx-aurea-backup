retrieveCollection (subscribableObject)

retrieveCollection method

The retrieveCollection method of the subscribableObject object searches the database and returns a collection of zero or more subscribableObject objects that match the search criteria.

retrieveCollection standard method information

Steps

Step type

Description

com

Validates input parameters

OnyxDM.DataManager

com

Retrieves a collection of subscribableObject objects
