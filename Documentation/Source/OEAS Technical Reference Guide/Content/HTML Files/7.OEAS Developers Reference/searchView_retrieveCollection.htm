retrieveCollection (searchView)

retrieveCollection method

The retrieveCollection method of the searchView object returns a collection of objects based on the search criteria supplied in the parameters XML.

retrieveCollection standard method information

Steps

Step type

Description

OnyxDataValidationStep.DataValidation

com

OnyxDM.DataManager

com
