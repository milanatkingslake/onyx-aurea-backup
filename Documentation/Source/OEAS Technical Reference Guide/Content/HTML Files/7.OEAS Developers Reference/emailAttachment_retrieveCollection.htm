retrieveCollection (emailAttachment)

retrieveCollection method

The retrieveCollection method of the emailAttachment object returns a collection of objects based on the search criteria supplied in the parameters XML.

retrieveCollection method standard behavior

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieves a collection of emailAttachments

Remarks

The stepAdditionalData node for this method contains the disableEmbeddedXml=1 string. The emailMessage send method constructs an email message by calling the emailMessage retrieve method, which in turn, calls this retrieveCollection method as a method step. The embeddedXml feature must be turned off for the send method to succeed.
