saveCollection (literatureOrderLine)

saveCollection method

The saveCollection method of the literatureOrderLine object performs multiple actions at once: the insert, update, and delete of literatureOrderLine objects, as appropriate.

saveCollection standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Data validation step

OnyxPSLabelStep.Label

com

When product has an associated mailing label, inserts label orderlines

OnyxDM.DataManager

com

Performs multiple actions with literatureOrderLine objects

On insert actions, calls a database function mapped to the lineNumber property to calculate the next sequential line number

Calls a post-procedure that sets a printer name when the printerName element is null and the fulfillment type is print document (6)
