associationType (workTicketAssociation)

associationType property

The associationType property of the workTicketAssociation object contains a unique numerical value that identifies the type of object that is associated with the work ticket.

Data type

i4

Access

Read/write

Remarks

Valid values of the associationType property include:

Valid value Object type

1

individual

2

company

6

incident

10

workTicket

These values are defined within the corresponding step component, not a database table. Attempts to pass in an invalid value fail.
