allowText (scriptControlType)

allowText property

The allowText property of the scriptControlType object specifies whether a control supports a text component in its display.

Data type

Boolean

Access

Read/write

Remarks

Most controls are not self-explanatory and require some label text to prompt the user before the user selects or enters a response. This property specifies that the control supports a text label. The default controls in Scripting all enable the text component except the divider control. The divider control is a graphical symbol and needs no explanation. In the default script delivery interface, a control is displayed in two columns with the text column to the left of the control. By disabling text on the divider control, the control can span both columns instead of only the right column.
