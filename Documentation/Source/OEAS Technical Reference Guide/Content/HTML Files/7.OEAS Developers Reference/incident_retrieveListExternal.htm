retrieveListExternal (incident)

retrieveListExternal method

The retrieveListExternal method of the incident object returns the incident data needed by OCP along with information on the incident's owner and reminder.

Parameters

Parameter Direction Data type Content type Description

incidentId

in

string

n/a

The incident's primaryId value. Note that a value must be provided for either the incidentId or the ownerId.

ownerId

in

string

n/a

The incident's ownerId value. Note that a value must be provided for either the incidentId or the ownerId.

categoryId

in

i4

n/a

The category of the incident

sortColumn

in

i4

n/a

The numerical position of the column to sort

sortOrder

in

string

n/a

The direction of the sort. Valid values are asc or desc..

numDays

in

i4

n/a

The number of days before the current date that should be used to search for incidents

rowSet

out

object

all

Contains the rowSet column definitions and the incident table rows matching with the owner and category criteria
