epDirection

epDirection enumerates the direction of parameters for a stored procedure call. Use this enumeration when configuring parameters for the CallStoredProcedure and CallStoredProcedureReturnExtendedInfo methods of the IEBusQE interface.

Element name (value)

Description

apdParameterInput (1)

Used for input

apdParameterOutput (2)

Used for output

apdParameterInputOutput (3)

Used for both input and output

Remarks

See the Stored procedure parameters topic for information on the preparation of parameters for use with the Query Engine.
