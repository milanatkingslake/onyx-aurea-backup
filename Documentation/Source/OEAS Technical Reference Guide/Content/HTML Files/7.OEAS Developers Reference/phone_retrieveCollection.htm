retrieveCollection (phone)

retrieveCollection method

The retrieveCollection method of the phone object returns a collection of objects based on the search criteria supplied in the parameters XML.

retrieveCollection standard method information

Steps

Step data Step type Description
OnyxDataValidationStep.DataValidation com Validates input parameters
OnyxDM.DataManager com Retrieves a collection of object instances

Example

Parameters

<parameters>

   <phones objectType="phone" collection="1">

      <phone objectType="phone">

         <ownerId>535722DD-E3BD-4A25-ABE9-6F24185CF74C</ownerId>

      </phone>

   </phones>

</parameters>
