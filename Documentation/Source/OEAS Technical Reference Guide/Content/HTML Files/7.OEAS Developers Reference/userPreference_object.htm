userPreference object

The userPreference object lets clients retrieve and maintain user preferences, which are stored in the Onyx Enterprise Database.

The userPreference object has three key properties, userId, preferenceName, and profileId, that uniquely identify the preference in the database. All of these properties are read/write.

Methods

Method Description

delete

Deletes a userPreference object

insert

Inserts a userPreference object

Returns a userPreference object

Returns a collection of object instances that match the search criteria

Returns a list of userPreference objects for a user

Searches the database by the userId and returns the list of the most recently used customer records

Returns a list of the most-recently-used customer records by a user

Properties

Property Description
