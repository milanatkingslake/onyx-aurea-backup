closeProbability (forecast)

closeProbability property

The closeProbability property of the forecast object contains a number (interpreted as a percentage) that represents the probability that the transaction represented by the forecast will actually occur.

Data type

int

Access

Read/write

Remarks

This weighting factor is applied to the final forecast total after discount and confidence factor have been applied to the individual forecastLine items.

The valid values range from 0.0 and 100.0, inclusive.
