literatureOrders (OPS)

literatureOrders

The literatureOrders node of the orderBatch XML contains the current set of literatureOrder objects that the fulfillment engine processes. The literatureOrders node includes a collection of literatureOrder child elements. The literatureOrder objects contain a collection of literatureOrderLine objects. The literatureOrder and literatureOrderLine objects contain the detailed information required to send a literature product to a customer. For more information on the literatureOrder and literatureOrderLine objects, see the Business Objects section of this reference.

Note: For a complete orderBatch example, you can look in the Utilities directory of the product CD for orderBatchEmailSample.xml and orderBatchPrintSample.xml.

The following is an example of the literatureOrders node of the orderBatch XML.

<literatureOrders objectType="literatureOrder" content="all">

   <literatureOrder objecttype="literatureOrder" content="all">

      <literatureOrderLines>

         <literatureOrderLine objecttype="literatureOrderLine" content="all"/>

      </literatureOrderLines>

   </literatureOrder>

   <literatureOrder/>

   ... 

   <literatureOrder/>

</literatureOrders>
