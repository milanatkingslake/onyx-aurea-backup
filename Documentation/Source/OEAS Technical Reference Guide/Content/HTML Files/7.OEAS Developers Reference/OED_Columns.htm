Columns

The columns collection is owned by a table entity. The collection holds all of the column entities that describe physical columns in the database table. Each column entity has attributes for name which must be unique within the table.

  • description
  • data type
  • length
  • "not null"
  • "installed by"

The length and the data type are not used but are there for completeness and could be used in later versions of OEAS. The data types used for the OEAS Oracle version are specific to the Oracle database engine.
