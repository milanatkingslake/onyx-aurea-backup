attendeeTypeCode (appointmentAttendee)

attendeeTypeCode property

The attendeeTypeCode property of the appointmentAttendee object identifies the type of attendee.

Data type

i4

Access

Read/write

Remarks

Attendees can be identified in one of the following categories.

Value Description
1 Meeting organizer
2 Required attendees
3 Optional attendees
4 Resources (rooms or other equipment)
