forecastLine object

The Forecast/Quotes feature's forecastLine object provides properties that capture the details of a forecast.

The forecastLine object has a key property, primaryId, that uniquely identifies the forecast data in the database. Once the object is created, the value of the primaryId property cannot be changed.

Methods

Method Description
retrieveCollection Returns a collection of forecastLine objects
saveCollection Performs multiple actions at once: inserting, updating, and deleting objects, as appropriate
retrieveListForecastLineMobile Retrieves forecast line items for Mobiles
