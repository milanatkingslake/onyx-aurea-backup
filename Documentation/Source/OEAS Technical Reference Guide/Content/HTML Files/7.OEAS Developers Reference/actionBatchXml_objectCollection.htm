actionBatchXml_objectCollection
[object collection node]

An element is only required in this location in the actionBatchXml if you are using a source list. There is no element named "object collection." You provide the element names and structure for your source list item objects in this location in the actionBatchXml. To define your source list item objects you can use any element name and structure, but the structure must agree with the paths provided in the sourceListMaps.

The source list item objects:

  • Do not need to be OEAS objects
  • Do not need to be structured like a standard OEAS collection
  • Do not need to include any data that is not used in the sourceListMaps

For an example of an actionBatchXml document using a source list, see Processing an actionBatchXml document.

Cardinality

1 (required if sourceList is present)

Default value

none
