literatureProducts

The literatureProducts node of the orderBatch XML contains the literatureProduct objects for the current batch of orders. Each literatureProduct corresponds to one or more orders in the literatureOrders collection. The subsystem process creates the literatureProducts dynamically based on the contents of the literatureOrders node.

Note: For a complete orderBatch example, you can look in the Utilities directory of the product CD for orderBatchEmailSample.xml and orderBatchPrintSample.xml.

The following is an example of the literatureProducts node of the orderBatch XML.

<literatureProducts objectType="literatureProduct" content="all">

   <literatureProduct objecttype="literatureProduct" contents="all"/>

</literatureProducts>

For more information on the literatureProduct object, see the Business Objects section of this reference.
