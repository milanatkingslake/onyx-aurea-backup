createDate (quote)

createDate property

The createDate property of the quote object contains the time and date on which the quote was initially created.

Data type

dateTime

Access

Read/write

Remarks

The createDate property value is a user input entry. Users can set its value to any date and time.

The dateTime data type has the following format:

YYYY-MM-DD hh:mm:ss

where:

YYYY = year

MM = month

DD = day

hh = hour

mm = minute

ss = second

Valid values for the dateTime data type range from 1900-01-01 00:00:00 to 9999-12-30 23:59:59.
