literatureServicePrinters (OPS)

literatureServicePrinters

The literatureServicePrinters node of the orderBatch XML contains the current set of literatureServicePrinter objects available to fulfill the current batch of orders. Each literatureServicePrinter in the literatureServicePrinters node corresponds to a physical printer configured in OEAS.

Note: For a complete orderBatch example, you can look in the Utilities directory of the product CD for orderBatchEmailSample.xml and orderBatchPrintSample.xml.

The following is an example of the literatureServicePrinters node of the orderBatch XML.

<literatureServicePrinters objectType="literatureServicePrinter" collection="1">

   <literatureServicePrinter objectType="literatureServicePrinter">

      <primaryId/>

      <printerShare/>

      <printerName/>

      <insertBy/>

      <insertDate/>

      <updateBy/>

      <updateDate/>

      <onyxTimestamp/>

      <active/>

   </literatureServicePrinter>

</literatureServicePrinters>

For more information on the literatureServicePrinter object, see the Business Objects section of this reference.
