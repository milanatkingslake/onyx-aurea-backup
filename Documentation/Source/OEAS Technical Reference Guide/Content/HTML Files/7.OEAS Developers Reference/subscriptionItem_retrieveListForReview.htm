Retrieves a list of subscriptionItem records

Remarks

This method returns information about currently subscribed records only. The syncAction values for the records in the result set are 0, 1, 3, or 5.

The rows of the result set are sorted by insertDate, in descending order.

Work notes and reminders are not included in the result set.

The following XML is an example of the objectTypeList input parameter:

 <objectTypeList>1;2;6,1;6,3;8,11</objectTypeList>

Note the following:

  • The category type is required to be specified for incidents (6) and tasks (8).
  • A comma must separate the object type from the category type for incidents and tasks.
  • For each incident and task category type requested, you need to repeat the object type. In this example, two categories of incidents are requested: 6,1 and 6,3.
