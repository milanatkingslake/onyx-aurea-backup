execute (OTM Interface Sample)
Calling the OTM

You call the Onyx TransactionManager (OTM) by the ITransactionManager Execute method. To use this method, declare an object variable. Use the object variable to create a TransactionManager object. Once you have created the object, use it to call its Execute method with the appropriate parameters. The following frame shows the code used to declare the variable, create the TransactionManager object, and call the Execute method. Click any of the links to see a description of the term.

This is the method signature for the ITransactionManager Execute method. Use the hyperlinks to display a definition of the term.

dim otm

set otm = server.createobject(

                             "OnyxTransactionManager.TransactionManager"

                             )

strParamsReturn = otm.Execute(

                             strApplicationName,

                             strLogicalObjectName,

                             strLogicalMethodName,

                             strSessionId,

                             strParamXML,

                             strCustomDataXML

                             )
