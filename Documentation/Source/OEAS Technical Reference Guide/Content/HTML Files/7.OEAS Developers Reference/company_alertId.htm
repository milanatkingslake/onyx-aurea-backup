alertId (company)

alertId property

The alertId property of the company object contains the numerical ID of the alert record that is associated with the customer record.

Data type

string (maximum length)

Access

Read-only; for more information, see Read Only attribute

Remarks

The numerical ID is a foreign key value that represents the association between the company table and the alert table.

The alertId property replaces the alertNumber property which was used in OEAS prior to version 4.0.
