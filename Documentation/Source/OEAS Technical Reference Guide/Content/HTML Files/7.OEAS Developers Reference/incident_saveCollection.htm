saveCollection (incident)

saveCollection method

The saveCollection method of the incident object inserts, updates, and/or deletes a collection of specified objects.

Parameters

Parameter Direction Data Type Content Type Description

incidents

inOut

object

any(in) keysOnly(out)

Contains the incident information to be inserted, updated, or deleted

Steps

Step Data Step Type Description

OnyxCollectionManager.CollectionManager

com

Forms individual standard method calls for each object instance in the parameters and executes them through the internal OTM

Remarks

Unlike the saveCollection standard method, this saveCollection uses the OnyxCollectionManager component. The main difference is that OnyxCollectionManager wraps each object action in a separate transaction.

Example

This example shows a saveCollection that inserts incident objects for two different individuals.

Input Parameter:

<parameters>

   <incidents objectType="incident" collection="1">

      <incident objectType="incident" action="insert">

         <ownerId>9DA65A6AF30540BD87C70235D8AB290E</ownerId>

         <ownerType>1</ownerType>

         <categoryId>1</categoryId>

         <typeId>151</typeId>

         <assignedTo>sa</assignedTo>

      </incident>

      <incident objectType="incident" action="insert">

         <ownerId>8C03CBED6A7B4BA98CB24D39505B35FF</ownerId>

         <ownerType>1</ownerType>

         <categoryId>1</categoryId>

         <typeId>151</typeId>

         <assignedTo>sa</assignedTo>

      </incident>

   </incidents>

</parameters>

<parametersReturn>

   <incidents objectType="incident" collection="1">

      <incident action="insert" objectType="incident" content="keysOnly">

         <primaryId>D1117DE5-88A7-4659-A384-1048DD3621B9</primaryId>

      </incident>

      <incident action="insert" objectType="incident" content="keysOnly">

      </incident>

   </incidents>

</parametersReturn>
