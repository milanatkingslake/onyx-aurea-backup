recipientGroupIdList (message)

recipientGroupIdList property

The recipientGroupIdList property of the message object contains groupId strings that indicate who receives the message.

Data type

string (maximum length)

Access

Write-only

Remarks

A list of groupIds must be semicolon-separated.

The onyx_message.insert_method procedure converts groupIds into a semicolon-separated userId list.

The recipientGroupIdList element is empty in the parameters on return because it is a write-only property. The recipientGroupIdList or recipientUserIdList properties are required only for message inserts.

Values in either the recipientUserIdList or recipientGroupIdList elements are required on input. Either element can be NULL if the other element contains one or more recipients. Both elements can contain values on input.

OEAS adds the recipient list for a message to the top of the message text.
