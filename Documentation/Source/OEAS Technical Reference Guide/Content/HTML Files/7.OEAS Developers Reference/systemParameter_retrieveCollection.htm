retrieveCollection (systemParameter)

retrieveCollection method

The retrieveCollection method of the systemParameter object returns a collection of objects based on the search criteria supplied in the parameters XML.

retrieveCollection standard method information

Steps

Step data

Step type

Description

OnyxDataValidationStep.DataValidation

com

Validates input parameters

OnyxDM.DataManager

com

Retrieves a collection of systemParameter object instances

Example

The following example shows the content of the input parameters and output parametersReturn to retrieve a collection of systemParameter objects. In this example, you search for systemParameter objects with an dataType of 'string'. This search does not match records with a dataType of 'String', because the search is case sensitive.

Parameters

<parameters>

   <systemParameters objectType="systemParameter" collection="1">

      <systemParameter objectType="systemParameter">

         <code/>

         <value/>

         <dataType>string</dataType>

         <minimumValue/>

         <maximumValue/>

         <suggestedValue/>

         <description/>

      </systemParameter>

   </systemParameters>

</parameters>
