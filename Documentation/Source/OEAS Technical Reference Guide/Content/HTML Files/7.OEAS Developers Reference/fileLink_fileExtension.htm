fileExtension (fileLink)

fileExtension property

The fileExtension property of the fileLink object is the file extension of the physical file located on the server.

Data type

string (maximum length)

Maximum length is 10 characters.

Access

Read/write

Remarks

This file extension indicates the type of file, such as .dll, .doc, or .txt. The value of this property includes the period that separates the file extension from the file name.
