closeDate (forecast)

closeDate property

The closeDate property of the forecast object contains the date on which the transaction represented by the forecast is expected to close.

Data type

dateTime

Access

Read/write

Remarks

The closeDate property value is a user-defined entry. Users can set to its value to any date and time they desire.

The dateTime data type has the following format:

YYYY-MM-DD hh:mm:ss

where:

YYYY = year

MM = month

DD = day

hh = hour

mm = minute

ss = second

Valid values for the dateTime data type range from 1900-01-01 00:00:00 to 9999-12-30 23:59:59.
