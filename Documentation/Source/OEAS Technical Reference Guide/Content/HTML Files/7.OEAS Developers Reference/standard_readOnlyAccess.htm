readOnlyAccess property

The readOnlyAccess property of an object indicates whether an object can be updated by any user or only by the last user to update the object.

This is a generic topic that applies to all properties with the same name. The remarks section may contain details specific to a given object.

Data type

boolean

Access

Read/write

Remarks

The readOnlyAccess property works in conjunction with the updateBy property. If readOnlyAccess is set to false, the object values can be changed. If the readOnlyAccess is set to true, the object is read-only. Only the user contained in the updateBy property can update the record or can set this property to false.

Valid values for the readOnlyAccess property are:

false (0): (Default) The object can be updated by all users.

true (1): The object can be updated only by the last user to update it as recorded in the updateBy property of the object.
