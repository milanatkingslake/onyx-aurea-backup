retrieveCollection method

A retrieveCollection method of the actionBatch object returns the collection of object instances based on search criteria.

Parameters

Parameter Direction Data type Content type Description

actionBatches

inOut

object

n/a

A collection of actionBatches

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates the input parameters

OnyxDM.DataManager

com

Retrieves the collection of objects
