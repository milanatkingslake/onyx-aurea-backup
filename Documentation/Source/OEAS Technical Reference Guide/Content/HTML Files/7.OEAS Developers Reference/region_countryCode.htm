countryCode (region)

countryCode property

The countryCode property of the region object identifies the country of a region using a two letter code.

Data type

string (maximum length)

Access

Read/write

Remarks

The countryCode is a unique two-letter text code assigned to each country. There is a unique mapping between a countryCode and a countryName. You can configure the countryCode values in the COUNTRY table using the Table administration tool.

This value represents the country code that is stored on the customer record (either Company or Individual) as part of the primary address.
