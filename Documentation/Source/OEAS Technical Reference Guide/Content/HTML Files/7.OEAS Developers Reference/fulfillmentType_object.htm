fulfillmentType Object

fulfillmentType object

The fulfillmentType object enables administrators to set and select processing types for literatureProduct objects. Each fulfillmentType object is directly associated with a fulfillmentEngine object that processes a literatureProduct order. The fulfillmentTypeCode property uniquely identifies the object in the database. Typical fulfillment types include email, print, and fax.

Methods

Method Description
retrieveCollection Retrieves a collection of fulfillmentTypes
saveCollection Saves a collection of fulfillmentTypes

Properties
