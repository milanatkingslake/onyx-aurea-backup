methodCallDetail object

The methodCallDetail object contains the definition of a single OEAS method input or output parameter or property for a given methodCall object. To be successful, a methodCall must include a detail object for every node in the parameters of the method. For an individual insert methodCall for example, you create a methodCallDetail for each required element in the input XML (primaryId, secondaryId, locked, individual, phones, and so on) and for each return value in the output XML (individual/primaryId, phone/primaryId, and address/primaryId). For conceptual information on method call details, see the Method Calls topic in the feature overview section of Customizing OEAS.

The methodCallDetail object has one key property, primaryId, that uniquely identifies the object instance. Once the object is created, the value of this property cannot be changed.
