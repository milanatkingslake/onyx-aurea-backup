Mapping with functions

In Object Designer, there are two ways that you can map a property to a database column using a database function:

  1. Specify the map type as "function" and enter the function name in the "Data" area. This causes SQL generation to compose the function call and pass it the property's value as the only parameter.
  2. For functions with complex signatures select the "complex function" type and select the function from the drop-down list. The function must have already been defined in the Physical Model section of Object Designer. The name of the function in the Physical Model should be prefaced with "[dbo]." For each function parameter, identify how it should be mapped and supply an appropriate value. Your choices of map type for the parameters are:
    • contextVariable
    • literal
    • methodParameter
    • property
    • XSL literal
