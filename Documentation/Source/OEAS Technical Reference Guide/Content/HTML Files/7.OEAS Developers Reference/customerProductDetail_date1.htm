date1 (customerProductDetail)

date1 property

The date1 property of the customerProductDetail object contains user input date information for a single customerProductDetail object.

Data type

dateTime

Access

Read-only; for more information, see Read Only attribute

Remarks

The date1 property is not mapped to a physical database column. To use this property, you must modify the database schema and provide the logical to physical mapping in the OED.

The dateTime data type has the following format:

YYYY-MM-DD hh:mm:ss

where:

YYYY = year

MM = month

DD = day

hh = hour

mm = minute

ss = second

Valid values for the dateTime data type range from 1900-01-01 00:00:00 to 9999-12-30 23:59:59.
