address1 (address)

address1 property

The address1 property of the address object reflects the first address line.

Data type

string (maximum length)

Access

Read/write

Remarks

A string used to store an address for the object, the first address line on an address record. There are a total of five address fields, address1, address2, address3, address4, and address5.

Default address1 is NULL.

The following demonstrates a possible setting of the address1 through address5 properties:

<address1>515 Main Street</address1>

<address2>Building 42B</address2>

<address3>Suite 13</address3>

<address4>% John Smith</address4>

<address5>Receptionist</address5>
