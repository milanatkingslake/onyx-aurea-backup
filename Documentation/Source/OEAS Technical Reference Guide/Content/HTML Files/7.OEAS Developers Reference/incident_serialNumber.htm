serialNumber (incident)

serialNumber property

The serialNumber property of the incident object contains the serial number of the product that is linked to the incident.

Data type

string (maximum length)

Access

Read/write

Remarks

OEAS does not track serial numbers of products, and this property does not reference any existing value in the database. Therefore, it can be any value supported by the data type. This value typically represents a serial number tracked in an integrated database or in a custom table.
