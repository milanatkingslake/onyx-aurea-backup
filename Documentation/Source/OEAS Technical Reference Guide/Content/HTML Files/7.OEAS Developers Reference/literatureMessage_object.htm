literatureMessage Object

literatureMessage object

The literatureMessage object represents subject and body text information that you can add to a literature order. You can make a literatureMessage object reusable, so users can choose from a list of predefined messages to add to a literature order. Typically, you use this object to include text entered by the end-user in the emails or printed documents output from the Onyx Processing Server.
