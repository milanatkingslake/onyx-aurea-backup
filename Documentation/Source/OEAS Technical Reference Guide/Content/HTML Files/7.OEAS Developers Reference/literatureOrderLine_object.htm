literatureOrderLine Object

literatureOrderLine object

A literatureOrderLine object contains information regarding a single item in a request for literature products including price, quantity, shipping status, and so on.

The literatureOrderLine object has one key property, primaryId, that uniquely identifies the order data in the database. This property is a system generated GUID and cannot be modified.

Because literatureOrderLine objects participate in a master detail relationship with a parent literatureOrder object, a client application should not call the literatureOrderLine methods directly. All manipulation of literatureOrderLine objects must be done through the parent object.
