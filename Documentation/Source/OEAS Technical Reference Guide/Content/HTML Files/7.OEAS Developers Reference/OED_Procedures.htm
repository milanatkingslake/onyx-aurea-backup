Procedures

The procedures collection is a container that holds all of the public stored procedure definitions for the Onyx Enterprise Database (OEDB).

A procedure entity is metadata describing a physical database stored procedure. It contains a collection of parameters as well as its own specific attributes. The procedure entity has two attributes, a procedure name and a validity flag.

The validity flag may be used by the Onyx Enterprise Dictionary Installer in a future release for upgrades. It is not implemented at this time.

Note: Only procedures defined as a procedure entity in the physical model can be used in a procedure map on a method step.
