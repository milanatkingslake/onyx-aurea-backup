extract (publishableObject)

extract method

The extract method of the publishableObject object filters and retrieves domain data using a database view. This method provides domain data retrieval when an object does not have a retrieveCollection method or when an existing retrieveCollection method does not filter the data as desired.

Remarks

This method appends "DP_VIEW" to the contents of the viewName parameter. For example, if the name of the view is REGION_EUROPE_DP_VIEW, specify REGION_EUROPE as the viewName parameter.

The publish method calls the extract method, which references the specified view. The publish method writes the view's domain data to an XML file in the OEP-Offline client file share. The name of the file is contained in the filename property.

Example

CREATE VIEW dbo.REGION_EUROPE_DP_VIEW

AS

SELECT    chRegionCode AS N'regionCode', chCountryCode AS N'countryCode',

                                         chRegionName AS N'regionName'

FROM      Region

WHERE     tiRecordStatus = 1 AND chCountryCode = 'BE' OR

                                 chCountryCode = 'DE' OR

                                 chCountryCode = 'FR'
