retrieveCollection method

The following table describes the behavior to expect for retrieveCollection methods.

Behavior based on the stored procedure return code value defines the entire result set. In this case, the return code column is listed as not applicable (N/A) because it is not included in the result set. For stored procedure return code values less than or equal to zero (<0), the result set must contain a column for each property defined in the OED for the given object. The columns in the result set must be returned in the order they are defined in the OED.  Because the middle tier depends on the order of columns in the result set, the column names and/or aliases are not critical.

Behavior only applies to the single row when the return code column is applicable.
