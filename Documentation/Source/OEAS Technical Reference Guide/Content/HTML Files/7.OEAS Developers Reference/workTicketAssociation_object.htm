workTicketAssociation object

The workTicketAssociation object enables users to insert and delete workTicketAssociation object instances. Instances of the workTicketAssociation object associate workTicket object instances with incident, company, and individual object instances, as well as other workTicket object instances. Work tickets are typically used to track customer support and product quality control issues.

The workTicketAssociation object has one key property, primaryId, that uniquely identifies object instances in the database. Once the object is created, the value of this property cannot be changed.

Method

Method Description

retrieveCollection

Returns the collection of object instances based on the search criteria

saveCollection

Saves a collection of workTicket object instances

Properties

Property Description

associationId

The primaryId of the object instance that is associated with a work ticket

associationType

A unique numerical value that identifies the type of object that is associated with the work ticket

insertBy

The ID of the Onyx user who initially saved the object instance

insertDate

The time and date on which the object instance was initially saved

primaryId

A unique value that identifies the workTicketAssociation object instance

updateBy

The Onyx user who last saved modifications to the object instance

updateDate

The time and date on which modifications to the object instance were last saved

workTicketId

The primaryId of the workTicket object instance that is associated with an individual, company, incident or another workTicket object instance
