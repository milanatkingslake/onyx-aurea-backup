retrieveListByGroup (productIndex)

retrieveListByGroup method

The retrieveListByGroup method of the productIndex object returns a list of all products of a given group.

retrieveList standard method information

Parameters

Parameter

Direction

Data type

Content type

Description

hierarchyId

in

i4

n/a

A number that identifies where a specified product group is in the tree hierarchy.

productActive

in

boolean

n/a

A value that indicates whether to return active, inactive, or both active and inactive products. The value of this parameter affects the selection of both products and literature products.

Possible values are:

null: return both active and inactive products and literature products.

1: return active products and literature products only.

0: return inactive products and literature products only.

literatureActive

in

boolean

n/a

A value that indicates whether to return active, inactive, or both active and inactive literature products. The value of this parameter affects the selection of literature products only.

Possible values are:

null: return both active and inactive literature products.

1: return active literature products only.

0: return inactive literature products only.

rowSet

out

object

all

An object that lists the products of a group.
