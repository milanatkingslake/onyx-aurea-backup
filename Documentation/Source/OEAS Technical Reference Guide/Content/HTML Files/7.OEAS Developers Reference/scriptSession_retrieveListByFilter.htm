retrieveListByFilter (scriptSession)

retrieveListByFilter method

The retrieveListByFilter method of the scriptSession object returns data about scriptSession objects. The data is returned as a rowSet object in the parametersReturn XML. The purpose of this method is to retrieve a list of scriptSession instances for one or more specified agents delivering scripts.

Remarks

Either the assignedToUserList or the assignedToGroupList parameter is required for this method to work. You can use additional parameters scriptIdList or statusIdList to further refine the search, but the status and script parameters cannot be used by themselves.

This method retrieves contact details based on the contactId of the scriptSession. Contact details include columnOrdinal 11 through 15 in the column definitions.
