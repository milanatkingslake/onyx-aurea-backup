retrieveListByPartnerAccess (customer)

retrieveListByPartnerAccess method

The retrieveListByPartnerAccess method of customer object retrieves the customers by partnerAccessIds.

Parameters

Parameter Direction Data type Content type Description

partnerAccessId

in

string

n/a

A list of the partnerAccess records to retrieve customer records

includeMask

in

string

n/a

Identifies whether the phone number mask is displayed on the output or not

rowSet

out

object

all

Contains the rowSet column definitions and the incident table rows matching with the owner and category criteria

Steps

Step data Step type Description

OnyxDataValidationStep.DataValidation

com

Validates the input parameters

OnyxDM.DataManager

com

Retrieves the information for customers
