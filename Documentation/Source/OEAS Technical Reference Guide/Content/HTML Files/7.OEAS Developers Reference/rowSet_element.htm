rowSet element
rowSet element

The rowSet element is returned within the parametersReturn XML when OEAS conducts complex search-type operations such as retrieveList methods. The rowSet element is constructed from the rowSet object, which provides a compressed way to send lists of data back to client applications. The rowSet object allows OEAS to dynamically define the list of data that is being returned.

The rowSet object is an object defined in OED (Onyx Enterprise Dictionary). You can view its definition using Object Designer. Its properties include a columnDefinition collection and a rows collection, but it has no methods. The rowSet object has no predefined collection of columnDefinition or row objects. With SQL generation, the columns returned for a particular retrieveList method are defined in OED. The only exception is when you mark the rowSet parameter as having a "dynamic" contentOut in the method definition. In this case, the rowset is determined by the stored procedure at runtime. The retrieveList methods must have the correct rowSet contentOut definition in the OED.  That is, the definition must be in agreement with what the stored procedure returns.

OEAS uses rowSet objects exclusively as output parameters in retrieveList methods.

Example

<parametersReturn>

   <rowSet objectType="rowSet" rowCount="3" columnCount="2">

      <columnDefinitions collection="1" objectType="columnDefinition">

         <columnDefinition>

            <columnHeading>Product Number</columnHeading>

            <columnName>chProductnumber</columnName>

            <columnOrdinal>1</columnOrdinal>

            <columnFlags>24</columnFlags>

            <columnSize>20</columnSize>

            <columnType>string</columnType>

            <columnPrecision>255</columnPrecision>

            <columnScale>255</columnScale>

         </columnDefinition>

         <columnDefinition>

            <columnHeading>Description</columnHeading>

            <columnName>vchDescription</columnName>

            <columnOrdinal>2</columnOrdinal>

            <columnFlags>8</columnFlags>

            <columnSize>50</columnSize>

            <columnType>string</columnType>

            <columnPrecision>255</columnPrecision>

            <columnScale>255</columnScale>

         </columnDefinition>

      </columnDefinitions>

      <rows collection="1" objectType="row">

         <row>

            <c0>USER-GD</c0>

            <c1>User Guide</c1>

         </row>

         <row>

            <c0>UPGRADE-PAK</c0>

            <c1>Upgrade Package</c1>

         </row>

         <row>

            <c0>ONLINE-REF</c0>

            <c1>Online Reference Guide</c1>

         </row>

      </rows>

   </rowSet>

</parametersReturn>
