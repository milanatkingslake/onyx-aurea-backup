workNoteHeader object

workNoteHeader object

The workNoteHeader object enables users to associate multiple workNoteDetail objects to a single workTicket object instance.

Note: The incident and task objects use the workNote object instead of the workNoteHeader and workNoteDetail objects.

Methods

Method

Description

retrieveCollection

Retrieves a collection of workNoteHeaders

insert

Saves a new workNoteHeader

Properties

Property

Description

description

A user input string that describes the object instance

insertBy

The name of the Onyx user who initially saved the workNoteHeader object

insertDate

The time and date on which the object was initially saved

ownerId

A unique value that identifies the parent object that owns the workNoteHeader object

ownerType

A unique numerical value that identifies the type of object that owns the workNoteHeader object

primaryId

A unique value that identifies the workNoteHeader object

updateBy

The Onyx user who last saved modifications to the object instance

updateDate

The time and date on which modifications to the object instance were last saved

workNoteDetails

A collection of workNoteDetail objects
