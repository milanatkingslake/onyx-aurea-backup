Supported date & time range

For data validation and UTC conversion purposes, the Query Engine should be passed date/time values within the ranges described below.

dateTime data type

Valid values for the dateTime data type range from:

1900-01-01 00:00:00

to

9999-12-30 23:59:59

Date range

The acceptable range for dates saved or retrieved by the Query Engine is 1900-01-01 to 9999-12-30, or:

January 1, 1900 to December 30, 9999

Time range

The range for times saved or retrieved by the Query Engine is accurate to the second:

hh:mm:ss
