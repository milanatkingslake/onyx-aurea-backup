Authenticate Method

The authenticate method of the externalUser object authenticates the logon credentials of the external user who is attempting to log on. This method returns the user access level information if authentication succeeds. If authentication fails, then it returns an error.

Parameters

Parameter Direction Data type Content type Description

userId

In

string

n/a

The userId of the user who is attempting to log on.

password

In

string

n/a

The password submitted by the user.

userAccessLevel

out

string

n/a

The userId returned if the logon attempt authenticates successfully.

Encrypt password

Remarks

The password is encrypted using standard OEAS encryption. It is then compared to the encrypted password stored in the table for that external user. If the password is correct, then a valid value for the userAccessLevel return parameter is provided. This userAccessLevel corresopnds with the userID to which the external user is mapped

External users are mapped to one of two user types, based on the selection used when creating the externalUser. Default values of userAccessLevel are:

  • If access level = 'Lead' then userAccessLevel = 'ExternalUserLead'
  • If access level = 'Customer' then userAccessLevel = 'ExternalUserCust'
