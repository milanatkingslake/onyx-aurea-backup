retrieveListMobileSearch

The retrieveListMobileSearch method of the Incident object searches the database by the input criteria and returns a list of incident records. This procedure is used by onyx mobile application.

Remarks

The retrieveListQuickSearch method is not case sensitive for the following input parameters:

  • secondaryId
  • firstName
  • lastName
  • companyName
  • description1

When you use either primaryId or secondaryId as a filter parameter, the retrieveListQuickSearch method ignores all other parameters that you pass to it.

This method supports wild card substitution in search criteria. For more information on wild card substitution, see the retrieveList method (standard behavior) topic.

This method supports paging functionality. For more information, see the Paging functionality.
