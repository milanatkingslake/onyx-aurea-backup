Welcome to Table Administration

You use the Table Administration tool to add, modify, and delete the data in certain tables in the Onyx Enterprise Database (OEDB).

For detailed instructions on using this tool, click to access the Help system.

 

Quick overview

Select a table in the navigation area of the Table Administration window. If the table selected has filters associated with it, you must select a filter.

To modify a recordâ€™s field displayed in the edit area, click the field you want to edit. The field's content is highlighted and any text that you type replaces the existing text in the field. If you want to add text without replacing the existing text, click the field again where you want to start adding new text. Certain fields in each of the tables are non-editable.

  • Click to add an empty row at the bottom of the table.
  • Select a row from the table and click to mark the row for deletion.
  • Click to save your changes to the OEDB.
