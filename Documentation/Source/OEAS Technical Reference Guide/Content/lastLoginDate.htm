lastLoginDate Property

The lastLoginDate property of the object individual retrieves the last login date from the user table to show it on the OEP Power Page.

Data type

datetime

Access

Read/write

Remarks

The context user  who  logged in   OEP will show last login  date on  the power  page.

Watch property

The watch property of the object individual retrieves watch information for logged in user.

Data type

Boolean

Access

Read/write

Remarks

 0 = not watched ,  1=  watched
