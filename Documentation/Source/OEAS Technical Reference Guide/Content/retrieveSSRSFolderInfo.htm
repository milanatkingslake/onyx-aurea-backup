retrieveSSRSFolderInfo Method

This method retrieves data from  SSRS reports by executing a series of method calls using the OTM. This is a custom method whose functionality will be implemented in a custom Step component called SSRSReports. This step component will retrieve the list of available reports and folders.

  • The webServiceURL paremeter contains the URL of the SSRS web service from which to retrieve the list of available reports and folders.
  • The folderNames parameter contains a semi colon separated list of folders from which the available reports and folders should be fetched.
  • If there are no reports available then the custom step component will not return any reports list.
