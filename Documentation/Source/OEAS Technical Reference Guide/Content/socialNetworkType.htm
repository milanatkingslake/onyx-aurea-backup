socialNetworkType Object

The socialNetworkType object contains information whether the social network integration is a link or a pull type of integration. This is a domain object which follows all domain business object restrictions. The socialNetworkType object has one key property, PrimaryId, that uniquely identifies the social media integration in the database. Once the object is created, the value of this property cannot be changed.

Methods

Method

Description
