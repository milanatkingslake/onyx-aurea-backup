Type

You can define the method call detail Type attribute for output method call details for a script method call. The Type attribute specifies whether the method call detail represents a single value or a list of values. The two valid values for this attribute are:

  • Scalar - A method call detail that can represent a single text value or a single ID/description pair.
  • List - A method call detail that can represent multiple text values or multiple ID/description pairs. A method call detail of type list is valid only for retrieveList and retrieveCollection method calls.

For input method call details for a script method call, “scalar” is the only valid value for this property.

For more information about how the ID/description pairs are defined, see ID Path and Description Path.
