Welcome to Campaign Administration

The Campaign Administration tool enables you to create, modify, activate or deactivate campaigns, tracking code types, and tracking codes.

For detailed instructions on using this tool, click to access the Help system.

Quick overview

  • Select an item in the navigation area to view or modify its attributes.
  • Click to expand an item within the navigation area.
  • Click to collapse an item within the navigation area.
  • Select an item in the navigation area, and click to create a new item of that type.
  • Select an item and click to mark it for deletion.
  • Select a tracking code and click to move it up one position in the tree.
  • Select a tracking code and click to move it down one position in the tree.
  • Click to save your changes.
