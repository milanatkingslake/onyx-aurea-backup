Welcome to Reference Table Administration

The Reference Table Administration tool is one of the tools that enables you to configure Onyx Enterprise Database fields.

For detailed instructions on using this tool, click to access the Help system.

Quick overview

  • Select an item in the navigation area to view or modify its attributes.
  • Click to expand an item within the navigation area.
  • Click to collapse an item within the navigation area.
  • Select a field or a lookup value in the navigation tree, and click to create a new item of that type.
  • Select an item and click to mark it for deletion.
  • Select a lookup value and click to move it up one position in the field drop down list in both the Reference Table Administration tool and the Onyx Employee Portal.
  • Select a lookup value and click to move it down one position in the field drop down list in both the Reference Table Administration tool and the Onyx Employee Portal.
  • Click to save your changes.
