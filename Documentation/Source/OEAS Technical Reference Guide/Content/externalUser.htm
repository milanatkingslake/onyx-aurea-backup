externalUser Object

The externalUser object provides a basic user management interface that can be used to grant and control access to a specific customer's information in .

Methods

Method Description

authenticate

Authenticates the logon credentials of the external user who is attempting to log on

delete

Deletes a specified externalUser object

insert

Inserts an externalUser object into the database

retrieve

Returns a specified externalUser object

retrieveCollection

Retrieves a collection of matching externalUser objects from the database

update

Modifies the external user's details.

Properties

Property Description

accessUserId

Contains a foreign key to the user record for the external user

inserrtBy

Contains the user ID of the user who initially saved the object

insertDate

Contains the date and time on which the object was inserted

languageCode

Contains the code for the preferred language for the external user

onyxTimestamp

Facilitates concurrency control with the database

ownerId

Contains the customer ID associated with the external user record

ownerType

Contains the type of customer for the external user

password

Contains the password for the external user

primaryId

A unique ID generated when the externalUser is saved

userId

Contains external user's user ID

updateBy

Contains the user ID of the user who last saved modifications to the object

updateDate

Contains the date and time on which the object was last updated
