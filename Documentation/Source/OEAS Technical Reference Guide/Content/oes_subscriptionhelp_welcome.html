Welcome to Subscription Designer

Subscription Designer enables you to:

  • View and configure subscription services.
  • View and configure subscribable objects, subscription relationships, and subscription rules.
  • View, configure, and publish domain data to the offline data store.

For detailed instructions on using this tool, click to access the Help system.

Quick overview

  • Select the Publishable Objects node to view a list of all of the publishable objects, select those objects you want to publish, and publish those records to the offline data store
  • Click to expand an item within the navigation area.
  • Click to collapse an item within the navigation area.
  • Select an item in the navigation area to view or modify its attributes.
  • Select an item in the navigation area and click to create a new item of that type.
  • Select an item and click to mark it for deletion.
  • Click to save your changes.
