retrieveByWorkNoteId

The retrieveByWorkNoteId method of the of the incident object searches the database by worknoteId dand returns a list of incident records. Owners include company and individual objects.

workNoteId (required)

in

String

n/a

out

all(out)

Contains rowSet column definitions and the incident table rows matching the owner and category criteria
