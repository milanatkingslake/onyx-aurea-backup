retrieveListByCustomer Method

The retrieveListByCustomer method of the task object searches the database by ownerId and ownerType and returns a list of task records. Task objects can be retrieved according to the incident object or customer object that owns them.

Remarks

The retrieveListByCustomer search is case sensitive.

This method retrieves only a limited list of data, and not always the entire list of data available. You can use the OEAS Configuration Utility to configure the maximum rows returned. For more information, see Paging functionality.
