Welcome to the Method Call Designer

The Method Call Designer enables you to add, modify, and delete method calls.

For detailed instructions on using this tool, click to access the Help system.

Quick overview

Select a method call type in the navigation area to view or modify the method calls of that type.

  • Click to expand an item within the navigation area.
  • Click to collapse an item within the navigation area.
  • Select a method call, or a method call type node, and click to add a new method call of that type. For example: To add a new Script method call, select the Script node and click .
  • Select a method call detail, or the Input or Output node of a method call, and click to add a new method call detail for the method call.
  • Select a method call or method call detail and click to mark it for deletion.
  • Select a method call detail and click to move it up one position in the order of the method call details for the method call.
  • Select a method call detail and click to move it down one position in the order of the method call details for the method call.
  • Click to save your changes.
