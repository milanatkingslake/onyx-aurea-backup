insertBy Property

The insertBy property of an object contains the user ID of the Onyx user who initially saves the object.

This is a generic topic that applies to all properties with the same name. The remarks section may contain details specific to a given object.

Data type

String

Access

Read-only; for more information,  see Read Only attribute.

Remarks

The insertBy property is set only once: when the object is first saved. Attempts to change this value are ignored. The user ID is an existing ID in the USERS table of the database and is obtained from the context XML when the record is inserted.
