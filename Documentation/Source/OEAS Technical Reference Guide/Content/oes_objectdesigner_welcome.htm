Welcome to the Object Designer

The Object Designer enables you to view, insert, update, delete, and otherwise modify the metadata that comprise the business objects stored in the Onyx Enterprise Dictionary (OED). The Object Designer is the only tool that Onyx supports for extending and customizing the business objects of the Onyx Enterprise Application Server.

The customization process usually involves more than simply making modifications to the OED using the Object Designer. In many cases, when you customize OEAS business objects, you must also write custom code (such as building your own step components) to take full advantage of the changes you made.

If you are new to this tool, preview the Getting Started section for helpful information.

Quick Overview:

  • Click to access the Help system.
  • Select an item in the navigation area to view or modify its attributes.
  • Click to expand an item within the navigation area.
  • Click to collapse an item within the navigation area.
  • Select an item, or the parent node of an item, and click to create a new item of that same type. (For example: To add a new object, select an existing object and click . To add a new method, open the object you want to add the method to, select the Methods node, or an existing method under the Methods node, and click .)
  • Select an existing object, property, method, parameter or step and click to create a new item of that type.
  • Select an item and click to mark it for deletion, or select an item marked for deletion and click to unmark it for deletion.
  • Select a property, method, parameter, or step and click to move it up one position in the tree.
  • Select a property, method, parameter, or step and click to move it down one position in the tree.
  • Click to save your changes to the working copy of the OED.
  • Click to validate the object data of the working copy of the OED.
  • Click to publish your changes, applying your changes in the working copy of the OED to the production copy of the OED.
