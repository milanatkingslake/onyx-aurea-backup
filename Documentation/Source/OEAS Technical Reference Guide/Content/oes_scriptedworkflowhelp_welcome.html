Welcome to Script Designer

The Script Designer enables you to create, modify, activate or deactivate script groups, scripts, script steps, or script step elements.

For detailed instructions on using the Script Designer, click to access the Help system.

Quick overview

To view or modify a script group, script, or script step, select it in the navigation area.

  • Click to expand an item within the navigation area.
  • Click to collapse an item within the navigation area.
  • Select an item in the navigation area and click to add a new element of the same type as that item, and on the same level as the item. You can do this in order to add:
  • A new script group or subgroup of a group
  • A new script
  • A new script step within a script
  • Select an item in the navigation area and click to mark it for deletion.
  • Click to save your changes.
