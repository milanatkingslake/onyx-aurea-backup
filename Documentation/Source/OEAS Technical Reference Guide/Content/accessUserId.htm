accessUserId Property

The accessUserId property of the object Exernaluser contains a foreign key to the user record for the external user.

Data type

String

Access

Read/Write

Remarks

The value for this property is an active user from the users table. By default, this is mapped to one of two users when the externalUser is created:

  • If access level = 'Lead' then userAccessLevel = 'ExternalUserLead'
  • If access level = 'Customer' then userAccessLevel = 'ExternalUserCust'
