Print Subsystem Logging

Log4Net is the Logger Service used for OPS Print SubSystem. Logger logs the information in <OPS installation folder>\Components\Logs\OPS_Onyx.log file.

The following code specifies the default configuration. The text in green specifies the level of logging which can be one of the following:

  • ALL
  • DEBUG
  • INFO
  • WARN
  • ERROR
  • FATAL
  • OFF

<configuration>

  <configSections>

    <section name="log4net" type="log4net.Config.Log4NetConfigurationSectionHandler, log4net" />

</configSections>

<log4net>

    <appender name="RollingFile" type="log4net.Appender.RollingFileAppender">

      <file value="Logs\OPS_Onyx.log"/>

      <lockingModel type="log4net.Appender.FileAppender+MinimalLock"/>

      <appendToFile value="true"/>

      <maximumFileSize value="500KB"/>

      <maxSizeRollBackups value="50"/>

      <layout type="log4net.Layout.PatternLayout">

        <IgnoresException value="False"/>

        <conversionPattern value="%newline%newlineDate: %date %newlineLogLevel: %level    %newlineMessage: %message%newline %newline%exception"/>

      </layout>

    </appender>

    <root>

      <level value="Error"/>

      <appender-ref ref="RollingFile"/>

    </root>

  </log4net>

</configuration>

For more information on Log4Net, refer to https://logging.apache.org/log4net/.

The same configuration holds good for OPS Print Template Test Utility as well.
