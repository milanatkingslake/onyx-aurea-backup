Using Advanced Subscription

This feature is available to OEP-Offline users who have been designated as Advanced Subscribers by the System Administrator.

Advanced Subscription allows designated users to subscribe data to other OEP-Offline users. Advanced subscription can be used in all feature areas that are enabled with subscription functionality.

To subscribe or unsubscribe items for another user:

  1. Select the data that you want to subscribe for another user by completing one of the following steps:
  • Click on the toolbar of the PowerPage or the Edit page to subscribe or unsubscribe a company or individual.
  • Click on the toolbar to subscribe or unsubscribe an incident or task, or on the Forecast and Quote toolbar to subscribe or unsubscribe a quote.
  • Click on the header or toolbar to subscribe or unsubscribe items in Advanced Search and Task Manager result lists. You must select the items that you want to subscribe from the list.
  1. In the Subscribed User menu in the Advanced Subscription dialog, select the user for whom you want to subscribe or unsubscribe the items.
  2. In the Available Services section, select the OEP-Offline check box to subscribe, or clear it to unsubscribe.
  3. Click to save the subscribed item. You can subscribe the same items to other users if desired.
  4. Click to save and close the item when you have finished subscribing data to OEP-Offline users.
