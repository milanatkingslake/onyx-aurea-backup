About List Manager

List Manager is a tool used for the bulk processing of OEP records. List Manager provides the ability to generate lists of customers (companies or individuals) or customer-related information. Using the available processes you can perform bulk actions on these lists, saving the time and effort of having to process records one at a time.

From the List Manager window you build searches based on customer search criteria. After generating a result list and configuring the columns of information returned by the result list, you select a process to apply to multiple records. Once a search and its results are configured to your liking, save it and reuse it as needed.

To view the List Manager window, click and then click List Manager.
