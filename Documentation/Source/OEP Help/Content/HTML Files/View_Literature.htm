Viewing Literature Order Information

Click Literature within the Customer submenu on the Navigation bar to view the literature orders for the active customer record. The list of literature orders for the customer appears in the Data Area frame. From here you can add and remove orders in the list, as well as access existing orders to edit or view the details of the order.

Managing literature information

The literature order information window appears any time you edit an existing order or add a new order to the list. For more information about this window, see Managing literature orders.

Deleting literature orders
