Work Ticket Search

Using the Work Ticket tab on the Search pane, you can perform work ticket searches.

To perform a work ticket search:

  1. Enter search delimiters in the QuickSearch text box and click .

The Search window appears.

  1. Click the Work Ticket tab.
  2. Specify search criteria.
  3. Click on the Search toolbar.

The results of the search appear below the search form.

  1. Click a work ticket record in the results list to open that work ticket in the work ticket edit window.

Use the supported search delimiters to broaden searches on the description field.

When you type a value in the Work Ticket ID text box, all other search criteria are ignored.

When you execute the work ticket search, if only one work ticket is found that matches the search criteria, OEP automatically opens the work ticket edit window for that work ticket.
