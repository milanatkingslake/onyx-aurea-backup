Linking a Primary Contact

Each task record has a single primary contact. By default, a task is assigned the primary contact of the parent incident. You can select a different contact if necessary.

To select a primary contact for a task:

  1. Click next to the Contact field.

A search window appears.

  1. Enter criteria, and click to start the search.
  2. Click a name from the results list to assign as the primary contact for the task.
