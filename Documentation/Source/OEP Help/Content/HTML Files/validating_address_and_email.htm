Validating Address and Email

This topic explains the requirements to validate address and email.

UI Permissions for OES

  • UI:OEP.address.validate
  • UI:OEP.email.validate

System Parameters

  • QASAddressTypesToValidate. Type the respective Address Field ID. Use the semi-colon mark (;) as separators if you have multiple address fields.
  • QASAllowSuggestions. Type Y or N.
  • QASEmailTypesToValidate. Type the respective Address Field ID. Use the semi-colon mark (;) as separators if you have multiple email fields.
  • QASNoOfRecordsPerPage. Type the number of records to be displayed per page in the Suggestion Box.
  • QASValidateAddress. Type Y or N.
  • QASValidateEmail. Type Y or N.

Config File Changesfor OEP

Enter the QASUser and QASPassword values in the QAS Section of OnyxGatewayWindowsServices.exe config file in the OnyxGatewaySerevices folder.

After updating the OnyxGatewayWindowsServices.exe config file with QASUser and QASPassword values, you must restart OGS services.

Specific Method calls have to be explicitly designed/configured using Method Call Designer. These method calls are used in script work-flows to perform address and email data changes.
Without the Method Calls, Address and Email data changes are not reflected in Script Work Flow.

The QASAllowSuggestions parameter is only applicable for address validation and not for email validation.
