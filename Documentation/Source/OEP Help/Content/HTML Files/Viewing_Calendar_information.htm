Viewing Calendar Information

You can view Calendar and appointment entries from the following areas in OEP:

From the Navigation bar

To view Outlook directly from the Navigation bar, click .

The view opened from the Navigation bar or Home Page is based on the Outlook profile currently configured on your local machine. OEP Calendar cannot be configured to display another user's calendar in the OEP environment, nor can it automatically select one profile over another when multiple profiles are configured. Outlook views are not stored in OEP, so any configurations made to the calendar must be done in the native Outlook application.

If Outlook is configured to prompt for a password on your local machine, it prompts for a password from the OEP Home Page as well. If Outlook is not currently running on your local machine, you may be asked for a user name and password before accessing it from OEP.

From the Home Page

To add Outlook to the Home Page layout, click (Select Sections). Select Calendar from the list on the left and click >>. Move Up and Move Down positions the Calendar on the Home Page.

The view launched from the Navigation bar or Home Page is based on the Outlook profile currently configured on your local machine. OEP Calendar cannot be configured to display another user's calendar in the OEP environment, nor can it automatically select one profile over another when multiple profiles are configured. Outlook views are not stored in OEP, so any configurations made to the Calendar must be done in the native Outlook application.

If Outlook is configured to prompt for a password on your local machine, it prompts for a password from the OEP Home Page as well. If Outlook is not currently running on your local machine, you may be asked for a user name and password before accessing it from OEP.

From the Appointments tab

The Appointments tab is located on the PowerPage and on the incident information page.

To view an appointment, locate and double-click the appointment you want. The Appointment page opens.  

To view calendar information for the active customer or incident, select the Appointments tab. The icons in the tab's toolbar enable you to view the associated appointments of the active customer or incident by day, week, or month.  

The icon updates the calendar to display new appointments and cancellations.

The icon opens the Outlook Appointment page to schedule a new appointment.

From the History tab

Appointments are listed in the history tab of the associated customer's PowerPage or incident's information page. Appointments are listed chronologically along with other recent interactions associated with the customer or incident.

To view an appointment, locate and click the appointment you want. The Appointment page opens.
