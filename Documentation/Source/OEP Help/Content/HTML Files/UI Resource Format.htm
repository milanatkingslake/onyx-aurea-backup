UI Resource Format 

When you create UI resource IDs for Navigator, we recommend that you follow the format: UI:OEP.<Entity_name.UI Element.Resource_Permission>, where:

  • Entity_name is the name of the search type that the field is added to.
  • UI Element is the UI resource name that you want to grant permissions for. This value is optional for some entity level UI resources.
  • Resource_Permission is the permission for the resource.
