General Preferences

Click the General tab to view the preference settings for the overall operation of OEP. Enter values in the text boxes and drop down lists.

General settings

The following table lists and describes the options available from the General tab.

Preference Description

Country

Select the country to use in the primary address when you create a new customer record.

Language

Select the language that appears in OEP.

Default Profile

Select the OEP profile that you use most often. If you have access to more than one OEP profile, OEP can prompt you to select an alternative to your default profile every time you log on to OEP.

Startup View

Select the view that you want to appear every time you log on to OEP.

Default Company Tab

Select the tab in the bottom half of the PowerPage that you want to have active each time  a company loads.

Default Individual Tab

Select the tab in the bottom half of the PowerPage that you want to have active each time an individual loads.

Default Search Tab

Select the tab that you want to have active when you open the general search window.

Default Task Manager Criteria Tab

Select the tab that you want to have active when you open Task Manager.

List Manager Maximum Result Rows

Type a value to limit the maximum number of records that OEP will return when you execute a List Manager query search. The default (and maximum recommended) value is 1000.

Expand Groups with Selected Users

Select this option to expand user groups to display all selected users when you load a feature that contains the user list. Leaving this option unselected leaves groups unexpanded, but identifies them as containing selected users.

Display Notification for New Messages

Click Yes to configure OEP to display a dialog box notifying you of new Messenger messages.

Display Preview Pane in Messenger Window

Select Yes to have the preview pane appear in the Messenger window when it loads. Select No to keep the preview pane hidden.

Default Forecast/Quote Currency

Select a currency that will be used as the default for new forecasts and quotes. Leave this preference blank to use the system's default currency.

Default Worknotes View

Select the layout in which you want worknotes to use when they appear in a record window.

Password changes

If you are required to type a password each time you connect to OEP, you can change the password from the General preferences tab.

To change your password:

  1. Click Change Password.

The Change Password dialog box appears.

  1. Type your old password in the Old Password box.
  2. Type your new password in the New Password and Confirm Password boxes.
  3. Click OK to save your new password.
