About Task Manager

Task Manager is a tool you can use to search for and administer the tasks, incidents, and work tickets assigned to users or user groups. Using Task Manager, you can perform batch updates to multiple incidents, tasks, or work tickets at one time.

The Task Manager screen is divided into two parts: the top half manages the search function, and the bottom half contains the search results.

Information for commonly used searches can be saved in items known as filters. A filter contains a set of search parameters that can be reused when needed.

Task Manager is accessible at any time by clicking on the Navigation bar.
