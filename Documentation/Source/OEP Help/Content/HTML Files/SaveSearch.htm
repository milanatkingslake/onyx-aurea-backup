Saving a Search

A search is saved as a combination of the selected search criteria fields and result grid columns. To access search results from Onyx Mobile on your iPhone, iPad or Android, you can save a search as a Mobile bookmark. For information about mobile bookmarks, see Mobile Bookmarks.

To save a search

  1. In the Search Criteria window, enter values for each search criteria.

Click , then click Result Columns, and in the Search Result Field Selection and Administration window, select the columns, sort order, and maximum rows to display, and page size for the search.

  1. On the Search Criteria window click . The Query Explorer window appears.

    2. You can also click after performing an ad-hoc search to save the search criteria and result grid columns to a filter.

  2. Enter a Query Name for your search.
  3. Select the Bookmark check box to bookmark the filter for easy access from the Header bar.
  4. Select the Mobile Bookmark check box to add the filter to Onyx Mobile. When you log on to Onyx, the filter is displayed on the Home Page. The filter name is same as the name of the saved Navigator query.
  5. Select the Homepage Query check box to add the filter to the Home page.
  6. Click to save the query.

When click , the query is saved under Personal Queries section. If you want to make a query as a public query, drag the query you created from Personal Queries section and drop it to the Public Queries section.

Depending on the filter's search type, the filter is available in the Bookmarks section under the , , , or buttons.

 

If you change the search criteria for a saved filter and save your changes, the Search Criteria panel is refreshed with the new search criteria and any previously entered values for the search criteria fields are cleared. To view the values you previously entered, reload the saved filter.

Right-click a query to perform the following actions:

  • Edit Query
  • Clone Query
  • Rename Query
  • Delete Query

To perform these actions, you need to have the UI:OEP.Navigator.PublicQuery.Manage permission.
