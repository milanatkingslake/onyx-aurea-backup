Using Subscription Review

Subscription Review is the primary place for viewing data to which you are subscribed. You can also view all data subscribed to other users if you are an advanced subscriber. Subscription review filters only data that is subscribed. To search for items that you want to subscribe to, you must use Search in OEP.

Similar to Advanced Search, Subscription Review provides filter criteria to help identify the type of items that you want to review, and provides a list of results that match your criteria.

All OEP-Offline users, and select OEP-Offline users called Advanced Subscribers, can use Subscription Review to manage the subscriptions of others.

Managing subscriptions

Subscription review enables you to view and manage subscriptions for yourself, or, if you are a advanced subscriber, on behalf of other users.

Subscription Review does not always display all subscribed data for a user because cascaded subscriptions do not automatically display. Each item with cascaded relationships must complete a full synchronization before its cascaded items will display in Subscription Review.

Viewing items subscribed to a user:

From Subscription on the Navigation bar:

  1. Click Review within the Subscription submenu. The Subscription Review page appears.
  2. Enter criteria in the search fields. The following list explains the search criteria:
  • Subscribed User is a required field. By default, your own user name appears in the list. Advanced Subscribers can select other users from the list.
  • Item Type is a required field. Select the type of item that you want to search for, or use the default.
  • Subscribed Service is a required field and can be set to OEP-Offline.
  • Subscribed is an optional field that allows you to search on the day or timeframe on which items were first subscribed.
  • Updated is an optional field that allows you to search on the day or timeframe on which items were most recently updated.
  1. Click . The subscribed items that meet your filter criteria appear in a list.
  2. Close Subscription Review.
Unsubscribing items from other users

To unsubscribe items from yourself or other users:

  1. Select Subscription on the Navigation bar and choose the Review submenu.
  2. On the Subscription Review dialog, choose the user whose subscribed data you want to manage.
  3. In the Item Type dialog, choose the type of item that you want to view.
  4. Select a day or date range if you want to filter on the date of initial subscription.
  5. Select a day or date range if you want to filter on the date of the most recent update.
  6. From the list, select the items that you want to unsubscribe for this user.
  7. Click . Click OK on the Confirmation dialog to proceed with the unsubscription.
  8. Close Subscription Review.

When you unsubscribe from an item, you may also unsubscribe additional, related items because of cascaded subscription rules established by your System Administrator.
