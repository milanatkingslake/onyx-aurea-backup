Search Delimiters

OEP supports two delimiters that can be used to expand or limit the text criteria of a search. The '%' character acts as a wild card specifying any possible character or list of characters. The '[]' characters contain a list of possible characters to restrict the search. The delimiters can be combined, although the '%' can only be placed at either end of a search string. All searches performed in OEP act as if a '%' wild card has been appended to the search strings.

You can use these delimiters in any search field that contains standard text. The delimiters are supported in QUICK! Search, and List Manager, but not in OEP user selection.

The following table shows some example uses of the delimiters when used in search criteria.

String Result

sm[i/y]th

Returns records that begin with 'smith' or 'smyth'

%smith

Returns records that contain 'smith' anywhere in the searched field

bl[a/o]cksm[i/y]th

Returns records that begin with 'blocksmith', 'blacksmith', 'blocksmyth', or 'blacksmyth'

[[]

Returns records that contain the '[' character

[%]

Returns records that contain the '%' character
