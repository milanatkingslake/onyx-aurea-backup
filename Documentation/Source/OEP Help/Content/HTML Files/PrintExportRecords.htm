Printing and Exporting Records

You can print the records displayed in the Search Results window to a printer, or you can export them to a CSV or Excel file.

To print records

  1. In the Search Results window, click . The Print Results window appears where you can preview the information before printing.
  2. In the Print window, select the printer, set printer preferences, and click Print.

To export records

  1. In the Search Results window, click , and then select Export to Excel or Export to CSV.
  2. In the Save As window, select the file location, type the file name and click Save.

Grouping and Filtering Records

To group the fields, in the Search Results window, click the arrow head on the column header, and click Group by this field.

To ungroup the fileds, in the Search Results window, click the arrow head on the column header, and clear the Show in groupscheckbox.
