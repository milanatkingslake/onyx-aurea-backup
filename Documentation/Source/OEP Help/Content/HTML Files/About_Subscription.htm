About Subscription

Subscription enables you to take existing OEP data offline so that you can access that data while you are not connected to the network. To take existing data offline, you must first subscribe to the data in OEP. Then, after you have synchronized with the OEP-Offline subscription service you can work with the data while you are disconnected from the network.

Offline subscription is available only to OEP users who also have permission to use OEP-Offline.

Subscription enables the following:

  • You can select specific OEP data to use in Outlook via OEP-Offline.
  • Certain OEP end-users (advanced subscribers) can select specific OEP data for you to use in OEP-Offline.

Subscription functionality is supported in the following areas:

  • Incidents
  • Tasks
  • Customers
  • Quotes
  • Lists in Task Manager
  • Advanced Search result lists

For users with subscription privileges, feature areas that support subscription display subscription icons:

  • Click to subscribe to items
  • Click to unsubscribe from subscribed items
  • Click   to subscribe items for other users if you have permissions as an advanced subscriber
  • All items that are subscribed to you display in the toolbar or header area

Remarks

Because many OEP items have close relationships with other data, subscriptions are cascaded to include subscription to related items when you subscribe to items in OEP. Cascading subscription rules are defined by your system administrator. It is important to note that when you subscribe to a set of items in OEP, you will actually have access to additional, related items when you view them in Subscription Review or OEP-Offline. Contact your system administrator for information on the cascading subscription rules that are used in your environment.
