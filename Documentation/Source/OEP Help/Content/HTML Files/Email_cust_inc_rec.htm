Emailing Customer and Incident Records

Customer and incident summary information can be added to emails as HTML attachments. Recipients can then view the information without having to access OEP.

To email a customer or incident record:

  1. From the PowerPage Details tab, click to open a new browser window that contains the active customer's summary information.

OR

  1. From the incident information window, click to open a new browser window that contains the active record's summary information.
  2. From the File menu, select Send and then Page by E-mail.

A blank email message appears with the customer or incident information attached in an HTML file. Complete the email as needed and send to the recipient.

This function is not integrated with the OEP Email feature.
