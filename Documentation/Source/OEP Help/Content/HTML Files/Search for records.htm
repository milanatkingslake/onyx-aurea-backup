Searching for Records

Using the Search Criteria window in Navigator, you can specify the search type and search criteria on which to base your search. The records that fulfill the specified criteria are displayed in the Search Results window.

The Search Criteria window provides a flexible way to:

  • Determine the criteria on which to base the search.
  • Enter parameters for each search criterion and run a search to view specific information.
  • Save and bookmark personal and public queries, and display them on the Home page or on Onyx Mobile.
