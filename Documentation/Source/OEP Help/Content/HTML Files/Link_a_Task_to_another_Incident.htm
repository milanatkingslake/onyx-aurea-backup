Managing Tasks

To view the list of tasks for the current customer or incident, click the Tasks tab.

You can sort the list by any of its columns. Click a column heading to sort the list by the contents of that column. To reverse sort the list, click the column heading a second time.

To add a task, click . See Managing task information for more information on editing tasks.

To delete one or more tasks, check the box in the far left column for each task you want to delete. Click to remove the tasks from the list.
