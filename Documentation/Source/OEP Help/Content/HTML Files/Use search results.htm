Viewing and Modifying Search Results

The Search Results window displays the results of a search based on the criteria you entered. From the Search Results window, you can select which fields to see in the Search Results grid, and you can select multiple rows to update multiple records simultaneously. You can also use this window to merge, print, and export records.

You can increase or decrease the size of the Search Criteria pane or the Search Results pane in the Search Results window or Detail Screen. Click the separator between the two panes to enable the drag bar. Click and pull the drag bar to re size the pane.
