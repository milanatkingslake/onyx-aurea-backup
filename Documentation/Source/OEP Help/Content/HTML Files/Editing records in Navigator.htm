Editing Records in Navigator

Editing in the Navigator enables you to quickly edit a record through the Detail screen.

Inline editing in Navigator enables you to quickly edit a record in the Navigator result grid. The columns available for editing depend on the search type selected.

Inline editing is UI resource controlled and therefore you must have appropriate permissions. For more details, contact your System Administrator.

  • You can edit only one record at a time.
  • You must save or cancel your changes before moving out of a record that you are editing and before refreshing the result grid.
  • You can edit the same record multiple times if desired.

Inline editing is not available for the search types Email, Appointments, Scripts, and Forecast.

Perform inline editing

To edit a record:

  1. Select a record in the result grid, right-click to activate the Action Menu, and click Edit Inline.

The focus moves to the first editable column in the Header section of the record.

  1. Edit the values in the desired columns for the selected record. Use the TAB key to move between editable columns of the selected record.
  2. To revert the changes you made to the record, click the Cancel button at the beginning of the row being edited. You can revert changes only before you click Save.
  3. To save your changes, click the Save button at the beginning of the row being edited.
  4. After you have saved your changes, the Save and Cancel buttons at the beginning of the row become unavailable. The saved record is highlighted in the result grid.

If an error occurs while saving, a message appears explaining the problem. The changes you made to the record are retained. You can make additional changes to fix the error and save the record.
