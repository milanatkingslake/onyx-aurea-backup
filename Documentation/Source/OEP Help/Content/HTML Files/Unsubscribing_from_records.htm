Unsubscribing from data

If you find that you no longer need items that you have subscribed to, you can unsubscribe them. After you unsubscribe an item, it is no longer available in OEP-Offline.

To unsubscribe from an item:

 Click on the PowerPage or Edit page toolbar for a company or individual.

 Open an incident, task, or quote and click on the toolbar.

To unsubscribe from multiple items at once:

  1. Run search in either Advanced Search or Task Manager.
  2. Select the items that you want to unsubscribe from the result list.
  3. Click on the toolbar.

To unsubscribe items from other users:

 If you are an advanced subscriber, you can use Subscription Review to unsubscribe one or multiple items from other users.

Unsubscribing data that you no longer need can decrease synchronization time because it prevents unnecessary data from synchronizing between OEP and OEP-Offline.
