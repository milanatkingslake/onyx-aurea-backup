Working with Saved Searches

Commonly used searches can be saved and recalled as needed. Searches can be saved for personal use, or marked as public to be made available to other OEP users.

Search administration is performed through the Search dialog box. This box lists the available public and private search in separate lists.

To save a search:

  1. Click .

The Search dialog box appears.

  1. Type a name for the query in the Select a Search from the Lists Below box.

OR

  1. To replace an existing search, click a name in the public or private list.
  2. To save the search as public and to make it available to all OEP users, check the Save as Public List box.
  3. Click to save the search and close the dialog box.

To load a saved search:

  1. Click .

The Search dialog box appears.

  1. Click a saved search from either the public or private lists.
  2. Click to load the search and close the dialog box.

To delete a search:

  1. Click .

The Search dialog box appears.

  1. Click a search from either the public or private lists.
  2. Click to delete the search and close the dialog box.
